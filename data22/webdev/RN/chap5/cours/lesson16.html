Bottom Tab & Drawer Navigation

Navigation mobile professionnelle avec React Navigation

Onglets • Menu latéral • Combinaison de navigateurs

Niveau: Intermédiaire Durée: 55 min

🎯 Objectifs de la leçon

  • Créer une navigation par onglets avec Bottom Tab Navigator
  • Utiliser Drawer Navigator pour un menu latéral personnalisable
  • Personnaliser l'apparence des onglets (icônes, couleurs, badges)
  • Combiner plusieurs navigateurs (Stack + Tabs / Stack + Drawer)
À la fin, vous saurez créer des applications avec une navigation professionnelle et fluide !

📱 Bottom Tab Navigator : Création

import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import { NavigationContainer } from '@react-navigation/native';

import HomeScreen from './screens/HomeScreen';
import ProfileScreen from './screens/ProfileScreen';
import SettingsScreen from './screens/SettingsScreen';

const Tab = createBottomTabNavigator();

function App() {
  return (
    <NavigationContainer>
      <Tab.Navigator>
        <Tab.Screen name="Accueil" component={HomeScreen} />
        <Tab.Screen name="Profil" component={ProfileScreen} />
        <Tab.Screen name="Paramètres" component={SettingsScreen} />
      </Tab.Navigator>
    </NavigationContainer>
  );
}

Résultat : 3 onglets en bas d'écran permettant de naviguer entre les écrans.

🎨 Personnaliser les onglets (icônes, couleurs)

import { Ionicons } from '@expo/vector-icons';

<Tab.Navigator
  screenOptions={({ route }) => ({
    tabBarIcon: ({ focused, color, size }) => {
      let iconName;
      if (route.name === 'Accueil') {
        iconName = focused ? 'home' : 'home-outline';
      } else if (route.name === 'Profil') {
        iconName = focused ? 'person' : 'person-outline';
      } else if (route.name === 'Paramètres') {
        iconName = focused ? 'settings' : 'settings-outline';
      }
      return <Ionicons name={iconName} size={size} color={color} />;
    },
    tabBarActiveTintColor: '#f59e0b',
    tabBarInactiveTintColor: 'gray',
    tabBarStyle: { backgroundColor: '#fff', paddingBottom: 5, height: 60 }
  })}
>
  <Tab.Screen name="Accueil" component={HomeScreen} />
  <Tab.Screen name="Profil" component={ProfileScreen} />
  <Tab.Screen name="Paramètres" component={SettingsScreen} />
</Tab.Navigator>

Options disponibles : `tabBarIcon`, `tabBarActiveTintColor`, `tabBarBadge`, `tabBarLabel`...

📂 Drawer Navigator (Menu latéral)

import { createDrawerNavigator } from '@react-navigation/drawer';

const Drawer = createDrawerNavigator();

function App() {
  return (
    <NavigationContainer>
      <Drawer.Navigator
        screenOptions={{
          drawerStyle: { width: 240 },
          drawerActiveTintColor: '#f59e0b',
          drawerInactiveTintColor: 'gray'
        }}
      >
        <Drawer.Screen name="Accueil" component={HomeScreen} />
        <Drawer.Screen name="Profil" component={ProfileScreen} />
        <Drawer.Screen name="Paramètres" component={SettingsScreen} />
      </Drawer.Navigator>
    </NavigationContainer>
  );
}

Le menu latéral s'ouvre en glissant depuis le bord gauche ou via `navigation.openDrawer()`.

🚀 Navigation programmatique

import { useNavigation } from '@react-navigation/native';

function HomeScreen() {
  const navigation = useNavigation();

  return (
    <View>
      <Button title="Aller au profil" onPress={() => navigation.navigate('Profil')} />
      <Button title="Ouvrir le drawer" onPress={() => navigation.openDrawer()} />
      <Button title="Fermer le drawer" onPress={() => navigation.closeDrawer()} />
      <Button title="Basculer drawer" onPress={() => navigation.toggleDrawer()} />
      <Button title="Retour arrière" onPress={() => navigation.goBack()} />
    </View>
  );
}

Astuce : `navigation.navigate('RouteName')` fonctionne même depuis un écran dans un Stack !

🔔 Badges et notifications sur les onglets

import { useBadgeStore } from '../stores/badgeStore';

function AppTabs() {
  const unreadCount = useBadgeStore(state => state.unreadCount);

  return (
    <Tab.Navigator>
      <Tab.Screen
        name="Messages"
        component={MessagesScreen}
        options={{
          tabBarBadge: unreadCount > 0 ? unreadCount : undefined,
          tabBarBadgeStyle: { backgroundColor: '#ef4444' }
        }}
      />
      <Tab.Screen name="Accueil" component={HomeScreen} />
    </Tab.Navigator>
  );
}

⚖️ Stack vs Tabs vs Drawer

NavigateurUtilisationExemple
StackNavigation hiérarchique (écrans qui s'empilent)Feed → Détail article → Commentaires
Bottom TabsSections principales toujours accessiblesAccueil, Recherche, Profil, Notifications
DrawerMenu secondaire / paramètres / optionsMon compte, Historique, Aide, Déconnexion

Bonnes pratiques : Combinez Stack + Tabs pour les écrans principaux, Drawer pour les options secondaires.

📌 Ce qu'il faut retenir

Bottom Tabs

  • Navigation principale toujours visible
  • Personnalisation avec icônes, couleurs, badges
  • Idéal pour les sections majeures

Drawer

  • Menu latéral accessible par glissement
  • Parfait pour paramètres et options secondaires
  • Personnalisable avec en-tête utilisateur
