📦 Qu'est-ce que AsyncStorage ?

AsyncStorage est une API de stockage clé-valeur, non chiffrée, asynchrone et persistante pour React Native.

Similaire à localStorage sur le web mais asynchrone

Capacité : Jusqu'à 6 Mo (généralement)

Cas d'usage : Thème sombre/clair, langue, token utilisateur, paramètres

⚠️ Attention : AsyncStorage n'est pas sécurisé pour les données sensibles comme les mots de passe ! Utilisez plutôt expo-secure-store pour cela.

🛠️ Installation

Avec Expo :

npx expo install @react-native-async-storage/async-storage

Avec React Native CLI :

npm install @react-native-async-storage/async-storage

Après installation, n'oubliez pas de relancer votre application !

🔑 Méthodes principales

setItem(key, value)

Stocke une donnée (value doit être une string)

getItem(key)

Récupère une donnée (retourne une Promise)

removeItem(key)

Supprime une donnée spécifique

clear()

Supprime toutes les données du stockage

💾 Stocker des données

import AsyncStorage from '@react-native-async-storage/async-storage';

// Stocker une chaîne simple
const storeData = async (key, value) => {
  try {
    await AsyncStorage.setItem(key, value);
    console.log('Donnée sauvegardée !');
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Stocker un objet (JSON)
const storeObject = async (key, objet) => {
  try {
    const jsonValue = JSON.stringify(objet);
    await AsyncStorage.setItem(key, jsonValue);
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Exemple d'utilisation
storeData('userName', 'Jean Dupont');
storeObject('userPrefs', { theme: 'dark', lang: 'fr' });

📖 Récupérer des données

// Récupérer une chaîne
const getData = async (key) => {
  try {
    const value = await AsyncStorage.getItem(key);
    if (value !== null) {
      console.log('Donnée récupérée:', value);
      return value;
    }
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Récupérer un objet JSON
const getObject = async (key) => {
  try {
    const jsonValue = await AsyncStorage.getItem(key);
    return jsonValue != null ? JSON.parse(jsonValue) : null;
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Exemple d'affichage
const userName = await getData('userName');
const prefs = await getObject('userPrefs');
console.log(`Bonjour ${userName}, thème: ${prefs.theme}`);

🗑️ Supprimer des données

// Supprimer une clé spécifique
const removeData = async (key) => {
  try {
    await AsyncStorage.removeItem(key);
    console.log(`Donnée ${key} supprimée`);
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Tout effacer (attention !)
const clearAll = async () => {
  try {
    await AsyncStorage.clear();
    console.log('Toutes les données ont été effacées');
  } catch (error) {
    console.error('Erreur:', error);
  }
};

// Supprimer plusieurs clés
const multiRemove = async (keys) => {
  try {
    await AsyncStorage.multiRemove(keys);
    console.log('Plusieurs données supprimées');
  } catch (error) {
    console.error('Erreur:', error);
  }
};

multiRemove(['userName', 'userPrefs', 'token']);

✅ Exemple concret : Todo List persistante

import AsyncStorage from '@react-native-async-storage/async-storage';
import React, { useState, useEffect } from 'react';
import { View, Text, TextInput, Button, FlatList } from 'react-native';

const TODOS_KEY = '@todos';

function TodoApp() {
  const [todos, setTodos] = useState([]);
  const [input, setInput] = useState('');

  // Charger les todos au démarrage
  useEffect(() => {
    loadTodos();
  }, []);

  // Sauvegarder les todos à chaque modification
  useEffect(() => {
    saveTodos(todos);
  }, [todos]);

  const loadTodos = async () => {
    try {
      const stored = await AsyncStorage.getItem(TODOS_KEY);
      if (stored) setTodos(JSON.parse(stored));
    } catch (error) {
      console.error('Erreur chargement:', error);
    }
  };

  const saveTodos = async (newTodos) => {
    try {
      await AsyncStorage.setItem(TODOS_KEY, JSON.stringify(newTodos));
    } catch (error) {
      console.error('Erreur sauvegarde:', error);
    }
  };

  const addTodo = () => {
    if (input.trim()) {
      setTodos([...todos, { id: Date.now(), text: input }]);
      setInput('');
    }
  };

  // Rendu du composant...
}

✨ Bonnes pratiques

Utilisez des clés nommées avec préfixe : @app_theme, @user_token pour éviter les collisions

Toujours utiliser try/catch car les opérations sont asynchrones

Stringifier les objets avant stockage avec JSON.stringify()

Parser les données à la récupération avec JSON.parse()

Ne stockez pas de données sensibles (mots de passe, infos bancaires)
