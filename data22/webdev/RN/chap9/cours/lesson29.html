📐 La fonction interpolate()

interpolate() permet de mapper une plage de valeurs d'entrée vers une plage de sortie (couleurs, angles, échelles).

const translateX = animatedValue.interpolate({
  inputRange: [0, 1],
  outputRange: [0, 100]  // de 0 à 100px
});

const bgColor = animatedValue.interpolate({
  inputRange: [0, 1],
  outputRange: ['rgb(255,99,71)', 'rgb(34,197,94)']
});
Exemples : rotation de 0° à 360°, opacité, échelle 0.5 → 1.5
Cas d'usage : barre de progression, animation de fondu au scroll

🔄 parallel / sequence / stagger

🔁 Animated.parallel([anim1, anim2])

Lance plusieurs animations en même temps.

➡️ Animated.sequence([anim1, anim2])

Exécute les animations l'une après l'autre.

⏱️ Animated.stagger(100, [anim1, anim2])

Démarre chaque animation avec un délai décalé (effet cascade).

Animated.parallel([
  Animated.timing(opacityAnim, { toValue: 1, duration: 500 }),
  Animated.spring(scaleAnim, { toValue: 1.2, friction: 2 })
]).start();

📜 Animations liées au défilement

On utilise Animated.ScrollView et onScroll avec Animated.event pour lier la position de défilement à une valeur animée.

const scrollY = new Animated.Value(0);

<Animated.ScrollView
  onScroll={Animated.event(
    [{ nativeEvent: { contentOffset: { y: scrollY } } }],
    { useNativeDriver: true }
  )}
>
  <Animated.View style={{
    opacity: scrollY.interpolate({
      inputRange: [0, 200],
      outputRange: [1, 0]
    })
  }}> ... </Animated.View>
</Animated.ScrollView>
Astuce: Utilisez interpolate pour animer la transparence du header ou l'échelle des cartes au scroll.

🌈 Exemple : interpolation de couleur + rotation

const spinValue = new Animated.Value(0);

const rotate = spinValue.interpolate({
  inputRange: [0, 1],
  outputRange: ['0deg', '360deg']
});

const color = spinValue.interpolate({
  inputRange: [0, 0.5, 1],
  outputRange: ['#FF5733', '#33FF57', '#3357FF']
});

// Lancement en boucle
Animated.loop(
  Animated.timing(spinValue, {
    toValue: 1,
    duration: 2000,
    useNativeDriver: false
  })
).start();
✨ Résultat: un carré qui tourne et change de couleur en continu.

⚡ Performances & useNativeDriver

Pour des animations 60 FPS, activez useNativeDriver: true sur les propriétés compatibles (transform, opacity).

Propriétés natives:
- opacity
- transform: translate, scale, rotate
- shadow (iOS)
Non natif:
- width, height
- backgroundColor, borderWidth
(nécessite JS thread)
📌 Toujours utiliser useNativeDriver: true quand c'est possible pour des animations fluides sur mobile.
