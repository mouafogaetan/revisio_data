Navigation dans React Native

Bottom Tab Navigator + Stack Navigator : la combinaison gagnante pour les apps mobiles

📱 Bottom Tabs 📚 Stack (pile) 🔁 Navigation imbriquée

🎯 Objectifs de la leçon

  • ✓ Créer un Bottom Tab Navigator (Accueil, Recherche, Panier, Profil)
  • ✓ Configurer un Stack Navigator pour gérer l'historique des écrans
  • ✓ Imbriquer un Stack dans un Tab pour une navigation fluide
  • ✓ Personnaliser les headers et la barre d'onglets (icônes, couleurs)
💡 Pourquoi ? Les tabs donnent une vue d'ensemble des sections principales, les stacks permettent de naviguer en profondeur (ex: produit → panier).

2. Bottom Tab Navigator

// navigation/BottomTabNavigator.js
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import HomeStack from './HomeStack';
import SearchScreen from '../screens/SearchScreen';
import CartScreen from '../screens/CartScreen';
import ProfileScreen from '../screens/ProfileScreen';
import { Ionicons } from '@expo/vector-icons';

const Tab = createBottomTabNavigator();

export default function BottomTabNavigator() {
  return (
    <Tab.Navigator
      screenOptions={({ route }) => ({
        tabBarIcon: ({ focused, color, size }) => {
          let iconName;
          if (route.name === 'Accueil') iconName = focused ? 'home' : 'home-outline';
          else if (route.name === 'Recherche') iconName = focused ? 'search' : 'search-outline';
          else if (route.name === 'Panier') iconName = focused ? 'cart' : 'cart-outline';
          else if (route.name === 'Profil') iconName = focused ? 'person' : 'person-outline';
          return <Ionicons name={iconName} size={size} color={color} />;
        },
        tabBarActiveTintColor: '#3b82f6',
        tabBarInactiveTintColor: 'gray',
        headerShown: false,
      })}
    >
      <Tab.Screen name="Accueil" component={HomeStack} />
      <Tab.Screen name="Recherche" component={SearchScreen} />
      <Tab.Screen name="Panier" component={CartScreen} />
      <Tab.Screen name="Profil" component={ProfileScreen} />
    </Tab.Navigator>
  );
}
✨ headerShown: false car chaque stack a son propre header.
📚

3. Stack Navigator (exemple: Accueil)

// navigation/HomeStack.js
import { createStackNavigator } from '@react-navigation/stack';
import HomeScreen from '../screens/HomeScreen';
import ProductDetailScreen from '../screens/ProductDetailScreen';
import CategoryScreen from '../screens/CategoryScreen';

const Stack = createStackNavigator();

export default function HomeStack() {
  return (
    <Stack.Navigator
      screenOptions={{
        headerStyle: { backgroundColor: '#3b82f6' },
        headerTintColor: '#fff',
        headerTitleStyle: { fontWeight: 'bold' },
      }}
    >
      <Stack.Screen name="Home" component={HomeScreen} options={{ title: 'Accueil' }} />
      <Stack.Screen name="ProductDetail" component={ProductDetailScreen} options={{ title: 'Détail produit' }} />
      <Stack.Screen name="Category" component={CategoryScreen} options={{ title: 'Catégorie' }} />
    </Stack.Navigator>
  );
}
🔁 Chaque onglet peut contenir son propre Stack Navigator. Cela permet une navigation indépendante par section.

4. Navigation avec paramètres

// Depuis un composant (ex: ProductCard)
import { useNavigation } from '@react-navigation/native';

function ProductCard({ product }) {
  const navigation = useNavigation();
  return (
    <TouchableOpacity onPress={() => navigation.navigate('ProductDetail', { 
      productId: product.id, 
      name: product.name 
    })}>
      <Text>{product.name}</Text>
    </TouchableOpacity>
  );
}

// Dans ProductDetailScreen
function ProductDetailScreen({ route }) {
  const { productId, name } = route.params;
  return <Text>Affichage produit {name}</Text>;
}
📤 navigation.navigate('Route', params)
📥 route.params récupère les données
🔙 Retour automatique avec bouton retour du stack.
🎨

5. Personnalisation des headers et onglets

Header dynamique :

<Stack.Screen 
  name="ProductDetail" 
  component={ProductDetailScreen} 
  options={({ route }) => ({ title: route.params.name })}
/>

Badge sur l'icône panier :

tabBarBadge: cartItems.length > 0 ? cartItems.length : undefined

Cacher l'onglet sur certains écrans :

navigation.getParent()?.setOptions({ tabBarStyle: { display: 'none' } });
✨ Bonnes pratiques : éviter les tabs trop nombreux (max 5). Utiliser des titres clairs.

🏁 Récapitulatif des concepts clés

  • Bottom Tab Navigator : navigation principale (onglets en bas)
  • Stack Navigator : historique d'écrans (pile) avec bouton retour
  • Navigation imbriquée : Stack à l'intérieur d'un Tab = meilleure UX
  • Paramètres : passer des données via route.params
  • useNavigation : hook pour naviguer programmatiquement

✨ Résultat : Vous maîtrisez la structure de navigation standard des applications mobiles modernes (Instagram, Amazon, etc.). Prêt pour intégrer l'authentification et les deep links !
