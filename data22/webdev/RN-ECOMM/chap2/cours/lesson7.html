⚙️ Installation de React Native Reanimated 3

Reanimated 3 est la bibliothèque d'animations recommandée pour React Native. Elle offre des performances optimales (60 FPS) et une syntaxe moderne.

# Installation dans un projet Expo
npx expo install react-native-reanimated

# Pour React Native CLI
npm install react-native-reanimated

Configuration Babel (expo ou RN CLI) :

// babel.config.js
module.exports = {
  presets: ['babel-preset-expo'],
  plugins: ['react-native-reanimated/plugin'], // 👈 important
};
⚠️ Note : Le plugin Babel doit être le dernier de la liste pour fonctionner correctement. Redémarrez l'application après modification.

🎨 Créer des animations fluides

Les hooks principaux : useSharedValue, useAnimatedStyle, withSpring et withTiming.

import Animated, { 
  useSharedValue, 
  useAnimatedStyle, 
  withSpring, 
  withTiming,
  Easing 
} from 'react-native-reanimated';

// Animation d'apparition d'un produit
const ProductCard = () => {
  const opacity = useSharedValue(0);
  const translateY = useSharedValue(50);
  
  useEffect(() => {
    opacity.value = withTiming(1, { duration: 500 });
    translateY.value = withSpring(0, { damping: 10, stiffness: 100 });
  }, []);
  
  const animatedStyle = useAnimatedStyle(() => ({
    opacity: opacity.value,
    transform: [{ translateY: translateY.value }]
  }));
  
  return (
    <Animated.View style={[styles.card, animatedStyle]}>
      <Text>Produit</Text>
    </Animated.View>
  );
};

🔄 Transitions entre les pages

Pour animer le changement d'écran, on combine Reanimated avec React Navigation via des transitions personnalisées.

// Configuration d'une transition personnalisée
import { TransitionPresets } from '@react-navigation/stack';

const screenOptions = {
  ...TransitionPresets.ModalSlideFromBottomIOS,
  gestureEnabled: true,
  cardStyleInterpolator: ({ current, layouts }) => ({
    cardStyle: {
      transform: [
        {
          translateX: current.progress.interpolate({
            inputRange: [0, 1],
            outputRange: [layouts.screen.width, 0],
          }),
        },
      ],
    },
  }),
};

// Alternative avec Reanimated directement dans les écrans
const AnimatedScreen = () => {
  const slideAnim = useSharedValue(300);
  
  useEffect(() => {
    slideAnim.value = withSpring(0, { damping: 15 });
  }, []);
  
  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateY: slideAnim.value }]
  }));
  
  return <Animated.View style={[styles.container, animatedStyle]}>...</Animated.View>;
};

📱 Astuce : Pour une expérience utilisateur optimale, utilisez des transitions de 250-350ms et des courbes d'accélération naturelles (spring pour les interactions directes, timing pour les transitions de pages).

🎬 Animer avec Lottie

Lottie permet d'intégrer des animations After Effects (fichiers JSON) pour des expériences utilisateur riches : loader, succès de commande, cœurs animés, etc.

# Installation
npx expo install lottie-react-native

# Utilisation
import LottieView from 'lottie-react-native';

// Animation de chargement
<LottieView
  source={require('./assets/loading.json')}
  autoPlay
  loop
  style={{ width: 200, height: 200 }}
/>

// Animation de succès (une fois)
<LottieView
  source={require('./assets/success.json')}
  autoPlay
  loop={false}
  onAnimationFinish={() => navigation.navigate('Orders')}
/>

📝 Exercices d'application

Exercice 1 (compétence – medium)

Créez un composant AnimatedButton qui :

  • S'anime au toucher (scale de 1 à 0.95 puis retour avec spring)
  • Change de couleur progressivement
  • Affiche un Lottie de succès après clic (simulation API)
🔍 Voir corrigé 
const AnimatedButton = ({ onPress, title }) => {
  const scale = useSharedValue(1);
  const backgroundColor = useSharedValue('#3b82f6');
  
  const handlePressIn = () => {
    scale.value = withSpring(0.95);
  };
  
  const handlePressOut = async () => {
    scale.value = withSpring(1);
    backgroundColor.value = withTiming('#22c55e', { duration: 300 });
    onPress();
  };
  
  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ scale: scale.value }],
    backgroundColor: backgroundColor.value,
  }));
  
  return (
    <AnimatedTouchableOpacity
      onPressIn={handlePressIn}
      onPressOut={handlePressOut}
      style={[styles.button, animatedStyle]}
    >
      <Text style={styles.buttonText}>{title}</Text>
    </AnimatedTouchableOpacity>
  );
};

Exercice 2 (ressource – hard)

Implémentez une animation de panier : lorsqu'on ajoute un produit, un petit cercle rouge part du bouton et se déplace vers l'icône du panier en haut à droite.

🔍 Voir corrigé

Utilisez deux useSharedValue pour les coordonnées X/Y, un withTiming pour le déplacement, et onAnimationFinish pour nettoyer. La mesure des positions se fait avec onLayout.

Exercice 3 (compétence – easy)

Ajoutez une animation d'entrée en cascade (stagger) pour une liste de 5 produits. Chaque produit doit apparaître avec un délai croissant de 100ms.

🔍 Voir corrigé 
products.forEach((_, index) => {
  opacityValues[index].value = withDelay(
    index * 100,
    withTiming(1, { duration: 400 })
  );
  translateYValues[index].value = withDelay(
    index * 100,
    withSpring(0, { damping: 12 })
  );
});

✅ Bonnes pratiques & récapitulatif

  • Préférez withSpring pour les interactions directes (boutons, gestes)
  • Utilisez withTiming avec Easing.inOut pour les transitions de pages
  • Évitez d'animer width/height (coûteux) → préférez scale ou opacity
  • Pour les listes longues, utilisez FlashList + animations sur les éléments visibles uniquement
  • Testez les animations sur des appareils réels (Android parfois moins fluide)
  • Lottie : compressez les fichiers JSON (lottie-compress) pour réduire la taille du bundle
