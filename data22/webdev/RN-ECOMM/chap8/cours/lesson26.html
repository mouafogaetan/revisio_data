📦 Installation & configuration

npm install react-hook-form zod @hookform/resolvers

Ces librairies permettent de créer des formulaires rapides, avec validation déclarative.

import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { z } from "zod";

const addressSchema = z.object({
  fullName: z.string().min(3, "Nom complet requis"),
  phone: z.string().regex(/^[6-9][0-9]{8}$/, "Téléphone invalide (9 chiffres)"),
  city: z.string().min(2, "Ville requise"),
  address: z.string().min(5, "Adresse précise obligatoire"),
});

const { register, handleSubmit, formState: { errors } } = useForm({
  resolver: zodResolver(addressSchema)
});
Zod valide côté client et côté serveur (cohérence).

🚚 Choix du mode de livraison

Utiliser le state local ou react-hook-form avec des radio buttons personnalisés.

const [delivery, setDelivery] = useState('standard');

const deliveryOptions = [
  { id: 'standard', label: 'Standard (3-5j)', price: 1500 },
  { id: 'express', label: 'Express (24h)', price: 3500 },
  { id: 'retrait', label: 'Retrait en magasin', price: 0 },
];

// JSX
{deliveryOptions.map(opt => (
  <TouchableOpacity key={opt.id} onPress={() => setDelivery(opt.id)}>
    <View style={{ flexDirection: 'row', alignItems: 'center' }}>
      <RadioButton value={opt.id} status={delivery===opt.id ? 'checked' : 'unchecked'} />
      <Text>{opt.label} - {opt.price === 0 ? 'Gratuit' : opt.price+' FCFA'}</Text>
    </View>
  </TouchableOpacity>
))}
Standard 1500 FCFA Express 3500 FCFA ✓ Retrait gratuit

🧾 Écran récapitulatif

Afficher un résumé des informations saisies + total du panier + frais de livraison.

const RecapScreen = ({ route }) => {
  const { address, deliveryMode } = route.params;
  const cartTotal = useCartStore(state => state.total);
  const deliveryFee = deliveryMode === 'express' ? 3500 : (deliveryMode === 'standard' ? 1500 : 0);
  const grandTotal = cartTotal + deliveryFee;

  return (
    <View>
      <Text>Livraison : {address.city}, {address.address}</Text>
      <Text>Frais livraison : {deliveryFee} FCFA</Text>
      <Text>Total TTC : {grandTotal} FCFA</Text>
      <Button title="Passer au paiement" onPress={goToPayment} />
    </View>
  );
};
$$ \text{Total} = \text{Sous-total panier} + \text{Frais livraison} - \text{Réduction} $$
