🎉 Animation de succès avec Lottie

Utiliser un fichier JSON d'animation (checkmark, confettis) depuis LottieFiles.

import LottieView from 'lottie-react-native';

const SuccessAnimation = () => {
  return (
    <LottieView
      source={require('../assets/success-check.json')}
      autoPlay
      loop={false}
      style={{ width: 200, height: 200 }}
      onAnimationFinish={() => console.log('Terminé')}
    />
  );
};
Animation personnalisable (coche, colis livré, feu d'artifice)

📄 Génération et partage de PDF

Créer un reçu PDF (facture) avec expo-print et le partager via expo-sharing.

import * as Print from 'expo-print';
import * as Sharing from 'expo-sharing';

const generateAndSharePDF = async (order) => {
  const html = `
    <html><body>
      <h1>AfriMarket - Facture #${order.id}</h1>
      <p>Date: ${order.date}</p>
      <p>Total: ${order.total} FCFA</p>
      <p>Merci pour votre achat !</p>
    </body></html>`;

  const { uri } = await Print.printToFileAsync({ html });
  await Sharing.shareAsync(uri, { mimeType: 'application/pdf' });
};
Alternative : capture d'écran (expo-screen-capture) ou partage d'image.

📸 Partager une capture du récapitulatif

Permet à l'utilisateur d'envoyer une image de sa confirmation (WhatsApp, etc.).

import { captureRef } from 'react-native-view-shot';
import * as Sharing from 'expo-sharing';

const viewRef = useRef();

const shareReceipt = async () => {
  const uri = await captureRef(viewRef, { format: 'png', quality: 0.8 });
  await Sharing.shareAsync(uri, { mimeType: 'image/png' });
};

return (
  <View ref={viewRef}>
    <Text>Merci pour votre commande #123</Text>
    <Text>Total: 25 000 FCFA</Text>
  </View>
);
Très utile pour les réseaux sociaux ou le support client.

🔄 Redirection automatique

Après quelques secondes ou sur bouton, naviguer vers l'écran "Mes Commandes".

import { useNavigation } from '@react-navigation/native';
import { useEffect } from 'react';

const OrderSuccessScreen = () => {
  const navigation = useNavigation();

  useEffect(() => {
    const timer = setTimeout(() => {
      navigation.replace('OrdersHistory');
    }, 4000);
    return () => clearTimeout(timer);
  }, []);

  return (
    <View>
      <SuccessAnimation />
      <Button title="Voir mes commandes" onPress={() => navigation.replace('OrdersHistory')} />
    </View>
  );
};
⚠️ Utilisez replace plutôt que navigate pour éviter de revenir à l'écran de succès.

✨ Bonnes pratiques & synthèse

  • ✔️ Animation visible mais non bloquante (max 2-3s).
  • ✔️ Proposer toujours un bouton d'action (partager, voir commandes, continuer).
  • ✔️ Générer un identifiant de commande unique pour le partage.
  • ✔️ Stocker le PDF ou l'image localement pour éviter de régénérer.
  • ✔️ Envoyer un email de confirmation en arrière-plan.

KPI à suivre :
