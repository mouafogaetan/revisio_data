⭐ Composant de notation par étoiles

Un composant réutilisable pour permettre à l'utilisateur de noter de 1 à 5 étoiles.

// StarRating.jsx
import { View, TouchableOpacity } from 'react-native';
import Icon from 'react-native-vector-icons/FontAwesome';

const StarRating = ({ rating = 0, onRatingChange, size = 32, readonly = false }) => {
  const stars = [1, 2, 3, 4, 5];
  
  return (
    <View className="flex-row gap-2">
      {stars.map((star) => (
        <TouchableOpacity
          key={star}
          onPress={() => !readonly && onRatingChange(star)}
          disabled={readonly}
        >
          <Icon
            name={star <= rating ? 'star' : 'star-o'}
            size={size}
            color={star <= rating ? '#fbbf24' : '#d1d5db'}
          />
        </TouchableOpacity>
      ))}
    </View>
  );
};

📌 Note : On peut aussi gérer les demi-étoiles avec 'star-half-o' pour plus de précision.

📝 Formulaire d'avis avec photos

Combinaison de React Hook Form, Zod, et expo-image-picker.

import * as ImagePicker from 'expo-image-picker';
import { useForm, Controller } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';

const reviewSchema = z.object({
  rating: z.number().min(1).max(5),
  comment: z.string().min(3, 'Commentaire trop court').max(500),
  images: z.array(z.string()).max(5, 'Maximum 5 photos'),
});

const ReviewForm = ({ productId, onSuccess }) => {
  const { control, handleSubmit, formState: { isSubmitting }, setValue, watch } = useForm({
    resolver: zodResolver(reviewSchema),
    defaultValues: { rating: 0, comment: '', images: [] }
  });
  
  const images = watch('images');
  
  const pickImages = async () => {
    const result = await ImagePicker.launchImageLibraryAsync({
      mediaTypes: ImagePicker.MediaTypeOptions.Images,
      allowsMultipleSelection: true,
      selectionLimit: 5,
      quality: 0.8,
    });
    if (!result.canceled) {
      const uris = result.assets.map(asset => asset.uri);
      setValue('images', [...images, ...uris].slice(0, 5));
    }
  };
  
  const removeImage = (index) => {
    const newImages = [...images];
    newImages.splice(index, 1);
    setValue('images', newImages);
  };
  
  const onSubmit = async (data) => {
    // Envoi du formulaire avec les images
    await submitReview(productId, data);
    onSuccess();
  };
  
  // Rendu du formulaire...
};

📋 Liste des avis avec options de modification

Affichage des avis avec photos, note moyenne, et actions si l'utilisateur est l'auteur.

const ReviewCard = ({ review, currentUserId, onEdit, onDelete }) => {
  const isAuthor = review.userId === currentUserId;
  
  return (
    <View className="bg-white p-4 rounded-lg mb-3 shadow-sm">
      <View className="flex-row justify-between items-start">
        <View>
          <Text className="font-semibold">{review.userName}</Text>
          <StarRating rating={review.rating} readonly size={16} />
        </View>
        {isAuthor && (
          <View className="flex-row gap-2">
            <TouchableOpacity onPress={() => onEdit(review)}>
              <Icon name="pencil" size={20} color="#3b82f6" />
            </TouchableOpacity>
            <TouchableOpacity onPress={() => onDelete(review.id)}>
              <Icon name="trash" size={20} color="#ef4444" />
            </TouchableOpacity>
          </View>
        )}
      </View>
      <Text className="text-gray-700 mt-2">{review.comment}</Text>
      {review.images?.length > 0 && (
        <ScrollView horizontal className="mt-3">
          {review.images.map((img, idx) => (
            <Image key={idx} source={{ uri: img }} className="w-20 h-20 rounded-md mr-2" />
          ))}
        </ScrollView>
      )}
      <Text className="text-xs text-gray-400 mt-2">{review.createdAt}</Text>
    </View>
  );
};

// Appel API

📝 Exercices d'application

Exercice 1 (compétence – medium)

Créez un composant AverageRating qui affiche la note moyenne d'un produit (ex: 4.5/5) avec des étoiles pleines, demi-étoiles et vides.

🔍 Voir corrigé 
const AverageRating = ({ average, totalReviews }) => {
  const fullStars = Math.floor(average);
  const hasHalfStar = average % 1 >= 0.5;
  const emptyStars = 5 - fullStars - (hasHalfStar ? 1 : 0);
  
  return (
    <View className="flex-row items-center">
      <View className="flex-row">
        {[...Array(fullStars)].map((_, i) => <Star key={i} filled />)}
        {hasHalfStar && <Star half />}
        {[...Array(emptyStars)].map((_, i) => <Star key={i} empty />)}
      </View>
      <Text className="ml-2 font-semibold">{average.toFixed(1)}/5</Text>
      <Text className="text-gray-500 text-sm ml-1">({totalReviews} avis)</Text>
    </View>
  );
};

Implémentez la pagination des avis avec FlatList et chargement infini (infinite scroll).

🔍 Voir corrigé 
const ReviewsList = ({ productId }) => {
  const [page, setPage] = useState(1);
  const [reviews, setReviews] = useState([]);
  const [hasMore, setHasMore] = useState(true);
  
  const loadMore = async () => {
    if (!hasMore) return;
    const response = await fetch(`/products/${productId}/reviews?page=${page}&limit=10`);
    const newReviews = await response.json();
    setReviews(prev => [...prev, ...newReviews.data]);
    setHasMore(newReviews.data.length === 10);
    setPage(prev => prev + 1);
  };
  
  return (
    <FlatList
      data={reviews}
      renderItem={({ item }) => <ReviewCard review={item} />}
      onEndReached={loadMore}
      onEndReachedThreshold={0.3}
      ListFooterComponent={hasMore && <ActivityIndicator />}
    />
  );
};

✅ Bonnes pratiques & récapitulatif

  • Vérifier que l'utilisateur a bien acheté le produit avant de pouvoir poster un avis
  • Limiter le nombre de photos (5 max) et leur taille (compression)
  • Afficher les avis les plus récents ou les mieux notés en premier (option de tri)
  • Modérer les avis : signaler les commentaires inappropriés
  • Répondre aux avis négatifs pour montrer l'engagement client
  • Utiliser React Query pour la gestion du cache des avis
