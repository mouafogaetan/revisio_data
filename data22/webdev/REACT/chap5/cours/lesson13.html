📝 L'input contrôlé

Un input contrôlé est un champ dont la valeur est gérée par l'état React via useState.

🔑 Principe :

La valeur affichée vient de l'état, et chaque modification déclenche une mise à jour de l'état via onChange.

const [nom, setNom] = useState('');

<input 
  type="text"
  value={nom}
  onChange={(e) => setNom(e.target.value)}
/>

Avantage : la source de vérité est unique (l'état React).

✏️ Input text & Textarea

Le fonctionnement est identique pour <input type="text"> et <textarea> : on utilise value et onChange.

// Input text
const [nom, setNom] = useState('');

<input 
  type="text"
  value={nom}
  onChange={(e) => setNom(e.target.value)}
  placeholder="Votre nom"
/>
// Textarea
const [bio, setBio] = useState('');

<textarea
  value={bio}
  onChange={(e) => setBio(e.target.value)}
  rows="3"
/>
En React, le textarea utilise value (pas le texte entre les balises comme en HTML).

☑️ Checkbox & Radio buttons

Pour les checkbox et radio, on utilise checked (et non value) avec onChange.

// Checkbox
const [accepte, setAccepte] = useState(false);

<input 
  type="checkbox"
  checked={accepte}
  onChange={(e) => setAccepte(e.target.checked)}
/>

// Radio buttons (même name, différentes valeurs)
const [genre, setGenre] = useState('');

<input type="radio" name="genre" value="homme" 
       checked={genre === 'homme'}
       onChange={(e) => setGenre(e.target.value)} />
<input type="radio" name="genre" value="femme" 
       checked={genre === 'femme'}
       onChange={(e) => setGenre(e.target.value)} />

📋 Select (dropdown)

Le composant <select> utilise value sur la balise select elle-même.

const [ville, setVille] = useState('douala');

<select value={ville} onChange={(e) => setVille(e.target.value)}>
  <option value="yaounde">Yaoundé</option>
  <option value="douala">Douala</option>
  <option value="garoua">Garoua</option>
</select>

Multiple select : utiliser value={[]} et e.target.selectedOptions.

📬 Formulaire complet & onSubmit

On utilise onSubmit sur la balise <form> avec preventDefault() pour éviter le rechargement de la page.

const [formData, setFormData] = useState({
  nom: '',
  email: '',
  abonnement: false
});

const handleSubmit = (e) => {
  e.preventDefault();
  console.log('Données soumises :', formData);
  // Envoi à une API...
};

<form onSubmit={handleSubmit}>
  <input value={formData.nom} 
         onChange={(e) => setFormData({...formData, nom: e.target.value})} />
  <button type="submit">Envoyer</button>
</form>

⚡ Optimisation : un seul état pour tout le formulaire

Pour les formulaires avec plusieurs champs, on utilise un objet d'état et une fonction générique.

const [form, setForm] = useState({
  nom: '',
  email: '',
  age: '',
  newsletter: false
});

const handleChange = (e) => {
  const { name, value, type, checked } = e.target;
  setForm(prev => ({
    ...prev,
    [name]: type === 'checkbox' ? checked : value
  }));
};

<input name="nom" value={form.nom} onChange={handleChange} />
<input name="email" value={form.email} onChange={handleChange} />
<input type="checkbox" name="newsletter" checked={form.newsletter} onChange={handleChange} />
