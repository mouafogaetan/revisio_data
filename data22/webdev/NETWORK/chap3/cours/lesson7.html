🏷️ Qu'est-ce qu'une adresse IP ?

Une adresse IP (Internet Protocol) est un numéro unique qui identifie chaque appareil connecté à un réseau (ordinateur, smartphone, imprimante, etc.).

C'est comme l'adresse postale d'une maison : elle permet de savoir où envoyer les données.

192.168.1.10   |   172.16.0.1   |   8.8.8.8

🧮 Structure d'IPv4 : 32 bits

Une adresse IPv4 fait 32 bits (0 ou 1), découpée en 4 octets (8 bits chacun).

11000000 . 10101000 . 00000001 . 00001010
↓
192 . 168 . 1 . 10

Forme décimale pointée : 192.168.1.10 (plus lisible pour nous).

Chaque octet varie de 0 à 255.

🔄 Conversion binaire ↔ décimal

Chaque bit a un poids (128, 64, 32, 16, 8, 4, 2, 1).

1 1 1 1 1 1 1 1 = 255
1 0 1 0 1 0 1 0 = 170
0 0 0 0 1 0 1 0 = 10

Exemple : \( 192 = 11000000_2 \) (128+64).

📌 Pour s'entraîner, décomposez chaque octet en puissances de 2.

🏷️ Les classes d'adresses IPv4

ClassePlageBits de débutUsage
A1.0.0.0 – 126.255.255.2550xxx xxxxTrès grands réseaux
B128.0.0.0 – 191.255.255.25510xx xxxxRéseaux moyens
C192.0.0.0 – 223.255.255.255110x xxxxPetits réseaux
D224.0.0.0 – 239.255.255.2551110 xxxxMulticast
E240.0.0.0 – 255.255.255.2551111 xxxxRecherche / réservé

Astuce : regardez le premier octet !

🔎 Identifier rapidement la classe

Classe A : premier octet entre 1 et 126 → 10.0.0.1 (A)
Classe B : premier octet entre 128 et 191 → 172.16.0.1 (B)
Classe C : premier octet entre 192 et 223 → 192.168.1.1 (C)
Classe D : 224-239 → multicast (ex: 224.0.0.1)
Classe E : 240-255 → réservé (recherche)

⚠️ Les adresses 127.x.x.x sont spéciales (loopback).

⭐ Adresses spéciales

  • Loopback (127.0.0.1) : test sur sa propre machine (localhost).
  • Adresse réseau : tous les bits d'hôte à 0 (ex: 192.168.1.0/24).
  • Adresse broadcast : tous les bits d'hôte à 1 (ex: 192.168.1.255). Envoie à tous les hôtes du réseau.
  • 0.0.0.0 : désigne un hôte non spécifié (ex: serveur DHCP).
  • 255.255.255.255 : broadcast limité (local).

🏠 Adresses privées (non routables sur Internet)

ClassePlage privée
A10.0.0.0 – 10.255.255.255
B172.16.0.0 – 172.31.255.255
C192.168.0.0 – 192.168.255.255

Ces adresses sont utilisées dans les réseaux locaux (WiFi, entreprise). Pour accéder à Internet, un routeur fait du NAT.

🏡 Exemple : ta box internet donne une adresse en 192.168.1.x

📋 Exemples d'analyse

🔹 10.0.0.5 → Classe A (privée)
🔹 172.20.10.2 → Classe B (privée)
🔹 192.168.0.100 → Classe C (privée)
🔹 8.8.8.8 → Classe A (publique, DNS Google)
🔹 127.0.0.1 → Loopback (soi-même)
🔹 224.0.0.1 → Multicast (tous les hôtes du segment)
