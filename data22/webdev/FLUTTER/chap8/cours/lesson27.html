✨ Qu'est-ce qu'une animation implicite ?

Flutter propose des widgets AnimatedXXX qui réagissent automatiquement lorsque leurs propriétés changent. Pas besoin de contrôler manuellement l'animation.

📦 AnimatedContainer

Anime la taille, couleur, marge, bordure, etc.

👻 AnimatedOpacity

Fait apparaître/disparaître avec fondu.

📍 AnimatedPositioned

Utilisé dans un Stack pour déplacer un widget.

🔄 AnimatedCrossFade

Croisement fondu entre deux enfants.

📘 Animation d'un conteneur

Avec AnimatedContainer, modifions sa hauteur, sa couleur et ses bordures. Il suffit de changer son color ou height dans un setState.

AnimatedContainer(
  duration: Duration(milliseconds: 500),
  curve: Curves.easeInOutCubic,
  width: 200,
  height: _expanded ? 200 : 100,
  color: _isBlue ? Colors.blue : Colors.orange,
  child: Center(child: Text('Cliquez-moi')),
)

🎯 Durée & Courbe :

  • duration : temps en millisecondes (ex: 300ms)
  • curve : l'accélération (Curves.easeIn, Curves.elasticOut, Curves.bounceIn…)

🌫️ AnimatedOpacity & AnimatedSwitcher

AnimatedOpacity : fait varier l'opacité (opacity: 0 → 1) pour faire apparaître/disparaître un widget.

AnimatedOpacity(
  duration: Duration(seconds: 1),
  opacity: _visible ? 1.0 : 0.0,
  child: Image.asset('assets/flower.png'),
)

AnimatedSwitcher : anime la transition entre deux widgets différents (scale, fade…)

AnimatedSwitcher(
  duration: Duration(milliseconds: 400),
  transitionBuilder: (child, anim) => FadeTransition(opacity: anim, child: child),
  child: _showText ? Text('Bonjour') : Icon(Icons.star, key: ValueKey('star')),
)
💡 Annoter chaque enfant avec un Key unique pour qu'AnimatedSwitcher détecte le changement.

🔄 AnimatedCrossFade & Hero

AnimatedCrossFade : anime le fondu croisé entre deux widgets lorsque le booléen change. Idéal pour les états de chargement vs contenu.

AnimatedCrossFade(
  duration: Duration(milliseconds: 600),
  firstChild: CircularProgressIndicator(),
  secondChild: Text('Données chargées'),
  crossFadeState: _isLoading ? CrossFadeState.showFirst : CrossFadeState.showSecond,
)

✨ Hero animations : transition spectaculaire entre deux écrans (partagé). On verra dans la leçon 30.

📝 À retenir & Exercice pratique

✓

AnimatedContainer : anime plusieurs propriétés à la fois (height, color, padding).

✓

AnimatedOpacity : idéal pour faire disparaître un widget (attention: l'espace reste réservé).

✓

AnimatedSwitcher / CrossFade : parfaits pour alterner entre deux vues.

✓

Durée & Curve : personnalisez le ressenti (ex: Curves.bounceOut pour un effet élastique).

Créez un bouton qui alterne la taille d'un carré rouge à vert et arrondi, avec une animation de 800ms et une courbe Curves.elasticOut. Utilisez AnimatedContainer. Bonus : ajoutez une ombre.
