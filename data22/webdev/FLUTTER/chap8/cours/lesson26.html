📖 TextTheme : la base typographique

TextTheme est une classe qui regroupe les styles de texte par défaut pour une application Material Design. Vous pouvez la définir dans ThemeData.

// Exemple de TextTheme personnalisé
ThemeData(
  textTheme: TextTheme(
    displayLarge: TextStyle(fontSize: 72, fontWeight: FontWeight.bold),
    titleLarge: TextStyle(fontSize: 36, fontStyle: FontStyle.italic),
    bodyMedium: TextStyle(fontSize: 16, color: Colors.grey.shade800),
  ),
)

Les propriétés principales : headline1→headline6 (ancien nom) deviennent displayLarge, displayMedium, headlineLarge, titleLarge, bodyLarge, bodyMedium, labelLarge etc. Facile à adapter !

⚠️ Note: Flutter 3+ utilise la nouvelle nomenclature TextTheme, mais l’ancienne (headline1) reste fonctionnelle.

📦 Google Fonts - Installation

Ajoutez la dépendance dans pubspec.yaml :

dependencies:
  flutter:
    sdk: flutter
  google_fonts: ^6.2.1  # dernière version stable

Puis exécutez flutter pub get. Ensuite importez le package :

import 'package:google_fonts/google_fonts.dart';

Ce package permet d’utiliser plus de 1000 polices open source, sans téléchargement manuel (accès direct via API ou asset).

✨ Appliquer une police personnalisée

Méthode 1 – Globale (dans ThemeData) :

MaterialApp(
  theme: ThemeData(
    textTheme: GoogleFonts.poppinsTextTheme(),   // Toute l'app en Poppins
    // Ou personnalisation partielle:
    // textTheme: GoogleFonts.latoTextTheme(Theme.of(context).textTheme),
  ),
  home: MyHomePage(),
);

Méthode 2 – Locale sur un widget :

Text(
  'Bonjour Cameroun !',
  style: GoogleFonts.montserrat(fontSize: 20, fontWeight: FontWeight.w600),
)

🚀 Résultat : votre texte adopte immédiatement la police choisie, sans gestion manuelle des fichiers .ttf.

🎨 Créer des styles réutilisables

Combinez TextTheme et Google Fonts pour des typographies homogènes :

final textTheme = Theme.of(context).textTheme;
return MaterialApp(
  theme: ThemeData(
    textTheme: TextTheme(
      displayLarge: GoogleFonts.roboto(
        fontSize: 96, fontWeight: FontWeight.w300,
        letterSpacing: -1.5,
      ),
      titleLarge: GoogleFonts.robotoSlab(
        fontSize: 32, fontWeight: FontWeight.w600,
      ),
      bodyMedium: GoogleFonts.inter(fontSize: 16),
      labelSmall: GoogleFonts.lato(fontSize: 12, color: Colors.grey),
    ),
  ),
);

Ensuite utilisez Text('Titre', style: Theme.of(context).textTheme.titleLarge) ou directement via TextTheme.

⚙️ Astuces & performances

  • GoogleFonts.getFont() : charge la police dynamiquement (léger délai initial).
  • Téléchargement local : pour éviter la latence internet, vous pouvez pré-télécharger les polices et les inclure dans les assets (moins courant).
  • Fallback : si la police Google Fonts ne charge pas, le style revient à la police par défaut du système (Roboto sur Android, San Francisco sur iOS).
  • Utilisez GoogleFonts.poppinsTextTheme() pour une application complète sans trop de code.
  • Créez un fichier app_theme.dart pour centraliser tous vos styles : meilleure maintenabilité.
💡 Exemple de fichier réutilisable :
class AppTypography { static TextStyle headline = GoogleFonts.playfairDisplay(...); }

🧪 À vous de jouer !

Modifiez l’application par défaut (compteur Flutter) pour utiliser la police Poppins ou Open Sans :

  1. Ajouter google_fonts dans pubspec.yaml
  2. Dans le MaterialApp, appliquer theme: ThemeData(textTheme: GoogleFonts.latoTextTheme())
  3. Créer un style spécifique pour le titre avec GoogleFonts.montserrat(fontWeight: FontWeight.bold)
  4. Vérifier que le hot reload affiche la nouvelle police !
