🎬 AnimationController : Le chef d'orchestre

🔍 Définition simple

AnimationController est un objet qui produit des valeurs entre 0.0 et 1.0 (par défaut) au cours du temps. Il contrôle la durée, la direction et l’état de l’animation.

AnimationController controller = AnimationController(
  vsync: this,      // éviter la consommation inutile
  duration: Duration(seconds: 2),
);

📈 Valeur générée :

0.0 → 0.75 → 1.0 (exemple de progression)

Avec controller.forward() l’animation démarre, reverse() l’inverse, repeat() la répète.

💡 Important : Toujours appeler dispose() sur le controller pour éviter les fuites mémoire !

🌀 Tween : Interpoler n'importe quelle valeur

Tween transforme la plage [0.0 → 1.0] en une autre plage (couleurs, tailles, positions, etc.)

final Animation<double> sizeAnimation = Tween<double>(
  begin: 50.0,
  end: 150.0,
).animate(controller);

final Animation<Color> colorAnimation = ColorTween(
  begin: Colors.blue,
  end: Colors.red,
).animate(controller);
🔵 begin: 50px
→
🔴 end: 150px

🎨 On peut même faire des tweens personnalisés : RectTween, AlignmentTween, Matrix4Tween…

⚙️ AnimatedBuilder : réactivité efficace

AnimatedBuilder écoute l’AnimationController et reconstruit uniquement le widget qui change, sans rebuild inutile.

AnimatedBuilder(
  animation: controller,
  builder: (context, child) {
    return Transform.scale(
      scale: controller.value, // 0→1
      child: child,
    );
  },
  child: FlutterLogo(size: 80),
);

Architecture : le builder réagit à chaque valeur.

🔄 Forward, Reverse, Repeat & Dispose

controller.forward()

Démarre l’animation de begin vers end (de 0.0 à 1.0).

controller.reverse()

Anime en sens inverse (1.0 → 0.0).

controller.repeat()

Répète l’animation en boucle (utile pour des indicateurs).

⚠️ Nettoyage indispensable :

@override
void dispose() {
  controller.dispose();
  super.dispose();
}

📱 Exemple : Cercle qui grandit et tourne

Associons AnimationController, Tween et AnimatedBuilder dans un projet simple.

✨ Simulation d’une animation Controller + Tween (taille & rotation) style Flutter

Controller value → interpolation taille: 50px ↔ 130px , rotation: 0deg ↔ 30deg

🎯 Code clé :

final controller = AnimationController(duration: 800ms, vsync: this);
final sizeTween = Tween<double>(begin: 50, end: 130);
final rotateTween = Tween<double>(begin: 0, end: 0.5); // radians
...
AnimatedBuilder(
  animation: controller,
  builder: (_, child) => Transform.rotate(
    angle: rotateTween.evaluate(controller),
    child: Container(
      width: sizeTween.evaluate(controller),
      height: sizeTween.evaluate(controller),
      ...
    ),
  ),
)

📌 Récapitulatif & Astuces

  • ✅ Utilisez AnimationController pour gérer la durée et le statut de l'animation.
  • ✅ Tween permet d'interpoler entre deux valeurs quelconques.
  • ✅ AnimatedBuilder sépare la logique d’animation du widget pour des performances optimales.
  • ✅ Toujours dispose() le controller dans State.dispose.
  • ✅ Associer vsync: this avec SingleTickerProviderStateMixin.
