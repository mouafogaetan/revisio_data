🧩 Les widgets de mise en page : la base de toute UI Flutter

En Flutter, tout est widget ! Les widgets de disposition (layout widgets) permettent d'agencer d'autres widgets. Voici les quatre piliers :

Container

Ajoute du padding, des marges, des bordures, une couleur de fond, des contraintes de taille. C'est une "boîte" décorative.

Row

Dispose ses enfants horizontalement (de gauche à droite). Idéal pour les barres d'outils, rangées d'icônes.

Column

Dispose ses enfants verticalement (de haut en bas). Parfait pour les formulaires, listes verticales.

Stack

Superpose les widgets comme des calques. Permet de placer un texte sur une image, des badges, etc.

💡 À retenir : Row et Column utilisent mainAxisAlignment (axe principal) et crossAxisAlignment (axe secondaire) pour contrôler l'alignement.

🔄 Row & Column : axes et alignements

Row : axe principal → horizontal. Column : axe principal → vertical.

  • mainAxisAlignment : start, end, center, spaceBetween, spaceAround, spaceEvenly.
  • crossAxisAlignment : start, end, center, stretch.
  • children : liste de widgets à organiser.

📌 Exemple simple (Row) :

Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [Icon(Icons.star), Text("Note"), Icon(Icons.favorite)],
)

Enfants répartis uniformément

✅ Expanded / Flexible

Permet à un enfant de Row ou Column de prendre l'espace restant. Expanded impose une taille proportionnelle.

📦 SizedBox & Spacer

SizedBox fixe une taille précise ; Spacer crée un espace élastique entre les enfants.

🎨 Container et Stack : décoration et superposition

📦 Container (boîte magique)

Ajoute margin, padding, decoration, width, height, alignment. Exemple typique :

Container(
  width: 150,
  height: 80,
  margin: EdgeInsets.all(10),
  padding: EdgeInsets.symmetric(horizontal: 12),
  decoration: BoxDecoration(
    color: Colors.blueAccent,
    borderRadius: BorderRadius.circular(20),
    boxShadow: [BoxShadow(blurRadius: 8)],
  ),
  child: Text('Hello', style: TextStyle(color: Colors.white)),
)

🥞 Stack (superposition de calques)

Place les widgets les uns sur les autres. Utile pour des badges, des images avec légende.

Stack(
  children: [
    Image.network("https://picsum.photos/150/150"),
    Positioned(
      bottom: 0, right: 0,
      child: Icon(Icons.camera_alt, color: Colors.white),
    ),
  ],
)
🔎 Conseil : Pour des mises en page complexes, combinez Row, Column et Container. Utilisez Expanded pour éviter les débordements.

🚀 Exemple concret : une carte produit avec Flutter

Code layout :

Column(
  children: [
    Container(/* image */),
    Row(
      mainAxisAlignment: MainAxisAlignment.spaceBetween,
      children: [Text("T-shirt"), Icon(Icons.shopping_cart)],
    ),
    Row(
      children: [Text("Prix: 25€"), Spacer(), Text("⭐⭐⭐⭐")],
    ),
    SizedBox(height: 8),
    ElevatedButton(child: Text("Acheter")),
  ],
)
