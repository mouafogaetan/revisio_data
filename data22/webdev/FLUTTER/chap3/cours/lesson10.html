Leçon 10 · Initiation à Flutter

Widgets essentiels : Text, Image, Icon, IconButton

Objectifs de la leçon

1. Afficher du texte avec Text et RichText

Le widget Text est le plus simple pour afficher une chaîne de caractères. On peut personnaliser le style, la taille, la couleur, la graisse, etc.

Text(
  'Bonjour mes élèves du Cameroun !',
  style: TextStyle(
    fontSize: 20,
    fontWeight: FontWeight.bold,
    color: Colors.blue,
  ),
)

Pour un texte plus riche avec plusieurs styles, on utilise RichText et TextSpan :

RichText(
  text: TextSpan(
    text: 'Flutter est ',
    style: TextStyle(fontSize: 18, color: Colors.black),
    children: [
      TextSpan(text: 'puissant', style: TextStyle(fontWeight: FontWeight.bold, color: Colors.red)),
      TextSpan(text: ' et '),
      TextSpan(text: 'multiplateforme', style: TextStyle(fontStyle: FontStyle.italic, color: Colors.green)),
    ],
  ),
)
💡 À retenir : Text() est idéal pour du texte simple. RichText permet de mixer couleurs, tailles et styles dans une même phrase.
🖼️

2. Afficher des images : Image.asset, Image.network, Image.file

Flutter propose plusieurs constructeurs pour afficher des images selon leur provenance :

  • Image.asset : pour les images incluses dans le dossier assets/ du projet (déclaré dans pubspec.yaml).
  • Image.network : pour charger une image depuis une URL (internet).
  • Image.file : pour afficher une image depuis le système de fichiers de l’appareil (après sélection).
Image.asset(
  'assets/logo.png',
  width: 100,
  height: 100,
  fit: BoxFit.cover,
)
Image.network(
  'https://picsum.photos/200',
  loadingBuilder: (context, child, progress) => CircularProgressIndicator(),
)
⚙️ Bonnes pratiques : Pensez toujours à gérer les erreurs de chargement (placeholder, erreur) et à utiliser fit: BoxFit.cover pour un rendu harmonieux.

3. Icônes et boutons interactifs : Icon & IconButton

Flutter intègre Material Icons : plus de mille icônes prêtes à l’emploi. Le widget Icon permet simplement d’afficher une icône, tandis que IconButton la rend cliquable.

Icon(Icons.favorite, color: Colors.red, size: 40)

IconButton(
  icon: Icon(Icons.thumb_up, size: 30),
  onPressed: () {
    print("J'aime !");
  },
  color: Colors.blue,
  iconSize: 32,
)

Exemple d’interface simple : une barre d’actions avec icônes.

4. Exemple concret : mini application

Voici un petit widget Flutter qui combine Texte stylisé, Image réseau et IconButton pour une interaction simple. (Code conceptuel)

class MonWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        RichText(
          text: TextSpan(
            text: 'Bienvenue au ',
            style: TextStyle(fontSize: 24, color: Colors.black),
            children: [
              TextSpan(text: 'Cameroun ❤️', style: TextStyle(fontWeight: FontWeight.bold, color: Colors.green)),
            ],
          ),
        ),
        Image.network('https://flagcdn.com/w320/cm.png', height: 100),
        IconButton(
          icon: Icon(Icons.camera_alt, color: Colors.blue),
          onPressed: () => print("Prendre une photo"),
        )
      ],
    );
  }
}

Ce code illustre parfaitement les objectifs de la leçon : affichage textuel riche, image distante et icône interactive.

⭐ Pratique : Modifiez les couleurs, remplacez l'url par une image locale et essayez d'ajouter un texte avec un effet de survol (flutter n'a pas de hover, mais on peut utiliser InkWell).

✨ Récapitulatif final

  • ✔️ Utiliser Text pour le texte basique, RichText+TextSpan pour les textes stylisés complexes.
  • ✔️ Afficher des images via Image.asset (locale), Image.network (web) ou Image.file (stockage).
  • ✔️ Les icônes Material rendent l’interface vivante. IconButton réagit aux clics.
  • ✔️ Combiner ces widgets permet des UI riches et réactives.
