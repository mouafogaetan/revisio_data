💡 Pourquoi Hive ? Pas de requêtes SQL, performances exceptionnelles, idéal pour les préférences, le cache, les données offline.

📦 Hive en résumé

Hive est une base de données clé-valeur écrite en pur Dart. Contrairement à SQLite, elle ne nécessite aucune couche native, ce qui la rend extrêmement rapide.

🔑 Clé (Key)

String, int, ou tout type supporté

💎 Valeur (Value)

Primitives, listes, maps ou objets personnalisés

// Exemple ultra simple
var box = await Hive.openBox('settings');
box.put('theme', 'dark');
String? theme = box.get('theme');

✔️ Aucune requête SQL, aucune latence interopérable -> idéale pour le offline-first.

⚙️ Installation & initialisation

1. Ajouter les dépendances dans pubspec.yaml

dependencies:
  flutter:
    sdk: flutter
  hive_flutter: ^1.1.0
  
dev_dependencies:
  hive_generator: ^2.0.0
  build_runner: ^2.4.0

2. Initialiser Hive dans main()

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Hive.initFlutter();       // Initialisation
  await Hive.openBox('userBox');  // Ouvre une box
  runApp(MyApp());
}
🔔 N'oubliez pas WidgetsFlutterBinding.ensureInitialized() avant d'appeler des méthodes asynchrones.

✍️ Opérations CRUD avec Hive

🔹 Écrire / Mettre à jour : box.put('key', valeur)

var box = Hive.box('userBox');
box.put('username', 'alice_dev');

🔹 Lire : box.get('key', defaultValue: 'inconnu')

String? name = box.get('username');
int score = box.get('score', defaultValue: 0);

🔹 Supprimer : box.delete('key') ou box.clear()

box.delete('username');   // supprime une clé
box.clear();              // vide toute la box

🔹 Vérifier l'existence : box.containsKey('key')

Toutes les opérations sont synchrones après ouverture ; Hive est instantané.

📂 Boxes et écoute en temps réel

Une Box est comme une table / un conteneur. Vous pouvez ouvrir plusieurs boxes pour organiser vos données.

🎯 Exemple : Liste de tâches réactive

final taskBox = await Hive.openBox<String>('tasks');
taskBox.add('Acheter du lait');       // Ajout avec clé auto-générée
taskBox.add('Réviser le cours');

Pour réagir aux modifications dans l'interface Flutter :

ValueListenableBuilder(
  valueListenable: taskBox.listenable(),
  builder: (context, Box<String> box, _) {
    return ListView.builder(
      itemCount: box.length,
      itemBuilder: (ctx, idx) => ListTile(
        title: Text(box.getAt(idx)!),
        onTap: () => box.deleteAt(idx),
      ),
    );
  },
);

💡 Utilisez box.listenable() pour rafraîchir automatiquement vos widgets.

🧩 Stocker des objets personnalisés

Hive peut stocker des classes Dart complexes grâce aux adaptateurs.

import 'package:hive/hive.dart';

part 'person.g.dart'; // généré automatiquement

@HiveType(typeId: 0)
class Person {
  @HiveField(0)
  String name;
  
  @HiveField(1)
  int age;
  
  Person(this.name, this.age);
}

Étape suivante : lancer flutter packages pub run build_runner build pour générer le fichier person.g.dart.

✅ Enregistrer l'adaptateur avant d'ouvrir la box : Hive.registerAdapter(PersonAdapter());

🔁 Une fois l'adaptateur enregistré, vous pouvez stocker Person directement : box.put('me', Person('Dupont', 28)).

🎓 Bonnes pratiques avec Hive

  • Nommer clairement les boxes : Hive.openBox('settings'), 'tasksBox'.
  • Fermer les boxes inutilisées : await box.close() (optionnel mais recommandé).
  • Utiliser box.values pour itérer sur toutes les valeurs.
  • Ne pas oublier d'enregistrer les adaptateurs avant la première utilisation des objets personnalisés.
  • Préférer Hive.box<Type>('boxName') pour typer vos opérations.

🚀 Astuce avancée :

Utilisez LazyBox pour stocker des millions d'objets sans charger tout en mémoire. Idéal pour des listes infinies.

final lazyBox = await Hive.openLazyBox('bigData');
await lazyBox.put('key', heavyObject);

📱 Exemple concret : compteur persistant

class CounterApp extends StatefulWidget {
  @override
  _CounterAppState createState() => _CounterAppState();
}

class _CounterAppState extends State<CounterApp> {
  late Box<int> counterBox;
  
  @override
  void initState() {
    super.initState();
    _initBox();
  }
  
  _initBox() async {
    counterBox = await Hive.openBox<int>('counter');
    setState(() {});
  }
  
  void _increment() {
    int current = counterBox.get('count', defaultValue: 0);
    counterBox.put('count', current + 1);
    setState(() {});
  }
  
  @override
  Widget build(BuildContext context) {
    if (counterBox == null) return CircularProgressIndicator();
    int count = counterBox.get('count', defaultValue: 0);
    return Scaffold(
      appBar: AppBar(title: Text('Compteur Hive')),
      body: Center(child: Text('Valeur: $count', style: TextStyle(fontSize: 40))),
      floatingActionButton: FloatingActionButton(
        onPressed: _increment,
        child: Icon(Icons.add),
      ),
    );
  }
}

✨ La valeur reste stockée après fermeture de l'application.
