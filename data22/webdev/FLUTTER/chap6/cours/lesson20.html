📝 Formulaires Flutter & Validation

Utiliser Form, GlobalKey, TextFormField et validateurs personnalisés pour créer des formulaires interactifs et fiables.

✅ Validation intégrée 🔑 GlobalKey<FormState> 📋 TextFormField

Comprendre Form et TextFormField

Form est un conteneur qui regroupe plusieurs champs et facilite la validation globale. TextFormField est un champ de texte intégré au système de formulaire.

Form(
  key: _formKey,
  child: Column(
    children: [
      TextFormField(
        decoration: InputDecoration(labelText: "Nom complet"),
        validator: (value) {
          if (value == null || value.isEmpty) {
            return "Veuillez entrer votre nom";
          }
          return null;
        },
      ),
      ElevatedButton(
        onPressed: _submitForm,
        child: Text("Soumettre"),
      ),
    ],
  ),
);
✔️ Chaque TextFormField peut avoir son propre validateur. Si une erreur est retournée, elle s'affiche sous le champ.

🔑 GlobalKey<FormState> & validation globale

GlobalKey<FormState> permet d'accéder à l'état du formulaire pour déclencher la validation et sauvegarder les données.

final _formKey = GlobalKey<FormState>();

void _submitForm() {
  if (_formKey.currentState!.validate()) {
    // Tous les champs sont valides
    _formKey.currentState!.save();
    // Traitement des données
    print("Formulaire validé !");
  } else {
    print("Erreur de validation");
  }
}
📌 validate() → appelle tous les validateurs, retourne true si tout est valide.
📌 save() → déclenche onSaved de chaque champ pour récupérer les valeurs.

⚙️ Créer des validateurs personnalisés

Un validateur est une fonction qui reçoit la valeur du champ et retourne String? (message d'erreur) ou null si tout est correct.

String? validateEmail(String? value) {
  if (value == null || value.isEmpty) {
    return "L'email est requis";
  }
  final emailRegex = RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$');
  if (!emailRegex.hasMatch(value)) {
    return "Email invalide";
  }
  return null;
}

// Utilisation
TextFormField(
  validator: validateEmail,
)
String? validatePassword(String? value) {
  if (value == null || value.length < 6) {
    return "Le mot de passe doit contenir au moins 6 caractères";
  }
  return null;
}
🎯 Astuce : vous pouvez combiner plusieurs règles dans un même validateur (ex: regex + longueur).

📱 Exemple complet : formulaire d'inscription

class InscriptionForm extends StatefulWidget {
  @override
  _InscriptionFormState createState() => _InscriptionFormState();
}

class _InscriptionFormState extends State<InscriptionForm> {
  final _formKey = GlobalKey<FormState>();
  String _nom = '', _email = '', _password = '';

  void _submit() {
    if (_formKey.currentState!.validate()) {
      _formKey.currentState!.save();
      // Appeler une API ou stocker les données
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(content: Text("Inscription réussie !")),
      );
    }
  }

  @override
  Widget build(BuildContext context) {
    return Form(
      key: _formKey,
      child: Column(children: [
        TextFormField(
          decoration: InputDecoration(labelText: "Nom"),
          validator: (v) => v!.isEmpty ? "Nom requis" : null,
          onSaved: (v) => _nom = v!,
        ),
        TextFormField(
          decoration: InputDecoration(labelText: "Email"),
          validator: validateEmail,
          onSaved: (v) => _email = v!,
        ),
        TextFormField(
          obscureText: true,
          decoration: InputDecoration(labelText: "Mot de passe"),
          validator: validatePassword,
          onSaved: (v) => _password = v!,
        ),
        ElevatedButton(onPressed: _submit, child: Text("S'inscrire")),
      ]),
    );
  }
}
✅ Bonnes pratiques
- Toujours utiliser GlobalKey
- Nettoyer les entrées avec trim()
- Donner des messages d'erreur clairs
⚠️ Erreurs fréquentes
- Oublier onSaved
- Ne pas gérer les types nullables
- Valider côté front sans back-up
