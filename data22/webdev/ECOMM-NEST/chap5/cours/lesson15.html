Gestion du panier d'achat

Entités Cart & CartItem, liaison utilisateur, panier anonyme

Objectifs de la leçon

  • ✅ Créer les entités Cart et CartItem
  • ✅ Lier le panier à l'utilisateur (relation OneToOne)
  • ✅ Gérer le panier anonyme avec session/visitorId
  • ✅ Fusionner le panier anonyme lors de la connexion
💡 Un panier peut exister sans utilisateur connecté (via un identifiant temporaire). À la connexion, on fusionne les deux paniers.

🛒 1. Entité Cart

// cart.entity.ts
import { Entity, PrimaryGeneratedColumn, OneToOne, OneToMany, Column, JoinColumn } from 'typeorm';
import { User } from './user.entity';
import { CartItem } from './cart-item.entity';

@Entity()
export class Cart {
    @PrimaryGeneratedColumn('uuid')
    id: string;

    @OneToOne(() => User, user => user.cart, { nullable: true })
    @JoinColumn()
    user: User | null;

    @Column({ nullable: true })
    visitorId: string;  // Pour le panier anonyme

    @OneToMany(() => CartItem, item => item.cart, { cascade: true })
    items: CartItem[];

    @Column({ type: 'timestamp', default: () => 'CURRENT_TIMESTAMP' })
    createdAt: Date;

    @Column({ type: 'timestamp', default: () => 'CURRENT_TIMESTAMP', onUpdate: 'CURRENT_TIMESTAMP' })
    updatedAt: Date;
}

📦 2. Entité CartItem

// cart-item.entity.ts
import { Entity, PrimaryGeneratedColumn, ManyToOne, Column } from 'typeorm';
import { Cart } from './cart.entity';
import { Product } from './product.entity';

@Entity()
export class CartItem {
    @PrimaryGeneratedColumn('uuid')
    id: string;

    @ManyToOne(() => Cart, cart => cart.items)
    cart: Cart;

    @ManyToOne(() => Product)
    product: Product;

    @Column({ default: 1 })
    quantity: number;

    // Variation (taille, couleur) - stockée en JSON
    @Column({ type: 'json', nullable: true })
    variation: Record;
}
📌 Le champ variation permet de stocker les options choisies (ex: {"size": "L", "color": "rouge"}).

✅ Chaque utilisateur a un panier unique (OneToOne). À l'inscription, on crée automatiquement son panier vide.

👤 4. Panier anonyme (visitorId)

Pour les visiteurs non connectés, on utilise un identifiant temporaire stocké dans le localStorage ou envoyé dans les headers.

// Récupération ou création du panier
async getOrCreateCart(visitorId?: string, userId?: string): Promise<Cart> {
    if (userId) {
        // Utilisateur connecté : panier lié à son compte
        const user = await this.userService.findOne(userId);
        if (user.cart) return user.cart;
        return this.createCartForUser(userId);
    }
    
    if (visitorId) {
        let cart = await this.cartRepository.findOne({ where: { visitorId }, relations: ['items', 'items.product'] });
        if (cart) return cart;
    }
    
    // Créer un nouveau panier anonyme
    return this.cartRepository.save({ visitorId: visitorId || uuidv4() });
}

🔄 5. Fusion panier anonyme → connecté

async mergeCarts(visitorId: string, userId: string): Promise<Cart> {
    const anonymousCart = await this.cartRepository.findOne({ where: { visitorId }, relations: ['items'] });
    const userCart = await this.getCartByUserId(userId);
    
    if (!anonymousCart) return userCart;
    
    // Fusionner les articles
    for (const item of anonymousCart.items) {
        await this.addItemToCart(userCart.id, item.productId, item.quantity, item.variation);
    }
    
    // Supprimer le panier anonyme
    await this.cartRepository.remove(anonymousCart);
    return this.getCartWithItems(userCart.id);
}
✨ Appeler cette méthode juste après la connexion (login ou register).

⚙️ 6. Service Cart - méthodes essentielles

// Ajouter un produit
async addItem(cartId: string, productId: string, quantity: number, variation?: any) {
    const cart = await this.getCart(cartId);
    const product = await this.productService.findOne(productId);
    
    let item = cart.items.find(i => i.product.id === productId && JSON.stringify(i.variation) === JSON.stringify(variation));
    if (item) {
        item.quantity += quantity;
    } else {
        item = this.cartItemRepository.create({ cart, product, quantity, variation });
    }
    return this.cartItemRepository.save(item);
}
