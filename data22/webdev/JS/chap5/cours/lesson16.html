🧱 Créer un tableau

Un tableau est une structure qui peut contenir plusieurs valeurs, même de types différents.

📐 Crochets (recommandé)

let fruits = ["🍎", "🍌", "🍒"];
let mixte = ["Bonjour", 42, true, null];
let vide = [];

🏗️ Constructeur Array

let nombres = new Array(1, 2, 3);
let taille5 = new Array(5); // 5 éléments vides

⚠️ Attention: new Array(3) crée un tableau de longueur 3, sans valeurs.

🎯 Essayez dans la console (ou ci-dessous) :

🔧 Manipulation de base

Accéder à un élément par son indice (tableau[index]) et modifier sa valeur.

let couleurs = ["rouge", "vert", "bleu"];
console.log(couleurs[0]); // "rouge"
couleurs[1] = "jaune"; // maintenant ["rouge","jaune","bleu"]

📥 Ajouter / Supprimer aux extrémités

MéthodeActionModification
push(élément)Ajoute à la fin→ nouvelle longueur
pop()Retire le dernier élément→ retourne l'élément retiré
unshift(élément)Ajoute au début→ nouvelle longueur
shift()Retire le premier élément→ retourne l'élément retiré

📏 La propriété length

length renvoie le nombre d’éléments. On peut aussi la modifier pour tronquer ou étendre le tableau.

let animaux = ["🐕", "🐈", "🐁"];
console.log(animaux.length); // 3
animaux.length = 2; // supprime le dernier élément → ["🐕","🐈"]
animaux.length = 5; // ajoute des emplacements vides

Tableaux hétérogènes

Un tableau peut contenir des nombres, chaînes, booléens, objets, voire d'autres tableaux !

let mixte = ["Jean", 25, true, ["sous-tableau", 99]];

💡 Bon à savoir : Les indices commencent toujours à 0 et le dernier index est tableau.length - 1.

📝 Récapitulons les essentiels

  • Les tableaux s’écrivent avec [valeur1, valeur2] et stockent des listes ordonnées.
  • Accès via l’indice : monTableau[0].
  • push/pop agissent à la fin ; unshift/shift agissent au début.
  • La propriété .length est automatiquement mise à jour.
  • Tableaux hétérogènes acceptent tout type de données (mélange possible).
  • Parcours fréquent : boucle for ou méthode forEach (vu dans prochaines leçons).
