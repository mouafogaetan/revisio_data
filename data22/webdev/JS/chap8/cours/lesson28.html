Les chaînes de caractères modernes

Template literals, interpolation, méthodes modernes (includes, startsWith, endsWith, repeat)

🎯 Niveau : Débutant / Intermédiaire ⏱️ Durée : 35 min 📜 ES6+ / JavaScript moderne

📌 Objectifs d'apprentissage

À la fin de cette leçon, vous serez capable de :

  • ✓ Utiliser les backticks (` `) pour créer des chaînes multi-lignes.
  • ✓ Intégrer des expressions dans des chaînes avec ${...} (interpolation).
  • ✓ Maîtriser les nouvelles méthodes : includes(), startsWith(), endsWith(), repeat().
  • ✓ Différencier les template literals des anciennes concaténations.

💡 Pourquoi c'est important ? Les template literals rendent le code plus lisible et moins sujet aux erreurs, surtout pour les chaînes dynamiques ou multi-lignes (HTML, messages, etc.).

📝 Template literals : la nouvelle façon d'écrire des chaînes

Avant ES6, on utilisait les guillemets simples ' ' ou doubles " ". Désormais, on peut utiliser les backticks (` `) pour créer des template literals.

// Ancienne méthode (concaténation)
let nom = "Alice";
let message = "Bonjour " + nom + " !";

// Nouvelle méthode (template literal)
let nouveauMessage = `Bonjour ${nom} !`;

✅ Avantages :

  • Plus besoin d'échapper les guillemets à l'intérieur.
  • Interpolation directe avec ${...}.
  • Supports des retours à la ligne sans concaténation.

🧮 Interpolation : ${expression}

À l'intérieur des ${}, on peut placer n'importe quelle expression JavaScript valide : variable, opération arithmétique, appel de fonction, etc.

let a = 5, b = 3;
let somme = `La somme de ${a} et ${b} est égale à ${a + b}.`;
console.log(somme); // "La somme de 5 et 3 est égale à 8."

let prix = 19.99;
let tva = 0.2;
let total = `Total TTC : ${prix * (1 + tva)} €`;
console.log(total); // "Total TTC : 23.988 €"
⚠️ Attention : Les backticks ne supportent pas l'échappement classique des guillemets à l'intérieur, mais vous pouvez utiliser les guillemets sans problème.

📄 Chaînes multi-lignes sans concaténation

Avec les backticks, on peut écrire du texte sur plusieurs lignes naturellement. Plus besoin de \n ou de concaténations.

let poeme = `
  Sur la route poussiéreuse,
  Le vent chante doucement.
  ${new Date().getFullYear()} - Poème moderne
`;
console.log(poeme);

💡 Cas pratique :

Idéal pour générer du HTML dynamique :

let html = `
  <div class="card">
    <h2>${titre}</h2>
    <p>${description}</p>
  </div>
`;

🔍 includes() : vérifier la présence d'une sous-chaîne

includes() retourne true si la chaîne contient la sous-chaîne spécifiée, false sinon.

let phrase = "Bonjour tout le monde !";
console.log(phrase.includes("tout"));     // true
console.log(phrase.includes("Bonjour"));  // true
console.log(phrase.includes("JavaScript")); // false

// Avec position de départ optionnelle
console.log(phrase.includes("monde", 10)); // true (à partir de l'index 10)

Utilité : Recherche de mots-clés, validation de contenu, filtrage.

🚦 startsWith() et endsWith()

Ces méthodes vérifient si une chaîne commence ou se termine par une sous-chaîne donnée.

let url = "https://mon-site.com/page";
console.log(url.startsWith("https"));  // true
console.log(url.endsWith(".com"));     // true

let email = "user@example.com";
if (email.endsWith("@example.com")) {
  console.log("Email professionnel");
}

// Avec position de départ pour startsWith
console.log(url.startsWith("mon-site", 8)); // true (à partir de l'index 8)

🔧 Utilisation courante :

Validation de protocole (http/https), vérification d'extension de fichier, routage simple.

🔄 repeat() : répéter une chaîne

repeat(n) retourne une nouvelle chaîne contenant la chaîne originale répétée n fois.

let etoiles = "*";
console.log(etoiles.repeat(5)); // "*****"

let ligne = "-";
console.log(ligne.repeat(10));  // "----------"

// Générer une barre de progression simple
let progression = 7;
let barre = "█".repeat(progression) + "░".repeat(10 - progression);
console.log(barre); // "███████░░░"

Note : si n = 0, la chaîne vide est retournée. Si n est négatif, une erreur est levée.

⚖️ Template literals : avantages concrets

❌ Ancienne façon (lourde)

let message = "Bonjour " + nom + ",\n" +
              "Votre commande n°" + id + " est prête.\n" +
              "Total : " + (prix * quantite) + " €";

✅ Moderne (lisible)

let message = `
  Bonjour ${nom},
  Votre commande n°${id} est prête.
  Total : ${prix * quantite} €
`;
  • Lisibilité accrue, moins d'erreurs de concaténation.
  • Gestion naturelle des sauts de ligne.
  • Interpolation de n'importe quelle expression (fonctions, calculs).
