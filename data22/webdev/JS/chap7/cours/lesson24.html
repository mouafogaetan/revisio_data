🎯 Objectifs d'apprentissage

  • 🔹 Comprendre les 3 méthodes pour attacher un événement click : attribut HTML (à éviter), propriété .onclick et addEventListener().
  • 🔹 Maîtriser addEventListener() et removeEventListener() comme meilleure pratique.
  • 🔹 Savoir gérer plusieurs événements sur un même élément et les retirer proprement.
  • 🔹 Différencier les approches pour écrire un code maintenable et moderne.
💡 Pourquoi addEventListener ? Il permet d'ajouter plusieurs gestionnaires, un meilleur contrôle et s'intègre parfaitement avec le flux d'événements.

📌 1. L'attribut HTML onclick

Méthode historique, simple pour des tests mais déconseillée dans des applications modernes :

<button onclick="alert('Clic!')">Clique-moi</button>

❌ Problèmes : mélange structure/comportement, difficile à maintenir, pas de séparation des couches, impossibilité d'ajouter plusieurs fonctions.

⚠️ À éviter pour des projets réels, mais utile pour des prototypes rapides.

📌 2. Propriété .onclick en JavaScript

On sélectionne l'élément et on lui assigne une fonction. C'est mieux que l'attribut, mais on ne peut associer qu'un seul gestionnaire par événement.

// Récupérer le bouton const monBtn = document.getElementById('btnEx'); monBtn.onclick = function() { console.log('Click via onclick'); };
✅ Avantage : Séparation HTML/JS.
❌ Inconvénient : si une autre partie du code écrase .onclick, la première fonction disparaît.

⭐ 3. addEventListener() & removeEventListener()

La méthode recommandée pour gérer les événements : elle permet d'ajouter plusieurs écouteurs sur le même événement, et de les retirer facilement.

element.addEventListener('click', maFonction); // Retirer l'écouteur plus tard element.removeEventListener('click', maFonction);

🎓 Avantages clés :

  • Ajout de multiples fonctions sur un même événement.
  • Contrôle fin avec des options (capture, une fois).
  • Meilleure gestion de la mémoire et du cycle de vie.
  • Respecte la séparation des couches.
💡 Bon réflexe : toujours utiliser addEventListener dans vos projets JavaScript modernes.
