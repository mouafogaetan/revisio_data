📜 Méthodes classiques de sélection

📌 document.getElementById('monId')

Sélectionne un unique élément par son attribut id. Rapide et précis.

// HTML : <div id="banniere">Bienvenue</div>
let element = document.getElementById('banniere');
element.style.backgroundColor = "lightblue";

📌 document.getElementsByClassName('maClasse')

Retourne une collection vivante (HTMLCollection) de tous les éléments ayant cette classe.

let items = document.getElementsByClassName('carte');
// items[0] → premier élément, attention pas de forEach direct (Array.from)

📌 document.getElementsByTagName('p')

Sélectionne tous les paragraphes, ou toutes les balises d’un type.

💡 Remarque importante : getElementsByClassName et getElementsByTagName retournent des collections vivantes (mises à jour automatiquement si le DOM change).

⚡ querySelector & querySelectorAll

Utilisez des sélecteurs CSS pour une flexibilité maximale !

🔍 document.querySelector('.ma-classe')

Retourne le premier élément correspondant au sélecteur CSS.

// Premier élément avec la classe 'menu-item'
const firstMenuItem = document.querySelector('.menu-item');

📚 document.querySelectorAll('div.article')

Retourne une NodeList statique (forEach possible directement).

const articles = document.querySelectorAll('.article');
articles.forEach(art => art.style.border = '1px solid red');

✅ Avantages :

  • Sélecteurs complexes : #menu li > a.active
  • Compatible avec tous les navigateurs modernes
  • Plus lisible et maintenable

🧭 Naviguer dans l’arbre DOM

Une fois un élément sélectionné, on peut se déplacer autour de lui.

PropriétéDescription
parentNodeL'élément parent direct
childrenCollection des éléments enfants (balises HTML)
nextSibling / previousSiblingNœud suivant/précédent (peut inclure textes/espaces)
nextElementSibling / previousElementSiblingÉlément suivant/précédent (ignore les textes)
firstChild / lastChildPremier/dernier nœud enfant

✨ Astuce : pour éviter les nœuds texte, préférez children, parentElement et nextElementSibling.
