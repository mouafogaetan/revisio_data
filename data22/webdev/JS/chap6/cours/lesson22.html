🏗️ Créer des éléments : createElement()

Le DOM (Document Object Model) nous permet de créer des noeuds HTML à la volée. La méthode reine est document.createElement(tagName).

✨ Exemple :

// Créer un nouveau paragraphe
let nouveauP = document.createElement('p');
nouveauP.textContent = 'Je suis un paragraphe dynamique !';

// Créer un texte séparé
let texte = document.createTextNode('Hello DOM');
divConteneur.appendChild(texte);

✅ Bon à savoir : Les éléments créés n’apparaissent pas immédiatement à l’écran tant qu'on ne les ajoute pas dans l’arbre DOM.

➕ Ajouter au DOM : les 4 méthodes essentielles

appendChild()

Ajoute un noeud comme dernier enfant.

append()

Ajoute plusieurs noeuds ou chaînes à la fin (moderne).

prepend()

Ajoute au début de l’élément cible.

insertBefore()

Insère avant un élément de référence.

// Exemples concrets
let parent = document.getElementById("maListe");
let nouvelItem = document.createElement("li");
nouvelItem.textContent = "Nouveau point";
parent.appendChild(nouvelItem);  // ajout à la fin

parent.prepend("Début du texte"); // ajoute du texte avant
parent.append("Fin", document.createElement("span"));
🎓 jQuery simplifie aussi : $("#conteneur").append("<p>Ajouté avec jQuery</p>"); mais on maîtrise le JS pur !

❌ Supprimer des éléments du DOM

  • removeChild() : méthode classique sur le parent pour supprimer un enfant spécifique.
  • remove() : méthode moderne (ES6+) directement sur l’élément à supprimer, plus intuitive !
// 1) Avec removeChild
let parent = document.getElementById("zone");
let enfant = document.getElementById("itemASupprimer");
parent.removeChild(enfant);

// 2) La méthode élégante (recommandée)
let element = document.querySelector(".a-supprimer");
element.remove();   // direct, simple !
⚠️ Attention : remove() n’est pas supportée par Internet Explorer, mais tous les navigateurs modernes (Chrome, Firefox, Edge) l’acceptent.

✨ jQuery simplifie aussi la manipulation

jQuery offre des méthodes très concises pour créer et injecter :

// Créer un élément avec jQuery
let $newDiv = $("<div>").text("jQuery power").addClass("alert");
$("#conteneur").append($newDiv);
// Supprimer facilement
$(".item").remove();

💡 Bonne pratique :

Préfère element.remove() en JS moderne. Si tu utilises jQuery, .remove() supprime aussi les événements associés.

📦 Performance :

Pour ajouter plusieurs éléments, utilise un DocumentFragment ou réduis les reflows.
