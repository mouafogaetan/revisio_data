🎨 Leçon 23 : Modifier les styles CSS en direct

Slide 1 /

Manipulation dynamique des styles CSS

Apprenez à modifier l'apparence d'une page web en direct grâce à JavaScript et jQuery, pour créer des interfaces vivantes et réactives.

🎯 .style 📐 getComputedStyle 📏 Dimensions

🎯 Objectifs de la leçon

À la fin de ce cours, vous serez capable de :

  • ✓ Modifier les styles inline avec la propriété .style en JavaScript / jQuery (.css()).
  • ✓ Obtenir les styles calculés (getComputedStyle()) pour lire les valeurs CSS effectives.
  • ✓ Accéder aux dimensions d’un élément (clientWidth, offsetWidth, scrollHeight) et comprendre leurs différences.
  • ✓ Appliquer ces techniques pour créer des animations simples et responsive.
💡 Pourquoi c'est utile ? Rendre une page interactive, changer de couleurs, afficher/cacher, agrandir/réduire, ou réagir aux actions de l'utilisateur.

✍️ 1. Modifier les styles inline

En JavaScript pur, chaque élément possède une propriété .style qui permet de lire/modifier les styles inline (ceux écrits dans l'attribut style="...").

// JS vanilla
const monElement = document.getElementById("demo");
monElement.style.backgroundColor = "lightblue";
monElement.style.fontSize = "20px";

// Avec jQuery : méthode .css()
$("#demo").css("color", "red");
$("#demo").css({
  "background-color": "yellow",
  "padding": "10px"
});
⚠️ Attention : les noms de propriétés CSS en camelCase (ex: backgroundColor au lieu de background-color).

🔍 2. Lire les styles calculés (getComputedStyle)

La propriété .style ne renvoie que les styles inline. Pour obtenir le style final appliqué par une feuille CSS ou par le navigateur, on utilise getComputedStyle().

const element = document.querySelector(".ma-classe");
const styles = getComputedStyle(element);
console.log(styles.color);        // couleur réelle
console.log(styles.marginTop);

Avec jQuery, on peut également récupérer la valeur calculée avec .css() (car jQuery lit les styles calculés).

📌 Exemple : $("#maDiv").css("width") retourne la largeur réelle (même si définie en CSS externe).

📏 3. Dimensions des éléments

JavaScript permet d'accéder aux dimensions réelles des éléments, utiles pour le positionnement, les animations ou la détection responsive.

clientWidth / clientHeight

Largeur/hauteur intérieure (contenu + padding, sans barre de défilement, sans bordure)

offsetWidth / offsetHeight

Dimensions totales (contenu + padding + bordure + barre défilement éventuelle)

scrollWidth / scrollHeight

Taille totale du contenu défilable (même si caché)

const box = document.getElementById("box");
console.log("clientWidth :", box.clientWidth);
console.log("offsetWidth :", box.offsetWidth);
console.log("scrollHeight :", box.scrollHeight);

// En jQuery : $element.width() / .outerWidth() / .innerWidth()

📌 Résumé des concepts clés

🎨 element.style.propriété – Modifie le style inline (prioritaire).
📐 getComputedStyle(element) – Lit le style réel après toutes les règles CSS.
📏 Dimensions : clientWidth/Height (padding inclus), offsetWidth/Height (+bordures), scrollWidth/Height (contenu défilable).
⚡ jQuery : .css() pour lire/écrire; .width(), .outerWidth() pour les dimensions.
🧠 Astuce professeur : manipulez les classes CSS via classList pour des changements groupés – mais la leçon 23 se concentre sur les styles inline et les dimensions calculées.

🏠 Petit exercice d'application

Créez une petite page où un carré change de taille et de couleur selon le mouvement de la souris, en utilisant getComputedStyle pour afficher ses dimensions réelles.

🔧 Défi :

  • Un div rond (ou carré) de 100px initial.
  • Au clic, double sa largeur/hauteur via .style.
  • Afficher son offsetWidth dans un paragraphe.
  • Bonus : un bouton "reset" qui restaure la taille d'origine.
✅ La pratique est essentielle ! Amusez-vous à modifier les styles en direct, c’est la base du JavaScript interactif.
