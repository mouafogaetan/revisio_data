📝 1. Modifier le contenu textuel / HTML

🔹 textContent

Permet de lire ou modifier le texte brut d’un élément. Il ignore les balises HTML (sécurité contre les injections XSS).

// Exemple : document.getElementById("demo").textContent = "Nouveau texte";

🔹 innerHTML

Lit ou définit le contenu HTML à l’intérieur d’un élément. Interprète les balises, donc plus puissant mais potentiellement risqué avec du contenu utilisateur.

⚠️ Attention : si vous insérez du contenu non fiable, préférez textContent ou désinfectez.

🏷️ 2. Manipuler les attributs HTML

Les méthodes getAttribute(), setAttribute() et removeAttribute() permettent de modifier dynamiquement les attributs comme src, href, alt ou même des attributs personnalisés.

// Récupérer la valeur
let href = lien.getAttribute('href');
// Modifier ou ajouter
img.setAttribute('src', 'photo.jpg');
// Supprimer un attribut
bouton.removeAttribute('disabled');

🎨 3. Modifier les classes CSS avec classList

classList est une interface moderne pour manipuler les classes. Elle fournit des méthodes puissantes sans toucher à la chaîne de classe brute.

  • element.classList.add('maClasse') – Ajoute une classe
  • element.classList.remove('maClasse') – Supprime une classe
  • element.classList.toggle('maClasse') – Ajoute si absente, supprime si présente
  • element.classList.contains('maClasse') – Vérifie la présence
