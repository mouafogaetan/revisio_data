La condition if : la base des décisions

Imaginez : "S'il pleut, je prends un parapluie". En JavaScript, c’est pareil : on teste une expression qui donne true ou false.

// Structure simple
if (condition) {
  // code exécuté si condition est vraie
}

🔍 Exemple concret :

let age = 18;
if (age >= 18) {
  console.log("Vous êtes majeur 🎉");
}

👉 Ici, la condition age >= 18 est vraie → le message s’affiche.

✅ À retenir : les conditions utilisent souvent les opérateurs de comparaison : >, <, >=, ===, != ...

🛤️

Ajouter else : l’alternative

Parfois on veut faire une chose si la condition est vraie, et une autre chose si elle est fausse. else permet d’exécuter un second bloc.

if (condition) {
  // si vrai
} else {
  // si faux
}

🌞 Exemple météo :

let heure = 14;
if (heure < 12) {
  console.log("Bonjour matin 🌅");
} else {
  console.log("Bonjour après-midi ☀️");
}

🎫 Exemple vérification :

let age = 16;
if (age >= 18) {
  console.log("Accès autorisé");
} else {
  console.log("Accès refusé, trop jeune");
}
🔀

else if : enchaîner les cas

Quand tu as plus de deux possibilités, else if vient à la rescousse. Il permet d’ajouter des conditions intermédiaires.

if (condition1) {
  // cas 1
} else if (condition2) {
  // cas 2
} else if (condition3) {
  // cas 3
} else {
  // aucun des cas ci-dessus
}

📊 Exemple : Appréciation d’une note scolaire

let note = 75;
if (note >= 90) {
  console.log("Excellent 🏆");
} else if (note >= 70) {
  console.log("Très bien 👍");
} else if (note >= 50) {
  console.log("Moyen 😐");
} else {
  console.log("Reprends les bases 📖");
}

➡️ Ici, la note 75 active le bloc else if (note >= 70).

⚠️ L'ordre compte : la première condition vraie s'exécute, puis on sort !
❓

L'opérateur ternaire ? : (condensed)

Pour des affectations simples, le ternaire raccourcit l’écriture. Syntaxe : condition ? valeurSiVrai : valeurSiFaux

Avec if/else classique:

let age = 20;
let statut;
if (age >= 18) {
  statut = "majeur";
} else {
  statut = "mineur";
}

✨ Version ternaire ✨

let age = 20;
let statut = (age >= 18) ? "majeur" : "mineur";
// statut vaut "majeur"

🔁 Autre exemple : afficher un message différent selon une variable booléenne

let estConnecte = true;
let message = estConnecte ? "Bienvenue membre" : "Veuillez vous connecter";
console.log(message);  // "Bienvenue membre"

⚠️ Attention : le ternaire rend le code plus court, mais ne l'utilisez pas pour des blocs complexes avec plusieurs instructions. Préférez if/else pour la lisibilité.

⚖️

Quand utiliser quoi ?

  • if / else if / else : pour des logiques complexes, plusieurs cas, plusieurs lignes d’action. Idéal pour les débutants et la clarté.
  • Ternaire ( ? : ) : pour des affectations simples, quand vous avez besoin de retourner une valeur sur une seule ligne. Éviter d’imbriquer les ternaires (rend le code difficile).

✍️ Petite astuce : si vous écrivez 3 ternaires dans le même code, demandez-vous si un if/else ne serait pas plus lisible.

📌 Exemple réel : let prixFinal = estMembre ? prix * 0.9 : prix; (réduction 10% si membre)
