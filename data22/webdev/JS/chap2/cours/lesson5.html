Les chaînes de caractères (string)

En JavaScript, on écrit du texte avec des guillemets simples '...', doubles "..." ou backticks \`...\` (template literals).

let prenom = 'Aicha';      // simple
let message = "Bonjour!";  // double
let pays = `Cameroun`;     // backtick

// Avec backticks, on peut insérer des variables: ${...}
let age = 20;
let phrase = `Je m'appelle ${prenom} et j'ai ${age} ans.`;
console.log(phrase);

💡 Astuce: Les backticks permettent aussi d'écrire du texte sur plusieurs lignes sans concaténation !

🔢

Le type Number (entiers, décimaux, spéciaux)

JavaScript utilise un seul type pour tous les nombres : number. Il gère aussi les valeurs spéciales :

  • NaN (Not a Number) : résultat d'une opération mathématique invalide.
  • Infinity et -Infinity : valeurs infinies.
let entier = 42;
let decimal = 3.14;
let negatif = -7;
let infini = 1 / 0;        // Infinity
let erreurMath = "salut" * 3;   // NaN

console.log(typeof entier);     // "number"
console.log(isNaN(erreurMath)); // true

✨

Symbol & BigInt

Symbol : permet de créer des identifiants uniques. BigInt : pour les très grands nombres entiers.

// Symbol : unicité
const id = Symbol("id");
const id2 = Symbol("id");
console.log(id === id2); // false

// BigInt : ajouter 'n' à la fin
let grandNombre = 9007199254740991n;
let operations = 10n + 5n;
console.log(typeof grandNombre); // "bigint"
⚡ Les BigInt sont utiles pour les gros calculs précis (identifiants, timestamp longs, etc.)
🔍

Inspecter les types avec typeof

typeof est un opérateur qui renvoie le type d'une valeur sous forme de chaîne.

console.log(typeof "Cameroun");   // "string"
console.log(typeof 2025);         // "number"
console.log(typeof true);         // "boolean"
console.log(typeof undefined);    // "undefined"
console.log(typeof Symbol());     // "symbol"
console.log(typeof 123n);         // "bigint"

📌 À retenir : typeof null retourne "object" (bug historique). Pour tester si une variable est vraiment null, on utilise valeur === null.
