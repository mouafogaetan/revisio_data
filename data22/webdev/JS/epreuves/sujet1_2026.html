2 Intégration & Débogage

8 points

On vous donne le code HTML suivant :

<!DOCTYPE html>
<html>
<head>
  <script src="script.js"></script>
</head>
<body>
  <h1>Bonjour</h1>
  <script>
    console.log("Message depuis le body");
  </script>
</body>
</html>

Questions :

  1. Le fichier script.js est chargé dans le <head>. Que va-t-il se passer si script.js tente d’accéder à l’élément <h1> ? Pourquoi ?
  2. Proposez deux solutions pour que le script puisse accéder à l’élément <h1>.
  3. À quoi sert la console du navigateur ? Citez deux méthodes de débogage utilisables dans la console.
  4. Quelle méthode JavaScript affiche une boîte de dialogue avec un message et un champ de saisie ?

3 Interactions utilisateur

6 points

Écrivez un petit script JavaScript qui :

  1. Demande à l’utilisateur son âge avec prompt().
  2. Si l’âge est supérieur ou égal à 18, affiche une boîte de confirmation (confirm()) lui demandant “Voulez-vous accéder au contenu adulte ?”.
  3. Si l’utilisateur confirme, affiche une alerte “Accès autorisé”. Sinon, affiche “Accès refusé”.
  4. Si l’âge est inférieur à 18, affiche simplement “Désolé, vous êtes mineur.”.

⚡ Commandes console utiles

  • console.log(variable) → affiche une valeur
  • console.error("erreur") → affiche en rouge
  • console.table(tableau) → affiche sous forme de tableau
  • console.time() / console.timeEnd() → mesure de performance
