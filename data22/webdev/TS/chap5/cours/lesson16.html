🏛️ Déclaration d'une classe

Une classe est un plan qui définit des propriétés (données) et des méthodes (comportements).

class Etudiant {
  // Propriétés
  nom: string;
  age: number;
  niveau: string;

  // Constructeur : appelé lors de la création d'un objet
  constructor(nom: string, age: number, niveau: string) {
    this.nom = nom;
    this.age = age;
    this.niveau = niveau;
  }

  // Méthode
  afficherInfos(): void {
    console.log(`${this.nom}, ${this.age} ans - Niveau ${this.niveau}`);
  }
}

// Instanciation
let eleve = new Etudiant("Fatima", 20, "Licence");
eleve.afficherInfos();
🔍 Remarque : this fait référence à l'instance courante.

⚡ Raccourci de propriétés (Parameter properties)

TypeScript offre une syntaxe concise : déclarez et assignez directement dans le constructeur grâce aux modificateurs (public, private, readonly).

// Version compacte, ultra-lisible
class Professeur {
  constructor(
    public nom: string,
    public matiere: string,
    public readonly anciennete: number
  ) {}

  enseigner() {
    console.log(`${this.nom} enseigne ${this.matiere}`);
  }
}

const prof = new Professeur("M. Kamga", "TypeScript", 5);
prof.enseigner();
✅ Avantage : on évite de répéter this.nom = nom;. TypeScript génère automatiquement les propriétés.

📚 Méthodes et mot-clé this

Les méthodes peuvent accéder aux propriétés via this et interagir entre elles.

class CompteBancaire {
  constructor(public titulaire: string, private solde: number) {}

  deposer(montant: number): void {
    this.solde += montant;
    console.log(`💰 Nouveau solde: ${this.solde} FCFA`);
  }

  retirer(montant: number): boolean {
    if (montant <= this.solde) {
      this.solde -= montant;
      return true;
    }
    return false;
  }

  consulterSolde(): number {
    return this.solde;
  }
}

let compte = new CompteBancaire("Alice", 50000);
compte.deposer(10000);

✍️ Comparaison : écriture classique vs moderne

📜 Classique

class Vehicule {
  marque: string;
  annee: number;
  constructor(m: string, a: number) {
    this.marque = m;
    this.annee = a;
  }
}

⚡ Raccourci (TypeScript)

class Vehicule {
  constructor(public marque: string, public annee: number) {}
}

Le raccourci réduit la redondance, améliore la lisibilité et reste parfaitement typé.

✅ Ce qu'il faut retenir

  • Une classe TypeScript = propriétés + constructeur + méthodes.
  • Le constructeur initialise l'état de l'objet (mot-clé new).
  • Le paramètre raccourci (constructor(public nom: string)) déclare et assigne automatiquement.
  • this est essentiel pour accéder aux membres de l'instance.
  • Les classes permettent un code modulaire, réutilisable et orienté objet.
