Authentification JWT avec NestJS

🎯 Objectifs de la leçon

  • ✓ Installer les packages nécessaires : @nestjs/jwt, @nestjs/passport, passport-jwt.
  • ✓ Créer un module Auth et un service d'authentification.
  • ✓ Générer et valider des tokens JWT.
  • ✓ Comprendre la structure d'un token (Header, Payload, Signature).
JWT (JSON Web Token) est une norme ouverte qui permet de transmettre des informations de manière sécurisée entre parties.

📦 Installation des dépendances

Dans un projet NestJS, exécutez :

npm install @nestjs/jwt @nestjs/passport passport passport-jwt
npm install @types/passport-jwt --save-dev

🔧 Configuration du module JWT

// auth.module.ts
import { JwtModule } from '@nestjs/jwt';

@Module({
  imports: [
    JwtModule.register({
      secret: 'maSuperCleSecrete',
      signOptions: { expiresIn: '1d' },
    }),
  ],
})
export class AuthModule {}

⚠️ En production, utilisez une variable d'environnement pour le secret !

🔐 Créer le service AuthService

// auth.service.ts
import { Injectable } from '@nestjs/common';
import { JwtService } from '@nestjs/jwt';

@Injectable()
export class AuthService {
  constructor(private jwtService: JwtService) {}

  async login(user: any) {
    const payload = { username: user.username, sub: user.userId };
    return {
      access_token: this.jwtService.sign(payload),
    };
  }

  // Vérification simplifiée (à remplacer par une vraie validation)
  async validateUser(username: string, pass: string) {
    if (username === 'admin' && pass === 'password') {
      return { userId: 1, username: 'admin' };
    }
    return null;
  }
}
La méthode sign() génère un token JWT à partir du payload.

🔍 Anatomie d'un token JWT

Un JWT est composé de 3 parties séparées par des points :

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.
eyJ1c2VybmFtZSI6ImFkbWluIiwic3ViIjoxfQ.
SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
Header
Algorithme + type
Payload
Données (username, sub...)
Signature
Vérification d'intégrité

La signature permet de garantir que le token n'a pas été modifié.

🌐 Créer le contrôleur Auth

// auth.controller.ts
import { Controller, Post, Body, UnauthorizedException } from '@nestjs/common';
import { AuthService } from './auth.service';

@Controller('auth')
export class AuthController {
  constructor(private authService: AuthService) {}

  @Post('login')
  async login(@Body() body: { username: string; password: string }) {
    const user = await this.authService.validateUser(body.username, body.password);
    if (!user) {
      throw new UnauthorizedException('Identifiants invalides');
    }
    return this.authService.login(user);
  }
}
Exemple de requête POST /auth/login avec un body JSON contenant {"username":"admin","password":"password"} → renvoie {"access_token":"eyJ..."}.
