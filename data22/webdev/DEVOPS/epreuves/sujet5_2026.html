📁 Structure du système de fichiers Linux

📌 Répertoire essentiels : /etc (configuration), /var (logs, données variables), /home (utilisateurs).

🔧 Exercice pratique : droits Linux (chmod)

Un fichier script.sh a les permissions actuelles : -rw-r--r-- (644). Quel est l'effet de chmod 750 script.sh ?

💡 Indice : 7 = rwx (4+2+1), 5 = r-x (4+0+1), 0 = ---
