2 Docker : Empaquetage & Orchestration (5 pts)

Vous devez containeriser une application Node.js (Express) qui écoute sur le port 3000. Elle utilise une base de données PostgreSQL.

3 Kubernetes : Déploiement & Exposition (5 pts)

Créez les manifests YAML pour déployer l'application précédente sur Kubernetes :

  • Un Deployment avec 3 réplicas de l'image monapp:latest, port 3000.
  • Un Service de type ClusterIP pour exposer l'application en interne.
  • Un Ingress pour exposer l'application sur app.mondomaine.com avec TLS (secret existant).
  • Un ConfigMap pour la variable d'environnement NODE_ENV=production.
