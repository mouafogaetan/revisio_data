2 Catalogue & Filtres (5 pts)

Vous devez implémenter un catalogue de produits avec barre de recherche et filtrage par catégorie. Écrivez le composant ProductCatalog qui :

  • Affiche une liste de produits reçue via props.
  • Intègre une barre de recherche (filtre par nom avec debounce de 300ms).
  • Intègre des boutons de filtre par catégorie (Tous, Électronique, Vêtements, Maison).
  • Affiche un message "Aucun produit trouvé" si la liste est vide.
  • Utilise un composant ProductCard pour chaque produit (affiche image, nom, prix, bouton "Ajouter au panier").

3 Panier & Processus d'achat (5 pts)

Répondez aux questions suivantes en vous basant sur le programme (chapitres 8 et 9).

🔹 3.1 Créez un store Zustand pour le panier avec les actions suivantes :

  • addItem(item) : ajoute ou incrémente la quantité
  • removeItem(id) : supprime un article
  • updateQuantity(id, quantity) : modifie la quantité (min 1)
  • clearCart() : vide le panier
  • total : propriété dérivée pour le montant total

🔹 3.2 Intégration Stripe : décrivez les étapes pour intégrer Stripe Elements dans React (composant CheckoutForm).
