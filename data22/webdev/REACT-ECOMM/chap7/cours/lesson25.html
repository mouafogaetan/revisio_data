Filtrage en temps réel

On écoute l'événement onChange d'un champ texte et on met à jour la liste des résultats. Dans React, on stocke la valeur dans un state local.

const [searchTerm, setSearchTerm] = useState('');
const filteredProducts = products.filter(product =>
  product.name.toLowerCase().includes(searchTerm.toLowerCase())
);

// Affichage dynamique
<input 
  type="text"
  value={searchTerm}
  onChange={(e) => setSearchTerm(e.target.value)}
  placeholder="Rechercher un produit..."
/>
<ul>
  {filteredProducts.map(p => <li key={p.id}>{p.name}</li>)}
</ul>
⚠️ Pour des grandes listes ou des appels API, on évite de filtrer à chaque frappe sans optimisations. D'où le debounce !
⏱️

Debounce : attendre la pause de frappe

Le debounce permet de déclencher une recherche uniquement après un délai d'inactivité (ex: 300ms). Cela réduit les appels serveur et les recalculs.

import { useState, useEffect, useRef } from 'react';

function SearchBar() {
  const [query, setQuery] = useState('');
  const [results, setResults] = useState([]);

  useEffect(() => {
    const delayDebounce = setTimeout(() => {
      if (query) {
        fetch(`/api/search?q=${query}`)
          .then(res => res.json())
          .then(setResults);
      }
    }, 300); // délai de 300ms

    return () => clearTimeout(delayDebounce);
  }, [query]);

  return (
    <input 
      value={query}
      onChange={(e) => setQuery(e.target.value)}
    />
  );
}
✅ Résultat : Quand l'utilisateur tape rapidement, seule la dernière requête compte. Performance garantie.

Autocomplétion intelligente

L'autocomplétion propose des mots-clés ou noms de produits pertinents pendant la saisie. On peut récupérer ces suggestions depuis une API ou un tableau local.

const fetchSuggestions = async (value) => {
  if (value.length < 2) return [];
  const res = await axios.get(`/api/suggest?q=${value}`);
  return res.data; // ex: ["Chemise", "Chaussures", "Chapeau"]
};

// Dans le composant :
const [suggestions, setSuggestions] = useState([]);

useEffect(() => {
  const timer = setTimeout(() => {
    fetchSuggestions(query).then(setSuggestions);
  }, 200);
  return () => clearTimeout(timer);
}, [query]);

// localStorage pour historique récent
const saveRecentSearch = (term) => {
  const recents = JSON.parse(localStorage.getItem('recentSearches') || '[]');
  const updated = [term, ...recents.filter(t => t !== term)].slice(0, 5);
  localStorage.setItem('recentSearches', JSON.stringify(updated));
};

🚀 Mise en œuvre globale

Une barre de recherche performante combine :

  • State local (useState)
  • Debounce avec useEffect + setTimeout
  • Suggestions (autocomplétion) et affichage conditionnel
  • Affichage des tendances lorsque l'input est vide

✨ Exemple d'ordre des étapes : 1. Saisie → 2. Debounce → 3. Requête API/ filtre local → 4. Mise à jour résultats + suggestions → 5. Sauvegarde recherche récente.
