Page produit : images, variations & avis

Afficher un carrousel d'images, un sélecteur de variations (taille/couleur) et la section des avis clients avec note moyenne.

👨‍🏫 Classe: Développement Web | React · Interface e-commerce

🎯 Objectifs de la leçon

  • ✓ Implémenter un carrousel / galerie d'images avec vue principale et miniatures
  • ✓ Créer un sélecteur de variations (taille, couleur, stock disponible)
  • ✓ Afficher la note moyenne et la liste des avis clients
  • ✓ Gérer l'ajout au panier avec la variation choisie

💡 Une page produit bien conçue augmente la conversion : donnez toutes les informations nécessaires !

🖼️ Galerie d'images interactive

Composant ProductGallery : affiche une image principale et des miniatures cliquables.

// components/ProductGallery.tsx
import { useState } from 'react';

export const ProductGallery = ({ images }: { images: string[] }) => {
  const [selected, setSelected] = useState(0);
  return (
    <div className="flex flex-col gap-3">
      <img src={images[selected]} alt="produit" className="w-full h-96 object-cover rounded-xl shadow" />
      <div className="flex gap-2 overflow-x-auto">
        {images.map((img, idx) => (
          <img key={idx} src={img} onClick={() => setSelected(idx)}
            className={`w-20 h-20 object-cover rounded-md cursor-pointer border-2 ${selected === idx ? 'border-indigo-500' : 'border-transparent'}`}
          />
        ))}
      </div>
    </div>
  );
};

🎨 Variations produit

Un produit peut avoir plusieurs variantes (taille S/M/L, couleur rouge/bleu). Il faut afficher les options et mettre à jour le prix / stock.

// composant VariationSelector
interface Variation {
  id: string;
  size: string;
  color: string;
  price: number;
  stock: number;
}

const VariationSelector = ({ variations, onChange, selectedVar }) => {
  return (
    <div className="space-y-3">
      <div>
        <span className="font-semibold">Taille :</span>
        <div className="flex gap-2 mt-1">
          {variations.map(v => (
            <button key={v.id} onClick={() => onChange(v)}
              className={`px-3 py-1 border rounded ${selectedVar?.id === v.id ? 'bg-indigo-600 text-white' : 'bg-gray-100'}`}>
              {v.size}
            </button>
          ))}
        </div>
      </div>
      <p>Stock disponible : {selectedVar?.stock || 0}</p>
      <p className="text-xl font-bold text-indigo-700">{selectedVar?.price} FCFA</p>
    </div>
  );
};

🔁 Dès qu'une variation change, le prix et le stock se mettent à jour. On désactive les options en rupture.

💡 Astuce : Gérer l'état de la variation sélectionnée dans le composant parent (ou store) pour l'ajout au panier.

⭐ Avis et évaluation

Affichage d'une moyenne d'étoiles et liste des commentaires avec React.

// composant RatingStars
const RatingStars = ({ rating }: { rating: number }) => {
  const fullStars = Math.floor(rating);
  const halfStar = rating % 1 >= 0.5;
  return (
    <div className="flex text-yellow-400">
      {[...Array(5)].map((_, i) => (
        <span key={i}>
          {i < fullStars ? '★' : (i === fullStars && halfStar ? '½' : '☆')}
        </span>
      ))}
      <span className="ml-2 text-gray-600">({rating.toFixed(1)}/5)</span>
    </div>
  );
};

// Afficher la liste des avis

🧩 Assemblage de la page produit

La page ProductPage utilise les composants précédents et récupère les données via React Query.

// pages/ProductPage.tsx
import { useParams } from 'react-router-dom';
import { useQuery } from '@tanstack/react-query';
import { ProductGallery } from '@/components/ProductGallery';
import { VariationSelector } from '@/components/VariationSelector';
import { ReviewList } from '@/components/ReviewList';
import { useCartStore } from '@/stores/cartStore';

export const ProductPage = () => {
  const { id } = useParams();
  const { data: product, isLoading } = useQuery({
    queryKey: ['product', id],
    queryFn: () => fetchProduct(id)
  });
  const [selectedVariation, setSelectedVariation] = useState(null);
  const addToCart = useCartStore(s => s.addItem);

  if (isLoading) return <div>Chargement...</div>;

  return (
    <div className="grid md:grid-cols-2 gap-8">
      <ProductGallery images={product.images} />
      <div>
        <h1 className="text-3xl font-bold">{product.name}</h1>
        <RatingStars rating={product.avgRating} />
        <VariationSelector variations={product.variations} onChange={setSelectedVariation} />
        <button onClick={() => addToCart({ ...product, variation: selectedVariation })}
          className="mt-4 bg-indigo-600 text-white px-6 py-2 rounded-full w-full">
          Ajouter au panier
        </button>
      </div>
      <div className="md:col-span-2">
        <h2 className="text-xl font-bold">Avis clients</h2>
        <ReviewList reviews={product.reviews} />
      </div>
    </div>
  );
};

➕ Quantité et ajout au panier

On ajoute un sélecteur de quantité avec validation du stock.

// QuantitySelector
const QuantitySelector = ({ stock, quantity, setQuantity }) => {
  return (
    <div className="flex items-center gap-3 mt-3">
      <button onClick={() => quantity > 1 && setQuantity(q => q-1)} className="border px-3 py-1">-</button>
      <span>{quantity}</span>
      <button onClick={() => quantity < stock && setQuantity(q => q+1)} className="border px-3 py-1">+</button>
      <span className="text-sm text-gray-500">Stock: {stock}</span>
    </div>
  );
};

✨ Optimisations & astuces

🖼️ Lazy loading des images

Utilisez loading="lazy" sur les images hors écran et des placeholders.

🎨 Préchargement des variations

Changez l'image principale selon la couleur sélectionnée.

🧪 A11y

Ajoutez aria-label aux boutons de variation, texte alternatif pour images.

📱 Mobile first

Sur mobile, afficher le carrousel en swipe et les options en colonne.

🧪 À retenir : le composant "Page produit" est l’un des plus critiques pour le SEO et la conversion. Soignez l'expérience !

📌 Résumé – Ce qu'il faut retenir

  • ✅ Galerie d'images : image principale + miniatures cliquables (évolutif vers carrousel)
  • ✅ Variations produit : sélecteurs dynamiques (taille/couleur) avec mise à jour prix/stock
  • ✅ Avis clients : affichage note moyenne étoiles + liste commentaires
  • ✅ Intégration : tous ces composants s'assemblent dans une page produit unique, alimentée par React Query

🎯 Exercice pratique : Ajoutez un zoom sur l'image principale et la possibilité de filtrer les avis par nombre d'étoiles.
