📦 Installation de Recharts

npm install recharts

Exemple simple : un graphique en barres.

import { BarChart, Bar, XAxis, YAxis, Tooltip } from 'recharts';

const data = [{ name: 'Jan', uv: 400 }, { name: 'Fév', uv: 300 }];

function MyChart() {
  return (
    <BarChart width={500} height={300} data={data}>
      <XAxis dataKey="name" />
      <YAxis />
      <Tooltip />
      <Bar dataKey="uv" fill="#8884d8" />
    </BarChart>
  );
}

Recharts est basé sur des composants SVG, facilement personnalisable.

📈 Évolution des ventes (LineChart)

import { LineChart, Line, CartesianGrid, XAxis, YAxis, Tooltip } from 'recharts';

const evolution = [
  { jour: 'Lun', ventes: 12 }, { jour: 'Mar', ventes: 19 },
  { jour: 'Mer', ventes: 15 }, { jour: 'Jeu', ventes: 22 }
];

<LineChart width={500} height={250} data={evolution}>
  <Line type="monotone" dataKey="ventes" stroke="#4f46e5" />
  <CartesianGrid stroke="#ccc" />
  <XAxis dataKey="jour" /> <YAxis /> <Tooltip />
</LineChart>
📈 Courbe des ventes sur la semaine : tendance à la hausse.

🎚️ Filtrage temporel dynamique

Ajoutez un sélecteur pour changer la période (jour, mois, année).

const [period, setPeriod] = useState('month');

const filteredData = useMemo(() => {
  if (period === 'day') return dailyData;
  if (period === 'month') return monthlyData;
  return yearlyData;
}, [period]);

return (
  <div>
    <select onChange={(e) => setPeriod(e.target.value)}>
      <option value="day">Jour</option>
      <option value="month">Mois</option>
      <option value="year">Année</option>
    </select>
    <LineChart data={filteredData}>...</LineChart>
  </div>
);

Les données sont récupérées depuis l'API selon la période choisie.

🥧 Top produits (PieChart)

import { PieChart, Pie, Cell, Legend } from 'recharts';

const topProducts = [
  { name: 'Tee-shirt', value: 450 },
  { name: 'Casque', value: 320 },
  { name: 'Montre', value: 210 }
];
const COLORS = ['#4f46e5', '#818cf8', '#a5b4fc'];

<PieChart width={300} height={300}>
  <Pie data={topProducts} dataKey="value" label>
    {topProducts.map((entry, index) => <Cell fill={COLORS[index % COLORS.length]} />)}
  </Pie>
  <Legend />
</PieChart>
Visualisez rapidement les produits qui génèrent le plus de revenus.

✏️ Exercice pratique

🧪 À réaliser :

  • Installer Recharts dans un projet React (admin dashboard).
  • Créer un jeu de données factices pour le CA des 6 derniers mois.
  • Afficher un graphique en barres (BarChart) et un graphique en lignes (LineChart).
  • Ajouter une carte KPI pour le nombre total de commandes et le panier moyen.
  • Permettre le filtrage entre vue mensuelle et vue trimestrielle (select).
💡 Utilisez `useState` pour le filtre et `useMemo` pour recalculer les données affichées.
