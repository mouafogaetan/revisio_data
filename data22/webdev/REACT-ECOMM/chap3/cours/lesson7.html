🧭 Qu'est‑ce que React Router DOM ?

React Router DOM est la bibliothèque standard pour gérer la navigation dans une application React. Elle permet de changer l'URL sans recharger la page (SPA) et d’afficher le composant correspondant à la route.

npm install react-router-dom

🔗 Avantages :
  • Historique HTML5
  • Routes dynamiques
  • Redirections, liens actifs
  • Code splitting natif
📦 Inclus :
  • <BrowserRouter>
  • <Routes>, <Route>
  • useNavigate, useParams
  • Link, NavLink

⚙️ Installation & Configuration

Dans un projet Vite + React, exécutez :

npm install react-router-dom

Ensuite, enveloppez votre application avec <RouterProvider> ou <BrowserRouter>. Exemple avec createBrowserRouter :

import { createBrowserRouter, RouterProvider } from 'react-router-dom';

const router = createBrowserRouter([
  { path: "/", element: <Home /> },
  { path: "/products", element: <Products /> },
]);

function App() {
  return <RouterProvider router={router} />;
}
✅ Cette configuration est moderne, recommandée par l'équipe React Router.

🗺️ Routes principales pour l'e‑commerce

On structure généralement avec un layout commun (Header, Footer) et des routes imbriquées.

const router = createBrowserRouter([
  {
    path: '/',
    element: <MainLayout />,
    children: [
      { index: true, element: <HomePage /> },
      { path: 'catalogue', element: <Catalog /> },
      { path: 'product/:id', element: <ProductDetail /> },
      { path: 'cart', element: <Cart /> },
      { path: 'checkout', element: <Checkout /> }
    ]
  },
  { path: '/login', element: <Login /> },
  { path: '/register', element: <Register /> }
]);

Le composant MainLayout contient un <Outlet /> pour afficher les enfants.

🧱 Layout & Outlet

Un layout est un composant qui partage la même interface (barre de navigation, pied de page).

import { Outlet, Link } from 'react-router-dom';

function MainLayout() {
  return (
    <div>
      <header>
        <Link to="/">Accueil</Link>
        <Link to="/catalogue">Produits</Link>
      </header>
      <main>
        <Outlet />   {/* ici s'affiche la page active */}
      </main>
      <footer>...</footer>
    </div>
  );
}
🔁 Link remplace les balises <a> pour éviter le rechargement complet.

🔀 Liens, paramètres & useNavigate

Paramètres d'URL : /product/:id → useParams()

import { useParams } from 'react-router-dom';
function ProductDetail() {
  const { id } = useParams();
  return <div>Produit #{id}</div>
}

Navigation programmatique (après action) :

import { useNavigate } from 'react-router-dom';
const navigate = useNavigate();
// après ajout au panier
navigate('/cart');
