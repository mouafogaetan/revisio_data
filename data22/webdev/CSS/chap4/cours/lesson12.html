🎯 Objectifs de la leçon

  • ✅ Centrer un élément bloc margin: auto efficacement.
  • ✅ Comprendre le phénomène de fusion des marges verticales (margin collapsing) et savoir l'anticiper.
  • ✅ Utiliser les raccourcis CSS pour margin et padding (1 à 4 valeurs) et gagner en productivité.
  • ✅ Appliquer ces notions sur des cas concrets d'agencement web.

💡 Astuce prof : Le centrage d’un bloc est essentiel pour les mises en page responsives. La fusion des marges surprend souvent les débutants, mais vous allez tout comprendre !

🧭 Centrer un élément bloc margin: auto

Pour centrer horizontalement un élément de type block (comme <div>), on utilise :

.bloc-centre {
  width: 80%;
  margin-left: auto;
  margin-right: auto;
}
/* raccourci */
.bloc-centre {
  width: 300px;
  margin: 0 auto; /* top/bottom 0, left/right auto */
}

⚠️ L'élément doit avoir une largeur définie (width ou max-width).

🔍 Exemple visuel :

🟦 Je suis centré (margin: auto)

.bloc { width: 80%; margin: auto; }

📌 À retenir : margin: 0 auto; centre un bloc dans son conteneur si le bloc a une largeur inférieure à celle du parent.

📉 Fusion des marges (Margin Collapsing)

En CSS, les marges verticales (margin-top / margin-bottom) adjacentes de deux éléments peuvent se combiner en une seule marge, égale à la plus grande des deux.

Bloc A (margin-bottom: 30px)
Bloc B (margin-top: 20px)

➡️ L'espace entre les deux = 30px (pas 50px) car la marge se "plie".

.blocA { margin-bottom: 30px; }
.blocB { margin-top: 20px; }
/* distance finale = 30px */

🔍 Cas où la fusion ne se produit pas :

  • Éléments flottants (float) ou position absolute
  • Éléments avec overflow: hidden/auto/scroll (crée un contexte)
  • Présence de padding ou border entre les marges
  • Éléments flex / grid enfants

💡 Conseil : Pour éviter l’effondrement, utilisez un padding ou border sur le parent, ou encore un overflow: auto.

⌨️ Raccourcis CSS : margin / padding

Les propriétés raccourcis vous permettent de définir les 4 côtés en une ligne. Très pratique pour un code propre !

Nombre de valeursSignificationExemple
1 valeurles 4 côtés identiquesmargin: 16px; → top,right,bottom,left = 16px
2 valeurs[haut/bas] [gauche/droite]margin: 10px 20px; → top/bot 10px; left/right 20px
3 valeurshaut, droite/gauche, basmargin: 8px 12px 4px; → top=8px, left/right=12px, bottom=4px
4 valeurshaut, droite, bas, gauche (sens horaire)margin: 5px 10px 15px 20px;
padding: 20px;
interne uniforme
padding: 12px 24px;
haut/bas 12px, côtés 24px
margin: 0 auto;
centrage (0 top/bot, auto L/R)

🎯 Les raccourcis margin et padding fonctionnent également avec des pourcentages, em, rem, etc.

🛠️ Mise en pratique : centrage & marges intelligentes

Carte produit

Un exemple de bloc centré avec margin auto + gestion des marges.

Image illustrative
.carte {
  width: 90%;
  max-width: 280px;
  margin: 1.5rem auto;      /* centrage horizontal + marge verticale */
  padding: 1rem;            /* raccourci: tous les côtés */
  border-radius: 1rem;
}

/* Éviter la fusion des marges avec le parent: 
   on peut ajouter un padding sur le conteneur ou overflow auto */
✅ Bonne pratique : Toujours définir une largeur ou max-width pour qu’un bloc centré avec margin auto fonctionne.
⚠️ Piège : margin collapsing peut réduire l’espace vertical entre éléments successifs, testez vos layouts.

📝 Récapitulatif & défi express

✔ Centrer un bloc : width + margin: auto.

✔ Fusion des marges : Les marges verticales adjacentes fusionnent (max). Solution: padding/border/overflow.

✔ Raccourcis margin/padding : 1,2,3 ou 4 valeurs pour un code plus lisible.
