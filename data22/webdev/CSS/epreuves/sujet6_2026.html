📐 Partie 2 : Maîtrise de Flexbox (12 points)

On souhaite créer une barre de navigation responsive avec Flexbox. Voici le HTML de base :

<nav class="navbar">
  <div class="logo">MonSite</div>
  <ul class="nav-links">
    <li>Accueil</li>
    <li>Services</li>
    <li>Contact</li>
  </ul>
  <div class="user">👤 Profil</div>
</nav>
  1. Écrivez le CSS pour que la barre de navigation utilise Flexbox, avec le logo à gauche, les liens centrés et le profil à droite. (4 pts)
  2. Ajoutez une propriété pour que les éléments s'adaptent sur mobile (flex-wrap) et que les liens aient un espacement égal. (4 pts)
  3. Quelle est la différence entre justify-content: space-between et justify-content: space-around ? Donnez un schéma explicatif simple. (4 pts)

📱 Partie 5 : Projet responsive - Galerie de cartes (9 points)

En utilisant soit Flexbox, soit Grid, concevez une galerie de 3 cartes qui s'affiche en colonne sur mobile, en deux colonnes sur tablette et en trois colonnes sur desktop. Ajoutez un effet de transition au survol.

Structure HTML suggérée :

 <div class="gallery"> <div class="card">Carte 1</div> <div class="card">Carte 2</div> <div class="card">Carte 3</div> </div>

Critères d'évaluation :

  • Media queries correctes (mobile-first ou desktop-first) (3 pts)
  • Utilisation cohérente de gap, padding, couleurs (3 pts)
  • Animation au survol et comportement responsive (3 pts)
