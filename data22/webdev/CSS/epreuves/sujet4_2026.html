Pratique

Mise en situation : Styliser un menu responsive

On vous donne la structure HTML suivante. Écrivez les règles CSS pour :

  • Les éléments <li> doivent s'afficher en ligne (inline-block) avec un espacement de 1rem.
  • Au survol (:hover), le fond devient gris clair et une transition douce.
  • Le lien actif (.active) doit avoir une bordure inférieure bleue.
  • Utilisez un sélecteur descendant pour cibler les liens <a> à l'intérieur des <li>.
<nav class="menu">
  <ul>
    <li><a href="#" class="active">Accueil</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
