🧠 Grid vs Flexbox

📦 Flexbox (1D)

Agencement sur une seule direction (ligne ou colonne)

→ alignement sur un axe

🧩 CSS Grid (2D)

Lignes ET colonnes simultanément

Grille complète : lignes + colonnes

💡 À retenir : Flexbox pour des composants ou une ligne unique, Grid pour la structure globale de la page.

📐 Créer une grille simple

Pour activer Grid, il suffit d'utiliser display: grid; sur un conteneur. Ensuite on découpe en colonnes grâce à grid-template-columns.

.galerie {
  display: grid;
  grid-template-columns: 200px 200px 200px;   /* trois colonnes fixes */
  gap: 16px;
}

✨ L'unité fr et repeat()

fr = fraction de l'espace disponible. Très utile pour des layouts fluides !

/* 3 colonnes: 1fr, 2fr, 1fr → la colonne du milieu est deux fois plus large */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
}

🎯 repeat(3, 1fr) équivaut à 1fr 1fr 1fr → plus concis !

Exemple : grid-template-columns: repeat(4, minmax(100px, 1fr)); → chaque colonne fait au moins 100px puis s'agrandit.

📏 Lignes : grid-template-rows & minmax()

On contrôle aussi la hauteur des lignes. Associée à minmax(), on obtient des rangées adaptatives.

.grid {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-rows: auto minmax(120px, auto);
}

🎨 minmax(min, max) : la rangée fera au moins min et au plus max. Idéal pour du contenu dynamique.

Ligne 1 (auto)
Ligne 2 (minmax: 120px, auto) → au moins 120px

🚀 Exemple pratique : galerie responsive

Grâce à repeat(auto-fit, minmax(150px, 1fr)) la grille s'adapte automatiquement !

.galerie {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 1rem;
}

📌 Récapitulons

display: grid → active la grille
grid-template-columns → définit les colonnes
grid-template-rows → définit les lignes
fr, repeat(), minmax() → outils puissants

⭐ Bon à savoir : Grid permet d'imbriquer des grilles et de les combiner avec Flexbox pour des résultats professionnels.
