Partie 2

Exercices pratiques (12 points)

📌 Exercice 1 : Pagination (4 pts)

On considère un système avec des pages de 4 Ko (4096 octets). Une adresse logique est 8 500.

  1. Calculer le numéro de page et le déplacement correspondant.
  2. Sachant que la table des pages indique que la page n°2 est chargée dans le cadre physique n°7 (adresse de base 7×4096), quelle est l'adresse physique réelle ?
  3. Donner la formule générale de traduction d'adresse en pagination.
Astuce : Déplacement = Adresse logique % Taille page ; Numéro page = Adresse logique // Taille page

⚙️ Exercice 2 : Ordonnancement Round Robin (4 pts)

Trois processus arrivent à l'instant 0 avec les durées d'exécution suivantes : P1=8, P2=5, P3=4. Quantum = 3 ms.

  1. Construire le diagramme de Gantt (ordre d'exécution).
  2. Calculer le temps d'attente moyen.
  3. Justifier l'impact du quantum sur les changements de contexte.

🐧 Exercice 3 : Commandes et Processus (4 pts)

Un utilisateur lance la commande find / -name "*.log" > resultats.txt &.

  1. Que signifie le symbole & à la fin de la commande ?
  2. Comment afficher l'état de ce processus dans le terminal ? (Donner 2 commandes)
  3. Si le processus se bloque, quelle commande permet de l'arrêter définitivement ?
