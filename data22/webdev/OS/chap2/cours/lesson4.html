🧠 Processus : Programme en action

Définition : Un processus est un programme en cours d'exécution. Il ne s'agit pas seulement du code, mais aussi de son contexte : compteur programme, registres, mémoire, fichiers ouverts, etc.

📄 Programme (passif)

Fichier exécutable stocké sur le disque (ex: firefox.exe)

⚙️ Processus (actif)

Programme chargé en RAM avec ses ressources → entité vivante gérée par l'OS.

📌 Exemple : Vous double-cliquez sur "Navigateur Web" → le système crée un processus pour ce navigateur. Chaque onglet peut être un processus ou un thread.

📋 Process Control Block (PCB)

Structure de données centrale que l'OS maintient pour chaque processus. Stockée dans la mémoire noyau.

🆔 PID (identifiant unique)ex: 1337
📌 État du processusprêt / élu / bloqué ...
📝 Compteur ordinal (PC)prochaine instruction
🖥️ Registres CPUcontexte sauvegardé
🗂️ Gestion mémoiretable des pages, segments
📂 Fichiers ouverts & ressourcesdescripteurs

🔍 Lors d'un changement de processus (commutation), l'OS sauvegarde le PCB du processus sortant et restaure celui du nouveau.

Nouveau → Prêt : admission, l'OS accepte le processus.
Prêt → Élu : ordonnanceur (scheduler) choisit ce processus.
Élu → Prêt : fin du quantum ou interruption → préemption.
Élu → Bloqué : demande d'opération I/O ou attente d'événement.
Bloqué → Prêt : fin de l'I/O, l'événement attendu est arrivé.
Élu → Terminé : fin normale ou erreur fatale.

✨ Diagramme simplifié : Prêt ↔ Élu (alternance) + bloqué en attente

⚙️ Changement de contexte

📖 À retenir : L'état "élu" signifie que le processus possède le CPU ; l'état "prêt" signifie qu'il pourrait s'exécuter dès que le CPU est libre.

🍳 Analogie: Chef cuisinier & recettes

Processus = une préparation active (ex: cuire une omelette). Programme = la fiche recette écrite.

  • Nouveau : vous décidez de cuisiner l'omelette.
  • Prêt : vous avez tous les ingrédients sur le plan de travail, attendez que la plaque soit libre.
  • Élu / Running : vous utilisez la plaque de cuisson.
  • Bloqué : vous attendez que les œufs cuisent (I/O) ou que l'eau bout.
  • Terminé : plat fini, vous quittez la cuisine.

🎓 Chaque processus possède son PCB qui stocke "la recette avancée" (état, compteur, registres).
