🍃 Exercice 2 : Modèle Mongoose (3 pts)

Définissez un schéma Mongoose pour un modèle `Product` avec les champs suivants :

  • name : String, requis
  • price : Number, requis, minimum 0
  • category : String, enum: ['electronique', 'vetement', 'maison']
  • createdAt : Date, valeur par défaut : Date.now

🔐 Exercice 3 : Middleware JWT (3 pts)

Écrivez un middleware Express nommé `authenticateToken` qui :

  • Récupère le token JWT depuis l’en-tête `Authorization` (format : `Bearer <token>`)
  • Vérifie le token avec `jsonwebtoken.verify()` et la clé secrète `process.env.JWT_SECRET`
  • Si le token est valide, attache l’utilisateur décodé à `req.user` et appelle `next()`
  • Sinon, renvoie une réponse 401 avec un message d’erreur JSON

  3. Quel code HTTP renverriez-vous pour une création réussie d’article (POST) ?
  4. Citez deux en-têtes HTTP importants pour la sécurité d’une API.
