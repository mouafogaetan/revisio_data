1. Définition et rôle du HTML

HTML (HyperText Markup Language) est un langage de balisage utilisé pour structurer le contenu d'une page web. Ce n'est PAS un langage de programmation, mais un langage qui organise textes, images, liens, vidéos, etc. Le navigateur interprète le HTML pour afficher la page correctement.

<!DOCTYPE html>
<html>
<head><title>Ma page</title></head>
<body>
<h1>Bonjour le monde !</h1>
</body>
</html>

📌 Rôle : donner une structure logique et sémantique au document, les navigateurs savent alors afficher titres, paragraphes et autres éléments.

Schéma simplifié : Le HTML structure la page en blocs logiques.

🔄

HTML vs CSS vs JavaScript

Ces trois technologies travaillent ensemble pour créer des sites web modernes :

📄 HTMLStructure & contenu (squelette)
🎨 CSSHabillage, couleurs, position (style)
⚡ JavaScriptInteractions, dynamisme (comportement)

💡 Exemple concret :

HTML = "Un bouton" → CSS = "Bouton bleu avec ombre" → JS = "Quand on clique, afficher un message".

📜 Historique et versions du HTML

  • HTML 1.0 - 1993 : premières balises basiques.
  • HTML 4.01 (1999) : standardisation avancée, séparation style/css.
  • XHTML (2000) : version plus stricte (XML), syntaxe rigoureuse.
  • HTML5 (2014) : révolution avec balises sémantiques (article, section), support natif vidéo/audio, API modernes.

🏷️ Aujourd'hui, HTML5 est la norme utilisée, compatible mobile et riche en fonctionnalités.
