✨ C# : Un langage moderne et polyvalent

C# (prononcé "C sharp") est un langage de programmation orienté objet créé par Microsoft en 2000. Il fait partie de la famille des langages de la plateforme .NET.

🎯 Pourquoi "moderne" ?

C# évolue constamment avec des fonctionnalités comme le typage sécurisé, la programmation asynchrone (async/await), le pattern matching, et il est utilisé aussi bien par des débutants que par des experts en logiciels critiques.

Le langage est fortement typé, ce qui permet d'éviter beaucoup d'erreurs avant même l'exécution. Il combine la puissance du C++ avec la simplicité de Java, mais avec des innovations propres à Microsoft.

🏛️ Plateformes .NET

📦 .NET Framework, .NET Core, .NET 5+ : quelles différences ?

.NET Framework

Ancienne version (2002-2019) uniquement pour Windows. Intégrée à Windows, utilisée pour des applications Desktop (WinForms, WPF).

.NET Core

Version open-source et multiplateforme (Windows, Linux, macOS). Optimisée pour le cloud et les performances.

.NET 5, 6, 7, 8 (et plus)

L'unification ! À partir de .NET 5, Microsoft a fusionné .NET Framework et .NET Core pour créer une seule plateforme multiplateforme : .NET 5 → .NET 6 (LTS) → .NET 7 → .NET 8 (LTS actuel). Un seul SDK pour tout : web, mobile, desktop, cloud, jeux.

💡 Pour l'étudiant·e : Aujourd'hui, on utilise .NET 8 (ou 6 pour les projets LTS). C'est le plus récent, rapide et compatible partout.
