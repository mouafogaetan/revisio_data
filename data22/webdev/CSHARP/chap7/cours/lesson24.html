⚡ Méthodes d'extension

Étendre des types existants sans modifier leur code source — une approche élégante et puissante en C#.

Classe : webdev · Chapitre 7 : POO avancée

3

4

👉 Glissez ou utilisez les boutons pour naviguer

🔍 Définition

Une méthode d’extension permet d’ajouter de nouvelles méthodes à un type existant sans créer une nouvelle classe dérivée et sans modifier le code original. Elles se présentent comme des méthodes natives du type.

📌 Exemple typique :

LINQ (Where(), Select()) sont des méthodes d'extension sur IEnumerable<T>.

// Déclaration
public static class StringExtensions
{
    public static bool IsEmail(this string str)
    {
        return str.Contains("@") && str.Contains(".");
    }
}

// Utilisation
string email = "eleve@example.com";
bool estValide = email.IsEmail(); // Magique !
✨ On appelle la méthode comme si elle appartenait au type d'origine.

📖 Syntaxe essentielle

  • La méthode doit être statique dans une classe statique.
  • Le premier paramètre utilise le mot-clé this suivi du type à étendre.
// Structure type
public static class NomExtensions
{
    public static ReturnType NomMethode(this TypeCible instance, autresParams)
    {
        // logique ...
    }
}

💡 Règle :

Le paramètre this représente l’objet sur lequel la méthode est appelée. Il n’est jamais passé explicitement.

🧩 Exemples pratiques

❶ Extension sur int :

public static bool EstPair(this int nombre) => nombre % 2 == 0;
// utilisation
int age = 22;
Console.WriteLine(age.EstPair()); // True

❷ Extension sur string : inversion de casse

public static string InverserCasse(this string chaine)
{
    char[] chars = chaine.ToCharArray();
    for (int i = 0; i < chars.Length; i++)
        chars[i] = char.IsUpper(chars[i]) ? char.ToLower(chars[i]) : char.ToUpper(chars[i]);
    return new string(chars);
}

✅ Utile pour uniformiser des traitements fréquents.

📦 Étendre les collections

Ajoutons une méthode Second() sur List<T> pour récupérer le deuxième élément (ou null).

public static class ListExtensions
{
    public static T Second<T>(this List<T> list)
    {
        if (list == null || list.Count < 2)
            throw new ArgumentException("Liste trop petite");
        return list[1];
    }
}

// Usage
var fruits = new List<string> { "mangue", "orange", "banane" };
string deuxieme = fruits.Second(); // "orange"
⚠️ Attention : Les méthodes d’extension ne peuvent pas accéder aux membres privés du type.

⚠️ Limitations à connaître

✖

Pas de priorité : Une méthode d’extension ne remplace jamais une méthode d’instance existante (même nom & signature).

✖

Visibilité restreinte : Elles ne peuvent pas accéder aux champs privés ou protégés.

✖

Portée via using : Les extensions ne sont disponibles que si l’espace de noms contenant la classe statique est importé.

📌 Bonne pratique : Regroupez vos méthodes d’extension dans des namespaces cohérents (ex: MyProject.Extensions)

🔗 LINQ : le plus grand exemple

Les méthodes LINQ comme .Where(), .OrderBy(), .Select() sont des méthodes d'extension de System.Linq.Enumerable.

using System.Linq;
int[] notes = { 12, 15, 8, 19, 10 };
var bonnesNotes = notes.Where(n => n >= 12);
// Where est une méthode d'extension sur IEnumerable<int> !
Observation : Les méthodes d'extension enrichissent l'expérience sans héritage complexe.

📁 Organisation du code

  • Créer une classe statique publique (ex: StringExtensions, CollectionExtensions).
  • Mettre les méthodes dans un namespace significatif.
  • Nommer clairement la méthode : verbe + objet (ex: ToShortDateStringCustom).
  • Ne pas abuser : utilisez une méthode d'extension seulement si elle est fréquemment réutilisable.
namespace MonProjet.Extensions
{
    public static class DateTimeExtensions
    {
        public static string FormatterCourt(this DateTime dt) => dt.ToString("dd/MM/yyyy");
    }
}

✏️ Pensez à importer using MonProjet.Extensions; pour utiliser les extensions.

🧪 Testez vos connaissances

Voici une méthode d'extension PremiereLettreMajuscule() pour string. Complétez-la mentalement, puis cliquez pour voir la solution.

public static string PremiereLettreMajuscule(this string input)
{
    if (string.IsNullOrEmpty(input)) return input;
    return char.ToUpper(input[0]) + input.Substring(1).ToLower();
}
💡 "bonjour le monde".PremiereLettreMajuscule() → "Bonjour le monde"
🎓 Maîtrisez les extensions = code plus expressif !
