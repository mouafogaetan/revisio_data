📊 Groupement & Jointures

LINQ : GroupBy, Join, GroupJoin et Zip — manipulez des collections comme un expert.

🔍 Transformer des données en groupes 🔗 Fusionner des séquences
⬅️ ➡️ Utilisez les boutons de navigation

🎯 Objectifs de la leçon

  • ✓ Grouper des éléments avec GroupBy pour créer des collections clé/valeur.
  • ✓ Réaliser des jointures entre séquences avec Join (comme en SQL).
  • ✓ Maîtriser GroupJoin pour joindre et grouper en une seule opération.
  • ✓ Utiliser Zip pour fusionner deux séquences en paires.
💡 Saviez-vous ? LINQ (Language Integrated Query) transforme la manipulation des données en C# : élégant, lisible et puissant.

📌 1. Grouper avec GroupBy

Regrouper des éléments selon une clé, comme trier des fruits par couleur ou des étudiants par niveau.

// Exemple : Liste d'étudiants regroupés par classe
var etudiants = new[] {
  new { Nom = "Alice", Classe = "BTS1" },
  new { Nom = "Bob", Classe = "BTS1" },
  new { Nom = "Charlie", Classe = "BTS2" }
};
var groupes = etudiants.GroupBy(e => e.Classe);
foreach(var groupe in groupes) {
  Console.WriteLine($"Classe {groupe.Key} : {groupe.Count()} élèves");
}

⭐ Chaque groupe a une propriété Key et est une collection d'éléments.

🔍 Résultat : Classe BTS1 → 2 élèves ; Classe BTS2 → 1 élève.

🔗 2. Jointure interne : Join

Associer des éléments de deux collections selon une clé commune. (ex: commandes + clients)

var clients = new[] { new { Id=1, Nom="Paul"}, new { Id=2, Nom="Marie"} };
var commandes = new[] { new { ClientId=1, Produit="Laptop"}, new { ClientId=1, Produit="Souris"}, new { ClientId=2, Produit="Clavier"} };
var jointure = clients.Join(commandes,
  client => client.Id,
  commande => commande.ClientId,
  (client, commande) => new { client.Nom, commande.Produit });
// Résultat: (Paul, Laptop), (Paul, Souris), (Marie, Clavier)
🧠 Syntaxe de méthode : outer.Join(inner, outerKey, innerKey, resultSelector)

🗂️ 3. GroupJoin — join + group by

Associe chaque élément de la première séquence à tous ceux de la seconde correspondants, sous forme de groupe.

var groupJoin = clients.GroupJoin(commandes,
  client => client.Id,
  commande => commande.ClientId,
  (client, commandesClient) => new {
    client.Nom,
    Produits = commandesClient.Select(c => c.Produit)
  });
// Paul -> [Laptop, Souris] , Marie -> [Clavier]

✨ Utile pour générer des structures hiérarchiques (client avec liste de commandes).

🤝 4. Zip : fusionner deux séquences

Associer élément par élément deux collections (comme une fermeture éclair).

var noms = new[] { "Alice", "Bob", "Charlie" };
var ages = new[] { 22, 25, 30 };
var personnes = noms.Zip(ages, (nom, age) => $"{nom} a {age} ans");
// ["Alice a 22 ans", "Bob a 25 ans", "Charlie a 30 ans"]
// Si tailles différentes : s'arrête à la plus petite séquence
📦 .NET Core 3+ / .NET 5+ : Zip à 3 arguments aussi

📈 Exemple pratique : GroupBy & agrégation

var produits = new[] {
  new { Nom="PC", Categorie="Electronique", Prix=1200 },
  new { Nom="Smartphone", Categorie="Electronique", Prix=800 },
  new { Nom="Chaise", Categorie="Mobilier", Prix=150 },
  new { Nom="Table", Categorie="Mobilier", Prix=300 },
};

var stats = produits.GroupBy(p => p.Categorie)
  .Select(g => new {
    Categorie = g.Key,
    Total = g.Sum(p => p.Prix),
    Moyenne = g.Average(p => p.Prix)
  });
// Electronique -> Total 2000, Moyenne 1000 | Mobilier -> Total 450, Moyenne 225

🎯 Idéal pour dashboards et rapports.

🧩 Combiner GroupJoin + SelectMany

Parfois, on veut une jointure classique mais avec groupement temporaire. GroupJoin est parfait pour les "one-to-many".

// Écoles et élèves : chaque école a plusieurs élèves
var ecoles = new[] { new { IdEcole=1, Nom="Lycée A"}, new { IdEcole=2, Nom="Collège B"} };
var eleves = new[] { new { EcoleId=1, EleveNom="Jean"}, new { EcoleId=1, EleveNom="Emma"}, new { EcoleId=2, EleveNom="Lucas"} };
var result = ecoles.GroupJoin(eleves, e => e.IdEcole, el => el.EcoleId,
  (ecole, listeEleves) => new { ecole.Nom, Effectif = listeEleves.Count() });
// Lycée A → 2 élèves, Collège B → 1 élève.

⚡ Zip avec trois séquences (C# 8+)

var letters = new[] { "A", "B", "C" };
var numbers = new[] { 1, 2, 3 };
var symbols = new[] { "!", "@", "#" };
var combos = letters.Zip(numbers, symbols, (l, n, s) => $"{l}{n}{s}");
// ["A1!", "B2@", "C3#"]
// Cette surcharge est disponible dans .NET 6+
💡 Très utile pour créer des tuples ou combiner des données issues de sources parallèles.

📌 Récapitulatif & Bonnes pratiques

  • GroupBy → Regroupement simple, souvent avec agrégation (Sum, Count).
  • Join → Relation "clé étrangère" comme SQL (produit cartésien filtré).
  • GroupJoin → Structure un-à-plusieurs, génial pour les arbres d'objets.
  • Zip → fusionner séquences de même longueur (ou moindre).
  • ⚠️ LINQ est paresseux : les requêtes sont exécutées lors de l'itération (foreach, ToList).
