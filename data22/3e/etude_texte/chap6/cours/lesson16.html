📖 Maîtriser la formation des mots

Français · Classe de 3ème · Module 3 – Chapitre 6 : Le vocabulaire

📌 Éléments clés à retenir

🔤 1. La structure du mot : radical + affixes

Un mot peut être simple (constitué d’un seul élément) ou dérivé. La plupart des mots sont formés à partir d’un radical (ou racine) qui porte le sens principal, auquel on ajoute des préfixes (avant le radical) et/ou des suffixes (après le radical).

  • Radical : élément de base, commun à une famille de mots. Ex : terr- (terre) → terrain, terrestre, atterrir.
  • Préfixe : s’ajoute devant le radical pour modifier le sens (souvent nuance négative, répétition, lieu…). Ex : in- (contraire) → invisible ; re- → refaire.
  • Suffixe : s’ajoute après le radical pour changer la classe grammaticale (nom, adjectif, verbe) ou ajouter une nuance. Ex : -tion → construction (verbe → nom).
🧩 Schéma de dérivation :
[PRÉFIXE] + RADICAL + [SUFFIXE] = mot dérivé
Exemple : in- + just + -ice → injustice.

🏗️ 2. La famille de mots

Une famille de mots regroupe tous les mots dérivés d’un même radical. Ils partagent une idée commune mais peuvent appartenir à différentes classes grammaticales.

  • Exemple avec le radical chant- : chanter (verbe), chanteur (nom), chant (nom), chanson (nom avec suffixe -on), déchanter (préfixe dé-).
  • Savoir identifier la famille permet de comprendre le sens d’un mot inconnu par analogie.

📦 3. Composition et mots composés

Certains mots sont formés par l’union de deux mots (ou plus) : ce sont les mots composés. Ils peuvent être soudés (portemanteau), reliés par un trait d’union (arc-en-ciel) ou séparés (pomme de terre).

  • Composition savante : avec des éléments grecs ou latins (bio-, télé-, -scope, -logue). Ex : téléphone (télé = loin, phone = son).
  • Composition populaire : union de mots français : chou-fleur, garde-robe.

💡 Exemples de compréhension

🔍 Analyser un mot : « malheureusement »

Observons le mot malheureusement. Décomposons-le et identifions chaque élément.

Démarche :
➤ Radical : heur (chance, bonheur).
➤ Préfixe mal- (contraire, mauvais) → malheur (contraire de bonheur).
➤ Suffixe -eux → transforme en adjectif : malheureux.
➤ Suffixe -ment → transforme l’adjectif en adverbe : malheureusement.
✅ Conclusion : c’est un adverbe dérivé construit par ajouts successifs.

💬 Astuce : Pour analyser, cherche d’abord le radical. Demande-toi : quel est le mot le plus simple de la famille ?

✍️ Identifier la famille de mots

À partir du mot terre, trouve au moins trois mots de la même famille en précisant les affixes.

Réponse :
▪ Terrestre (radical terr- + suffixe -estre) → adjectif relatif à la terre.
▪ Atterrir (préfixe a- + radical terr + suffixe -ir) → verbe : venir sur terre.
▪ Terrain (radical terr- + suffixe -ain) → nom : étendue de terre.
▪ Souterrain (préfixe sous- + radical terr + suffixe -ain) → adjectif/nom : sous la terre.

💡 On peut aussi trouver terrier, terrasse, etc. L’important est de reconnaître le radical commun.

🧩 Mots composés : repérage

Relève deux mots composés dans la phrase suivante et explique leur formation :
« Le porte-avions est un navire de guerre qui transporte des aéronefs. »

Analyse :
• porte-avions : formé du verbe porter + nom avions (trait d’union). Désigne un bâtiment qui porte des avions.
• aéronefs : composition savante : aéro- (air) + -nef (navire, du latin navis). Machine qui navigue dans l’air.
→ Ce sont des mots composés, l’un populaire (verbe+nom) et l’autre savant (préfixe grec + radical latin).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre préfixe et suffixe. Rappelle-toi : préfixe = placé avant le radical ; suffixe = placé après.
  • Attention : Un même radical peut avoir plusieurs orthographes (ex : chant / chanson). Pense à la famille élargie.
  • Ne pas confondre : mot dérivé (radical + affixe) et mot composé (deux mots indépendants soudés). Ex : portefeuille (composé) ≠ portable (dérivé du verbe porter).
  • Piège : Certains préfixes changent de forme selon la première lettre du radical (ex : in- devient im- devant m/p : impossible, il- devant l : illisible).

✏️ Entraîne-toi ! (Auto-évaluation)

Défi : décompose et trouve la famille

1. Décompose le mot « incroyablement » : donne préfixe(s), radical, suffixe(s) et classe grammaticale.
2. Trouve trois mots de la famille du radical « lumi- » (lumière).
3. Relève le mot composé dans : « Le magnétoscope a été remplacé par le lecteur DVD. »

🔍 Voir les corrigés

1. incroyablement : préfixe in- (négation) + radical croy (croire) + suffixe -able (possibilité) + suffixe -ment (adverbe). Adverbe de manière.
2. lumineux (adjectif), illuminer (verbe), luminescence (nom), luminosité (nom).
3. magnétoscope (composé savant : magnéto- + scope) et lecteur DVD (composé avec apposition).

📚 Tableau mémo : préfixes & suffixes utiles

Préfixes courants

  • in-, im-, il-, ir- : négation (invisible, immoral, illégal, irréel).
  • re-, ré- : répétition, retour (refaire, réécrire).
  • pré- : avant (préhistoire, prévenir).
  • post- : après (postposer, postface).
  • anti- : contre (antibactérien).
  • co-, con- : avec (copilote, collaborer).

Suffixes courants

  • -tion, -ation, -ion → nom (construction, invention).
  • -able, -ible → adjectif (possible, mangeable).
  • -eur, -euse, -rice → nom de personne (chanteur, actrice).
  • -ment → adverbe (rapidement) ou nom (châtiment).
  • -iser, -ifier → verbe (moderniser, clarifier).
