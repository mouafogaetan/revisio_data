Leçon 23 : Réciproque du théorème de Thalès

Mathématiques · Troisième · Prouver que deux droites sont parallèles

📌 Éléments clés à retenir

🔁 Énoncé de la réciproque de Thalès

Soient deux droites $(d_1)$ et $(d_2)$ sécantes en un point $A$. Soient $B$ et $M$ deux points distincts de $A$ sur $(d_1)$, et $C$ et $N$ deux points distincts de $A$ sur $(d_2)$. Si les points $A, B, M$ sont alignés dans le même ordre que les points $A, C, N$ et si \[ \frac{AB}{AM} = \frac{AC}{AN} \] alors les droites $(BC)$ et $(MN)$ sont parallèles.

Figure type : A, B, M alignés ; A, C, N alignés ; si AB/AM = AC/AN alors (BC) // (MN).
  • Condition indispensable : les points doivent être alignés dans le même ordre (souvent A entre B et M ? Attention à bien vérifier la configuration).
  • Utilité : Démontrer que deux droites sont parallèles sans mesurer d'angle.
  • Formule clé : $\frac{AB}{AM} = \frac{AC}{AN}$ ou $\frac{AB}{BM} = \frac{AC}{CN}$ selon la configuration.

🧠 Distinction directe / réciproque

Propriété directe (Thalès) : Si $(BC) // (MN)$, alors $\frac{AB}{AM} = \frac{AC}{AN}$.
Réciproque : Si $\frac{AB}{AM} = \frac{AC}{AN}$ (avec le bon alignement), alors $(BC) // (MN)$.
La réciproque sert à prouver le parallélisme à partir de rapports de longueurs.

💡 Astuce : Pour t’en souvenir → “Si on a l’égalité des rapports, les droites sont copines (parallèles).”

💡 Exemples de compréhension

Exemple 1 : Prouver que (BC) // (MN)

Sur la figure, on sait que $AB = 3$ cm, $AM = 7,5$ cm, $AC = 2$ cm, $AN = 5$ cm. Les points A, B, M sont alignés dans cet ordre et A, C, N sont alignés dans le même ordre.

Résolution :
Calculons les rapports : $\frac{AB}{AM} = \frac{3}{7,5} = 0,4$ et $\frac{AC}{AN} = \frac{2}{5} = 0,4$.
On constate que $\frac{AB}{AM} = \frac{AC}{AN}$ et l’ordre des points est respecté. D’après la réciproque du théorème de Thalès, les droites $(BC)$ et $(MN)$ sont parallèles.

⚡ Vérifie toujours que les rapports sont bien dans le même ordre (parcours sur chaque droite depuis A).

Exemple 2 : Application avec des longueurs littérales

Soit un triangle $ABC$. $M \in [AB]$ tel que $AM = 4$ cm, $AB = 10$ cm. $N \in [AC]$ tel que $AN = 6$ cm et $AC = 15$ cm. Démontrer que $(MN) // (BC)$.

Calculs : $\frac{AM}{AB} = \frac{4}{10} = 0,4$ et $\frac{AN}{AC} = \frac{6}{15} = 0,4$.
L’égalité $\frac{AM}{AB} = \frac{AN}{AC}$ est vérifiée, et les points $A,M,B$ sont dans le même ordre que $A,N,C$. La réciproque de Thalès nous permet de conclure $(MN) // (BC)$.

📐 On peut utiliser indifféremment $\frac{AB}{AM} = \frac{AC}{AN}$ ou $\frac{AM}{AB} = \frac{AN}{AC}$ selon ce qu’on connaît.

Exemple 3 : Piège fréquent – ordre des points

Dans une configuration où $A=4$ cm, $B=7$ cm, $M=12$ cm sur $(d_1)$ (A, B, M alignés dans cet ordre) et $A=3$ cm, $C=5$ cm, $N=9$ cm sur $(d_2)$ (A, C, N alignés). Les rapports $\frac{AB}{AM} = \frac{4}{12} = \frac{1}{3}$ et $\frac{AC}{AN} = \frac{3}{9} = \frac{1}{3}$. L'égalité est vérifiée donc $(BC) // (MN)$.

Attention : Si les points n'étaient pas dans le même ordre (par exemple B entre A et M, mais N entre A et C), même avec l’égalité des rapports, la réciproque ne s’applique pas ! Toujours vérifier l’ordre.

✏️ Méthodologie : Comment utiliser la réciproque ?

Étapes clés pour une démonstration réussie

  • 1. Identifier les droites sécantes (en A) et les points sur chaque droite.
  • 2. Vérifier l’alignement et l’ordre : les points A, B, M sont alignés dans le même ordre que A, C, N.
  • 3. Calculer les deux rapports (ex: $\frac{AB}{AM}$ et $\frac{AC}{AN}$ ou autre combinaison).
  • 4. Constater l’égalité des rapports.
  • 5. Conclure : “D’après la réciproque du théorème de Thalès, les droites $(BC)$ et $(MN)$ sont parallèles.”
Méthode visuelle : repérer le "petit triangle" ABC et le "grand triangle" AMN, les rapports identiques → parallélisme.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente n°1 : Utiliser la réciproque sans vérifier l’ordre des points. Cela peut conduire à une conclusion fausse si la configuration est du type “papillon” ou points non correspondants.
  • Attention : Ne pas confondre la propriété directe (on sait que les droites sont parallèles pour trouver des longueurs) et la réciproque (on connaît des rapports pour prouver le parallélisme).
  • Piège : Appliquer la réciproque avec des longueurs qui ne correspondent pas aux segments homologues (ex: $\frac{AB}{BM}$ au lieu de $\frac{AB}{AM}$). Bien repérer les côtés correspondants.
  • À ne pas oublier : Les rapports doivent être pris sur les deux sécantes partant du point d’intersection A.

🏆 Exercice d'application guidé

Entraînement : parallélisme ou pas ?

Dans le triangle $ABC$, on place $M$ sur $[AB]$ tel que $AM = 2,5$ cm et $AB = 6$ cm. Sur $[AC]$, on place $N$ tel que $AN = 3,75$ cm et $AC = 9$ cm. Les droites $(MN)$ et $(BC)$ sont-elles parallèles ?

Correction détaillée :
• Points alignés : $A, M, B$ sont dans cet ordre (M entre A et B) ; $A, N, C$ dans cet ordre (N entre A et C).
• Calcul du rapport $\frac{AM}{AB} = \frac{2,5}{6} = \frac{5}{12}$ (environ 0,4167).
• Calcul du rapport $\frac{AN}{AC} = \frac{3,75}{9} = \frac{375}{900} = \frac{5}{12}$ (car 3,75/9 = 0,4167).
• On a bien $\frac{AM}{AB} = \frac{AN}{AC}$, donc d’après la réciproque du théorème de Thalès, $(MN) // (BC)$.

✨ Pense à simplifier les fractions pour comparer les rapports facilement.

📖 Récapitulatif – à retenir absolument

Réciproque de Thalès : Si dans une configuration de deux droites sécantes, les rapports de longueurs sur chaque droite sont égaux (dans le même ordre), alors les deux autres droites sont parallèles.

$$\text{Si } \frac{AB}{AM} = \frac{AC}{AN} \text{ et points alignés dans le même ordre, alors } (BC) \parallel (MN).$$

📌 Conditions sine qua non : mêmes sécantes, même ordre des points, égalité des rapports.
