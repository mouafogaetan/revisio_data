Leçon 10 : Opérations sur les expressions littérales

Identités remarquables · Développement · Réduction | Troisième

📌 Éléments clés à retenir

🧠 Les trois identités remarquables

Ces trois formules permettent de développer ou factoriser rapidement des expressions. Elles sont à connaître parfaitement.

  • Carré d'une somme : $(a + b)^2 = a^2 + 2ab + b^2$
  • Carré d'une différence : $(a - b)^2 = a^2 - 2ab + b^2$
  • Produit de la somme par la différence : $(a + b)(a - b) = a^2 - b^2$
💡 Astuce mnémotechnique : Pour $(a+b)^2$ on dit « carré du premier, plus le double produit, plus le carré du second ».

📐 Développer une expression

Développer, c’est transformer un produit en une somme (ou différence). On utilise la distributivité simple ou les identités remarquables.

  • $k(a + b) = ka + kb$ (Distributivité simple)
  • $(a+b)(c+d) = ac + ad + bc + bd$ (Double distributivité)
  • Appliquer les identités remarquables pour gagner du temps et éviter les erreurs.

✂️ Réduire une expression

Réduire, c’est regrouper les termes de même nature (les constantes entre elles, les termes en $x$, $x^2$, etc.) pour obtenir une écriture plus simple.

  • $3x^2 + 5x - 2x^2 + 4x = (3x^2-2x^2) + (5x+4x) = 1x^2 + 9x$
  • On n’oublie pas les signes qui précèdent chaque terme !

💡 Exemples de compréhension

Exemple 1 : Développer avec $(a+b)^2$

Développer et réduire $A = (3x + 5)^2$.

Méthode : On reconnaît $(a+b)^2$ avec $a = 3x$ et $b = 5$.
$(3x)^2 = 9x^2$, $2ab = 2 \times 3x \times 5 = 30x$, $b^2 = 5^2 = 25$.
Donc $A = 9x^2 + 30x + 25$.

💬 Pensez à bien mettre les parenthèses pour $a^2$ : $(3x)^2 = 9x^2$ (et non $3x^2$).

Exemple 2 : Développer avec $(a-b)^2$

Développer et réduire $B = (4 - 2x)^2$.

Méthode : On utilise $(a - b)^2 = a^2 - 2ab + b^2$ avec $a = 4$ et $b = 2x$.
$a^2 = 16$, $2ab = 2 \times 4 \times (2x) = 16x$, $b^2 = (2x)^2 = 4x^2$.
Donc $B = 16 - 16x + 4x^2$. On ordonne : $B = 4x^2 - 16x + 16$.

💡 Attention au signe : $ -2ab = - 2 \times 4 \times 2x = -16x$.

Exemple 3 : Produit (somme × différence)

Développer $C = (2x + 3)(2x - 3)$.

Méthode : On reconnaît $(a+b)(a-b)=a^2-b^2$ avec $a = 2x$ et $b = 3$.
$a^2 = (2x)^2 = 4x^2$, $b^2 = 3^2 = 9$. Donc $C = 4x^2 - 9$.

⚡ Résultat direct : c'est la différence des deux carrés. Pas de terme en $x$.

Exemple 4 : Développement complexe (double distributivité)

Développer et réduire $D = (x + 2)(3x - 5)$.

Méthode : On applique la double distributivité :
$x \times 3x = 3x^2$, $x \times (-5) = -5x$, $2 \times 3x = 6x$, $2 \times (-5) = -10$.
On additionne : $3x^2 + (-5x + 6x) - 10 = 3x^2 + x - 10$.

📌 Pour réduire, on regroupe les $x$ ensemble : $-5x + 6x = +x$.

Exemple 5 : Réduction après développement

Développer puis réduire $E = (2x - 1)^2 + (x + 3)(x - 3)$.

Étape 1 : $(2x - 1)^2 = 4x^2 - 4x + 1$ (identité remarquable).
Étape 2 : $(x+3)(x-3) = x^2 - 9$ (différence de carrés).
Étape 3 : On additionne : $ (4x^2 - 4x + 1) + (x^2 - 9) = 5x^2 - 4x - 8$.

🧩 Toujours réduire l’expression finale en regroupant les $x^2$, les $x$ et les constantes.

⚠️ Pièges courants & erreurs à éviter

  • Erreur classique : Écrire $(a+b)^2 = a^2 + b^2$ (oubli du double produit $2ab$). Toujours vérifier avec un exemple : $(3+2)^2 = 25$, alors que $3^2+2^2=13$ → faux.
  • Attention aux signes : Dans $(a-b)^2$, le double produit est négatif : $(5-3)^2 = 4$, alors que $5^2 - 2*5*3 + 3^2 = 25 -30 +9 =4$. Ne pas écrire $+2ab$.
  • Confusion produit/somme : $(a+b)(a-b) = a^2 - b^2$ (il n’y a pas de $2ab$).
  • Oubli des parenthèses : Dans $(3x)^2$, on doit élever $3$ et $x$ au carré : $9x^2$ (et non $3x^2$).
  • Réduction incomplète : Toujours regrouper les termes semblables après développement.

📖 Aide-mémoire (à découper mentalement)

🌟 $(a+b)^2 = a^2 + 2ab + b^2$
🌟 $(a-b)^2 = a^2 - 2ab + b^2$
🌟 $(a+b)(a-b) = a^2 - b^2$

💪 Entraînez-vous à les reconnaître dans les deux sens : développement et factorisation.
