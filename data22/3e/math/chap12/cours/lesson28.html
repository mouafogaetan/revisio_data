Leçon 28 : Calcul des coordonnées d’un point

Mathématiques · Troisième · Vecteurs et repérage dans le plan

📌 Éléments clés à retenir

Coordonnées d’un vecteur défini par deux points

Dans un repère orthonormé ou quelconque, si on connaît les coordonnées de deux points $A(x_A; y_A)$ et $B(x_B; y_B)$, alors le vecteur $\overrightarrow{AB}$ a pour coordonnées :

$\boxed{\overrightarrow{AB} \begin{pmatrix} x_B - x_A \\ y_B - y_A \end{pmatrix}}$
  • On soustrait les coordonnées de l'extrémité B par celles de l'origine A.
  • L'ordre des points est essentiel : $\overrightarrow{AB} = -\overrightarrow{BA}$.
  • Exemple : $A(2;3)$ et $B(5;7)$ donne $\overrightarrow{AB}(3;4)$.

Figure 1 : Calcul de $\overrightarrow{AB} (x_B-x_A; y_B-y_A)$.

Coordonnées du milieu d’un segment

Si $A(x_A; y_A)$ et $B(x_B; y_B)$ sont deux points du plan, alors le milieu $M$ du segment $[AB]$ a pour coordonnées :

$\boxed{M\left( \frac{x_A + x_B}{2} \; ; \; \frac{y_A + y_B}{2} \right)}$
  • La formule est une moyenne des abscisses et des ordonnées.
  • Cette propriété sert à vérifier qu’un point est le milieu d’un segment ou à trouver l’extrémité manquante.
  • Cas particulier : Si $M$ est le milieu de $[AB]$, alors $\overrightarrow{AM} = \overrightarrow{MB}$.

Figure 2 : Le milieu M est le point d’équidistance des extrémités.

Lien avec les vecteurs égaux et parallélisme

Savoir calculer les coordonnées d’un vecteur permet de :

  • Montrer que deux vecteurs sont égaux (mêmes coordonnées).
  • Démontrer qu’un quadrilatère est un parallélogramme : $\overrightarrow{AB} = \overrightarrow{DC}$.
  • Déterminer les coordonnées d’un point pour qu’une condition vectorielle soit vérifiée.

💡 Exemples de compréhension

Exemple 1 : Calcul des coordonnées d’un vecteur

On donne $A(-2 ; 5)$ et $B(4 ; -1)$ dans un repère orthonormé. Calculer les coordonnées de $\overrightarrow{AB}$ et $\overrightarrow{BA}$.

Résolution :
Pour $\overrightarrow{AB}$ : $x_B - x_A = 4 - (-2) = 6$ et $y_B - y_A = -1 - 5 = -6$.
Donc $\boxed{\overrightarrow{AB}(6 ; -6)}$.

Pour $\overrightarrow{BA}$ : $x_A - x_B = -2 - 4 = -6$ et $y_A - y_B = 5 - (-1) = 6$.
Donc $\boxed{\overrightarrow{BA}(-6 ; 6)}$.
On constate bien que $\overrightarrow{BA} = -\overrightarrow{AB}$.
💡 Astuce : Vérifiez toujours le sens : $\overrightarrow{AB}$ part de A vers B, donc on soustrait les coordonnées de A à celles de B.

Exemple 2 : Calcul du milieu d’un segment

Soit $C(3 ; 8)$ et $D(-5 ; 2)$. Déterminer les coordonnées du milieu $M$ du segment $[CD]$.

Calcul :
$x_M = \dfrac{x_C + x_D}{2} = \dfrac{3 + (-5)}{2} = \dfrac{-2}{2} = -1$.
$y_M = \dfrac{y_C + y_D}{2} = \dfrac{8 + 2}{2} = \dfrac{10}{2} = 5$.
Ainsi, $\boxed{M(-1 ; 5)}$.
📌 À retenir : Le milieu est le « point d’équilibre » ; ses coordonnées sont les moyennes arithmétiques.

Exemple 3 : Application - Trouver un point manquant

Dans un parallélogramme $EFGH$, on sait que $E(1;2)$, $F(4;3)$, $G(6;7)$. Déterminer les coordonnées de $H$ sachant que $\overrightarrow{EF} = \overrightarrow{HG}$.

Méthode vectorielle :
Calculons $\overrightarrow{EF} (4-1 ; 3-2) = (3 ; 1)$.
On veut $\overrightarrow{HG} = (x_G - x_H ; y_G - y_H) = (3 ; 1)$.
Soit $H(x_H;y_H)$ : $6 - x_H = 3 \Rightarrow x_H = 3$ ; $7 - y_H = 1 \Rightarrow y_H = 6$.
Donc $\boxed{H(3;6)}$.
Autre méthode (milieu) : Les diagonales d’un parallélogramme se coupent en leur milieu. Le milieu de $[EG]$ est $\left( \frac{1+6}{2};\frac{2+7}{2} \right) = (3.5;4.5)$ qui est aussi milieu de $[FH]$. On retrouve $H$ par $x_H = 2×3.5 - 4 = 3$, etc.
✨ Le lien vecteurs ↔ milieu est très puissant pour résoudre des exercices de géométrie.

⚠️ Pièges courants & erreurs fréquentes

  • Inverser l'ordre : Pour $\overrightarrow{AB}$, ne pas calculer $A - B$ ! La formule est extrémité moins origine.
  • Oublier la moyenne : Pour le milieu, on divise bien par 2 la somme des abscisses ET des ordonnées.
  • Confondre coordonnées d’un point et coordonnées d’un vecteur : Un point a une position fixe, un vecteur représente un déplacement.
  • Ne pas simplifier : Toujours réduire les fractions obtenues pour les coordonnées du milieu.

✏️ Petit entraînement express

Testez-vous !

  • Soit $A(0;4)$ et $B(5;0)$. Calculez $\overrightarrow{AB}$ et les coordonnées du milieu de $[AB]$.
  • Si $\overrightarrow{CD} ( -2 ; 3 )$ et $C(1; -1)$, quelles sont les coordonnées de $D$ ?
  • Le point $I(2; -3)$ est le milieu de $[JK]$ avec $J(5; -7)$. Trouvez $K$.
🔎 Réponses : $\overrightarrow{AB}(5;-4)$, milieu $(2,5;2)$ ; $D(-1;2)$ ; $K(-1;1)$. Revoir les formules si nécessaire.
