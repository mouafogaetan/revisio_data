Inéquations & systèmes du 1er degré dans IR²

Mathématiques · Troisième · Résolution graphique

📌 Éléments clés à retenir

Inéquation linéaire à deux inconnues

Une inéquation du premier degré à deux inconnues \(x\) et \(y\) peut se mettre sous la forme : \[ ax + by + c \le 0 \quad \text{ou} \quad ax + by + c \ge 0 \quad (\text{avec } (a,b) \neq (0,0)). \] L'ensemble des solutions est un demi-plan (frontière incluse si l'inégalité est large, exclue si stricte).

  • La droite d'équation \(ax+by+c = 0\) sépare le plan en deux demi-plans.
  • Pour déterminer le demi-plan solution, on teste un point témoin (souvent l'origine \((0;0)\) si elle n'appartient pas à la droite).
  • Graphiquement, on hachure le demi-plan non solution pour faire apparaître la zone solution.
  • Si l'inégalité est stricte (\(<\) ou \(>\)), la droite frontière est tracée en pointillés.

Système de deux inéquations linéaires

Un système du type \[ \begin{cases} a_1x + b_1y + c_1 \le 0 \\ a_2x + b_2y + c_2 \le 0 \end{cases} \] admet pour ensemble solution l'intersection des demi-plans solutions de chaque inéquation.

  • On trace les deux droites frontières.
  • On détermine le demi-plan solution pour chaque inéquation (en hachurant le côté non valide).
  • La région solution est la partie du plan qui n'est jamais hachurée (zone commune non hachurée).
  • Cette zone peut être polygonale, non bornée ou vide (si les contraintes sont incompatibles).

Méthode pas à pas (graphique)

1️⃣ Transformer chaque inéquation en équation de droite.
2️⃣ Tracer les droites dans un repère orthonormé (graduation régulière).
3️⃣ Choisir un point test (ex: (0;0)) pour chaque inéquation.
4️⃣ Hachurer la région non vérifiant l'inéquation.
5️⃣ L'intersection des demi-plans non hachurés est l'ensemble des solutions.

💡 Astuce : Si le point test vérifie l'inégalité, le demi-plan solution est celui qui contient ce point. Sinon, c'est l'autre demi-plan.

💡 Exemples de compréhension

Exemple 1 : Inéquation simple \(y \ge -2x + 3\)

Représenter graphiquement l'ensemble des points \((x ; y)\) tels que \(y + 2x - 3 \ge 0\).

Résolution : On trace la droite \(d : y = -2x + 3\) (frontière).
Point test : \((0;0)\) donne \(0 + 0 - 3 = -3 \ge 0\) ? Faux. Donc le demi-plan contenant (0;0) n'est pas solution. On hachure ce côté. La zone solution est le demi-plan au-dessus (incluant la droite car \(\ge\)).

💬 Vérification : Prendre un point de la zone, ex: \((1;2)\). \(2 \ge -2(1)+3 \Rightarrow 2 \ge 1\) : vrai.

Exemple 2 : Système de deux inéquations

Résoudre graphiquement : \[ \begin{cases} x + y \le 4 \\ x \ge 0 \\ y \ge 0 \end{cases} \]

Démarche : On trace \(d_1 : x+y=4\) (frontière oblique), \(d_2 : x=0\) (axe des ordonnées), \(d_3 : y=0\) (axe des abscisses).
- Pour \(x+y \le 4\) : point test (0;0) → \(0 \le 4\) vrai → on hachure le côté opposé (là où se trouve (2;3) par exemple).
- Pour \(x \ge 0\) : on hachure la partie gauche de l'axe vertical.
- Pour \(y \ge 0\) : on hachure la partie sous l'axe horizontal.
La région solution est le triangle rectangle (incluant les bords) dans le premier quadrant sous la droite \(x+y=4\).

📐 Les contraintes \(x \ge 0\) et \(y \ge 0\) correspondent au premier quadrant (y compris axes).

Exemple 3 : Système avec inéquations strictes

\[ \begin{cases} 2x - y > 1 \\ x + y < 5 \end{cases} \]

Étapes : Tracer \(d_1 : y = 2x - 1\) (pointillés car >) et \(d_2 : y = -x + 5\) (pointillés car <).
Test (0;0) pour \(2x - y > 1\) : \(0 > 1\) ? faux → hachurer le demi-plan contenant (0;0).
Test (0;0) pour \(x + y < 5\) : \(0 < 5\) vrai → hachurer le demi-plan qui ne contient pas (0;0).
La solution est l'intersection des deux zones non hachurées (bande ouverte entre les deux droites sans les bords).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de tester un point lorsque la droite passe par l'origine. Dans ce cas, utiliser (1;0) ou (0;1) comme point témoin.
  • Confusion : Hachurer la zone solution au lieu de la zone non solution. Rappel : on hachure ce qui ne convient pas pour que la zone claire soit l'ensemble des solutions.
  • Droite mal tracée : Ne pas respecter l'échelle du repère ou confondre pente et ordonnée à l'origine.
  • Inégalité stricte : Oublier les pointillés sur la frontière (solution = droite exclue).
  • Système incompatible : Si les deux demi-plans ne se rencontrent pas (ex: \(y \ge 2x+3\) et \(y \le 2x-1\)), la solution est l'ensemble vide.

📐 Marche à suivre – fiche réflexe

Synthèse graphique pour un système

  • Étape 1 : Écrire chaque inéquation sous forme \(ax+by+c \ge 0\) ou \(\le 0\).
  • Étape 2 : Tracer la droite frontière (\(=0\)).
  • Étape 3 : Point test → déterminer quel demi-plan satisfait l'inéquation.
  • Étape 4 : Hachurer le demi-plan rejeté.
  • Étape 5 : Répéter pour chaque inéquation. La région solution est la partie jamais hachurée.
  • Étape 6 : Exprimer la solution : description géométrique (polygone, demi-droite) ou par un système équivalent.
🧠 Pour contrôler : Choisir un point dans la zone supposée solution et vérifier qu'il satisfait toutes les inégalités.

✏️ Petit exercice d'entraînement

Déterminer le système correspondant à la zone

Une région solution est délimitée par les droites \(y = x+1\) (trait plein), \(y = -x+4\) (trait plein) et l'axe des abscisses \(y=0\) (trait plein). La zone est un triangle situé au-dessus de l'axe des x, en dessous de la droite \(y = x+1\) et en dessous de \(y = -x+4\). Écrire le système d'inéquations correspondant.

Correction : On traduit chaque condition :
- Au-dessus de l'axe des abscisses → \(y \ge 0\).
- En dessous de \(y = x+1\) → \(y \le x+1\) soit \(-x + y -1 \le 0\).
- En dessous de \(y = -x+4\) → \(y \le -x+4\) soit \(x + y -4 \le 0\).
Le système est : \[ \begin{cases} y \ge 0 \\ y \le x+1 \\ y \le -x+4 \end{cases} \]
