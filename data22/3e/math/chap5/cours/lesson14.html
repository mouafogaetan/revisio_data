Leçon 14 : Inéquations du premier degré dans IR

Mathématiques · Troisième · Organisation et gestion des données

📌 Éléments clés à retenir

1. Qu'est-ce qu'une inéquation du premier degré ?

Une inéquation du premier degré à une inconnue \(x\) est une inégalité qui peut se mettre sous l'une des formes suivantes : \(ax + b > 0\), \(ax + b \ge 0\), \(ax + b < 0\) ou \(ax + b \le 0\), où \(a\) et \(b\) sont des nombres réels avec \(a \neq 0\). Résoudre une inéquation, c'est trouver toutes les valeurs de \(x\) (dans \(\mathbb{R}\)) qui vérifient l'inégalité. L'ensemble des solutions est généralement un intervalle ou une réunion d'intervalles.

  • Exemple : \(2x - 5 \le 3\) est une inéquation du premier degré.
  • Lorsqu'on multiplie ou divise les deux membres par un nombre négatif, le sens de l'inégalité s'inverse.
  • Les solutions se représentent sur une droite graduée et s'écrivent sous forme d'intervalle.

2. Règles de transformation (propriétés)

Pour résoudre une inéquation, on applique des transformations qui conservent (ou inversent) le sens de l'inégalité.

  • On peut ajouter ou soustraire un même nombre aux deux membres, sans changer le sens.
  • On peut multiplier ou diviser par un même nombre positif, sans changer le sens.
  • ⚠️ Règle cruciale : Si on multiplie ou divise par un nombre négatif, on doit inverser le sens de l'inégalité.
  • Exemple : \(-3x < 6\) équivaut à \(x > -2\) (division par -3, le < devient >).

3. Représentation des solutions sur une droite graduée

L'ensemble des solutions d'une inéquation se représente graphiquement sur une droite réelle. On utilise :

  • Crochet [ ou ] : lorsque la borne est incluse (\(\ge\) ou \(\le\)).
  • Parenthèse ( ou ) : lorsque la borne est exclue (\(>\) ou \(<\)).
  • Les flèches indiquent que l'intervalle se prolonge indéfiniment.
⚫ pour inclusion ● , ○ pour exclusion
Exemple : \(x \ge 2\) → [2 ; +∞[ (cercle fermé sur 2, flèche vers la droite)

4. Écriture sous forme d'intervalle

Une inéquation du premier degré a généralement une solution qui s'écrit sous forme d'intervalle. Les notations à maîtriser :

  • \(x > a\) → \(]a ; +\infty[\)
  • \(x \ge a\) → \([a ; +\infty[\)
  • \(x < a\) → \(]-\infty ; a[\)
  • \(x \le a\) → \(]-\infty ; a]\)
  • Parfois, on obtient des intervalles bornés : \(2 \le x < 5\) → \([2 ; 5[\).

💡 Exemples de compréhension

Exemple n°1 : Résolution simple (multiplication par un négatif)

Résoudre l'inéquation : \(-4x + 7 \le 15\) dans \(\mathbb{R}\).

Résolution pas à pas :
1. On isole les termes en \(x\) : \(-4x + 7 \le 15\) → \(-4x \le 15 - 7\) → \(-4x \le 8\).
2. On divise par \(-4\) (nombre négatif) → On inverse le sens : \(x \ge \dfrac{8}{-4}\) → \(x \ge -2\).
3. L'ensemble des solutions est l'intervalle \([-2 ; +\infty[\).
Représentation graphique :

←---|---|---|---|---|---|---|---|---→
  -3    -2    -1    0    1    2
[●===================→

💬 Astuce : Chaque fois que vous divisez par un nombre négatif, pensez à “changer le signe de l’inégalité”. Vérifiez avec une valeur test (exemple : x = 0 donne -4(0)+7=7 ≤ 15 ✅).

Exemple n°2 : Inéquation avec parenthèses et fractions

Résoudre \(3(2x - 4) > 5x - 7\).

Étapes de calcul :
1. Développer : \(6x - 12 > 5x - 7\).
2. Regrouper les \(x\) : \(6x - 5x > -7 + 12\) → \(x > 5\).
3. Solution : \(]5 ; +\infty[\).
Vérification : Pour \(x = 6\), on a \(3(12-4)=24 > 30-7=23\) → vrai.

💡 Il est souvent utile de faire un test avec une valeur simple pour valider l'intervalle trouvé.

Exemple n°3 : Représentation graphique et intervalle

Résoudre l'inéquation \(-2x + 1 \ge 3\) et donner la représentation graphique.

Calcul : \(-2x + 1 \ge 3\) → \(-2x \ge 2\) → on divise par \(-2\) (négatif) → \(x \le -1\).
Intervalle : \(]-\infty ; -1]\).
Représentation sur la droite :

←====●]---|---|---|---|---→
    -3   -2   -1   0   1
Le crochet fermé en -1 indique que -1 est inclus.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente n°1 : Oublier d'inverser le sens de l'inégalité en multipliant ou divisant par un nombre négatif.
    ➜ Toujours vérifier le signe du coefficient avant de diviser.
  • Erreur fréquente n°2 : Confondre les crochets ouverts et fermés. Rappel : ≤ ou ≥ → crochet fermé [ ou ] ; < ou > → parenthèse ( ou ).
  • Attention particulière : Dans la représentation graphique, ne pas oublier que \(+\infty\) et \(-\infty\) sont toujours exclus (parenthèse ouverte).
  • Confusion : Ne pas assimiler une inéquation à une équation : une inéquation donne souvent un ensemble infini de solutions (intervalle) alors qu'une équation donne un nombre fini de solutions.

📐 Exercices d’entraînement rapide

À résoudre seul(e) (corrigés ci-dessous)

  • 1) \(5x - 3 \ge 2x + 6\)    →   Intervalle : [3 ; +∞[
  • 2) \(-2(x + 4) < 3x - 2\)    →   Intervalle : ]-6/5 ; +∞[ soit \(]-1,2 ; +\infty[\)
  • 3) Représenter sur une droite : \(x \le -2\) et \(x > 1\) (attention, c'est une réunion)
Voir les corrigés détaillés

1) \(5x - 3 \ge 2x + 6\) → \(5x-2x \ge 6+3\) → \(3x \ge 9\) → \(x \ge 3\) → Intervalle \([3;+\infty[\).

2) \(-2x -8 < 3x -2\) → \(-2x -3x < -2 + 8\) → \(-5x < 6\) → division par -5 (négatif) → \(x > -\frac{6}{5}\) → \(]-\frac{6}{5}; +\infty[\).

3) Réunion : \(x \le -2\) donne \(]-\infty ; -2]\) et \(x>1\) donne \(]1; +\infty[\) → la représentation comporte deux parties disjointes.

📊 Tableau récapitulatif : inégalité ↔ intervalle ↔ droite

Un petit mémo visuel pour ne plus se tromper :

Astuce mnémotechnique : "Crochet vers le nombre quand on l’inclut, parenthèse quand on l’exclut". Le sens de l’intervalle se lit toujours de la plus petite valeur à la plus grande.

🎯 Méthodologie pas à pas (pour réussir en devoir)

Étapes pour résoudre une inéquation du 1er degré

  • 1. Simplifier chaque membre (développer, réduire).
  • 2. Regrouper les termes en \(x\) dans un membre et les constantes dans l’autre (en ajoutant ou soustrayant).
  • 3. Isoler \(x\) en divisant par le coefficient. Attention au signe de ce coefficient : s'il est négatif, inverser le sens de l’inégalité.
  • 4. Écrire l’ensemble solution sous forme d’intervalle.
  • 5. Tracer la droite graduée pour vérifier (cercle ouvert/fermé).
✅ Petit contrôle : prendre une valeur dans l’intervalle et une valeur hors intervalle pour valider la solution.

🧠 Pour aller plus loin (sans panique)

  • Cas particulier : Une inéquation comme \(0x \le 5\) donne \(\mathbb{R}\) car toujours vraie ; \(0x \ge 7\) donne \(\varnothing\) (pas de solution).
  • Lien avec les fonctions : Résoudre \(ax+b > 0\) revient à chercher les abscisses où la droite \(y=ax+b\) est au-dessus de l’axe des abscisses.
  • Maîtriser les inéquations est essentiel pour les systèmes d’inéquations (Leçon 16) et les problèmes de seuil (optimisation).
