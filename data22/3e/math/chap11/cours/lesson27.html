Leçon 27 : Utilisation des formules trigonométriques

Mathématiques · Troisième · Module 3 – Trigonométrie dans le triangle rectangle

📌 Éléments clés à retenir

Relation fondamentale : $ \cos^2 x + \sin^2 x = 1$

Pour tout angle aigu $x$ (mesuré en degrés ou radians) dans un triangle rectangle, le carré du cosinus additionné au carré du sinus est toujours égal à 1. Cette formule découle du théorème de Pythagore appliqué au cercle trigonométrique ou au triangle rectangle.

  • Elle permet de calculer $\sin x$ quand on connaît $\cos x$ (et inversement).
  • Elle est valable pour tout angle, pas seulement aigus, mais en 3e on l'utilise dans le triangle rectangle.
  • $\sin x = \sqrt{1 - \cos^2 x}$ ou $\cos x = \sqrt{1 - \sin^2 x}$ (avec $x$ aigu donc positif).

Relation avec la tangente : $ \tan x = \dfrac{\sin x}{\cos x}$

La tangente d’un angle aigu peut être exprimée comme le quotient du sinus par le cosinus. C'est une définition équivalente à celle du rapport $\frac{\text{côté opposé}}{\text{côté adjacent}}$.

  • On peut ainsi calculer $\tan x$ à partir de $\sin x$ et $\cos x$.
  • Ou bien retrouver $\sin x$ ou $\cos x$ si l’on connaît $\tan x$ et une des deux autres valeurs (à l’aide de la première formule).
  • $\tan x = \frac{\sin x}{\cos x}$ est aussi très utile pour simplifier des expressions.

Lien pratique avec le triangle rectangle

Dans un triangle rectangle, si on note $\widehat{A}$ un angle aigu, alors : \[ \sin A = \frac{\text{côté opposé}}{\text{hypoténuse}}, \quad \cos A = \frac{\text{côté adjacent}}{\text{hypoténuse}}, \quad \tan A = \frac{\text{côté opposé}}{\text{côté adjacent}}. \] Les formules $\cos^2 A + \sin^2 A = 1$ et $\tan A = \frac{\sin A}{\cos A}$ sont alors automatiquement vérifiées.

Figure : Triangle rectangle ABC rectangle en B, angle aigu en A → côté opposé = BC, adjacent = AB, hypoténuse = AC.

💡 Exemples de compréhension

Exemple 1 : Calculer $\sin x$ connaissant $\cos x$

On sait que $\cos x = 0,6$ pour un angle aigu $x$. Déterminer la valeur exacte de $\sin x$ puis celle de $\tan x$.

Résolution :
On utilise $\cos^2 x + \sin^2 x = 1$
$(0,6)^2 + \sin^2 x = 1$ → $0,36 + \sin^2 x = 1$ → $\sin^2 x = 0,64$
Comme $x$ est aigu, $\sin x > 0$ donc $\sin x = \sqrt{0,64} = 0,8$.
Ensuite $\tan x = \dfrac{\sin x}{\cos x} = \dfrac{0,8}{0,6} = \dfrac{4}{3} \approx 1,333$.

💬 Astuce : Pensez toujours au signe. Dans le triangle rectangle, sinus et cosinus d’un angle aigu sont positifs. La racine carrée donne donc la valeur positive.

Exemple 2 : Calculer $\cos x$ à partir de $\tan x$

Soit $x$ un angle aigu tel que $\tan x = 2$. Calculer $\cos x$ et $\sin x$.

Méthode : On sait que $\tan x = \frac{\sin x}{\cos x} = 2$ donc $\sin x = 2 \cos x$.
On remplace dans $\cos^2 x + \sin^2 x = 1$ :
$\cos^2 x + (2\cos x)^2 = 1$ → $\cos^2 x + 4\cos^2 x = 1$ → $5\cos^2 x = 1$ → $\cos^2 x = \frac{1}{5}$.
Comme $x$ aigu, $\cos x = \sqrt{\frac{1}{5}} = \frac{\sqrt{5}}{5}$.
Ensuite $\sin x = 2 \times \frac{\sqrt{5}}{5} = \frac{2\sqrt{5}}{5}$.

💡 On peut vérifier : $\left(\frac{\sqrt{5}}{5}\right)^2 + \left(\frac{2\sqrt{5}}{5}\right)^2 = \frac{1}{25} + \frac{4}{25} = 1$.

Exemple 3 : Problème concret avec l’utilisation combinée

Dans un triangle rectangle, un angle aigu $\alpha$ vérifie $\sin \alpha = \frac{3}{5}$. Déterminer $\cos \alpha$ et $\tan \alpha$, puis calculer la longueur du côté adjacent si l'hypoténuse mesure 20 cm.

Étape 1 – Calcul de cosinus : $\cos^2 \alpha = 1 - \sin^2 \alpha = 1 - \frac{9}{25} = \frac{16}{25}$ → $\cos \alpha = \frac{4}{5}$ (positif car angle aigu).
Étape 2 – Tangente : $\tan \alpha = \frac{\sin \alpha}{\cos \alpha} = \frac{3/5}{4/5} = \frac{3}{4}$.
Étape 3 – Longueur du côté adjacent : $\cos \alpha = \frac{\text{côté adjacent}}{\text{hypoténuse}}$ → $\frac{4}{5} = \frac{\text{adjacent}}{20}$ → adjacent $= 20 \times \frac{4}{5} = 16$ cm.

📐 Rappel : Dans un triangle rectangle, une fois que l’on connaît deux rapports trigonométriques, le troisième s’en déduit facilement.

📐 Visualisation des formules

Cercle trigonométrique (angle aigu)

Plaçons un angle $x$ dans le cercle de rayon 1. Les coordonnées du point M sont $(\cos x, \sin x)$. Par le théorème de Pythagore, la distance OM vaut 1, d’où $\cos^2 x + \sin^2 x = 1$. De plus, la pente de la droite (OM) est $\frac{\sin x}{\cos x} = \tan x$ (lorsque $\cos x \neq 0$).

Cercle trigonométrique : lien direct entre $\cos x$, $\sin x$ et $\tan x$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier que $\cos^2 x + \sin^2 x = 1$ n'est pas $\cos^2 x + \sin^2 x = 0$ ! Et surtout ne pas confondre avec $(\cos x + \sin x)^2 = 1$ (cette égalité est fausse en général).
  • Attention : Pour utiliser $\tan x = \frac{\sin x}{\cos x}$, il faut que $\cos x \neq 0$ (ce qui est toujours vrai pour un angle aigu).
  • Ne pas oublier : Dans la relation $\sin x = \sqrt{1 - \cos^2 x}$, on prend la racine positive uniquement parce que l’angle est aigu. Si l’angle était obtus, le sinus pourrait être négatif, mais ce n’est pas au programme de 3e.
  • Confusion entre degrés et radians : Les formules sont indépendantes de l’unité, mais assurez-vous que votre calculatrice est en mode degré (DEG) lorsque vous travaillez avec des mesures en degrés.
  • Piège : Utiliser la relation $\tan x = \frac{\sin x}{\cos x}$ pour calculer un angle sans connaître $\sin$ et $\cos$ précisément. Préférer la définition directe avec les côtés du triangle.

Exercice 1 : Sachant que $\sin \beta = \frac{5}{13}$ pour un angle aigu $\beta$, calculez $\cos \beta$ et $\tan \beta$.
Exercice 2 : Si $\tan \gamma = 0,75$, déterminez $\sin \gamma$ et $\cos \gamma$ (valeurs exactes sous forme fractionnaire).
Exercice 3 : Dans un triangle rectangle, $\cos \theta = 0,28$. Quelle est la valeur de $\sin \theta$ ? En déduire $\tan \theta$.

Voir les réponses

Réponse 1 : $\cos \beta = \frac{12}{13}$, $\tan \beta = \frac{5}{12}$.
Réponse 2 : $\cos \gamma = \frac{4}{5}$, $\sin \gamma = \frac{3}{5}$ (car $\tan = 3/4$ puis système).
Réponse 3 : $\sin \theta = 0,96$ (car $\sqrt{1-0,0784}=0,96$), $\tan \theta = 0,96/0,28 \approx 3,4286$.

🧠 Pour l’exercice 2, posez $\sin x = k\cos x$ avec $k = \tan x$, puis remplacez dans l’identité fondamentale.

📋 Tableau récapitulatif

Les trois formules trigonométriques essentielles

  • $\cos^2 x + \sin^2 x = 1$ (Identité fondamentale)
  • $\tan x = \dfrac{\sin x}{\cos x}$ (Définition liant les trois rapports)
  • $\tan x = \dfrac{\text{côté opposé}}{\text{côté adjacent}}$ (définition géométrique, cohérente avec la précédente)

Grâce à ces deux relations, on peut passer de l’une à l’autre des trois lignes trigonométriques lorsqu’on connaît une seule valeur (et l’angle est aigu). Elles sont aussi utiles pour simplifier des expressions littérales.

🎯 Stratégie de résolution

  • 1. Si on connaît $\sin x$ ou $\cos x$, utiliser $\cos^2 x + \sin^2 x = 1$ pour trouver l’autre.
  • 2. Si on connaît $\tan x$, écrire $\sin x = \tan x \times \cos x$, substituer dans l’identité fondamentale pour obtenir $\cos x$.
  • 3. Vérifier la cohérence des signes (dans le triangle rectangle, tout est positif).
  • 4. Pour un problème concret, faire un schéma et identifier les rapports sinus, cosinus ou tangente utiles avant d’utiliser les formules.
💪 Astuce de professeur : Maîtrisez d'abord les définitions dans le triangle rectangle. Ensuite, les formules $\cos^2 + \sin^2 = 1$ et $\tan = \sin / \cos$ deviennent des outils puissants pour vérifier ou compléter vos calculs.
