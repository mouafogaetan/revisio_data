🎨 Infographie : Images bitmap & vectorielle

Informatique – 3ème (Cameroun) · UE 15 : Notions de base sur l'infographie

📌 Éléments clés à retenir

1. Image bitmap (matricielle) vs vectorielle

En infographie, on distingue principalement deux types d’images : les images bitmap (ou matricielles) et les images vectorielles. Le choix du type dépend de l’usage (photographie, logo, impression, web…).

🖼️ BITMAP (matricielle)

  • Composée d’une grille de pixels (points colorés).
  • Chaque pixel possède une couleur définie.
  • Perd en qualité quand on l’agrandit → pixelisation.
  • Idéale pour les photographies, images complexes.
  • Formats courants : JPEG, PNG, GIF, BMP, TIFF.

✒️ VECTORIELLE

  • Composée d’objets mathématiques (courbes, lignes, formes).
  • Définie par des formules (points, vecteurs).
  • Agrandissement sans perte de qualité (redimensionnable à l’infini).
  • Idéale pour les logos, illustrations, caractères typographiques.
  • Formats courants : SVG, AI (Adobe Illustrator), CDR (CorelDRAW), EPS.

2. Formats d’images et propriétés techniques

Chaque format possède des caractéristiques spécifiques : définition, résolution et poids (taille du fichier). Ces trois notions sont essentielles pour maîtriser l’infographie.

  • 📏 Définition : nombre total de pixels qui composent l’image (ex: 1920 × 1080). Plus elle est élevée, plus l’image est détaillée.
  • 🔍 Résolution : densité de pixels par unité de longueur (souvent en ppp = points par pouce). Pour l’impression, on utilise 300 ppp ; pour l’écran, 72 ppp suffisent.
  • ⚖️ Poids (taille fichier) : quantité de données stockées (en Ko, Mo). Une haute définition + absence de compression = poids élevé.

Formats bitmap principaux :

JPEG/JPG : compression avec perte, idéal pour photos (poids réduit). PNG : compression sans perte, supporte la transparence, parfait pour logos/icônes web. GIF : limité à 256 couleurs, permet des animations simples. BMP : non compressé, très lourd, peu utilisé aujourd’hui.

Formats vectoriels principaux :

SVG : standard ouvert pour le web, redimensionnable, léger. AI : format natif d’Adobe Illustrator (studio professionnel). CDR : format natif de CorelDRAW.

3. Résolution & impact visuel

La résolution influence directement la qualité d’affichage et d’impression. Une image avec une faible résolution apparaîtra pixelisée ou floue lorsqu’elle sera agrandie.

  • 72 ppp (points par pouce) : résolution standard pour les écrans (web, mobile).
  • 300 ppp ou plus : résolution recommandée pour l’impression (flyers, magazines).
  • Augmenter la résolution d’une image bitmap ne peut pas créer de nouveaux détails réels : on risque d’obtenir un effet flou.
🖨️ Gauche : rendu écran (72 ppp) – Droite : rendu imprimé (300 ppp).

💡 Exemples de compréhension

🎯 Exemple 1 : Logo d’entreprise ou photo de classe ?

Contexte : Le club informatique du collège veut créer deux supports : un logo pour ses affiches (qu’il pourra agrandir sur un grand panneau) et une affiche souvenir avec les photos des membres.

✅ Résolution :
• Pour le logo, il faut choisir un format vectoriel (SVG, AI). Ainsi, on pourra l’agrandir sans perte de qualité, que ce soit sur un badge ou sur un panneau 2 mètres.
• Pour l’affiche souvenir avec les photos, le format bitmap (JPEG ou PNG) est adapté. Il faut une résolution suffisante (au moins 150–300 ppp) pour que l’impression soit nette.

💡 Astuce : Demandez toujours le format vectoriel pour un logo destiné à l’impression ou au web responsive.

📊 Exemple 2 : Comprendre la relation définition / poids

Une image au format BMP (non compressé) a une définition de 800 × 600 pixels. Chaque pixel utilise 3 octets (rouge, vert, bleu). Quel est son poids approximatif ?

Calcul : Nombre total de pixels = 800 × 600 = 480 000 pixels.
Poids = 480 000 × 3 octets = 1 440 000 octets ≈ 1,44 Mo.
Si on enregistre cette image en JPEG (compression avec perte), le poids peut descendre à 200–300 Ko sans perte de qualité perceptible pour un usage courant.

🧠 À retenir : les formats compressés (JPEG, PNG) réduisent la taille des fichiers, ce qui est essentiel pour le stockage et le partage sur Internet.

🔎 Exemple 3 : Pourquoi mon image est-elle floue après agrandissement ?

Marie a téléchargé une petite photo de 200 × 150 pixels depuis un site web pour l’imprimer en format A4. À l’impression, l’image apparaît très floue et on voit des « carrés ».

Explication : L’image est une bitmap avec trop peu de pixels. En l’agrandissant pour remplir une page A4, le logiciel d’impression doit « inventer » des pixels, ce qui provoque la pixelisation. Pour imprimer en bonne qualité, il faut au minimum 1500 × 1000 pixels environ (selon la résolution visée).
➜ Solution : utiliser des images vectorielles quand on doit agrandir ou trouver une photo avec une définition plus élevée.

📌 Bonne pratique : toujours vérifier la définition d’une image avant de l’utiliser dans un projet imprimé.

⚠️ Pièges courants / À ne pas confondre

  • Confondre résolution et définition : la définition c’est le nombre de pixels (largeur × hauteur). La résolution exprime la densité (ppp). Une image peut avoir une grande définition et une faible résolution d’affichage.
  • Croire qu’un JPEG vectoriel existe : le JPEG est toujours une image bitmap. On ne peut pas le transformer en véritable vectoriel sans retracer les contours.
  • Oublier la transparence : pour un logo sur fond coloré, utiliser PNG ou SVG plutôt que JPEG qui ne gère pas la transparence.
  • Poids = qualité automatique ? Non, un fichier très lourd n’est pas toujours de meilleure qualité ; parfois une compression mal réglée alourdit inutilement sans gain visuel.
  • Utiliser une image web pour l’impression : les images prévues pour le web sont souvent en 72 ppp. Pour un flyer, préférer du 300 ppp ou une source vectorielle.

🖌️ Outils courants en infographie

Logiciels bitmap et vectoriels

  • Logiciels bitmap (matriciels) : Adobe Photoshop, GIMP (gratuit), Paint.NET, Krita.
  • Logiciels vectoriels : Adobe Illustrator, CorelDRAW, Inkscape (gratuit et open source), Affinity Designer.
  • Utilité : Pour la création de flyers, logos, affiches, maquettes, on associe parfois les deux types (vectoriel pour les formes, bitmap pour les textures/effets).
💡 À l’école : on peut utiliser Inkscape (gratuit) pour s’entraîner à la création vectorielle et GIMP pour retoucher des photos.
