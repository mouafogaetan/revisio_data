UE 11 : Utilisation des fonctions d'insertions et le correcteur automatique

Traitement de Texte (Word, Writer) · Classe de 3ème · Mise en page professionnelle

📌 Éléments clés à retenir

1. Insertion d'objets graphiques

Un logiciel de traitement de texte permet d'enrichir un document avec des éléments visuels : images, formes géométriques, tableaux, graphiques, etc.

  • Images : Insérer depuis un fichier, redimensionner, rogner, ajuster le texte autour (encadré, serré, derrière le texte…).
  • Formes : Flèches, organigrammes, zones de texte. Permettent de schématiser des idées.
  • Tableaux : Organiser des données en lignes et colonnes (fusion de cellules, bordures, alignement).
  • Graphiques : Représenter des données chiffrées (diagrammes en barres, circulaires). Souvent liés à un tableau de données.

Exemple visuel : Onglet "Insertion" → groupe "Illustrations" → Image.

2. Manipulation des objets insérés

Une fois inséré, un objet peut être modifié pour s'adapter à la mise en page.

  • Habillage du texte : options « Carré », « Serré », « Derrière le texte », « Devant le texte », « En ligne ».
  • Redimensionnement : utiliser les poignées d'angle pour garder les proportions.
  • Rognage (recadrage) : couper les parties inutiles d’une image sans supprimer définitivement.
  • Styles d’image : contours, effets (ombre, reflet), bordures pour harmoniser.

3. Le correcteur automatique : orthographe et grammaire

Un outil indispensable pour améliorer la qualité d’un document et éviter les fautes.

  • Soulignements : rouge = faute d’orthographe, vert/bleu = faute de grammaire ou de style.
  • Correction automatique : remplace automatiquement des erreurs fréquentes (ex : « j'ai » corrige « jai ») et ajoute des majuscules en début de phrase.
  • Dictionnaire personnel : permet d’ajouter des mots techniques (ex: "Cameroun", "informatique") pour ne plus qu’ils soient signalés.
  • Vérification grammaticale : accords sujet-verbe, accords des participes passés, etc.

4. Utilisation avancée : insertion de légendes et notes

Pour un document professionnel, on peut ajouter des légendes aux images/tableaux et des notes de bas de page.

  • Légende : texte descriptif sous une image ou un tableau (ex: « Figure 1 : Architecture de Von Neumann »).
  • Note de bas de page : commentaire placé en bas de page pour préciser une information sans alourdir le corps du texte.

💡 Exemples de compréhension

Exemple 1 : Créer un document avec image et légende

Consigne : Dans un rapport sur les composants d’un ordinateur, insérer une image de la carte mère, ajouter une légende et appliquer un habillage « Serré ».

Étapes :
  1. Placer le curseur à l'endroit souhaité.
  2. Aller dans l’onglet Insertion → Image → choisir le fichier.
  3. Sélectionner l’image, puis dans Options d’habillage choisir Serré.
  4. Cliquer droit sur l’image → Insérer une légende → taper « Figure 1 : Carte mère (vue d’ensemble) ».
💬 Astuce : Maintenir la touche Ctrl (ou Cmd) tout en redimensionnant permet de conserver les proportions de l’image.

Exemple 2 : Correction d’un texte fautif

Phrase d’origine : "L'informatique est une matiere fascinante. j'aime beaucoup les cour de programation."
Utiliser le correcteur automatique et la vérification grammaticale pour corriger les erreurs.

Correction après utilisation du correcteur :
"L'informatique est une matière fascinante. J'aime beaucoup les cours de programmation."
Explication : Le correcteur détecte « matiere » → propose « matière », « j'aime » → « J'aime » (majuscule automatique), « cour » → « cours » (orthographe), « programation » → « programmation » (double m).
💡 Pour vérifier tout le document rapidement : F7 (sous Word) ou Outils → Vérification orthographique (sous Writer).

Exemple 3 : Insérer un tableau et un graphique simple

Réaliser un tableau des effectifs d’élèves de 3e (classes A, B, C) avec les notes de passage, puis générer un graphique en colonnes.

Procédure :
  1. Insertion → Tableau (3x2). Saisir les données : Classe | Taux de réussite.
  2. Sélectionner le tableau → Insertion → Graphique (ou Diagramme).
  3. Choisir « Histogramme » (colonnes) → valider. Le graphique apparaît et peut être redimensionné.
Résultat : un graphique illustrant les données pour un rendu plus visuel.
🧠 Si les données du tableau changent, le graphique peut se mettre à jour automatiquement (si lié).

⚠️ Pièges courants / À ne pas confondre

  • Ne pas confondre habillage et position : L'habillage « En ligne » traite l'image comme un caractère, tandis que « Carré » permet de déplacer librement l'image.
  • Attention au rognage : Rogner une image ne réduit pas son poids (la partie rognée reste cachée). Pour réduire la taille du fichier, il faut compresser l’image.
  • Correction automatique excessive : Parfois, le correcteur transforme des mots techniques valides (ex: « onglet » devient « onglet » ? non, mais des noms propres peuvent être mal corrigés). Toujours relire.
  • Oublier les légendes : Dans un rapport, une image sans légende perd de son contexte. Penser à ajouter une courte description.
  • Graphique non adapté : Utiliser un diagramme circulaire pour des évolutions dans le temps est souvent inapproprié. Préférer un histogramme ou une courbe.

⌨️ Raccourcis et astuces utiles

  • F7 : Lancer la vérification orthographique/grammaticale (Word, Writer).
  • Ctrl + C / Ctrl + V : Copier/coller des objets (images, formes).
  • Maintenir Alt (ou Ctrl sous certains logiciels) en déplaçant une image : désactiver l’aimantation pour un positionnement précis.
  • Double-clic sur une image : ouvre rapidement les options de formatage (couleur, contour, effets).
  • Utiliser le volet de navigation (ou styles) pour mieux gérer les objets si le document est long.
