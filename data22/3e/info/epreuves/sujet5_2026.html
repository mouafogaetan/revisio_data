❓ QUESTION 1 (3 pts)
Quelles sont les trois grandes parties qui structurent généralement un algorithme (selon le programme vu en classe) ?
  • A. Entête, Traitement, Fin
  • B. Nom, Déclarations, Corps (instructions)
  • C. Début, Boucle, Condition
  • D. Initialisation, Itération, Sortie

    • ✅ Bonne réponse : B Un algorithme possède un nom (identifiant), une section de déclarations (variables, constantes) et le corps (séquence d'instructions).

    ❓ QUESTION 2 (3 pts)
    Parmi ces instructions simples, laquelle permet de stocker une valeur dans une variable en algorithmique ?
  • A. Lire(variable)
  • B. Ecrire(variable)
  • C. variable ← valeur
  • D. variable == valeur

    • ✅ Bonne réponse : C L'affectation se note généralement ← (ou = dans certains langages).
    "Lire" sert à saisir une donnée, "Ecrire" à afficher.

    ❓ QUESTION 3 (4 pts)
    Un élève souhaite écrire un algorithme qui affiche "Majeur" si l’âge est supérieur ou égal à 18 ans, sinon "Mineur". Quelle structure doit-il utiliser ?
  • A. Une boucle POUR
  • B. Une structure alternative SI...ALORS...SINON
  • C. Une boucle TANT QUE
  • D. Une simple séquence d'affectations

    • ✅ Bonne réponse : B La structure conditionnelle SI condition ALORS instructions SINON autres instructions permet de prendre une décision selon une condition logique.

    Exemple :
    SI age >= 18 ALORS Ecrire("Majeur") SINON Ecrire("Mineur")

    ❓ QUESTION 4 (3 pts)
    Laquelle de ces affirmations est VRAIE concernant la boucle "POUR" (ou "FOR") ?
  • A. On l'utilise quand on ne connaît pas le nombre d'itérations à l'avance.
  • B. On l'utilise principalement quand on connaît le nombre exact de répétitions.
  • C. Elle ne peut contenir qu'une seule instruction.
  • D. Elle permet uniquement de trier des données.

    • ✅ Bonne réponse : B La boucle POUR (itération déterminée) est adaptée quand le nombre de répétitions est connu d'avance (ex: afficher les nombres de 1 à 10). La boucle TANT QUE est utilisée pour les répétitions conditionnelles.

    ❓ QUESTION 5 (3 pts)
    Dans un algorigramme (organigramme), quel symbole représente généralement une condition ou un test logique (branchement conditionnel) ?
  • A. Un parallélogramme
  • B. Un rectangle
  • C. Un losange
  • D. Un cercle

    • ✅ Bonne réponse : C Le losange est le symbole normalisé pour représenter un test conditionnel (SI ... ALORS). Le rectangle correspond aux traitements, le parallélogramme aux entrées/sorties.

    📌 (Rappel : losange = décision / embranchement)

    ❓ QUESTION 6 (2 pts)
    Vrai ou Faux : Un langage de programmation de haut niveau (comme Python ou Java) doit être traduit en langage machine par un compilateur ou un interpréteur pour être exécuté par le processeur.
  • A. Vrai
  • B. Faux

    • ✅ Réponse : A. Vrai Le processeur ne comprend que le langage binaire (langage machine). Un compilateur (ou interpréteur) convertit le code source en instructions compréhensibles par la machine.

    ❓ QUESTION 7 (2 pts)
    Remettez dans l'ordre chronologique les grandes étapes du cycle de développement logiciel :
    1) Codage 2) Tests 3) Analyse des besoins 4) Maintenance 5) Conception
  • A. 3 → 5 → 1 → 2 → 4
  • B. 1 → 2 → 3 → 4 → 5
  • C. 5 → 3 → 1 → 2 → 4
  • D. 3 → 1 → 5 → 2 → 4

    • ✅ Bonne réponse : A On commence par l'analyse des besoins, puis la conception, ensuite le codage, les tests et enfin la maintenance.

    ✏️ QUESTION 8 (Exercice pratique – 5 pts)
    On souhaite écrire un algorithme qui calcule la moyenne de trois notes (N1, N2, N3) saisies par l'utilisateur, puis affiche "Félicitations" si la moyenne est supérieure ou égale à 10, sinon "Travail à fournir".
  • A. L'algorithme doit contenir une boucle TANT QUE.
  • B. L'algorithme doit utiliser une structure conditionnelle SI..ALORS..SINON après avoir calculé la moyenne.
  • C. Il n'est pas nécessaire de déclarer des variables.
  • D. Il faut obligatoirement une fonction avancée.

    • ✅ Bonne approche : B L'algorithme doit :

    • ✔ Déclarer les variables N1, N2, N3, moyenne (réel)
    • ✔ Lire(N1, N2, N3)
    • ✔ moyenne ← (N1+N2+N3)/3
    • ✔ SI moyenne >= 10 ALORS Ecrire("Félicitations") SINON Ecrire("Travail à fournir")

    La condition SI..SINON est indispensable pour l'affichage personnalisé. Aucune boucle n'est nécessaire ici.

    ❓ QUESTION 9 (2 pts)
    Quel est le rôle principal d'un compilateur ?
  • A. Exécuter le code ligne par ligne.
  • B. Traduire l'intégralité d'un programme source en langage machine avant l'exécution.
  • C. Gérer la mémoire vive uniquement.
  • D. Créer des interfaces graphiques.

    • ✅ Bonne réponse : B Le compilateur traduit tout le code source en un fichier exécutable (langage machine). L'interpréteur exécute instruction par instruction.

    📐 QUESTION 10 (Bonus / 1 pt supplémentaire)
    Dans un algorigramme, pour représenter une boucle "TANT QUE condition", on utilise généralement un losange (test) avec une flèche de retour en arrière. Cette affirmation est-elle correcte ?
  • A. Oui, le test conditionnel est dans un losange et on reboucle tant que la condition est vraie.
  • B. Non, les boucles se représentent uniquement par des rectangles.

    • ✅ Bonne réponse : A En organigramme, le test de la boucle TANT QUE se place dans un losange, et une flèche de retour permet de répéter le bloc d'instructions tant que la condition est vraie.
