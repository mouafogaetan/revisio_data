🔊 Lesson 2: Pronunciation and Silent Letters

Anglais · 3ème · Module 3 : Advanced Structures

📌 Éléments clés à retenir

1. Qu'est-ce qu'une "lettre silencieuse" ?

En anglais, de nombreux mots contiennent des lettres qui ne se prononcent pas. Ces lettres sont appelées silent letters. Les ignorer est une erreur courante pour les apprenants. Les reconnaître est essentiel pour une bonne prononciation et une bonne compréhension orale.

  • Pourquoi ? Raisons historiques, emprunts à d'autres langues (français, grec, latin).
  • Astuce mnémotechnique : Il faut souvent "voir" le mot avant de le prononcer.
  • Exemple : Dans "knee" (genou), le k est silencieux. On prononce /niː/.

2. Les familles de lettres silencieuses

Certaines lettres sont silencieuses dans des contextes bien précis. Voici les cas les plus fréquents à connaître pour le BEPC et le lycée.

  • 🔇 Silent 'b' : après 'm' (comb, thumb, climb) ou devant 't' (debt, doubt).
  • 🔇 Silent 'k' : devant 'n' en début de mot (knee, know, knife, knock).
  • 🔇 Silent 'w' : devant 'r' (write, wrong, wrist) ou dans certains mots (answer, sword).
  • 🔇 Silent 'h' : après 'g' (ghost, ghastly), après 'r' (rhino, rhyme) ou dans certains mots comme hour, honest, honor.
  • 🔇 Silent 't' : dans castle, listen, whistle, Christmas, often (parfois prononcé, mais souvent silencieux).
  • 🔇 Silent 'gh' : généralement silencieux, il allonge la voyelle (night, light, thought, through).
💡 Mémorise ces paires : write / right (homophones) – la lettre 'w' change l'orthographe mais pas le son.

3. Les différents phonèmes (sons) à ne pas confondre

En anglais, une même lettre peut produire des sons différents. Il faut savoir distinguer les voyelles courtes/longues et les consonnes voisées/non voisées.

  • 🎵 cease /siːs/ vs pleasure /ˈpleʒ.ər/ : Le 's' peut être /s/ (sourd) ou /z/ (voisé) ; 'ea' peut être /iː/ (long) ou /e/ (court).
    "cease" a le son doux /s/ tandis que "pleasure" a le son /ʒ/ (comme en français "jardin").
  • 🐻 bear /beər/ vs fear /fɪər/ : Même terminaison "-ear" mais prononciation différente. "Bear" rime avec "air", "fear" rime avec "ear". Il faut apprendre chaque mot.
  • 🎯 Les sons du 'th' : /θ/ sourd (think, thanks) et /ð/ voisé (this, that, mother).

4. L'importance de l'API (Alphabet Phonétique International)

Pour ne plus se tromper, il est utile de connaître quelques symboles phonétiques qui apparaissent dans le dictionnaire.

  • /θ/ : 'th' sourd (three, bath).
  • /ð/ : 'th' voisé (the, father).
  • /ʃ/ : 'sh' (she, action).
  • /ʒ/ : son 'j' français (vision, pleasure).
  • /ɪ/ vs /iː/ : ship vs sheep.

💡 Exemples de compréhension

🔊 Exemple 1 : Identifier les lettres silencieuses

Lis les mots suivants à voix haute, puis identifie la lettre qui ne se prononce pas : knight, calm, half, autumn, island.

✅ Correction :
- knight : le 'k' est silencieux → prononcé /naɪt/.
- calm : le 'l' est silencieux → /kɑːm/.
- half : le 'l' est silencieux → /hæf/.
- autumn : le 'n' est silencieux → /ˈɔː.təm/.
- island : le 's' est silencieux → /ˈaɪ.lənd/.
💬 Astuce : Quand tu rencontres un nouveau mot, regarde sa transcription phonétique dans le dictionnaire. Les lettres silencieuses n'apparaissent pas dans la transcription.

🎯 Exemple 2 : Distinguer les paires minimales (sons proches)

Prononce correctement ces paires de mots. La différence de son change le sens.

✅ Prononciation ciblée :
- ship /ʃɪp/ (bateau) vs sheep /ʃiːp/ (mouton).
- live /lɪv/ (vivre) vs leave /liːv/ (partir/quitter).
- thin /θɪn/ (mince) vs thing /θɪŋ/ (chose).
- pleasure /ˈpleʒ.ər/ vs pressure /ˈpreʃ.ər/.
💡 Place un doigt sous ta gorge : si ça vibre, le son est "voisé" (/ð/, /z/, /ʒ/). Si ça ne vibre pas, le son est "sourd" (/θ/, /s/, /ʃ/).

📖 Exemple 3 : Application en contexte

Lis le petit texte suivant en faisant attention aux lettres silencieuses et aux sons.
"I know the hour is late. Let’s listen to the whistle of the train. We write a knife in the story, but it’s wrong."

🔍 Décorticage :
- know : 'k' silencieux → /noʊ/.
- hour : 'h' silencieux → /aʊər/.
- listen : 't' silencieux → /ˈlɪs.ən/.
- whistle : 't' silencieux → /ˈwɪs.əl/.
- write : 'w' silencieux → /raɪt/.
- knife : 'k' silencieux → /naɪf/.
- wrong : 'w' silencieux → /rɒŋ/.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Prononcer le 'k' dans know, le 'b' dans climb ou le 't' dans castle.
  • Attention : Ne pas confondre le son /θ/ (think) et /ð/ (this). Un mauvais placement de la langue change le mot (ex: "tree" vs "three").
  • Piège visuel : Les mots comme bear et fear s'écrivent pareil à la fin mais ne se prononcent pas de la même manière. Il faut apprendre la phonétique par cœur.
  • Confusion orthographique : "Though" (bien que) se prononce /ðoʊ/, "thought" (pensée) /θɔːt/ et "through" (à travers) /θruː/. Ce sont des mots très différents malgré une orthographe proche.
🚨 Conseil de révision : Entraîne-toi avec un dictionnaire audio (type WordReference ou Cambridge Dictionary) en écoutant la prononciation des mots contenant des lettres silencieuses.
