Conditional Sentences: Zero, Type 1 & Type 2

Anglais · Classe de 3ème · Module 2 - Chapitre 4, Leçon 2

📌 Éléments clés à retenir

1. Zero Conditional (Condition Zéro) : Vérités Générales

On utilise le Zero Conditional pour parler de faits scientifiques, de vérités générales ou de routines. La condition est toujours vraie. On peut remplacer "if" par "when" (quand).

  • Structure : If + Présent Simple , Présent Simple
  • Exemple : If you heat water, it boils. (Si tu chauffes l'eau, elle bout.)
  • Exemple : If it rains, the ground gets wet. (S'il pleut, le sol devient mouillé.)

2. First Conditional (Type 1) : Possibilité Réelle

On utilise le First Conditional pour parler d'une situation future possible et de sa conséquence probable. La condition a une chance de se réaliser.

  • Structure : If + Présent Simple , will + Verbe (base)
  • Exemple : If you study hard, you will pass your exam. (Si tu étudies sérieusement, tu réussiras ton examen.)
  • On peut utiliser may, might, can à la place de will pour exprimer une probabilité moindre ou une possibilité.

3. Second Conditional (Type 2) : Situation Hypothétique

On utilise le Second Conditional pour parler d'une situation imaginaire, irréelle ou improbable dans le présent ou le futur. Elle exprime souvent un souhait ou un conseil.

  • Structure : If + Prétérit (Past Simple) , would + Verbe (base)
  • Attention ! Avec le verbe "to be" au prétérit, on utilise souvent "were" pour toutes les personnes (I, he, she, it).
  • Exemple : If I were rich, I would travel the world. (Si j'étais riche, je voyagerais autour du monde.)
  • Exemple : If she studied more, she would get better grades. (Si elle étudiait plus, elle aurait de meilleures notes.)

💡 Exemples de compréhension

Exemple concret : Distinguer Type 1 et Type 2

Contexte 1 (Type 1): John a un examen demain. Il n'a pas encore révisé.

Phrase correcte : "If you don't revise tonight, you will fail the exam."
Explication : C'est une situation réelle et possible. Il peut encore réviser ce soir, donc la conséquence est probable.

💬 Astuce : Pour le Type 1, on imagine un scénario qui pourrait arriver.

Exemple concret : Utiliser le Second Conditional

Contexte 2 (Type 2): John n'aime pas l'école et il ne peut pas s'empêcher de rêver. Il n'est pas un génie.

Phrase correcte : "If I were a genius, I would not need to study."
Explication : C'est une situation hypothétique. John n'est pas un génie, c'est un rêve. On utilise "were" (prétérit) et "would".

💡 Pour le Type 2, la condition est imaginaire ou contraire à la réalité. On utilise souvent "were" pour "if I were you" (à ta place) pour donner un conseil.

Zero Conditional : Une vérité scientifique

Contexte : Le professeur explique un phénomène naturel aux élèves.

Phrase correcte : "If there is a lot of pollution, global warming increases."
Explication : C'est un fait avéré et constant. On utilise le présent simple dans les deux parties de la phrase.

💬 Astuce : Pour le Zero Conditional, on peut remplacer "if" par "when" (quand). "When there is a lot of pollution, global warming increases."

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente n°1 : Utiliser "will" dans la partie "if". La condition (if) ne prend jamais "will". On dit "If it rains, I will stay home." et non "If it will rain...".
  • Erreur fréquente n°2 : Confondre le Type 1 et le Type 2. Rappelez-vous : Type 1 = Réel/Possible → If + présent, will + base. Type 2 = Irréel/Imaginaire → If + prétérit, would + base.
  • Attention à la forme : Dans le Type 2, "was" devient souvent "were". "If I were you" est une expression figée très courante pour donner un conseil.
  • Important : L'ordre des propositions peut être inversé. Si la condition est en second, pas de virgule. "I will stay home if it rains."

💡 Pour bien choisir, posez-vous la question : "Est-ce que cette situation a une chance réelle d'arriver ?" Si oui → Type 1. Si c'est un rêve ou une situation impossible → Type 2.
