Équations & inéquations du second degré

Mathématiques · Première C · Outil de révision

📌 Éléments clés à retenir

1. Discriminant Δ et racines réelles

Pour un trinôme $ax^2 + bx + c = 0$ avec $a \neq 0$, on calcule le discriminant $\Delta = b^2 - 4ac$. Il détermine l’existence et le nombre de solutions réelles :

  • Δ > 0 : deux racines distinctes $x_1 = \dfrac{-b - \sqrt{\Delta}}{2a}$ et $x_2 = \dfrac{-b + \sqrt{\Delta}}{2a}$.
  • Δ = 0 : une racine double $x_0 = -\dfrac{b}{2a}$.
  • Δ < 0 : aucune solution réelle.
  • Formule à retenir : $x = \frac{-b \pm \sqrt{\Delta}}{2a}$

2. Factorisation et signe du trinôme

Si $\Delta \ge 0$, on factorise le trinôme : $ax^2 + bx + c = a(x - x_1)(x - x_2)$ (ou $a(x - x_0)^2$ si Δ=0).
Le signe de $ax^2+bx+c$ suit la règle :

  • $a > 0$ : trinôme positif à l’extérieur des racines, négatif entre les racines.
  • $a < 0$ : trinôme négatif à l’extérieur des racines, positif entre les racines.
  • Cas Δ=0 : le trinôme est toujours du signe de $a$ (sauf en $x_0$ où il s’annule).
  • Cas Δ<0 : le trinôme a toujours le signe de $a$ pour tout $x \in \mathbb{R}$.
💡 Tableau de signe systématique : placer les racines dans l’ordre croissant puis appliquer la règle.

3. Relations somme et produit des racines

Si $x_1$ et $x_2$ sont les racines (réelles ou complexes), alors sans calculer $\Delta$ :

  • Somme : $S = x_1 + x_2 = -\dfrac{b}{a}$
  • Produit : $P = x_1 \cdot x_2 = \dfrac{c}{a}$
  • Utile pour former une équation connaissant $S$ et $P$ : $X^2 - SX + P = 0$.

💡 Exemples de compréhension

Exemple 1 : Résoudre une équation complète

Résoudre dans $\mathbb{R}$ : $2x^2 - 5x + 2 = 0$

Résolution :
On identifie $a=2$, $b=-5$, $c=2$.
$\Delta = (-5)^2 - 4 \times 2 \times 2 = 25 - 16 = 9$ → $\sqrt{\Delta}=3$.
$x_1 = \frac{5 - 3}{4} = \frac{2}{4} = \frac{1}{2}$, $x_2 = \frac{5 + 3}{4} = \frac{8}{4} = 2$.
Les solutions sont $\boxed{\frac{1}{2}}$ et $\boxed{2}$.

💬 Astuce : Vérifier rapidement la somme $\frac{1}{2}+2 = \frac{5}{2}$ et produit $1$ → $\frac{c}{a}=\frac{2}{2}=1$ OK.

Exemple 2 : Inéquation & tableau de signes

Résoudre $-x^2 + 3x + 4 \ge 0$

Démarche :
$a=-1$, $b=3$, $c=4$. $\Delta = 9 - 4(-1)(4) = 9+16 = 25$. Racines : $x_1 = \frac{-3 -5}{-2}= \frac{-8}{-2}=4$ ; $x_2 = \frac{-3+5}{-2}= \frac{2}{-2}=-1$.
Comme $a=-1<0$, le trinôme est positif entre les racines.
Tableau de signe :

Donc $S = [-1; 4]$.

⚡ Toujours ordonner les racines : $x_1 \le x_2$ pour éviter les erreurs de signe.

Exemple 3 : Utiliser la somme et le produit

Trouver deux nombres réels dont la somme est 7 et le produit 12.

Ces nombres sont les racines de $X^2 - 7X + 12 = 0$.
$\Delta = 49 - 48 = 1$ → $X_1 = \frac{7-1}{2}=3$, $X_2=\frac{7+1}{2}=4$.
Les deux nombres sont $3$ et $4$.

📐 Schéma récapitulatif

Allure de la parabole selon le signe de $a$ et la valeur de $\Delta$. Cela permet de visualiser directement le signe du trinôme.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de vérifier que $a \neq 0$ avant de calculer $\Delta$.
  • Attention : Ne pas confondre la formule des racines : $x = \frac{-b \pm \sqrt{\Delta}}{2a}$ (pas $\frac{-b \pm \sqrt{\Delta}}{a}$).
  • Piège : Pour une inéquation, ne jamais multiplier par un facteur qui change de signe sans étudier le cas. Toujours utiliser le tableau de signes.
  • À surveiller : Dans les relations somme/produit, bien vérifier que les racines existent ($\Delta \ge 0$).

🧠 Méthodes pas à pas

Pour résoudre une équation du second degré

  1. Écrire l’équation sous forme $ax^2+bx+c=0$.
  2. Calculer $\Delta = b^2-4ac$.
  3. Si $\Delta < 0$ : pas de solution réelle → ensemble vide.
  4. Si $\Delta = 0$ : solution unique $x_0 = -\frac{b}{2a}$.
  5. Si $\Delta > 0$ : deux solutions $x_{1,2}=\frac{-b\pm\sqrt{\Delta}}{2a}$.

Pour résoudre une inéquation du second degré

  1. Ramener tous les termes dans un membre (0 dans l’autre).
  2. Calculer les racines du trinôme si elles existent.
  3. Dresser un tableau de signes en plaçant les racines ordonnées.
  4. Déterminer l’intervalle solution selon le signe demandé ($\ge 0$, $\le 0$, $>0$, $<0$).
📌 Astuce globale : Pour factoriser rapidement quand $\Delta$ est un carré parfait, on peut aussi utiliser la forme canonique.

✏️ Teste-toi (mini exercice)

Résoudre l’inéquation $3x^2 - 5x - 2 < 0$

Voir la solution

$\Delta = 25 + 24 = 49$, racines $x_1 = \frac{5-7}{6} = -\frac{1}{3}$, $x_2 = \frac{5+7}{6}=2$.
$a=3>0$ ⇒ trinôme négatif entre les racines. Donc $S = \left]-\frac{1}{3}; 2\right[$.
