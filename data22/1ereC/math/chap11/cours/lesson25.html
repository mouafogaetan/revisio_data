📊 Séries statistiques regroupées en classes

Mathématiques · Première C · Statistiques et probabilités

📌 Éléments clés à retenir

🔢 1. Organisation en classes

Lorsque l'effectif est grand ou que la variable est continue, on regroupe les valeurs en intervalles de classe (souvent de même amplitude). Chaque classe est définie par sa borne inférieure et sa borne supérieure. On utilise alors le centre de classe pour les calculs.

  • Centre de classe $c_i = \frac{\text{borne inf} + \text{borne sup}}{2}$
  • Amplitude d'une classe : $\text{borne sup} - \text{borne inf}$ (souvent constante).
  • Effectif total $N = \sum n_i$ (somme des effectifs de chaque classe).
  • Fréquence : $f_i = \frac{n_i}{N}$ ; fréquence cumulée croissante/décroissante.

📈 2. Histogramme et polygone

Deux outils visuels indispensables pour analyser la répartition des données.

  • Histogramme : diagramme en barres jointives. La hauteur de chaque barre est proportionnelle à l'effectif (ou à la densité si les amplitudes sont inégales).
  • Polygone des effectifs : on joint les points d'abscisse le centre de classe et d'ordonnée l'effectif (ou fréquence).
  • Polygone des effectifs cumulés croissants (ECC) : utile pour déterminer la médiane graphiquement.

📍 3. Moyenne, mode et médiane

Pour des données regroupées en classes, on utilise des formules approchées (sauf le mode qui est la classe avec le plus grand effectif).

  • Moyenne : $\overline{x} = \dfrac{\sum n_i \cdot c_i}{N}$ où $c_i$ est le centre de la classe.
  • Mode (classe modale) : classe ayant l'effectif le plus élevé.
  • Médiane : valeur qui partage la série en deux parties égales. On la détermine par interpolation linéaire dans la classe médiane (celle où l'ECC dépasse N/2).
💡 Astuce : Pour la médiane, on utilise souvent : $M_e = L + \frac{\frac{N}{2} - F}{f} \times a$ (avec $L$ = borne inf de la classe médiane, $F$ = effectif cumulé avant la classe, $f$ = effectif de la classe médiane, $a$ = amplitude).

🎲 4. Variance et écart-type

Indicateurs de la dispersion autour de la moyenne.

  • Variance : $V = \dfrac{\sum n_i (c_i - \overline{x})^2}{N}$ (formule de la moyenne pondérée).
  • Écart-type : $\sigma = \sqrt{V}$ (même unité que la variable).
  • Formule de König : $V = \dfrac{\sum n_i c_i^2}{N} - (\overline{x})^2$ (plus rapide pour les calculs).

💡 Exemples de compréhension

📐 Exemple 1 : Tailles des élèves

On a mesuré la taille (en cm) de 50 élèves et regroupé en classes : [150, 160[ : 5 élèves ; [160, 170[ : 15 élèves ; [170, 180[ : 20 élèves ; [180, 190[ : 10 élèves. Calculer la moyenne, la classe modale et la médiane.

Résolution :
1. Centres : $c_1=155$, $c_2=165$, $c_3=175$, $c_4=185$.
2. Moyenne : $\overline{x} = \frac{5\times155 + 15\times165 + 20\times175 + 10\times185}{50} = \frac{775+2475+3500+1850}{50} = \frac{8600}{50} = 172$ cm.
3. Classe modale : [170, 180[ (effectif 20).
4. Médiane : $N/2 = 25$. Effectifs cumulés : 5, 20, 40, 50. La classe [170,180[ contient la 25e valeur.
$M_e = 170 + \frac{25-20}{20}\times10 = 170 + \frac{5}{20}\times10 = 170 + 2,5 = 172,5$ cm.

💬 Astuce : Pour les ECC, on additionne progressivement. La médiane se trouve dans la classe où l'effectif cumulé atteint ou dépasse N/2.

📉 Exemple 2 : Dispersion des notes

Dans un devoir, les notes (sur 20) sont réparties ainsi : [0;5[ : 4 élèves, [5;10[ : 8 élèves, [10;15[ : 12 élèves, [15;20] : 6 élèves. Calculer la variance et l'écart-type.

Calcul :
Centres : 2,5 ; 7,5 ; 12,5 ; 17,5. Effectif total $N=30$.
Moyenne : $\overline{x} = \frac{4\times2,5 + 8\times7,5 + 12\times12,5 + 6\times17,5}{30} = \frac{10+60+150+105}{30} = \frac{325}{30} \approx 10,83$.
Utilisons la formule de König : $\sum n_i c_i^2 = 4\times(2,5^2) + 8\times(7,5^2) + 12\times(12,5^2) + 6\times(17,5^2) = 4\times6,25 + 8\times56,25 + 12\times156,25 + 6\times306,25 = 25 + 450 + 1875 + 1837,5 = 4187,5$.
Variance : $V = \frac{4187,5}{30} - (10,83)^2 \approx 139,583 - 117,289 \approx 22,294$.
Écart-type : $\sigma = \sqrt{22,294} \approx 4,72$ (sur 20).

🧠 Rappel : L'écart-type mesure la dispersion moyenne autour de la moyenne. Plus il est grand, plus les valeurs sont étalées.

📊 Exemple 3 : Construction d'histogramme

À partir du tableau des tailles de l'exemple 1, construire l'histogramme et le polygone des effectifs.

Méthode : En abscisse, placer les classes (150,160,...). En ordonnée, les effectifs. Dessiner des rectangles adjacents de hauteurs 5, 15, 20, 10. Pour le polygone, placer les points aux centres de classes (155 ;5), (165 ;15), (175 ;20), (185 ;10) et les relier.

✏️ À savoir : Le polygone des effectifs cumulés croissants permet de lire directement la médiane (intersection avec la droite horizontale d'ordonnée N/2).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Utiliser les bornes au lieu des centres pour calculer la moyenne. On doit toujours prendre le centre de classe.
  • Attention : Si les amplitudes de classe sont inégales, l'histogramme se construit avec des aires proportionnelles aux effectifs (hauteur = effectif / amplitude).
  • Piège : Confondre la classe médiane (celle qui contient la médiane) et la médiane (une valeur numérique). La médiane se calcule par interpolation.
  • À ne pas oublier : L'écart-type s'exprime dans l'unité de la variable (contrairement à la variance).

📐 Formulaire rapide

  • Centre : $c_i = \frac{L_i + L_{i+1}}{2}$
  • Moyenne : $\overline{x} = \frac{1}{N}\sum n_i c_i$
  • Variance (définition) : $V = \frac{1}{N}\sum n_i (c_i - \overline{x})^2$
  • Variance (König) : $V = \frac{1}{N}\sum n_i c_i^2 - (\overline{x})^2$
  • Écart-type : $\sigma = \sqrt{V}$
  • Médiane (interpolation) : $M_e = L + \frac{\frac{N}{2} - F}{f} \times a$
