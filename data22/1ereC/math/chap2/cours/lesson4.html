Systèmes d'équations linéaires

Mathématiques · Première C · Leçon 4 | 2 et 3 inconnues

📌 Éléments clés à retenir

🔹 Définition & structure d’un système linéaire

Un système d’équations linéaires est un ensemble d’équations du premier degré à plusieurs inconnues.
Exemple (2 inconnues) : \[ \begin{cases} ax + by = e \\ cx + dy = f \end{cases} \] où \(a,b,c,d,e,f\) sont des réels. Une solution est un couple \((x;y)\) qui vérifie les deux équations simultanément.

  • Pour 2 inconnues : une solution unique, une infinité ou aucune.
  • Pour 3 inconnues : un triplet \((x;y;z)\) solution.
  • Interprétation géométrique : chaque équation = une droite (dans le plan) ou un plan (dans l’espace).

🔹 Méthodes de résolution (2 inconnues)

Deux méthodes principales sont utilisées en Première C :

Substitution Combinaison (addition) Graphique (interprétation)
  • Substitution : Isoler une inconnue dans une équation, remplacer dans l’autre.
  • Combinaison : Multiplier les équations par des coefficients puis additionner/soustraire pour éliminer une inconnue.
  • Graphique : Tracer les deux droites. Leur intersection donne la solution.
  • Formule à connaître (système \( \begin{cases} a x + b y = e \\ c x + d y = f \end{cases} \)) : le déterminant \( \Delta = a d - b c \). Si \( \Delta \neq 0 \) : solution unique.

🔹 Systèmes à 3 inconnues (méthode du pivot / substitution)

Résoudre un système du type : \[ \begin{cases} a_1 x + b_1 y + c_1 z = d_1 \\ a_2 x + b_2 y + c_2 z = d_2 \\ a_3 x + b_3 y + c_3 z = d_3 \end{cases} \] On utilise soit la substitution progressive (exprimer x puis y) soit la méthode de combinaison (éliminer une variable, se ramener à un système 2x2).

  • 💡 Astuce : Commencer par éliminer la même inconnue dans deux paires d’équations.
  • Vérifier toujours la solution trouvée dans les trois équations.
  • Interprétation dans l’espace : trois plans sécants en un point (solution unique), une droite commune (infinité) ou aucun point commun.

🔹 Interprétation géométrique (essentiel)

Dans le plan (2 inconnues) :
✔️ Droites sécantes → solution unique.
✔️ Droites parallèles distinctes → aucune solution.
✔️ Droites confondues → infinité de solutions.

Dans l’espace (3 inconnues) : chaque équation représente un plan. L’ensemble des solutions peut être un point, une droite, un plan ou vide.

💡 Exemples de compréhension

Exemple 1 : Système 2×2 par substitution

Résoudre dans \(\mathbb{R}^2\) le système : \[ \begin{cases} 2x + 3y = 7 \quad (1) \\ x - y = 1 \quad (2) \end{cases} \]

Résolution par substitution :
• De (2) : \(x = 1 + y\).
• On remplace dans (1) : \(2(1+y) + 3y = 7 \) → \(2 + 2y + 3y = 7\) → \(5y = 5\) → \(y = 1\).
• Puis \(x = 1 + 1 = 2\).
• Vérification : (1) \(2×2+3×1 = 4+3=7\) ; (2) \(2-1=1\). ✅
Solution unique : \((x;y) = (2;1)\).
💬 Astuce : Toujours vérifier dans l’équation non utilisée pour éviter les erreurs de calcul.

Exemple 2 : Système 2×2 par combinaison

Résoudre par combinaison linéaire : \[ \begin{cases} 3x + 2y = 8 \\ 5x - 4y = 6 \end{cases} \]

Méthode : Multiplions la première équation par 2 : \(6x + 4y = 16\).
Ajoutons à la seconde équation : \((6x+4y) + (5x-4y) = 16+6 \) → \(11x = 22 \) → \(x=2\).
Remplaçons \(x\) dans \(3x+2y=8\) : \(6+2y=8 \) → \(2y=2 \) → \(y=1\).
Solution : \((2;1)\). Vérifions dans la 2e : \(5×2 - 4×1 = 10-4=6\) ✅.
💡 La combinaison est très efficace quand les coefficients sont bien choisis.

Exemple 3 : Système 3×3 (substitution progressive)

Résoudre le système : \[ \begin{cases} x + y + z = 6 \quad (L_1) \\ 2x - y + z = 3 \quad (L_2) \\ x + 2y - z = 3 \quad (L_3) \end{cases} \]

Étapes :
• De \(L_1\) : \(x = 6 - y - z\).
• Remplacer dans \(L_2\) : \(2(6 - y - z) - y + z = 3 \) → \(12 - 2y -2z - y + z = 3\) → \(12 - 3y - z = 3\) → \(-3y - z = -9\) → \(3y + z = 9\) (équation A).
• Remplacer dans \(L_3\) : \((6 - y - z) + 2y - z = 3\) → \(6 + y - 2z = 3\) → \(y - 2z = -3\) (équation B).
• Système 2×2 : \( \begin{cases} 3y + z = 9 \\ y - 2z = -3 \end{cases} \).
• De B : \(y = 2z - 3\). Substituer dans A : \(3(2z-3) + z = 9\) → \(6z - 9 + z = 9\) → \(7z = 18\) → \(z = \frac{18}{7}\).
• \(y = 2×\frac{18}{7} - 3 = \frac{36}{7} - \frac{21}{7} = \frac{15}{7}\).
• \(x = 6 - \frac{15}{7} - \frac{18}{7} = \frac{42}{7} - \frac{33}{7} = \frac{9}{7}\).
Solution : \( \left(\frac{9}{7}; \frac{15}{7}; \frac{18}{7}\right) \). Vérification rapide ✅.
📌 Dans les systèmes 3×3, la clé est d’éliminer méthodiquement la même variable pour obtenir un système 2×2.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de vérifier les solutions dans toutes les équations, surtout pour les systèmes 3×3.
  • Attention : Dans la méthode de combinaison, multiplier par un facteur négatif peut inverser les signes, vérifiez l’addition.
  • Piège : Confondre « aucune solution » (droites parallèles) et « infinité de solutions » (droites confondues). Pour un système 2×2 : Si \( \frac{a}{c} = \frac{b}{d} \neq \frac{e}{f} \) → pas de solution ; si tous les rapports égaux → infinité.
  • Rappel : Une équation comme \(0x + 0y = 5\) est impossible → système incompatible.

📐 Synthèse : démarche de résolution

Étapes à suivre pour tout système linéaire

  • 1. Identifier le nombre d’inconnues (2 ou 3).
  • 2. Choisir la méthode adaptée : substitution si une variable est facile à isoler, combinaison pour éliminer rapidement.
  • 3. Pour 3 inconnues : réduire progressivement jusqu’à un système 2×2, puis résoudre.
  • 4. Toujours vérifier la solution dans le système d’origine.
  • 5. Interpréter géométriquement si demandé : intersection de droites ou de plans.

Cas particulier : Un système linéaire homogène (second membre nul) admet toujours au moins la solution nulle.

Règles d’or pour les équivalences

  • On ne modifie pas les solutions en : multipliant une équation par un réel non nul ; ajoutant (ou soustrayant) deux équations ; échangeant deux équations.
  • Ne jamais « diviser » par une expression contenant l’inconnue (risque de perte de solutions).

Solution (à cacher puis vérifier) :
• Combinaison : multiplier la 2e équation par 2 → \(6x + 2y = 10\).
