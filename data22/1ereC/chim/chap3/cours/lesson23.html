Leçon 23 : Tests d'identification des aldéhydes et cétones

Chimie organique · Première C & D · Composés oxygénés

📌 Éléments clés à retenir

1. Principe général des tests

Les aldéhydes et les cétones sont deux familles de composés carbonylés ($C=O$). Les aldéhydes possèdent un atome d'hydrogène sur le carbone du carbonyle ($R-CHO$) alors que les cétones ont deux groupements alkyles ou aryles ($R-CO-R'$). Cette différence structurale permet de les distinguer par des tests chimiques spécifiques.

  • Test à la 2,4-DNPH : commun aux aldéhydes et aux cétones (présence du carbonyle).
  • Tests de Schiff, Fehling, Tollens : réservés aux aldéhydes (pouvoir réducteur).
  • Les cétones ne réagissent pas avec les réactifs de Schiff, Fehling ou Tollens.

2. Test à la 2,4-DNPH (dinitrophénylhydrazine)

C'est le test de reconnaissance de la fonction carbonyle ($C=O$). Tous les aldéhydes et cétones donnent une réaction positive.

  • Réactif : 2,4-dinitrophénylhydrazine (solution dans l'acide phosphorique ou sulfurique dilué).
  • Résultat : formation d'un précipité jaune à orangé (dinitrophénylhydrazone).
  • Équation : $R_1R_2C=O + H_2N-NH-C_6H_3(NO_2)_2 \rightarrow R_1R_2C=N-NH-C_6H_3(NO_2)_2 + H_2O$
  • Utilisation : mettre en évidence la présence d'un aldéhyde ou d'une cétone.
💡 Astuce : Un précipité jaune indique la présence d'un carbonyle. Ce test ne permet pas de distinguer aldéhyde et cétone.

3. Test au réactif de Schiff (fuchsine décolorée)

Le réactif de Schiff est une solution incolore de fuchsine décolorée par l'acide sulfureux. Il réagit spécifiquement avec les aldéhydes.

  • Résultat avec un aldéhyde : apparition d'une coloration rose magenta caractéristique.
  • Résultat avec une cétone : aucune réaction (solution reste incolore).
  • Mécanisme : l'aldéhyde réagit avec l'acide sulfonique formé pour régénérer la fuchsine colorée.

4. Test à la liqueur de Fehling

La liqueur de Fehling est un oxydant doux à base de cuivre(II) en milieu basique (complexe tartrate cuivrique).

  • Aspect initial : solution bleue (ions $Cu^{2+}$ complexés).
  • Avec un aldéhyde (sauf aromatiques) : formation d'un précipité rouge brique d'oxyde de cuivre(I) $Cu_2O$.
  • Équation bilan : $R-CHO + 2Cu^{2+} + 5OH^- \rightarrow R-COO^- + Cu_2O \downarrow + 3H_2O$
  • Avec une cétone : aucune réaction (la solution reste bleue).
  • Remarque : les aldéhydes aromatiques (comme le benzaldéhyde) ne réagissent pas avec Fehling.

5. Test au réactif de Tollens (miroir d'argent)

Le réactif de Tollens est une solution ammoniacale de nitrate d'argent $[Ag(NH_3)_2]^+$ , qui est un oxydant doux.

  • Aspect initial : solution incolore.
  • Avec un aldéhyde : réduction des ions argent en argent métallique qui se dépose sur les parois du tube : miroir d'argent caractéristique.
  • Équation : $R-CHO + 2[Ag(NH_3)_2]^+ + 2OH^- \rightarrow R-COO^- + 2Ag \downarrow + 4NH_3 + H_2O$
  • Avec une cétone : aucune réaction (pas de miroir d'argent).
  • Ce test fonctionne avec tous les aldéhydes (aliphatiques et aromatiques).

💡 Exemples de compréhension

Exemple 1 : Distinguer éthanal (aldéhyde) et propanone (cétone)

On dispose de deux flacons notés A et B contenant respectivement de l'éthanal ($CH_3CHO$) et de la propanone ($CH_3COCH_3$). Proposer un protocole simple pour identifier chaque flacon.

Résolution :
1. On prélève un échantillon de chaque flacon.
2. On ajoute quelques gouttes de liqueur de Fehling à chaque tube et on chauffe doucement.
  - Le tube qui donne un précipité rouge brique contient l'éthanal (aldéhyde).
  - Le tube qui reste bleu contient la propanone (cétone).
3. On peut confirmer avec le réactif de Tollens : l'éthanal donne un miroir d'argent, la propanone ne réagit pas.
💬 Astuce : Le test de Fehling est plus rapide pour distinguer un aldéhyde aliphatique d'une cétone. Attention : les aldéhydes aromatiques ne réagissent pas avec Fehling mais réagissent avec Tollens.

Exemple 2 : Analyse d'un composé inconnu X

Un composé organique X donne un précipité jaune avec la 2,4-DNPH. Par contre, il ne modifie ni la liqueur de Fehling ni le réactif de Tollens. Que peut-on conclure ?

Explication :
- Test 2,4-DNPH positif : X possède une fonction carbonyle ($C=O$).
- Tests de Fehling et Tollens négatifs : X n'est pas un aldéhyde (ou aldéhyde très particulier ? mais ici négatif).
- Conclusion : X est très probablement une cétone. Les cétones possèdent le carbonyle mais ne sont pas oxydables par les réactifs doux (Fehling, Tollens, Schiff).

Exemple 3 : Identifier un sucre réducteur

Les sucres réducteurs (comme le glucose) possèdent un groupement aldéhyde libre. Comment les mettre en évidence ?

Méthode : On réalise le test de Fehling ou de Tollens sur une solution de glucose. On observe un précipité rouge brique (Fehling) ou un miroir d'argent (Tollens), confirmant le caractère réducteur dû à la fonction aldéhyde. Les cétones ne réagissent pas.
💡 Les tests de Fehling et Tollens sont utilisés en biochimie pour détecter les sucres réducteurs.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Penser que le test à la 2,4-DNPH distingue aldéhydes et cétones → Faux, il est positif pour les deux.
  • Attention : Les aldéhydes aromatiques (comme le benzaldéhyde) ne réagissent pas avec la liqueur de Fehling mais réagissent avec Tollens.
  • Piège : Certaines cétones alpha-hydroxylées peuvent donner un faux positif avec Fehling par réarrangement. Mais dans le cadre du programme, on considère les tests comme spécifiques.
  • Ne pas confondre : Le réactif de Schiff donne une couleur rose uniquement avec les aldéhydes ; les cétones ne colorent pas.
  • Condition expérimentale : Pour le test de Tollens, le tube doit être parfaitement propre pour obtenir un beau miroir d'argent. La réaction doit être réalisée sans excès d'ammoniac.

📝 Équations clés à connaître

  • Test à la 2,4-DNPH (général) : $R_2C=O + H_2N-NH-C_6H_3(NO_2)_2 \rightarrow R_2C=N-NH-C_6H_3(NO_2)_2 + H_2O$
  • Test de Fehling (aldéhyde aliphatique) : $R-CHO + 2Cu^{2+} + 5OH^- \rightarrow R-COO^- + Cu_2O \downarrow + 3H_2O$
  • Test de Tollens (aldéhyde) : $R-CHO + 2[Ag(NH_3)_2]^+ + 2OH^- \rightarrow R-COO^- + 2Ag \downarrow + 4NH_3 + H_2O$
