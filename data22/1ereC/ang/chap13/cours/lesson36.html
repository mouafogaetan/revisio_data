📘 So / Such...that : Exprimer la conséquence

Grammaire – 1ère CD – Leçon 23 (Programme Probatoire)

📌 Éléments clés à retenir

1️⃣ So + adjectif / adverbe + that

On utilise so + adjectif/adverbe + that pour exprimer une conséquence extrême. So renforce l'adjectif ou l'adverbe, et that introduit le résultat.

  • Structure : Sujet + verbe + so + adjectif/adverbe + that + conséquence
  • Exemple : The movie was so boring that I fell asleep.
    → (Le film était tellement ennuyeux que je me suis endormi.)
  • Exemple avec adverbe : He ran so fast that nobody could catch him.
    → (Il courait tellement vite que personne ne pouvait l'attraper.)
💡 Astuce : En français, "so...that" se traduit souvent par "tellement...que" ou "si...que".

2️⃣ Such + (adjectif) + nom + that

On utilise such + (adjectif) + nom + that quand on veut mettre l'accent sur un nom (avec ou sans adjectif). Such signifie "un tel / une telle / de tels".

  • Structure : Sujet + verbe + such + (adjectif) + nom + that + conséquence
  • Avec nom seul : It was such a storm that trees fell down.
    → (C'était une telle tempête que des arbres sont tombés.)
  • Avec adjectif + nom : She is such a talented singer that everyone admires her.
    → (Elle est une chanteuse tellement talentueuse que tout le monde l'admire.)
  • Pour les noms indénombrables ou pluriels : on omet "a/an".
    It was such bad weather that the flight was cancelled.
    (C'était un temps tellement mauvais que le vol a été annulé.)

⚖️ La différence essentielle : So vs Such

So + adjectif/adverbeSuch + (adjectif) + nom
🔹 So accompagne un adjectif (ou adverbe)
so interesting (si intéressant)
so quickly (si rapidement)		 🔸 Such accompagne un nom (parfois précédé d’un adjectif)
such an interesting book (un livre tellement intéressant)
such kindness (une telle gentillesse)
🎯 Pas de nom après l’adjectif :
The coffee is so hot that...		 🎯 Nom obligatoire (ou groupe nominal) :
It’s such hot coffee that...
📝 Moyen mnémotechnique : "So" + adjectif (pas de nom) ; "Such" + nom (un tel NOM). Pensez à "SO + Adjective / SUCH + Noun".

📌 Cas particuliers : so much / so many / so little / so few

Devant un nom (quantité), on utilise so much/so many (et non such) à cause de l’idée de quantité.

  • So much + indénombrable : He had so much work that he stayed late. (Il avait tellement de travail qu’il est resté tard.)
  • So many + dénombrable pluriel : There were so many people that we couldn’t move. (Il y avait tellement de monde qu'on ne pouvait pas bouger.)
  • So little / so few : She had so little time that she didn’t finish. (Elle avait si peu de temps qu’elle n’a pas fini.)

⚠️ Remarque : On n’utilise pas "such" devant much/many/little/few. C’est une exception fréquente au Bac/Probatoire.

💡 Exemples de compréhension

📖 Exemple 1 : Conséquence avec so + adjectif

Contexte : Pendant les examens, un élève raconte son état d’épuisement.

Phrase : "I was so tired that I couldn’t answer any question correctly."
🔍 Traduction et explication : "J'étais tellement fatigué que je n'ai pas pu répondre correctement à aucune question."
→ So + adjectif "tired" exprime l'intensité, that introduit la conséquence (incapacité à répondre).

💬 Variante possible : On peut remplacer "so" par "really" mais "so...that" insiste sur le lien cause → conséquence.

📖 Exemple 2 : Such + adjectif + nom + that

Sujet : Un discours sur la pollution dans une ville.

Phrase : "It was such a toxic smoke that people wore masks."
🔍 Analyse : Such a + adjectif (toxic) + nom (smoke) + that. On insiste sur le nom "smoke" qualifié par l'adjectif.
Traduction : "C'était une fumée tellement toxique que les gens portaient des masques."

🌍 À retenir pour l'écrit : Si le nom est singulier et commence par une consonne, on écrit "such a" ; pour une voyelle : "such an" (such an amazing performance).

📖 Exemple 3 : Différence entre so et such (piège classique)

Transformer la phrase avec "so" ou "such" correctement.

❌ Erreur fréquente : "It was so a beautiful day that we went to the beach." → incorrect.
✅ Correction : "It was such a beautiful day that we went to the beach." (car "day" est un nom).
✅ Alternative avec so : "The day was so beautiful that we went to the beach." (on utilise so + adjectif, pas de nom après beautiful).

🎯 Règle d’or : Si vous voyez "a/an" + nom → il faut "such". Si l’adjectif est seul → "so".

📖 Exemple 4 : so many / so much (quantité)

Situation : Après une fête, on décrit l’affluence.

"There were so many guests that the host ran out of food."
→ "Il y avait tellement d'invités que l'hôte a manqué de nourriture."
"She had so much energy that she danced all night."
→ "Elle avait tellement d'énergie qu'elle a dansé toute la nuit."
Remarque : Jamais "such many" ou "such much" !

⚠️ Pièges courants / À ne pas confondre

  • ❌ Confusion so / such : "He is so a nice person" → Faux ! Dire "He is such a nice person" ou "He is so nice".
  • ❌ Oublier "a/an" après such + adjectif + nom singulier : "such important decision" → manque "a" → "such an important decision".
  • ❌ Utiliser such devant much/many : "such many problems" est incorrect → "so many problems".
  • ❌ Inversion de l’ordre : "That so hot the soup was that..." → Structure anglaise stricte : "The soup was so hot that...".
  • 📌 Attention aux noms indénombrables : "such + adjectif + indénombrable" sans "a/an".
    Exemple correct : "It was such fresh water that we drank it all." (pas de 'a' car 'water' indénombrable).

  • 1. The movie was _______ boring that we left the cinema.
  • 2. It was _______ a wonderful surprise that she cried.
  • 3. There were _______ mistakes in his copy that he failed.
  • 4. He has _______ money that he can buy anything.
  • 5. She spoke _______ quickly that nobody understood.

2. such
3. so many
4. so much
5. so
