So / Such...that (Conséquence)

Anglais · 1ère CD · Exprimer une cause et sa conséquence

📌 Éléments clés à retenir

🔹 Structure avec "SO ... that"

On utilise so + adjectif/adverbe + that pour exprimer une conséquence.
En français : « tellement ... que ». L’adjectif ou l’adverbe se place directement après so.

  • Formule : So + adj/adv + that + proposition (conséquence)
  • Exemple : The movie was so boring that I fell asleep.
    ➡️ Le film était tellement ennuyeux que je me suis endormi.
  • Avec adverbe : He ran so fast that nobody could catch him.
    ➡️ Il a couru tellement vite que personne n’a pu l’attraper.
💡 Astuce : Après "so", on met uniquement un adjectif (so beautiful) ou un adverbe (so quickly). Jamais un nom seul !

🔹 Structure avec "SUCH ... that"

On utilise such + (adjectif) + nom + that pour exprimer la conséquence.
Such accompagne toujours un nom (parfois précédé d’un adjectif). En français : « un tel / une telle ... que ».

  • Formule : Such + (adj) + nom (singulier/pluriel) + that + conséquence
  • Exemple (nom singulier) : It was such a hot day that we stayed inside.
    ➡️ C’était une journée tellement chaude que nous sommes restés à l’intérieur.
  • Exemple (nom pluriel) : They are such nice people that everyone loves them.
    ➡️ Ce sont des gens tellement gentils que tout le monde les aime.
  • Avec "such" + adjectif + nom indénombrable : It’s such good news that I called my family.
💡 Attention à l’article : Si le nom est singulier et qu’il y a un adjectif, on écrit "such a/an" (such a beautiful car). Pour les pluriels ou indénombrables, pas d’article (such beautiful cars, such luck).

📈 Relation cause (intensité) → conséquence

💡 Exemples de compréhension

Exemple 1 : So + adjectif

Contexte : Le professeur explique un examen difficile. Les élèves n’ont pas réussi.

Phrase correcte : The test was so difficult that most students failed.
Traduction : Le test était tellement difficile que la plupart des élèves ont échoué.
Explication : "so difficult" (adjectif) + "that" + conséquence (échec).

💬 On pourrait aussi dire : "It was such a difficult test that..." (variante avec such).

Exemple 2 : So + adverbe (action)

Un match de football au Cameroun : l’équipe marque rapidement.

Phrase : The team played so well that they scored three goals in the first half.
Traduction : L’équipe a joué tellement bien que elle a marqué trois buts en première mi-temps.
Analyse : "so well" (adverbe de manière) → conséquence directe.

Exemple 3 : Such + nom (avec adjectif)

À l’occasion d’une fête traditionnelle, les habitants sont très accueillants.

Phrase : The villagers showed such great hospitality that we felt at home.
Traduction : Les villageois ont fait preuve d’une telle hospitalité que nous nous sommes sentis chez nous.
Remarque : "hospitality" est indénombrable, donc pas d’article. "such + adjectif + nom".

Exemple 4 : Such a/an + singulier

Un élève lit un livre passionnant.

Phrase : It was such an inspiring book that I read it twice.
Traduction : C’était un livre tellement inspirant que je l’ai lu deux fois.
Règle : "such a" (car book est singulier) + adjectif "inspiring".

⚠️ Pièges courants & différences à maîtriser

  • Erreur fréquente : Utiliser "so" devant un nom. ❌ "so a good student" → ✅ "such a good student" ou "so good a student" (littéraire).
  • Attention : Ne pas oublier "that" pour lier la conséquence. Dans le langage parlé, on peut l’omettre parfois, mais à l’écrit ou à l’examen, gardez-le.
  • Confusion so / such :
    • So + adjectif seul (so tall) / adverbe (so quickly).
    • Such + (adjectif) + nom (such a tall building).
  • Piège avec "many/much" : On dit "so many" (so many people that...) et "so much" (so much noise that...). Jamais "such many" ❌.
  • Ne pas inverser l’ordre : "that" exprime toujours la conséquence après l’intensité.

📊 Tableau récapitulatif : So vs Such

Structure Emploi Exemple
so + adj/adv + that Adjectif ou adverbe seuls The coffee was so hot that I burned my tongue.
such + (adj) + nom + that Nom (toujours), parfois avec adjectif It was such hot coffee that I burned my tongue.
so many / much + that Quantité (dénombrable / indénombrable) There were so many people that we couldn't enter.

🎯 À retenir : So insiste sur la qualité (adj/adv). Such insiste sur le nom.

✏️ Petit entraînement

Complétez avec so / such / such a

  • 1. The movie was ______ boring that I left the cinema.
  • 2. They are ______ kind children that everyone likes them.
  • 3. It was ______ beautiful day that we went to the beach.
  • 4. He drives ______ fast that it’s dangerous.
🔍 Voir les réponses

1. so (so boring)
2. such (such kind children)
3. such a (such a beautiful day)
4. so (so fast)

📘 N’oubliez pas : "such a" devant adjectif + nom singulier ; "so" devant adjectif seul.
