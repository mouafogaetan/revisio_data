🔹 Question 1 (2 pts)
Décrivez la structure générale d’un programme en langage C. Quels sont les éléments obligatoires et leur rôle ?

✅ Éléments attendus Un programme C se structure ainsi :

  • Bibliothèques (#include) : Permettent d'utiliser des fonctions prédéfinies (ex: stdio.h pour printf/scanf).
  • Fonction principale (int main() ou void main()) : Point d'entrée du programme. Toute instruction s'exécute à partir de main.
  • Déclarations de variables : Réservation d'espace mémoire (ex: int a;).
  • Instructions / Corps : Les opérations à réaliser (calculs, affichages, appels).
  • return (0) : Indique la fin correcte du programme (si main est de type int).

Barème : 0,5 pt par élément correctement cité et expliqué.

Exemple int : int age = 17; (on ne mettra pas 17.5).
Exemple float : (moyenne avec décimales).

🔹 Question 3 (2 pts)
Citez et expliquez brièvement les 3 principales structures de contrôle qui permettent de modifier le flux d’exécution d’un programme en C.

✅ Réponse Les 3 structures sont :

  1. Structure conditionnelle (if/else) : Exécute un bloc si une condition est vraie, sinon un autre bloc.
  2. Structure itérative (boucles) : Répète un bloc (for, while, do...while).
  3. Structure séquentielle : Les instructions sont exécutées les unes après les autres (par défaut).

✏️ Partie 2 : Application et traduction (8 points)

Traduisez les algorithmes en langage C.

🔹 Exercice 1 (3 pts)
On donne l’algorithme suivant :
Algorithme CalculSomme
Variables A, B, S : Entiers
Début
  Ecrire("Entrez le premier nombre : ")
  Lire(A)
  Ecrire("Entrez le deuxième nombre : ")
  Lire(B)
  S ← A + B
  Ecrire("La somme est : ", S)
Fin
Traduisez cet algorithme en un programme C complet (avec #include, main, scanf, printf).

✅ Code C correspondant

#include <stdio.h>

int main() {
    int A, B, S;
    printf("Entrez le premier nombre : ");
    scanf("%d", &A);
    printf("Entrez le deuxième nombre : ");
    scanf("%d", &B);
    S = A + B;
    printf("La somme est : %d\n", S);
    return 0;
}

🔹 On utilise %d pour les entiers et & pour l'adresse mémoire de la variable dans scanf.

🔹 Exercice 2 (2,5 pts)
Écrivez un programme en C qui demande l’âge d’un utilisateur et affiche "Mineur" si l’âge est inférieur à 18, sinon "Majeur".

✅ Code attendu

#include <stdio.h>

int main() {
    int age;
    printf("Entrez votre age : ");
    scanf("%d", &age);

    if (age < 18) {
        printf("Mineur\n");
    } else {
        printf("Majeur\n");
    }
    return 0;
}

🔹 La structure conditionnelle if...else est parfaitement adaptée.

🔹 Exercice 3 (2,5 pts)
Écrivez un programme C qui affiche les nombres de 1 à 10 à l’aide d’une boucle `for`.

✅ Programme

#include <stdio.h>

int main() {
    int i;
    for(i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}

🔹 La boucle `for` initialise i à 1, tourne tant que i <= 10, et incrémente i de 1 à chaque itération.

🔹 Problème : Gestion de notes (6 pts)
On souhaite écrire un programme permettant de :
1. Saisir les notes de 5 élèves (entiers entre 0 et 20).
2. Calculer et afficher la moyenne de la classe.
3. Afficher le nombre d’élèves ayant une note supérieure ou égale à 10.

Consignes : Utilisez un tableau pour stocker les notes, une boucle pour la saisie, et une fonction `calculMoyenne` si possible.

✅ Corrigé complet en C

#include <stdio.h>

float calculMoyenne(int notes[], int taille) {
    int somme = 0;
    for(int i = 0; i < taille; i++) {
        somme += notes[i];
    }
    return (float)somme / taille;
}
