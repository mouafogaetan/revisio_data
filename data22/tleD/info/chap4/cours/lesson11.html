📌 Définition et déclaration des tableaux

Qu’est-ce qu’un tableau ?

Un tableau est une structure de données qui permet de stocker plusieurs valeurs de même type sous un même nom. Chaque élément est accessible via son indice (position dans le tableau). En C, les indices commencent toujours à 0.

  • Tableau 1D (vecteur) : une seule ligne d’éléments.
  • Tableau 2D (matrice) : organisation en lignes et colonnes.
  • Taille fixe : la dimension du tableau doit être connue à la compilation (constante ou littéral).

Déclaration et initialisation

Syntaxe générale : type nom_tableau[taille];

  • Déclaration simple : int notes[5]; (tableau de 5 entiers, valeurs non initialisées).
  • Initialisation à la déclaration : int notes[5] = {10, 12, 15, 8, 14};
  • Initialisation partielle : int notes[5] = {10, 12}; → les autres valent 0.
  • Sans préciser la taille : int notes[] = {10, 12, 15}; → taille = 3.
  • Tableau 2D : int matrice[3][4]; (3 lignes, 4 colonnes).
    Initialisation : int mat[2][3] = {{1,2,3},{4,5,6}};
💡 Astuce : Pour connaître la taille d’un tableau (en nombre d’éléments) : int taille = sizeof(tableau) / sizeof(tableau[0]);

🔄 Parcours et manipulation des tableaux

Accès aux éléments

On accède à un élément en utilisant son indice entre crochets : nom[indice]. Les indices vont de 0 à taille-1. 

int notes[4] = {15, 12, 18, 10};
printf("%d", notes[0]);   // affiche 15
notes[2] = 20;            // modifie la 3e valeur (devient 20)

Parcours avec une boucle for

La boucle for est l’outil idéal pour parcourir un tableau séquentiellement. 

int notes[5] = {10, 12, 8, 15, 9};
int i;
for(i = 0; i < 5; i++) {
    printf("note[%d] = %d\n", i, notes[i]);
}
  • On utilise souvent une variable constante pour la taille : #define TAILLE 5 ou const int TAILLE = 5;
  • Parcours d’une matrice : double boucle (lignes, puis colonnes).

💡 Exemples de compréhension (algorithmique → C)

Exemple 1 : Calculer la somme des éléments d’un tableau

On dispose d’un tableau contenant les notes de 6 élèves : {14, 11, 16, 9, 13, 15}. Calculer la somme totale.

Algorithme (pseudo-code) :
somme ← 0
Pour i de 0 à 5 faire
    somme ← somme + tab[i]
Fin Pour
Programme en C : 
#include <stdio.h>
int main() {
    int notes[6] = {14, 11, 16, 9, 13, 15};
    int somme = 0, i;
    for(i = 0; i < 6; i++) {
        somme = somme + notes[i];
    }
    printf("Somme = %d\n", somme);
    return 0;
}

Résultat : 14+11+16+9+13+15 = 78

💬 Astuce : Pensez à initialiser la variable accumulateur (somme) à 0 avant la boucle.

Exemple 2 : Rechercher la plus grande valeur d’un tableau

Soit le tableau températures[7] = {23, 19, 21, 24, 22, 26, 20}. Trouver la température maximale.

Principe : On suppose que le premier élément est le maximum, puis on compare avec les suivants.
#include <stdio.h>
int main() {
    int temp[7] = {23, 19, 21, 24, 22, 26, 20};
    int max = temp[0];
    int i;
    for(i = 1; i < 7; i++) {
        if(temp[i] > max) {
            max = temp[i];
        }
    }
    printf("Température max = %d\n", max);
    return 0;
}

Sortie : Température max = 26

Exemple 3 : Parcours d’une matrice (tableau 2D)

Afficher tous les éléments d’une matrice 2x3.

#include <stdio.h>
int main() {
    int matrice[2][3] = {{1, 2, 3}, {4, 5, 6}};
    int i, j;
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            printf("%d ", matrice[i][j]);
        }
        printf("\n");
    }
    return 0;
}

Affichage :
1 2 3
4 5 6

⚠️ Pièges courants et bonnes pratiques

  • Erreur fréquente : Dépasser les indices du tableau (index out of bounds). Ex : accéder à tab[taille] alors que le dernier indice est taille-1.
  • Attention : Les tableaux ne se copient pas directement en C. int b = a; est interdit pour les tableaux. Il faut copier élément par élément.
  • Piège : Ne pas confondre l’indice et la valeur. L’indice commence à 0, donc tab[0] est le premier élément.
  • Bonnes pratiques : Utiliser une constante pour la taille du tableau (#define N 10) afin de faciliter les modifications.
  • À retenir : Un tableau 2D se parcourt avec deux boucles imbriquées (lignes et colonnes).
