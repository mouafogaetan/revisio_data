Leçon 5 : Équations différentielles du premier ordre

Terminale D · Mathématiques · Révision BAC (session 2023 D, Exercice 2)

📌 Éléments clés à retenir

1. Forme générale et solution homogène

Une équation différentielle linéaire du premier ordre homogène (sans second membre) s'écrit : $$ y' - a y = 0 \quad \text{ou} \quad y' = a y $$ où \(a\) est une constante réelle. La solution générale sur \(\mathbb{R}\) est : $$ y(x) = K e^{a x}, \quad K \in \mathbb{R} $$

  • On retrouve cette forme dans les phénomènes de croissance exponentielle (radioactivité, intérêts composés).
  • Si \(a > 0\) : croissance exponentielle ; si \(a < 0\) : décroissance exponentielle.
  • La constante \(K\) est déterminée par une condition initiale du type \(y(x_0) = y_0\).
  • Formule à retenir : $y(x) = y_0 e^{a(x - x_0)}$ (solution particulière avec condition initiale).

2. Équation avec second membre (cas constant)

Pour une équation complète du type : \( y' - a y = b \) où \(a\) et \(b\) sont des constantes réelles (\(a \neq 0\)) :

La solution générale est la somme de la solution générale de l'équation homogène et d'une solution particulière constante. La solution particulière constante est \( y_p = -\dfrac{b}{a} \).

Ainsi, la solution générale s'écrit : $$ y(x) = K e^{a x} - \frac{b}{a}, \quad K \in \mathbb{R} $$

  • Cette structure est fondamentale : Solution générale = Solution homogène + Solution particulière.
  • Pour vérifier qu’une fonction est solution, on la dérive et on remplace dans l’équation.
  • La condition initiale permet de calculer \(K\) de manière unique.

3. Méthode de résolution pas à pas

Pour résoudre \(y' - a y = b\) (avec condition initiale éventuelle) :

  • Étape 1 : Résoudre l’équation homogène \(y' - a y = 0\) → \(y_h(x) = C e^{ax}\).
  • Étape 2 : Trouver une solution particulière \(y_p\) : si le second membre est constant, \(y_p\) est constante = \(-\frac{b}{a}\).
  • Étape 3 : La solution générale : \(y(x) = y_h(x) + y_p = C e^{ax} - \frac{b}{a}\).
  • Étape 4 : Utiliser la condition initiale pour déterminer \(C\).

💡 Exemples de compréhension

Examen BAC 2023 D, Exercice 2 (inspiré)

On considère l’équation différentielle \( y' - 2y = 6 \). Déterminer la solution \(y\) vérifiant \(y(0) = -5\).

Résolution détaillée :
  1. Équation homogène : \(y' - 2y = 0\) → \(y_h(x) = C e^{2x}\).
  2. Second membre constant = 6, on cherche \(y_p\) constante : \(y_p = k\). Alors \(y_p' = 0\). En remplaçant : \(0 - 2k = 6\) → \(k = -3\). Donc \(y_p = -3\).
  3. Solution générale : \(y(x) = C e^{2x} - 3\).
  4. Condition initiale \(y(0) = -5\) : \(C e^{0} - 3 = -5\) → \(C - 3 = -5\) → \(C = -2\).
  5. Solution particulière : \(y(x) = -2 e^{2x} - 3\).
💡 Astuce : Vérifiez toujours que la solution particulière choisie satisfait l’équation. Pour un second membre constant, la solution particulière constante est rapide.

Modélisation : désintégration radioactive

La loi de décroissance radioactive s'écrit \( \frac{dN}{dt} = -\lambda N \), où \(\lambda > 0\) et \(N(t)\) est le nombre de noyaux à l'instant \(t\). À \(t=0\), \(N(0)=N_0\). Exprimer \(N(t)\).

Résolution : L’équation est \(N' + \lambda N = 0\) (forme \(N' - (-\lambda) N = 0\)). Solution générale : \(N(t) = K e^{-\lambda t}\).
Avec \(N(0)=N_0\) : \(K = N_0\). Donc \(N(t) = N_0 e^{-\lambda t}\). C’est la loi exponentielle décroissante.
💬 Interprétation : la demi-vie \(T_{1/2}\) vérifie \(N(T_{1/2}) = N_0/2\) → \(e^{-\lambda T_{1/2}} = 1/2\) → \(T_{1/2} = \frac{\ln 2}{\lambda}\).

Équation avec second membre et condition initiale

Résoudre \(y' + 3y = 9\) avec \(y(1) = 2\).

Correction : Équation homogène \(y' + 3y = 0\) → \(y_h(x) = C e^{-3x}\).
Second membre constant = 9, on pose \(y_p = k\) constante : \(0 + 3k = 9\) → \(k = 3\).
Solution générale : \(y(x) = C e^{-3x} + 3\).
Condition \(y(1)=2\) : \(C e^{-3} + 3 = 2\) → \(C e^{-3} = -1\) → \(C = -e^{3}\).
Donc \(y(x) = -e^{3} e^{-3x} + 3 = -e^{3(1-x)} + 3\).
📌 Pensez à bien gérer les coefficients : Si l’équation est sous la forme \(y' + ay = b\), la solution particulière constante est \(\frac{b}{a}\) (attention au signe !).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier la constante d'intégration \(K\) dans la solution homogène → solution incomplète.
  • Attention : Dans \(y' - ay = b\), la solution particulière constante est \(-\frac{b}{a}\) (et non \(\frac{b}{a}\)). Vérifier le signe !
  • Piège : Ne pas confondre la forme \(y' = ay + b\) (qui se ramène à \(y' - ay = b\)) avec une équation à variables séparables.
  • Conseil : Lorsqu’on utilise une condition initiale \(y(x_0)=y_0\), remplacer directement dans \(y(x) = C e^{a(x-x_0)} + y_p\) peut simplifier le calcul de \(C\).
  • À vérifier : Si le second membre n’est pas constant, la méthode change (variation de la constante, ou forme particulière adaptée). En Terminale D, on se limite souvent au cas constant.

Application : valeur moyenne d’une fonction et suites

D’après le programme Terminale D (Chapitre 2, Leçon 7), les équations différentielles du premier ordre sont souvent reliées à des problèmes de suites et de valeur moyenne. Par exemple, une suite définie par \(u_{n+1} = e^{a} u_n + b\) peut être liée à la discrétisation d’une équation différentielle.

La valeur moyenne d’une fonction solution de \(y' - a y = b\) sur un intervalle peut être calculée et permet d’étudier la convergence d’une suite associée.

  • Lorsqu’on étudie \( \frac{1}{T} \int_0^T y(t) dt \) avec \(y(t) = K e^{at} - \frac{b}{a}\), la limite éventuelle fait intervenir le comportement exponentiel.
  • Voir session 2023 D, Exercice 2 : exploitation d’une équation différentielle pour déterminer une suite géométrique.
📖 Révision croisée : Revoir le calcul de la somme des termes d’une suite géométrique (Chapitre 7) pour traiter les problèmes de modèles discrets issus d’EDO.

📝 Synthèse en un coup d’œil

  • Équation : \(y' - a y = b\) (avec \(a, b\) réels, \(a \neq 0\))
  • Solution générale : \(y(x) = C e^{a x} - \dfrac{b}{a}\)
  • Avec condition initiale \(y(x_0)=y_0\) : \(y(x) = \left(y_0 + \dfrac{b}{a}\right) e^{a(x-x_0)} - \dfrac{b}{a}\)
  • Cas particulier \(b=0\) : \(y(x) = C e^{a x}\) (croissance/décroissance exponentielle).
