Leçon 6 : Équations Différentielles du Second Ordre

Terminale D (Sciences Expérimentales) · Mathématiques · Bac 2025-2026

📌 Éléments clés à retenir

1. Forme générale et notations

Une équation différentielle linéaire du second ordre à coefficients constants s'écrit :

$$ y'' + a\,y' + b\,y = c $$

où $a$ et $b$ sont des constantes réelles et $c$ est une constante (second membre constant).
L'équation homogène associée est : $y'' + a y' + b y = 0$.

  • $y$ est la fonction inconnue, définie sur $\mathbb{R}$ ou un intervalle.
  • $y'$ sa dérivée première, $y''$ sa dérivée seconde.
  • On cherche des solutions $y(x)$.

2. Résolution de l'équation homogène $y'' + a y' + b y = 0$

On introduit l'équation caractéristique : $r^2 + a r + b = 0$. La nature des solutions dépend du discriminant $\Delta = a^2 - 4b$.

  • Si $\Delta > 0$ : deux racines réelles distinctes $r_1$ et $r_2$.
    $y_h(x) = A e^{r_1 x} + B e^{r_2 x}$.
  • Si $\Delta = 0$ : une racine double $r_0 = -\frac{a}{2}$.
    $y_h(x) = (A x + B) e^{r_0 x}$.
  • Si $\Delta < 0$ : deux racines complexes conjuguées $\alpha \pm i\beta$.
    $y_h(x) = e^{\alpha x}\big(A\cos(\beta x) + B\sin(\beta x)\big)$.

$A$ et $B$ sont des constantes réelles déterminées par les conditions initiales.

3. Solution particulière $y_p$ pour $y'' + a y' + b y = c$ (c constante)

On cherche une solution particulière constante $y_p = k$ (car second membre constant). En substituant $y_p = k$ dans l'équation : $0 + 0 + b k = c$.

  • Si $b \neq 0$ : $k = \dfrac{c}{b}$ ⇒ $y_p = \dfrac{c}{b}$.
  • Si $b = 0$ et $a \neq 0$ : L'équation devient $y'' + a y' = c$. On cherche alors $y_p$ de la forme $y_p = \lambda x$ (polynôme de degré 1). En dérivant : $y_p' = \lambda$, $y_p'' = 0$ ⇒ $0 + a\lambda = c$ ⇒ $\lambda = \frac{c}{a}$ ⇒ $y_p = \frac{c}{a}x$.
  • Si $b = 0$ et $a = 0$ : Équation $y'' = c$. On intègre deux fois : $y_p = \frac{c}{2}x^2$.
💡 Astuce : La solution générale de l'équation complète est $y(x) = y_h(x) + y_p(x)$.

💡 Exemples de compréhension

Examen 2024 D (extrait) : résolution complète

Résoudre l'équation différentielle : $y'' - 3y' + 2y = 4$.

Résolution étape par étape :
  1. Équation homogène : $y'' - 3y' + 2y = 0$.
    Équation caractéristique : $r^2 - 3r + 2 = 0$.
    Discriminant : $\Delta = 9 - 8 = 1 > 0$. Racines : $r_1 = 1$, $r_2 = 2$.
    Donc $y_h(x) = A e^{x} + B e^{2x}$.
  2. Solution particulière : second membre constant $=4$, $b=2 \neq 0$ ⇒ $y_p = \dfrac{4}{2} = 2$.
  3. Solution générale : $y(x) = A e^{x} + B e^{2x} + 2$.
📌 Vérification : remplacer $y_p = 2$ dans l'équation donne $0 - 0 + 2×2 = 4$ → correct.

Cas particulier : $y'' + 2y' = 6$

Résoudre $y'' + 2y' = 6$ avec conditions initiales $y(0)=1$ et $y'(0)=2$.

Démarche :
  • Équation homogène : $y''+2y'=0$ → $r^2+2r=0$ → $r(r+2)=0$ ⇒ $r_1=0$, $r_2=-2$.
    $y_h(x) = A + B e^{-2x}$.
  • Particulière : $b=0$, $a=2 \neq 0$ ⇒ on pose $y_p = \lambda x$.
    $y_p' = \lambda$, $y_p''=0$ ⇒ $0 + 2\lambda = 6$ ⇒ $\lambda = 3$. Donc $y_p = 3x$.
  • Solution générale : $y(x) = A + B e^{-2x} + 3x$.
  • Conditions initiales : $y(0)= A + B = 1$ ; $y'(x)= -2B e^{-2x} + 3$ ⇒ $y'(0)= -2B+3=2$ ⇒ $-2B = -1$ ⇒ $B=0.5$.
    Alors $A + 0.5 = 1$ ⇒ $A=0.5$. Solution : $y(x) = 0.5 + 0.5 e^{-2x} + 3x$.
💡 Lorsque $b=0$, ne jamais prendre $y_p$ constante (car elle serait solution de l'homogène) ; essayer $y_p = \lambda x$ ou $y_p = \lambda x^2$ selon le cas.

Second membre constant avec $\Delta < 0$

Résoudre $y'' + 2y' + 5y = 10$.

  • Équation caractéristique : $r^2 + 2r +5=0$, $\Delta = 4 - 20 = -16$. Racines : $r = -1 \pm 2i$.
  • $y_h(x) = e^{-x}\big(A\cos(2x) + B\sin(2x)\big)$.
  • Particulière : $b=5 \neq 0$ ⇒ $y_p = \frac{10}{5}=2$.
  • Solution générale : $y(x) = e^{-x}\big(A\cos(2x) + B\sin(2x)\big) + 2$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de traiter le cas particulier où $b=0$ pour la solution particulière. Dans ce cas, $y_p$ n'est pas constante mais un polynôme de degré 1 (ou 2 si $a=0$).
  • Attention : Ne pas confondre l'équation caractéristique $r^2+ar+b=0$ avec l'équation différentielle. Vérifier toujours le signe des coefficients.
  • Piège : Dans le cas $\Delta < 0$, les racines complexes sont $\alpha \pm i\beta$ ; la partie réelle $\alpha$ apparaît dans l'exponentielle $e^{\alpha x}$.
  • À surveiller : L'écriture de la solution générale pour une équation avec second membre : $y = y_h + y_p$ (ne jamais oublier $y_p$).
  • Conseil examen : Toujours commencer par écrire l'équation caractéristique et déterminer $\Delta$ avant de donner la forme de $y_h$.

📖 Synthèse : Marche à suivre

1️⃣ Écrire l'équation sous forme normalisée : $y'' + a y' + b y = c$.
2️⃣ Résoudre l'équation homogène : $r^2 + a r + b = 0$ → forme de $y_h$ selon $\Delta$.
3️⃣ Chercher une solution particulière $y_p$ (constante si $b\neq0$ ; sinon polynôme adapté).
4️⃣ Solution générale : $y(x) = y_h(x) + y_p(x)$.
5️⃣ Utiliser les conditions initiales (si données) pour déterminer $A$ et $B$.
