Leçon 32 : Acides α-aminés

Définition · Structure générale · Nomenclature | Chimie Terminale C & D (Programme 2025-2026)

📌 Éléments clés à retenir

1. Définition d’un acide α-aminé

Un acide α-aminé est un acide carboxylique qui possède une fonction amine ($-NH_2$) fixée sur le carbone portant le groupe carboxyle ($-COOH$). Ce carbone est appelé carbone α (alpha).

Formule générale : $H_2N - CHR - COOH$
  • R : représente une chaîne carbonée ou un atome d'hydrogène (chaîne latérale variable selon l'acide aminé).
  • Ils constituent les monomères de base des protéines.
  • Lien Bac : Reconnaître un acide α-aminé dans une molécule donnée.

2. Structure et groupes fonctionnels

Une molécule d'acide α-aminé comporte deux groupes fonctionnels essentiels sur le même carbone (le carbone α) :

  • Un groupe carboxyle ($-COOH$), responsable du caractère acide.
  • Un groupe amine primaire ($-NH_2$), responsable du caractère basique.

Cette double fonctionnalité leur confère un comportement amphotère (à la fois acide et basique) et la capacité à former des zwitterions.

3. Nomenclature des acides α-aminés

Deux systèmes de nomenclature sont utilisés pour nommer ces molécules :

  • Nomenclature systématique (IUPAC) : acide 2-amino... (le carbone fonctionnel porte le n°1).
    Exemple : $H_2N-CH_2-COOH$ → acide 2-aminoéthanoïque.
  • Nomenclature courante (biologique) : acide α-amino... ou noms triviaux.
    Exemple : $H_2N-CH_2-COOH$ → glycine (nom usuel).
💡 À retenir : La glycine est le seul acide α-aminé non chiral car R = H.

💡 Exemples de compréhension

Exemple 1 : Identifier un acide α-aminé

Molécule proposée : $CH_3 - CH(NH_2) - COOH$

Analyse : Cette molécule possède un groupe $-COOH$ et un groupe $-NH_2$ attachés au même atome de carbone (le carbone central). Le carbone porte également un groupement $R = CH_3$. C'est donc un acide α-aminé.

✔️ Nom IUPAC : acide 2-aminopropanoïque.
✔️ Nom courant : alanine.

Exemple 2 : Écrire la formule d'un acide α-aminé à partir de son nom

Nom : acide 2-amino-3-méthylbutanoïque.

Démarche :

  1. La chaîne principale est l'acide butanoïque (4 carbones dont le COOH).
  2. Le carbone 2 porte la fonction $-NH_2$.
  3. Le carbone 3 porte un groupement méthyle ($-CH_3$).
Formule développée : $H_2N - CH(CH_3) - CH_2 - COOH$ ? Non, attention : le carbone α est le n°2. La bonne formule est :
$HOOC - CH(NH_2) - CH(CH_3) - CH_3$ ?
Réécriture correcte : $H_2N-CH(COOH)-CH(CH_3)-CH_3$ ? Non, il faut que le COOH soit terminal.
Formule finale : $ (CH_3)_2CH - CH(NH_2) - COOH $

💡 Astuce : Placez toujours le groupe $COOH$ en bout de chaîne (carbone n°1) et le groupe $NH_2$ sur le carbone n°2 pour respecter la nomenclature.

⚠️ Pièges courants / À ne pas confondre

  • 🔴 Erreur fréquente : Confondre un acide α-aminé avec un acide β-aminé (amine sur le carbone β).
    ✅ Rappel : Dans un acide α-aminé, la fonction $-NH_2$ est sur le carbone directement lié au $-COOH$.
  • 🔴 Attention : Oublier que la glycine n'a pas de carbone asymétrique. Elle est donc achirale et ne présente pas d'énantiomères.
  • 🔴 Ne pas confondre : La nomenclature systématique (acide 2-amino...) et la nomenclature usuelle (alanine, valine, etc.) sont toutes deux exigibles au baccalauréat.
