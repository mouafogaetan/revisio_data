Leçon 30 : Nomenclature des amines

Chimie · Terminale C & D · Programme Officiel (2025-2026) | Baccalauréat

📌 Nomenclatures officielles (UICPA)

1. Nomenclature substitutive (préfixe "amino")

La nomenclature substitutive traite l'amine comme un dérivé du composé parent (alcane, alcène, etc.). Le groupe $-NH_2$ est indiqué par le préfixe "amino" placé devant le nom de la chaîne carbonée principale.

  • On numérise la chaîne la plus longue pour donner l'indice le plus bas possible au carbone portant le groupe amino.
  • Exemple : $CH_3-CH_2-CH_2-NH_2$ → propan-1-amine (ou 1-aminopropane).
  • Pour les amines secondaires ou tertiaires, on utilise le préfixe "N" pour indiquer les substituants sur l'azote.
  • Formule à retenir : Alcane + suffixe "amine" ou "amino" + alcane.

2. Nomenclature radico-fonctionnelle (alkylamine)

Cette nomenclature, plus ancienne mais encore courante, nomme l'amine comme un dérivé de l'ammoniac ($NH_3$) dans lequel un ou plusieurs atomes d'hydrogène sont remplacés par des groupements alkyles.

  • Amine primaire : alkylamine. Ex : $CH_3NH_2$ → méthylamine.
  • Amine secondaire : dialkylamine. Ex : $(CH_3)_2NH$ → diméthylamine.
  • Amine tertiaire : trialkylamine. Ex : $(CH_3)_3N$ → triméthylamine.
  • Règle alphabétique : on cite les substituants dans l'ordre alphabétique (éthyl, méthyl, propyl...).
💡 Astuce : Pour les amines complexes, la nomenclature substitutive (avec "amino") est préférée. Pour les amines simples, les deux sont acceptées.

3. Cas particuliers : amines aromatiques et sels d'ammonium

Les amines dérivées du benzène ont des noms spécifiques. Les sels d'ammonium quaternaire suivent également des règles précises.

  • $C_6H_5-NH_2$ → aniline (nom usuel conservé).
  • $C_6H_5-NH-CH_3$ → N-méthylaniline (nomenclature substitutive).
  • Ion ammonium quaternaire : $[R_4N]^+$ → on nomme les substituants suivis de "ammonium". Ex : $[(CH_3)_4N]^+$ → tétraméthylammonium.
  • Pour les sels d'amine : $RNH_3^+Cl^-$ → chlorure d'alkylammonium. Ex : $CH_3NH_3^+Cl^-$ → chlorure de méthylammonium.

💡 Exemples commentés (Baccalauréat)

Exemple 1 : Nommer l'amine suivante

Molécule : $CH_3 - CH(NH_2) - CH_2 - CH_3$

Résolution :
- Chaîne carbonée la plus longue : 4 carbones (butane).
- Fonction amine ($-NH_2$) sur le carbone n°2.
- Nomenclature substitutive : butan-2-amine (ou 2-aminobutane).
- Nomenclature radico-fonctionnelle : c'est une amine secondaire ? Non, c'est primaire (un seul groupe alkyle lié à N). Le groupe alkyle est un butyle secondaire → sec-butylamine.

💬 Astuce : Repérez toujours la chaîne la plus longue portant la fonction $-NH_2$ pour la nomenclature substitutive.

Exemple 2 : Nommer une amine secondaire symétrique

Molécule : $(CH_3-CH_2)_2NH$

Nomenclature radico-fonctionnelle : deux groupements éthyle → diéthylamine.
Nomenclature substitutive (UICPA) : éthanamine avec un substituant N-éthyle → N-éthyléthanamine.
(On prend l'éthanamine comme base et on indique que l'hydrogène de l'azote est remplacé par un éthyle).

Exemple 3 : Amine tertiaire mixte

Molécule : $CH_3 - N(CH_3) - CH_2 - CH_3$

Nom UICPA : N-éthyl-N-méthylméthanamine.
Explication : La chaîne la plus longue liée à N est méthane (1C), donc "méthanamine". Les deux autres substituants (méthyle et éthyle) sont placés devant avec le préfixe N- (ordre alphabétique : éthyl puis méthyl).
Nom radico-fonctionnel : éthyldiméthylamine (car éthyl + deux méthyles sur N).

📚 Lien Bac : Les exercices de nomenclature d'amines sont fréquents en début d'année (QCM ou questions ouvertes).

Représentation schématique des familles d'amines

Différence entre amine primaire, secondaire, tertiaire et ammonium quaternaire.

À retenir : Le nombre de groupements alkyles liés à l'atome d'azote détermine la classe de l'amine (primaire, secondaire, tertiaire). Le cation ammonium quaternaire porte une charge positive permanente.

⚠️ Erreurs fréquentes (Bac & examens)

  • ❌ Confondre classe de l'amine et fonction : Une amine tertiaire n'a pas d'hydrogène sur l'azote, elle ne peut pas former de liaison hydrogène aussi forte qu'une amine primaire.
  • ❌ Oublier le préfixe "N" : Dans la nomenclature substitutive, pour les amines secondaires ou tertiaires, il faut absolument noter les substituants sur l'azote avec "N-". Ex : $CH_3-NH-CH_3$ est N-méthylméthanamine, pas "méthanamine" tout court.
  • ❌ Mauvais ordre alphabétique : En nomenclature radico-fonctionnelle, on range les alkyles par ordre alphabétique (éthyl, méthyl, propyl...).
  • ❌ Ne pas numéroter correctement la chaîne : En nomenclature substitutive, on doit donner le numéro le plus bas possible au carbone portant l'amine (ou "amino").

🎯 Application type (sujet de Bac)

Question extraite d'un sujet 2024

Énoncé : Donner le nom (selon les règles UICPA) de l'amine de formule développée : $$CH_3 - CH_2 - CH(CH_3) - NH - CH_2 - CH_3$$

Correction détaillée :
1. Identifier la chaîne la plus longue liée à l'azote : on a un groupe $CH_3-CH_2-CH(CH_3)-$ (4 carbones, avec un méthyle sur le carbone 2) et un groupe $CH_2-CH_3$ (2 carbones).
2. La chaîne principale est celle qui contient le plus de carbones : butane (4C). La fonction amine secondaire, donc on doit nommer le substituant sur l'azote.
3. Nomenclature : La chaîne butane porte la fonction amine ? Non, car l'azote n'est pas dans la chaîne principale. On choisit le plus long alkyle lié à N comme base : 2-méthylbutane ? Attention, le groupe principal est "butanamine" mais l'azote n'est pas sur le butane. En fait, on prend l'amine comme fonction prioritaire : on considère la molécule comme une amine secondaire où les deux substituants sont "but-2-yle" (car le carbone portant la liaison avec N est le n°2 du butane) et "éthyle".
4. Nom UICPA : N-éthylbutan-2-amine.
Vérification : la chaîne la plus longue liée à N est le butane (4C) avec la fonction amine sur le carbone 2, et un groupement éthyle sur l'azote (indiqué par N-éthyl).
Nom radico-fonctionnel : éthyl(sec-butyl)amine (ou éthylbutan-2-ylamine).

⚡ Méthode : Pour une amine secondaire ou tertiaire, identifiez d'abord la chaîne carbonée la plus longue liée à N. C'est elle qui donne le nom de base (suffixe -amine). Les autres chaînes sont des substituants sur N avec le préfixe "N-".
