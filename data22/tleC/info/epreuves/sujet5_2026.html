➜ Questions de cours et QCM sur l’environnement de développement et la compilation

📝 Exercice 1 : Définitions et IDE (4 pts)
1. Donnez la définition d’un Environnement de Développement Intégré (IDE) et citez deux exemples d’IDE utilisés pour programmer en langage C. (2 pts)
2. Expliquez en une phrase le rôle du compilateur dans le processus de traduction d’un programme C. (1 pt)
3. Vrai ou Faux ? (1 pt)
  a) Un IDE intègre uniquement un éditeur de texte et ne gère pas l’exécution.
  b) Le fichier source en C a généralement l’extension .exe avant compilation.

✅ Corrigé

1. Un IDE (Integrated Development Environment) est un logiciel regroupant des outils pour développer : éditeur de code, compilateur, débogueur. Exemples : Code::Blocks, Dev-C++, Visual Studio Code (avec extensions).
2. Le compilateur traduit le code source (fichier .c) en langage machine (fichier objet .obj ou exécutable).
3. a) Faux — un IDE intègre plusieurs outils (compilation, débogage). b) Faux — le fichier source en C est .c ; le fichier exécutable est généré après compilation.

💡 Exercice 2 : QCM sur la compilation (4 pts)
Choisissez la ou les bonne(s) réponse(s) :
1. La compilation d’un programme C permet de :
A. Vérifier les erreurs logiques
B. Transformer le code source en code objet
C. Exécuter directement le programme sans étapes
D. Générer un fichier avec extension .c
2. Parmi ces logiciels, lesquels sont des environnements de développement ?
A. Turbo C++
B. Microsoft Word
C. Visual Studio Code
D. Code::Blocks
3. La phase de "link" (édition de liens) :
A. Combine les fichiers objets en un exécutable
B. Modifie la syntaxe du code
C. Supprime les commentaires du code source
D. Crée des bibliothèques dynamiques
4. Lequel de ces éléments n’est PAS présent dans un IDE typique ?
A. Compilateur
B. Antivirus intégré
C. Débogueur
D. Coloration syntaxique

✅ Réponses QCM
1. B (Transformer le code source en code objet).
2. A, C, D — Turbo C++, VS Code, Code::Blocks sont des IDE. Word est un traitement de texte.
3. A (Linkeur combine les fichiers objets).
4. B (Un IDE n’inclut pas d’antivirus par défaut).
