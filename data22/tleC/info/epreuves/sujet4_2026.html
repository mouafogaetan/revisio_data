❓ 1. En langage C, quelle est la sortie de printf("%d", 7 / 2); ?
  • A) 3.5
  • B) 3
  • C) 4
  • D) Erreur de compilation
✅ Réponse B (3)
En C, la division de deux entiers donne un résultat entier (troncature). 7/2 = 3.
❓ 2. Laquelle de ces instructions est une boucle conditionnelle en C ?
  • A) for(int i=0;i<10;i++)
  • B) do{ ... } while(x>0);
  • C) if(age>=18)
  • D) switch(choix)
✅ Réponse B
do...while est une boucle qui s'exécute au moins une fois. Les boucles conditionnelles incluent while, do...while et for.
❓ 3. Quelle bibliothèque standard est indispensable pour utiliser printf() et scanf() ?
  • A) <conio.h>
  • B) <math.h>
  • C) <stdio.h>
  • D) <stdlib.h>
✅ Réponse C (<stdio.h>)
Standard Input Output header.
❓ 4. Quelle structure de contrôle permet de choisir entre plusieurs blocs selon la valeur d'une expression ?
  • A) if-else if
  • B) switch-case
  • C) boucle for

✅ Réponse D
if-else if et switch-case permettent la sélection multiple.

📝 Exercice 1 : Manipulation de tableaux (6 pts)

On dispose d’un tableau d’entiers nommé notes contenant les scores de 5 élèves : [12, 15, 8, 19, 10].

  1. Écrire un algorithme (ou pseudo-code) qui calcule et affiche la moyenne des notes.
  2. Traduire cet algorithme en langage C (écrire un programme complet avec main).
  3. Quelle sera la valeur de notes[3] ? Justifier.
Correction détaillée
// 1. Algorithme :
// Variable somme, moyenne, i
// Initialiser somme = 0
// Pour i de 0 à 4 : somme = somme + notes[i]
// moyenne = somme / 5
// Afficher moyenne

// 2. Programme C complet :
#include <stdio.h>
int main() {
int notes[5] = {12, 15, 8, 19, 10};
int somme = 0;
for(int i=0; i<5; i++) { somme += notes[i]; }
float moyenne = (float)somme / 5;
printf("Moyenne : %.2f\n", moyenne);
return 0;
}
// 3. notes[3] vaut 19 (car l'indice 3 correspond à la 4ème case, le tableau commence à l'indice 0).

📝 Exercice 2 : Analyse et correction (4 pts)

On donne l’extrait de code suivant :

int x = 5;
while(x >= 0){
   if(x % 2 == 0){
      printf("%d ", x);
   }
   x--;
}
  1. Qu’affiche ce programme ? Expliquer le rôle de la condition x % 2 == 0.
  2. Modifier la boucle pour afficher uniquement les nombres impairs.
✅ Solution
1. Affichage : 4 2 0 (les nombres pairs décroissants de 5 à 0). La condition x % 2 == 0 teste si le reste de la division par 2 est nul (pair).
2. Pour les impairs, il faut inverser : if(x % 2 != 0) ou if(x % 2 == 1). Le programme afficherait alors 5 3 1.

🖥️ Exercice 3 : Environnement de développement (2 pts)

Un élève a écrit un programme C parfait, mais en compilant avec GCC, il obtient une erreur "fatal error: stdio.h: No such file or directory".

Question : Quelle est la cause probable de cette erreur ? Comment y remédier ?

🔧 Diagnostic
Cause : L'environnement ne trouve pas les headers standards (bibliothèques). Soit l'installation du compilateur est incomplète, soit l'IDE n'est pas configuré.
Remédiation : Installer correctement GCC via MinGW (sous Windows) ou build-essential (Linux). Sous Code::Blocks, vérifier les paramètres du compilateur.
