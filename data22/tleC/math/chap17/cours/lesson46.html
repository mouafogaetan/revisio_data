📊 Leçon 46 : Problèmes avec ajustement linéaire

Mathématiques · Terminale C · Préparation Bac (Séries statistiques doubles)

📌 Éléments clés à retenir

🔹 1. Objectif de l’ajustement linéaire

Lorsqu’on étudie deux variables statistiques $x$ et $y$ (par exemple : année et chiffre d’affaires), on cherche souvent à modéliser la relation par une droite appelée droite de régression. Cette droite permet de prévoir des valeurs de $y$ pour des $x$ donnés (extrapolation ou interpolation). Deux méthodes sont au programme : la droite de Mayer (méthode des points moyens) et la droite des moindres carrés (plus précise).

  • Nuage de points : représentation des couples $(x_i ; y_i)$ dans un repère.
  • Droite d’ajustement : $y = ax + b$ qui résume la tendance.
  • Coefficient de corrélation linéaire $r$ : mesure la qualité de l’ajustement ($-1 \le r \le 1$). Si $|r|$ proche de 1, la corrélation est forte.

🔹 2. Méthode de Mayer (points moyens)

Technique simple pour tracer une droite “à la main” :

  1. Ordonner les données selon $x$ croissant.
  2. Partager la série en deux groupes de même effectif (ou presque).
  3. Calculer le point moyen de chaque groupe : $G_1(\overline{x_1} ; \overline{y_1})$ et $G_2(\overline{x_2} ; \overline{y_2})$.
  4. La droite de Mayer est la droite $(G_1G_2)$.

Équation : $y = a_M x + b_M$ où $a_M = \dfrac{\overline{y_2} - \overline{y_1}}{\overline{x_2} - \overline{x_1}}$

⚠️ Avantage : rapide, résiste aux valeurs aberrantes. Inconvénient : moins précise que la méthode des moindres carrés.

🔹 3. Méthode des moindres carrés (droite de régression)

Elle minimise la somme des carrés des résidus (écarts verticaux). On obtient la “meilleure” droite au sens mathématique.

  • Covariance : $\operatorname{cov}(x,y) = \dfrac{1}{n}\sum_{i=1}^{n} (x_i - \overline{x})(y_i - \overline{y}) = \overline{xy} - \overline{x}\,\overline{y}$.
  • Variance de $x$ : $V(x) = \dfrac{1}{n}\sum (x_i - \overline{x})^2 = \overline{x^2} - (\overline{x})^2$.
  • Pente : $a = \dfrac{\operatorname{cov}(x,y)}{V(x)}$.
  • Ordonnée à l’origine : $b = \overline{y} - a\,\overline{x}$.
  • Coefficient de corrélation : $r = \dfrac{\operatorname{cov}(x,y)}{\sigma_x \sigma_y}$ avec $\sigma_x = \sqrt{V(x)}$.
Droite de régression de $y$ en $x$ : $y = a x + b$ avec $a = \dfrac{\sum (x_i - \overline{x})(y_i - \overline{y})}{\sum (x_i - \overline{x})^2}$.

🔹 4. Interprétation et prévisions

Une fois la droite $y = ax + b$ obtenue :

  • Pour un nouvel $x$ (exemple : année suivante), on calcule $y_{\text{estimé}} = a x + b$.
  • Attention à la validité : l’extrapolation (prédire loin des données) est risquée si la tendance change.
  • Le signe de $a$ indique le sens de la relation : $a>0$ → corrélation positive ; $a<0$ → corrélation négative.

💡 Exemples de compréhension

📈 Exemple 1 : Ajustement par la méthode des moindres carrés (inspiré Bac 2023)

Le chiffre d'affaires $y$ (en millions de FCFA) d'une entreprise selon l'année $x$ (codée : $x=0$ pour 2020) est donné :
$x_i$ : 0, 1, 2, 3, 4
$y_i$ : 12, 15, 18, 22, 27.

1. Calcul des moyennes : $\overline{x} = \frac{0+1+2+3+4}{5}=2$ ; $\overline{y} = \frac{12+15+18+22+27}{5}= \frac{94}{5}=18.8$.
2. Covariance : $\operatorname{cov}(x,y) = \frac{1}{5}\sum x_i y_i - \overline{x}\,\overline{y}$.
$\sum x_i y_i = 0\cdot12 + 1\cdot15 + 2\cdot18 + 3\cdot22 + 4\cdot27 = 0+15+36+66+108 = 225$.
$\overline{xy}=225/5 = 45$ → $\operatorname{cov}=45 - (2 \times 18.8) = 45 - 37.6 = 7.4$.
3. Variance de $x$ : $V(x)= \frac{0^2+1^2+2^2+3^2+4^2}{5} - (2)^2 = \frac{30}{5} - 4 = 6-4=2$.
4. Pente : $a = \frac{7.4}{2} = 3.7$.
5. Ordonnée : $b = \overline{y} - a\overline{x} = 18.8 - 3.7\times 2 = 18.8 - 7.4 = 11.4$.
Donc droite : $\boxed{y = 3.7x + 11.4}$.
Prévision pour 2025 (x=5) : $y = 3.7\times 5 + 11.4 = 18.5 + 11.4 = 29.9$ millions FCFA.

💡 Astuce : Pour éviter les erreurs, construisez un tableau avec les colonnes $x_i$, $y_i$, $x_i y_i$, $x_i^2$. Vérifiez la cohérence de la pente avec le nuage.

📉 Exemple 2 : Méthode de Mayer pour une série avec effectifs

Une étude mesure la température $y$ (en °C) en fonction de l'altitude $x$ (en km). Relevés : (0, 25) ; (1, 20) ; (2, 16) ; (3, 12) ; (4, 9) ; (5, 7). Déterminer la droite de Mayer.

• Groupes : 3 premiers points $G_1$ : $(0,25), (1,20), (2,16)$ ; 3 derniers points $G_2$ : $(3,12), (4,9), (5,7)$.
$\overline{x_1} = \frac{0+1+2}{3}=1$ ; $\overline{y_1} = \frac{25+20+16}{3}= \frac{61}{3} \approx 20.33$.
$\overline{x_2} = \frac{3+4+5}{3}=4$ ; $\overline{y_2} = \frac{12+9+7}{3}= \frac{28}{3} \approx 9.33$.
Pente : $a_M = \frac{9.33 - 20.33}{4 - 1} = \frac{-11}{3} \approx -3.667$.
Ordonnée avec $G_1$ : $20.33 = -3.667 \times 1 + b \Rightarrow b = 24$.
Droite : $y = -3.667x + 24$.
Prévision à 6 km : $y \approx -22 + 24 = 2$°C.

💬 Interprétation : La pente négative confirme que la température diminue avec l'altitude (environ -3,7°C par km).

🏆 Problème de synthèse (type Bac Partie B)

🎯 Estimation d’une production agricole

Un agriculteur observe la production $y$ (en tonnes) de maïs en fonction de la quantité d’engrais $x$ (en kg/ha). Relevés : $(x_i ; y_i)$ : (10 ; 3.2), (15 ; 4.0), (20 ; 4.5), (25 ; 5.1), (30 ; 5.8), (35 ; 6.2).

1. Représenter le nuage de points (schéma mental).
2. Calculer la droite de régression de $y$ en $x$ par la méthode des moindres carrés (arrondir à $10^{-3}$).
3. Donner une estimation de la production pour un apport de 40 kg/ha. Cet ajustement est-il fiable ?
4. Déterminer le coefficient de corrélation $r$ et interpréter.

📖 Voir la correction détaillée

Moyennes : $\overline{x} = \frac{10+15+20+25+30+35}{6} = \frac{135}{6}=22.5$ ; $\overline{y} = \frac{3.2+4.0+4.5+5.1+5.8+6.2}{6} = \frac{28.8}{6}=4.8$.

Covariance : $\sum x_i y_i = 10\cdot3.2 + 15\cdot4 + 20\cdot4.5 + 25\cdot5.1 + 30\cdot5.8 + 35\cdot6.2 = 32+60+90+127.5+174+217 = 700.5$ ; $\overline{xy}=700.5/6 = 116.75$ ; $\operatorname{cov}=116.75 - (22.5)(4.8)=116.75-108=8.75$.

Variance de $x$ : $\overline{x^2} = (100+225+400+625+900+1225)/6 = 3475/6 \approx 579.167$ ; $V(x)=579.167 - (22.5)^2 = 579.167 - 506.25 = 72.917$.

Pente : $a = 8.75 / 72.917 \approx 0.120$ ; $b = 4.8 - 0.120\times 22.5 = 4.8 - 2.7 = 2.1$ → $y = 0.120x + 2.1$.

Prévision pour $x=40$ : $y=0.120\times40+2.1=4.8+2.1=6.9$ tonnes.

Coefficient $r$ : $\sigma_x = \sqrt{72.917}\approx 8.54$ ; $\sigma_y$ : $\overline{y^2} = (10.24+16+20.25+26.01+33.64+38.44)/6 = 144.58/6=24.0967$ ; $V(y)=24.0967 - (4.8)^2 = 24.0967-23.04=1.0567$ ; $\sigma_y\approx 1.028$ ; $r = 8.75/(8.54\times 1.028) \approx 8.75/8.777 \approx 0.997$. Très proche de 1, ajustement excellent.

🧠 Rappel examen : Dans les problèmes de synthèse, on combine souvent ajustement linéaire avec pourcentages, probabilités ou équations différentielles. Toujours vérifier l’unité des prévisions.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Utiliser la droite de régression pour des $x$ hors du domaine observé sans mentionner les limites de l’extrapolation.
  • Attention : Confondre la droite de régression de $y$ en $x$ avec celle de $x$ en $y$ (pentes différentes sauf corrélation parfaite).
  • Piège : Oublier de centrer les données pour le calcul de la covariance → appliquer la formule $\frac{1}{n}\sum x_i y_i - \overline{x}\,\overline{y}$.
  • À ne pas oublier : Un coefficient de corrélation proche de 0 n’implique pas forcément l’absence de lien (peut être non linéaire). L’ajustement linéaire n’est pertinent que si le nuage a une forme allongée.
  • Méthode de Mayer : Attention lorsque le nombre de points est impair, les groupes doivent avoir des effectifs équilibrés (sinon approximation).

📐 Formulaire récapitulatif (à savoir par cœur)

📌 Moyennes : $\displaystyle \overline{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$    $\displaystyle \overline{y} = \frac{1}{n}\sum_{i=1}^{n} y_i$

📌 Covariance : $\operatorname{cov}(x,y) = \frac{1}{n}\sum_{i=1}^{n} (x_i - \overline{x})(y_i - \overline{y}) = \overline{xy} - \overline{x}\,\overline{y}$

📌 Variance : $V(x) = \frac{1}{n}\sum (x_i - \overline{x})^2 = \overline{x^2} - (\overline{x})^2$

📌 Droite de régression (moindres carrés) : $y = ax + b$ avec $a = \dfrac{\operatorname{cov}(x,y)}{V(x)}$, $b = \overline{y} - a\overline{x}$

📌 Coefficient de corrélation : $r = \dfrac{\operatorname{cov}(x,y)}{\sigma_x \sigma_y}$ avec $\sigma_x = \sqrt{V(x)}$, $\sigma_y = \sqrt{V(y)}$

📌 Droite de Mayer : $a_M = \dfrac{\overline{y_2} - \overline{y_1}}{\overline{x_2} - \overline{x_1}}$, puis $b_M$ par substitution.
