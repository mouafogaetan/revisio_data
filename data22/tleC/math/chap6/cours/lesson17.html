Leçon 17 : Racines n‑ièmes d’un nombre complexe

Terminale C · Mathématiques · Approche géométrique et algébrique

📌 Éléments clés à retenir

1. Racine n‑ième d’un complexe – Définition

Soit \(n \in \mathbb{N}^*\) et \(Z \in \mathbb{C}\). Un nombre complexe \(z\) est une racine n‑ième de \(Z\) si : \[ z^n = Z. \] Dans \(\mathbb{C}\), tout nombre complexe non nul possède exactement \(n\) racines n‑ièmes distinctes.

  • Si \(Z = 0\), l’unique racine n‑ième est \(z = 0\).
  • Les racines n‑ièmes se trouvent grâce à la forme exponentielle : \(Z = r e^{i\theta}\)
  • Formule générale : \[ z_k = r^{1/n} \, e^{\,i\left(\frac{\theta + 2k\pi}{n}\right)}, \quad k = 0,1,\dots, n-1 \]
  • Le module de chaque racine est \(\sqrt[n]{r}\) (racine réelle positive).

2. Racines n‑ièmes de l’unité

Cas particulier fondamental : \(Z = 1\). Les racines n‑ièmes de l’unité sont les complexes : \[ \omega_k = e^{i\frac{2k\pi}{n}} = \cos\left(\frac{2k\pi}{n}\right) + i\sin\left(\frac{2k\pi}{n}\right), \quad k=0,1,\dots,n-1. \]

  • Elles sont situées sur le cercle trigonométrique (module 1) et forment un polygone régulier à \(n\) côtés.
  • Propriété : \(\omega_k^n = 1\) et \(1 + \omega_1 + \omega_2 + \dots + \omega_{n-1} = 0\).
  • Notation : souvent \(\omega = e^{2i\pi/n}\) et les racines sont \(1, \omega, \omega^2, \dots, \omega^{n-1}\).
Figure : Racines 5èmes de l’unité (polygone régulier).

3. Représentation géométrique et interprétation

Si \(Z = r e^{i\theta}\), ses \(n\) racines n‑ièmes sont les sommets d’un polygone régulier à \(n\) côtés, centré à l’origine, de rayon \(\sqrt[n]{r}\). Les arguments sont en progression arithmétique de raison \(\frac{2\pi}{n}\).

  • Les points \(M_k\) d’affixes \(z_k\) sont régulièrement espacés sur le cercle de centre O et de rayon \(r^{1/n}\).
  • La somme des racines n‑ièmes de \(Z\) est nulle (si \(Z \neq 0\)).
  • Utile pour résoudre \(z^n = Z\) : on passe d’abord à la forme exponentielle.

💡 Exemples de compréhension

Exemple 1 : Racines cubiques de \(8i\)

Déterminer les racines cubiques du nombre complexe \(Z = 8i\) (c’est-à-dire résoudre \(z^3 = 8i\)).

Résolution :
1. Écriture exponentielle : \(8i = 8 e^{i\pi/2}\) (car \(\cos\frac{\pi}{2}=0, \sin\frac{\pi}{2}=1\)).
2. Module : \(r=8\), \(\sqrt[3]{8} = 2\). Argument : \(\theta = \frac{\pi}{2}\).
3. Racines : \(z_k = 2 \cdot e^{\,i\left(\frac{\pi/2 + 2k\pi}{3}\right)}\), \(k=0,1,2\).
- \(k=0\) : \(z_0 = 2 e^{i\pi/6} = 2\left(\cos\frac{\pi}{6}+i\sin\frac{\pi}{6}\right) = \sqrt{3}+i\).
- \(k=1\) : \(z_1 = 2 e^{i5\pi/6} = 2\left(-\frac{\sqrt{3}}{2} + i\frac12\right) = -\sqrt{3}+i\).
- \(k=2\) : \(z_2 = 2 e^{i9\pi/6}=2 e^{i3\pi/2} = -2i\).
Vérification : \((-2i)^3 = -8i^3 = -8(-i)=8i\).

💬 Astuce : Toujours mettre \(Z\) sous forme \(r e^{i\theta}\) avec \(\theta\) dans \([0,2\pi[\) pour éviter les erreurs.

Exemple 2 : Racines quatrièmes de \(-1\)

Résoudre dans \(\mathbb{C}\) l’équation \(z^4 = -1\).

Démarche : \(-1 = 1 \times e^{i\pi} = e^{i\pi}\).
Module des racines : \(1^{1/4}=1\). Arguments : \(\displaystyle \frac{\pi + 2k\pi}{4}\) pour \(k=0,1,2,3\).
\(z_0 = e^{i\pi/4} = \frac{\sqrt{2}}{2} + i\frac{\sqrt{2}}{2}\)
\(z_1 = e^{i3\pi/4} = -\frac{\sqrt{2}}{2} + i\frac{\sqrt{2}}{2}\)
\(z_2 = e^{i5\pi/4} = -\frac{\sqrt{2}}{2} - i\frac{\sqrt{2}}{2}\)
\(z_3 = e^{i7\pi/4} = \frac{\sqrt{2}}{2} - i\frac{\sqrt{2}}{2}\)
Ces 4 points forment un carré inscrit dans le cercle unité.

💡 Penser à la formule de Moivre pour vérifier : \((e^{i\pi/4})^4 = e^{i\pi} = -1\).

Exemple 3 : Utilisation des racines n‑ièmes pour factoriser

Factoriser \(z^4 - 1\) dans \(\mathbb{C}\) puis dans \(\mathbb{R}\).

Les racines quatrièmes de l’unité : \(1, i, -1, -i\).
Dans \(\mathbb{C}\) : \(z^4-1 = (z-1)(z-i)(z+1)(z+i)\).
En regroupant les facteurs conjugués : \((z-1)(z+1) = z^2-1\) et \((z-i)(z+i)=z^2+1\).
D’où dans \(\mathbb{R}\) : \(z^4-1 = (z^2-1)(z^2+1)\).

Racines 4èmes de l'unité : sommets d’un carré.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier que le module d’une racine n‑ième est la racine réelle positive \(\sqrt[n]{r}\) et non pas \(r/n\). Exemple : racine cubique de 8 est 2, pas \(8/3\).
  • Attention : Confondre racines n‑ièmes dans \(\mathbb{R}\) et \(\mathbb{C}\). Dans \(\mathbb{R}\), \(x^3 = 8\) a une solution unique (2). Dans \(\mathbb{C}\), on a 3 solutions.
  • Piège géométrique : Ne pas diviser correctement l’argument : \( \frac{\theta + 2k\pi}{n} \) et non \( \frac{\theta}{n} + 2k\pi\).
  • Ne pas oublier : Pour un nombre complexe donné, il y a exactement \(n\) racines distinctes (sauf \(Z=0\)).

📐 Formulaire pratique

Synthèse des formules

  • Forme exponentielle : \(Z = r e^{i\theta}\) avec \(r=|Z|\), \(\theta = \arg Z\)
  • Racines n‑ièmes : \(\displaystyle z_k = \sqrt[n]{r} \cdot e^{i\frac{\theta + 2k\pi}{n}}\), \(k=0,\dots,n-1\)
  • Racines de l’unité : \(\displaystyle U_n = \left\{ e^{i\frac{2k\pi}{n}} \mid k=0,\dots,n-1\right\}\)
  • Propriété de somme : \(\displaystyle \sum_{k=0}^{n-1} z_k = 0\) (si \(Z\neq0\))
  • Lien polynôme : \(z^n - a^n = (z-a)(z^{n-1}+ a z^{n-2} + \dots + a^{n-1})\) (factorisation utilisant les racines n‑ièmes).

🏋️ Exercice d’entraînement (inspiré examen)

Résoudre \(z^6 = -64\)

Donner les solutions sous forme algébrique et trigonométrique.

Éléments de correction :
\(-64 = 64 e^{i\pi} = 2^6 e^{i\pi}\).
Module : \(\sqrt[6]{64} = 2\). Argument : \(\frac{\pi + 2k\pi}{6}\), \(k=0,1,2,3,4,5\).
Solutions : \(z_k = 2 e^{i(\pi+2k\pi)/6} = 2 e^{i\pi/6} e^{ik\pi/3}\).
Soit : \(k=0: 2e^{i\pi/6} = \sqrt{3}+i\) ; \(k=1: 2e^{i\pi/2}=2i\) ; \(k=2: 2e^{i5\pi/6}=-\sqrt{3}+i\) ;
\(k=3: 2e^{i7\pi/6}=-\sqrt{3}-i\) ; \(k=4: 2e^{i3\pi/2}=-2i\) ; \(k=5: 2e^{i11\pi/6}=\sqrt{3}-i\).
Ces 6 points forment un hexagone régulier de rayon 2.

💪 Pensez à contrôler que le produit des racines vaut \((-1)^{n-1} Z\) (théorème de Viète).
