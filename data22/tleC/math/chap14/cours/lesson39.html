📐 Leçon 39 : Étude de fonctions avec paramètre

Terminale C · Analyse · Révision examen 2025-2026

📌 Éléments clés à retenir

🎯 1. Qu’est-ce qu’un paramètre ?

Un paramètre (noté souvent \(m, k, \alpha, a\)) est une constante réelle qui n’est pas fixée une fois pour toutes. L’étude de la fonction dépend de la valeur de ce paramètre. On ne peut pas conclure universellement sans discuter selon les valeurs du paramètre.

  • Exemple : \(f(x) = \dfrac{x^2 + m}{x-1}\) avec \(m \in \mathbb{R}\).
  • L’étude (domaine, variations, extremums, signe) peut changer radicalement suivant que \(m > 0\), \(m = 0\) ou \(m < 0\).
  • 📌 Objectif : Déterminer le comportement de la fonction en fonction du paramètre.

🧠 2. Démarche générale pour une étude paramétrée

Suivre la trame classique de l'étude de fonction, mais en gardant le paramètre littéral. Chaque calcul (dérivée, signe, limites) doit être traité comme une expression contenant le paramètre.

  • 1. Domaine de définition : \(D_f\) (peut dépendre du paramètre si racine carrée, dénominateur, logarithme).
  • 2. Dérivée \(f'(x)\) : Calculer en fonction du paramètre puis factoriser au maximum.
  • 3. Signe de \(f'(x)\) : Étudier le signe des facteurs contenant le paramètre → discuter selon les valeurs critiques.
  • 4. Tableau de variations : Construire un tableau par cas (ex : si \(m < 2\), si \(m=2\), si \(m > 2\)).
  • 5. Éventuelles asymptotes ou branches infinies : paramètre peut influencer les limites en l’infini ou aux points singuliers.
💡 Astuce : Toujours factoriser l’expression de \(f'(x)\) pour faire apparaître un terme "paramètre dépendant" séparément.

⚙️ 3. Cas particuliers fréquents au baccalauréat

Dans les sujets récents (2022-2025), les paramètres apparaissent dans :

  • Fonctions rationnelles : \(f(x) = \dfrac{ax + b}{cx + d}\) avec paramètre au numérateur ou dénominateur.
  • Fonctions avec exponentielle : \(f(x) = e^{kx} - kx\) (discussion du signe de la dérivée selon \(k\)).
  • Fonctions avec logarithme : \(f(x) = \ln(x) + \frac{\alpha}{x}\), paramètre \(\alpha\) modifie la position de l’extremum.
  • Polynômes paramétrés : \(f(x) = x^3 + px + q\) (étude des racines, variations).

💡 Exemples de compréhension

📐 Exemple 1 : fonction rationnelle \(f(x) = \frac{x^2 + m}{x-1}\)

On considère \(f_m(x) = \dfrac{x^2 + m}{x-1}\) définie pour \(x \neq 1\), où \(m\) est un paramètre réel. Étudier les variations selon \(m\).

🔍 Résolution pas à pas :
1️⃣ Dérivée (quotient) : \(f'(x) = \dfrac{2x(x-1) - (x^2+m)}{(x-1)^2} = \dfrac{2x^2 - 2x - x^2 - m}{(x-1)^2} = \dfrac{x^2 - 2x - m}{(x-1)^2}\).
2️⃣ Le signe de \(f'(x)\) est celui du numérateur \(N(x)=x^2 - 2x - m\) car \((x-1)^2 > 0\).
3️⃣ Discriminant : \(\Delta = 4 + 4m = 4(1+m)\).
   ▪️ Si \(m < -1\) : \(\Delta < 0\) ⇒ \(N(x) > 0\) pour tout \(x\) ⇒ \(f\) strictement croissante sur chaque intervalle \(]-\infty,1[\) et \(]1,+\infty[\).
   ▪️ Si \(m = -1\) : \(\Delta = 0\) ⇒ \(N(x) = (x-1)^2 \ge 0\) et s’annule en \(x=1\) (valeur exclue) ⇒ \(f\) croissante.
   ▪️ Si \(m > -1\) : deux racines \(x_{1,2} = 1 \pm \sqrt{1+m}\). Tableau de signe : \(f'\) positive à l’extérieur des racines, négative entre. Variations adaptées.
💬 À retenir : La valeur \(m = -1\) est la frontière entre monotonicité et extremums locaux. Pensez toujours au signe du discriminant.

📈 Exemple 2 : fonction exponentielle \(g(x) = e^{kx} - kx\)

Soit \(g_k(x) = e^{kx} - kx\) définie sur \(\mathbb{R}\), \(k\) paramètre réel non nul. Déterminer la nature du point critique selon \(k\).

Étapes clés :
• Dérivée : \(g'(x) = k e^{kx} - k = k(e^{kx} - 1)\).
• Signe de \(g'\) :
   - Si \(k > 0\) : \(g'(x) = 0 \iff e^{kx}=1 \iff x=0\). Pour \(x<0\), \(e^{kx}<1\) ⇒ \(g'(x)<0\) ; pour \(x>0\), \(g'(x)>0\). Donc \(g\) décroît puis croît : minimum en \(x=0\).
   - Si \(k < 0\) : \(g'(x) = k(e^{kx}-1)\) ; attention \(k\) négatif change le sens. Pour \(x<0\), \(kx>0\) (car \(k<0\)) ⇒ \(e^{kx}>1\) ⇒ \(e^{kx}-1>0\) mais \(k<0\) ⇒ \(g'(x)<0\) ? À refaire proprement : étudier le signe. En réalité pour \(k<0\), \(g'(x) = 0 \iff x=0\) également. Pour \(x>0\), \(kx<0\) ⇒ \(e^{kx}<1\) ⇒ \(e^{kx}-1<0\) et \(k<0\) ⇒ produit \(>0\) ? On conclut : si \(k<0\), \(g\) croissante sur \(]-\infty,0[\) et décroissante sur \(]0,+\infty[\) ⇒ maximum en \(0\).
➡️ Conclusion : La nature de l’extremum dépend du signe de \(k\) : \(k>0\) → minimum ; \(k<0\) → maximum.
📌 Piège fréquent : ne pas oublier que multiplier par \(k\) (négatif) renverse les inégalités lorsqu’on étudie le signe de \(g'\).

🔁 Exemple 3 : fonction logarithme \(h(x) = \ln x + \dfrac{\alpha}{x}\)

\(h_\alpha(x) = \ln x + \frac{\alpha}{x}\) définie sur \(]0, +\infty[\), \(\alpha\) paramètre réel. Discuter l’existence d’un extremum.

Dérivée : \(h'(x) = \frac{1}{x} - \frac{\alpha}{x^2} = \frac{x - \alpha}{x^2}\).
Signe de \(h'(x)\) = signe de \(x - \alpha\) (car \(x^2>0\)).
• Si \(\alpha \le 0\) : \(x-\alpha > 0\) pour tout \(x>0\) ⇒ \(h' > 0\) ⇒ \(h\) strictement croissante (pas d’extremum local).
• Si \(\alpha > 0\) : \(h'(x) = 0 \iff x=\alpha\). Pour \(0 < x < \alpha\), \(h'<0\) ; pour \(x>\alpha\), \(h'>0\) ⇒ minimum global en \(x = \alpha\) valant \(h(\alpha)=\ln\alpha + 1\).
Résumé : paramètre seuil \(\alpha=0\) sépare deux comportements.

📊 Tableau de discussion type (baccalauréat)

Quand le paramètre influence le signe de la dérivée, on organise la réponse sous forme de tableau de discussion. Voici un modèle général utilisé dans les épreuves (session 2025 C, exercice 4).

Valeur du paramètre \(m\)Signe de \(f'(x)\)Variations de \(f\)
\(m < m_0\)\(f'(x) \ge 0\) (par exemple)\(f\) croissante
\(m = m_0\)\(f'(x) \ge 0\) avec annulation isolée\(f\) croissante (point d’inflexion)
\(m > m_0\)\(f'(x)\) change de signe → extremum\(f\) décroît puis croît (minimum)

✨ Les valeurs critiques du paramètre (\(m_0\)) proviennent souvent de l’annulation d’un discriminant, d’un facteur ou d’une condition d’existence.

⚠️ Pièges courants & astuces pour le jour J

  • Erreur fréquente 1 : Oublier que le paramètre peut rendre le dénominateur nul ou modifier le domaine de définition.
    ➜ Vérifier toujours les valeurs interdites en fonction du paramètre.
  • Erreur fréquente 2 : Manipuler une inégalité en multipliant par le paramètre sans connaître son signe.
    ➜ Pour étudier le signe de \(f'(x) = k \times g(x)\), discuter d’abord du signe de \(k\).
  • Erreur fréquente 3 : Donner une conclusion unique alors qu’il faut distinguer des intervalles de paramètre.
    ➜ Toujours structurer la réponse avec "Si ... alors ...".
  • Astuce gagnante : Pour les fonctions paramétrées, tracer sur GeoGebra en faisant varier un curseur (représentation mentale). Cela aide à anticiper les différents cas.
  • Lien avec l’examen : Sujet 2025 C, Partie A, Exercice 4 – une fonction avec paramètre \(\alpha\) : étude complète (limites, variations, position de courbe). Maîtriser la discussion est indispensable.

📌 Synthèse express

✔️ Une fonction paramétrée = une famille de courbes.
✔️ Les étapes : domaine, dérivée, signe de dérivée → discussion selon paramètre.
✔️ Ne jamais remplacer le paramètre par une valeur avant la fin de l’étude littérale.
✔️ Pour chaque cas, donner le tableau de variation complet.
✔️ Vérifier la cohérence avec les limites (elles aussi peuvent dépendre du paramètre).

🔥 Examen : les correcteurs attendent une discussion claire, des "si ... alors ..." et éventuellement une représentation graphique synthétique.
