📈 Leçon 40 : Étude de fonctions avec logarithme

Terminale C (Scientifique) – Mathématiques · Révision ciblée · Examen 2025-2026

📌 Éléments clés à retenir

1. Domaine de définition et limites de base

La fonction logarithme népérien \( \ln x \) est définie sur \( ]0; +\infty[ \). Toute fonction du type \( f(x) = \ln(u(x)) \) est définie lorsque \( u(x) > 0 \). Les limites fondamentales à connaître :

  • \(\displaystyle \lim_{x \to 0^{+}} \ln x = -\infty\)
  • \(\displaystyle \lim_{x \to +\infty} \ln x = +\infty\)
  • \(\displaystyle \lim_{x \to 0^{+}} x \ln x = 0\)   (croissances comparées)
  • \(\displaystyle \lim_{x \to +\infty} \frac{\ln x}{x} = 0\)   et   \(\displaystyle \lim_{x \to 0^{+}} x \ln x = 0\)
💡 Croissances comparées : En \(+\infty\), \(x^\alpha\) l’emporte sur \(\ln x\) ; en \(0^+\), \(\ln x\) l’emporte sur \(x^\alpha\) (pour \(\alpha > 0\)).

2. Dérivée et variations (cas type \(f(x)=x\ln x - x\))

Soit \(f(x) = x\ln x - x\) définie sur \(]0; +\infty[\).
Calcul de la dérivée : \(f'(x) = \ln x + 1 - 1 = \ln x\). D’où le signe de \(f'(x)\) est celui de \(\ln x\).

  • \(f'(x) < 0 \iff \ln x < 0 \iff 0 < x < 1\) → \(f\) strictement décroissante sur \(]0;1[\).
  • \(f'(x) = 0 \iff x = 1\) → minimum (ou maximum selon les cas).
  • \(f'(x) > 0 \iff x > 1\) → \(f\) strictement croissante sur \(]1; +\infty[\).

Pour une fonction générale avec logarithme, on applique les règles de dérivation : \(\frac{d}{dx}[\ln(u(x))] = \frac{u'(x)}{u(x)}\) et produit \( (u v)' = u'v + uv'\).

3. Convexité et point d’inflexion

La convexité s’étudie avec la dérivée seconde \(f''(x)\). Exemple pour \(f(x)=x\ln x - x\) : \(f'(x)=\ln x\) donc \(f''(x)=\frac{1}{x}\). Comme \(f''(x) > 0\) sur \(]0;+\infty[\), la fonction est convexe sur tout son domaine. Pas de point d’inflexion (car la convexité ne change pas de signe). Pour d’autres fonctions, \(f''(x)\) peut s’annuler et donner un point d’inflexion.

📐 Rappel : Si \(f''(x) > 0\) sur un intervalle, la courbe est convexe (en forme de cuvette) ; si \(f''(x) < 0\) elle est concave.

4. Étude complète d’une fonction avec logarithme (démarche type)

  • Étape 1 : Déterminer le domaine de définition \(D_f\).
  • Étape 2 : Calculer les limites aux bornes de \(D_f\) (asymptotes éventuelles).
  • Étape 3 : Dériver et étudier le signe de \(f'(x)\) pour les variations.
  • Étape 4 : Calculer la dérivée seconde pour la convexité et points d’inflexion.
  • Étape 5 : Dresser le tableau de variations complet (avec limites et extremums).
  • Étape 6 : Tracer l’allure de la courbe (points particuliers, tangentes).

💡 Exemples de compréhension

Exemple n°1 : Étude de \(f(x) = x\ln x - x\) (sujet type bac)

On considère la fonction \(f(x) = x\ln x - x\) pour \(x > 0\).

1. Limites : \(\lim_{x\to 0^{+}} f(x) = 0 - 0 = 0\) (car \(x\ln x \to 0\)). \(\lim_{x\to +\infty} f(x) = +\infty\) (car \(x\ln x\) domine \(-x\)).
2. Dérivée : \(f'(x) = \ln x + 1 - 1 = \ln x\). 3. Signe et variations : \(f'(x) < 0\) sur \(]0;1[\) → décroissante ; \(f'(x) > 0\) sur \(]1;+\infty[\) → croissante. Minimum en \(x=1\), \(f(1) = -1\). 4. Convexité : \(f''(x) = 1/x > 0\) → fonction convexe sur tout son domaine. 5. Tableau de variations : (voir ci-dessous)

💬 Astuce examen : pour les fonctions du type \(x\ln x - kx\), la dérivée se simplifie souvent en \(\ln x + 1 - k\). Pensez à bien vérifier la constante.

Exemple n°2 : Étude de \(g(x) = \ln(x^2 + 1)\)

Déterminer le domaine, les variations, convexité et tracer l’allure de \(g(x) = \ln(x^2+1)\).

Domaine : \(x^2+1 > 0\) toujours vrai, donc \(D_g = \mathbb{R}\). Parité : \(g(-x) = \ln(x^2+1) = g(x)\) → fonction paire (axe des ordonnées). Limites : \(\lim_{x\to \pm\infty} \ln(x^2+1) = +\infty\). Dérivée : \(g'(x) = \frac{2x}{x^2+1}\). Signe : \(g'(x) > 0\) si \(x>0\), \(g'(x)<0\) si \(x<0\). Minimum en \(x=0\), \(g(0)=0\). Dérivée seconde : \(g''(x) = \frac{2(x^2+1) - 2x(2x)}{(x^2+1)^2} = \frac{2 - 2x^2}{(x^2+1)^2}\). \(g''(x) \ge 0 \iff 1 - x^2 \ge 0 \iff x \in [-1;1]\). Donc \(g\) est convexe sur \([-1;1]\), concave ailleurs. Points d’inflexion en \(x = \pm 1\). Résultat : courbe symétrique, croissance sur \([0;+\infty[\), décroissance sur \(]-\infty;0]\).

💡 Pour les fonctions composées \(\ln(u(x))\), on utilise \(\frac{u'}{u}\) et on étudie le signe de \(u'\) car \(u>0\).

Exemple n°3 : Fonction paramétrée avec logarithme – extrait session 2024

Soit \(h(x) = \frac{\ln x}{x}\) pour \(x>0\). Étudier les variations, limites, et trouver l’extremum.

Limites : \(\lim_{x\to 0^{+}} h(x) = -\infty\) (car \(\ln x \to -\infty\), \(x\to 0^{+}\)), \(\lim_{x\to +\infty} h(x) = 0\) (croissances comparées). Dérivée : \(h'(x) = \frac{1\cdot x - \ln x \cdot 1}{x^2} = \frac{1 - \ln x}{x^2}\). Signe : \(h'(x) > 0 \iff 1 - \ln x > 0 \iff \ln x < 1 \iff 0 < x < e\). Donc \(h\) croissante sur \(]0; e[\), décroissante sur \(]e; +\infty[\). Maximum en \(x=e\) : \(h(e) = \frac{1}{e}\). Convexité (optionnelle) : \(h''(x) = \frac{-3 + 2\ln x}{x^3}\) qui change de signe en \(x = e^{3/2}\). Utile pour tracer avec précision.

🧠 La fonction \( \frac{\ln x}{x} \) est souvent utilisée pour modéliser des décroissances ou pour comparer des croissances. Retenez son maximum en \(e\).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier le domaine de définition : \(\ln(u(x))\) impose \(u(x) > 0\). Ne pas écrire \(\ln(0)\) ou \(\ln(négatif)\).
  • Attention : Confondre \(\ln(x^2)\) et \(2\ln x\) : la deuxième expression n’est valable que si \(x>0\) ; pour tout \(x\neq 0\), \(\ln(x^2) = 2\ln|x|\).
  • Piège classique : Dériver \(\ln(u)\) en oubliant le facteur \(u'\) : \(\frac{d}{dx}\ln(u) = \frac{u'}{u}\).
  • Méprise sur les limites : \(\lim_{x\to 0^{+}} x\ln x = 0\) (lever l’indétermination par croissance comparée). Ne pas écrire \(0\times (-\infty) = 0\) sans justification.
  • Confusion convexité / variation : Une fonction peut être croissante et concave (ex: \(\sqrt{x}\) sur \(]0,+\infty[\)) ; utilisez \(f''\) et non \(f'\).

📐 Formulaire de référence (logarithme et exponentielle)

Dérivées & primitives à maîtriser

  • \(\frac{d}{dx} [\ln x] = \frac{1}{x}\)    (pour \(x>0\))
  • \(\frac{d}{dx} [\ln(u(x))] = \frac{u'(x)}{u(x)}\)
  • \(\frac{d}{dx} [x\ln x] = \ln x + 1\)
  • \(\int \frac{1}{x} dx = \ln|x| + C\)
  • \(\int \ln x \, dx = x\ln x - x + C\) (par parties)
  • \(\lim_{x\to 0^{+}} x^{\alpha} \ln x = 0 \; (\alpha>0)\)   ;   \(\lim_{x\to +\infty} \frac{\ln x}{x^{\alpha}} = 0\)
🔁 Rappel : Croissances comparées : pour tout \( \alpha > 0\), \( \ln x = o(x^{\alpha})\) au voisinage de \(+\infty\) et \(x^{\alpha} = o\left(\frac{1}{|\ln x|}\right)\) au voisinage de \(0^+\).

Exercice type Bac : Étude d’une fonction avec paramètre

On donne \(f(x) = \ln(1 + e^{-x})\) définie sur \(\mathbb{R}\). 1) Déterminer les limites en \(+\infty\) et \(-\infty\). 2) Calculer \(f'(x)\) et étudier les variations. 3) Montrer que \(f\) est décroissante et déterminer sa convexité. 4) Interpréter graphiquement.

Voir les solutions détaillées
1. Limites : \(\lim_{x\to +\infty} e^{-x}=0\) donc \(f(x) \to \ln(1)=0\). \(\lim_{x\to -\infty} e^{-x}=+\infty\) donc \(1+e^{-x} \sim e^{-x}\) → \(f(x) \sim \ln(e^{-x}) = -x \to +\infty\).
2. Dérivée : \(f'(x) = \frac{-e^{-x}}{1+e^{-x}} = \frac{-1}{e^{x}+1}\) (après factorisation). \(f'(x) < 0\) pour tout \(x\) → fonction strictement décroissante.
3. Dérivée seconde : \(f''(x) = \frac{e^{x}}{(e^{x}+1)^2} > 0\) → fonction convexe sur \(\mathbb{R}\).
4. Interprétation : Courbe décroissante, convexe, asymptote horizontale \(y=0\) en \(+\infty\) et asymptote oblique (direction \(-x\)) en \(-\infty\).
