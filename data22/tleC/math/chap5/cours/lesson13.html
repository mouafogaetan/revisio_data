📊 Leçon 13 : Tableaux d'effectifs et nuage de points

Mathématiques · Terminale C (Scientifique) · Statistiques à deux variables

📌 Éléments clés à retenir

1. Organisation des données : tableau d'effectifs à double entrée

Une série statistique à deux variables \((X, Y)\) associe à chaque individu deux caractères quantitatifs. Les données sont souvent présentées dans un tableau d'effectifs (ou tableau de contingence) où chaque ligne correspond à une modalité de \(X\) et chaque colonne à une modalité de \(Y\).

  • Effectif total \(n\) : somme de tous les effectifs du tableau.
  • Effectifs marginaux : totaux par ligne (distribution de \(X\)) ou par colonne (distribution de \(Y\)).
  • Fréquences : \(f_{ij} = \frac{n_{ij}}{n}\).
  • Objectif : mettre en évidence d’éventuelles liaisons entre les deux variables.
Figure 1 : Exemple de tableau d’effectifs croisés (X = âge, Y = note).

2. Nuage de points : représentation graphique

Le nuage de points est une représentation dans le plan où chaque couple \((x_i, y_i)\) correspond à un point. Les valeurs de \(X\) sont portées en abscisse, celles de \(Y\) en ordonnée.

  • Point moyen \(G(\bar{x}, \bar{y})\) : centre de gravité du nuage, défini par : \[ \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i, \quad \bar{y} = \frac{1}{n}\sum_{i=1}^{n} y_i \]
  • Le nuage permet de visualiser la forme du lien (linéaire, parabolique, etc.) et l’existence d’éventuelles valeurs aberrantes.
  • Pour des séries pondérées (effectifs répétés), les formules intègrent les effectifs : \[ \bar{x} = \frac{\sum n_i x_i}{\sum n_i},\quad \bar{y} = \frac{\sum n_i y_i}{\sum n_i} \]

3. Caractéristiques numériques d’un nuage

En plus du point moyen, on calcule des indicateurs de dispersion et de liaison :

  • Variance et écart-type de \(X\) : \(V(X) = \frac{1}{n}\sum (x_i - \bar{x})^2\) et \(\sigma_X = \sqrt{V(X)}\).
  • Covariance : mesure la liaison linéaire entre \(X\) et \(Y\) : \[ \text{Cov}(X,Y) = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y}) \]
  • Ces calculs sont essentiels pour la droite de régression (Leçon 14).

💡 Exemples de compréhension

Exemple 1 : Construction d’un tableau d’effectifs et nuage de points

Une classe de Terminale C a relevé pour 8 élèves le nombre d’heures de révision par semaine (\(X\)) et la note obtenue à un contrôle (\(Y\)) :
Élève A : (4 h, 12/20) ; B : (6 h, 14/20) ; C : (2 h, 9/20) ; D : (5 h, 13/20) ;
E : (7 h, 16/20) ; F : (3 h, 11/20) ; G : (5 h, 15/20) ; H : (8 h, 17/20).
Construire le tableau d’effectifs (regroupement en classes si souhaité) puis le nuage de points.

Résolution :
1. Tableau des données brutes (valeurs individuelles) :
\(x_i\) : 4, 6, 2, 5, 7, 3, 5, 8
\(y_i\) : 12, 14, 9, 13, 16, 11, 15, 17
2. On peut regrouper les \(x_i\) par tranches (optionnel) mais ici on garde les points distincts.
3. Calcul du point moyen : \(\bar{x} = \frac{4+6+2+5+7+3+5+8}{8} = \frac{40}{8}=5\) heures. \(\bar{y} = \frac{12+14+9+13+16+11+15+17}{8} = \frac{107}{8}=13,375\).
4. Le nuage de points se construit en plaçant chaque couple \((x_i, y_i)\) sur un repère cartésien.

Figure 2 : Nuage de points des heures de révision vs note, avec point moyen \(G\).
💬 Astuce : Pour dessiner à la main, utilisez une échelle adaptée. Placez toujours le point moyen \(G\) ; il sera utile pour l’ajustement linéaire.

Exemple 2 : Calcul du point moyen et covariance (préparation régression)

Reprenons les données de l’exemple 1. Calculez la covariance entre \(X\) et \(Y\), ainsi que les variances.

Calcul : On a \(n=8\), \(\bar{x}=5\), \(\bar{y}=13,375\).
\[ \begin{aligned} \text{Cov}(X,Y) &= \frac{1}{8}\sum_{i=1}^{8} (x_i-5)(y_i-13,375)\\ &= \frac{1}{8}\big[(4-5)(12-13,375)+(6-5)(14-13,375)+(2-5)(9-13,375)\\ &\quad +(5-5)(13-13,375)+(7-5)(16-13,375)+(3-5)(11-13,375)\\ &\quad +(5-5)(15-13,375)+(8-5)(17-13,375)\big]\\ &= \frac{1}{8}\big[(-1)(-1,375)+(1)(0,625)+(-3)(-4,375)+0+(2)(2,625)+(-2)(-2,375)+0+(3)(3,625)\big]\\ &= \frac{1}{8}\big[1,375+0,625+13,125+0+5,25+4,75+0+10,875\big] = \frac{36}{8}=4,5. \end{aligned} \] La covariance est positive : les deux variables tendent à varier dans le même sens.
Variance de \(X\) : \(V(X)=\frac{1}{8}\sum (x_i-5)^2 = \frac{1}{8}(1+1+9+0+4+4+0+9)=\frac{28}{8}=3,5\).

💡 Une covariance positive indique que les points du nuage sont globalement orientés vers le haut à droite.

Le nuage ci-dessous représente la taille (cm) et le poids (kg) de 12 adultes. Décrivez la liaison observée.

Figure 3 : Nuage de points taille-poids.

Analyse : Les points s’allongent globalement selon une direction croissante : plus la taille est grande, plus le poids a tendance à être élevé. Il n’y a pas de point aberrant majeur. La liaison semble linéaire positive, ce qui justifiera un ajustement linéaire.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre effectifs et fréquences. L’effectif total \(n\) sert à calculer les fréquences, mais le nuage de points utilise les valeurs brutes (ou les centres de classes).
  • Attention : Un nuage de points ne montre pas une relation de cause à effet, seulement une corrélation éventuelle.
  • Piège : Oublier de pondérer par les effectifs quand les données sont regroupées (classes avec effectifs). Toujours utiliser \(\bar{x} = \frac{\sum n_i x_i}{\sum n_i}\).
  • À ne pas confondre : Le point moyen \(G\) n’est pas forcément un point observé dans la série ; c’est une moyenne.

📐 Méthode : Construire un nuage de points à partir d’un tableau

Étapes clés

  • Étape 1 : Lire le tableau d’effectifs ou la liste des couples \((x_i, y_i)\).
  • Étape 2 : Déterminer les échelles sur l’axe des abscisses (min \(x\) à max \(x\)) et ordonnées (min \(y\) à max \(y\)).
  • Étape 3 : Placer chaque point avec soin. Si plusieurs individus ont le même couple, on peut annoter l’effectif ou utiliser une taille de point variable.
  • Étape 4 : Calculer et matérialiser le point moyen \(G(\bar{x},\bar{y})\) pour faciliter l’analyse (souvent représenté par un losange ou une croix).
  • Étape 5 : Observer la forme du nuage : tendance linéaire, forme en U, dispersion, points aberrants.
🧠 Pour les calculs à la main, un tableau des écarts \((x_i-\bar{x})\) et \((y_i-\bar{y})\) facilite la covariance.

📐 Formulaire à retenir (Statistiques à deux variables)

  • Point moyen : \( \displaystyle \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i \), \( \displaystyle \bar{y} = \frac{1}{n}\sum_{i=1}^{n} y_i \)
  • Covariance : \( \displaystyle \text{Cov}(X,Y) = \frac{1}{n}\sum_{i=1}^{n} (x_i-\bar{x})(y_i-\bar{y}) \)
  • Variance de \(X\) : \( V(X) = \frac{1}{n}\sum_{i=1}^{n} (x_i-\bar{x})^2 \)
  • Écart-type : \(\sigma_X = \sqrt{V(X)}\)
  • Relation : \(\sum (x_i-\bar{x})(y_i-\bar{y}) = \sum x_i y_i - n\bar{x}\bar{y}\) (formule de calcul pratique).
