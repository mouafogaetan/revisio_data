Leçon 7 : Équations dans ℂ

Mathématiques · Terminale C · Résolution dans l’ensemble des nombres complexes

📌 Éléments clés à retenir

1. Équation du second degré à coefficients réels : $az^2 + bz + c = 0$ ($a \neq 0$)

On calcule le discriminant $\Delta = b^2 - 4ac$. Le signe de $\Delta$ détermine les solutions dans $\mathbb{C}$ :

  • Si $\Delta > 0$ : deux solutions réelles distinctes $z_1 = \dfrac{-b - \sqrt{\Delta}}{2a}$, $z_2 = \dfrac{-b + \sqrt{\Delta}}{2a}$.
  • Si $\Delta = 0$ : une solution réelle double $z_0 = -\dfrac{b}{2a}$.
  • Si $\Delta < 0$ : deux solutions complexes conjuguées : $$ z_1 = \dfrac{-b - i\sqrt{|\Delta|}}{2a}, \quad z_2 = \dfrac{-b + i\sqrt{|\Delta|}}{2a} = \overline{z_1} $$
💡 Astuce : On écrit $\sqrt{\Delta} = i\sqrt{|\Delta|}$ quand $\Delta < 0$. Ne jamais oublier le $i$ !

2. Factorisation d’un polynôme de degré $n$ dans $\mathbb{C}$

Tout polynôme $P(z)$ à coefficients réels ou complexes se factorise en produit de facteurs du premier degré dans $\mathbb{C}$ (théorème fondamental de l’algèbre). Pour un polynôme de degré 3, si $z_0$ est une racine (réelle ou complexe), on peut factoriser par $(z - z_0)$.

  • Méthode pratique : Division euclidienne ou identification pour trouver le quotient de degré 2.
  • Si les coefficients sont réels et $z_0$ complexe non réel, son conjugué $\overline{z_0}$ est aussi racine.
  • Exemple : $P(z)=z^3 - 1$ a pour racine évidente $z=1$. Après factorisation : $(z-1)(z^2+z+1)$. Ensuite on résout $z^2+z+1=0$.

3. Équations du type $z^n = a$ (racines n‑ièmes)

Pour résoudre $z^n = a$ avec $a \in \mathbb{C}^*$, on utilise la forme exponentielle : $a = |a| e^{i\theta}$. Les solutions sont : $$ z_k = |a|^{1/n} \, e^{i\left(\frac{\theta + 2k\pi}{n}\right)} \quad \text{pour } k = 0, 1, \dots, n-1 $$

  • Cas particulier : $z^n = 1$ (racines n‑ièmes de l’unité) : $z_k = e^{i\frac{2k\pi}{n}}$.
  • Les solutions sont régulièrement réparties sur un cercle de rayon $\sqrt[n]{|a|}$.

💡 Exemples de compréhension

Exemple 1 : Équation du second degré à discriminant négatif

Résoudre dans $\mathbb{C}$ : $z^2 - 4z + 13 = 0$.

Résolution : $\Delta = (-4)^2 - 4 \times 1 \times 13 = 16 - 52 = -36$. $\Delta < 0$, donc $\sqrt{\Delta} = i\sqrt{36} = 6i$.
Les solutions sont : $$ z_1 = \frac{4 - 6i}{2} = 2 - 3i, \quad z_2 = \frac{4 + 6i}{2} = 2 + 3i. $$ On vérifie que $z_2 = \overline{z_1}$.

💬 Astuce examen : Toujours présenter les solutions sous forme algébrique $a+ib$. Bien distinguer $\Delta$ et $|\Delta|$.

Exemple 2 : Factorisation d’un polynôme de degré 3 connaissant une racine

Soit $P(z) = z^3 + 2z^2 + 2z + 1$. Montrer que $z = -1$ est une racine, puis factoriser et résoudre $P(z)=0$ dans $\mathbb{C}$.

Étape 1 : $P(-1) = -1 + 2 -2 + 1 = 0$ donc $z=-1$ est racine.
Étape 2 : On effectue la division de $P(z)$ par $(z+1)$ : on obtient $z^2 + z + 1$.
Donc $P(z) = (z+1)(z^2+z+1)$.
Étape 3 : Résolution de $z^2+z+1=0$ : $\Delta = 1-4 = -3$, $\sqrt{\Delta}= i\sqrt{3}$.
$z = \dfrac{-1 \pm i\sqrt{3}}{2}$. Finalement les solutions sont : $-1$, $\; -\dfrac12 + i\dfrac{\sqrt{3}}{2}$, $\; -\dfrac12 - i\dfrac{\sqrt{3}}{2}$.

📌 Lien avec l’examen : Session 2023 C, Partie A, Exercice 1 – équations dans $\mathbb{C}$ avec factorisation.

Exemple 3 : Équation $z^3 = 8i$ (racines troisièmes)

Résoudre dans $\mathbb{C}$ l’équation $z^3 = 8i$.

On écrit $8i = 8 e^{i\frac{\pi}{2}}$ (module $8$, argument $\frac{\pi}{2}$).
Les racines cubiques : $z_k = \sqrt[3]{8} \, e^{i\left(\frac{\pi/2 + 2k\pi}{3}\right)} = 2\, e^{i\left(\frac{\pi}{6} + \frac{2k\pi}{3}\right)}$ pour $k=0,1,2$.
$k=0$ : $z_0 = 2e^{i\pi/6} = 2\left(\cos\frac{\pi}{6}+i\sin\frac{\pi}{6}\right) = \sqrt{3} + i$.
$k=1$ : $z_1 = 2e^{i5\pi/6} = 2\left(-\frac{\sqrt{3}}{2} + i\frac12\right) = -\sqrt{3} + i$.
$k=2$ : $z_2 = 2e^{i3\pi/2} = -2i$.

🧠 Représentation géométrique : les trois points forment un triangle équilatéral sur le cercle de rayon 2.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier le $i$ dans $\sqrt{\Delta}$ lorsque $\Delta<0$ et donner une solution réelle erronée.
  • Attention : Ne pas confondre la résolution dans $\mathbb{C}$ avec la résolution dans $\mathbb{R}$ : pour $\Delta<0$, il existe deux solutions complexes, pas “aucune solution”.
  • Piège : Une équation comme $z^2 = -4$ donne $z = \pm 2i$ et non $\pm 2$.
  • Astuce : Vérifier systématiquement que la somme et le produit des racines d’un trinôme vérifient $S = -b/a$ et $P = c/a$, même dans $\mathbb{C}$.

📐 Méthode pas à pas pour une équation polynomiale

Algorithme de résolution

  • 1. Recherche de racines évidentes (essayer $z = 0, \pm 1, \pm i$, etc.) pour abaisser le degré.
  • 2. Factorisation par $(z - z_0)$ (division euclidienne ou identification).
  • 3. Résolution de l’équation réduite (degré 2 ou équation binôme $z^n = a$).
  • 4. Présentation des solutions sous forme algébrique ou exponentielle suivant la consigne.

📖 Formulaire express

Second degré

$\Delta = b^2-4ac$
$\Delta < 0 \;\Rightarrow\; z = \dfrac{-b \pm i\sqrt{|\Delta|}}{2a}$

Factorisation

$P(z) = (z - z_0)Q(z)$
Si $z_0$ racine de $P$.

Racines $n$-ièmes

$z^n = a = \rho e^{i\theta}$
$z_k = \rho^{1/n} e^{i\frac{\theta+2k\pi}{n}}$
