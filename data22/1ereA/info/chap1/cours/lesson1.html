🖥️ UE 1 : Optimiser un environnement de travail

Compétence : Maîtriser les bases de l'utilisation d'un ordinateur, optimiser son environnement de travail et protéger les données.

🔧 Partitionnement 💾 Formatage 🔄 Pilotes 📀 Organisation données

Bienvenue dans cette première unité d'enseignement ! Un environnement de travail bien optimisé vous permet d'être plus efficace, de mieux organiser vos fichiers et d'éviter les pertes de données. Nous allons découvrir les concepts fondamentaux : partitionnement, formatage et pilotes. À la fin de cette leçon, vous saurez préparer un disque et comprendre l'architecture logique d'un ordinateur.

📚 1. Définitions essentielles

Partitionnement

Le partitionnement consiste à diviser un disque dur physique en plusieurs parties logiques appelées partitions. Chaque partition fonctionne comme un disque indépendant. Cela permet de séparer le système d'exploitation des données personnelles, ou d'installer plusieurs OS (Windows, Linux).

💡 Exemple : Disque de 500 Go partitionné en C: (200 Go pour Windows) et D: (300 Go pour documents).
Formatage

Le formatage est l'opération qui prépare une partition (ou un disque) à recevoir des fichiers en créant un système de fichiers (NTFS, FAT32, exFAT). Il efface les données existantes et organise l'espace pour que le système d'exploitation puisse lire/écrire.

⚠️ Attention : le formatage supprime définitivement les données (sauf restauration complexe).
Pilote (Driver)

Un pilote est un logiciel qui permet au système d'exploitation de communiquer avec un périphérique matériel (carte graphique, imprimante, disque dur). Sans pilote, le matériel ne fonctionne pas correctement.

🔧 Mise à jour régulière des pilotes = meilleures performances

🗂️ 2. Pourquoi partitionner un disque dur ?

  • ✔️ Organisation des données : Séparer système, applications et fichiers personnels facilite les sauvegardes.
  • ✔️ Multi-boot : Installer Windows + Linux sur le même disque sans conflit.
  • ✔️ Sécurité : En cas de corruption d'une partition, l'autre reste accessible.
  • ✔️ Performances : Réduire la fragmentation et améliorer la vitesse d'accès.
📢 Astuce enseignants : Avant d'installer un OS, il est recommandé de partitionner son disque pour isoler les données critiques.

🔨 3. Préparer un lecteur à recevoir des données

Avant de pouvoir utiliser un disque dur neuf ou réutiliser une clé USB, deux actions majeures sont nécessaires :

1

Partitionnement

Créer une ou plusieurs partitions sur le disque via un outil comme GParted, DiskPart (Windows) ou l'utilitaire de gestion des disques.

2

Formatage

Choisir un système de fichiers (NTFS pour Windows, exFAT pour compatibilité, ext4 pour Linux) et lancer le formatage logique. Le disque est alors prêt à stocker des données.

📌 Ordre logique :
Disque physique → Partitionnement → Formatage → Prêt à l'emploi ✅

💻 4. Exemple pratique : Simulation de commandes (Windows/Linux)

Voici à quoi ressemble la préparation d'un disque en ligne de commande :

# Windows (DiskPart)

DISKPART> list disk
DISKPART> select disk 1
DISKPART> clean
DISKPART> create partition primary size=51200
DISKPART> format fs=ntfs quick
DISKPART> assign letter=E

# Linux (fdisk + mkfs)

$ sudo fdisk /dev/sdb
> n (nouvelle partition)
> w (écriture)
$ sudo mkfs.ext4 /dev/sdb1
$ sudo mount /dev/sdb1 /mnt/data

Ces commandes effectuent le partitionnement puis le formatage. À ne pas exécuter sans précaution !
