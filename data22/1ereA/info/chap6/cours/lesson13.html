🌐 UE 13 : Insertion des liens, images, sons et vidéos en HTML

Informatique · 1ère A · Création de pages web interactives

📌 Éléments clés à retenir

🔗 1. Insérer un lien hypertexte

La balise <a> (ancre) permet de créer un lien vers une autre page web, un fichier ou une ancre interne. L'attribut obligatoire href définit la destination.

  • Lien externe : <a href="https://www.exemple.com">Texte du lien</a>
  • Lien vers une page interne : <a href="page2.html">Aller à la page 2</a>
  • Ouvrir dans un nouvel onglet : ajouter l'attribut target="_blank".
  • Lien vers une ancre : <a href="#section1">Aller à la section 1</a> (définir l'ID cible avec <h2 id="section1">).

🖼️ 2. Insérer une image

La balise <img> est auto-fermante. Elle nécessite les attributs src (source du fichier) et alt (texte alternatif pour accessibilité et si l'image ne s'affiche pas).

  • <img src="photo.jpg" alt="Description de la photo">
  • On peut ajuster la largeur et la hauteur : width="300" height="200" (en pixels) ou en CSS.
  • Formats courants : JPEG, PNG, GIF, SVG, WebP.
  • Bonnes pratiques : toujours renseigner alt pour l'accessibilité et le référencement.

🎵 3. Insérer un fichier audio

La balise <audio> permet d'intégrer des sons. On utilise <source> pour indiquer différents formats.

  • Exemple basique : 
    <audio controls>
  <source src="musique.mp3" type="audio/mpeg">
  <source src="musique.ogg" type="audio/ogg">
  Votre navigateur ne supporte pas l'audio.
</audio>
  • L'attribut controls affiche les boutons lecture/pause/volume.
  • Attributs utiles : autoplay (déconseillé sans précaution), loop (répétition), muted (muet au départ).

🎬 4. Insérer une vidéo

La balise <video> fonctionne comme <audio> mais pour des contenus vidéo.

  • Structure typique : 
    <video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Votre navigateur ne supporte pas la vidéo.
</video>
  • Attributs supplémentaires : poster="vignette.jpg" (image avant lecture), autoplay, loop.
  • Pour intégrer une vidéo YouTube, on utilise plutôt l'iframe fourni par la plateforme (embed).

📁 Bonnes pratiques & accessibilité

Toujours optimiser les médias pour le web (taille, résolution). Fournir des formats alternatifs (MP4 + WebM pour vidéo, MP3 + OGG pour audio). Le texte alternatif (alt) est obligatoire pour les images. Penser aux sous-titres pour les vidéos (<track>).

💡 Exemples de compréhension

Exemple 1 : Page web avec galerie et lien

On souhaite créer une page "portfolio.html" contenant une image cliquable qui redirige vers un site externe, et un lien de retour à l'accueil.

Code HTML : 

<!-- image cliquable -->
<a href="https://www.unsplash.com" target="_blank">
    <img src="paysage.jpg" alt="Paysage naturel" width="400">
</a>

<br>
<a href="index.html">← Retour à l'accueil</a>
💬 Astuce : Envelopper une balise <img> dans une balise <a> rend l'image cliquable. Pensez à préciser target="_blank" pour ne pas quitter votre page.

Exemple 2 : Intégrer un fichier audio avec lecture automatique (silencieux) et boucle

Ajouter un fond sonore discret (loop) mais sans déclencher le son brutalement.

Code : 

<audio controls autoplay muted loop>
  <source src="ambiance.mp3" type="audio/mpeg">
</audio>
Explication : muted + autoplay permet la lecture automatique sans bruit (autorisé par les navigateurs). L'utilisateur peut ensuite désactiver le muet.

Exemple 3 : Vidéo avec affiche personnalisée et commandes

Intégrer une vidéo de démonstration pour un cours.

<video controls width="100%" poster="apercu-cours.jpg">
  <source src="cours_html.mp4" type="video/mp4">
  <source src="cours_html.ogv" type="video/ogg">
  Désolé, votre navigateur ne lit pas les vidéos HTML5.
</video>

🧠 À retenir : L'attribut poster améliore l'expérience utilisateur avant que la vidéo ne soit lancée. Toujours proposer plusieurs formats pour compatibilité.

⚠️ Pièges courants / À ne pas confondre

  • ❌ Erreur fréquente : Oublier l'attribut alt sur <img> → perte d'accessibilité et validation HTML incorrecte.
  • ⚠️ Attention : Les chemins des fichiers (src, href) doivent être corrects. Distinguer chemin relatif (images/photo.jpg) et absolu (/dossier/photo.jpg ou URL complète).
  • 🔊 Erreur : Utiliser autoplay sur un fichier audio sans muted → bloqué par les navigateurs modernes (politique de lecture automatique).
  • 📁 Formats incompatibles : Tous les navigateurs ne supportent pas les mêmes formats (ex: anciens .avi). Préférer MP4 pour vidéo et MP3 pour audio.
  • 🔗 Lien brisé : Un lien href="#" rafraîchit la page. Utiliser href="javascript:void(0)" ou bien une ancre réelle.
