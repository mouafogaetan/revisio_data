UE 16 : Présentation des outils d’un logiciel d’infographie

Informatique · Première A, ABI, AC, SH · Module 3 : Infographie, multimédia et usage socioculturel du numérique

📌 Éléments clés à retenir

🎨 Qu'est-ce qu'un logiciel d'infographie ?

Un logiciel d'infographie (ou de traitement d'image) permet de créer, modifier, retoucher et améliorer des images numériques. Il est utilisé dans de nombreux domaines : photographie, design graphique, web, publicité, cinéma, etc.

  • Logiciels professionnels : Adobe Photoshop, CorelDRAW, Affinity Photo.
  • Logiciels libres / gratuits : GIMP (GNU Image Manipulation Program), Krita, Paint.NET.
  • Logiciels en ligne : Photopea, Pixlr, Canva (version gratuite).

🖌️ Présentation d'Adobe Photoshop : interface et outils fondamentaux

Adobe Photoshop est le logiciel de référence pour la retouche d'images et la création graphique. Son interface est organisée en plusieurs zones :

  • Barre de menus : Fichier, Édition, Image, Calque, Sélection, Filtre, etc.
  • Barre d'outils (panneau outils) : contient les principaux outils de sélection, peinture, retouche, etc.
  • Panneau des calques : gère les différents calques de l'image (superposition, fusion).
  • Zone de travail / canevas : zone où s'affiche et se modifie l'image.
  • Panneau des propriétés / réglages : ajuste les paramètres de l'outil sélectionné.

🔧 Outils indispensables de Photoshop

Voici les outils de base que tout élève doit connaître pour débuter en infographie.

🔍 Zoom (Z) ✋ Main (H) 🖱️ Déplacement (V) 📦 Rectangle de sélection (M) ✨ Lasso (L) 🪄 Baguette magique (W) ✂️ Recadrage (C) 🖌️ Pinceau (B) 🧽 Tampon de clonage (S) 💧 Doigt / Densité (flou) 📝 Texte (T) 🎨 Pot de peinture (G) 🌅 Dégradé (G) 🩹 Correcteur (J)

Raccourcis clavier essentiels : Ctrl+Z (Annuler), Ctrl+T (Transformation libre), Ctrl+S (Enregistrer), Ctrl+Alt+Z (Annuler plusieurs étapes).

💡 Opérations infographiques courantes (selon UE16)

✂️ 1. Recadrage d'une image (Crop)

Situation : Vous avez une photo trop large ou avec un arrière-plan gênant. Vous voulez ne garder que le sujet principal.

Procédure dans Photoshop :

  1. Sélectionner l'outil Recadrage (C) dans la barre d'outils.
  2. Cliquer-glisser pour délimiter la zone à conserver.
  3. Ajuster les poignées si nécessaire.
  4. Appuyer sur Entrée ou cliquer sur la coche dans la barre d'options pour valider.

💡 Astuce : Maintenir la touche Shift en recadrant pour conserver les proportions de l'image.

🩹 2. Correction des imperfections (taches, boutons, poussières)

Contexte : Un portrait présente une petite tache sur le visage ou un élément indésirable.

Méthode avec l'outil Correcteur (J) :

  1. Choisir l'outil Correcteur (spot healing brush) ou Tampon de clonage (S).
  2. Pour le tampon : Alt+clic sur une zone saine proche pour définir la source.
  3. Cliquer ou peindre sur l'imperfection pour la remplacer par la texture source.

💬 Le tampon de clonage est idéal pour supprimer un objet entier (ex: une personne dans un paysage).

🎨 3. Montage simple : superposition d'images (utilisation des calques)

Vous souhaitez incruster le logo d'un établissement sur une photo de classe.

Étapes :

  1. Ouvrir la photo de fond (Fichier → Ouvrir).
  2. Importer le logo : Fichier → Placer (ou glisser-déposer le fichier logo).
  3. Le logo apparaît sur un nouveau calque. Utiliser l'outil Déplacement (V) pour le positionner.
  4. Ajuster la taille avec Ctrl+T (transformation libre).
  5. Dans le panneau des calques, modifier l'opacité si besoin pour un effet de transparence.

🧠 À retenir : Les calques sont indépendants. On peut modifier, masquer ou supprimer chaque calque sans affecter les autres.

⚠️ Pièges courants & bonnes pratiques

  • Erreur fréquente : Travailler directement sur le calque d'arrière-plan sans le dupliquer. Bon réflexe : Dupliquer le calque (Ctrl+J) avant toute modification lourde.
  • Attention : Enregistrer au format JPEG après avoir travaillé avec des calques → la superposition est perdue (aplatissement). Solution : Conserver une copie au format PSD (format natif Photoshop) pour garder les calques.
  • Piège visuel : Utiliser un pinceau trop grand ou trop petit pour les retouches. Adapter la taille du pinceau avec les touches [ et ].
  • À ne pas confondre : Outil Déplacement (V) et outil Recadrage (C). Le premier déplace le contenu, le second recadre le canevas.

🎯 Ce qu'il faut savoir pour l'évaluation

  • Citer des logiciels d'infographie : Photoshop, GIMP, Photopea, etc.
  • Lister les opérations infographiques : recadrage, correction des imperfections, montage (calques), ajustement des couleurs, redimensionnement.
  • Connaître les outils de base de Photoshop : sélection, pinceau, tampon, correcteur, recadrage, texte, dégradé.
  • Expliquer l'utilité des calques : permettent de travailler sur des éléments indépendants et de réaliser des montages complexes.
  • Différencier les formats : PSD (modifiable, avec calques) / JPEG (image finale, sans calques).
