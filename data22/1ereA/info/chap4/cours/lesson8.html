📽️ PréAO : MS PowerPoint

Informatique · Première (A, ABI, AC, SH) · Module 1 – UA4 : Produire une présentation

📌 Éléments clés à retenir

🎯 Qu'est-ce que la préAO ?

La préAO (Préparation Assistée par Ordinateur) désigne l'ensemble des techniques et logiciels permettant de concevoir des diaporamas ou des supports de présentation. MS PowerPoint est le logiciel phare de la suite Microsoft Office dédié à la création de présentations dynamiques (slides, animations, multimédia).

  • Utilité : Exposés scolaires, réunions professionnelles, soutenances, rapports visuels.
  • Fonctions de base : Créer des diapositives, insérer du texte, des images, des formes, des graphiques.
  • Formats courants : .pptx (éditable), .pdf (diffusion), .ppsx (diaporama automatique).
🖥️ À savoir : PowerPoint fait partie des outils de préAO les plus utilisés au Cameroun dans les lycées et universités. Il permet d’intégrer des animations, vidéos et sons.

🧩 Interface et éléments essentiels

Pour bien maîtriser PowerPoint, il faut identifier les zones clés :

  • Ruban (onglets) : Accueil, Insertion, Création, Transitions, Animations, Diaporama, Révision, Affichage.
  • Zone de diapositive : espace central où l'on conçoit le contenu.
  • Volet des diapositives (gauche) : aperçu miniature de toutes les slides.
  • Zone de notes : pour ajouter des commentaires personnels (orateur).
  • Barre d’état : numéro de diapositive, zoom, mode de visualisation.

✨ Animer un objet dans une diapositive

L’animation permet de contrôler l’apparition, la mise en avant ou le déplacement des éléments (texte, images, formes). PowerPoint distingue 4 types d’animations :

  • 🔄 Apparition : l’objet entre dans la diapositive (ex: fondu, apparition progressive).
  • 🎯 Accentuation : modification sur place (clignotement, changement de couleur, grossissement).
  • 🚪 Disparition : l’objet sort de la diapositive (ex: fondu, envol).
  • 🛤️ Trajectoire personnalisée : l’objet se déplace selon un chemin défini.

Comment animer un objet ?
➡️ Sélectionner l'objet → Onglet Animations → Choisir un effet (par ex. "Fondu") → Régler la durée, le délai et l’ordre via Volet d'animation.

💡 Astuce : évitez les animations trop rapides ou trop nombreuses pour que la présentation reste professionnelle. Utilisez l’option “Déclencher” (au clic, avec précédent, après précédent).

📁 Fonctions de base – Créer une présentation complète

Les compétences minimales attendues pour produire un diaporama cohérent :

  • Créer / supprimer une diapositive : Onglet Accueil → Nouvelle diapositive ou via Ctrl+M.
  • Mise en page : choisir parmi les dispositions (titre + contenu, deux colonnes, comparaison, etc.).
  • Insérer du texte : zones de texte automatiques ou via Insertion → Zone de texte.
  • Insérer des images, formes, SmartArt : Insertion → Images / Formes / SmartArt (organigrammes).
  • Appliquer un thème / variante : Onglet Création → choisir un modèle de couleurs et polices homogènes.
  • Transitions entre diapositives : Onglet Transitions → choisir un effet (fondu, poussée, coulisseau) et gérer la durée.
  • Lancer le diaporama : F5 (début) ou Shift+F5 (diapo courante).

💡 Exemples de compréhension

Exemple 1 : Animer le titre d’une diapositive d’accueil

Contexte : Tu prépares un exposé sur "Les technologies vertes au Cameroun". Tu souhaites que le titre principal "Énergies renouvelables" apparaisse avec un effet de zoom et que le sous-titre s’affiche 2 secondes après.

Procédure :
1️⃣ Sélectionner la zone de texte du titre principal.
2️⃣ Onglet Animations → choisir Zoom (catégorie Apparition).
3️⃣ Dans Volet d'animation (ouvrir via le bouton "Volet d'animation"), vérifier l’ordre.
4️⃣ Sélectionner le sous-titre → lui attribuer l’animation Fondu.
5️⃣ Régler le délai du sous-titre : Démarrer : Après précédent avec un délai de 1,5 s.
6️⃣ Aperçu avec Aperçu ou lancer le diaporama.

💬 Astuce pratique : Pour que l’animation soit fluide, utilisez “Démarrer : Avec précédent” pour que plusieurs objets s’animent simultanément.

Exemple 2 : Construire un mini-diaporama « Les règles d’or d’une bonne présentation »

Ton professeur d’informatique demande de réaliser 3 diapositives : (1) titre, (2) conseils avec puces, (3) image avec citation. Il faut appliquer un thème, une transition et animer la citation.

Démarche détaillée :
- Étape 1 : Ouvrir PowerPoint → Nouvelle présentation → choisir thème "Ion" (onglet Création).
- Étape 2 : Diapo 1 : "Les règles d’or" (titre principal) + sous-titre "Réussir son oral".
- Étape 3 : Ajouter diapositive (disposition "Titre et contenu") → saisir 3 conseils avec puces (ex: "1. 6 lignes max par slide", "2. Police lisible", "3. Images pertinentes").
- Étape 4 : Diapo 3 : Insertion → Image (choisir une illustration de présentation) → ajouter zone de texte avec citation : "Un diaporama n’est qu’un support, l’orateur reste essentiel."
- Étape 5 : Appliquer une transition Fondu sur toutes les diapositives (onglet Transitions → Fondu → Appliquer à toutes).
- Étape 6 : Animer la citation : sélectionner la zone texte → animation "Apparition" avec durée 0,75 s.
- Étape 7 : Sauvegarder sous expose_regles.pptx → tester avec F5.

Exemple 3 : Présenter des données (statistiques classe)

Dans le cadre d’un projet sur les résultats du baccalauréat, tu dois insérer un graphique à barres montrant le taux de réussite par série (A, ABI, AC, SH). Comment faire efficacement ?

Résolution : Aller dans l’onglet Insertion → Graphique → choisir Histogramme. Une feuille Excel s’ouvre. Saisir les séries et les valeurs. Une fois terminé, le graphique apparaît sur la diapositive. On peut le mettre en forme (couleurs, titres d’axes). Astuce : utiliser l’onglet Création du graphique pour personnaliser.

📊 À retenir : Un graphique bien conçu renforce la compréhension. PowerPoint permet de lier les données à un fichier Excel ou de les intégrer statiquement.

⚠️ Pièges courants / À ne pas confondre

  • ❌ Erreur fréquente : Utiliser trop d’animations ou de transitions différentes qui fatiguent le public. Préférer la cohérence (1 ou 2 types d’animations max).
  • ⚠️ Attention : Oublier de vérifier la lisibilité : texte trop petit, police fantaisiste ou couleur contrastée insuffisante (ex: jaune sur fond blanc).
  • 💢 Ne pas confondre : Transition (effet entre deux diapositives) vs Animation (effet sur un objet à l’intérieur d’une diapositive).
  • 🧠 Erreur de manipulation : Ne pas enregistrer la présentation au bon format. Pour une modification ultérieure, utiliser .pptx ; pour une diffusion sans modification, exporter en PDF ou PPSX.
  • ⚡ Astuce anti-piège : Toujours tester le diaporama sur un autre ordinateur si possible (vérifier les polices et les médias intégrés).

⌨️ Raccourcis et trucs pratiques (PowerPoint)

Gagner du temps avec les raccourcis

  • Ctrl + N : Nouvelle présentation
  • Ctrl + M : Nouvelle diapositive
  • Ctrl + D : Dupliquer l’objet / la diapositive sélectionnée
  • F5 : Démarrer le diaporama depuis le début
  • Shift + F5 : Démarrer depuis la diapositive courante
  • Echap : Quitter le mode diaporama
  • Ctrl + G : Grouper des objets (pour les animer ensemble)

Insérer des médias : Il est possible d’insérer des vidéos YouTube (via Insertion → Vidéo en ligne) ou des fichiers audio. Pour que la présentation reste légère, pensez à compresser les images (onglet Format de l’image → Compresser les images).

🎯 Compétences visées (examen probatoire)

À l’issue de cette leçon sur PowerPoint, tu dois être capable de :

  • ✔ Créer une présentation structurée d’au moins 5 diapositives sur un thème donné.
  • ✔ Animer des objets (textes, images) en maîtrisant les 4 types d’animations.
  • ✔ Appliquer des transitions cohérentes et paramétrer la durée du diaporama.
  • ✔ Insérer des formes, images, graphiques et éventuellement du son/vidéo (conformément au module multimédia).
  • ✔ Exporter la présentation dans différents formats et l’imprimer (notes, plan, diapositives).
📌 Conseil pour l'examen théorique : On peut te demander d’expliquer la différence entre animation et transition, ou de décrire les étapes pour insérer un graphique. Révise le vocabulaire spécifique (volet d’animation, déclencheur, minuterie).
