Cloud Computing

Définition : Le Cloud (ou « informatique en nuage ») désigne l’accès à des ressources informatiques (serveurs, stockage, bases de données, logiciels) via Internet, sans gestion locale. On loue ou utilise ces services à la demande, depuis n’importe quel appareil connecté.

🌍 Exemple : Google Drive, iCloud, Dropbox, Microsoft OneDrive.
📀

Stockage en ligne

C’est une forme de Cloud dédiée au stockage de fichiers (documents, photos, vidéos) sur des serveurs distants. Accessible depuis tout appareil connecté, les données sont synchronisées et sécurisées par des fournisseurs.

Avantages du Cloud

  • Accessibilité permanente : Accès aux données depuis n'importe quel appareil (PC, tablette, smartphone) et où que l'on soit.
  • Partage simplifié : Envoyer un lien à des collègues ou amis pour consulter/modifier des fichiers.
  • Économie matérielle : Pas besoin de disques durs externes coûteux; mise à l'échelle flexible.
  • Sauvegarde automatique : Réduction des risques de perte de données (vol, incendie, panne).
⚠️

Inconvénients du Cloud

  • Dépendance à Internet : Pas de connexion = pas d'accès aux données (sauf mode hors-ligne limité).
  • Sécurité / confidentialité : Risque de piratage, de violation de données ou d’accès non autorisé par les fournisseurs.
  • Coûts cachés : L’espace gratuit est limité; les abonnements peuvent devenir chers pour les gros volumes.
  • Délocalisation des données : Les lois sur la protection diffèrent selon les pays où se trouvent les serveurs.

🏢 Les 3 grands modèles de service Cloud

ModèleDescriptionExemple
IaaSInfrastructure (serveurs, réseaux)AWS EC2, Google Compute Engine
PaaSPlateforme de développementHeroku, Google App Engine
SaaSLogiciels accessibles en ligneGoogle Drive, Microsoft 365, Dropbox
💡 Pour un élève ou enseignant, le Cloud est surtout utilisé sous forme SaaS (Google Drive, iCloud, OneDrive) pour stocker cours, partages et travaux collaboratifs.
