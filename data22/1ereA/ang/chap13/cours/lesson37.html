Lesson 23: So / Such ... that

Exprimer la conséquence (cause et effet) · Anglais Première CD · Révision Probatoire

📌 Éléments clés à retenir

1. SO + adjectif/adverbe + THAT

On utilise so ... that pour exprimer une conséquence extrême. La structure relie une cause (l'intensité d'une qualité) à un effet (la conséquence).

  • Formule : so + adjectif / adverbe + that + phrase (conséquence)
  • Avec adjectif : The movie was so boring that I fell asleep. (Le film était tellement ennuyeux que je me suis endormi.)
  • Avec adverbe : He ran so fast that nobody could catch him. (Il courait tellement vite que personne n'a pu l'attraper.)
  • On peut aussi employer so + many/much + nom + that pour les quantités.
  • She had so much work that she stayed up all night.
✨ Idée visuelle : "She was so tired that she fell asleep on her desk." → Cause (fatigue intense) → Conséquence (s'endormir).

2. SUCH + (adjectif) + nom + THAT

Such ... that s'utilise devant un nom (parfois précédé d'un adjectif). On parle d'une chose ou d'une personne si particulière que cela entraîne une conséquence.

  • Formule : such + (adjectif) + nom (dénombrable/indénombrable) + that + conséquence
  • Avec adjectif + nom : It was such a beautiful day that we decided to go to the beach. (C'était une si belle journée que nous avons décidé d'aller à la plage.)
  • Attention au déterminant : devant un nom singulier dénombrable, on met "such a/an".
  • They are such nice people that everyone loves them. (Ce sont des gens tellement gentils que tout le monde les aime.)
  • Nom indénombrable ou pluriel : It's such nonsense that nobody believes it.
💡 Astuce mnémotechnique :
→ SO + adjectif (sans nom) → so hot, so quickly
→ SUCH + (adjectif) + NOM → such a hot day, such quick results

3. Différence essentielle : SO vs SUCH

Beaucoup d'élèves confondent ces deux structures. La règle d'or concerne la présence d'un nom.

  • SO + adjectif/adverbe (pas de nom directement après so).
  • SUCH + (adjectif) + NOM (toujours un nom après such).
  • Exemple identique : The cake was so delicious that I ate it all. → pas de nom après delicious.
  • Exemple identique : It was such a delicious cake that I ate it all. → "cake" est le nom.
  • ⚠️ Piège : Ne pas dire "so delicious cake" ❌ ni "such delicious" sans nom ❌.

💡 Exemples de compréhension

Exemple 1 : Conséquence avec SO + adjectif

Contexte : Après un long match de football, les joueurs sont épuisés.

Phrase en anglais : "The players were so exhausted that they could not celebrate their victory."
📖 Traduction : "Les joueurs étaient tellement épuisés qu'ils n'ont pas pu célébrer leur victoire."
🔍 Explication : L'adjectif "exhausted" exprime l'état extrême. So + exhausted → intensité. La conséquence est qu'ils ne peuvent pas célébrer.

💬 Astuce : Quand l'adjectif exprime un degré très fort (très fatigué, très heureux, très rapide), pensez à "so ... that".

Exemple 2 : Such + adjectif + nom (conséquence)

Une élève a obtenu des résultats exceptionnels à l'examen.

Phrase : "She got such good marks that she received a scholarship."
Traduction : "Elle a eu de si bonnes notes qu'elle a reçu une bourse."
Analyse : "such" + adjectif "good" + nom "marks" → la conséquence est l'obtention de la bourse. Remarquez que "marks" est pluriel, on n'utilise pas "a".

✨ Variante avec singulier : "It was such a difficult exam that many students failed." (C'était un examen tellement difficile que beaucoup d'élèves ont échoué.)

Exemple 3 : So much / so many + that

Situation : Pendant les fêtes, une famille a préparé énormément de nourriture.

Phrase : "There was so much food that they couldn't finish everything."
Traduction : "Il y avait tellement de nourriture qu'ils n'ont pas pu tout finir."
Règle : "so much" + nom indénombrable (food, water, time) ; "so many" + nom dénombrable pluriel (so many people, so many problems).

📌 Ne pas confondre : so many books that... (beaucoup de livres) vs such interesting books that... (livres tellement intéressants).

⚠️ Pièges courants & erreurs fréquentes

  • ❌ Erreur : "She is so a nice girl that everyone likes her." → Correction : "She is such a nice girl that..." (car "nice girl" = adjectif + nom).
  • ❌ Erreur : "It was such hot that we stayed inside." → Correction : "It was so hot that..." (hot est adjectif, pas de nom).
  • ⚠️ Attention : Après "such", si le nom est singulier dénombrable, il faut "such a / such an". Ex: such an amazing story that ...
  • 📖 Oubli de "that" : Dans un style très familier, on peut omettre "that", mais à l'écrit et aux examens, gardez-le pour plus de clarté. so tired (that) ... → préférez inclure "that".
  • 🎯 Confusion so/such avec "very" : "Very" exprime juste l'intensité sans conséquence explicite. "So ... that" montre un rapport cause/effet.

✍️ Mini exercices (auto-évaluation)

Complétez avec SO ou SUCH

  • 1. The weather was _______ terrible that we cancelled the picnic.
  • 2. They are _______ friendly people that we immediately felt welcome.
  • 3. He drove _______ fast that the police stopped him.
  • 4. It was _______ an interesting documentary that I watched it twice.
  • 5. There were _______ many mosquitoes that we couldn't sleep.
📌 Voir les réponses

1. so (terrible = adjectif)
2. such (friendly people = adjectif + nom pluriel)
3. so (fast = adverbe)
4. such (such an interesting documentary → nom singulier)
5. so (so many + nom dénombrable)

📚 Phrases modèles pour le bac / probatoire

  • The novel was so captivating that I read it in one night. (Le roman était tellement captivant que je l'ai lu en une nuit.)
  • She has such a powerful voice that she doesn't need a microphone.
  • There was so much noise that we couldn't concentrate.
  • It was such a moving speech that everyone applauded.
🎭 Scène : "It was such an inspiring speech that the audience gave a standing ovation."
🧠 Récapitulatif visuel (à retenir absolument) :
➕ SO + adjectif/adverbe + THAT + conséquence
➕ SUCH + (adjectif) + NOM + THAT + conséquence
🎯 Exemple clé : The exam was so difficult that many students failed. → It was such a difficult exam that many students failed.
