📘 So / Such...that : L'intensité qui mène à une conséquence

Anglais · Première C · Examen Probatoire

📌 Éléments clés à retenir

🔹 1. Structure avec so : so + adjectif/adverbe + that

On utilise so devant un adjectif (qualificatif) ou un adverbe pour exprimer un degré extrême, et that introduit la conséquence.
En français, cela correspond à « si ... que » ou « tellement ... que ».

  • Formule : So + adjectif/adverbe + that + sujet + verbe (conséquence)
  • 📌 Exemple : The movie was so boring that I fell asleep.
    (Le film était tellement ennuyeux que je me suis endormi.)
  • 📌 Exemple : He ran so fast that nobody could catch him.
    (Il a couru si vite que personne n'a pu l'attraper.)
  • ⚠️ Important : Après so, on ne met jamais de nom directement. Si on a un nom, on utilise such.

🔹 2. Structure avec such : such + (adjectif) + nom + that

On utilise such devant un nom (avec ou sans adjectif). La conséquence est toujours introduite par that.
Traduction : « un tel ... que » ou « une telle ... que ».

  • Formules :
    • Such + a/an + (adjectif) + nom singulier + that
    • Such + (adjectif) + nom pluriel / indénombrable + that
  • 📌 Exemple (nom singulier) : It was such a beautiful day that we went to the beach.
    (C'était une si belle journée que nous sommes allés à la plage.)
  • 📌 Exemple (nom pluriel) : They are such nice people that everyone loves them.
    (Ce sont des gens tellement sympas que tout le monde les aime.)
  • 📌 Exemple (nom indénombrable) : She has such energy that she works 12 hours a day.
    (Elle a une telle énergie qu'elle travaille 12 heures par jour.)

🔍 Règle d'or : Distinguer so et such

Le choix dépend du mot qui suit immédiatement so ou such :

  • ✅ SO + adjectif/adverbe (pas de nom) → so interesting, so quickly
  • ✅ SUCH + (adjectif) + NOM → such an interesting book, such quick progress
  • ⚠️ Astuce mnémotechnique : "Such" contient "noun" (nom) dans sa sonorité ? Pensez "Such + Nom".
  • 🎯 Remarque : Si on a "so + adjectif + nom" c'est une erreur fréquente. Exemple incorrect : so beautiful girl → such a beautiful girl (car "girl" est un nom).

💡 Exemples de compréhension

📖 Exemple 1 : Le succès d'un élève

Contexte : Paul a obtenu des résultats exceptionnels au Probatoire. Sa famille est très fière.

Phrase avec "such" : Paul got such good results that his parents organized a big party.
🔎 Explication : "results" est un nom pluriel → on utilise such. L'adjectif "good" se place avant le nom.

Phrase avec "so" (variante possible) : Paul's results were so good that his parents organized a big party.
🔎 Explication : Ici, "so" est suivi de l'adjectif "good" (pas de nom directement). On a transformé la phrase en mettant le verbe "were".

💬 Astuce : Si vous pouvez supprimer le nom et garder du sens, utilisez "so". Sinon, utilisez "such". Ex: "so good" (adjectif seul) ≠ "such good results" (nom).

🚗 Exemple 2 : Situation de circulation à Douala

Contexte : Les embouteillages sont très longs. Beaucoup de personnes arrivent en retard au travail.

Construction correcte : There was such heavy traffic that many employees were late.
🔎 Explication : "traffic" est un nom indénombrable → such + adjectif + nom. On ne peut pas dire "so heavy traffic".

Variante avec "so" : The traffic was so heavy that many employees were late.
🔎 Explication : On change la structure : "The traffic was so heavy" → adjectif "heavy" après le verbe "was". C'est aussi correct.

💡 Retenez : Such + nom / So + adjectif. Quand vous hésitez, réécrivez la phrase en plaçant le nom en premier.

🎓 Exemple 3 : Événement culturel au Cameroun (Fêtes traditionnelles)

Lors d'un festival, il y a une foule immense et la musique est très entraînante.

Phrase 1 (such) : It was such an exciting festival that tourists come back every year.
Phrase 2 (so) : The festival was so exciting that tourists come back every year.
👉 Les deux sont justes, mais la structure grammaticale change. Entraînez-vous à passer de l'une à l'autre.

🧠 Pour les examens : On utilise très souvent "so...that" et "such...that" dans les rédactions pour exprimer la cause et la conséquence de façon riche.

⚠️ Pièges courants & Erreurs fréquentes

  • ❌ Erreur n°1 : Utiliser "so" devant un nom seul.
    Incorrect : She has so beautiful voice that...
    Correct : She has such a beautiful voice that...
  • ❌ Erreur n°2 : Oublier l'article indéfini "a/an" avec "such" + nom singulier.
    Incorrect : It was such interesting book that...
    Correct : It was such an interesting book that...
  • ❌ Erreur n°3 : Confondre l'ordre des mots avec "so many / so much".
    Note : "so many + nom pluriel" et "so much + nom indénombrable" existent aussi !
    Exemple correct : He had so many problems that he gave up. (Ici "so" + many + nom, c'est une exception car "many" est un quantifieur).
    Mais pour le reste, suivez la règle générale.
  • ⚠️ Attention : Dans le langage courant, on peut parfois omettre "that", mais à l'écrit et à l'examen, gardez "that" pour plus de clarté.

📊 Tableau récapitulatif : quand choisir So ou Such ?

Structure Exemple Pourquoi ?
So + adjectif The coffee is so hot that I can't drink it. "hot" = adjectif, pas de nom derrière so.
So + adverbe She spoke so quickly that nobody understood. "quickly" = adverbe de manière.
Such + a/an + adj + nom singulier It's such an expensive car that I can't afford it. "car" = nom singulier → such + a/an
Such + adj + nom pluriel They are such kind people that everyone likes them. "people" = nom pluriel, pas de "a".

✍️ Mini exercice : Testez-vous !

Complétez avec so / such / such a

  • 1. It was _______ beautiful weather that we stayed outside all day.
  • 2. The problem was _______ complicated that nobody could solve it.
  • 3. He has _______ amazing talent that he will become famous.
  • 4. They made _______ noise that the neighbors complained.
🔎 Voir les réponses

1. such (weather nom indénombrable) → such beautiful weather
2. so (complicated adjectif) → so complicated
3. such (talent nom) → such amazing talent (indénombrable)
4. such (noise nom) → such noise / such a noise (les deux possibles, mais "such a noise" courant)

💬 Conseil examen : Relisez toujours le mot qui suit la structure. Si c'est un adjectif seul → so. Si c'est un nom (même avec adjectif) → such.

🎯 Pour briller à l'écrit (Probatoire)

Utilisez so...that et such...that dans vos dissertations pour exprimer des conséquences avec emphase. Cela enrichit votre style et montre une bonne maîtrise grammaticale.

Exemple de phrase de transition : "Corruption has become such a serious issue that citizens have lost faith in the system."

"The government acted so slowly that the population protested."
