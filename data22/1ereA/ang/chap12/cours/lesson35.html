Lesson 23: So / Such...that

Anglais · Première CD · Révision Probatoire

📌 Éléments clés à retenir

1. Structure avec "So + adjectif/adverbe + that"

On utilise so devant un adjectif ou un adverbe pour exprimer une conséquence extrême. La structure signifie "tellement... que" en français.
➡️ So + adjectif/adverbe + that + proposition de conséquence.

  • Exemple : The movie was so boring that I fell asleep.
    (Le film était tellement ennuyeux que je me suis endormi.)
  • Exemple : He ran so fast that nobody could catch him.
    (Il courait tellement vite que personne n’a pu l’attraper.)
  • 💡 Astuce : "So" est toujours suivi d'un adjectif ou d'un adverbe (pas d'un nom).

2. Structure avec "Such + (adjectif) + nom + that"

On utilise such devant un nom (seul ou accompagné d'un adjectif). Cela signifie aussi "un tel / une telle... que" ou "tellement... que" mais en mettant l'accent sur le nom.

  • Exemple : It was such a hot day that we stayed inside.
    (C'était une journée tellement chaude que nous sommes restés à l'intérieur.)
  • Exemple : She has such good friends that she is never lonely.
    (Elle a des amis si formidables qu'elle n'est jamais seule.)
  • ⚠️ Attention à l'article : devant un nom singulier dénombrable, on ajoute "a/an" : such a nice car ; pour les noms indénombrables ou pluriels, pas d'article : such nice weather, such beautiful flowers.

3. Différence essentielle : so vs such

✔️ SO + ADJECTIF / ADVERBE
so + adj/adv + that
The coffee was so hot that I burned my tongue.
✔️ SUCH + (ADJ) + NOM
such + (adj) + noun + that
It was such hot coffee that I burned my tongue.

📝 Règle mnémotechnique :
- "SO" s'utilise SANS nom juste après.
- "SUCH" s'utilise AVEC un nom (parfois avec adjectif). Imagine : "SO" est court → pas de nom ; "SUCH" a plus de lettres → accueille le nom.

🎨 Illustration conceptuelle : "so hard that he passed" / "such success that everyone applauded"

💡 Exemples de compréhension

Exemple 1 : So + adjectif → conséquence logique

Contexte : Paul a très mal à la tête. Il ne peut pas se concentrer.

Construction avec "so" :
Paul had so a headache ? → NON ! On ne dit pas "so a headache" car "headache" est un nom.
On dit plutôt : Paul's headache was so severe that he couldn't concentrate.
(La migraine de Paul était tellement sévère qu'il ne pouvait pas se concentrer.)
✔️ On utilise so severe (adjectif) + that.

💬 Astuce : Quand vous voyez un adjectif comme "tired, hungry, beautiful, fast", pensez à "so". Ex: The music was so loud that the neighbors complained.

Exemple 2 : Such + nom (avec ou sans adjectif) → résultat frappant

Contexte : C'était un orage violent. Toute l'école a été privée d'électricité.

Construction avec "such" :
It was such a violent storm that the whole school lost power.
(C'était un orage tellement violent que toute l'école a perdu l'électricité.)
👉 such a violent storm (nom "storm" précédé de l'adjectif "violent" + article "a").

Autre variante (nom sans adjectif) : It was such bad weather that the match was cancelled.
(Il faisait un temps tellement mauvais que le match a été annulé.)

💡 Pensez à "such a/an + adjectif + nom" pour les noms dénombrables singuliers. Pour les noms pluriels : "such + adjectif + nom pluriel" (such interesting books).

Exemple 3 : Transformation d'une phrase simple en "so...that" ou "such...that"

Phrase simple : "The suitcase is very heavy. I cannot lift it."

🔁 Deux possibilités :
- Avec so : The suitcase is so heavy that I cannot lift it.
- Avec such : It is such a heavy suitcase that I cannot lift it.
✅ Les deux sont corrects, mais la structure change : so + adjectif (heavy) // such + (a) + nom (suitcase).

🌳 Infographie mentale : "so tall" → attribut de l'arbre ; "such a tall tree" → met l'accent sur l'objet "tree".

⚠️ Pièges courants / À ne pas confondre

  • ❌ Erreur fréquente 1 : Utiliser "so" devant un nom seul.
    Incorrect : I have so money that I can buy anything.
    Correct : I have such a lot of money that I can buy anything. (ou "so much money").
    Rappel : "so" + adj/adv ; "such" + nom.
  • ❌ Erreur fréquente 2 : Oublier l'article "a/an" avec "such" devant un nom singulier dénombrable.
    Incorrect : It was such beautiful day that we went out.
    Correct : It was such a beautiful day that we went out.
  • 🧩 Confusion avec "so...that" et "so that" : "So that" exprime le but (pour que), tandis que "so...that" exprime la conséquence (tellement... que).
    Exemple : He studied hard so that he could pass (but) ≠ He studied so hard that he fell asleep (conséquence).
  • 📌 Attention : Avec "much/many/little/few", on utilise "so" : so much work, so many people.
    Mais "such" peut aussi être employé avec "a lot of" : such a lot of work.

✍️ Petit test éclair (auto-évaluation)

Complétez avec "so" ou "such" :

  1. The movie was _______ boring that we left early.
  2. It was _______ an interesting story that everyone listened carefully.
  3. She has _______ beautiful eyes that everybody admires her.
  4. He drove _______ fast that the police stopped him.
📖 Voir les réponses

1. so (boring = adjectif)
2. such (such an interesting story : nom avec article)
3. such (such beautiful eyes : nom pluriel)
4. so (fast = adverbe)

💪 Pour réussir, posez-vous la question : est-ce que le mot qui suit est un adjectif/adverbe → so ; est-ce que c’est un nom (parfois avec adjectif) → such.
