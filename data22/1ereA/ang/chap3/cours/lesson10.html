🎓 Lesson 10 : Type 2 & Type 3 Conditionals

Anglais · Première CD · Révision Probatoire

📌 Éléments clés à retenir

🔁 Type 2 : Hypothèse irréelle du présent / futur

Le conditionnel type 2 exprime une situation improbable ou imaginaire dans le présent ou le futur. On l'utilise pour parler de ce qui contredit la réalité actuelle.

  • Structure : If + past simple (prétérit) , would + base verbale (infinitif sans to)
  • Exemple : If I were rich, I would travel the world. (Si j'étais riche, je voyagerais autour du monde → Mais je ne le suis pas.)
  • ⚠️ Verbe "to be" : On utilise souvent were pour toutes les personnes (I, he, she, it) dans un contexte formel ou hypothétique. If she were here...
  • Dans la proposition principale on peut aussi utiliser could ou might pour exprimer la capacité ou la possibilité.
💡 Astuce en français : Le type 2 correspond à notre « si + imparfait, conditionnel présent » en français. Pensez à des rêves ou des situations contraires à la réalité.

⏳ Type 3 : Regret & situation irréelle du passé

Le conditionnel type 3 exprime une hypothèse sur le passé, quelque chose qui ne s'est pas produit. On l'utilise souvent pour exprimer des regrets ou des critiques sur le passé.

  • Structure : If + past perfect (had + participe passé) , would have + participe passé
  • Exemple : If I had studied harder, I would have passed the exam. (Si j'avais étudié davantage, j'aurais réussi l'examen → Mais je n'ai pas assez étudié.)
  • Variantes : could have (aurait pu) ou might have (aurait peut-être).
  • Note importante : Le type 3 décrit un passé irréversible. On ne peut pas changer le passé, seulement l'imaginer différemment.
💡 Équivalence française : « Si + plus-que-parfait, conditionnel passé ». Exemple : "Si j'avais su, je serais venu."

⚖️ Comparaison rapide : Type 2 vs Type 3

✅ Type 2 (Présent/Futur imaginaire)

If + past simple → would + BV

If I had money, I would buy a car.
(Mais je n'ai pas d'argent maintenant.)

🕒 Quand ? Situations hypothétiques actuelles ou futures.

✅ Type 3 (Passé irréel / regret)

If + past perfect → would have + V-ed/part. passé

If I had had money, I would have bought a car.
(Je n'avais pas d'argent à ce moment-là.)

🕒 Quand ? Regret sur un événement passé non réalisé.

💡 Exemples de compréhension

🌟 Exemple Type 2 – Situation imaginaire actuelle

Contexte : Tu n'as pas de connexion internet en ce moment. Exprime une hypothèse.

✅ Phrase correcte : "If I had internet at home, I would do my online research easily."
🔍 Explication : "had" est le prétérit de "have" (même si je parle du présent). La situation est fausse : je n'ai pas internet, donc j'imagine le résultat.

💬 Astuce : Pour le type 2, on utilise toujours le prétérit dans la proposition if, même si on parle du présent.

😔 Exemple Type 3 – Regret après un examen

Hier, tu n'as pas révisé le chapitre sur les modaux. Aujourd'hui, l'examen était difficile sur ce sujet.

📘 Phrase : "If I had revised modal verbs yesterday, I would have answered all the questions correctly."
🧠 Démarche : On utilise "had revised" (past perfect) car l'action de réviser est antérieure à l'échec. La conséquence est "would have answered".

💡 Pense à la chronologie : la condition (réviser) est avant le résultat (bien répondre). Dans le passé, tout est déjà fini → past perfect + would have + participe passé.

🔄 Comparaison directe : Type 2 vs Type 3 avec le même verbe

Le verbe "wake up" (se réveiller). Observe la différence de temps et de sens.

Type 2 (maintenant / futur) : "If I woke up early tomorrow, I would not miss the bus." (Il est peu probable que je me réveille tôt, mais c'est encore possible demain.)
Type 3 (passé) : "If I had woken up early yesterday, I would not have missed the bus." (Je ne me suis pas réveillé tôt hier → c'est trop tard.)

🎯 Astuce mnémotechnique : Type 2 = rêve dans le présent ; Type 3 = remords dans le passé.

⚠️ Pièges courants & erreurs fréquentes

  • ❌ Erreur 1 : Mettre "would" dans la proposition "if".
    Incorrect : "If I would be rich, I would travel." → Corrigé : "If I were rich, I would travel." (jamais de would après if dans les conditionnels classiques).
  • ❌ Erreur 2 : Confondre type 2 et type 3 dans le choix du temps.
    Exemple faux : "If I had known the answer, I would tell you." (Si j'avais su (passé) → j'aurais dit (passé)).
    Correction : "If I had known the answer, I would have told you." (type 3)
  • ❌ Erreur 3 : Oublier la virgule ou l'inverser.
    Quand la condition (if) est en premier, on met une virgule. Si la conséquence est en premier, pas de virgule.
    ✅ If it rained, I would stay home. / I would stay home if it rained.
  • ⚠️ Attention spéciale : Le verbe "to be" au type 2 : "If I were" (et non was dans un anglais soigné, surtout à l'écrit).
    If he were the president, he would change the law. (même pour he/she/it).

🎯 Entraînement mental : Transforme les phrases

Passer du Type 2 au Type 3 (ou l'inverse)

Pour bien maîtriser la nuance, essaie mentalement ces transformations. Le temps change tout le sens.

  • Type 2 → Type 3 : "If I had free time now, I would help you." (imaginaire présent) → Version passé : "If I had had free time yesterday, I would have helped you."
  • Type 3 → Type 2 : "If she had taken the train, she would have arrived on time." (regret passé) → Version présent : "If she took the train (regularly), she would arrive on time."
  • Conseil : Repère l’indice de temps (now, every day → Type 2 ; yesterday, last week, in 2010 → Type 3).

📖 Rappel des objectifs officiels (Programme Probatoire)

✔️ Utiliser le type 2 pour des situations hypothétiques (présent/futur)
✔️ Utiliser le type 3 pour des regrets et des situations irréelles du passé
✔️ Maîtriser la concordance des temps : if + past simple → would + BV ; if + past perfect → would have + past participle

📚 Tableau mémo – Formules clés

Type 2
If + past simple
would + infinitif (sans to)
Type 3
If + past perfect (had + V-ed/3e colonne)
would have + past participle
Exemple type 2
If I knew, I would tell you.
Exemple type 3
If I had known, I would have told you.
