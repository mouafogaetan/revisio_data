Leçon 5 : Formules de transformation

Trigonométrie · Première D & TI · Outils pour équations et études de fonctions

📌 Formules fondamentales à retenir

1. Formules d'addition

Elles permettent d'exprimer le cosinus et le sinus d'une somme ou d'une différence.

  • $\cos(a + b) = \cos a \cos b - \sin a \sin b$
  • $\cos(a - b) = \cos a \cos b + \sin a \sin b$
  • $\sin(a + b) = \sin a \cos b + \cos a \sin b$
  • $\sin(a - b) = \sin a \cos b - \cos a \sin b$
💡 Astuce mnémotechnique : "Casser le sinus" → sinus change de signe dans la différence, cosinus garde le même signe.

2. Formules de duplication (double angle)

Cas particulier des formules d'addition lorsque $a = b$. Très utiles pour simplifier ou transformer des expressions.

  • $\cos(2a) = \cos^2 a - \sin^2 a$
  • $\cos(2a) = 2\cos^2 a - 1$
  • $\cos(2a) = 1 - 2\sin^2 a$
  • $\sin(2a) = 2 \sin a \cos a$

On en déduit aussi les formules de linéarisation (puissance) : $\cos^2 a = \dfrac{1+\cos(2a)}{2}$ et $\sin^2 a = \dfrac{1-\cos(2a)}{2}$.

3. Transformation $a\cos x + b\sin x$ en $R\cos(x-\varphi)$ ou $R\sin(x+\varphi)$

Technique essentielle pour étudier des fonctions, résoudre des équations ou déterminer une amplitude maximale.

  • $a\cos x + b\sin x = R \cos(x - \varphi)$ avec $R = \sqrt{a^2 + b^2}$
  • Détermination de $\varphi$ : $\cos\varphi = \frac{a}{R}$ et $\sin\varphi = \frac{b}{R}$ (donc $\tan\varphi = \frac{b}{a}$ si $a\neq 0$).
  • Variante : $a\cos x + b\sin x = R \sin(x + \psi)$ avec $\sin\psi = \frac{a}{R}$ et $\cos\psi = \frac{b}{R}$.
📌 Interprétation : $R$ est l'amplitude maximale, $\varphi$ le déphasage.

💡 Exemples guidés

Exemple 1 : Calcul exact avec les formules d'addition

Calculer $\cos\left(\frac{5\pi}{12}\right)$ sachant que $\frac{5\pi}{12} = \frac{\pi}{4} + \frac{\pi}{6}$.

Résolution :
$\cos\left(\frac{\pi}{4} + \frac{\pi}{6}\right) = \cos\frac{\pi}{4}\cos\frac{\pi}{6} - \sin\frac{\pi}{4}\sin\frac{\pi}{6}$
$= \frac{\sqrt{2}}{2} \times \frac{\sqrt{3}}{2} - \frac{\sqrt{2}}{2} \times \frac{1}{2} = \frac{\sqrt{6}}{4} - \frac{\sqrt{2}}{4} = \frac{\sqrt{6} - \sqrt{2}}{4}$.

💬 Astuce : Décomposer un angle inconnu en somme d'angles remarquables ($\frac{\pi}{6}, \frac{\pi}{4}, \frac{\pi}{3}$).

Exemple 2 : Utilisation des formules de duplication

Simplifier l'expression $f(x) = \cos^2(3x) - \sin^2(3x)$.

Simplification : On reconnaît $\cos^2 X - \sin^2 X = \cos(2X)$ avec $X = 3x$.
Donc $f(x) = \cos(2 \times 3x) = \cos(6x)$.

✨ La duplication permet de réduire la fréquence : elle transforme un carré en cosinus simple.

Exemple 3 : Transformation $a\cos x + b\sin x$

Écrire $g(x) = \sqrt{3}\cos x + \sin x$ sous la forme $R\cos(x - \varphi)$.

Étapes :
1. $R = \sqrt{(\sqrt{3})^2 + 1^2} = \sqrt{3+1}=2$.
2. On a $\cos\varphi = \frac{\sqrt{3}}{2}$ et $\sin\varphi = \frac{1}{2}$.
3. L'angle $\varphi$ qui vérifie ces deux conditions est $\varphi = \frac{\pi}{6}$.
Donc $g(x) = 2\cos\left(x - \frac{\pi}{6}\right)$.

🎯 Cette forme permet de connaître directement l'amplitude (2) et le déphasage ($\pi/6$), très utile pour tracer la courbe.

🔧 Applications typiques au programme

Résolution d'équations trigonométriques

Les formules de transformation permettent de ramener une équation complexe à une forme basique $\cos X = \cos Y$ ou $\sin X = \sin Y$.

  • Exemple : $\cos(2x) + \sin(2x) = 1$ se transforme en $\sqrt{2}\cos\left(2x - \frac{\pi}{4}\right) = 1$.
  • Exemple : $\sin(3x)\cos(x) - \cos(3x)\sin(x) = \frac12$ → forme $\sin(2x) = \frac12$ (formule d'addition inverse).

Étude de fonctions

Pour une fonction du type $f(x) = a\cos(\omega x) + b\sin(\omega x)$, la forme $R\cos(\omega x - \varphi)$ donne directement :

  • Amplitude $|R|$
  • Période $T = \frac{2\pi}{\omega}$
  • Maximum $R$ et minimum $-R$

📐 Illustration : transformation $a\cos x + b\sin x$

Figure : Représentation géométrique de $R = \sqrt{a^2+b^2}$ et de l'angle $\varphi$ tel que $\cos\varphi = a/R$, $\sin\varphi = b/R$.

🔍 Interprétation : $(a,b)$ sont les coordonnées d'un point sur le cercle de rayon $R$. L'angle $\varphi$ est l'argument de ce point.

⚠️ Pièges courants et erreurs fréquentes

  • Erreur de signe : dans $\sin(a-b) = \sin a \cos b - \cos a \sin b$ (et non $+$).
  • Oublier le facteur 2 : $\sin(2a) = 2\sin a \cos a$ (et non $\sin a \cos a$).
  • Mauvaise détermination de $\varphi$ : Vérifier simultanément $\cos\varphi$ et $\sin\varphi$ pour placer $\varphi$ dans le bon quadrant.
  • Confondre $R\cos(x-\varphi)$ et $R\cos(x+\varphi)$ : bien vérifier le signe dans la transformation.

📖 Formulaire synthétique

  • Addition : $\cos(a\pm b) = \cos a \cos b \mp \sin a \sin b$ ; $\sin(a\pm b) = \sin a \cos b \pm \cos a \sin b$
  • Duplication : $\cos 2a = \cos^2 a - \sin^2 a = 2\cos^2 a -1 = 1-2\sin^2 a$ ; $\sin 2a = 2\sin a \cos a$
  • Linéarisation : $\cos^2 a = \frac{1+\cos2a}{2}$ ; $\sin^2 a = \frac{1-\cos2a}{2}$
  • Forme réduite : $a\cos x + b\sin x = \sqrt{a^2+b^2}\cos(x-\varphi)$ avec $\tan\varphi = \frac{b}{a}$ (en tenant compte du quadrant).
