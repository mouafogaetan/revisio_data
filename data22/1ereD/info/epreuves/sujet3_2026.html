📐 Exercice 1 : Structures algorithmiques (6 points)
Chap 7 – Structures itératives, conditionnelles et tableaux
1. (1,5 pt) Définissez les termes suivants :
a. Structure itérative (boucle)
b. Structure conditionnelle
c. Tableau (en algorithmique)

✅ Correction

a. Une structure itérative permet de répéter une séquence d'instructions un nombre défini ou indéfini de fois (boucles POUR, TANT QUE, REPETER).

b. Une structure conditionnelle permet d'exécuter des instructions selon qu'une condition est vraie ou fausse (SI...ALORS...SINON, SELON...CAS).

c. Un tableau est une structure de données permettant de stocker plusieurs valeurs de même type, accessibles par un indice.

2. (2 pts) On considère l'algorithme suivant :
Variable: tab : tableau[1..5] d'entier
Variable: i, somme : entier
Début
    somme ← 0
    Pour i allant de 1 à 5 faire
        Ecrire("Entrer l'élément ", i)
        Lire(tab[i])
        somme ← somme + tab[i]
    FinPour
    Ecrire("La somme est : ", somme)
Fin
a. (0,5 pt) Quel est le rôle de cet algorithme ?
b. (0,5 pt) Combien de fois la boucle est-elle exécutée ?
c. (1 pt) Modifier cet algorithme pour qu'il affiche également la moyenne des éléments.

✅ Correction

a. Cet algorithme permet de remplir un tableau de 5 entiers saisis par l'utilisateur, puis calcule et affiche leur somme.

b. La boucle est exécutée 5 fois (i de 1 à 5).

c. Algorithme modifié :

Variable: tab : tableau[1..5] d'entier
Variable: i, somme : entier
Variable: moyenne : réel
Début
    somme ← 0
    Pour i allant de 1 à 5 faire

3. (1,5 pt) Écrivez un algorithme qui lit un entier N et affiche s'il est pair ou impair en utilisant une structure conditionnelle.

✅ Correction

Variable: N : entier
Début
    Ecrire("Entrez un entier :")
    Lire(N)
    Si (N mod 2 = 0) Alors
        Ecrire(N, " est pair")
    Sinon
        Ecrire(N, " est impair")
    FinSi
Fin
4. (1 pt) Citez les trois types de boucles existant en algorithmique et précisez leur différence principale.

✅ Correction

POUR (FOR) : Utilisée quand le nombre d'itérations est connu à l'avance.

TANT QUE (WHILE) : Exécute la boucle tant qu'une condition est vraie (0 ou plusieurs itérations).

REPETER JUSQU'A (DO...WHILE) : Exécute la boucle puis teste la condition (au moins une itération).

⚙️ Exercice 2 : Fonctions et procédures (4 points)
Chap 8 – Sous-programmes
1. (1 pt) Quelle est la différence fondamentale entre une fonction et une procédure ?

✅ Correction

Une fonction retourne une valeur unique au programme appelant (ex: calculer la racine carrée).

Une procédure ne retourne pas de valeur ; elle exécute simplement une série d'actions (ex: afficher un menu).

2. (1,5 pt) Définissez les termes suivants :
a. Variable locale
b. Variable globale

✅ Correction

a. Une variable locale est déclarée à l'intérieur d'un sous-programme et n'est accessible que dans ce sous-programme.

b. Une variable globale est déclarée dans le programme principal et est accessible par tous les sous-programmes.

3. (1,5 pt) Écrivez une fonction en algorithmique nommée carre qui prend un entier en paramètre et retourne son carré.

✅ Correction

Fonction carre(x: entier): entier
Début
    retourner x * x
Fin

Exemple d'appel : carre(5) retourne 25.

💻 Exercice 3 : Programmation en langage C (5 points)
Chap 11 – Syntaxe, structures de contrôle, tableaux
1. (1 pt) À quoi sert la ligne #include <stdio.h> dans un programme en C ?

✅ Correction

#include <stdio.h> permet d'inclure la bibliothèque standard d'entrée/sortie (Standard Input Output). Elle donne accès aux fonctions printf() (affichage) et scanf() (saisie).

2. (1,5 pt) Traduisez en langage C l'algorithme suivant :
Variable: N, i : entier
Variable: somme : réel
Début
    somme ← 0
    Ecrire("Entrez un entier N :")
    Lire(N)
    Pour i allant de 1 à N faire
        somme ← somme + i
    FinPour
    Ecrire("La somme des ", N, " premiers entiers est : ", somme)
Fin

✅ Correction

#include <stdio.h>

int main() {
    int N, i;
    float somme = 0;
    
    printf("Entrez un entier N : ");
    scanf("%d", &N);
    
    for(i = 1; i <= N; i++) {
        somme = somme + i;
    }
    
    printf("La somme des %d premiers entiers est : %.0f\n", N, somme);
    
    return 0;
}
3. (1,5 pt) Écrivez en C la déclaration et l'initialisation d'un tableau d'entiers nommé notes de taille 10. Accédez ensuite à son 3ème élément (indice 2 en C).

✅ Correction

// Déclaration d'un tableau d'entiers de taille 10
int notes[10];

// Initialisation du 3ème élément (indice 2)
notes[2] = 15;

// Accès au 3ème élément
printf("La 3ème note est : %d", notes[2]);

Rappel : En C, les indices des tableaux commencent à 0 (premier élément : notes[0], deuxième : notes[1], troisième : notes[2]).

4. (1 pt) Nommez les étapes de compilation et d'exécution d'un programme en C.

✅ Correction

Les étapes sont : Édition (écriture du code source) → Compilation (transformation en code objet) → Édition de liens (liaison avec les bibliothèques) → Exécution.

🌐 Exercice 4 : Programmation HTML et JavaScript (5 points)
Chap 9 & 10 – Balises HTML, événements, fonctions JS
1. (1,5 pt) Complétez le code HTML suivant pour créer une page web contenant :
  • - Un titre principal (h1) "Ma Page Web"
  • - Une image (nommez le fichier "logo.png" avec le texte alternatif "Logo du site")
  • - Un paragraphe décrivant brièvement la page

✅ Correction

<!DOCTYPE html>
<html>
<head>
    <title>Ma Page Web</title>
</head>
<body>
    <h1>Ma Page Web</h1>
    <img src="logo.png" alt="Logo du site" />
    <p>Ceci est une page web d'exemple créée en HTML.</p>
</body>
</html>
2. (1,5 pt) Expliquez le rôle des attributs suivants en HTML :
a. colspan
b. rowspan

✅ Correction

colspan : Permet de fusionner plusieurs colonnes dans un tableau HTML. Exemple : <td colspan="2"> fusionne 2 colonnes.

rowspan : Permet de fusionner plusieurs lignes dans un tableau HTML.

3. (2 pts) Écrivez une fonction JavaScript afficherMessage() qui affiche une boîte de dialogue "Bienvenue sur mon site !" lorsqu'un bouton est cliqué. Donnez le code HTML complet.

✅ Correction

<!DOCTYPE html>
<html>
<head>
    <script language="Javascript">
        function afficherMessage() {
            alert("Bienvenue sur mon site !");
        }
    </script>
</head>
<body>
    <button onClick="afficherMessage();">Cliquez ici</button>
</body>
</html>

Explication : La balise <script> contient la fonction. L'événement onClick du bouton déclenche l'exécution de afficherMessage().
