📖 Lesson 52: Word Formation

Formation des mots : suffixes, préfixes & construction lexicale · English · Première CD (Probatoire)

📌 Éléments clés à retenir

🔹 1. Suffixes pour former des NOMS (Noun suffixes)

En anglais, on ajoute des suffixes à des verbes ou des adjectifs pour créer des noms (personne, chose, qualité, action). Voici les suffixes les plus fréquents au programme officiel :

  • -tion / -sion → action ou état.
    Exemples : educate (éduquer) → education ; decide (décider) → decision.
  • -ment → résultat d’une action.
    Exemples : govern (gouverner) → government ; develop (développer) → development.
  • -ness → qualité ou état (surtout à partir d’adjectifs).
    Exemples : happy (heureux) → happiness ; dark (sombre) → darkness.
  • -ity / -ty → qualité abstraite.
    Exemples : able (capable) → ability ; certain (certain) → certainty.
💡 Astuce : Pour repérer le nom, demandez-vous : « Puis-je mettre “the” ou “a/an” devant ? » Les noms se réfèrent souvent à une chose, une idée ou une personne.

🔸 2. Suffixes pour former des ADJECTIFS (Adjective suffixes)

On transforme un nom ou un verbe en adjectif pour décrire une qualité. Suffixes typiques :

  • -ful → « plein de » (caractérise une qualité positive).
    Exemples : care (soin) → careful ; hope (espoir) → hopeful.
  • -less → « sans » (exprimant l’absence).
    Exemples : care → careless (négligent) ; home → homeless (sans abri).
  • -ive → tendance à faire qqch.
    Exemples : act (agir) → active ; create → creative.
  • -ous → caractérise une qualité (souvent scientifique ou abstraite).
    Exemples : danger → dangerous ; fame (célébrité) → famous.

🚫 3. Préfixes négatifs (Negative prefixes)

Les préfixes changent le sens d’un mot en son contraire. Très importants pour enrichir son vocabulaire.

  • un- → le plus courant pour adjectifs/verbes.
    Exemples : happy → unhappy ; known → unknown ; lock → unlock.
  • in- / im- / il- / ir- → variantes selon la lettre suivante.
    Exemples : possible → impossible ; legal → illegal ; regular → irregular ; active → inactive.
  • dis- → souvent pour verbes et adjectifs (opposition).
    Exemples : agree → disagree ; honest → dishonest.
📌 À retenir : Im- s’utilise devant les mots commençant par m ou p (impatient, immature). In- devant la plupart des autres (incomplete). Il- devant l (illegal). Ir- devant r (irresponsible).

💡 Exemples de compréhension

📝 Exemple 1 : Transformer un verbe en nom (-tion, -ment)

Complétez les phrases avec le nom formé à partir du verbe entre parenthèses.

Phrase 1 : The government announced a new ______ (develop).
➡️ Correction : development (développement).
Phrase 2 : His ______ (decide) surprised everyone.
➡️ Correction : decision (décision).
Explication : On ajoute -ment à "develop" et on remplace "de" par "sion" pour decide → decision.

💬 Méthode : Cherchez d’abord le radical du verbe (ex: educate → educat-), puis ajoutez le suffixe -ion (education).

🎯 Exemple 2 : Former des adjectifs avec -ful / -less / -ive

Complétez avec l’adjectif correspondant au mot donné.

1. use (utiliser) → adjectif avec "plein de" : useful (utile) / useless (inutile).
2. create (créer) → adjectif qui décrit une personne imaginative : creative.
3. care (attention) → antonyme : careless (négligent).
➡️ Phrase modèle : “She is very creative and always makes useful objects from waste.”

✨ Petit truc : -ful = « full of », -less = « without ». Ainsi « hopeful » = plein d’espoir, « hopeless » = sans espoir.

🔁 Exemple 3 : Utiliser les préfixes négatifs (un-, in-, im-, dis-)

Rendez les adjectifs suivants négatifs en ajoutant le préfixe correct.

- possible → impossible (attention, p → im)
- regular → irregular
- legal → illegal
- comfortable → uncomfortable
- honest → dishonest
Phrase d’exemple : It is illegal to drive without a license, and it’s uncomfortable to travel in a crowded bus.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre -tion et -sion. Exemples : verbe “decide” → decision (avec s) ; verbe “educate” → education (avec t). Il faut connaître la forme.
  • Attention : Ne pas ajouter un suffixe directement sans ajustement orthographique : « happy » perd son y pour devenir « happiness » ; « create » perd son e pour « creative ».
  • Piège : Le préfixe in- change selon la consonne : impossible (pas *inpossible), illegal (pas *inlegal).
  • Faux-amis en formation : “Actually” n’est pas « actuellement » (c’est « en fait ») ; la formation de l’adverbe se fait souvent avec -ly mais ce n’est pas l’objet ici. Restez concentré sur les noms/adjectifs.
  • Conseil : Un mot ne peut pas avoir deux suffixes négatifs. On ne dit pas “unimpossible” mais “impossible”. Apprenez les combinaisons usuelles.

✍️ Petit test express (auto-évaluation)

Transformez les mots suivants :

  1. pollute (verbe) → nom en -tion : ______________
  2. happy (adjectif) → nom en -ness : ______________
  3. responsible → adjectif négatif avec préfixe ir- : ______________
  4. comfort (nom) → adjectif avec -able (idée de confortable) : ______________
  5. employ (verbe) → nom en -ment : ______________
📌 Voir les réponses

1. pollution
2. happiness
3. irresponsible
4. comfortable
5. employment
