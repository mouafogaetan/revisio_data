📊 Leçon 15 : Point moyen

Mathématiques · Terminale A (Littéraire) · Statistiques à une variable

📌 Éléments clés à retenir

🎯 Définition du point moyen

Dans une série statistique à deux variables $(x_i, y_i)$ pour $i = 1, 2, \dots, n$, le point moyen noté $G(\bar{x}, \bar{y})$ est le point dont l’abscisse est la moyenne des $x_i$ et l’ordonnée la moyenne des $y_i$.

$$ \bar{x} = \frac{x_1 + x_2 + \dots + x_n}{n} = \frac{1}{n}\sum_{i=1}^{n} x_i \quad ; \quad \bar{y} = \frac{y_1 + y_2 + \dots + y_n}{n} = \frac{1}{n}\sum_{i=1}^{n} y_i $$
  • Le point moyen représente le « centre de gravité » du nuage de points.
  • Il est indispensable pour la méthode d’ajustement linéaire (Mayer, moindres carrés).
  • Les coordonnées sont toujours les moyennes arithmétiques des séries.

📐 Propriété fondamentale

La droite de régression (ou droite d’ajustement) passe toujours par le point moyen $G(\bar{x}, \bar{y})$. Que ce soit la droite de Mayer ou la droite des moindres carrés, le point moyen appartient à la droite.

Illustration : Nuage de points et position du point moyen G (croix rouge).

Cette propriété permet de vérifier la cohérence d’un ajustement : toute droite d’ajustement doit passer par $G$.

🧮 Calcul pratique étape par étape

Pour déterminer le point moyen d’une série statistique :

  1. Additionner toutes les valeurs de la variable $x$ : $S_x = \sum x_i$.
  2. Additionner toutes les valeurs de la variable $y$ : $S_y = \sum y_i$.
  3. Compter l’effectif total $n$ (nombre de couples).
  4. Calculer $\bar{x} = S_x / n$ et $\bar{y} = S_y / n$.
  5. Placer le point $G(\bar{x}; \bar{y})$ dans le repère.

💡 Exemples de compréhension

Exemple 1 : Relevé de notes et heures d’étude

On étudie le lien entre le nombre d’heures de révision par semaine ($x$) et la note obtenue à un examen ($y$) pour 5 élèves.
$(x_i; y_i)$ : (2 ; 9), (3 ; 11), (5 ; 14), (7 ; 16), (8 ; 18).

Résolution :
Effectif $n = 5$.
Somme des $x$ : $2+3+5+7+8 = 25$ → $\bar{x} = 25/5 = 5$.
Somme des $y$ : $9+11+14+16+18 = 68$ → $\bar{y} = 68/5 = 13,6$.
Point moyen : $G(5\ ;\ 13,6)$.

💬 Astuce : Vérifier que $G$ se situe bien au « milieu » des données. Ici, les heures varient de 2 à 8, la moyenne 5 est cohérente ; les notes de 9 à 18, moyenne 13,6.

Exemple 2 (Sujet type examen - Session 2023/2024)

Une entreprise suit l’évolution de son chiffre d’affaires (en milliers d’euros) en fonction du rang de l’année.
Années : rang 1 → 45, rang 2 → 50, rang 3 → 53, rang 4 → 58, rang 5 → 64.
Calculer le point moyen.

Calcul :
$x_i$ (rang) : 1, 2, 3, 4, 5 → $\sum x_i = 15$ → $\bar{x} = 15/5 = 3$.
$y_i$ (CA) : 45, 50, 53, 58, 64 → $\sum y_i = 270$ → $\bar{y} = 270/5 = 54$.
Donc $G(3\ ;\ 54)$.

💡 Ce point moyen servira ensuite à tracer la droite de Mayer ou à faire des prévisions (extrapolation). Dans les examens, on demande souvent de le placer sur le nuage.

Exemple 3 : Cas particulier - données répétées

Une enquête sur le nombre d’enfants par famille donne :
(1 enfant, 2 familles) ; (2 enfants, 5 familles) ; (3 enfants, 3 familles).
Attention ici $x$ = nombre d’enfants, $y$ = effectif ? Non : pour le point moyen on considère un tableau de deux variables statistiques. En réalité, pour une seule variable, la notion de point moyen n’a pas de sens. Mais dans un ajustement, on peut avoir $x$ = rang, $y$ = effectif cumulé. Prenons un vrai exemple à deux variables :
Production ($x$) et coût de revient ($y$) : (10, 200), (12, 215), (10, 205), (14, 230), (12, 218).
Calculer $\bar{x}$ et $\bar{y}$.

$n=5$.
$\sum x = 10+12+10+14+12 = 58$ → $\bar{x}=58/5=11,6$.
$\sum y = 200+215+205+230+218 = 1068$ → $\bar{y}=1068/5=213,6$.
Point moyen $G(11,6\ ;\ 213,6)$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre le point moyen avec la médiane du nuage. Le point moyen est une moyenne arithmétique, pas une valeur centrale au sens de la médiane.
  • Attention : Oublier de diviser par l’effectif total (ne pas confondre avec une somme cumulée). Toujours vérifier $n$ = nombre de couples.
  • Piège : Dans un tableau avec effectifs (données regroupées), il faut d’abord calculer les produits $n_i \times x_i$ et $n_i \times y_i$ pour obtenir les bonnes sommes, puis diviser par l’effectif total $N = \sum n_i$.
  • À ne pas oublier : Le point moyen ne suffit pas à lui seul pour déterminer la droite d’ajustement ; il faut une deuxième condition (un autre point, ou la méthode de Mayer avec deux sous-groupes).

✏️ Mini-exercice d’entraînement

À toi de jouer !

Voici les relevés de température moyenne mensuelle (en °C) et du nombre de glaces vendues (en centaines) dans une ville :
Mois : Jan (x=1, y=12), Fév (2, 15), Mars (3, 18), Avr (4, 21), Mai (5, 25), Juin (6, 30).

Consigne : Calcule les coordonnées du point moyen $G(\bar{x}, \bar{y})$. Vérifie avec la correction ci-dessous (cache puis révèle).

Voir la correction

$n = 6$.
$\sum x = 1+2+3+4+5+6 = 21$ → $\bar{x}=21/6 = 3,5$.
$\sum y = 12+15+18+21+25+30 = 121$ → $\bar{y}=121/6 \approx 20,17$.
Donc $G(3,5\ ;\ 20,17)$.

💪 Astuce : Si tu trouves $\bar{x}=3,5$ et $\bar{y}\approx 20,17$, c’est juste ! Ce point sera utile pour tracer la droite de Mayer dans un second temps.
