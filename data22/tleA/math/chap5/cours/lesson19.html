📈 Leçon 20 : Inéquations avec exponentielles

Terminale A (Littéraire) · Mathématiques · Changement de variable $X = e^x$

📌 Éléments clés à retenir

🔁 Principe du changement de variable

Pour résoudre une inéquation de la forme $e^{2x} + 2e^{x} - 15 \ge 0$ (ou avec $e^{2x}$, $e^{x}$, constante), on pose $X = e^{x}$ avec $X > 0$. L’inéquation devient alors une inéquation du second degré en $X$. Une fois résolue dans $\mathbb{R}$ pour $X$, on revient à la variable $x$ en utilisant la bijection de la fonction exponentielle : $$ X = e^{x} \;\Longleftrightarrow\; x = \ln X \quad (\text{avec } X>0)$$

  • Domaine de validité : $e^x > 0$ pour tout réel $x$, donc $X$ appartient toujours à $]0, +\infty[$.
  • Transformation : $e^{2x} = (e^x)^2 = X^2$ et $e^{kx} = (e^x)^k = X^k$.
  • Comparaison : Si $X \ge a$ avec $a>0$, alors $x \ge \ln a$ (car $\ln$ est croissante). Si $X \le b$, $x \le \ln b$.
📊 Illustration : de $e^{2x} + 2e^x -15 \ge 0$ à $X^2 + 2X -15 \ge 0$, résolution puis $x \ge \ln 3$.

⚡ Résolution type : inéquation $a e^{2x} + b e^{x} + c \ge 0$ (ou $\le 0$)

Méthode en 4 étapes systématiques :

Étape 1 : Poser $X = e^{x}$ ($X > 0$). Remplacer $e^{2x}$ par $X^2$ et $e^{x}$ par $X$.
Étape 2 : Résoudre l'inéquation du second degré en $X$ : $aX^2 + bX + c \ge 0$ (ou $\le 0$) dans $\mathbb{R}$.
Étape 3 : Intersection de l’ensemble solution avec la condition $X > 0$ (car $e^x > 0$). On obtient un intervalle ou une réunion d’intervalles de la forme $[X_1, X_2]$ ou $]-\infty, X_1] \cup [X_2, +\infty[$ (après ajustement avec $X>0$).
Étape 4 : Revenir à $x$ : $X = e^x$ donc $x = \ln X$ (croissance de $\ln$). Transformer les bornes : $X \ge \alpha \iff x \ge \ln \alpha$ ; $X \le \beta \iff x \le \ln \beta$.

💡 Astuce examen : Toujours vérifier que les bornes en $X$ sont positives avant d’appliquer $\ln$, sinon l’inéquation n’a pas de solution ou bien est toujours vraie.

🧠 Cas particuliers et pièges

  • Si après résolution on obtient $X \le \text{nombre négatif}$, cette branche est impossible car $X>0$.
  • Penser à factoriser si possible : $e^{2x} - 3e^{x} \ge 0$ peut se factoriser par $e^x$ mais le changement de variable reste efficace.
  • Ne pas oublier que $\ln(1) = 0$, $\ln(e) = 1$, etc. pour simplifier les solutions.
  • Lien avec l'examen : Session 2023, Exercice 1 (inéquation $e^{2x} + 2e^{x} - 15 \ge 0$). Savoir écrire l’ensemble solution sous forme d’intervalle ou de réunion.

💡 Exemples de compréhension

Examen type : $e^{2x} + 2e^{x} - 15 \ge 0$ (Cameroun, Session 2023)

Résoudre dans $\mathbb{R}$ l’inéquation : $e^{2x} + 2e^{x} - 15 \ge 0$.
🔎 Résolution détaillée :
1. Changement de variable : $X = e^{x}$, $X>0$. Alors $e^{2x} = X^2$. L’inéquation devient :
$X^2 + 2X - 15 \ge 0$.
2. Résolution du trinôme : $\Delta = 2^2 - 4(1)(-15) = 4 + 60 = 64$, $\sqrt{\Delta}=8$.
Racines : $X_1 = \frac{-2 - 8}{2} = -5$, $\quad X_2 = \frac{-2 + 8}{2} = 3$.
Signe : $X^2+2X-15 \ge 0$ pour $X \le -5$ ou $X \ge 3$ (car $a=1>0$).
3. Intersection avec $X>0$ : $X \le -5$ impossible (car $X>0$). Seule branche $X \ge 3$.
Ensemble solution en $X$ : $[3, +\infty[$.
4. Retour à $x$ : $e^{x} \ge 3 \iff x \ge \ln 3$ (car $\ln$ strictement croissante).
Solution finale : $S = [\ln 3, +\infty[$.
💬 À retenir : $\ln 3 \approx 1.0986$. Toujours vérifier le signe de la racine négative ($-5$) qu’on ignore car $X>0$.

Exemple 2 : inéquation $e^{2x} - 4e^{x} + 3 \le 0$

Résoudre $e^{2x} - 4e^{x} + 3 \le 0$.
Étape 1 : $X = e^x > 0$, on a $X^2 - 4X + 3 \le 0$.
Étape 2 : $\Delta = 16 - 12 = 4$, racines $X_1 = 1$, $X_2 = 3$. Trinôme négatif entre les racines : $X \in [1, 3]$.
Étape 3 : $X>0$ déjà vérifié, on conserve $[1,3]$.
Étape 4 : $1 \le e^{x} \le 3 \iff \ln 1 \le x \le \ln 3 \iff 0 \le x \le \ln 3$.
$S = [0, \ln 3]$
📌 $\ln 1 = 0$ : les bornes s’expriment simplement.

Exemple 3 : forme plus complexe $e^{2x} - 5e^{x} + 6 > 0$

Résoudre $e^{2x} - 5e^{x} + 6 > 0$ dans $\mathbb{R}$.
➡️ Changement $X=e^x$ : $X^2 -5X + 6 >0$
Racines : $\Delta = 25-24=1$, $X_1=2$, $X_2=3$. Signe positif à l’extérieur : $X<2$ ou $X>3$.
Avec $X>0$, on obtient $03$.
Retour : $00$ toujours) et $e^x>3 \iff x>\ln 3$.
$S = ]-\infty, \ln 2[ \;\cup\; ]\ln 3, +\infty[$

⚠️ Pièges courants / À ne pas confondre

  • ❌ Oublier $X>0$ : Ne jamais garder une solution $X$ négative ou nulle car $e^x >0$ strictement.
  • ❌ Inverser le sens de l’inégalité : $\ln$ est croissante, donc $e^x \ge a \iff x \ge \ln a$ (avec $a>0$). Ne pas changer le sens.
  • ❌ Confondre $e^{2x}$ et $(e^x)^2$ : C’est la même chose, mais attention aux parenthèses en posant $X^2$.
  • ❌ Ne pas traiter les cas où le discriminant est négatif : Si le trinôme en $X$ est toujours positif, alors l’inéquation est vérifiée pour tout $X>0$ donc pour tout $x$ réel.
  • ❌ Erreur d’écriture des intervalles : $\ln(\text{nombre})$ n’existe que si le nombre est strictement positif. Vérifier les bornes avant de passer au logarithme.

📋 Méthode résumée (aide-mémoire)

1️⃣ $X=e^x$ ( $X>0$ ) → transformer en $aX^2+bX+c \ge 0$
2️⃣ Résoudre dans $\mathbb{R}$ → obtenir intervalles en $X$
3️⃣ Garder uniquement $X>0$
4️⃣ $x = \ln X$ → écrire solution avec $\ln$.
