🔹 Exercice 1 : QCM – Concepts des systèmes d'information (4 pts)
1. Un système d'information automatisé se compose principalement de :
  • A. Uniquement des ordinateurs et des imprimantes
  • B. Des ressources matérielles, logicielles, des données et des acteurs
  • C. Des fichiers Excel et des bases de données uniquement
  • D. Des serveurs web uniquement

Réponse B Un SI automatisé intègre matériel, logiciels, données et utilisateurs (acteurs).

2. Laquelle de ces actions correspond à la fonction "Collecte" d’un système d’information ?
  • A. Générer un graphique de ventes
  • B. Saisir les notes des élèves dans un fichier
  • C. Archiver les anciennes commandes
  • D. Transmettre un rapport au directeur

Réponse B La collecte consiste à saisir ou acquérir des données (ex : saisie des notes).

🗄️ Exercice 2 : Bases de données – Concepts fondamentaux (4 pts)
3. Dans une table "Élèves", quel champ serait le meilleur choix comme clé primaire ?
  • A. Nom
  • B. Date de naissance
  • C. Numéro_matricule (unique)
  • D. Ville

Réponse C Une clé primaire doit être unique et ne change pas. Le matricule est idéal.

4. Un SGBD (Système de Gestion de Base de Données) permet :
  • A. Uniquement créer des présentations
  • B. Définir, manipuler et contrôler les accès aux données
  • C. Naviguer sur Internet
  • D. Écrire des feuilles de style CSS

Réponse B SGBD : MySQL, Access, Oracle – permet de gérer les données efficacement.

📝 Exercice 3 : Maîtrise du langage SQL (8 points)

Contexte : Une base de données nommée Bibliotheque contient la table Livres :

Livres (id_livre INT PRIMARY KEY, titre VARCHAR(100), auteur VARCHAR(80), annee_publication INT, genre VARCHAR(40))
5. Écrire la requête SQL qui affiche tous les livres de l’auteur "Émile Zola".

Requête SELECT SELECT * FROM Livres WHERE auteur = 'Émile Zola';

On utilise WHERE pour filtrer les lignes selon l’auteur.

6. Écrire la requête pour insérer un nouveau livre : "Les Misérables", Victor Hugo, 1862, Roman.

Requête INSERT INSERT INTO Livres (titre, auteur, annee_publication, genre) VALUES ('Les Misérables', 'Victor Hugo', 1862, 'Roman');

L'id_livre est auto-incrémenté ou à renseigner si nécessaire.

7. Modifier l’année de publication du livre "Germinal" (d'Émile Zola) pour la corriger en 1885.

Requête UPDATE UPDATE Livres SET annee_publication = 1885 WHERE titre = 'Germinal' AND auteur = 'Émile Zola';

Attention à bien cibler le bon enregistrement.

8. Expliquer brièvement l’effet de la requête DELETE FROM Livres WHERE annee_publication < 1900;

Effet : Supprime tous les livres publiés avant l’année 1900. Cette opération est irréversible sans sauvegarde.

🏢 Exercice 4 : Système d’information en entreprise (4 points)
9. Dans une entreprise, le système de pilotage (SI décisionnel) a pour rôle principal :
  • A. Gérer les stocks quotidiens
  • B. Aider les dirigeants à prendre des décisions stratégiques (tableaux de bord, analyses)
  • C. Saisir les feuilles de présence
  • D. Imprimer les factures clients

Réponse B Le système de pilotage (ex : Business Intelligence) fournit des indicateurs pour la direction.

10. Citez deux ressources matérielles d’un système d’information automatisé.

Exemples : Serveurs, postes clients, disques de stockage, imprimantes réseau, switchs, etc.
