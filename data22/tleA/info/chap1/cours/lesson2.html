Schéma fonctionnel de l'ordinateur

Informatique · Terminale A (Littéraire) · Unité 1 : Architecture et protection de l'ordinateur

📌 Éléments clés à retenir

Le modèle Entrée-Traitement-Sortie (E-T-S)

L'ordinateur fonctionne selon un principe universel : il reçoit des données en entrée, les traite grâce à son unité centrale, puis fournit un résultat en sortie. Ce modèle s'applique à toutes les actions réalisées par un ordinateur.

  • ENTRÉE : Les données ou instructions fournies à l'ordinateur (via clavier, souris, microphone, scanner).
  • TRAITEMENT : Les opérations effectuées par le processeur (calculs, comparaisons, transformations).
  • SORTIE : Le résultat obtenu après traitement (affiché à l'écran, imprimé, sonore).

Le flux de données dans l'ordinateur

Les données circulent entre les différents composants de l'ordinateur via des bus (circuits de communication). Comprendre ce flux permet de visualiser le cheminement de l'information.

📡 Cheminement typique des données :
Périphérique d'entrée → Mémoire vive (stockage temporaire) → Processeur (traitement) → Mémoire vive → Périphérique de sortie

Le processeur ne communique pas directement avec les périphériques. Il utilise la mémoire vive (RAM) comme zone de stockage intermédiaire pour les données en attente de traitement ou de sortie.

Les échanges entre composants

L'unité centrale (processeur + mémoire) est le cœur de l'ordinateur. Elle échange en permanence des données avec :

  • Les périphériques d'entrée : pour recevoir des informations de l'utilisateur.
  • Les périphériques de sortie : pour restituer les résultats du traitement.
  • Les périphériques de stockage (disque dur, SSD) : pour sauvegarder des données à long terme.

📊 Schéma du modèle Entrée → Traitement → Sortie

Légende : Les flèches représentent le flux de données. L'unité centrale (processeur + mémoire) reçoit les données des périphériques d'entrée, les traite, et envoie le résultat aux périphériques de sortie.

Exemple concret n°1 : Saisie d'un texte

Vous tapez "Bonjour" sur le clavier, puis ce mot s'affiche à l'écran.

📝 Cheminement des données :
1️⃣ ENTRÉE : Le clavier (périphérique d'entrée) envoie les codes des touches tapées.
2️⃣ TRAITEMENT : Le processeur reçoit ces codes, les interprète et prépare l'affichage.
3️⃣ SORTIE : L'écran (périphérique de sortie) affiche le mot "Bonjour".

💬 Astuce : Dans ce cas, la mémoire vive stocke temporairement chaque lettre avant que l'écran ne les affiche.

Exemple concret n°2 : Impression d'un document

Vous cliquez sur "Imprimer" dans un logiciel de traitement de texte.

🖨️ Cheminement des données :
1️⃣ Le document est chargé depuis le disque dur (stockage) vers la mémoire vive.
2️⃣ Le processeur traite les données (mise en page, police, etc.).
3️⃣ Les données formatées sont envoyées à l'imprimante (périphérique de sortie).
4️⃣ L'imprimante produit le document papier.

💡 L'imprimante a parfois sa propre mémoire pour stocker les données en attente (spoule d'impression).

Exemple concret n°3 : Lecture d'une musique en ligne

Vous écoutez une chanson sur Internet via des écouteurs branchés à l'ordinateur.

🎵 Cheminement des données :
1️⃣ ENTRÉE : Clic de souris pour lancer la lecture.
2️⃣ Les données audio sont téléchargées depuis Internet vers la mémoire vive.
3️⃣ TRAITEMENT : Le processeur et la carte son décodent le fichier audio.
4️⃣ SORTIE : Le signal audio est envoyé aux écouteurs / haut-parleurs.

🎯 Liens avec l'examen (Session 2025)

  • Question type possible : "Établissez le schéma fonctionnel simplifié de l'ordinateur en identifiant les échanges de données entre l'unité centrale, les entrées et les sorties."
  • Savoir-faire attendu : Être capable de placer correctement chaque élément (entrée, unité centrale, sortie) et de décrire le flux de données avec des flèches.
  • Piège à éviter : Ne pas confondre le rôle du disque dur (stockage permanent) avec celui de la mémoire vive (stockage temporaire pour le traitement).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Penser que l'écran est une entrée alors que c'est une sortie. L'écran affiche des informations, il ne les envoie pas à l'ordinateur.
  • Attention : Certains périphériques sont à la fois entrée ET sortie (exemple : un écran tactile = entrée via le toucher + sortie via l'affichage).
  • À retenir : Le processeur ne peut pas traiter des données directement depuis le disque dur ; elles doivent d'abord passer par la mémoire vive.

📖 Vocabulaire essentiel

  • Flux de données : Cheminement des informations entre les différents composants de l'ordinateur.
  • Unité centrale (UC) : Ensemble constitué du processeur et de la mémoire vive.
  • Bus informatique : Ensemble de liaisons qui permettent le transfert de données entre les composants.
  • Périphérique d'entrée/sortie : Appareil qui peut à la fois envoyer et recevoir des données.
  • Traitement : Opérations effectuées par le processeur (calculs, comparaisons, décisions).
