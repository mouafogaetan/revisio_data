Leçon 1 : Caractéristiques des composants et périphériques

Informatique - Terminale A (Littéraire) · Architecture et protection de l'ordinateur

📌 Éléments clés à retenir

1. Composants internes (logés dans l'unité centrale)

Ce sont les éléments physiques indispensables au fonctionnement de l'ordinateur. On les trouve à l'intérieur du boîtier.

  • Processeur (CPU) : Cerveau de l'ordinateur, il exécute les instructions et traite les données. Sa vitesse se mesure en GHz (gigahertz).
  • Mémoire vive (RAM) : Mémoire temporaire qui stocke les données en cours d'utilisation. Plus la RAM est grande, plus l'ordinateur est réactif (ex: 4 Go, 8 Go, 16 Go).
  • Disque dur (HDD ou SSD) : Mémoire de masse (stockage permanent). Le SSD est plus rapide et silencieux que le HDD. Capacité en Go ou To.
  • Carte mère : Grande carte qui relie tous les composants (processeur, RAM, disques, périphériques). C'est le "système nerveux".
Figure 1 : Architecture interne simplifiée (carte mère, CPU, RAM, disque).

2. Périphériques externes : Entrée / Sortie

Appareils connectés à l'ordinateur pour interagir avec lui. On distingue trois familles :

🖱️ Périphériques d'entrée

Permettent d'envoyer des informations à l'ordinateur.
Exemples : Clavier, souris, scanner, microphone, webcam.

🖥️ Périphériques de sortie

Reçoivent des informations depuis l'ordinateur pour les restituer à l'utilisateur.
Exemples : Écran (moniteur), imprimante, haut-parleurs, casque.

🔄 Périphériques mixtes (entrée/sortie)

Assurent à la fois la réception et l'envoi de données.
Exemples : Clé USB, disque dur externe, carte réseau, écran tactile.

Figure 2 : Flux des données entre l'ordinateur et les périphériques.

3. Lire une fiche technique (examen session 2024)

Une fiche technique résume les caractéristiques essentielles d'un ordinateur. À savoir identifier :

  • Processeur : modèle et fréquence (ex: Intel Core i5, 2.4 GHz).
  • Mémoire RAM : capacité (ex: 8 Go DDR4).
  • Stockage : type (SSD/HDD) et capacité (ex: SSD 256 Go).
  • Périphériques inclus : clavier, souris, écran, etc.
💡 Astuce examen : On peut vous demander : "Identifiez le composant qui stocke les données à long terme" → réponse : disque dur. "Quel composant exécute les programmes ?" → processeur.

💡 Exemples de compréhension

Exemple 1 : Distinguer composant interne / périphérique externe

Question type Bac : "Parmi les éléments suivants, lesquels sont des composants internes ? Clavier, RAM, imprimante, disque dur, souris, processeur."

Résolution : Composants internes = RAM, disque dur, processeur. Périphériques externes = clavier, imprimante, souris.
La RAM et le processeur sont situés sur la carte mère, le disque dur dans le boîtier. Le clavier et la souris sont reliés par câble ou Bluetooth.

💬 Mémo technique : "Ce qui se trouve à l'intérieur du boîtier (sauf alimentation) = composant interne. Ce qui se branche à l'extérieur = périphérique."

Exemple 2 : Rôle de chaque périphérique

"Lors d'une visioconférence, quels périphériques d'entrée et de sortie sont utilisés ?"

Explication :
• Entrée : Webcam (capture vidéo), microphone (capture son), éventuellement clavier/souris pour contrôler.
• Sortie : Écran (affiche les participants), haut-parleurs/casque (restituent le son).
Ces échanges montrent la complémentarité entrée/sortie.

Exemple 3 : Identifier sur une fiche technique

Voici une fiche technique simplifiée :
Ordinateur X : Processeur Intel Core i7-1165G7 (2,8 GHz), RAM 16 Go, SSD 512 Go, écran 15 pouces, souris optique incluse.
Question : "Quelle est la capacité de stockage permanent ?"

Réponse : SSD 512 Go (car le SSD est le support de stockage permanent, contrairement à la RAM qui est volatile).
La RAM (16 Go) sert à l'exécution des programmes en cours.

📘 Différence clé : RAM = mémoire vive (perdue à l'extinction). Disque dur/SSD = mémoire de masse (conservée).

⚠️ Pièges courants / À ne pas confondre

  • Confondre RAM et disque dur : La RAM est volatile et rapide ; le disque dur est permanent mais plus lent. Ne pas dire "le disque dur est la mémoire vive".
  • Périphériques d'entrée vs sortie : L'écran tactile est un périphérique mixte (il affiche et reçoit les touches). L'imprimante est uniquement sortie.
  • Carte mère = processeur ? Non, la carte mère accueille le processeur, mais ce sont deux éléments distincts.
  • Oublier les unités : En examen, on peut vous demander "Go, GHz, Mo". Savoir que 1 Go = 1024 Mo, et que GHz mesure la vitesse du processeur.
