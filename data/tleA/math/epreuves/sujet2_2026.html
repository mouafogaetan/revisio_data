📖 Exercice 1 : Équations, inéquations et systèmes (7 points)

1.1 Équation du second degré (2 points)

Résoudre dans ℝ l'équation suivante : 2x² - 5x + 2 = 0

✅ Correction

On calcule le discriminant Δ = b² - 4ac = (-5)² - 4×2×2 = 25 - 16 = 9.

Δ = 9 > 0, donc l'équation admet deux racines réelles distinctes :

x₁ = (5 - √9)/(2×2) = (5 - 3)/4 = 2/4 = 1/2

x₂ = (5 + √9)/(2×2) = (5 + 3)/4 = 8/4 = 2

L'ensemble solution est S = {1/2 ; 2}.

1.2 Équation polynomiale de degré 3 (2 points)

Soit P(x) = 2x³ - 3x² - 17x + 30.
1. Montrer que 2 est une racine évidente de P(x).
2. Factoriser P(x) par (x - 2) puis résoudre l'équation P(x) = 0.

✅ Correction

1. P(2) = 2×8 - 3×4 - 17×2 + 30 = 16 - 12 - 34 + 30 = 0. Donc 2 est bien une racine de P.

2. La factorisation donne : P(x) = (x - 2)(2x² + x - 15).

On résout 2x² + x - 15 = 0 : Δ = 1 - 4×2×(-15) = 1 + 120 = 121, √Δ = 11.

x₁ = (-1 - 11)/(4) = -12/4 = -3 ; x₂ = (-1 + 11)/(4) = 10/4 = 5/2.

L'ensemble solution est S = {-3 ; 2 ; 5/2}.

1.3 Système linéaire à trois inconnues (3 points)

Résoudre le système suivant par la méthode du pivot de Gauss :

⎧ x + y + z = 6
⎨ 2x - y + z = 3
⎩ x + 2y - z = 3

✅ Correction

On écrit la matrice augmentée :

L₁ : [1 1 1 | 6]
L₂ : [2 -1 1 | 3]
L₃ : [1 2 -1 | 3]

L₂ ← L₂ - 2L₁ : [0 -3 -1 | -9]
L₃ ← L₃ - L₁ : [0 1 -2 | -3]

On obtient : y = 2, puis x = 1, z = 3.

La solution est le triplet (x ; y ; z) = (1 ; 2 ; 3).

🎲 Exercice 2 : Dénombrement et probabilités (6 points)

2.1 Combinaisons (2 points)

Une classe de Terminale A compte 25 élèves. On doit choisir 5 élèves pour former une commission.
1. Combien de commissions différentes peut-on former ?
2. Si la classe compte 15 filles et 10 garçons, combien de commissions sont composées uniquement de filles ?

✅ Correction

1. Le nombre de commissions est le nombre de combinaisons de 5 élèves parmi 25 :

C(25,5) = 25!/(5!×20!) = (25×24×23×22×21)/(5×4×3×2×1) = 53130/120 = 53 130 commissions.

2. Commissions uniquement de filles : C(15,5) = 15!/(5!×10!) = (15×14×13×12×11)/(120) = 360360/120 = 3 003 commissions.

2.2 Arrangements (2 points)

On dispose des chiffres 1, 2, 3, 4, 5.
1. Combien de nombres à 3 chiffres distincts peut-on former ?
2. Combien de ces nombres sont pairs ?

✅ Correction

1. Le nombre d'arrangements de 3 chiffres parmi 5 est : A(5,3) = 5×4×3 = 60 nombres.

2. Pour qu'un nombre soit pair, le dernier chiffre doit être pair (2 ou 4).

Cas 1 : dernier chiffre = 2 → choix des 2 premiers : A(4,2) = 4×3 = 12 nombres.
Cas 2 : dernier chiffre = 4 → idem : 12 nombres.

Total = 12 + 12 = 24 nombres pairs.

2.3 Calcul de probabilité (2 points)

On lance un dé équilibré à 6 faces deux fois de suite.
Quelle est la probabilité d'obtenir une somme supérieure ou égale à 10 ?

✅ Correction

L'univers Ω contient 6×6 = 36 issues équiprobables.

Les couples donnant une somme ≥ 10 sont :
(4,6), (5,5), (5,6), (6,4), (6,5), (6,6) → 6 issues.

La probabilité est donc P = 6/36 = 1/6.

📈 Exercice 3 : Étude de fonctions (7 points)

3.1 Domaine de définition et limites (2 points)

Soit la fonction f définie par f(x) = (2x² - 3x + 1)/(x - 1).
1. Déterminer le domaine de définition Df de f.
2. Calculer les limites de f aux bornes de son domaine.

✅ Correction

1. Le dénominateur s'annule pour x = 1. Donc Df = ℝ \ {1} = ]-∞ ; 1[ ∪ ]1 ; +∞[.

2. Limites :

En ±∞ : lim f(x) = lim (2x²/x) = lim 2x = -∞ en -∞ et +∞ en +∞.

En 1⁻ : x → 1⁻, x-1 → 0⁻, numérateur → 2-3+1=0, forme indéterminée. Factorisation : 2x²-3x+1 = (x-1)(2x-1).

f(x) = 2x-1 pour x≠1, donc lim f(x) en 1 = 1. La fonction est prolongeable par continuité en 1.

3.2 Dérivée et variations (3 points)

Soit g(x) = x² - 4x + 3 sur ℝ.
1. Calculer g'(x) et étudier son signe.
2. Dresser le tableau de variations de g.

✅ Correction

1. g'(x) = 2x - 4.

Signe de g'(x) :
g'(x) = 0 ⇔ 2x - 4 = 0 ⇔ x = 2.
g'(x) < 0 pour x < 2 (fonction décroissante).
g'(x) > 0 pour x > 2 (fonction croissante).

2. Tableau de variations :

g(2) = 4 - 8 + 3 = -1 (minimum).

g est décroissante sur ]-∞ ; 2] et croissante sur [2 ; +∞[. g admet un minimum de -1 en x=2.

3.3 Problème de synthèse (2 points)

Un agriculteur veut clôturer un champ rectangulaire de 400 m². Le terrain est bordé par une rivière sur un côté (qui ne nécessite pas de clôture). Les trois autres côtés nécessitent une clôture.
Déterminer les dimensions du champ pour minimiser la longueur de clôture utilisée.

✅ Correction

Soit x la largeur perpendiculaire à la rivière et y la longueur parallèle. Aire : x × y = 400.

Longueur de clôture : L(x) = x + x + y = 2x + y.

Comme y = 400/x, on a L(x) = 2x + 400/x pour x > 0.

L'(x) = 2 - 400/x². L'(x) = 0 ⇔ 2 = 400/x² ⇔ x² = 200 ⇔ x = √200 = 10√2 ≈ 14,14 m.

y = 400/(10√2) = 40/√2 = 20√2 ≈ 28,28 m.

Les dimensions optimales sont : largeur 10√2 m, longueur 20√2 m.
