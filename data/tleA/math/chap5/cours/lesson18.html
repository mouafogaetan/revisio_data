Leçon 18 : Équations avec logarithmes

Mathématiques · Terminale A (Littéraire) · Baccalauréat 2026

📌 Éléments clés à retenir

1. Domaine de définition (condition d'existence)

La fonction logarithme népérien $\ln(x)$ est définie uniquement pour $x > 0$. Dans une équation comme $\ln(ax+b) = c$, on doit donc imposer $ax+b > 0$ avant toute résolution.

  • Si l'équation contient $\ln(u(x))$, la condition est $u(x) > 0$.
  • Les solutions trouvées doivent obligatoirement vérifier cette condition.
  • Formule fondamentale : $\ln(a) = \ln(b) \iff a = b > 0$.

2. Propriétés algébriques du logarithme

Pour tous réels $a>0$ et $b>0$, et pour tout entier $n$ :

  • $\ln(ab) = \ln(a) + \ln(b)$
  • $\ln\left(\frac{a}{b}\right) = \ln(a) - \ln(b)$
  • $\ln(a^n) = n \ln(a)$
  • $\ln(1) = 0$ et $\ln(e) = 1$
  • Équation type : $\ln(ax+b) = c \iff ax+b = e^c$ (après vérification $ax+b>0$).

3. Résolution d'équations polynomiales en $\ln x$

Lorsque l'équation contient $\ln x$, $\ln^2 x$, etc., on utilise le changement de variable $X = \ln x$ (avec $x>0$). On résout l'équation polynomiale en $X$, puis on revient à $x = e^{X}$.

Exemple : $\ln^3 x - \ln^2 x - 22\ln x + 40 = 0$ → poser $X = \ln x$.

💡 Exemples de compréhension

Exemple 1 : Équation simple $\ln(2x-1) = 3$

Résoudre dans $\mathbb{R}$ : $\ln(2x-1) = 3$.

Étape 1 : Condition d'existence. $2x-1 > 0 \iff x > \frac{1}{2}$.
Étape 2 : Application de la définition. $\ln(2x-1)=3 \iff 2x-1 = e^{3}$.
Étape 3 : Résolution. $2x = e^{3}+1 \iff x = \frac{e^{3}+1}{2}$.
Étape 4 : Vérification. $\frac{e^{3}+1}{2} \approx \frac{20,0855+1}{2} \approx 10,54 > \frac{1}{2}$, donc solution valide.
💬 Astuce : Ne jamais oublier de vérifier la condition $2x-1>0$ avant d'écrire la solution.

Exemple 2 : Équation avec changement de variable inspirée du Bac 2023

Résoudre dans $]0; +\infty[$ : $\ln^3 x - \ln^2 x - 22\ln x + 40 = 0$ (Session 2023, Exercice 1).

1. Changement de variable : Posons $X = \ln x$ (avec $x>0$). L'équation devient :
$X^3 - X^2 - 22X + 40 = 0$.
2. Recherche de racine évidente : $X=4$ : $64 - 16 - 88 + 40 = 0$ → $0=0$, donc $X=4$ est racine.
3. Factorisation : $(X-4)(X^2 + 3X -10)=0$.
4. Résolution du trinôme : $\Delta = 9+40=49$, $\sqrt{\Delta}=7$. Racines : $X = \frac{-3 \pm 7}{2}$ soit $X=2$ et $X=-5$.
5. Solutions en $X$ : $X_1 = 4$, $X_2 = 2$, $X_3 = -5$.
6. Retour à $x$ : $\ln x = 4 \Rightarrow x = e^{4}$ ; $\ln x = 2 \Rightarrow x = e^{2}$ ; $\ln x = -5 \Rightarrow x = e^{-5}$.
7. Ensemble solution : $S = \{ e^{-5} ; e^{2} ; e^{4} \}$.
💡 Penser à vérifier que les solutions appartiennent bien à l'ensemble de définition $]0;+\infty[$ (c'est le cas ici).

Exemple 3 : Équation avec somme de logarithmes

Résoudre $\ln(x-1) + \ln(x+2) = \ln(4)$.

Condition d'existence : $x-1 > 0$ et $x+2 > 0$ → $x>1$.
Utilisation de la propriété : $\ln[(x-1)(x+2)] = \ln(4)$.
Simplification : $(x-1)(x+2) = 4$ → $x^2 + x -2 = 4$ → $x^2 + x -6 = 0$.
Racines : $\Delta = 1+24=25$, $x_1 = \frac{-1-5}{2} = -3$ (rejeté car $x>1$), $x_2 = \frac{-1+5}{2}=2$.
Solution unique : $S = \{2\}$.
⚠️ Attention : On ne peut pas appliquer $\ln(a)+\ln(b)=\ln(ab)$ sans avoir vérifié $a>0$ et $b>0$ au préalable.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente 1 : Oublier le domaine de définition. Une solution trouvée algébriquement peut être négative ou nulle et doit être rejetée.
  • Erreur fréquente 2 : Confondre $\ln(a+b)$ et $\ln a + \ln b$. Il n'y a aucune simplification pour $\ln(a+b)$.
  • Erreur fréquente 3 : Appliquer $e^{\ln u} = u$ sans avoir vérifié $u>0$. La fonction exponentielle permet d'éliminer le logarithme mais la condition reste impérative.
  • Attention spéciale : Dans un changement de variable $X = \ln x$, ne pas oublier que $x = e^{X}$ et que $x>0$ est automatiquement vérifié.
