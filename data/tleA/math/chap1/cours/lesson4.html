Leçon 4 : Équations du type P(x) = 0 avec P factorisable

Mathématiques · Terminale A (Littéraire) · Baccalauréat (Sessions 2023, 2025)

📌 Éléments clés à retenir

Méthode générale : résoudre P(x) = 0 (degré ≥ 3)

Pour une équation polynomiale de degré 3 ou plus, on cherche d'abord une racine évidente (souvent parmi -2, -1, 0, 1, 2). Ensuite, on factorise le polynôme par (x - a) où a est cette racine. On obtient un produit : (x - a) × Q(x) = 0. Enfin, on résout Q(x) = 0 (équation du second degré généralement).

  • Étape 1 : Tester les valeurs simples (diviseurs du terme constant) pour trouver une racine évidente x = a.
  • Étape 2 : Effectuer la division euclidienne ou identifier les coefficients pour factoriser par (x - a).
  • Étape 3 : Résoudre Q(x) = 0 (second degré → discriminant Δ).
  • Étape 4 : Donner l’ensemble des solutions (union des solutions de chaque facteur).
💡 Astuce examen : Dans les sujets du baccalauréat (Cameroun), les polynômes sont choisis pour que la racine évidente soit un entier simple (-2, 1, 2, 3…). Testez systématiquement ces valeurs !

Factorisation et division par (x - a)

Si P(a) = 0, alors P(x) = (x - a) × Q(x), avec Q(x) un polynôme de degré inférieur d’une unité. Pour déterminer Q(x), on peut :

  • Utiliser la division synthétique (méthode de Horner).
  • Développer (x - a)(bx² + cx + d) et identifier les coefficients avec P(x).
  • Dans les cas simples, factoriser directement par regroupement.

Exemple : Pour P(x) = x³ - x² - 2x + 2, on trouve P(1)=0 ⇒ factorisation par (x-1). On obtient (x-1)(x²-2).

Résolution de Q(x) = 0 (second degré)

Après factorisation, le polynôme Q(x) est généralement du second degré : Q(x) = Ax² + Bx + C. On calcule le discriminant : Δ = B² - 4AC.

  • Si Δ > 0 : deux racines réelles x₁ = \frac{-B - \sqrt{Δ}}{2A}, x₂ = \frac{-B + \sqrt{Δ}}{2A}.
  • Si Δ = 0 : une racine double x₀ = \frac{-B}{2A}.
  • Si Δ < 0 : pas de solution réelle (dans ℝ).

Les solutions finales de P(x)=0 sont la racine évidente a et les racines de Q(x) (si elles existent).

💡 Exemples de compréhension

Examen 2023 (Cameroun) – Équation factorisable

Résoudre dans ℝ l’équation : x³ - x² - 2x + 2 = 0.

🔍 Résolution détaillée :
• Recherche d’une racine évidente : Testons x = 1 : 1³ - 1² - 2×1 + 2 = 1 - 1 - 2 + 2 = 0. Donc 1 est racine.
• Factorisation par (x - 1) : On écrit x³ - x² - 2x + 2 = (x - 1)(ax² + bx + c). En développant : ax³ + bx² + cx - ax² - bx - c = ax³ + (b - a)x² + (c - b)x - c.
  Par identification : a = 1 ; b - a = -1 ⇒ b - 1 = -1 ⇒ b = 0 ; c - b = -2 ⇒ c - 0 = -2 ⇒ c = -2 ; vérifions la constante : -c = 2 ⇒ -(-2)=2 OK.
  Donc P(x) = (x - 1)(x² - 2).
• Résoudre (x - 1)(x² - 2) = 0 → x - 1 = 0 ou x² - 2 = 0.
  x = 1 ou x = \sqrt{2} ou x = -\sqrt{2}.
• Ensemble solution : S = \left\{ -\sqrt{2},\ 1,\ \sqrt{2} \right\}.
📌 À retenir : La factorisation par (x - 1) transforme un polynôme de degré 3 en produit d’un facteur linéaire et d’un trinôme du second degré facile à résoudre.

Examen 2025 (Cameroun) – Équation de degré 3 classique

Résoudre dans ℝ : 2x³ - 3x² - 17x + 30 = 0.

🔍 Résolution pas à pas :
• Racine évidente : Testons x = 2 : 2×8 - 3×4 - 17×2 + 30 = 16 - 12 - 34 + 30 = 0. Donc 2 est racine.
• Factorisation : On pose 2x³ - 3x² - 17x + 30 = (x - 2)(Ax² + Bx + C).
  Développons : Ax³ + Bx² + Cx - 2Ax² - 2Bx - 2C = Ax³ + (B - 2A)x² + (C - 2B)x - 2C.
  Identification : A = 2 ; B - 2A = -3 ⇒ B - 4 = -3 ⇒ B = 1 ; C - 2B = -17 ⇒ C - 2 = -17 ⇒ C = -15 ; constante -2C = 30 ⇒ -2×(-15)=30 OK.
  Donc P(x) = (x - 2)(2x² + x - 15).
• Résoudre l’équation produit : x - 2 = 0 ou 2x² + x - 15 = 0.
  Pour le trinôme : Δ = 1² - 4×2×(-15) = 1 + 120 = 121 → √Δ = 11.
  Racines : x₁ = \frac{-1 - 11}{4} = \frac{-12}{4} = -3 ; x₂ = \frac{-1 + 11}{4} = \frac{10}{4} = \frac{5}{2}.
• Ensemble solution : S = \left\{ -3,\ 2,\ \frac{5}{2} \right\}.
🧠 Pensez-y : Parfois la racine évidente est une fraction simple ou un entier négatif. N’oubliez pas de tester 2, -2, 3, -3, 1, -1.

Exemple supplémentaire – Cas d’une racine double

Résoudre x³ - 4x² + 5x - 2 = 0.

Démarche :
• Test de x = 1 : 1 - 4 + 5 - 2 = 0 ⇒ 1 est racine.
• Factorisation par (x - 1) : on obtient (x - 1)(x² - 3x + 2).
• Trinôme x² - 3x + 2 : Δ = 9 - 8 = 1 > 0 → racines \frac{3 ± 1}{2} soit 2 et 1.
• Finalement P(x) = (x - 1)(x - 1)(x - 2) = (x - 1)²(x - 2).
• Solutions : S = \{1 ; 2\} (1 est racine double).
✨ Remarque : Une racine double apparaît quand la factorisation donne un carré. Dans l’ensemble des solutions, on la note une seule fois.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de tester les valeurs négatives pour la racine évidente. Parfois la racine est x = -2 ou x = -3.
  • Attention : Vérifier que la factorisation est complète. Si Q(x) est encore factorisable (par exemple une différence de carrés), il faut poursuivre.
  • Piège : Ne pas confondre la division par (x - a) avec une simplification abusive. On ne peut pas simplifier (x - a) si on ne sait pas qu’il est facteur commun.
  • À surveiller : Ne pas oublier les solutions issues du facteur (x - a) quand on résout Q(x)=0. L’ensemble solution est l’union des deux.
  • Conseil pour l’examen : Toujours présenter le tableau de factorisation ou l’identification des coefficients. Les correcteurs apprécient la clarté.
