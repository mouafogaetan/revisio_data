Leçon 5 : Systèmes linéaires (3 équations, 3 inconnues)

Mathématiques · Terminale A (Littéraire) · Pivot de Gauss & substitution

📌 Éléments clés à retenir

🔹 Qu'est-ce qu'un système linéaire de 3 équations à 3 inconnues ?

C’est un ensemble de trois équations du premier degré avec trois inconnues (souvent notées \(x, y, z\)). L'objectif est de trouver l'unique triplet \((x ; y ; z)\) qui vérifie les trois équations simultanément.

  • Forme générale : \[ \begin{cases} a_1x + b_1y + c_1z = d_1 \\ a_2x + b_2y + c_2z = d_2 \\ a_3x + b_3y + c_3z = d_3 \end{cases} \]
  • Solution unique si les équations sont "indépendantes".
  • Deux méthodes principales : substitution et pivot de Gauss (celle-ci est recommandée en Terminale A).

🔹 Méthode du pivot de Gauss (étape par étape)

Elle consiste à transformer le système en un système triangulaire équivalent, plus facile à résoudre. On élimine progressivement les inconnues en combinant les équations.

  • Étape 1 : Choisir une équation comme pivot (généralement la première) et éliminer \(x\) des autres lignes.
  • Étape 2 : Utiliser la nouvelle équation (sans \(x\)) pour éliminer \(y\) de la troisième ligne.
  • Étape 3 : Résoudre le système triangulaire obtenu :
    - D'abord \(z\) (dernière équation) → puis \(y\) → enfin \(x\).
  • Étape 4 : Vérifier la solution dans le système d'origine (indispensable !).
💡 Astuce mnémotechnique : « Élimine x, puis élimine y, remonte pour trouver z, y, x ».

🔹 Méthode par substitution (alternative)

On exprime une inconnue en fonction des autres dans une équation, puis on la remplace dans les deux autres équations. On obtient un système 2×2, puis on termine par substitution simple.

  • Utile quand une équation donne directement une inconnue (exemple : \(x = 5 - y\)).
  • Attention : peut être plus longue que le pivot de Gauss pour des coefficients compliqués.

💡 Exemples types (inspirés des examens 2024 & 2025)

Exemple 1 : Résolution par pivot de Gauss (session 2025)

Résoudre le système suivant : \[ \begin{cases} x + y - z = 0 \quad &(L_1)\\ 2x - y + 3z = 9 \quad &(L_2)\\ -x + 2y + z = 2 \quad &(L_3) \end{cases} \]

Correction détaillée (pivot de Gauss) :
1️⃣ On conserve \(L_1\) comme pivot. On élimine \(x\) dans \(L_2\) et \(L_3\) :
  • \(L_2 \leftarrow L_2 - 2L_1\) → \((2x-2x) + (-y-2y?) \) Attention : calcul précis : \(2x - 2x = 0\) ; \(-y - 2(-y?)\) Non : \(L_2 - 2L_1 = (2x - x) \) Non: reprenons proprement :
  \(L_2 - 2L_1 : (2x - 2x) + (-y - 2y?) \) plutôt : coefficient de y: \(-y - 2(1)y = -y - 2y = -3y\). Pour z: \(3z - 2(-z) = 3z +2z = 5z\). Second membre: \(9 - 2×0 = 9\).
  Donc nouvelle \(L_2\) : \(-3y + 5z = 9\).
  • \(L_3 \leftarrow L_3 + L_1\) (car \( -x + x = 0\)) → \( (2y + y) = 3y\) ; \( (z + (-z)) = 0\) ; second membre: \(2 + 0 = 2\). Donc \(3y = 2 \Rightarrow y = \frac{2}{3}\).
2️⃣ On remplace \(y = \frac{2}{3}\) dans \(L_2'\) : \(-3×\frac{2}{3} + 5z = 9 \Rightarrow -2 + 5z = 9 \Rightarrow 5z = 11 \Rightarrow z = \frac{11}{5}\).
3️⃣ On utilise \(L_1\) : \(x + \frac{2}{3} - \frac{11}{5} = 0 \Rightarrow x = \frac{11}{5} - \frac{2}{3} = \frac{33 - 10}{15} = \frac{23}{15}\).
4️⃣ Solution unique : \(\displaystyle \left( \frac{23}{15} ; \frac{2}{3} ; \frac{11}{5} \right)\). Vérification rapide : OK.
💬 Conseil examen : Écrivez toujours les opérations sur les lignes (ex: \(L_2 \leftarrow L_2 - 2L_1\)). Cela évite les erreurs de calcul et facilite la relecture.

Exemple 2 : Problème de mise en équation (inspiré session 2024 – ferme)

Dans une ferme, il y a 30 animaux (poules, chèvres, vaches). Le nombre de chèvres est le double du nombre de vaches. Le nombre total de pattes est 80 (une poule a 2 pattes, une chèvre 4, une vache 4). Déterminer le nombre de chaque animal.

Mise en équation : Soit \(p\) = poules, \(c\) = chèvres, \(v\) = vaches.
\[ \begin{cases} p + c + v = 30 \quad (1) \\ c = 2v \quad (2) \\ 2p + 4c + 4v = 80 \quad (3) \end{cases} \] On remplace \(c\) par \(2v\) dans (1) et (3) :
(1') : \(p + 2v + v = 30 \Rightarrow p + 3v = 30\).
(3') : \(2p + 4×(2v) + 4v = 2p + 8v + 4v = 2p + 12v = 80\).
On divise (3') par 2 : \(p + 6v = 40\).
On soustrait (1') de cette nouvelle équation : \((p+6v) - (p+3v) = 40 - 30 \Rightarrow 3v = 10 \Rightarrow v = \frac{10}{3}\) → impossible car v doit être entier.
Ceci montre une incohérence ? En réalité les énoncés officiels sont cohérents. Avec \(c=2v\), on trouve \(v\) non entier. On peut corriger l'exemple : supposons que le nombre total de pattes est 92 (pour que cela tombe juste).
Version corrigée : \(2p+4c+4v=92\) alors \(p+6v=46\). Avec \(p+3v=30\) → \(3v=16 \Rightarrow v\) non entier. Autre jeu :
Pour rester réaliste, prenons \(p=12, c=12, v=6\) → total 30 animaux, chèvres = 2×vaches (12=2×6), pattes: 12×2 + 12×4 + 6×4 = 24+48+24=96. Donc système adapté : \[ \begin{cases} p+c+v=30 \\ c=2v \\ 2p+4c+4v=96 \end{cases} \] ➜ Solution : \(v=6, c=12, p=12\). La résolution par substitution ou pivot mène au triplet \((12;12;6)\).
📐 Réflexe : Avant de résoudre, vérifier la cohérence des unités (entiers positifs pour des animaux). Lire l'énoncé en détail.

⚠️ Pièges courants & astuces pour l'examen

  • Erreur fréquente : Oublier de multiplier TOUS les termes d’une équation quand on fait une combinaison. Exemple : \(L_2 \leftarrow L_2 - 2L_1\) ⇒ multiplier chaque coefficient de \(L_1\) par 2, y compris le second membre.
  • Attention aux signes : \( - (a - b) = -a + b\). Une erreur de signe peut tout fausser.
  • Ne pas vérifier la solution : On gagne des points en remplaçant rapidement dans le système initial. Cela prend 30 secondes et évite les pertes de points.
  • Confondre substitution et pivot : Pour les systèmes 3×3, la méthode du pivot est systématique et bien adaptée aux coefficients décimaux ou fractionnaires.
  • Conseil rédaction : Présentez toujours le système triangulaire intermédiaire et les opérations sur les lignes. Cela montre votre raisonnement.
