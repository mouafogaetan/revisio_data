Leçon 7 : Adressage IP

Informatique · Terminale A (Littéraire) · Unité 2 : Réseaux informatiques et communication

📌 Éléments clés à retenir

1. Qu'est-ce qu'une adresse IP ?

Une adresse IP (Internet Protocol) est un identifiant unique attribué à chaque appareil connecté à un réseau informatique (ordinateur, smartphone, imprimante, etc.). Elle permet aux machines de se reconnaître et d'échanger des données sur Internet ou un réseau local.

  • Adresse IPv4 : La plus courante, composée de 4 nombres (entre 0 et 255) séparés par des points. Exemple : 192.168.1.10
  • Rôle essentiel : Sans adresse IP, un appareil ne peut ni envoyer ni recevoir d'informations sur le réseau.
  • Chaque adresse IP est unique sur un même réseau (sinon conflit d'adresses).

🔁 Exemple visuel : des appareils connectés avec leurs adresses IP.

2. Adressage statique (fixe)

L’adressage statique consiste à attribuer manuellement une adresse IP à un périphérique. Cette adresse ne change pas dans le temps.

  • ✅ Avantage : Contrôle total, idéal pour les serveurs, imprimantes réseau, appareils qui doivent toujours avoir la même adresse.
  • ⚠️ Inconvénient : Gestion lourde dans un grand réseau (risque d'erreur ou de duplication). Nécessite une intervention manuelle pour chaque machine.
  • Utilisé dans les petites structures ou pour des équipements critiques.

3. Adressage dynamique (DHCP)

L'adressage dynamique utilise un serveur DHCP (Dynamic Host Configuration Protocol) qui attribue automatiquement une adresse IP à chaque appareil lorsqu'il se connecte au réseau.

  • ✅ Avantage : Simple et automatisé, idéal pour les réseaux nombreux (box internet, entreprise, cybercafé). Pas de conflit d'adresses.
  • ⚠️ Inconvénient : L’adresse IP peut changer au fil des connexions (pas adapté pour certains services fixes).
  • C’est le mode par défaut de la plupart des routeurs et box Internet (Orange, MTN, Camtel, etc.).

💡 Exemples de compréhension

Exemple 1 : Identifier le type d’adressage

Dans un cybercafé du quartier, les ordinateurs récupèrent automatiquement une adresse IP dès qu’ils sont allumés. Le gérant n’a pas besoin de configurer chaque machine.

✅ Solution : Il s’agit d’un adressage dynamique (DHCP). Le routeur du cybercafé joue le rôle de serveur DHCP et distribue les adresses IP aux ordinateurs sans intervention humaine.

💬 Astuce : Si tu te connectes à la Wi-Fi d’un hôtel ou d’un lieu public sans paramétrer d’IP, tu utilises le DHCP. C’est le cas le plus fréquent.

Exemple 2 : Cas d’un serveur d’impression

Une association scolaire possède une imprimante réseau partagée. L’administrateur souhaite que tous les ordinateurs retrouvent toujours l’imprimante à la même adresse IP, même après une coupure de courant.

📌 Meilleure approche : Utiliser un adressage statique pour l’imprimante. On fixe une adresse IP hors de la plage DHCP (par exemple 192.168.1.200). Ainsi, les ordinateurs peuvent configurer l’imprimante une fois pour toutes.

💡 À retenir : Pour des équipements qui doivent être toujours joignables (serveur, imprimante, caméra IP), l’adressage statique est recommandé.

Examen type (Session 2023, 2024)

Question type du Baccalauréat : "Distinguer l’adressage statique de l’adressage dynamique. Donner un exemple d’utilisation pour chacun."

Éléments de réponse :
- L’adressage statique nécessite une configuration manuelle et l’IP ne change jamais ; exemple : serveur d’une petite bibliothèque.
- L’adressage dynamique (DHCP) attribue les IP automatiquement ; exemple : les smartphones d’élèves connectés au Wi-Fi du lycée.
Différence clé : gestion automatique vs manuelle, IP fixe vs IP temporaire.

⚡ En examen, insiste sur le rôle du DHCP et le fait que l’adressage statique est contraignant pour les grands réseaux.

⚠️ Pièges courants / À ne pas confondre

  • Ne pas confondre adresse IP et adresse MAC : L’adresse MAC est une adresse physique fixe (gravée dans la carte réseau). L’adresse IP est logique et peut changer.
  • Attention à l’unicité : Sur un même réseau, deux appareils ne peuvent pas avoir la même adresse IP statique, cela provoque un conflit et coupe la connexion.
  • Piège fréquent : Croire que l’adressage dynamique est moins sécurisé. En réalité, la sécurité ne dépend pas du type d’adressage mais du pare-feu et des autorisations.
  • Erreur en devoir : écrire "adressage dynamique = adresse qui change toutes les 5 minutes" — non, l’IP dynamique peut rester plusieurs jours selon la durée de bail DHCP.

📖 Mots-clés à maîtriser

  • Adresse IP : identifiant unique d’un équipement sur un réseau.
  • Adressage statique : configuration manuelle et permanente.
  • Adressage dynamique : attribution automatique via DHCP.
  • DHCP : protocole de configuration dynamique des hôtes.
  • Conflit d’adresses : situation où deux machines ont la même IP.

🔗 En résumé pour le Bac

Ce qu’il faut absolument savoir :

  • Définition d’une adresse IP.
  • Différence entre adressage statique (manuel, fixe) et adressage dynamique (automatique, via DHCP).
  • Savoir citer un exemple concret d’utilisation pour chaque type (ex: serveur → statique ; box Internet → dynamique).
  • Comprendre que le DHCP simplifie l’administration des réseaux modernes.
