📄 Leçon 12 : Insertion d'objets dans un document

Informatique · Terminale A (Littéraire) · Traitement de texte

📌 Éléments clés à retenir

1. Qu'est-ce qu'un « objet » dans un traitement de texte ?

En bureautique, un objet est un élément qui n'est pas du texte brut, mais qui peut être inséré dans un document pour enrichir son contenu : images, tableaux, formes géométriques, graphiques, zones de texte, etc. Ces objets permettent de rendre un document plus visuel et professionnel.

  • Images : photos, illustrations, captures d'écran, logos.
  • Tableaux : organisation des données en lignes et colonnes.
  • Formes : flèches, cercles, rectangles, schémas.
  • Graphiques : diagrammes (secteurs, barres, courbes).

2. Procédure générale pour insérer un objet

Dans la plupart des logiciels de traitement de texte (Microsoft Word, LibreOffice Writer, Google Docs), la démarche suit un modèle similaire.

📌 Étapes communes :

  1. Placer le curseur à l'endroit où l'objet doit apparaître.
  2. Cliquer sur l'onglet « Insertion » (ou menu « Insérer »).
  3. Choisir le type d'objet souhaité : Image, Tableau, Forme, Graphique, etc.
  4. Sélectionner la source (fichier image, nombre de lignes/colonnes, type de forme) et valider.
  5. Ajuster l'objet (taille, position, habillage) après insertion.
💡 Raccourci pratique : Ctrl + V permet de coller une image copiée depuis le presse-papier (autre document, navigateur…).

3. Insertion d’une image pas à pas

Pour ajouter une image depuis un fichier ou depuis une source en ligne :

  • Étape 1 : Menu Insertion → Image → À partir d’un fichier (ou Image en ligne).
  • Étape 2 : Parcourir le dossier et sélectionner l’image.
  • Étape 3 : Cliquer sur « Insérer ». L’image apparaît dans le document.
  • Étape 4 : Redimensionner avec les poignées situées aux coins (maintenir Maj pour garder les proportions).
Illustration : menu Insertion → Image (simulation d’interface bureautique).

4. Insertion d’un tableau – méthode pratique

Un tableau permet de structurer des informations (notes, planning, comparatifs). Voici la procédure standard :

  • Via la grille visuelle : Insertion → Tableau → survoler les cases (ex: 3×4) → cliquer.
  • Via la boîte de dialogue : Insertion → Tableau → « Insérer un tableau » → choisir nombre de colonnes et lignes.
  • Dessiner un tableau : Outil « Dessiner un tableau » pour créer des cellules non standard.

🎯 Exemple concret :

  1. Cliquer sur Insertion → Tableau.
  2. Sélectionner 3 colonnes et 4 lignes.
  3. Saisir les données (exemple : noms, notes, appréciations).
  4. Appliquer un style de tableau (bordures, couleurs).

💡 Exemples de compréhension

Examen pratique n°1 : Insérer une image dans un rapport de stage

Contexte : Vous rédigez un rapport de stage pour une association culturelle. Vous souhaitez insérer le logo de l’association (fichier "logo_association.png") après le titre.

Résolution / Explication :
1. Placer le curseur juste après le titre principal.
2. Aller dans le menu Insertion → Image → À partir d’un fichier.
3. Naviguer jusqu’au dossier contenant logo_association.png et double-cliquer sur le fichier.
4. L’image s’affiche. Utiliser les poignées pour la réduire (taille raisonnable).
5. Pour un meilleur rendu, choisir l’option d’habillage « Serré » (clic droit sur l’image → habillage) afin que le texte l’entoure.

💬 Astuce : Si l’image est trop grande, maintenez la touche Shift (Maj) en glissant un coin pour conserver les proportions.

Exemple n°2 : Créer un tableau des moyennes pour 5 élèves

Vous êtes délégué de classe et devez présenter les résultats du trimestre. Construisez un tableau avec les colonnes : Nom, Mathématiques, Français, Moyenne.

Procédure détaillée :
→ Insertion → Tableau → choisir 4 colonnes et 6 lignes (5 élèves + en-tête).
→ Saisir les titres : Nom, Maths, Français, Moyenne.
→ Remplir les notes des élèves.
→ Sélectionner la dernière colonne (Moyenne) et utiliser l’outil « Formule » (ou calculer mentalement) pour ajouter une moyenne simple.
→ Mettre en gras la ligne des en-têtes et appliquer des bordures légères.

Exemple de tableau de notes dans un traitement de texte.
📌 À retenir : On peut ajouter ou supprimer des lignes/colonnes après création (clic droit → Insérer / Supprimer).

Comparaison rapide : Image vs Tableau

Il est utile de savoir quand utiliser un objet plutôt qu’un autre.

ObjetUtilisation typiqueAvantage
🖼️ ImageIllustrations, logos, captures d'écranVisuel attractif, explique des concepts
📊 TableauDonnées chiffrées, comparatifs, planningsStructure claire, facile à lire

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Insérer une image mais ne pas gérer l’habillage. Résultat : l’image « pousse » le texte de manière inattendue. Toujours vérifier l’option « habillage du texte » (carré, serré, devant/derrière le texte).
  • Attention : Confondre « copier/coller » une image depuis internet avec l’insertion par le menu. Le collage direct est rapide, mais peut causer des problèmes de résolution ou de droits. Mieux vaut utiliser Insertion → Image lorsque l’image est sauvegardée.
  • Piège du tableau : Oublier de définir les en-têtes de colonnes. Les correcteurs (examen) vérifient la lisibilité. Toujours mettre la première ligne en gras ou avec un fond coloré.
  • À ne pas faire : Redimensionner une image uniquement en largeur (déformation). Maintenir Maj (Shift) pour garder les proportions.

📎 Procédures essentielles pour l'examen

Insérer un tableau – version ultra-rapide (Word/LibreOffice)

  • 1. Onglet Insertion → groupe Tableau.
  • 2. Glisser sur la grille pour choisir le nombre de colonnes/lignes.
  • 3. Relâcher : le tableau apparaît.
  • 4. Pour fusionner des cellules (examen 2025) : sélectionner les cellules → clic droit → Fusionner les cellules.

Insérer une image – questions typiques du Bac

  • • « Proposer une procédure pour insérer une image depuis un fichier. » → Réponse structurée : position du curseur, menu Insertion, Image, choisir le fichier, valider.
  • • « Comment améliorer l’intégration d’une image dans le texte ? » → Utiliser l’habillage « serré » ou « carré » et ajuster la taille.
🧠 Mémo visuel : Imaginez les trois étapes : 1. Curseur au bon endroit → 2. Menu Insertion → 3. Sélection de l’objet (Image/Tableau).

📖 Mots-clés à maîtriser

  • Objet : Élément insérable non textuel (image, tableau, forme).
  • Habillage : Façon dont le texte contourne l’objet (carré, serré, aucun).
  • Poignées de redimensionnement : Petits cercles/carrés autour de l’objet pour modifier sa taille.
  • Fusion de cellules : Combiner plusieurs cellules adjacentes pour former une seule cellule (notion vue dans le programme).
  • Tableau : Grille organisée en lignes et colonnes.
