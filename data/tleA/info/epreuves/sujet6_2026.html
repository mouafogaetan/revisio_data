📖 Exercice 1 : QCM (6 pts)
1. Quelle est la principale différence entre une image bitmap et une image vectorielle ?
  • A. L'image bitmap est composée de pixels, l'image vectorielle de formules mathématiques
  • B. L'image vectorielle est plus lourde que l'image bitmap
  • C. L'image bitmap ne peut être visualisée que sur ordinateur
  • D. Il n'y a pas de différence
    • 2. Lequel de ces formats est un format d'image VECTORIEL ?
  • A. .jpg
  • B. .png
  • C. .svg
  • D. .gif
    • 3. En PAO (Publication Assistée par Ordinateur), qu'est-ce qu'un "calque" ?
  • A. Une police d'écriture spéciale
  • B. Un niveau transparent permettant de superposer des éléments sans les mélanger
  • C. Un type d'imprimante
  • D. Une extension de fichier
    • 4. Parmi ces logiciels, lequel est spécifiquement dédié à la PAO ?
  • A. Adobe Photoshop (éditeur d'images)
  • B. Microsoft Word (traitement de texte)
  • C. Adobe InDesign / Scribus
  • D. Visual Studio Code (éditeur de code)

    • ✅ Réponses : 1-A (Bitmap = pixels, Vectoriel = maths) ; 2-C (SVG est le standard vectoriel pour le web) ; 3-B (Les calques permettent de travailler sur des éléments indépendants) ; 4-C (InDesign et Scribus sont des logiciels de mise en page professionnelle).

    🎨 Exercice 2 : Analyse d'image (4 pts)

    a) Identifiez et nommez deux éléments principaux de l'interface visibles sur cette image.

    ✏️ Exercice 3 : Production (6 pts)
    Contexte : Vous êtes chargé de créer une carte de visite pour le club d'informatique de votre lycée.
    Tâches :
  • • Décrivez les étapes de réalisation de cette carte à l'aide d'un logiciel de PAO.
  • • Citez trois informations indispensables qui doivent figurer sur cette carte.
  • • Proposez une mise en page simple (texte + image).

    • ✅ Proposition de correction :
    Étapes : 1. Créer un nouveau document (format 9x5 cm). 2. Définir les marges. 3. Importer un logo (image vectorielle). 4. Créer des blocs texte pour le nom, le titre et les coordonnées. 5. Appliquer des styles de caractère. 6. Exporter au format PDF pour l'impression.
    Informations : Nom du club, Nom du membre/president, Adresse mail/WhatsApp, Logo du lycée.
    Mise en page : Logo à gauche, Texte aligné à droite, fond de couleur unie.

  • 1. .mp3      A. Image bitmap
