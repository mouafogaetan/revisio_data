Leçon 3 : Logiciels informatiques

Informatique · Terminale A (Littéraire) · Unité 1 : Architecture et protection de l'ordinateur

📌 Éléments clés à retenir

1. Qu'est-ce qu'un logiciel informatique ?

Un logiciel est un ensemble d'instructions (programmes) qui indiquent à l'ordinateur comment exécuter des tâches spécifiques. Sans logiciel, un ordinateur n'est qu'un ensemble de composants matériels inutilisables. On distingue deux grandes familles de logiciels.

2. Le système d'exploitation (OS)

Le système d'exploitation est le logiciel principal de l'ordinateur. Il est responsable de :

  • Gérer les ressources matérielles (processeur, mémoire, disque dur, périphériques).
  • Fournir une interface utilisateur (graphique ou texte) pour interagir avec la machine.
  • Lancer et superviser l'exécution des autres logiciels (applications).
  • Assurer la gestion des fichiers et des dossiers.

Exemples de systèmes d'exploitation : Windows (Microsoft), Linux (Ubuntu, Fedora, Debian), macOS (Apple), Android, iOS.

Figure 1 : Le système d'exploitation sert d'intermédiaire entre le matériel et les applications

3. Les logiciels d'application

Les logiciels d'application (ou applications) sont des programmes conçus pour réaliser des tâches spécifiques pour l'utilisateur. Ils s'exécutent sur le système d'exploitation.

  • Bureautique : Microsoft Word, Excel, PowerPoint ; LibreOffice Writer, Calc, Impress ; Google Docs, Sheets, Slides.
  • Navigation Internet : Google Chrome, Mozilla Firefox, Microsoft Edge, Safari.
  • Sécurité : Antivirus (Avast, Kaspersky, Bitdefender), pare-feu.
  • Communication : Zoom, WhatsApp, Telegram, Outlook.
  • Création / Multimédia : Photoshop, Audacity, VLC.

4. Différence essentielle à connaître

Critère de distinction : Le système d'exploitation est indispensable au fonctionnement de l'ordinateur. Sans lui, aucun logiciel d'application ne peut s'exécuter. Les logiciels d'application, quant à eux, répondent à des besoins spécifiques de l'utilisateur et peuvent être installés ou désinstallés sans remettre en cause le fonctionnement de base de la machine.

💡 Exemples de compréhension

Exercice type 1 : Distinguer OS et application

Question (d'après Bac 2025) : Parmi les éléments suivants, lesquels sont des systèmes d'exploitation et lesquels sont des logiciels d'application ?
Windows 11 · Microsoft Word · Linux Ubuntu · Antivirus Kaspersky · Google Chrome · macOS · Zoom

Résolution :
✅ Systèmes d'exploitation : Windows 11, Linux Ubuntu, macOS.
✅ Logiciels d'application : Microsoft Word, Antivirus Kaspersky, Google Chrome, Zoom.
💬 Astuce : Un système d'exploitation porte souvent le nom d'une version (Windows 10, macOS Sonoma) et est le premier logiciel qui se lance au démarrage. Une application se lance après le démarrage, quand l'utilisateur en a besoin.

Exercice type 2 : Rôle du système d'exploitation

Question : Pourquoi ne peut-on pas installer Microsoft Word sur un ordinateur qui ne possède pas de système d'exploitation ?

Explication : Microsoft Word est un logiciel d'application. Il a besoin d'un système d'exploitation pour fonctionner, car c'est le système d'exploitation qui :
  • Gère l'allocation de la mémoire nécessaire à Word.
  • Permet à Word d'interagir avec le processeur et le disque dur.
  • Fournit les bibliothèques logicielles (DLL, frameworks) dont Word a besoin.
Sans système d'exploitation, l'ordinateur ne peut même pas démarrer et aucun logiciel ne peut être exécuté.
💡 Pensez à la métaphore : le système d'exploitation est le conducteur d'orchestre et les applications sont les musiciens. Sans conducteur, les musiciens ne peuvent pas jouer ensemble de manière coordonnée.

Exercice type 3 : Cas concret du laboratoire informatique

Situation : Dans un lycée au Cameroun, les ordinateurs du laboratoire d'informatique sont équipés de Windows 11. Les élèves souhaitent utiliser un tableur pour calculer des moyennes. Le professeur propose d'utiliser LibreOffice Calc (gratuit) car la licence Microsoft Office est trop chère.

Analyse :
- Système d'exploitation présent : Windows 11 (permet le fonctionnement des ordinateurs).
- Logiciel d'application utilisé : LibreOffice Calc (permet de faire des calculs, des tableaux).
- Alternative possible : Google Sheets (application en ligne, nécessite une connexion Internet).
💬 Remarque : Un même système d'exploitation (Windows) peut exécuter différents logiciels d'application (Microsoft Office, LibreOffice, Google Chrome, etc.).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente n°1 : Confondre un logiciel d'application (comme un antivirus) avec le système d'exploitation. Rappelez-vous : l'antivirus s'installe sur le système d'exploitation.
  • Erreur fréquente n°2 : Croire qu'un ordinateur peut fonctionner uniquement avec des logiciels d'application. Non ! Le système d'exploitation est indispensable.
  • Attention : Un même nom peut parfois désigner une suite logicielle contenant plusieurs applications. Exemple : "Microsoft Office" n'est pas un logiciel unique mais une collection d'applications (Word, Excel, PowerPoint).
  • À retenir pour l'examen : Les questions consistent souvent à classer des éléments donnés dans deux colonnes : "Système d'exploitation" et "Logiciel d'application". Soyez méthodique.

🎯 Ce qu'il faut absolument savoir pour le Bac

Synthèse en 5 points

  • Définition : Un logiciel est un ensemble d'instructions exécutables par un ordinateur.
  • Catégorie 1 : Le système d'exploitation (Windows, Linux) gère le matériel et permet le fonctionnement de l'ordinateur.
  • Catégorie 2 : Les logiciels d'application (antivirus, navigateur, tableur) répondent à des besoins spécifiques des utilisateurs.
  • Relation : Les applications s'exécutent sur le système d'exploitation.
  • Exemples à citer : Savoir donner au moins deux exemples pour chaque catégorie (Windows/Linux pour les OS ; Word/Antivirus pour les applications).
