📄 PARTIE A : HTML (6 points)
Exercice 1 : Structure et balises (3 pts)

1.1 (1 pt) - Quelle est la structure de base d'une page HTML ? Donnez les balises principales dans l'ordre.

✅ Réponse La structure de base est :

<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    Contenu visible de la page
  </body>
</html>

Les balises principales : <html> (racine), <head> (métadonnées), <title> (titre onglet), <body> (contenu visible).

1.2 (1 pt) - Comment insère-t-on une image dans une page HTML ? Donnez la syntaxe complète avec les attributs nécessaires.

✅ Réponse On utilise la balise auto-fermante <img> :

<img src="chemin/vers/image.jpg" alt="Description de l'image" />

Attributs obligatoires : src (source de l'image) et alt (texte alternatif pour l'accessibilité).

Exemple concret : <img src="logo.png" alt="Logo de l'école" />

1.3 (1 pt) - Donnez la syntaxe pour créer un tableau à 2 lignes et 3 colonnes en HTML.

✅ Réponse Tableau 2×3 :

<table border="1">
  <tr>
    <td>Ligne1-Col1</td>
    <td>Ligne1-Col2</td>
    <td>Ligne1-Col3</td>
  </tr>
  <tr>
    <td>Ligne2-Col1</td>
    <td>Ligne2-Col2</td>
    <td>Ligne2-Col3</td>
  </tr>
</table>

Balises : <table> (conteneur), <tr> (ligne), <td> (cellule).

Pour fusionner des colonnes : attribut colspan="n".

Exercice 2 : Formulaires HTML (3 pts)

2.1 (1.5 pts) - Écrivez le code HTML d'un formulaire contenant :

  • Un champ "Nom" (type texte)
  • Un champ "Email" (type email)
  • Un bouton d'envoi "Envoyer"

✅ Réponse Code du formulaire :

<form action="traitement.php" method="post">
  <label for="nom">Nom :</label>
  <input type="text" id="nom" name="nom" required>
  <br>
  <label for="email">Email :</label>
  <input type="email" id="email" name="email" required>
  <br>
  <button type="submit">Envoyer</button>
</form>

Explication : <form> définit le formulaire, action précise où envoyer les données, method la méthode HTTP. L'attribut required rend le champ obligatoire.

2.2 (1.5 pts) - À quoi sert l'attribut colspan dans un tableau HTML ? Donnez un exemple concret.

✅ Réponse colspan permet de fusionner plusieurs colonnes d'une même ligne.

Exemple :

<table border="1">
  <tr>
    <td colspan="2">Cellule qui occupe 2 colonnes</td>
  </tr>
  <tr>
    <td>Colonne 1</td>
    <td>Colonne 2</td>
  </tr>
</table>

Utilité : créer des titres de tableau qui s'étendent sur plusieurs colonnes.

⚡ PARTIE B : JavaScript (6 points)
Exercice 3 : Concepts de base (3 pts)

3.1 (0.5 pt) - Qu'est-ce que JavaScript et à quoi sert-il dans une page web ?

✅ Réponse JavaScript est un langage de programmation côté client qui permet de rendre les pages web interactives et dynamiques (gestion des événements, modification du DOM, validation de formulaires, animations, etc.).

3.2 (1 pt) - Donnez la syntaxe pour déclarer une fonction en JavaScript qui calcule la somme de deux nombres.

✅ Réponse Fonction en JavaScript :

function somme(a, b) {
    return a + b;
}

// Appel de la fonction
var resultat = somme(5, 3);  // resultat vaut 8

Syntaxe : function nomFonction(paramètres) { instructions; return valeur; }

3.3 (0.75 pt) - À quoi servent les fonctions parseInt() et prompt() en JavaScript ?

✅ Réponse

  • prompt() : Affiche une boîte de dialogue permettant à l'utilisateur de saisir une valeur (retourne une chaîne de caractères).
  • parseInt() : Convertit une chaîne de caractères en nombre entier.

Exemple : var age = parseInt(prompt("Entrez votre âge :"));

3.4 (0.75 pt) - Citez trois événements JavaScript courants et expliquez à quoi chacun correspond.

✅ Réponse Événements courants :

  • onClick : Se déclenche au clic sur un élément (bouton, lien, etc.)
  • onLoad : Se déclenche lorsque la page ou un élément a fini de charger
  • onMouseOver : Se déclenche quand la souris survole un élément
  • onChange : Se déclenche quand la valeur d'un champ de formulaire change

Exemple : <button onClick="maFonction()">Cliquez</button>

Exercice 4 : Analyse de code (3 pts)

4.1 (1.5 pts) - Analysez le code JavaScript suivant et dites ce qu'il affiche :

var notes = [12, 15, 8, 17, 10];
var somme = 0;
for(var i = 0; i < notes.length; i++) {
    somme = somme + notes[i];
}
var moyenne = somme / notes.length;
document.write("La moyenne est : " + moyenne);

✅ Réponse Ce programme :

  1. Déclare un tableau notes contenant 5 valeurs
  2. Calcule la somme des éléments du tableau (12+15+8+17+10 = 62)
  3. Calcule la moyenne (62 ÷ 5 = 12,4)
  4. Affiche : "La moyenne est : 12.4"

Il illustre l'utilisation des tableaux, de la boucle for et de l'affichage avec document.write().

4.2 (1.5 pts) - Écrivez un programme JavaScript qui demande à l'utilisateur son âge via prompt() et affiche "Majeur" si l'âge est ≥ 18, "Mineur" sinon.

✅ Réponse Code :

var age = parseInt(prompt("Entrez votre âge :"));

if (age >= 18) {
    document.write("Majeur");
} else {
    document.write("Mineur");
}

Explication : parseInt() convertit la saisie en nombre, la condition if...else permet l'affichage selon la valeur.

Exercice 5 : Structure d'un programme C (2 pts)

5.1 (1 pt) - Écrivez la structure complète d'un programme C qui affiche "Bonjour Cameroun !" à l'écran.

✅ Réponse Programme C :

#include <stdio.h>

int main() {
    printf("Bonjour Cameroun !");
    return 0;
}

Explication : #include <stdio.h> inclut la bibliothèque d'entrée/sortie, main() est la fonction principale, printf() affiche le texte, return 0 indique une exécution correcte.

5.2 (1 pt) - À quoi sert la directive #include <stdio.h> et que signifie return 0 ?

✅ Réponse

  • #include <stdio.h> : Inclut la bibliothèque standard d'entrée/sortie qui contient les fonctions printf() (affichage) et scanf() (saisie).
  • return 0 : Indique au système d'exploitation que le programme s'est terminé correctement (code de retour 0 = succès).
Exercice 6 : Traduction algorithmique (3 pts)

6.1 (1.5 pts) - Traduisez l'algorithme suivant en langage C :

Variable: N : entier
Début
    Ecrire("Entrez un nombre : ")
    Lire(N)
    Si (N > 0) Alors
        Ecrire("Positif")
    Sinon Si (N == 0) Alors
        Ecrire("Nul")
    Sinon
        Ecrire("Négatif")
    FinSi
Fin

✅ Réponse Traduction en C :

#include <stdio.h>

int main() {
    int N;
    printf("Entrez un nombre : ");
    scanf("%d", &N);
    
    if (N > 0) {
        printf("Positif");
    } else if (N == 0) {
        printf("Nul");
    } else {
        printf("Négatif");
    }
    return 0;
}

scanf("%d", &N) permet de lire un entier saisi par l'utilisateur.

6.2 (1.5 pts) - Écrivez un programme C qui calcule et affiche la somme des entiers de 1 à 100 (boucle for).

✅ Réponse Programme :

#include <stdio.h>

int main() {
    int somme = 0;
    int i;
    
    for (i = 1; i <= 100; i++) {
        somme = somme + i;
    }
    
    printf("La somme de 1 à 100 est : %d", somme);
    return 0;
}

Explication : La boucle for commence avec i=1, tourne tant que i≤100, et incrémente i de 1 à chaque itération (i++).

Résultat : La somme est 5050.

Exercice 7 : Tableaux en C (3 pts)

7.1 (1 pt) - Déclarez en langage C un tableau capable de stocker 50 notes (nombres réels). Quelle est la particularité des indices en C ?

✅ Réponse Déclaration :

float notes[50];

Particularité : En langage C, les indices des tableaux commencent à 0 (et non à 1 comme en algorithmique).

Pour un tableau de taille 50, les indices vont de 0 à 49.

Accès au premier élément : notes[0] ; au dernier : notes[49].

7.2 (2 pts) - Écrivez un programme C qui :

  • Déclare un tableau d'entiers de taille 5
  • Saisit les 5 valeurs au clavier
  • Affiche la plus grande valeur

✅ Réponse Programme complet :

#include <stdio.h>

int main() {
    int tab[5];
    int i, max;
    
    // Saisie des valeurs
    for (i = 0; i < 5; i++) {
        printf("Entrez la valeur %d : ", i+1);
        scanf("%d", &tab[i]);
    }
    
    // Recherche du maximum
    max = tab[0];
    for (i = 1; i < 5; i++) {
        if (tab[i] > max) {
            max = tab[i];
        }
    }
    
    printf("La plus grande valeur est : %d", max);
    return 0;
}

Explication : On initialise max avec le premier élément, puis on parcourt le tableau en comparant chaque élément avec max.
