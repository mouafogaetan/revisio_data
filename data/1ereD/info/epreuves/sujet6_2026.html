1.3) Explique l’utilité de l’attribut colspan dans un tableau HTML. Donne un exemple d’utilisation. (1.5 pts)
✔ Réponse colspan permet de fusionner plusieurs colonnes horizontales. Exemple : <td colspan="2">Titre large</td> [citation:9].
1.4) Cite deux attributs essentiels de la balise <img> et explique leur rôle. (1.5 pts)
✔ Réponse src : chemin de l’image ; alt : texte alternatif (accessibilité, référencement).
⚡ EXERCICE 2 : JAVASCRIPT – INTERACTIONS (6 points)
2.1) Quelle est la différence entre parseInt() et parseFloat() en JavaScript ? (1 pt)
✔ Réponse parseInt() convertit une chaîne en nombre entier, parseFloat() garde la partie décimale (nombre réel). [cf. programme chap10].
2.2) Écris une fonction JavaScript nommée calculerSurface() qui prend la longueur et la largeur, puis affiche le résultat dans une boîte alert(). (2 pts)
✔ Code JS
function calculerSurface(L, l) {
let surface = L * l;
alert("Surface = " + surface + " m²");
}
2.3) Un évènement onClick : définis cette notion et écris un bouton HTML qui déclenche l’appel à la fonction précédente. (2 pts)
✔ Explication Évènement = action utilisateur (clic, survol...). <button onClick="calculerSurface(5,3);">Calculer</button>
2.4) Que fait la méthode prompt() ? Donne un extrait qui demande le nom de l’utilisateur et le stocke. (1 pt)
✔ Exemple let nom = prompt("Entrez votre nom :"); → boîte de dialogue avec saisie.

✔ Réponse for = nombre d'itérations connu (compteur) ; while = dépend d'une condition. Exemple for : parcours d’un tableau.
