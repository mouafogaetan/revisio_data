💿 Chapitre 1 : Installation (4 pts)
1. Laquelle de ces propositions est une étape cruciale APRÈS l'installation du système d’exploitation ?
  • A. Le partitionnement du disque dur
  • B. L’installation des pilotes (drivers)
  • C. Le choix du type de système de fichiers (NTFS/FAT32)
  • D. La définition de la mémoire RAM allouée

    • ✅ Réponse : B L’installation des pilotes est une opération post-installation essentielle pour que les périphériques fonctionnent.

    💿 Chapitre 1 : Partition
    2. En informatique, qu’est-ce qu’une partition secondaire ?
  • A. Une partition qui contient obligatoirement le système d’exploitation
  • B. Une partie du disque dur dédiée à une utilisation spécifique, distincte de la partition principale
  • C. Un espace de stockage sur une clé USB
  • D. La mémoire cache du processeur

    • ✅ Réponse : B Une partition secondaire est une section du disque dur réservée à un usage particulier (données, sauvegarde).

    💿 Chapitre 1 : Restauration
    3. Quel est le principal intérêt de créer un point de restauration système ?
  • A. Réinstaller complètement Windows
  • B. Ramener les éléments du système à un état antérieur après un problème
  • C. Supprimer tous les fichiers personnels de l’utilisateur
  • D. Augmenter la vitesse de la RAM

    • ✅ Réponse : B La restauration système permet de revenir en arrière sans perdre vos documents, contrairement à une réinstallation.

    ⌨️ Chapitre 2 : Commandes DOS (6 pts)
    4. Quelle commande utilise-t-on sous Windows (CMD) pour afficher la liste des fichiers et dossiers dans le répertoire courant ?
  • A. LS
  • B. LIST
  • C. DIR
  • D. SHOW

    • ✅ Réponse : C La commande DIR (Directory) liste le contenu d’un dossier.

    ⌨️ Chapitre 2 : Navigation
    5. Pour changer de répertoire (se déplacer dans l’arborescence) sous DOS, la commande est :
  • A. RD
  • B. CD
  • C. MD
  • D. REN

    • ✅ Réponse : B CD (Change Directory) permet de se déplacer dans les dossiers.

    ⌨️ Chapitre 2 : Création
    6. Quel est le bon exemple pour créer un répertoire nommé "COURS_INFO" ?
  • A. CREATE COURS_INFO
  • B. MD COURS_INFO
  • C. MKFILE COURS_INFO
  • D. NEW COURS_INFO

    • ✅ Réponse : B MD (Make Directory) ou MKDIR crée un dossier.

    ⌨️ Chapitre 2 : Manipulation
    7. Associez chaque action à la bonne commande (Réponse unique) : Pour SUPPRIMER un fichier "test.txt".
  • A. DEL test.txt
  • B. RD test.txt
  • C. COPY test.txt
  • D. RMDIR test.txt

    • ✅ Réponse : A La commande DEL supprime un ou plusieurs fichiers. RD supprime un dossier vide.

    🔒 Chapitre 3 : Cybersécurité (6 pts)
    8. Dans le modèle de sécurité informatique, quel principe garantit que les données ne sont accessibles qu’aux personnes autorisées ?
  • A. Intégrité
  • B. Disponibilité
  • C. Confidentialité
  • D. Authentification

    • ✅ Réponse : C La Confidentialité assure que seules les personnes habilitées voient les données.

    🔒 Chapitre 3 : Menaces
    9. "Wannacry" est un exemple célèbre de logiciel malveillant qui chiffre vos données et demande une rançon. Il s’agit d’un :
  • A. Ver (Worm)
  • B. Ransomware
  • C. Cheval de Troie
  • D. Spyware

    • ✅ Réponse : B Un Ransomware prend les données en otage contre de l’argent.

    🔒 Chapitre 3 : Phishing
    10. Que désigne le terme "Hameçonnage" (Phishing) ?
  • A. Technique qui consiste à surcharger un serveur de requêtes
  • B. Envoi de courriels frauduleux pour récolter des informations personnelles (mots de passe, cartes bancaires)
  • C. Logiciel qui espionne les frappes au clavier
  • D. Défaut matériel d’un disque dur

    • ✅ Réponse : B Le phishing est une arnaque visant à usurper l’identité de marques pour voler des données sensibles.

    🔒 Chapitre 3 : Protection
    11. Laquelle de ces actions est la PLUS efficace pour se protéger contre les menaces en ligne ?
  • A. Désactiver le pare-feu pour que l’ordinateur aille plus vite
  • B. Utiliser le même mot de passe partout pour ne pas l’oublier
  • C. Maintenir ses logiciels (OS, antivirus) à jour et activer l’authentification à deux facteurs
  • D. Ouvrir toutes les pièces jointes des emails pour vérifier leur contenu

    • ✅ Réponse : C Les mises à jour corrigent les failles de sécurité, et le double facteur renforce l’authentification.

    🔒 Chapitre 3 : Disponibilité
    12. Une attaque par Déni de service (DoS) vise principalement à compromettre quel pilier de la sécurité ?
  • A. La Confidentialité
  • B. L’Intégrité
  • C. La Disponibilité
  • D. La Non-répudiation

    • ✅ Réponse : C Le DoS rend un service inaccessible, détruisant ainsi la Disponibilité.
