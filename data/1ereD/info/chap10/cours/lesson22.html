🎯 Les événements en JavaScript

Informatique · Première C, D, E · Programme officiel (Unité 22)

📌 Éléments clés à retenir

🔹 Qu'est-ce qu'un événement ?

Un événement est une action de l'utilisateur (ou du système) détectée par le navigateur. En JavaScript, les événements permettent de rendre une page web interactive : on peut exécuter une fonction en réaction à un clic, une saisie, le chargement de la page, etc.

  • Principe : "Quand X se produit, alors exécute Y".
  • Le code JavaScript "écoute" les événements via des gestionnaires d'événements (event handlers).
  • Les événements sont au cœur de l'interactivité des sites modernes (formulaires, animations, jeux...).

🔹 Principaux événements à connaître

Le navigateur peut détecter de nombreuses actions. Voici les événements les plus utilisés en classe de Première :

  • onClick → Se produit lorsque l'utilisateur clique sur un élément (bouton, lien, image...).
  • onLoad → Se produit lorsque la page ou un élément (image, frame) a fini de charger.
  • onMouseOver → Se produit lorsque la souris survole un élément.
  • onChange → Se produit lorsque la valeur d'un champ (input, select, textarea) est modifiée et que l'utilisateur quitte le champ.
  • onMouseOut → Inverse de onMouseOver : quand la souris quitte l'élément.
  • onKeyPress / onKeyDown → Quand une touche du clavier est pressée.

🔹 Comment utiliser un événement en HTML/JS ?

Deux méthodes principales (nous nous concentrons sur la plus simple et directe pour le niveau Première) :

  • Attribut HTML : Ajouter directement l'événement dans la balise, par exemple : <button onClick="maFonction();">Cliquez</button>
  • Propriété d'élément : En JavaScript, sélectionner l'élément et lui affecter une fonction (ex: document.getElementById("monBouton").onclick = maFonction;).

✅ À retenir : Le nom de l'événement s'écrit en camelCase avec un 'on' devant : onClick, onLoad, onMouseOver.

💡 Exemples de compréhension

Exemple 1 : Bouton avec onClick

On souhaite afficher un message "Bonjour !" dans une boîte de dialogue (alerte) quand l'utilisateur clique sur un bouton.

Code HTML + JavaScript :
<button onClick="afficherMessage();">Dire bonjour</button>
<script>
  function afficherMessage() {
    alert("Bonjour !");
  }
</script>

💬 Astuce : La fonction appelée par l'événement doit être définie AVANT que l'utilisateur ne clique, sinon une erreur se produit. Placez vos fonctions dans la balise <script> dans le <head> ou avant la fermeture de <body>.

Exemple 2 : onMouseOver et onMouseOut (effet visuel)

Créer un paragraphe qui change de couleur lorsque la souris passe dessus, et retrouve sa couleur d'origine quand la souris quitte.

Code :
<p id="texte" onMouseOver="surligner();" onMouseOut="normal();">Passez la souris ici !</p>
<script>
  function surligner() {
    document.getElementById("texte").style.backgroundColor = "yellow";
  }
  function normal() {
    document.getElementById("texte").style.backgroundColor = "transparent";
  }
</script>

💡 Compréhension : onMouseOver déclenche la mise en surbrillance, onMouseOut rétablit l'état initial. On peut aussi modifier la taille, la police, etc.

Exemple 3 : onLoad (action au chargement de la page)

Afficher une alerte "Bienvenue sur mon site !" dès que la page est complètement chargée.

Code :
<body onLoad="afficherBienvenue();">
<script>
  function afficherBienvenue() {
    alert("Bienvenue sur mon site !");
  }
</script>

⚡ Utilité : onLoad permet de préparer des animations, pré-remplir des champs, ou charger des données sans attendre l'action de l'utilisateur.

Exemple 4 : onChange (formulaire interactif)

Un menu déroulant (select) qui affiche une phrase différente selon l'option choisie.

Code :
<select id="couleur" onChange="afficherChoix();">
  <option value="rouge">Rouge</option>
  <option value="bleu">Bleu</option>
</select>
<p id="message"></p>
<script>
  function afficherChoix() {
    let select = document.getElementById("couleur");
    let valeur = select.value;
    document.getElementById("message").innerHTML = "Vous aimez le " + valeur + " !";
  }
</script>

📌 Remarque : onChange se déclenche quand la valeur change ET que l'utilisateur quitte le champ (click ailleurs ou tabulation).

⚠️ Pièges courants & À ne pas confondre

  • Confusion entre onClick et onChange : onClick réagit à un clic immédiat, onChange nécessite un changement de valeur + perte de focus.
  • Oublier les parenthèses : Dans l'attribut HTML, on écrit onClick="maFonction()" et non onClick="maFonction" (sinon la fonction n'est pas appelée).
  • Fonction non définie : Si la fonction appelée par l'événement n'existe pas, le navigateur génère une erreur silencieuse (rien ne se passe). Vérifiez le nom et l'ordre des scripts.
  • Cas de onLoad : On ne peut pas utiliser document.getElementById sur un élément qui n'est pas encore chargé. Placez le script après l'élément ou utilisez onLoad.
  • Attention à la casse : onclick (minuscules) fonctionne parfois, mais la convention standard est onClick (C majuscule) en HTML. En JavaScript moderne on utilise plutôt click avec addEventListener, mais au programme officiel on retient onClick.

🎮 Application directe : Calcul de surface avec onClick

Cas pratique : "Prix de vente d'un rectangle"

D'après le programme officiel : « Lors du clic, la fonction surfaceRect() est exécutée, affichant les boîtes de dialogue pour saisir longueur, largeur et prix, puis calculant et affichant le prix de vente. »

Voici un exemple complet utilisant l'événement onClick sur un bouton. La fonction surfaceRect() utilise prompt() pour saisir les données, calcule la surface, puis le prix total.

👆 Cliquez sur le bouton pour tester l'événement onClick.
Vous serez invité à saisir la longueur (m), largeur (m) et prix au m² (FCFA).

💻 Extrait du code utilisé :
function surfaceRect() {
   let longueur = parseFloat(prompt("Entrez la longueur (en mètres) :"));
   let largeur = parseFloat(prompt("Entrez la largeur (en mètres) :"));
   let prixM2 = parseFloat(prompt("Entrez le prix au m² (FCFA) :"));
   let surface = longueur * largeur;
   let prixVente = surface * prixM2;
   alert("Surface : " + surface + " m²\nPrix de vente : " + prixVente + " FCFA");
}
// Lier l'événement onClick via JavaScript (équivalent à l'attribut HTML) document.getElementById("demoSurfaceBtn").onclick = surfaceRect;

✅ Ce qui se produit lors du clic : La fonction surfaceRect() est exécutée → trois boîtes de dialogue prompt() s'affichent → le calcul est effectué → une alerte montre le résultat.
