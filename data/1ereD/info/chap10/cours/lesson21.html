📘 Structures de contrôle, tableaux et fonctions en JavaScript

Informatique · Première C, D, E · Unité d'Apprentissage 10 (UE 21)

📌 Éléments clés à retenir

1. Structures conditionnelles en JS

Elles permettent d'exécuter des blocs d'instructions selon qu'une condition est vraie ou fausse.

  • if (condition) { ... } : exécute le bloc si la condition est vraie.
  • if (condition) { ... } else { ... } : exécute un bloc si vrai, un autre si faux.
  • else if : enchaîner plusieurs conditions.
  • switch (expression) { case valeur: ... } : alternative pour multiples cas.
// Exemple condition
let age = 18;
if (age >= 18) {
    console.log("Majeur");
} else {
    console.log("Mineur");
}

2. Structures itératives (boucles)

Répéter un bloc d'instructions un nombre défini ou tant qu'une condition est vérifiée.

  • for (initialisation; condition; incrément) { ... } : quand le nombre d'itérations est connu.
  • while (condition) { ... } : tant que la condition est vraie, peut ne pas s'exécuter.
  • do { ... } while (condition); : exécute au moins une fois.
// Boucle for : afficher les nombres de 1 à 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// Boucle while : somme des entiers jusqu'à 10
let somme = 0, n = 1;
while (n <= 10) {
    somme += n;
    n++;
}

3. Tableaux en JavaScript

Un tableau est une structure de données qui stocke une liste ordonnée d'éléments (de tout type : nombres, chaînes, objets...).

  • Déclaration : let notes = [12, 15, 18, 10]; ou let tableau = new Array(5);
  • Accès aux éléments : notes[0] (indice 0 pour premier élément).
  • Propriété .length : longueur du tableau.
  • Parcours classique : for (let i = 0; i < tab.length; i++)
  • Méthodes utiles : push() (ajout fin), pop() (supprime dernier), forEach().
let fruits = ["Mangue", "Orange", "Banane"];
fruits.push("Ananas");   // ajoute à la fin
console.log(fruits.length); // 4
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

4. Fonctions en JavaScript

Une fonction est un bloc d'instructions réutilisable, qui peut recevoir des paramètres et retourner une valeur.

  • Déclaration classique : function nomFonction(param1, param2) { return result; }
  • Fonction anonyme / expression : const addition = function(a,b){ return a+b; };
  • Fonction fléchée (ES6) : const carre = x => x*x;
  • Appel : let result = nomFonction(valeur1, valeur2);
// Calcul de la moyenne d'un tableau
function moyenne(tableau) {
    let somme = 0;
    for (let i = 0; i < tableau.length; i++) {
        somme += tableau[i];
    }
    return somme / tableau.length;
}
let notes = [14, 16, 12, 18];
let moy = moyenne(notes);
console.log("Moyenne : " + moy);

💡 Exemples de compréhension

Exemple 1 : Utilisation d'une condition et d'une boucle

Écrire une fonction nombresPairs(n) qui affiche tous les nombres pairs de 0 à n (inclus) et retourne leur quantité.

Résolution : On utilise une boucle for pour parcourir les nombres, un test if (i % 2 === 0) pour détecter les pairs, et un compteur. 

function nombresPairs(n) {
    let compteur = 0;
    for (let i = 0; i <= n; i++) {
        if (i % 2 === 0) {
            console.log(i);
            compteur++;
        }
    }
    return compteur;
}
let total = nombresPairs(10);
console.log("Quantité de nombres pairs : " + total);
💬 Astuce : L'opérateur modulo % donne le reste de la division. i % 2 === 0 signifie que i est divisible par 2.

Exemple 2 : Manipulation d'un tableau et fonction

On dispose d'un tableau de températures en degrés Celsius. Créer une fonction convertirEnFahrenheit(tableauC) qui retourne un nouveau tableau contenant les conversions en Fahrenheit (formule : °F = °C × 9/5 + 32).

Démarche : Parcourir le tableau d'origine avec une boucle, appliquer la formule à chaque élément, et stocker le résultat dans un nouveau tableau. 

function convertirEnFahrenheit(tableauC) {
    let tableauF = [];
    for (let i = 0; i < tableauC.length; i++) {
        let f = tableauC[i] * 9/5 + 32;
        tableauF.push(f);
    }
    return tableauF;
}
let celsius = [0, 20, 30, 40];
let fahrenheit = convertirEnFahrenheit(celsius);
console.log(fahrenheit); // [32, 68, 86, 104]
📌 À retenir : push() ajoute un élément à la fin du tableau. On peut aussi modifier directement un tableau existant.

Exemple 3 : Boucle while et recherche dans un tableau

Écrire une fonction rechercherValeur(tab, cible) qui retourne l'indice de la première occurrence de cible dans le tableau, ou -1 si non trouvée.

Solution : Utiliser une boucle while ou for. On parcourt jusqu'à trouver ou avoir épuisé le tableau. 

function rechercherValeur(tab, cible) {
    let i = 0;
    while (i < tab.length) {
        if (tab[i] === cible) {
            return i;   // indice trouvé
        }
        i++;
    }
    return -1; // non trouvé
}
let couleurs = ["rouge", "vert", "bleu"];
console.log(rechercherValeur(couleurs, "vert")); // 1
console.log(rechercherValeur(couleurs, "jaune")); // -1
🎯 Attention : Les indices des tableaux commencent à 0 en JavaScript. La fonction retourne l'indice correspondant.

⚠️ Pièges courants & Bonnes pratiques

  • Confusion entre = et == / === : = est l'affectation ; == compare les valeurs (avec conversion de type) ; === compare valeur ET type (recommandé).
  • Indices hors limites : Accéder à tab[tab.length] retourne undefined, mais n'affiche pas d'erreur. Toujours vérifier i < tab.length.
  • Boucle infinie : Oublier d'incrémenter le compteur ou une condition toujours vraie (while(true) sans break).
  • Fonction sans return : Si une fonction ne possède pas d'instruction return, elle retourne undefined.
  • Variables globales par inadvertance : Oublier let, const, var crée une variable globale. Toujours déclarer ses variables.
