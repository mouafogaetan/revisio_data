📀 Installation d'un système d'exploitation

Informatique · Première C, D, E · Module 1 - Environnement numérique

📌 Éléments clés à retenir

💻 Qu'est-ce qu'un système d'exploitation (SE) ?

Un système d'exploitation est un logiciel qui gère les ressources matérielles d'un ordinateur et fournit des services aux logiciels applicatifs. Il fait le lien entre l'utilisateur, les programmes et le matériel.

  • Windows (Microsoft) : le plus répandu sur PC, interface graphique conviviale.
  • Linux (open source) : libre, sécurisé, utilisé dans les serveurs et supercalculateurs.
  • macOS (Apple) : système des ordinateurs Mac, réputé pour sa stabilité.

⚙️ Prérequis matériels et supports d'installation

Avant d'installer un SE, il faut vérifier la compatibilité matérielle (processeur, mémoire RAM, espace disque, architecture 32/64 bits). L'installation se fait généralement à partir d'un support bootable :

  • Clé USB bootable (méthode la plus rapide actuellement)
  • DVD d'installation (pour les ordinateurs équipés d'un lecteur)
  • Image ISO téléchargée depuis le site officiel de l'éditeur
Figure 2 : Supports d'installation courants (clé USB, DVD)

🗂️ Notion de partition (primaire, secondaire, logique)

Une partition est une subdivision logique d'un disque dur. Cela permet d'organiser les données et d'installer plusieurs SE sur une même machine.

  • Partition primaire : partition qui peut contenir un système d'exploitation (au maximum 4 sur un disque MBR).
  • Partition secondaire : partie du disque dur dédiée à une utilisation spécifique (stockage de données, sauvegarde).
  • Partition logique : subdivision d'une partition étendue, utilisée pour contourner la limite des 4 primaires.

💡 À retenir : Lors de l'installation, il faut souvent créer une partition pour le SE (ex: C:\ sous Windows) et éventuellement une partition pour les données personnelles.

🔧 Étapes d'installation d'un système d'exploitation

Procédure générale (exemple avec Windows/Linux)

L'installation suit un déroulement logique, que l'on retrouve avec des variantes selon le SE choisi.

  1. Configuration du BIOS/UEFI : régler l'ordre de démarrage pour booter sur la clé USB ou le DVD.
  2. Démarrage sur le support d'installation : l'ordinateur charge l'environnement d'installation.
  3. Choix de la langue, du clavier et des préférences régionales.
  4. Partitionnement du disque : créer/supprimer/formater les partitions nécessaires.
  5. Copie des fichiers système : le SE se déploie sur la partition cible.
  6. Configuration initiale : nom de l'ordinateur, mot de passe administrateur, fuseau horaire.
  7. Finalisation et redémarrage : l'ordinateur démarre sur le nouveau système.
💬 Astuce : Avant toute installation, sauvegardez vos données importantes ! Une manipulation sur les partitions peut effacer définitivement des fichiers.

Exemple concret : Partitionnement sous Windows

Lors de l'installation de Windows 10/11, l'utilisateur arrive à l'écran "Où voulez-vous installer Windows ?". Il voit un disque non partitionné de 500 Go.

Solution : Il clique sur "Nouveau" pour créer une partition système (ex: 150 Go pour C:\) qui deviendra la partition primaire. Le reste (350 Go) peut rester non alloué pour une future partition de données ou être formaté immédiatement. Windows crée automatiquement des partitions supplémentaires (EFI, MSR) sur les disques GPT.

💡 La partition système doit être formatée en NTFS (sous Windows) ou en ext4 (sous Linux). Le format détermine la façon dont les fichiers sont organisés.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de rendre la clé USB bootable (simple copie de l'ISO ne suffit pas, il faut utiliser un outil comme Rufus ou Etcher).
  • Attention : Confondre partition primaire et partition secondaire. La secondaire n'est pas destinée à héberger un SE, c'est une zone de stockage.
  • Méprise : Penser que "formater" efface définitivement les données (en réalité, des outils de récupération peuvent les retrouver tant qu'elles ne sont pas écrasées).
  • Ne pas négliger : Vérifier l'architecture du processeur (32 bits ou 64 bits) pour télécharger la bonne version du SE.

📝 Récapitulatif pour l'examen

Définitions incontournables

  • Système d'exploitation : logiciel de base qui gère le matériel et lance les applications.
  • Partition secondaire : partie d'un disque dur dédiée à une utilisation spécifique (données, sauvegarde).
  • Partition primaire : partition sur laquelle on peut installer un OS et qui est reconnue au démarrage.
  • Boot (amorçage) : processus de démarrage de l'ordinateur qui charge le SE en mémoire.
