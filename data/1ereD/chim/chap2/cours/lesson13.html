Leçon 13 : Halogénation des Alcènes & Alcynes

Chimie Organique · Première C & D · Réactions d'addition

📌 Éléments clés à retenir

1. Addition du chlorure d’hydrogène (HCl) sur les alcènes

Les alcènes (hydrocarbures insaturés avec une double liaison C=C) réagissent avec HCl pour former un chloroalcane. La réaction obéit à la règle de Markovnikov : l'atome d'hydrogène se fixe sur le carbone le plus hydrogéné (celui qui porte déjà le plus d'atomes H).

$R\text{-}CH=CH_2 + HCl \rightarrow R\text{-}CHCl\text{-}CH_3$

Exemple type : Propène + HCl → 2-chloropropane (majoritaire)

$CH_3\text{-}CH=CH_2 + HCl \rightarrow CH_3\text{-}CHCl\text{-}CH_3$
  • Réaction d'addition électrophile.
  • Le produit majoritaire est celui où le chlore est porté par le carbone le plus substitué.

2. Addition du chlorure d’hydrogène (HCl) sur les alcynes

Les alcynes (triple liaison C≡C) subissent une addition successive de deux molécules de HCl.

$R\text{-}C \equiv CH + HCl \rightarrow R\text{-}CCl=CH_2$ (chloroalcène)
$R\text{-}CCl=CH_2 + HCl \rightarrow R\text{-}CCl_2\text{-}CH_3$ (dichloroalcane)

Exemple avec l’acétylène :

$HC \equiv CH + HCl \xrightarrow{\text{HgCl}_2} CH_2=CHCl$ (chlorure de vinyle)

Le chlorure de vinyle est le monomère du PVC (polychlorure de vinyle).

3. Addition du dichlore (Cl₂) sur les alcènes et alcynes

En l'absence de lumière (obscurité), le dichlore s'additionne sur la double liaison pour former un dérivé 1,2-dichloré. C'est une réaction caractéristique des insaturations.

$CH_2=CH_2 + Cl_2 \rightarrow CH_2Cl\text{-}CH_2Cl$ (1,2-dichloroéthane)

Sur un alcyne : addition successive de deux moles de Cl₂ :

$HC \equiv CH + Cl_2 \rightarrow CHCl=CHCl$ (dichloroéthène)
$CHCl=CHCl + Cl_2 \rightarrow CHCl_2\text{-}CHCl_2$ (tétrachloroéthane)
  • Réaction stéréospécifique : en général addition anti (produit trans majoritaire).
  • Le dibrome (Br₂) donne une décoloration caractéristique (test des insaturations).

4. Test d’identification des doubles liaisons : décoloration du dibrome

Le dibrome en solution (eau de brome de couleur orange-brun) se décolore rapidement en présence d’un alcène ou d’un alcyne. Cette réaction permet de distinguer un hydrocarbure saturé (alcane) d’un insaturé.

$C_2H_4 + Br_2 \rightarrow CH_2Br\text{-}CH_2Br$ (décoloration)
🧪 Protocole simple : Ajouter quelques gouttes d’eau de dibrome à l’échantillon. Si la coloration brune disparaît rapidement → présence de double ou triple liaison.

Remarque : Les alcynes décolorent aussi le dibrome, mais parfois plus lentement selon la structure.

💡 Exemples de compréhension

Exemple 1 : Application de la règle de Markovnikov

On fait réagir du 2-méthylpropène (isobutène) avec du chlorure d’hydrogène gazeux. Écrire l’équation de la réaction et identifier le produit majoritaire.

Résolution : Le 2-méthylpropène a pour formule : $(CH_3)_2C=CH_2$. La double liaison se situe entre C2 (porteur de deux groupes méthyle) et C1 (porteur de deux hydrogènes).
Selon Markovnikov, H s’ajoute sur le carbone le plus hydrogéné (C1), et Cl sur le carbone le plus substitué (C2).
Équation :
$(CH_3)_2C=CH_2 + HCl \rightarrow (CH_3)_2CCl\text{-}CH_3$ (2-chloro-2-méthylpropane).
💡 Astuce : "Le riche devient plus riche" : le carbone qui a déjà plus d'alkyles reçoit l'halogène.

Exemple 2 : Test d’identification par le dibrome

Un laborantin dispose de deux tubes : l’un contient de l’hexane (C₆H₁₄), l’autre du cyclohexène (C₆H₁₀). Comment distinguer les deux liquides incolores ?

Méthode : Ajouter de l’eau de dibrome (brun-rouge) dans chaque tube.
- Dans l’hexane (alcane) : aucune réaction, la couleur brune persiste.
- Dans le cyclohexène (alcène) : décoloration immédiate due à l’addition de Br₂ sur la double liaison.
Équation :
$\text{Cyclohexène} + Br_2 \rightarrow \text{1,2-dibromocyclohexane}$ (décoloration).
🔬 Le dibrome est aussi un test rapide pour détecter les faux "saturés" dans les huiles (indice de brome).

Exemple 3 : Addition du dichlore sur l’éthylène (obscurité)

On fait barboter du dichlore gazeux dans une solution d’éthylène dans le tétrachlorure de carbone, à l’abri de la lumière. Quel est le produit obtenu ? Écrire l’équation bilan.

Réponse : Il se forme du 1,2-dichloroéthane (liquide), un solvant chloré industriel important.
Équation :
$CH_2=CH_2 + Cl_2 \xrightarrow{\text{obscurité}} CH_2Cl\text{-}CH_2Cl$
Mécanisme : addition électrophile en deux étapes via un ion chloronium.
⚠️ Attention : à la lumière, le chlore peut provoquer des réactions de substitution radicalaire (comme avec les alcanes). C’est pourquoi l’obscurité est spécifiée.

⚠️ Pièges courants & Confusions à éviter

  • Erreur fréquente : Penser que la décoloration du dibrome est spécifique aux seuls alcènes. En réalité, les alcynes et les composés conjugués décolorent aussi.
  • Attention à la lumière : L’addition de Cl₂ se fait à l’obscurité. Sous lumière UV, on observe une substitution radicalaire (comme avec le méthane).
  • Ne pas confondre : L’addition de HCl sur un alcène asymétrique donne toujours un seul produit majoritaire (Markovnikov), mais attention aux réarrangements possibles en milieu très acide.
  • Oubli des indices : Lors de la nomenclature des produits halogénés, il faut indiquer la position des atomes de chlore (ex: 2-chloropropane, pas chloropropane).
  • Piège sur l’alcyne : L’addition de HCl sur un alcyne terminal ne s’arrête pas toujours au chloroalcène ; selon les conditions, on peut obtenir le dichloroalcane.

📝 Équations essentielles à mémoriser

Alcène + HCl (Markovnikov)  →  $R\text{-}CHCl\text{-}CH_3$
Alcyne terminal + HCl (1er équiv.)  →  $R\text{-}CCl=CH_2$
Alcène + Br₂ (test)  →  $CH_2Br\text{-}CH_2Br$ (décoloration)
Éthylène + Cl₂ (obscurité)  →  $CH_2Cl\text{-}CH_2Cl$
Acétylène + Cl₂ (excès)  →  $CHCl_2\text{-}CHCl_2$

🔗 Application industrielle : PVC

L’addition de HCl sur l’acétylène (ou sur l’éthylène via un procédé moderne) produit le chlorure de vinyle (monomère). Ce dernier est polymérisé pour donner le PVC (polychlorure de vinyle), utilisé dans les tuyaux, isolants électriques, revêtements.

$CH_2=CHCl \xrightarrow{\text{polymérisation}} \text{-(CH}_2\text{-CHCl)-}_n$
🧪 Lien avec la Leçon 14 : la polymérisation des dérivés halogénés permet d’obtenir des plastiques courants.
