📝 Corrigé officiel – Évaluation séquentielle N°4

Partie A – Savoirs

A1 :
1. Oxydant : espèce qui capte des électrons ; Réducteur : espèce qui cède des électrons ; Réaction redox : transfert d'électrons entre oxydant et réducteur.
2. Règle de gamma : Un réducteur (Red1) réagit spontanément avec un oxydant (Ox2) si le potentiel standard du couple (Ox2/Red2) est supérieur à celui du couple (Ox1/Red1). Soit $E°(Ox2/Red2) > E°(Ox1/Red1)$.
3. Exemples voie sèche : $2Mg + O_2 \to 2MgO$ ; $Mg + Cl_2 \to MgCl_2$.

A2 – Nombres d'oxydation :
$Mn$ dans $MnO_4^-$ : +7 ; $Cr$ dans $Cr_2O_7^{2-}$ : +6 ; $S$ dans $S_2O_3^{2-}$ : +2 (moyenne) ; $N$ dans $NO_3^-$ : +5 ; $Cl$ dans $Cl_2$ : 0 (corps simple). Règles : n.o total d’une molécule = 0 ; d’un ion = charge ; O = -2 sauf peroxydes ; H = +1 généralement.

A3 – Demi-équations en milieu acide :
$MnO_4^- + 8H^+ + 5e^- \to Mn^{2+} + 4H_2O$
$Cr_2O_7^{2-} + 14H^+ + 6e^- \to 2Cr^{3+} + 7H_2O$
$NO_3^- + 4H^+ + 3e^- \to NO + 2H_2O$
$SO_4^{2-} + 4H^+ + 2e^- \to SO_2 + 2H_2O$.

Partie B – Piles

1. $E°(Cu^{2+}/Cu) = 0,34V$ ; $E°(Zn^{2+}/Zn) = -0,76V$. $0,34 > -0,76$ donc $Cu^{2+}$ oxyde Zn : réaction spontanée. Équation : $Zn + Cu^{2+} \to Zn^{2+} + Cu$.

2. Pile Argent-Cuivre : $E°_{pile} = E°(Ag^+/Ag) - E°(Cu^{2+}/Cu) = 0,80 - 0,34 = 0,46 V$. Anode (oxydation) : $Cu \to Cu^{2+} + 2e^-$ ; Cathode (réduction) : $2Ag^+ + 2e^- \to 2Ag$ ; Bilan : $Cu + 2Ag^+ \to Cu^{2+} + 2Ag$. Notation : $\Theta Cu/Cu^{2+} // Ag^+/Ag \oplus$.

3. ESH : électrode de platine plongeant dans une solution $H^+$ (1M) et saturée en dihydrogène (1 bar). $E°=0$ par convention ; sert de référence pour mesurer les potentiels standards.

4. Pont salin : assure la neutralité électrique des solutions, permet la migration des ions et ferme le circuit électrique.

Partie C – Permanganate & Fer

1. $MnO_4^-$ (oxydant) est réduit ; $Fe^{2+}$ (réducteur) est oxydé.
2. Bilan : $MnO_4^- + 5Fe^{2+} + 8H^+ \to Mn^{2+} + 5Fe^{3+} + 4H_2O$.
3. $E°(Fe^{3+}/Fe^{2+}) = 0,72V > E°(I_2/I^-)=0,54V$ donc $Fe^{3+}$ oxyde $I^-$ en $I_2$ spontanément : $2Fe^{3+} + 2I^- \to 2Fe^{2+} + I_2$.
4. Dismutation : une même espèce chimique est à la fois oxydée et réduite. Exemple : $2H_2O_2 \to 2H_2O + O_2$ (peroxyde d’hydrogène).

Partie D – Industrie

1. Aluminothermie : $Fe_2O_3 + 2Al \to Al_2O_3 + 2Fe$. L’alumine ($Al_2O_3$) est le sous-produit. Procédé exothermique utilisé pour souder les rails.

2. Fabrication $H_2SO_4$ :
- Combustion : $S + O_2 \to SO_2$
- Oxydation catalytique : $2SO_2 + O_2 \rightleftharpoons 2SO_3$ (catalyseur $V_2O_5$)
- Hydratation : $SO_3 + H_2O \to H_2SO_4$.

3. Haut-fourneau : $Fe_2O_3 + 3CO \to 2Fe + 3CO_2$. On obtient la fonte (fer + carbone) puis l’acier par affinage.
