Leçon 35 : Oxydoréduction – Définitions fondamentales

Chimie · Première C & D (Cameroun) · Module 2 : Oxydoréduction

📌 Définitions fondamentales à maîtriser

⚡ Oxydation et Réduction (selon le transfert d'électrons)

🔹 Oxydation : perte d’électrons par une espèce chimique.
🔹 Réduction : gain d’électrons par une espèce chimique.

Ces deux phénomènes sont indissociables : toute perte d’électrons s’accompagne nécessairement d’un gain d’électrons par une autre espèce. On parle de réaction d’oxydoréduction (redox) lorsqu’il y a transfert simultané d’électrons d’une espèce vers une autre.

  • Moyen mnémotechnique : "Oxyde = perd des électrons" (Oxyde, on perd le "o" ? pensez à perte). Réduction = réception d’électrons.
  • Exemple type : Zn → Zn²⁺ + 2e⁻ (oxydation du zinc) et Cu²⁺ + 2e⁻ → Cu (réduction des ions cuivre).
Illustration du transfert d'électrons lors d'une réaction redox.

🧪 Oxydant et Réducteur : les acteurs de la réaction

🔹 Oxydant : espèce chimique capable de capter des électrons (elle est réduite).
🔹 Réducteur : espèce chimique capable de céder des électrons (elle est oxydée).

Dans une réaction redox, l’oxydant d’un couple réagit avec le réducteur d’un autre couple. L’oxydant se transforme en réducteur conjugué ; le réducteur se transforme en oxydant conjugué.

  • Exemple simple : dans Zn + Cu²⁺ → Zn²⁺ + Cu :
    • L’ion Cu²⁺ est l’oxydant (il capte 2 e⁻).
    • Le métal Zn est le réducteur (il cède 2 e⁻).
  • Astuce : L’oxydant oxyde le réducteur ; le réducteur réduit l’oxydant.

🔄 Réaction d'oxydoréduction (redox)

Une réaction d’oxydoréduction est une transformation chimique au cours de laquelle il y a transfert d’électrons d’une espèce chimique (le réducteur) vers une autre espèce (l’oxydant). Elle met en jeu deux couples oxydant/réducteur.

📌 Écriture symbolique : Red₁ + Ox₂ → Ox₁ + Red₂
où Ox₁/Red₁ et Ox₂/Red₂ sont les deux couples.

On peut toujours décomposer la réaction globale en deux demi-équations électroniques :

  • Oxydation : Red → Ox + n e⁻
  • Réduction : Ox' + n e⁻ → Red'

💡 Exemples de compréhension (expériences du programme)

Exemple n°1 : Action de l’acide chlorhydrique sur le zinc

On plonge un ruban de zinc (Zn) dans une solution d’acide chlorhydrique (H₃O⁺, Cl⁻). On observe un dégagement de dihydrogène (test à la flamme : pop caractéristique).

🔍 Analyse redox :
- Oxydation : Zn (métal) perd 2 électrons → Zn → Zn²⁺ + 2 e⁻ (Zn est le réducteur).
- Réduction : Les ions oxonium H₃O⁺ captent des électrons pour former H₂ : 2 H₃O⁺ + 2 e⁻ → H₂ + 2 H₂O (H₃O⁺ est l’oxydant).
- Équation bilan : Zn + 2 H₃O⁺ → Zn²⁺ + H₂ + 2 H₂O.
💡 Astuce : Le test au dégagement de dihydrogène (bulles) indique la présence d’une réaction redox avec un métal actif. Le métal est toujours le réducteur.
Expérience : action de l’acide chlorhydrique sur le zinc.

Exemple n°2 : Dépôt de cuivre sur une lame de zinc

Une lame de zinc est plongée dans une solution bleue de sulfate de cuivre (Cu²⁺). La couleur bleue disparaît progressivement et un dépôt rouge-brun de cuivre métallique apparaît.

🧪 Identification des rôles :
- Le zinc (Zn) s’oxyde : Zn → Zn²⁺ + 2 e⁻ → réducteur.
- Les ions cuivre (Cu²⁺) se réduisent : Cu²⁺ + 2 e⁻ → Cu → oxydant.
- Équation bilan : Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s).
🧠 À retenir : Un métal peut réduire les ions d’un métal moins réducteur. Ici, le zinc (réducteur) cède ses électrons à Cu²⁺.
Observation : le dépôt de cuivre prouve la réduction des ions Cu²⁺.

Exemple n°3 : Réaction entre ions fer (III) et iodure

On ajoute une solution d’iodure de potassium (I⁻) à une solution jaune d’ions fer (III) (Fe³⁺). La solution devient brune (apparition de diiode I₂). Test à l’empois d’amidon : coloration bleu-noir.

⚖️ Identification :
- Les ions I⁻ (réducteur) sont oxydés en I₂ : 2 I⁻ → I₂ + 2 e⁻
- Les ions Fe³⁺ (oxydant) sont réduits en Fe²⁺ : Fe³⁺ + 1 e⁻ → Fe²⁺
- Après équilibrage : 2 Fe³⁺ + 2 I⁻ → 2 Fe²⁺ + I₂
🔬 L’oxydant n’est pas forcément un ion métallique : ici Fe³⁺ joue le rôle d’oxydant, I⁻ celui de réducteur.

⚠️ Pièges courants & erreurs à éviter

  • Confondre oxydant et réducteur : Retenez que l’oxydant est celui qui capte des électrons (donc son nombre d’oxydation diminue). Le réducteur cède des électrons (son nombre d’oxydation augmente).
  • Oublier que toute oxydation s’accompagne d’une réduction : Une réaction ne peut être uniquement une oxydation sans réduction simultanée. Les deux sont indissociables.
  • Écrire une demi-équation sans respecter la conservation des charges : Dans une demi-équation, la charge totale doit être équilibrée (ex: Zn → Zn²⁺ + 2e⁻). Vérifiez toujours la somme des charges.
  • Croire que seul un métal peut être réducteur : De nombreuses espèces chimiques (ions iodure I⁻, fer II Fe²⁺, etc.) sont des réducteurs courants.
  • Négliger l’importance du milieu (acide, basique) : Certaines demi-équations nécessitent H₃O⁺ ou H₂O pour être équilibrées (cela sera vu dans les leçons suivantes).

📖 Tableau récapitulatif – Les 4 notions clés

Notion Définition simple Variation (perte/gain) Exemple
Oxydation Perte d’électrons Le réducteur s’oxyde Zn → Zn²⁺ + 2e⁻
Réduction Gain d’électrons L’oxydant est réduit Cu²⁺ + 2e⁻ → Cu
Oxydant Espèce qui capte des e⁻ Son nombre d’oxydation ↓ Cu²⁺, H₃O⁺, Fe³⁺
Réducteur Espèce qui perd des e⁻ Son nombre d’oxydation ↑ Zn, I⁻, Fe²⁺

🔁 Une réaction redox met toujours en jeu un oxydant (qui se réduit) et un réducteur (qui s’oxyde).

✏️ Exercice rapide d’application

S’entraîner : identifier l’oxydant et le réducteur

Soit la réaction : 2 Ag⁺(aq) + Cu(s) → 2 Ag(s) + Cu²⁺(aq)
Cette réaction est utilisée dans la pile cuivre-argent.

✅ Correction :
- Le cuivre Cu(s) perd 2 électrons : Cu → Cu²⁺ + 2e⁻ → c’est le réducteur (il s’oxyde).
- L’ion argent Ag⁺ capte 1 électron par atome : Ag⁺ + 1e⁻ → Ag → c’est l’oxydant (il est réduit).
- Donc oxydant = Ag⁺ ; réducteur = Cu(s).
📌 Pour vous entraîner, repérez toujours quel élément voit son nombre d’oxydation augmenter (oxydation) et lequel le voit diminuer (réduction).
