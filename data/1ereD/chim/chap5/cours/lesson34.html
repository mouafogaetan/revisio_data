Leçon 34 : Généralités sur les fonctions

Mathématiques · Seconde (Cameroun) · Révisions BAC/Examen

📌 Éléments clés à retenir

1. Définition d'une fonction

Une fonction $f$ est une relation qui associe à chaque nombre $x$ d'un ensemble $D$ (appelé ensemble de définition) un unique nombre $y = f(x)$. On note : $f : D \rightarrow \mathbb{R}$.

  • $x$ est la variable ou l'antécédent.
  • $y = f(x)$ est l'image de $x$ par $f$.
  • Exemple : $f(x) = x^2$ ; l'image de 3 est $f(3)=9$.
  • Image = $f(x)$    Antécédent(s) = solution(s) de $f(x)=k$

2. Ensemble de définition

L'ensemble des réels $x$ pour lesquels $f(x)$ existe (calculable). Règles à respecter :

  • Pas de division par zéro : le dénominateur $\neq 0$.
  • Expression sous une racine carrée $\sqrt{u}$ : $u \geq 0$.
  • Pour une fonction définie par morceaux, prendre l'intersection des intervalles.
  • Exemple : $f(x) = \frac{1}{x-2}$ → $D_f = \mathbb{R} \setminus \{2\}$.
  • $f(x) = \sqrt{x+3}$ → $D_f = [-3; +\infty[$.

3. Courbe représentative et variations

La courbe $\mathcal{C}_f$ est l'ensemble des points $(x ; f(x))$ dans un repère. Les variations décrivent où la fonction croît ou décroît.

  • Fonction croissante : si $a < b$ alors $f(a) \le f(b)$.
  • Fonction décroissante : si $a < b$ alors $f(a) \ge f(b)$.
  • Tableau de variations : outil essentiel pour visualiser la tendance.
  • Extremum : maximum ou minimum sur un intervalle.

💡 Exemples de compréhension

Exemple 1 : Déterminer image et antécédents

Soit $f(x) = 3x - 5$. Calculez l'image de $-2$ puis le(s) antécédent(s) de $4$.

Résolution :
• Image de $-2$ : $f(-2) = 3 \times (-2) - 5 = -6 - 5 = -11$.
• Antécédent(s) de $4$ : on résout $f(x)=4$ → $3x-5=4$ → $3x=9$ → $x=3$.
Donc l'antécédent de $4$ est $3$.
💬 Astuce : Pour trouver l'image, on remplace $x$ par la valeur donnée. Pour l'antécédent, on résout $f(x)=k$.

Exemple 2 : Ensemble de définition

Déterminez $D_f$ pour $f(x) = \frac{\sqrt{x+5}}{x-1}$.

Calcul :
• Condition de la racine : $x+5 \ge 0 \Rightarrow x \ge -5$.
• Condition du dénominateur : $x-1 \neq 0 \Rightarrow x \neq 1$.
• Donc $D_f = [-5 ; 1[ \cup ]1 ; +\infty[$.
💡 On représente sur un axe gradué pour visualiser l'intersection des conditions.

Exemple 3 : Lecture graphique et variations

On donne la courbe d'une fonction $g$ définie sur $[-3; 4]$. Déterminer les intervalles où $g$ est croissante et le maximum.

Explication : En observant la courbe, $g$ est croissante de $x=-3$ jusqu'à $x=0$, puis décroissante de $0$ à $4$. Le maximum est $g(0)=5$ (valeur la plus haute sur l'intervalle).
On peut également dresser un tableau de signes de la dérivée (en classe supérieure).
📈 Lorsqu'on lit une courbe, on regarde le sens de la courbe de gauche à droite : "qui monte" → croissante, "qui descend" → décroissante.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre image et antécédent. L'image se lit sur l'axe des ordonnées, l'antécédent sur l'axe des abscisses.
  • Attention : Oublier d'exclure les valeurs qui annulent un dénominateur ou qui rendent négatif un radicande (racine carrée).
  • Ne pas confondre : Croissante n'implique pas que la fonction soit toujours positive ; cela concerne seulement l'ordre.
  • Piège graphique : Ne pas oublier les points "trous" ou exclus dans la courbe (cercle vide).

📊 Fonctions de référence à connaître

Fonctions usuelles

  • $f(x)=ax+b$ (affine) : droite, variations selon le signe de $a$.
  • $f(x)=x^2$ (carré) : décroissante sur $]-\infty;0]$, croissante sur $[0;+\infty[$.
  • $f(x)=\frac{1}{x}$ (inverse) : décroissante sur $]-\infty;0[$ et sur $]0;+\infty[$ ; $D_f = \mathbb{R}^*$.
  • $f(x)=\sqrt{x}$ : définie sur $[0;+\infty[$, croissante.
🧠 Mémoriser leurs courbes et ensembles de définition facilite la résolution de nombreux problèmes.
