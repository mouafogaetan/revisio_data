Leçon 51 : Notion de nombre d'oxydation (n.o.)

Chimie · Première C & D · Oxydoréduction par voie sèche

📌 Définition et règles de calcul

Nombre d’oxydation (n.o.) : charge fictive

Le nombre d’oxydation (ou degré d’oxydation) est la charge électrique fictive qu’un atome posséderait si toutes ses liaisons avec d’autres atomes étaient totalement ioniques (les électrons de liaison sont attribués à l’élément le plus électronégatif). Il permet de suivre les transferts d’électrons lors d’une réaction d’oxydoréduction.

  • On le note n.o. ou deg. ox.
  • Il peut être positif, négatif ou nul selon l’atome et son environnement.
  • Une augmentation du n.o. correspond à une oxydation (perte d’électrons).
  • Une diminution du n.o. correspond à une réduction (gain d’électrons).

Règles de détermination (à connaître par cœur)

  • Règle 1 – Corps simple : n.o. = 0 (ex : $O_2$, $Cl_2$, $Na$, $Fe$, $S_8$).
  • Règle 2 – Ion monoatomique : n.o. = charge de l’ion (ex : $Na^+$ → +1 ; $Cl^-$ → -1 ; $Al^{3+}$ → +3).
  • Règle 3 – Atome d’hydrogène : n.o.(H) = +1 (sauf dans les hydrures métalliques comme $NaH$ où n.o.(H) = -1).
  • Règle 4 – Atome d’oxygène : n.o.(O) = -2 (sauf dans les peroxydes $H_2O_2$ où n.o.(O) = -1, et dans $OF_2$ où n.o.(O) = +2).
  • Règle 5 – Molécule neutre : somme des n.o. de tous les atomes = 0.
  • Règle 6 – Ion polyatomique : somme des n.o. = charge de l’ion.
  • Priorité : En cas de conflit, les règles 5 et 6 priment (neutralité électrique).
💡 Astuce mnémotechnique : “Corps simple zéro, ion = sa charge, H presque toujours +1 (sauf hydrure), O presque toujours -2 (sauf peroxyde)”.

Schéma illustratif – transfert électronique et n.o.

Un atome dont le nombre d’oxydation augmente est oxydé (perd des e⁻). S’il diminue, il est réduit (gagne des e⁻).

💡 Calculs pas à pas

Exemple 1 : Déterminer n.o. du soufre dans $H_2SO_4$

Acide sulfurique : molécule neutre, atomes H, S, O.

Résolution :
• Règle H : n.o.(H) = +1 → pour 2H : contribution totale = +2.
• Règle O : n.o.(O) = -2 → pour 4O : contribution totale = -8.
• Somme des n.o. = 0 (molécule neutre) → $2 \times (+1) + n.o.(S) + 4 \times (-2) = 0$.
• $+2 + n.o.(S) - 8 = 0$ → $n.o.(S) - 6 = 0$ → $n.o.(S) = +6$.
✅ Le soufre est au degré d’oxydation +VI.
💬 Toujours commencer par les atomes dont le n.o. est fixé (H, O sauf exceptions) puis résoudre l’équation.

Exemple 2 : Ion permanganate $MnO_4^-$

Ion polyatomique de charge -1, constitué de Mn et O.

Résolution :
• Oxygène : n.o.(O) = -2 (pas de peroxyde ici). Contribution 4O = -8.
• Charge totale de l’ion = -1 → $n.o.(Mn) + 4 \times (-2) = -1$.
• $n.o.(Mn) - 8 = -1$ → $n.o.(Mn) = +7$.
✅ Le manganèse est au degré +VII (très oxydant).

Exemple 3 : Cas particulier – peroxyde d’hydrogène $H_2O_2$

Molécule neutre, mais l’oxygène est dans un état particulier (peroxyde).

Résolution :
• Règle H : n.o.(H) = +1 → 2H = +2.
• Somme n.o. = 0 → $+2 + 2 \times n.o.(O) = 0$ → $2 \times n.o.(O) = -2$ → $n.o.(O) = -1$.
✅ Ici l’oxygène a un n.o. -1 (exception à la règle -2). À mémoriser pour les peroxydes.
🧪 Dans $H_2O$ (eau) : n.o.(O) = -2. Dans $H_2O_2$ (eau oxygénée) : n.o.(O) = -1.

Exemple 4 : Ion dichromate $Cr_2O_7^{2-}$

Ion polyatomique courant en oxydoréduction (couleur orangée).

Résolution :
• Oxygène : n.o.(O) = -2 → 7O = -14.
• Charge de l’ion = -2 → $2 \times n.o.(Cr) + (-14) = -2$.
• $2 \times n.o.(Cr) = +12$ → $n.o.(Cr) = +6$.
✅ Le chrome est au degré +VI.

⚡ Utilisation des nombres d’oxydation

Identifier une réaction d’oxydoréduction

Une réaction est une réaction redox si et seulement si les nombres d’oxydation des atomes d’au moins un élément varient entre les réactifs et les produits.

  • Oxydant : élément dont le n.o. diminue (capte des électrons).
  • Réducteur : élément dont le n.o. augmente (perd des électrons).
  • Dismutation : un même élément à la fois s’oxyde et se réduit (son n.o. augmente chez certains atomes et diminue chez d’autres). Exemple : $2H_2O_2 \rightarrow 2H_2O + O_2$ (l’oxygène passe de -1 à 0 et à -2).

Méthode d’équilibrage par variation du n.o.

Pour équilibrer une équation redox (voie sèche ou humide) :

  • 1️⃣ Déterminer les n.o. de tous les atomes dans réactifs et produits.
  • 2️⃣ Identifier les éléments dont le n.o. change.
  • 3️⃣ Calculer la variation du n.o. par atome (Δn.o.).
  • 4️⃣ Égaliser le nombre total d’électrons échangés : multiplier chaque demi-réaction par un coefficient approprié.
  • 5️⃣ Ajuster les autres atomes (hors H et O) puis compléter avec H⁺, H₂O (milieu acide) ou OH⁻, H₂O (milieu basique).
✨ En voie sèche, on utilise directement les coefficients stoechiométriques pour que le gain total d’électrons soit égal à la perte totale.

Application rapide : combustion du magnésium dans $O_2$

$2Mg + O_2 \rightarrow 2MgO$

• n.o.(Mg) : 0 → +2 (augmentation → Mg est réducteur, s’oxyde).
• n.o.(O) : 0 → -2 (diminution → O₂ est oxydant, se réduit).
• Transfert de 4 électrons (2 Mg perdent 2e⁻ chacun, O₂ gagne 4e⁻).
✅ La variation des n.o. confirme la nature redox.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier que dans un ion polyatomique, la somme des n.o. doit être égale à la charge de l’ion (et pas zéro).
  • Attention : Confondre n.o. avec la charge réelle. Le n.o. est une fiction comptable utile pour le suivi des électrons.
  • Piège : Dans $OF_2$, l’oxygène a un n.o. = +2 (car le fluor plus électronégatif impose n.o.(F) = -1). Ne pas appliquer -2 systématiquement.
  • Erreur classique : Penser que le nombre d’oxydation d’un atome dans une molécule est toujours égal à sa valence. Ce n’est pas équivalent !
  • À retenir : Un même élément peut avoir plusieurs n.o. différents selon le composé ($MnO_4^-$ : Mn VII ; $Mn^{2+}$ : Mn II).
