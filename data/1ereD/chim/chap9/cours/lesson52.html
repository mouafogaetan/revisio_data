Leçon 52 : Utilisation des nombres d'oxydation

Chimie – Première C & D | Module 2 : Oxydoréduction par voie sèche

📌 Identifier une réaction redox grâce au nombre d'oxydation

🔍 Variation du n.o. : signature d'une réaction redox

Une réaction d'oxydoréduction s'accompagne toujours d'une modification des nombres d'oxydation de certains atomes. Si au cours d'une transformation chimique les nombres d'oxydation ne changent pas, il s'agit d'une réaction non redox (acido-basique, précipitation…).

  • Oxydation : augmentation du nombre d'oxydation d'un élément (perte d'électrons).
  • Réduction : diminution du nombre d'oxydation d'un élément (gain d'électrons).
  • L'oxydant est l'espèce dont un élément voit son n.o. diminuer ; le réducteur voit son n.o. augmenter.

🧩 Dismutation (auto-oxydoréduction)

Une dismutation est un cas particulier où le même élément à la fois s'oxyde et se réduit. Exemple classique : le chlore en milieu basique : $Cl_2 + 2OH^- \rightarrow ClO^- + Cl^- + H_2O$. Le chlore (n.o. 0) donne des ions hypochlorite $ClO^-$ (n.o. +I) et chlorure $Cl^-$ (n.o. -I).

💡 Astuce : Pour reconnaître une dismutation, vérifiez si le même élément apparaît avec au moins deux n.o. différents dans les produits.

⚖️ Équilibrer une équation redox par la méthode des variations du n.o.

Méthode systématique pour équilibrer sans demi-équations en milieu acide ou basique (voie sèche ou humide). Étapes :

  • 1. Attribuer les nombres d'oxydation à tous les atomes.
  • 2. Identifier les éléments dont le n.o. change.
  • 3. Calculer la variation totale du n.o. par atome (Δ positive pour oxydation, négative pour réduction).
  • 4. Égaliser les variations totales en multipliant par des coefficients appropriés.
  • 5. Ajuster les atomes hors oxygène/hydrogène (si besoin) puis équilibrer O avec $H_2O$ et H avec $H^+$ (milieu acide) ou $OH^-$ (milieu basique).

Cette méthode est particulièrement utile pour des équations complexes sans détailler les demi-réactions.

💡 Applications pratiques & exemples

Exemple 1 : Oxydation du fer par le dioxygène (formation de rouille)

Équation non équilibrée : $Fe + O_2 \rightarrow Fe_2O_3$

🔎 Variation des n.o. :
- $Fe$ (corps simple) : n.o. = 0 → dans $Fe_2O_3$, $Fe$ est au n.o. +III (augmentation de 3 par atome).
- $O_2$ (corps simple) : n.o. = 0 → dans $Fe_2O_3$, $O$ est au n.o. -II (diminution de 2 par atome).
Pour équilibrer les variations : chaque atome $Fe$ perd 3 électrons (oxydation), chaque atome $O$ gagne 2 électrons (réduction).
On cherche le PPCM entre 3 et 2 = 6.
- Il faut donc $2$ atomes de $Fe$ (variation totale $2 \times 3 = 6$) et $3$ atomes d'oxygène ($3 \times 2 = 6$).
➜ Équation équilibrée : $4Fe + 3O_2 \rightarrow 2Fe_2O_3$.
📌 À retenir : L'oxygène moléculaire est toujours réduit (n.o. 0 → -II). Le métal s'oxyde.

Exemple 2 : Dismutation de l'eau oxygénée $H_2O_2$

Réaction de décomposition : $H_2O_2 \rightarrow H_2O + O_2$

📐 Calcul des n.o. :
- Dans $H_2O_2$ : H = +I, donc O = -I (car $2 \times (+I) + 2x = 0 \Rightarrow x = -I$).
- Dans $H_2O$ : O = -II (réduction de O : -I → -II).
- Dans $O_2$ : O = 0 (oxydation de O : -I → 0).
Le même élément (oxygène) est à la fois réduit et oxydé → dismutation.
Équation équilibrée : $2H_2O_2 \rightarrow 2H_2O + O_2$.

Exemple 3 : Équilibrage avec la méthode des n.o. – Réaction $Cu + HNO_3 \rightarrow Cu(NO_3)_2 + NO + H_2O$

Réaction classique de dissolution du cuivre dans l'acide nitrique dilué.

1. Attribution des n.o. :
Cu (0) → Cu(+II) dans $Cu(NO_3)_2$ : augmentation de 2.
N dans $HNO_3$ : H = +I, O = -II → $+I + x + 3\times(-II)=0$ → $x = +V$.
N dans $NO$ : O = -II → $x + (-II)=0$ → $x = +II$. Variation : +V → +II = diminution de 3.
2. Égaliser les variations : PPCM(2,3) = 6. Multiplier Cu par 3 (variation totale 3×2=6) et N (NO) par 2 (2×3=6).
➜ $3Cu + 2HNO_3 \rightarrow 3Cu(NO_3)_2 + 2NO + H_2O$ (provisoire).
3. Équilibrer les autres atomes : N : côté gauche 2 atomes N, droite : $3Cu(NO_3)_2$ donne 6 N + 2N de NO = 8 N → ajuster $HNO_3$ à 8. Oxygène et hydrogène : $3Cu + 8HNO_3 \rightarrow 3Cu(NO_3)_2 + 2NO + 4H_2O$. Vérification : O (gauche 8×3=24, droite 3×6=18 + 2×1=2 + 4 = 24). Équation finale correcte.
💬 Conseil : après avoir fixé les coefficients des espèces dont les n.o. varient, équilibrez d'abord les métaux puis les non-métaux, terminez par H et O.

💡 La somme des n.o. dans une molécule neutre est nulle ; dans un ion polyatomique, elle est égale à la charge de l'ion.

⚠️ Pièges courants / Erreurs fréquentes

  • Confondre oxydant et réducteur : L'oxydant est l'espèce dont le n.o. diminue (il est réduit). Le réducteur voit son n.o. augmenter.
  • Oublier les corps simples : Dans un corps simple diatomique ($O_2$, $Cl_2$, $H_2$), le n.o. des atomes est toujours zéro.
  • Mauvaise attribution dans les peroxydes : Dans $H_2O_2$ ou $Na_2O_2$, l'oxygène a un n.o. de -I et non -II. Ne pas appliquer mécaniquement -II.
  • Ne pas vérifier la neutralité électrique globale : Après équilibrage par variation du n.o., pensez à ajuster les $H^+$ ou $OH^-$ et l'eau pour respecter la conservation de la matière et de la charge.
  • Dismutation non détectée : Penser à examiner si un élément apparaît avec des n.o. différents dans les produits alors qu'il est unique dans le(s) réactif(s).

🏭 Application industrielle : réduction du minerai de fer

Haut fourneau – Utilisation des n.o. pour comprendre la réaction

La réduction de l'hématite ($Fe_2O_3$) par le monoxyde de carbone ($CO$) produit du fer métallique et du dioxyde de carbone.

Équation : $Fe_2O_3 + 3CO \rightarrow 2Fe + 3CO_2$
Suivi des nombres d'oxydation :
- Fer : dans $Fe_2O_3$, n.o. = +III ; dans $Fe$ métal, n.o. = 0 → réduction (gain d'électrons).
- Carbone : dans $CO$, O = -II ⇒ C = +II ; dans $CO_2$, C = +IV → oxydation (perte d'électrons).
➜ $CO$ est le réducteur, $Fe_2O_3$ l'oxydant. Cette analyse par les n.o. confirme le transfert d'électrons sans écrire les demi-équations.
