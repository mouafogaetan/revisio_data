Leçon 9 : Les modes verbaux

Français · Première (Séries A, C, D, E, TI) · Probatoire

📌 Éléments clés à retenir

Qu'est-ce qu'un mode verbal ?

Le mode est la manière dont le locuteur présente l'action ou l'état exprimé par le verbe. Il indique l'attitude du sujet par rapport à ce qu'il dit (réalité, doute, ordre, souhait, etc.). En français, on distingue les modes personnels (varient selon les personnes) et impersonnels (invariables).

  • Modes personnels : indicatif, subjonctif, conditionnel, impératif.
  • Modes impersonnels : infinitif, participe, gérondif (étudiés en classe supérieure).
  • Astuce mnémotechnique : "Indicatif = réel, Subjonctif = doute/souhait, Conditionnel = hypothèse/politesse, Impératif = ordre/conseil"

L'impératif : ordre, conseil, prière

L'impératif exprime une injonction (ordre, conseil, demande, prière). Il ne se conjugue qu'à trois personnes : 2e personne du singulier (tu), 1re personne du pluriel (nous) et 2e personne du pluriel (vous). Il n'a pas de sujet exprimé (sauf pour insister).

  • Formation : Présent de l'indicatif sans le pronom sujet (sauf pour les verbes en -er : tu chantes → chante !).
  • Exemples : "Mange tes légumes !" (ordre), "Aidez-moi, s'il vous plaît." (prière), "Conseille-lui de se reposer." (conseil).
  • Attention : Pour les verbes "avoir", "être", "savoir" et "vouloir", l'impératif a des formes particulières : aie, sois, sache, veuille.

Le conditionnel : hypothèse, souhait, atténuation, futur dans le passé

Le conditionnel exprime une action ou un état qui dépend d'une condition, un souhait, une information incertaine ou encore une politesse. Il a deux temps principaux : conditionnel présent et conditionnel passé.

  • Valeur d'hypothèse : "Si j'étais riche, je voyagerais autour du monde." (condition non réalisée)
  • Valeur de souhait : "Je voudrais réussir mon examen." (formule de politesse ou désir)
  • Valeur d'atténuation : "Je désirerais vous parler." (plus poli que "je veux")
  • Valeur de futur dans le passé : "Il a annoncé qu'il viendrait demain." (futur par rapport au passé)
  • Formation : infinitif + terminaisons de l'imparfait (-ais, -ais, -ait, -ions, -iez, -aient).

Le subjonctif : subjectivité, doute, souhait, incertitude

Le subjonctif est le mode du sujet pensant. Il exprime une action envisagée, incertaine, souhaitée ou dépendant d'un sentiment. Il est généralement employé dans une subordonnée introduite par "que".

  • Après des verbes exprimant la volonté, le souhait, l'ordre : "Je veux que tu viennes." / "Il faut que nous partions."
  • Après des verbes de sentiment (joie, peur, regret) : "J'ai peur qu'il ne pleuve." / "Elle regrette que tu sois absent."
  • Après des conjonctions comme : "bien que", "quoique", "pour que", "avant que", "sans que". Exemple : "Bien qu'il soit fatigué, il travaille."
  • Dans des expressions impersonnelles : "Il est important que", "Il est nécessaire que".

L'indicatif : le mode du réel et de la certitude

L'indicatif présente les actions comme réelles, objectives, situées dans le temps. Il est le mode de la déclaration et de l'affirmation. Ses principaux temps sont : présent, passé composé, imparfait, plus-que-parfait, passé simple, futur simple, futur antérieur.

  • Emploi dans les phrases déclaratives : "Le Cameroun est un pays d'Afrique centrale."
  • Emploi dans l'interrogation directe : "Où vas-tu ?"
  • Contrairement au subjonctif, l'indicatif exprime une action certaine : "Je sais qu'il viendra." (certitude) vs "Je doute qu'il vienne." (incertitude → subjonctif)

💡 Exemples de compréhension

Exemple 1 : Distinguer impératif, conditionnel et subjonctif

Consigne : Identifiez le mode des verbes soulignés et justifiez leur emploi.

"Il faut que tu finisses ton devoir. Si tu le terminais à temps, tu pourrais sortir. Dépêche-toi !"

Résolution :
- finisses → subjonctif présent (après "il faut que", exprime la nécessité/obligation).
- pourrais → conditionnel présent (hypothèse liée à une condition : "si tu le terminais").
- Dépêche-toi → impératif présent (ordre ou conseil urgent).

💬 Astuce : Pour repérer le subjonctif, cherchez "que" + un verbe exprimant un souhait, un doute ou une obligation. Pour le conditionnel, repérez "si" ou une idée d'irréel.

Exemple 2 : Valeurs du conditionnel

Texte : "Je voudrais un café, s'il vous plaît. Dans mon rêve, je serais médecin. Il m'avait promis qu'il m'aiderait."

Explication :
- "voudrais" : conditionnel de politesse (atténuation de la demande).
- "serais" : conditionnel de souhait / rêve (irréel du présent).
- "aiderait" : futur dans le passé (action postérieure par rapport à un passé).

💡 Pensez à la valeur de politesse dans les demandes ("je voudrais", "j'aimerais") – c'est du conditionnel, pas de l'imparfait !

Exemple 3 : Subjonctif après les conjonctions

Complétez avec le mode qui convient (indicatif ou subjonctif) : "Bien qu'il (pleuvoir) ______, nous sortirons. Pour que vous (comprendre) ______, je vais expliquer."

Correction :
- "Bien qu'il pleuve" → subjonctif (conjonction d'opposition "bien que" + subjonctif).
- "Pour que vous compreniez" → subjonctif (conjonction de but "pour que" + subjonctif).
Règle : Les conjonctions comme "bien que", "quoique", "pour que", "avant que", "sans que" sont toujours suivies du subjonctif.

📚 Mémorisez la liste des conjonctions qui imposent le subjonctif : "afin que, avant que, bien que, pour que, quoique, sans que, à moins que, pourvu que".

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente 1 : Confondre le conditionnel et le futur. Le futur exprime une certitude ("Je viendrai demain"), le conditionnel une hypothèse ou une politesse ("Je viendrais si j'avais le temps").
  • Erreur fréquente 2 : Utiliser l'indicatif après "il faut que". Toujours le subjonctif : "Il faut que tu ailles" (et non "tu vas").
  • Erreur fréquente 3 : Oublier la terminaison de l'impératif pour les verbes en -er (pas de -s à la 2e pers. sing.) : "Parle !" (et non "parles"). Sauf exceptions : "va, as, aie, sois".
  • Attention : "Si" + imparfait n'introduit jamais le conditionnel après. On dit "Si j'étais riche, je voyagerais" (pas "si je serais").
  • Ne pas confondre : "Je veux que vous partiez" (subjonctif) vs "Je veux partir" (infinitif). Le subjonctif est toujours dans une subordonnée avec "que".

✍️ Entraînement : Reconnaître les modes

Exercice : Identifiez le mode et sa valeur

Lisez les phrases suivantes. Pour chaque verbe conjugué, donnez le mode et justifiez son emploi (valeur).

  1. "Pourvu qu'il ne pleuve pas pendant la cérémonie !"
  2. "Si j'avais su, je n'aurais pas fait cette erreur."
  3. "Fermez la porte et venez ici immédiatement."
  4. "Il est certain que le Cameroun participera à la prochaine Coupe d'Afrique."
  5. "Je souhaiterais obtenir une bourse pour mes études supérieures."

Corrigé :
1. "pleuve" → subjonctif (expression de souhait / crainte avec "pourvu que").
2. "aurais fait" → conditionnel passé (hypothèse irréelle du passé).
3. "Fermez", "venez" → impératif (ordre).
4. "participera" → indicatif futur (certitude, réalité).
5. "souhaiterais" → conditionnel présent (atténuation / politesse).

⭐ Astuce pour l'examen : Lisez la phrase en cherchant les indices (conjonctions, "si", marques de politesse, ponctuation exclamative). Chaque mode a ses mots déclencheurs.
