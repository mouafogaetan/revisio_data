PARTIE 1 : ÉVALUATION DES RESSOURCES (14 points)
📌 Exercice 1 : Second degré et Polynômes (4 pts)
1 Résoudre dans ℝ l'équation : -2x² + 5x - 2 = 0
✅ Correction : Δ = 5² - 4(-2)(-2) = 25 - 16 = 9. √Δ = 3.
x₁ = (-5 -3)/(-4) = (-8)/(-4) = 2 ; x₂ = (-5+3)/(-4) = (-2)/(-4) = 1/2. S = {1/2 ; 2}.
2 Déterminer le signe du trinôme P(x) = -x² + 4x - 5.
✅ Correction : Δ = 16 - 20 = -4. Δ < 0 et a = -1 < 0 ⇒ P(x) est strictement négatif pour tout réel x.
3 Soit Q(x) = 2x³ - 3x² - 5x + 6. Vérifier que 2 est une racine, puis factoriser Q(x).
✅ Correction : Q(2) = 2(8) - 3(4) -10 + 6 = 16 -12 -10 + 6 = 0.
On factorise par (x-2) : Q(x) = (x-2)(2x² + x -3). Δ = 1+24=25 ⇒ racines : 1 et -3/2.
Donc Q(x) = (x-2)(x-1)(2x+3).
📌 Exercice 2 : Systèmes linéaires et non linéaires (4 pts)
1 Résoudre le système : \[ \begin{cases} 3x + 2y = 8 \\ x - y = 1 \end{cases} \]
✅ Correction : Par substitution : x = y+1 ⇒ 3(y+1)+2y=8 ⇒ 5y = 5 ⇒ y=1, x=2. S = {(2 ; 1)}.
2 Résoudre le système suivant (se ramenant au second degré) : \[ \begin{cases} x + y = 7 \\ xy = 10 \end{cases} \]
✅ Correction : x et y sont solutions de X² - 7X + 10 = 0. Δ = 49 - 40 = 9.
X₁ = (7-3)/2 = 2, X₂ = (7+3)/2 = 5. Donc couples (2 ; 5) ou (5 ; 2).
📌 Exercice 3 : Cercle trigonométrique et équations (6 pts)
1 Calculer cos(75°) en utilisant les formules d'addition (cos(a+b)).
✅ Correction : cos(75°) = cos(45°+30°) = cos45.cos30 - sin45.sin30
= (√2/2)(√3/2) - (√2/2)(1/2) = √2/4 (√3 - 1).
2 Simplifier l'expression : E = sin(π + x) + cos(π/2 - x).
✅ Correction : sin(π+x) = -sin x ; cos(π/2 - x) = sin x. Donc E = -sin x + sin x = 0.
3 Résoudre dans ℝ l'équation : 2cos²(x) + cos(x) - 1 = 0 (Indice : poser X = cos x).
✅ Correction : Soit X = cos x. 2X² + X -1 = 0. Δ = 1+8=9 ⇒ X₁ = (-1-3)/4 = -1 ; X₂ = (-1+3)/4 = 1/2.
cos x = -1 ⇒ x = π + 2kπ ; cos x = 1/2 ⇒ x = ± π/3 + 2kπ. (k ∈ ℤ)
PARTIE 2 : ÉVALUATION DES COMPÉTENCES (6 points)
🌽 Situation problème : Gestion d’un groupement agricole

Contexte : À l’Ouest du Cameroun, un groupement de femmes cultive du maïs et du haricot. Elles souhaitent maximiser leurs bénéfices. Le bénéfice (en milliers de FCFA) en fonction du nombre x (en centaines de kg) de maïs vendu suit la loi : \[ B(x) = -x^2 + 10x - 16 \] (pour x compris entre 0 et 8).

1 Déterminer la forme canonique de B(x) et le bénéfice maximal.
✅ Correction : B(x) = -(x² -10x) -16 = -[(x-5)² -25] -16 = -(x-5)² + 25 -16 = -(x-5)² + 9.
Le maximum est 9 (milliers FCFA) atteint pour x = 5 (500 kg de maïs).
2 Pour quelles quantités de maïs le bénéfice est-il positif ?
✅ Correction : On résout -x²+10x-16 ≥ 0 ⇔ x²-10x+16 ≤ 0. Δ = 100-64=36, racines : (10-6)/2=2 et (10+6)/2=8.
Le trinôme est positif entre les racines (a=1>0). Donc bénéfice positif pour x ∈ [2 ; 8] (soit entre 200 et 800 kg).
