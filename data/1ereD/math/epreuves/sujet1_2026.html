Un groupement d'agriculteurs de la région de l'Ouest souhaite acheter un terrain pour étendre une plantation de maïs. Le terrain est rectangulaire ; sa longueur dépasse sa largeur de 12 mètres. Son aire est de 588 m². Pour clôturer le terrain, ils doivent également prévoir un portail.

📐 Exercice 1 : Second degré et forme canonique (6 pts)
1. Résoudre dans ℝ l’équation : 2x² - 7x + 3 = 0 (calcul du discriminant Δ).
✅ Correction Δ = b² - 4ac = (-7)² - 4·2·3 = 49 - 24 = 25. √Δ = 5. Solutions : x₁ = (7-5)/(4) = 0,5 ; x₂ = (7+5)/4 = 3. S = {0,5 ; 3}
2. Déterminer le signe du trinôme -x² + 4x - 4 et résoudre l’inéquation -x² + 4x - 4 ≥ 0.
✅ Correction Δ = 16 - 16 = 0 → racine double x₀ = 2. Le coefficient a = -1 < 0 → trinôme négatif sauf en x=2 où il est nul. Solution de l’inéquation : x = 2. S = {2}
3. Écrire le trinôme 3x² + 6x - 5 sous forme canonique (a(x - α)² + β).
✅ Correction 3x²+6x-5 = 3(x²+2x) -5 = 3[(x+1)² -1] -5 = 3(x+1)² -3 -5 = 3(x+1)² - 8. Forme canonique : 3(x+1)² -8.
4. Vérifier que 2 est une racine du polynôme P(x) = x³ - 4x² + x + 6, puis factoriser complètement P(x).
✅ Correction P(2)=8-16+2+6=0 → 2 est racine. Division euclidienne: (x-2)(x²-2x-3) = (x-2)(x-3)(x+1). Factorisation: P(x) = (x-2)(x-3)(x+1).
🔁 Exercice 2 : Systèmes linéaires et non linéaires (5 pts)
1. Résoudre le système linéaire suivant :
\(\begin{cases} 3x + 2y = 13 \\ 5x - y = 13 \end{cases}\)
✅ Correction Par substitution : y = 5x - 13 → 3x + 2(5x-13) = 13 → 3x+10x-26=13 → 13x=39 → x=3, puis y=5·3-13=2. Solution (3;2).
2. Résoudre le système non linéaire (par substitution) : \(\begin{cases} x + y = 8 \\ x·y = 15 \end{cases}\)
✅ Correction De y = 8 - x, alors x(8-x)=15 → -x²+8x-15=0 → x²-8x+15=0 → Δ=64-60=4 → x=3 ou x=5. Couples (3;5) ou (5;3).
3. Problème concret : Un commerçant achète 5 cahiers et 3 stylos pour 2650 FCFA. Le prix d’un cahier dépasse celui d’un stylo de 250 FCFA. Trouver le prix unitaire.
✅ Correction Soit c=cahier, s=stylo : c = s+250 et 5c+3s=2650 → 5(s+250)+3s=2650 → 8s+1250=2650 → 8s=1400 → s=175 FCFA, c=425 FCFA.
📐 Exercice 3 : Formules trigonométriques & équations (6 pts)
1. Simplifier l’expression : \( A = \cos\left(\frac{\pi}{3} + x\right) \cos x + \sin\left(\frac{\pi}{3} + x\right) \sin x \) (utiliser formule d’addition).
✅ Correction cos(a-b)=cos a cos b + sin a sin b ⇒ A = cos[(π/3 + x) - x] = cos(π/3) = 1/2.
2. Établir la formule de duplication : sin(2a) = 2 sin a cos a. Puis calculer sin(2π/3) en connaissant sin(π/3) et cos(π/3).
✅ Correction sin(2a)=2 sin a cos a. sin(2π/3)= 2·sin(π/3)·cos(π/3)= 2·(√3/2)·(1/2)= √3/2.
3. Résoudre dans ℝ l’équation trigonométrique : 2 cos²x - 3 cos x + 1 = 0 (poser X = cos x).
✅ Correction X = cos x → 2X²-3X+1=0, Δ=9-8=1 → X₁=1, X₂=1/2. cos x=1 ⇒ x=2kπ ; cos x=1/2 ⇒ x= ±π/3 + 2kπ. S = {2kπ ; π/3+2kπ ; -π/3+2kπ, k∈ℤ}.
4. Transformer l’expression \( \sqrt{3} \sin x + \cos x \) sous la forme \( R \cos(x - \varphi) \) (R>0).
✅ Correction R = √( (√3)²+1² ) = √4 = 2. cos φ = 1/2, sin φ = √3/2 ⇒ φ=π/3. donc 2 cos(x - π/3).
🎯 Exercice 4 : QCM conceptuel et problème (3 pts)
❶ (QCM) Le discriminant de l’équation 4x² - 12x + 9 = 0 vaut :
A) 0 B) 144 C) 36 D) -144
✅ Réponse A Δ = (-12)² - 4·4·9 = 144 - 144 = 0 (racine double).
❷ Une expression de la forme a cos x + b sin x peut s’écrire :
A) R cos(x - φ) B) R sin(x + φ) C) Les deux sont possibles D) Aucune
✅ Réponse C On peut écrire sous les deux formes avec un déphasage approprié (transformation classique).
❸ Sachant que α est une racine de P(x) = x³ - 2x² - 5x + 6, et α=2 est racine, déterminer le quotient Q(x) puis les autres racines.
✅ Correction Division par (x-2) : x²+0x-3 = x²-3 → racines x=√3 et x=-√3. Les solutions sont 2 ; √3 ; -√3.
