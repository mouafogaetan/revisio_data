📐 Polynômes de degré 3

Algèbre · Première D & TI · Racines, factorisation et signe

📌 Éléments clés à retenir

🔍 Racine d’un polynôme

Un réel $\alpha$ est une racine (ou zéro) d’un polynôme $P(x)$ si et seulement si $P(\alpha) = 0$. Pour un polynôme de degré 3 : $P(x) = ax^3 + bx^2 + cx + d$ avec $a \neq 0$.

  • Si $\alpha$ est une racine, alors $P(x)$ peut s’écrire $(x - \alpha) \times Q(x)$ où $Q(x)$ est un polynôme de degré 2.
  • Racine évidente : on teste souvent $\alpha = -2, -1, 0, 1, 2$ (diviseurs de $d$).
  • Pour vérifier rapidement : $P(\alpha) = 0$.
💡 Astuce : La somme des coefficients d’un polynôme est $P(1)$ ; si elle est nulle, $1$ est racine.

✖️ Factorisation d’un polynôme de degré 3

Connaissant une racine $\alpha$, on factorise $P(x)$ par $(x - \alpha)$. On obtient : $$P(x) = (x - \alpha)(Ax^2 + Bx + C)$$ Le trinôme du second degré $Ax^2 + Bx + C$ se factorise ensuite si son discriminant $\Delta$ est positif ou nul.

  • Méthode pratique : division euclidienne ou identification des coefficients.
  • Forme factorisée complète : $P(x) = a(x - \alpha)(x - \beta)(x - \gamma)$ si trois racines réelles.
  • Sinon, on garde $(x - \alpha)$ × (trinôme irréductible sur $\mathbb{R}$).

📈 Signe d’un polynôme de degré 3

Pour étudier le signe de $P(x)$, on factorise totalement $P(x)$ en produit de facteurs du premier degré et éventuellement d’un trinôme de signe constant.

  • On dresse un tableau de signes en plaçant les racines dans l’ordre croissant.
  • Le coefficient $a$ (terme $ax^3$) donne le signe « à l’infini » : si $a>0$, $P(x) \to +\infty$ quand $x\to +\infty$ ; si $a<0$, $P(x) \to -\infty$ quand $x\to +\infty$.
  • On applique la règle des signes produit (comme pour les polynômes du second degré mais avec trois facteurs).
📌 Exemple : $P(x) = 2(x-1)(x+2)(x-3)$ → racines : $-2, 1, 3$ ; $a>0$ donc signe : $-$ entre $-2$ et $1$, $+$ à l’extérieur des racines extrêmes.

💡 Exemples de compréhension

Exemple 1 : Vérifier une racine évidente et factoriser

Soit $P(x) = x^3 - 2x^2 - 5x + 6$. Vérifier que $x=1$ est une racine, puis factoriser $P(x)$ complètement.

Résolution :
$P(1) = 1 - 2 - 5 + 6 = 0$ ⇒ $1$ est racine.
On effectue la division de $P(x)$ par $(x-1)$ ou on identifie : $P(x) = (x-1)(x^2 + bx + c)$.
Développement : $(x-1)(x^2+bx+c)=x^3 + bx^2 + cx - x^2 - bx - c = x^3 + (b-1)x^2 + (c-b)x - c$.
Par identification avec $x^3 - 2x^2 -5x + 6$ :
$b-1 = -2 \Rightarrow b = -1$ ;
$c - b = -5 \Rightarrow c - (-1) = -5 \Rightarrow c+1 = -5 \Rightarrow c = -6$ ;
$-c = 6 \Rightarrow c = -6$ (cohérent).
Donc $P(x) = (x-1)(x^2 - x - 6)$.
Factorisons $x^2 - x - 6 = 0$ → $\Delta = 1 + 24 = 25$, racines $x = \frac{1 \pm 5}{2} = 3$ et $-2$.
Ainsi $P(x) = (x-1)(x-3)(x+2)$.

💬 Astuce : La division synthétique (schéma de Horner) permet d’aller plus vite. On teste les diviseurs du terme constant $\pm1, \pm2, \pm3, \pm6$.

Exemple 2 : Étude du signe d’un polynôme de degré 3

Étudier le signe de $Q(x) = -2x^3 + 4x^2 + 2x - 4$ sur $\mathbb{R}$.

Étape 1 – Recherche d’une racine évidente :
$Q(1) = -2+4+2-4=0$ ⇒ $1$ est racine.
Étape 2 – Factorisation : On factorise $(x-1)$.
En procédant par identification : $Q(x) = (x-1)(-2x^2 + Bx + C)$.
Développement : $(x-1)(-2x^2+Bx+C) = -2x^3 + Bx^2 + Cx + 2x^2 - Bx - C$
$= -2x^3 + (B+2)x^2 + (C - B)x - C$.
Identification avec $-2x^3 + 4x^2 + 2x - 4$ :
$B+2 = 4 \Rightarrow B=2$ ; $C - B = 2 \Rightarrow C-2=2 \Rightarrow C=4$ ; $-C=-4 \Rightarrow C=4$ ok.
D’où $Q(x) = (x-1)(-2x^2+2x+4)$. On met $-2$ en facteur : $-2x^2+2x+4 = -2(x^2 - x -2)$.
Factorisons $x^2 - x - 2$ : $\Delta = 1+8=9$, racines $x = \frac{1 \pm 3}{2} = 2$ et $-1$.
Donc $Q(x) = (x-1)(-2)(x-2)(x+1) = -2(x-1)(x-2)(x+1)$.
Étape 3 – Tableau de signes : Les racines sont $-1, 1, 2$ (ordre croissant).
Coefficient dominant $-2 < 0$ ⇒ $Q(x) \to -\infty$ en $+\infty$ et $Q(x) \to +\infty$ en $-\infty$.
On place les racines et on applique la règle des signes pour $(x+1)(x-1)(x-2)$ puis on multiplie par $-2$.
Sur $]-\infty ; -1[$ : $(x+1)<0, (x-1)<0, (x-2)<0$ ⇒ produit des 3 facteurs négatif ⇒ $(-2) \times (\text{négatif}) = \text{positif}$.
Sur $]-1 ; 1[$ : $(x+1)>0, (x-1)<0, (x-2)<0$ ⇒ deux négatifs ⇒ produit positif ⇒ $(-2) \times \text{positif} = \text{négatif}$.
Sur $]1 ; 2[$ : $(x+1)>0, (x-1)>0, (x-2)<0$ ⇒ un négatif ⇒ produit négatif ⇒ $(-2) \times \text{négatif} = \text{positif}$.
Sur $]2 ; +\infty[$ : tous positifs ⇒ produit positif ⇒ $(-2) \times \text{positif} = \text{négatif}$.
Conclusion : $Q(x) > 0$ sur $]-\infty ; -1[ \cup ]1 ; 2[$ ; $Q(x) < 0$ sur $]-1 ; 1[ \cup ]2 ; +\infty[$ ; $Q(x)=0$ en $x=-1,1,2$.

📐 Représentation graphique : une cubique avec trois racines réelles traverse l’axe des abscisses en changeant de signe selon la multiplicité.

Exemple 3 : Racine évidente et factorisation avec reste

Soit $R(x) = 2x^3 + 3x^2 - 8x + 3$. Montrer que $x = 1$ est racine, factoriser et déterminer le signe.

$R(1) = 2+3-8+3 = 0$ ⇒ $x=1$ racine.
Effectuons la division par $(x-1)$ :
$R(x) = (x-1)(2x^2 + 5x - 3)$ (vérification : développement $2x^3+5x^2-3x -2x^2-5x+3 = 2x^3+3x^2-8x+3$).
Résolvons $2x^2+5x-3 = 0$ : $\Delta = 25+24 = 49$, racines $x = \frac{-5 \pm 7}{4}$ soit $x = \frac{1}{2}$ et $x = -3$.
Ainsi $R(x) = (x-1)(2x-1)(x+3)$.
Signe : racines $-3, \frac{1}{2}, 1$. Coefficient $a=2>0$ → $R(x)$ est positif à l’extérieur des racines extrêmes (après ordre croissant).
Tableau : $+$ sur $]-3 ; \frac12[$ et $]1; +\infty[$ ; $-$ sur $]-\infty ; -3[ \cup ]\frac12 ; 1[$.

⚠️ Pièges courants & astuces gagnantes

  • Erreur fréquente : Oublier de vérifier que le polynôme de degré 2 obtenu après factorisation peut encore se factoriser (calculer $\Delta$).
  • Ne pas confondre : Une racine évidente ne garantit pas que toutes les racines sont entières. Penser aux racines fractionnaires du type $\frac{p}{q}$ où $p$ divise $d$ et $q$ divise $a$.
  • Astuce : Pour factoriser rapidement par $(x-\alpha)$, utilisez le schéma de Horner (coefficients du polynôme) : on abaisse le premier coefficient, on multiplie par $\alpha$, on additionne au suivant, etc.
  • Attention au signe : Si vous factorisez avec un facteur négatif, n’oubliez pas de le reporter dans le tableau de signes. Exemple : $P(x) = -(x-1)(x+2)(x-3)$ change les signes attendus.
  • Pour les polynômes de degré 3 sans racine évidente : On peut utiliser le théorème des valeurs intermédiaires pour localiser les racines, mais au programme Première D, on se concentre sur les racines « simples » (entières ou rationnelles évidentes).

📐 Méthode pas à pas : factoriser un polynôme de degré 3

✅ Algorithme de résolution

  1. Tester les racines évidentes : $\pm 1, \pm 2, \pm 3, \pm \frac12$, etc. (diviseurs du terme constant $d$ / diviseurs de $a$).
  2. Vérifier $P(\alpha)=0$.
  3. Factoriser par $(x-\alpha)$ en utilisant l’identification ou la division polynomiale.
  4. Résoudre le trinôme du second degré obtenu (calcul $\Delta$, racines éventuelles).
  5. Écrire la forme factorisée complète : $P(x) = a(x-\alpha)(x-\beta)(x-\gamma)$ (si 3 racines réelles).
  6. Étudier le signe via tableau de signes avec toutes les racines ordonnées.

🏋️ Exercice d’entraînement (auto-correction)

À vous de jouer !

Soit $F(x) = x^3 - 3x^2 - 4x + 12$.
1) Montrer que $2$ est une racine de $F$.
2) Factoriser $F(x)$ complètement.
3) Résoudre $F(x) \ge 0$ sur $\mathbb{R}$.

🔎 Voir la correction

1) $F(2) = 8 - 12 - 8 + 12 = 0$ ⇒ racine.
2) Factorisation : $(x-2)(x^2 - x - 6)$. Or $x^2 - x -6 = (x-3)(x+2)$. Donc $F(x) = (x-2)(x-3)(x+2)$.
3) Racines ordonnées : $-2, 2, 3$. Coefficient $a=1>0$ ⇒ signe $+$ sur $[-2;2] \cup [3;+\infty[$ et $-$ sur $]-\infty;-2] \cup [2;3]$. Donc $F(x) \ge 0$ pour $x \in [-2;2] \cup [3;+\infty[$.

💪 Conseil : Pour vérifier votre factorisation, développez $(x-2)(x-3)(x+2)$ et vous devez retrouver $x^3-3x^2-4x+12$.

📚 Rappels utiles (second degré)

Discriminant $\Delta = b^2 - 4ac$ pour $ax^2+bx+c=0$ :

  • $\Delta > 0$ : deux racines réelles $x_{1,2} = \frac{-b \pm \sqrt{\Delta}}{2a}$.
  • $\Delta = 0$ : une racine double $x_0 = -\frac{b}{2a}$.
  • $\Delta < 0$ : pas de racine réelle, le trinôme est toujours du signe de $a$.

Penser à l’utiliser après avoir factorisé par $(x-\alpha)$.
