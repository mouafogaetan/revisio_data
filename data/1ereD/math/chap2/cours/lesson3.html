Systèmes d'équations linéaires

Mathématiques · Première D & TI · Module 1 – Chapitre 2

📌 Éléments clés à retenir

🔷 1. Définition et forme générale

Un système d'équations linéaires est un ensemble de plusieurs équations du premier degré à plusieurs inconnues. La solution d’un tel système est l’ensemble de tous les couples (ou triplets) qui vérifient simultanément toutes les équations.

  • Système 2×2 : \[ \begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases} \]
  • Système 3×3 : \[ \begin{cases} a_1 x + b_1 y + c_1 z = d_1 \\ a_2 x + b_2 y + c_2 z = d_2 \\ a_3 x + b_3 y + c_3 z = d_3 \end{cases} \]

🧮 2. Méthodes de résolution

Pour résoudre un système linéaire, on utilise principalement deux méthodes algébriques :

  • Méthode par substitution : on exprime une inconnue en fonction des autres dans une équation, puis on remplace dans les autres équations.
  • Méthode par combinaison (ou élimination) : on additionne ou soustrait des équations après les avoir multipliées par des coefficients afin d'éliminer une inconnue.
  • Pour les systèmes 3×3, on applique ces méthodes de façon itérative (réduction triangulaire).
💡 Astuce : Vérifiez toujours votre solution en la replaçant dans toutes les équations initiales.

📐 3. Interprétation graphique (système 2×2)

Chaque équation linéaire à deux inconnues représente une droite dans le plan. Résoudre le système revient à trouver leur intersection.

  • Solution unique : les droites sont sécantes (coefficients directeurs différents).
  • Aucune solution : les droites sont parallèles distinctes.
  • Une infinité de solutions : les droites sont confondues.

🔁 4. Systèmes non linéaires par substitution

Certains systèmes ne sont pas linéaires mais peuvent se ramener à une équation du second degré grâce à la substitution. On isole une variable dans une équation (souvent la plus simple) et on remplace dans l’autre.

Exemple : \( \begin{cases} y = 2x + 1 \\ x^2 + y^2 = 25 \end{cases} \) → remplacer \(y\) dans la seconde → équation en \(x\).

💡 Exemples de compréhension

Exemple 1 : Substitution (système 2×2)

Résoudre par substitution : \[ \begin{cases} 2x + y = 7 \\ 3x - 2y = 0 \end{cases} \]

Résolution détaillée :
• Dans la première équation, on isole \(y\) : \(y = 7 - 2x\).
• On remplace \(y\) dans la deuxième équation : \(3x - 2(7 - 2x) = 0\) → \(3x - 14 + 4x = 0\) → \(7x = 14\) → \(x = 2\).
• On calcule \(y\) : \(y = 7 - 2 \times 2 = 3\).
• La solution est \((x; y) = (2; 3)\).
💬 Astuce : Choisissez l’équation où une variable a un coefficient 1 ou –1 pour simplifier l’isolement.

Exemple 2 : Combinaison linéaire (système 2×2)

Résoudre par combinaison : \[ \begin{cases} 4x - 3y = 1 \\ 2x + y = 3 \end{cases} \]

Élimination de \(y\) :
• On multiplie la deuxième équation par 3 : \(6x + 3y = 9\).
• On additionne avec la première : \((4x - 3y) + (6x + 3y) = 1 + 9\) → \(10x = 10\) → \(x = 1\).
• On remplace \(x = 1\) dans \(2x + y = 3\) : \(2 + y = 3\) → \(y = 1\).
• Solution : \((1; 1)\).

Exemple 3 : Système 3×3 (méthode par étapes)

Résoudre le système : \[ \begin{cases} x + y + z = 6 \\ 2x - y + z = 3 \\ x + 2y - z = 2 \end{cases} \]

Élimination pas à pas :
• (L1) \(x + y + z = 6\)
• (L2) \(2x - y + z = 3\)
• (L3) \(x + 2y - z = 2\)

On élimine \(z\) : L1 + L3 → \((x+y+z)+(x+2y-z) = 6+2\) → \(2x + 3y = 8\) (E4)
L2 + L3 → \((2x-y+z)+(x+2y-z) = 3+2\) → \(3x + y = 5\) (E5)
On résout le système (E4, E5) : de E5, \(y = 5 - 3x\). Dans E4 : \(2x + 3(5-3x) = 8\) → \(2x+15-9x=8\) → \(-7x = -7\) → \(x=1\).
Alors \(y = 5 - 3 = 2\). Enfin dans L1 : \(1 + 2 + z = 6\) → \(z = 3\).
Solution : \((1; 2; 3)\).
📌 Vérification rapide : \(1+2+3=6\), \(2-2+3=3\), \(1+4-3=2\) → juste.

Exemple 4 : Système non linéaire → second degré

Résoudre le système : \[ \begin{cases} x + y = 5 \\ xy = 6 \end{cases} \]

Substitution : de la première équation, \(y = 5 - x\).
On remplace dans \(xy = 6\) : \(x(5 - x) = 6\) → \(5x - x^2 = 6\) → \(-x^2 + 5x - 6 = 0\) → \(x^2 - 5x + 6 = 0\).
Discriminant \(\Delta = 25 - 24 = 1\) → \(x = \frac{5 \pm 1}{2}\) soit \(x=3\) ou \(x=2\).
Les solutions : \((x;y) = (3;2)\) ou \((2;3)\).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier de multiplier TOUS les termes d’une équation lorsqu’on utilise la combinaison linéaire.
  • Attention : Dans un système 3×3, une erreur de signe en additionnant peut fausser toute la résolution. Vérifiez chaque étape.
  • Piège : Confondre solution unique, aucune solution et infinité de solutions. Penser à l’interprétation graphique.
  • Conseil : Pour les systèmes non linéaires, s’assurer que l’équation obtenue après substitution est bien du second degré avant d’appliquer le discriminant.

📐 Méthodes en un clin d'œil

Procédure pour un système 2×2

  • Substitution : Isoler une inconnue → remplacer → résoudre → trouver l’autre inconnue.
  • Combinaison : Multiplier les équations pour obtenir des coefficients opposés → additionner → résoudre l’équation à une inconnue.

Procédure pour un système 3×3

1. Éliminer une variable (par exemple \(z\)) en combinant deux paires d’équations.
2. Obtenir un système 2×2 en \(x\) et \(y\).
3. Résoudre ce système 2×2.
4. Déterminer la troisième variable à l’aide d’une équation initiale.
