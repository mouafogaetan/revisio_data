📘 Lesson 24 : In order to / So as to

Exprimer le but (purpose) en anglais · Première CD · Révision Probatoire

📌 Éléments clés à retenir

🔹 1. Pourquoi utiliser "in order to" et "so as to" ?

Ces deux expressions servent à exprimer le but (l'objectif) d'une action. Elles signifient « afin de / pour » en français. On les utilise devant un verbe à l'infinitif. Elles rendent le discours plus formel et plus précis que le simple "to".

  • In order to : plus courant, neutre (soutenu ou non).
  • So as to : légèrement plus formel, très fréquent à l'écrit.
  • Astuce : On peut souvent les remplacer par "to", mais pour les examens, on privilégie ces formes élégantes.
  • 📌 Structure : Sujet + verbe + (complément) + in order to / so as to + base verbale
🎨 Schéma visuel : l’action (gauche) mène à un but (droite) via "in order to / so as to".

🔹 2. La forme négative : "in order not to" / "so as not to"

Pour exprimer un but négatif (« pour ne pas »), on ajoute not avant l'infinitif. Attention : on ne met jamais "don't" ou "doesn't", mais directement not + verbe.

  • In order not to : afin de ne pas ...
  • So as not to : de façon à ne pas ...
  • ✔ Exemple : She left early so as not to miss the bus. (Elle est partie tôt pour ne pas rater le bus.)
  • ⚠️ Piège fréquent : ❌ She left early so as to not miss (moins élégant, à éviter). On préfère so as not to.

🔹 3. Différence avec "to" seul et "for"

"to" (infinitif) exprime aussi le but mais de façon plus brève. "For" est suivi d’un NOM (pas d’un verbe). En révision, on utilise "in order to" / "so as to" pour plus de clarté et un style soigné.

  • to + verbe : I went to the market to buy fruits. (correct mais basique)
  • for + nom : I went to the market for fruits.
  • in order to + verbe : I went to the market in order to buy fresh fruits. (plus précis)
  • so as to + verbe : I went to the market so as to buy fresh fruits. (équivalent formel)
💡 Mémo examen : Si la phrase a un but clair (réponse à "Why ?"), utilisez "in order to" ou "so as to". Pour le négatif, pensez "in order NOT TO".

💡 Exemples de compréhension

Exemple 1 : But positif avec "in order to"

Contexte : Un étudiant travaille beaucoup le soir.

Phrase : « John studies hard every evening in order to pass his Probatoire exam. »
Traduction française : John étudie sérieusement chaque soir afin de réussir son examen probatoire.

✨ Analyse : "in order to" introduit clairement le but (passer l’examen). On pourrait aussi dire "so as to pass".

Exemple 2 : But négatif avec "so as not to"

Contexte : Pendant une réunion, on coupe son téléphone pour ne pas être dérangé.

Phrase : « The teacher switched off his phone so as not to be disturbed during the staff meeting. »
Traduction : Le professeur a éteint son téléphone pour ne pas être dérangé pendant la réunion des professeurs.

⚠️ À retenir : On n'écrit pas "so as to not be disturbed" (bien que compris, mais moins standard dans un examen).

Exemple 3 : Comparaison "in order to" / "so as to" dans un même contexte

Contexte : Une association plante des arbres pour protéger l’environnement.

✅ The NGO plants trees in order to reduce pollution.
✅ The NGO plants trees so as to reduce pollution.
💬 Les deux sont parfaitement corrects. Le deuxième peut être un peu plus formel.

💪 Conseil pour l’écrit : variez les expressions (parfois "in order to", parfois "so as to") pour montrer votre maîtrise.

✍️ Application rapide (à faire sur feuille)

Transforme les phrases suivantes avec "in order to" ou "so as to"

  • 1. She wakes up early. She wants to catch the first bus. → She wakes up early ______ catch the first bus.
  • 2. They speak slowly. They don’t want to confuse the students. → They speak slowly ______ confuse the students. (négatif)
  • 3. He saves money. He wants to buy a new laptop. → He saves money ______ buy a new laptop.
  • 4. We left the party early. We didn’t want to miss the last taxi. → We left early ______ miss the last taxi.
🔍 Corrigés : 1. in order to / so as to  |  2. so as not to / in order not to  |  3. in order to / so as to  |  4. so as not to / in order not to

⚠️ Pièges courants / À ne pas confondre

  • ❌ Erreur fréquente : Utiliser "for + verbe".
    Exemple incorrect : I study hard for to pass. → Il faut dire I study hard in order to pass.
  • ⚠️ Attention à la négation : "so as to not" est parfois entendu mais considéré comme moins académique. Privilégiez so as not to / in order not to.
  • 🧩 Confusion possible : Ne pas mélanger "so as to" (but) avec "so that" (but + sujet + can/will).
    ✔ He spoke loudly so as to be heard. (même sujet)
    ✔ He spoke loudly so that everyone could hear. (sujet différent)
  • 📌 Rappel : Après "in order to" / "so as to", on utilise toujours la base verbale (infinitif sans to).

🎯 Astuce visuelle pour le jour de l'examen

Pour ne pas oublier la structure, imaginez une flèche qui part de l'action et va vers l'objectif. "in order to" / "so as to" = la flèche du but. Négatif = flèche barrée 🚫.

📚 Illustration conceptuelle : les deux formes (positive et négative) en contexte scolaire.
  • Mots-clés pour le probatoire : purpose, aim, objective → répondez avec "in order to" ou "so as to".
  • 💬 À l'oral ou à l'écrit, ces structures vous donneront un style plus mature.
