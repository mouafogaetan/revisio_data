🧠 Partie 1 – Restitution (6 pts)
❓ Question 1 (1,5 pt)
En langage C, laquelle de ces structures permet d'exécuter un bloc d'instructions au moins une fois, même si la condition est fausse ?
  • A. for ( ... )
  • B. while ( ... )
  • C. do ... while ( ... )
  • D. if ... else

    • ✅ Réponse correcte : C. do ... while ( ... )

    🔹 La boucle do...while teste la condition après l'exécution du bloc, garantissant une exécution minimale.

    ❓ Question 2 (1,5 pt)
    Quelle est la sortie du code suivant ?
    int t[3] = {5, 8, 12}; printf("%d", t[2]);
  • A. 5
  • B. 8
  • C. 12
  • D. Erreur de compilation

    • ✅ Réponse correcte : C. 12

    🔹 En C, l'indice du tableau commence à 0. t[2] correspond au troisième élément (valeur 12).

    ❓ Question 3 (1,5 pt)
    Vrai ou faux : une variable déclarée const int MAX = 100; ne peut pas être modifiée après son initialisation.
  • A. Vrai
  • B. Faux

    • ✅ Réponse correcte : A. Vrai

    🔹 Le qualificatif const rend la variable en lecture seule. Toute tentative de modification génère une erreur.

    💻 Partie 2 – Application (14 pts)
    📝 Exercice 1 : Analyse de programme (4 pts)
    On donne le programme C suivant : 
    #include <stdio.h>
int main() {
    int notes[4] = {15, 10, 18, 8};
    int i, somme = 0;
    for(i = 0; i < 4; i++) {
        somme = somme + notes[i];
    }
    printf("Moyenne = %.2f", somme / 4.0);
    return 0;
}
    a) (2 pts) Quelle valeur est affichée ?
    b) (2 pts) Modifier la boucle pour qu'elle affiche uniquement les notes supérieures ou égales à 10.

    ✅ Corrigé

    a) La somme vaut 15+10+18+8 = 51, divisée par 4 donne 12.75.

    b) Ajouter une condition dans la boucle :
    for(i = 0; i < 4; i++) {
        if(notes[i] >= 10)
            printf("%d ", notes[i]);
    }
    Affichage : 15 10 18

    📝 Exercice 2 : Structure conditionnelle (4 pts)
    Écrire un programme en C qui demande l'âge d'une personne et affiche :
    • "Enfant" si âge < 12
    • "Adolescent" si 12 ≤ âge < 18
    • "Adulte" si âge ≥ 18
    On utilisera if ... else if ... else.

    ✅ Solution attendue

    #include <stdio.h>
int main() {
    int age;
    printf("Entrez l'âge : ");
    scanf("%d", &age);
    
    if(age < 12)
        printf("Enfant\n");
    else if(age < 18)
        printf("Adolescent\n");
    else
        printf("Adulte\n");
    return 0;
}
    📝 Exercice 3 : Recherche séquentielle (6 pts)
    On dispose d'un tableau d'entiers : int valeurs[6] = {4, 9, 2, 15, 7, 10};
    a) (2 pts) Écrire une boucle qui affiche tous les éléments du tableau.
    b) (4 pts) Écrire l'algorithme de recherche séquentielle permettant de trouver si la valeur 7 existe dans le tableau. Afficher "Trouvé" ou "Non trouvé".

    ✅ Proposition de correction

    a) Affichage simple :
    for(int i = 0; i < 6; i++) printf("%d ", valeurs[i]);

    b) Recherche de 7 :
    int trouve = 0;
    for(int i = 0; i < 6; i++) {
        if(valeurs[i] == 7) { trouve = 1; break; }
    }
    if(trouve) printf("Trouvé"); else printf("Non trouvé");
    ➤ 7 est présent à l'indice 4 → "Trouvé"
