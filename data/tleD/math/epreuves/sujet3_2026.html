📌 Exercice 1 : Tirage avec remise et variable aléatoire (5 points)
🔹 Partie A – Tirage avec remise
Une urne contient 5 boules rouges, 3 boules vertes et 2 boules bleues. On tire successivement deux boules avec remise.
1. Construire l’arbre pondéré des probabilités.
2. Calculer la probabilité d’obtenir deux boules de couleurs différentes.

✅ Corrigé 1. Arbre : chaque tirage a 3 issues : R(0,5), V(0,3), B(0,2). Avec remise, les probabilités sont identiques au second tirage.
2. P(couleurs différentes) = 1 – P(même couleur) = 1 – (0,5² + 0,3² + 0,2²) = 1 – (0,25 + 0,09 + 0,04) = 1 – 0,38 = 0,62.

🔹 Partie B – Loi de probabilité
On considère une variable aléatoire \(X\) qui prend les valeurs -2, 0 et 5 avec les probabilités : \(P(X=-2)=0,3\) ; \(P(X=0)=0,4\) ; \(P(X=5)=0,3\).
Calculer l’espérance mathématique \(E(X)\) et interpréter ce résultat dans un jeu équitable.

✅ Corrigé \(E(X) = (-2)\times0,3 + 0\times0,4 + 5\times0,3 = -0,6 + 0 + 1,5 = 0,9\). L’espérance étant positive, le jeu est favorable au joueur (gain moyen 0,9 unité par partie).

📊 Exercice 2 : Ajustement linéaire – Méthode de Mayer (5 points)
Une entreprise agricole étudie l’évolution de sa production de cacao (en tonnes) selon le rang de l’année.
Rang \(x_i\)12345
Production \(y_i\) (t)1215182125
1. Partager la série en deux sous-groupes égaux (1;2) et (4;5) puis déterminer les points moyens \(G_1\) et \(G_2\).
2. Déterminer l’équation de la droite de Mayer \((D)\) sous la forme \(y = ax + b\).
3. Estimer la production pour le rang 7 (année suivante).

✅ Corrigé 1. \(G_1 = \left(\frac{1+2}{2} ; \frac{12+15}{2}\right) = (1,5 ; 13,5)\) ; \(G_2 = \left(\frac{4+5}{2} ; \frac{21+25}{2}\right) = (4,5 ; 23)\).
2. Coefficient directeur \(a = \frac{23-13,5}{4,5-1,5} = \frac{9,5}{3} \approx 3,167\). Ordonnée à l’origine \(b = 13,5 - 3,167\times1,5 \approx 13,5 - 4,75 = 8,75\). Droite : \(y = 3,167x + 8,75\).
3. Pour \(x=7\) : \(y \approx 3,167\times7 + 8,75 = 22,169 + 8,75 \approx 30,92\) tonnes.

📈 Exercice 3 : Étude complète d’une fonction (5 points)
Soit \(f(x) = x\ln x - x\) définie sur \(]0; +\infty[\).
1. Calculer les limites aux bornes du domaine.
2. Déterminer \(f'(x)\) et étudier le signe de la dérivée.
3. Dresser le tableau de variations complet.
4. Donner l’équation de la tangente au point d’abscisse \(x=1\).

✅ Corrigé 1. \(\lim_{x\to 0^+} f(x) = 0\) (car \(x\ln x \to 0\)) ; \(\lim_{x\to +\infty} f(x) = +\infty\).
2. \(f'(x) = \ln x + 1 - 1 = \ln x\). Signe : \(f'(x) > 0 \iff x>1\) ; \(f'(x) < 0 \iff 0 3. Tableau : décroissante sur \(]0,1]\), minimum \(f(1) = -1\), croissante sur \([1,+\infty[\).
4. \(f(1) = -1\), \(f'(1)=0\) → tangente horizontale : \(y = -1\).

🧮 Exercice 4 : Primitives et intégration par parties (5 points)
On considère la fonction \(g(x) = (2x+1)e^{x}\).
1. Déterminer une primitive \(G\) de \(g\) sur \(\mathbb{R}\) en utilisant une intégration par parties ou une identification.
2. Calculer l’intégrale \(I = \displaystyle\int_{0}^{1} (2x+1)e^{x} \,dx\).

✅ Corrigé 1. On pose \(u = 2x+1\), \(dv = e^x dx\) → \(du=2dx\), \(v=e^x\). Par parties : \(\int (2x+1)e^x dx = (2x+1)e^x - \int 2e^x dx = (2x+1)e^x - 2e^x + C = (2x-1)e^x + C\). Une primitive est \(G(x) = (2x-1)e^x\).
2. \(I = \left[(2x-1)e^x\right]_{0}^{1} = (2\cdot1 -1)e^1 - (2\cdot0-1)e^0 = (1)e - (-1)\cdot1 = e + 1\).

🔹 Décomposition en éléments simples : Soit \(h(x) = \frac{2x+1}{x^2+x}\). Décomposer \(h(x)\) en éléments simples puis calculer \(\int_{1}^{2} h(x) dx\).

✅ Corrigé \(h(x) = \frac{2x+1}{x(x+1)} = \frac{A}{x} + \frac{B}{x+1}\). On trouve \(A=1, B=1\) car \(2x+1 = A(x+1)+Bx\) → \(A=1,B=1\). Donc \(h(x)=\frac{1}{x}+\frac{1}{x+1}\). L’intégrale \(\int_{1}^{2} h(x) dx = [\ln|x| + \ln|x+1|]_{1}^{2} = \ln2 + \ln3 - (\ln1+\ln2) = \ln3\).

🌟 Problème de synthèse : Application économique (5 points bonus / compétences)
Une entreprise de production d’huile de palme modélise son bénéfice marginal (en milliers de FCFA) par la fonction \(B'(q) = -0,3q^2 + 6q - 5\), où \(q\) est la quantité produite en tonnes ( \(0 \le q \le 12\) ).
Le bénéfice total \(B(q)\) est une primitive de \(B'(q)\) telle que \(B(0) = -20\) (coûts fixes).
1. Déterminer l’expression de \(B(q)\).
2. Étudier les variations de \(B\) sur \([0;12]\) et déterminer la production qui rend le bénéfice maximal.
3. Calculer la valeur de ce bénéfice maximum.

✅ Corrigé 1. \(B(q) = \int (-0,3q^2+6q-5)dq = -0,1q^3 + 3q^2 -5q + C\). Avec \(B(0)=-20\) ⇒ \(C=-20\). Donc \(B(q) = -0,1q^3 + 3q^2 -5q -20\).
2. \(B'(q) = -0,3q^2+6q-5\). Racines : \(\Delta=36-6=30\) ⇒ \(q_1 \approx 0,92\) , \(q_2 \approx 18,08\) (hors domaine). Donc \(B'(q) \ge 0\) sur \([0;0,92]\) et négative ensuite ? Non, re-calcul : polynôme du second degré concave, positif entre les racines. La seule racine dans l’intervalle est \(q_0 \approx 0,92\) ⇒ \(B\) croissante sur \([0;0,92]\), décroissante sur \([0,92;12]\). Maximum en \(q \approx 0,92\).
3. \(B(0,92) \approx -0,1(0,78) + 3(0,8464) -5(0,92) -20 \approx -0,078 + 2,539 -4,6 -20 = -22,139\) → bénéfice négatif ? L’énoncé semble peu réaliste, mais mathématiquement le maximum local est en 0,92 avec une perte faible. On corrige : meilleure production \(q=0,92\) t.
