Exercice 1 : Nombres complexes (4 points)
Le plan complexe est muni d'un repère orthonormé (O; \vec{u}, \vec{v}).
1. Résoudre dans \mathbb{C} l'équation : z^2 - (5 - 3i)z + 4 - 7i = 0. (1,5 point)
2. On considère les points A, B et C d'affixes respectives z_A = 2 - i, z_B = 3 - 2i et z_C = 1 - 2i. Déterminer l'écriture complexe de la similitude directe s de centre C qui transforme A en B. (1,5 point)
3. Donner les éléments caractéristiques de cette similitude (rapport et angle). (0,5 point)
4. Quelle est l'image de la droite (AC) par la similitude s ? Justifier. (0,5 point)

✅ Corrigé

1. Résolution de l'équation : z^2 - (5-3i)z + 4-7i = 0

Calcul du discriminant : \Delta = (5-3i)^2 - 4(4-7i) = (25 - 30i - 9) - (16 - 28i) = (16 - 30i) - 16 + 28i = -2i

On cherche \delta tel que \delta^2 = -2i. On pose \delta = x + iy. On obtient x^2 - y^2 = 0 et 2xy = -2, donc x = y ou x = -y. Avec x = y, 2x^2 = -2 impossible. Avec x = -y, -2x^2 = -2 \Rightarrow x^2 = 1 donc x = 1, y = -1 ou x = -1, y = 1. On prend \delta = 1 - i.

Solutions : z_1 = \frac{(5-3i) + (1-i)}{2} = \frac{6-4i}{2} = 3 - 2i et z_2 = \frac{(5-3i) - (1-i)}{2} = \frac{4-2i}{2} = 2 - i.

Donc S = \{2 - i\ ;\ 3 - 2i\}.

2. Similitude directe : s(z) = a(z - z_C) + z_C avec a \in \mathbb{C}^*. On a s(z_A) = z_B donc a(z_A - z_C) = z_B - z_C.

z_A - z_C = (2-i) - (1-2i) = 1 + i ; z_B - z_C = (3-2i) - (1-2i) = 2.

Donc a = \frac{2}{1+i} = \frac{2(1-i)}{(1+i)(1-i)} = \frac{2(1-i)}{2} = 1 - i.

L'écriture complexe est : s(z) = (1-i)(z - (1-2i)) + (1-2i) = (1-i)z + (1-2i) - (1-i)(1-2i).

Calcul de (1-i)(1-2i) = 1 -2i -i + 2i^2 = 1 - 3i - 2 = -1 - 3i.

Donc s(z) = (1-i)z + (1-2i) - (-1-3i) = (1-i)z + 2 + i.

3. Éléments caractéristiques : Rapport k = |a| = |1-i| = \sqrt{2}. Angle \theta = \arg(a) = \arg(1-i) = -\frac{\pi}{4}.

4. Image de (AC) : La similitude transforme une droite ne passant pas par le centre en une autre droite. Comme C est le centre, s(C)=C. L'image de la droite (AC) est la droite passant par s(A)=B et s(C)=C, donc la droite (BC).

Exercice 2 : Probabilités (3 points)
Une urne contient 5 boules rouges, 3 boules vertes et 2 boules bleues, toutes indiscernables au toucher. On tire successivement deux boules sans remise.
1. Construire un arbre de probabilités illustrant cette expérience. (1 point)
2. Soit X la variable aléatoire égale au nombre de boules rouges obtenues. Déterminer la loi de probabilité de X. (1,5 point)
3. Calculer l'espérance mathématique de X. (0,5 point)

✅ Corrigé

1. Arbre de probabilités :

Premier tirage : P(R) = \frac{5}{10} = \frac{1}{2}, P(V) = \frac{3}{10}, P(B) = \frac{2}{10} = \frac{1}{5}.

Second tirage (sans remise) :

• Si R au 1er : P(R|R) = \frac{4}{9}, P(V|R) = \frac{3}{9}, P(B|R) = \frac{2}{9}

• Si V au 1er : P(R|V) = \frac{5}{9}, P(V|V) = \frac{2}{9}, P(B|V) = \frac{2}{9}

• Si B au 1er : P(R|B) = \frac{5}{9}, P(V|B) = \frac{3}{9}, P(B|B) = \frac{1}{9}

2. Loi de probabilité de X (nombre de boules rouges) :

• P(X=0) : aucune boule rouge = P(VV) + P(VB) + P(BV) + P(BB)
= \frac{3}{10} \times \frac{2}{9} + \frac{3}{10} \times \frac{2}{9} + \frac{2}{10} \times \frac{3}{9} + \frac{2}{10} \times \frac{1}{9} = \frac{6}{90} + \frac{6}{90} + \frac{6}{90} + \frac{2}{90} = \frac{20}{90} = \frac{2}{9}

• P(X=2) : deux boules rouges = P(RR) = \frac{5}{10} \times \frac{4}{9} = \frac{20}{90} = \frac{2}{9}

• P(X=1) = 1 - \frac{2}{9} - \frac{2}{9} = \frac{5}{9}

Tableau de la loi :

3. Espérance : E(X) = 0 \times \frac{2}{9} + 1 \times \frac{5}{9} + 2 \times \frac{2}{9} = \frac{5}{9} + \frac{4}{9} = 1

Exercice 3 : Équations différentielles et intégration (5 points)
On considère l'équation différentielle (E) : y'' + 4y' + 4y = 0.
1. Résoudre l'équation (E). (1 point)
2. Soit l'équation (E') : y'' + 4y' + 4y = -4. Déterminer une solution particulière constante de (E'). (0,5 point)
3. En déduire la solution générale de (E'). (0,5 point)
4. Déterminer la solution f de (E') vérifiant f(0) = -1,5 et f'(0) = 0. (1 point)
5. Calculer la limite de f(x) quand x \to +\infty. (0,5 point)
6. Calculer l'intégrale I = \int_0^2 (x + 0,5)e^{-2x} dx par intégration par parties. (1,5 point)

✅ Corrigé

1. Résolution de (E) : Équation caractéristique r^2 + 4r + 4 = (r+2)^2 = 0, racine double r = -2.

Donc y(x) = (Ax + B)e^{-2x} avec A, B \in \mathbb{R}.

2. Solution particulière constante de (E') : On cherche y = k (constante). Alors y''=0, y'=0, donc 0 + 0 + 4k = -4 \Rightarrow k = -1.

Donc y_p(x) = -1.

3. Solution générale de (E') : y(x) = (Ax + B)e^{-2x} - 1.

4. Solution avec conditions initiales : f(0) = B - 1 = -1,5 \Rightarrow B = -0,5.

f'(x) = Ae^{-2x} - 2(Ax + B)e^{-2x} = (-2Ax + A - 2B)e^{-2x}.

f'(0) = A - 2B = A - 2(-0,5) = A + 1 = 0 \Rightarrow A = -1.

Donc f(x) = (-x - 0,5)e^{-2x} - 1.

5. Limite en +\infty : \lim_{x \to +\infty} e^{-2x} = 0 et \lim_{x \to +\infty} (-x - 0,5)e^{-2x} = 0 (croissance comparée).

Donc \lim_{x \to +\infty} f(x) = -1.

6. Intégration par parties : I = \int_0^2 (x + 0,5)e^{-2x}dx

Posons u = x + 0,5, dv = e^{-2x}dx. Alors du = dx, v = -\frac{1}{2}e^{-2x}.

I = \left[(x+0,5) \times \left(-\frac{1}{2}e^{-2x}\right)\right]_0^2 - \int_0^2 \left(-\frac{1}{2}e^{-2x}\right) dx

= -\frac{1}{2}\left[(x+0,5)e^{-2x}\right]_0^2 + \frac{1}{2}\int_0^2 e^{-2x}dx

= -\frac{1}{2}\left[(2,5)e^{-4} - (0,5)e^0\right] + \frac{1}{2}\left[-\frac{1}{2}e^{-2x}\right]_0^2

= -\frac{1}{2}(2,5e^{-4} - 0,5) - \frac{1}{4}(e^{-4} - 1)

= -1,25e^{-4} + 0,25 - 0,25e^{-4} + 0,25 = -1,5e^{-4} + 0,5

Donc I = \frac{1}{2} - \frac{3}{2}e^{-4}.

Exercice 4 : Ajustement linéaire - Méthode de Mayer (3 points)
Le tableau suivant donne l'évolution de la production mensuelle d'une usine (en milliers de pièces) :
Rang du mois x_i12345
Production y_i (en milliers)1215182227
1. Représenter le nuage de points dans un repère orthogonal. (1 point)
2. Déterminer l'équation de la droite de Mayer (ajustement affine en deux groupes). (1,5 point)
3. Estimer la production prévue pour le mois de rang 7. (0,5 point)

✅ Corrigé

1. Nuage de points : À tracer sur papier millimétré avec x en abscisse (de 1 à 5) et y en ordonnée (de 10 à 30).

2. Méthode de Mayer : On partage la série en deux groupes d'effectifs égaux.

Groupe 1 : mois 1, 2, 3 → points moyens : \bar{x}_1 = \frac{1+2+3}{3}=2, \bar{y}_1 = \frac{12+15+18}{3}=15

Groupe 2 : mois 4, 5 → points moyens : \bar{x}_2 = \frac{4+5}{2}=4,5, \bar{y}_2 = \frac{22+27}{2}=24,5

Coefficient directeur : a = \frac{\bar{y}_2 - \bar{y}_1}{\bar{x}_2 - \bar{x}_1} = \frac{24,5 - 15}{4,5 - 2} = \frac{9,5}{2,5} = 3,8

Ordonnée à l'origine : b = \bar{y}_1 - a\bar{x}_1 = 15 - 3,8 \times 2 = 15 - 7,6 = 7,4

Équation : y = 3,8x + 7,4

3. Estimation pour x=7 : y = 3,8 \times 7 + 7,4 = 26,6 + 7,4 = 34 milliers de pièces.

Partie B : Évaluation des compétences (5 points)
Situation problème : Optimisation d'un espace rectangulaire

Un agriculteur dispose d'un terrain ayant la forme d'un demi-disque de centre O et de rayon 100 mètres. Il souhaite délimiter un espace rectangulaire à l'intérieur de ce terrain pour y installer une serre. Les dimensions du rectangle sont telles que deux de ses sommets sont situés sur le diamètre [OA] et les deux autres sommets sont sur l'arc de cercle du demi-disque.

On note x la distance OM (en mètres), où M est le pied de la hauteur du rectangle sur le diamètre. La largeur du rectangle est donc 2x et sa hauteur est h = \sqrt{R^2 - x^2} avec R=100.

1. Exprimer l'aire \mathcal{A}(x) du rectangle en fonction de x. (0,5 point)
2. Étudier les variations de la fonction \mathcal{A} sur l'intervalle [0; 100]. (1,5 point)
3. Déterminer la valeur de x pour laquelle l'aire du rectangle est maximale. Calculer cette aire maximale. (1 point)
4. L'épouse de l'agriculteur souhaite que l'espace rectangulaire n'occupe que la moitié de l'aire totale du demi-disque. Existe-t-il une position du point M permettant d'obtenir cette aire ? Justifier. (2 points)

✅ Corrigé

1. Expression de l'aire : \mathcal{A}(x) = 2x \times \sqrt{100^2 - x^2} = 2x\sqrt{10000 - x^2} pour x \in [0; 100].

2. Étude des variations : On pose f(x) = 2x\sqrt{10000 - x^2}.

Dérivée : f'(x) = 2\sqrt{10000 - x^2} + 2x \times \frac{-x}{\sqrt{10000 - x^2}} = 2\sqrt{10000 - x^2} - \frac{2x^2}{\sqrt{10000 - x^2}}

On réduit au même dénominateur : f'(x) = \frac{2(10000 - x^2) - 2x^2}{\sqrt{10000 - x^2}} = \frac{20000 - 4x^2}{\sqrt{10000 - x^2}} = \frac{4(5000 - x^2)}{\sqrt{10000 - x^2}}

Le signe de f'(x) est celui de 5000 - x^2.

f'(x) \ge 0 \Leftrightarrow x^2 \le 5000 \Leftrightarrow 0 \le x \le \sqrt{5000} = 50\sqrt{2} \approx 70,71

f est donc croissante sur [0; 50\sqrt{2}] et décroissante sur [50\sqrt{2}; 100].

3. Aire maximale : Maximum atteint pour x_0 = 50\sqrt{2} \approx 70,71 m.

\mathcal{A}_{max} = 2 \times 50\sqrt{2} \times \sqrt{10000 - (50\sqrt{2})^2} = 100\sqrt{2} \times \sqrt{10000 - 5000} = 100\sqrt{2} \times \sqrt{5000}

\sqrt{5000} = \sqrt{2500 \times 2} = 50\sqrt{2}. Donc \mathcal{A}_{max} = 100\sqrt{2} \times 50\sqrt{2} = 100 \times 50 \times 2 = 10000 m².

4. Demi-disque et condition de l'épouse : Aire du demi-disque = \frac{1}{2}\pi R^2 = \frac{1}{2}\pi \times 10000 = 5000\pi m².

La moitié de cette aire est 2500\pi \approx 7854 m².

L'aire maximale du rectangle est 10000 m² > 7854 m².

De plus, \mathcal{A}(0)=0 et \mathcal{A} est continue sur [0; x_0] (où elle est croissante de 0 à 10000).

D'après le théorème des valeurs intermédiaires, il existe une unique valeur x_1 \in ]0; x_0[ telle que \mathcal{A}(x_1)=2500\pi.

Donc oui, il existe une position du point M permettant d'obtenir une aire rectangulaire égale à la moitié de l'aire du demi-disque.
