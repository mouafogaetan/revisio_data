📝 EXERCICE 1 (4 points) • Équations différentielles
1. Résoudre dans ℝ l'équation différentielle : y'' + 4y' + 4y = 0
(Rappel : il s'agit d'une équation homogène du second ordre à coefficients constants)

✅ Corrigé

Équation caractéristique : r² + 4r + 4 = 0
Δ = 16 - 16 = 0 → racine double r₀ = -2

La solution générale est donc : y(x) = (Ax + B)e^{-2x} où A et B sont des constantes réelles.

📌 D'après le cours sur les équations différentielles du second ordre, lorsque le discriminant est nul, les solutions sont de la forme (Ax+B)e^{r₀x}.
📝 EXERCICE 1 (suite)
2. Déterminer la solution particulière de l'équation y'' + 4y' + 4y = 8 vérifiant les conditions initiales y(0) = 1 et y'(0) = 0.

✅ Corrigé

Étape 1 : Solution particulière
Le second membre est constant (8). On cherche une solution constante y = k.
y'' = 0, y' = 0 → 0 + 0 + 4k = 8 → k = 2.
Donc yₚ = 2.

Étape 2 : Solution générale
y(x) = y_h(x) + yₚ = (Ax + B)e^{-2x} + 2

Étape 3 : Conditions initiales
y(0) = B + 2 = 1 → B = -1
y'(x) = Ae^{-2x} - 2(Ax + B)e^{-2x}
y'(0) = A - 2B = 0 → A - 2(-1) = A + 2 = 0 → A = -2

Solution : y(x) = (-2x - 1)e^{-2x} + 2

📊 EXERCICE 2 (5 points) • Ajustement linéaire
Une entreprise agricole au Cameroun étudie l'évolution de sa production de cacao (en tonnes) en fonction du rang de l'année.

Rang (x)12345
Production (y)1215192430
1. Déterminer les points moyens G₁ des deux premières années et G₂ des trois dernières années.
2. Déterminer l'équation de la droite de Mayer (D) passant par G₁ et G₂.
3. Estimer la production prévue pour le rang 8 (année suivante).

✅ Corrigé

1. Points moyens
G₁ (x̄₁, ȳ₁) : x̄₁ = (1+2)/2 = 1,5 ; ȳ₁ = (12+15)/2 = 13,5 → G₁(1,5 ; 13,5)
G₂ (x̄₂, ȳ₂) : x̄₂ = (3+4+5)/3 = 4 ; ȳ₂ = (19+24+30)/3 = 24,33 → G₂(4 ; 24,33)

2. Équation de la droite
Coefficient directeur a = (24,33 - 13,5)/(4 - 1,5) = 10,83/2,5 = 4,332
Ordonnée à l'origine : b = 13,5 - 4,332×1,5 = 13,5 - 6,498 = 7,002
y = 4,33x + 7,00

3. Prévision pour x = 8
y = 4,33×8 + 7 = 34,64 + 7 = 41,64 tonnes

📌 Méthode de Mayer : partage de la série en deux sous-groupes de même effectif. Référence : programme Chap4 "Ajustement linéaire par la méthode de Mayer" [citation:2]
🎲 EXERCICE 3 (5 points) • Probabilités
Une urne contient 5 boules rouges et 3 boules vertes. On tire successivement 3 boules avec remise.

1. Quelle est la probabilité d'obtenir exactement 2 boules rouges ?
2. Soit X la variable aléatoire égale au nombre de boules rouges obtenues. Déterminer la loi de probabilité de X.
3. Calculer l'espérance mathématique E(X).

✅ Corrigé

1. Probabilité d'obtenir exactement 2 rouges
p(Rouge) = 5/8 ; p(Vert) = 3/8
C'est un schéma de Bernoulli (n=3, p=5/8).
P(X=2) = C₃² × (5/8)² × (3/8)¹ = 3 × (25/64) × (3/8) = 3 × 75/512 = 225/512 ≈ 0,439

2. Loi de probabilité de X
X suit la loi binomiale ℬ(3 ; 5/8)
P(X=0) = (3/8)³ = 27/512 ≈ 0,053
P(X=1) = C₃¹ × (5/8)¹ × (3/8)² = 3 × 5/8 × 9/64 = 135/512 ≈ 0,264
P(X=2) = 225/512 ≈ 0,439
P(X=3) = (5/8)³ = 125/512 ≈ 0,244

3. Espérance
E(X) = n × p = 3 × (5/8) = 15/8 = 1,875

📌 Tirage avec remise → épreuves indépendantes → loi binomiale. Référence : programme Chap3 "Tirage avec remise"
🏦 EXERCICE 4 (6 points) • Problème de synthèse
Un entrepreneur camerounais place un capital de 1 000 000 FCFA à un taux d'intérêt composé annuel de 5%.

1. Calculer le capital obtenu après 3 ans.
2. Au bout de combien d'années le capital aura-t-il doublé ? (Arrondir à l'unité)
3. L'entrepreneur souhaite obtenir 1 500 000 FCFA. Déterminer la durée nécessaire (en années) pour atteindre cet objectif.

✅ Corrigé

Formule des intérêts composés : C = C₀ × (1 + t)ⁿ avec t = 0,05

1. Après 3 ans
C = 1 000 000 × (1,05)³ = 1 000 000 × 1,157625 = 1 157 625 FCFA

2. Temps de doublement
2 000 000 = 1 000 000 × (1,05)ⁿ → 2 = (1,05)ⁿ
n = ln(2)/ln(1,05) = 0,6931/0,04879 ≈ 14,2 → 15 ans

3. Pour atteindre 1 500 000 FCFA
1 500 000 = 1 000 000 × (1,05)ⁿ → 1,5 = (1,05)ⁿ
n = ln(1,5)/ln(1,05) = 0,4055/0,04879 ≈ 8,31 → 9 ans
