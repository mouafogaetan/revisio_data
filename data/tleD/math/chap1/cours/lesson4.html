📐 Lieux géométriques dans le plan complexe

Terminale D · Mathématiques · Fiche de révision (Leçon 4)

📌 Éléments clés à retenir

Lien entre affixe et géométrie

Dans le plan complexe, tout point $M$ est repéré par son affixe $z = x + iy$ (coordonnées cartésiennes). Un ensemble de points $M$ dont l'affixe vérifie une condition complexe constitue un lieu géométrique (droite, cercle, médiatrice, etc.). Les outils fondamentaux sont le module $|z|$ (distance à l'origine) et l'argument $\arg(z)$ (angle avec l'axe réel).

  • Module : $|z| = \sqrt{x^2 + y^2} = OM$ où $O$ est l'origine.
  • Argument : $\arg(z) = \theta$ tel que $z = |z|(\cos\theta + i\sin\theta)$.
  • Distance entre deux points : $AB = |z_B - z_A|$.
Illustration : affixe, module et argument dans le plan complexe.

Cercle défini par une égalité de modules

L'ensemble des points $M$ d'affixe $z$ tels que $|z - a| = R$ (avec $a \in \mathbb{C}$ et $R > 0$) est le cercle de centre $A(a)$ et de rayon $R$.

  • Forme générale : $|z - z_A| = r$ (équation d'un cercle).
  • Cas particulier : $|z| = r$ → cercle centré à l'origine.
  • Lorsque $|z - a| = |z - b|$, il s'agit de la médiatrice du segment $[AB]$.
Exemple : $|z - (2 + i)| = 3$ → cercle de centre $A(2,1)$, rayon 3.

Droites et demi-droites par argument constant

Une condition sur l'argument définit une demi-droite ou une droite privée d'un point.

  • $\arg(z - a) = \theta \pmod{\pi}$ → droite passant par $A(a)$ (sauf $A$) avec un angle $\theta$ par rapport à l'axe réel.
  • $\arg(z - a) = \theta \pmod{2\pi}$ → demi-droite d'origine $A$ et de direction $\theta$.
  • La condition $\arg\left(\dfrac{z - a}{z - b}\right) = 0 \pmod{\pi}$ signifie que les points $M, A, B$ sont alignés.

Lieux issus de transformations

L'image d'un cercle ou d'une droite par une similitude directe (vue au chapitre 1) reste un cercle ou une droite. Pour déterminer un lieu, on peut parfois utiliser l'écriture complexe de la transformation.

💡 Exemples de compréhension

Exemple 1 : Déterminer un cercle à partir d'une équation complexe

Soit $z \in \mathbb{C}$. Déterminer l'ensemble $\mathcal{E}$ des points $M$ d'affixe $z$ vérifiant $|z - 3 + 2i| = 4$.

Résolution : On écrit $z - (3 - 2i)$ (car $3 - 2i$ est l'affixe du centre). En effet $z - 3 + 2i = z - (3 - 2i)$.
$|z - (3 - 2i)| = 4$ signifie que la distance de $M$ au point $A(3;-2)$ est égale à $4$.
$\Rightarrow \mathcal{E}$ est le cercle de centre $A(3, -2)$ et de rayon $4$.

💡 Astuce : Toujours mettre sous la forme $|z - z_A| = R$ pour identifier le centre $A(z_A)$ et le rayon $R$.

Exemple 2 : Médiatrice comme lieu géométrique

Déterminer l'ensemble des points $M(z)$ tels que $|z - 2i| = |z + 1 - i|$.

On pose $A(0+2i) = A(0;2)$ et $B(-1 + i) = B(-1;1)$. L'égalité des modules $|z - z_A| = |z - z_B|$ signifie $MA = MB$.
L'ensemble est donc la médiatrice du segment $[AB]$.
Calcul rapide : coordonnées de $A(0,2)$, $B(-1,1)$ → milieu $I(-0.5 ; 1.5)$ et pente de $(AB)$ : $\frac{1-2}{-1-0}=1$, donc médiatrice de pente $-1$ : $y - 1.5 = -1(x + 0.5)$ soit $y = -x + 1$.

💬 La condition $|z - a| = |z - b|$ équivaut à $\dfrac{z-a}{z-b}$ de module 1, soit une transformation purement géométrique.

Exemple 3 : Demi-droite (argument fixe)

Déterminer et décrire le lieu $\mathcal{F} = \{ M(z) \mid \arg(z - (1+i)) = \dfrac{\pi}{4} \pmod{2\pi} \}$.

$\arg(z - (1+i)) = \pi/4 \;(2\pi)$ signifie que le vecteur $\overrightarrow{AM}$ (avec $A(1,1)$) fait un angle de $45^\circ$ avec l'axe des abscisses positives.
Ainsi $\mathcal{F}$ est la demi-droite d'origine $A$ (exclue) dirigée selon l'angle $\pi/4$.

🔍 Attention à la différence entre $(\text{mod } \pi)$ (droite entière) et $(\text{mod } 2\pi)$ (demi-droite). Ne pas confondre!

Exemple 4 : Application avec racines cubiques et lieu

(Inspiré Bac 2023 D) Soit $z$ un complexe non nul. On pose $Z = z + \frac{1}{z}$. Déterminer l'ensemble des points $M(z)$ tels que $Z$ soit réel.

$Z$ réel $\Longleftrightarrow \overline{Z} = Z$ $\Longleftrightarrow \overline{z} + \frac{1}{\overline{z}} = z + \frac{1}{z}$.
Après calculs on aboutit à $(z - \overline{z})(1 - \frac{1}{|z|^2}) = 0$. Donc soit $z - \overline{z}=0$ (axe réel, $z$ réel) soit $|z|=1$ (cercle unité).
L'ensemble est la réunion de l'axe réel et du cercle trigonométrique.

📐 Penser à utiliser la conjugaison pour traduire une condition de réel ou d'imaginaire pur.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre $|z - a| = R$ (cercle) avec $|z - a| \le R$ (disque). L'égalité stricte donne le contour uniquement.
  • Attention : $\arg(z-a)=\theta \pmod{\pi}$ représente une droite (privée du point $a$), tandis que $\pmod{2\pi}$ représente une demi-droite. Ne pas oublier le modulo.
  • Malentendu : L'équation $|z - i| = |z + 1|$ n'est pas un cercle mais une médiatrice (droite).
  • Oubli fréquent : Vérifier les points exclus : dans une demi-droite, l'origine du lieu ($A$) n'appartient pas à l'ensemble car $\arg(z-a)$ n'est pas défini en $z=a$.
  • Astuce inverse : Pour reconnaître un cercle, rechercher $|z - a| = k|z - b|$ avec $k\neq 1$ (cercle d'Apollonius).

📖 Tableau récapitulatif des lieux usuels

Condition complexeLieu géométrique
$|z - a| = r$Cercle de centre $A(a)$, rayon $r$
$|z - a| = |z - b|$Médiatrice du segment $[AB]$
$\arg(z - a) = \theta \pmod{2\pi}$Demi-droite d'origine $A$ de direction $\theta$
$\arg(z - a) = \theta \pmod{\pi}$Droite passant par $A$ faisant un angle $\theta$ avec l'axe réel
$|z - a| = k|z - b|$, $k>0, k\neq 1$Cercle d'Apollonius
$\dfrac{z - a}{z - b}$ est réel positifSegment ouvert $[AB]$ privé des extrémités ? Non : demi-droite [AB) ? Mieux : alignement et même sens → demi-droite
🧠 Retenir : module ↔ distance ; argument ↔ angle. Un lieu se "lit" géométriquement à partir de ces deux notions.
