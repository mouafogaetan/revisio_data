Racines cubiques d'un nombre réel

Mathématiques · Terminale D (Sciences Expérimentales) · Baccalauréat

📌 Éléments clés à retenir

Définition : Racine cubique d'un réel

Soit $a$ un nombre réel. On appelle racine cubique de $a$ tout nombre complexe $z$ tel que : $$z^3 = a$$ Dans $\mathbb{R}$, il existe une unique racine cubique réelle notée $\sqrt[3]{a}$ (si $a \ge 0$, $\sqrt[3]{a} \ge 0$ ; si $a \le 0$, $\sqrt[3]{a} \le 0$). Mais dans $\mathbb{C}$, l'équation $z^3 = a$ admet trois solutions distinctes (sauf si $a=0$).

  • Pour $a > 0$ : une racine réelle positive $\sqrt[3]{a}$ et deux racines complexes conjuguées.
  • Pour $a < 0$ : une racine réelle négative $\sqrt[3]{a}$ et deux racines complexes non réelles.
  • Pour $a = 0$ : une seule racine (triple) $z = 0$.

Forme trigonométrique & méthode de calcul

Pour déterminer les trois racines cubiques d'un réel $a$, on utilise la forme trigonométrique des nombres complexes. On écrit $a = \rho e^{i\theta}$ avec $\rho = |a|$ et $\theta = 0$ si $a>0$, $\theta = \pi$ si $a<0$. Les solutions de $z^3 = a$ sont données par : $$z_k = \sqrt[3]{\rho} \; e^{i\frac{\theta + 2k\pi}{3}} \quad \text{pour } k = 0, 1, 2$$

Figure : Les trois racines cubiques de 8 dans le plan complexe (sommets d'un triangle équilatéral).
  • Cas $a>0$ : $z_0 = \sqrt[3]{a}$ (réel positif), $z_1 = \sqrt[3]{a} \cdot e^{i\frac{2\pi}{3}}$, $z_2 = \sqrt[3]{a} \cdot e^{-i\frac{2\pi}{3}}$.
  • Cas $a<0$ : poser $a = -b$ avec $b>0$. Alors $z_0 = -\sqrt[3]{b}$ (réel négatif), $z_1 = \sqrt[3]{b} \cdot e^{i\frac{\pi}{3}}$, $z_2 = \sqrt[3]{b} \cdot e^{-i\frac{\pi}{3}}$.
  • Formule algébrique : si $a \in \mathbb{R}^*$, les racines cubiques sont de la forme $\sqrt[3]{|a|} \times \omega$ où $\omega$ est une racine cubique complexe de $1$ ou $-1$ selon le signe.

Lien avec les racines de l'unité

Les racines cubiques complexes de $1$ (souvent notées $j$ et $j^2$) jouent un rôle fondamental. $$j = e^{i\frac{2\pi}{3}} = -\frac{1}{2} + i\frac{\sqrt{3}}{2}, \quad j^2 = e^{i\frac{4\pi}{3}} = -\frac{1}{2} - i\frac{\sqrt{3}}{2}$$ Elles vérifient : $1 + j + j^2 = 0$ et $j^3 = 1$. Ainsi, les racines cubiques d'un réel $a>0$ sont $\sqrt[3]{a}$, $\sqrt[3]{a}\,j$, $\sqrt[3]{a}\,j^2$.

💡 Exemples de compréhension

Exemple 1 : Racines cubiques de 8 (session 2023 D)

Résoudre dans $\mathbb{C}$ l'équation $z^3 = 8$.

Résolution :
On a $8 = 8 \cdot e^{i0}$. Le module est $\rho = 8$, donc $\sqrt[3]{\rho} = 2$. Les trois racines cubiques sont données pour $k = 0,1,2$ : $$z_k = 2 \cdot e^{i\frac{0+2k\pi}{3}}$$
- $k=0$ : $z_0 = 2 e^{i0} = 2$ (racine réelle)
- $k=1$ : $z_1 = 2 e^{i\frac{2\pi}{3}} = 2\left(-\frac12 + i\frac{\sqrt3}{2}\right) = -1 + i\sqrt{3}$
- $k=2$ : $z_2 = 2 e^{i\frac{4\pi}{3}} = 2\left(-\frac12 - i\frac{\sqrt3}{2}\right) = -1 - i\sqrt{3}$
Ensemble solution : $\left\{2,\; -1 + i\sqrt{3},\; -1 - i\sqrt{3}\right\}$.

💬 Astuce : Vérifiez que $z_1$ et $z_2$ sont conjugués. Leur somme vaut $-2$ et leur produit vaut $4$.

Exemple 2 : Racines cubiques de $-27$

Déterminer les trois nombres complexes $z$ tels que $z^3 = -27$.

Méthode trigonométrique : $-27 = 27 \cdot e^{i\pi}$ (module $27$, argument $\pi$). $\sqrt[3]{27}=3$. Les racines : $z_k = 3 \cdot e^{i\frac{\pi + 2k\pi}{3}}$, $k=0,1,2$.
- $k=0$ : $z_0 = 3 e^{i\frac{\pi}{3}} = 3\left(\frac12 + i\frac{\sqrt3}{2}\right) = \frac32 + i\frac{3\sqrt3}{2}$
- $k=1$ : $z_1 = 3 e^{i\pi} = -3$ (racine réelle)
- $k=2$ : $z_2 = 3 e^{i\frac{5\pi}{3}} = 3\left(\frac12 - i\frac{\sqrt3}{2}\right) = \frac32 - i\frac{3\sqrt3}{2}$
On retrouve bien une racine réelle négative $-3$ et deux complexes conjuguées.

💡 Pensez à utiliser la forme exponentielle : c'est la méthode la plus sûre et rapide pour les racines $n$-ièmes.

Exemple 3 : Application avec équation

Résoudre $(z+1)^3 = 8$ dans $\mathbb{C}$.

Démarche : Posons $Z = z+1$. L'équation devient $Z^3 = 8$. D'après l'exemple 1, les solutions en $Z$ sont : $Z_1 = 2$, $Z_2 = -1 + i\sqrt{3}$, $Z_3 = -1 - i\sqrt{3}$. On revient à $z = Z - 1$ : $$z_1 = 2-1 = 1,\quad z_2 = (-1 + i\sqrt{3}) - 1 = -2 + i\sqrt{3},\quad z_3 = (-1 - i\sqrt{3}) - 1 = -2 - i\sqrt{3}$$ L'ensemble solution est $\left\{1,\; -2 + i\sqrt{3},\; -2 - i\sqrt{3}\right\}$.

🔁 Cette technique de changement de variable fonctionne pour toute équation de la forme $(az+b)^n = c$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Ne donner que la racine cubique réelle dans $\mathbb{C}$. En terminale D, on attend toujours les trois racines complexes (sauf mention contraire).
  • Attention : Pour $a$ négatif, ne pas écrire $\sqrt[3]{a}$ comme une racine réelle négative puis appliquer la formule des racines $n$-ièmes sans passer par l'argument $\pi$.
  • Confusion : Ne pas mélanger les racines cubiques de l'unité ($j$ et $j^2$) avec les racines cubiques d'un réel positif : on multiplie par $\sqrt[3]{a}$.
  • Piège des modules : Lorsque $a<0$, le module est $|a|$ et l'argument est $\pi$ (et non $0$).

🧠 Méthode pas à pas

Marche à suivre pour trouver les trois racines cubiques d'un réel $a$

  1. Déterminer le signe de $a$.
  2. Calculer le module $|a|$ et la racine cubique réelle $\sqrt[3]{|a|}$.
  3. Écrire $a$ sous forme exponentielle : $a = |a| e^{i\theta}$ avec $\theta = 0$ si $a>0$, $\theta = \pi$ si $a<0$.
  4. Appliquer la formule des racines $n$-ièmes avec $n=3$ : $z_k = \sqrt[3]{|a|} \cdot e^{i\frac{\theta + 2k\pi}{3}}$ pour $k = 0,1,2$.
  5. Exprimer chaque solution sous forme algébrique ($x+iy$) si demandé.
  6. Vérifier éventuellement que $z_1$ et $z_2$ sont conjugués lorsque $a$ est réel.
Organigramme mental : de l'équation $z^3=a$ aux trois racines dans $\mathbb{C}$.
