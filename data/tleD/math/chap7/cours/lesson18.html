📈 Leçon 18 : Suites arithmétiques et géométriques

Mathématiques · Terminale D · Préparation baccalauréat

📌 Éléments clés à retenir

🔢 Suite arithmétique

Une suite $(u_n)$ est arithmétique si la différence entre deux termes consécutifs est constante. Cette constante est appelée raison et notée $r$.

  • Relation de récurrence : $u_{n+1} = u_n + r$ pour tout $n \in \mathbb{N}$.
  • Terme général : $u_n = u_0 + n \cdot r$ ou $u_n = u_p + (n-p) \cdot r$.
  • Sens de variation :
    • Si $r > 0$ : suite strictement croissante.
    • Si $r < 0$ : suite strictement décroissante.
    • Si $r = 0$ : suite constante.
  • Représentation graphique : points alignés sur une droite de pente $r$.
💡 Astuce : Dans une suite arithmétique, la différence $u_{n+1} - u_n$ ne dépend pas de $n$.

🔁 Suite géométrique

Une suite $(v_n)$ est géométrique si le quotient entre deux termes consécutifs est constant (non nul). Cette constante est appelée raison et notée $q$.

  • Relation de récurrence : $v_{n+1} = v_n \times q$ pour tout $n \in \mathbb{N}$.
  • Terme général : $v_n = v_0 \cdot q^{n}$ ou $v_n = v_p \cdot q^{\,n-p}$.
  • Sens de variation (si $v_0 > 0$) :
    • $q > 1$ : suite strictement croissante.
    • $0 < q < 1$ : suite strictement décroissante.
    • $q = 1$ : suite constante.
    • $q < 0$ : suite alternée (non monotone).
  • Cas particulier : Si $v_0 = 0$ alors tous les termes sont nuls.
💡 Astuce : Pour une suite géométrique, le rapport $\frac{v_{n+1}}{v_n}$ est constant (pour $v_n \neq 0$).

📊 Récapitulatif des formules essentielles

Propriété Suite arithmétique Suite géométrique
Récurrence $u_{n+1}=u_n+r$ $v_{n+1}=v_n \times q$
Terme général $u_n = u_0 + nr$ $v_n = v_0 \cdot q^n$
Somme des $n$ premiers termes $S_n = \frac{n}{2}(u_0+u_{n-1})$ $S_n = v_0\frac{1-q^n}{1-q}\ (q\neq1)$

⚡ Pour les suites arithmétiques : $S_n = \frac{\text{nombre de termes} \times (\text{premier terme} + \text{dernier terme})}{2}$.
⚡ Pour les suites géométriques : bien vérifier $q \neq 1$ avant d'utiliser la formule.

💡 Exemples de compréhension

Exemple 1 : Suite arithmétique

Soit $(u_n)$ la suite définie par $u_0 = 5$ et $u_{n+1} = u_n - 3$. Déterminer $u_5$ et le sens de variation.

🔍 Résolution :
- La raison est $r = -3$ (car on soustrait 3 à chaque étape).
- Terme général : $u_n = u_0 + n \times r = 5 - 3n$.
- Donc $u_5 = 5 - 3 \times 5 = 5 - 15 = -10$.
- Puisque $r = -3 < 0$, la suite $(u_n)$ est strictement décroissante.
💬 À retenir : Une raison négative entraîne une décroissance. Pour calculer rapidement un terme, on utilise $u_n = u_0 + nr$.

Exemple 2 : Suite géométrique

Une population de bactéries triple chaque heure. Initialement, on compte 200 bactéries. Soit $v_n$ le nombre de bactéries après $n$ heures. Exprimer $v_n$ et calculer $v_4$.

🧫 Résolution :
- $v_0 = 200$, et chaque heure la population est multipliée par 3 donc $q = 3$.
- Relation : $v_{n+1} = 3 v_n$ → suite géométrique.
- Terme général : $v_n = v_0 \times q^n = 200 \times 3^n$.
- $v_4 = 200 \times 3^4 = 200 \times 81 = 16\,200$ bactéries.
💡 Astuce : Dans les problèmes concrets, identifiez d'abord si on ajoute une même valeur (arithmétique) ou si on multiplie par un facteur constant (géométrique).

Exemple 3 : Reconnaître la nature d'une suite

Étudier la nature de la suite $(w_n)$ définie par $w_n = 2^{n+1}$.

📐 Résolution :
$w_n = 2^{n+1} = 2 \times 2^n$. On a $w_{n+1} = 2^{(n+1)+1} = 2^{n+2} = 2 \times 2^{n+1} = 2 w_n$.
Donc $w_{n+1} = 2 \cdot w_n$ : $(w_n)$ est une suite géométrique de raison $q=2$ et de premier terme $w_0 = 2^{1}=2$.
⚡ Une expression du type $a \times b^n$ (avec $b>0$) correspond toujours à une suite géométrique de raison $b$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre la raison $r$ d'une suite arithmétique et la raison $q$ d'une suite géométrique. Une suite arithmétique s'obtient par addition (ou soustraction), une suite géométrique par multiplication (ou division).
  • Attention : Pour une suite géométrique, la formule de la somme $\frac{v_0(1-q^n)}{1-q}$ n'est valable que si $q \neq 1$. Si $q = 1$, la somme vaut $n \times v_0$.
  • Piège : Ne pas oublier que $n$ dans le terme général $u_n = u_0 + n r$ représente le nombre de pas à partir de l'indice 0. Si le premier terme est $u_1$, alors $u_n = u_1 + (n-1)r$.
