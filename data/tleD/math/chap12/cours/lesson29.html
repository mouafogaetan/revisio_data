Intégration des fonctions rationnelles

Terminale D · Mathématiques · Méthode de décomposition en éléments simples

📌 Éléments clés à retenir

1. Principe général

Une fonction rationnelle est de la forme $F(x) = \frac{P(x)}{Q(x)}$ où $P$ et $Q$ sont des polynômes. Pour l’intégrer sur un intervalle ne contenant pas les pôles, on la décompose en éléments simples (somme d’une partie polynomiale + fractions simples de types $\frac{A}{(x-a)^k}$ et $\frac{Bx+C}{(x^2+px+q)^n}$).

  • Étape 1 : Vérifier le degré : si $\deg(P) \ge \deg(Q)$, effectuer la division euclidienne.
  • Étape 2 : Factoriser $Q(x)$ sur $\mathbb{R}$ (facteurs linéaires ou quadratiques irréductibles).
  • Étape 3 : Décomposer en éléments simples (coefficients à déterminer).
  • Étape 4 : Intégrer terme à terme (primitives classiques : $\ln|x-a|$, $\arctan$, fractions puissance).
💡 Astuce globale : La décomposition transforme une intégrale complexe en somme d'intégrales immédiates.

2. Primitives de base pour éléments simples

Voici les intégrales usuelles à connaître pour intégrer une fraction rationnelle décomposée :

  • $\displaystyle\int \frac{A}{x-a}\,dx = A \ln|x-a| + C$
  • $\displaystyle\int \frac{A}{(x-a)^n}\,dx = A \cdot \frac{(x-a)^{-n+1}}{-n+1} + C \quad (n \ge 2)$
  • $\displaystyle\int \frac{Bx+C}{x^2+px+q}\,dx$ avec $\Delta<0$ : on fait apparaître la dérivée du dénominateur et une forme $\frac{1}{u^2+1}$ (arctan).
  • Rappel : $\displaystyle\int \frac{dx}{x^2 + a^2} = \frac{1}{a}\arctan\left(\frac{x}{a}\right) + C$
Pour les éléments simples de deuxième espèce (dénominateur quadratique), on utilise la technique de « dérivée du dénominateur » : $$ \frac{Bx+C}{x^2+px+q} = \frac{B}{2}\cdot \frac{2x+p}{x^2+px+q} + \frac{C - \frac{Bp}{2}}{x^2+px+q} $$

3. Schéma récapitulatif de la méthode

Appliquer systématiquement ces quatre étapes garantit une intégration correcte. En contrôle, on attend de maîtriser la décomposition pour des fractions rationnelles simples (degré du dénominateur ≤ 3).

💡 Exemples de compréhension

Exemple 1 : Fraction simple avec facteurs linéaires distincts

Calculer $I = \displaystyle\int \frac{3x+1}{x^2 - x - 2}\,dx$.

Résolution :
1. Factorisation : $x^2 - x - 2 = (x-2)(x+1)$.
2. Décomposition : $\displaystyle \frac{3x+1}{(x-2)(x+1)} = \frac{A}{x-2} + \frac{B}{x+1}$.
Par identification : $A(x+1)+B(x-2)=3x+1$ → $A= \frac{7}{3},\; B= \frac{2}{3}$.
3. Intégration :
$$ I = \int \frac{7/3}{x-2} dx + \int \frac{2/3}{x+1} dx = \frac{7}{3}\ln|x-2| + \frac{2}{3}\ln|x+1| + C. $$
💬 Vérification rapide : dériver le résultat pour retrouver la fraction initiale.

Exemple 2 : Fraction avec dénominateur quadratique irréductible

Déterminer $J = \displaystyle\int \frac{2x+1}{x^2+4x+13}\,dx$.

Méthode :
Le discriminant du dénominateur est $16-52 = -36 <0$. On écrit $\frac{2x+1}{x^2+4x+13}$.
Dérivée du dénominateur : $(x^2+4x+13)' = 2x+4$. On ajuste le numérateur : $2x+1 = (2x+4) - 3$.
$$ J = \int \frac{2x+4}{x^2+4x+13}dx - 3\int \frac{dx}{x^2+4x+13}. $$ La première intégrale donne $\ln|x^2+4x+13|$ (car numérateur = dérivée du dénominateur).
Pour la seconde : $x^2+4x+13 = (x+2)^2 + 9$. On pose $u = x+2$, $du=dx$ : $$ \int \frac{dx}{(x+2)^2+9} = \frac13 \arctan\left(\frac{x+2}{3}\right) + C. $$ Finalement : $J = \ln(x^2+4x+13) - 3\cdot \frac13 \arctan\left(\frac{x+2}{3}\right) + C = \ln(x^2+4x+13) - \arctan\left(\frac{x+2}{3}\right) + C$.
🔍 Toujours vérifier la présence de la dérivée exacte du dénominateur pour simplifier.

Exemple 3 : Fraction avec partie polynomiale (division préalable)

Évaluer $K = \displaystyle\int \frac{x^3+2}{x^2+1}\,dx$.

Division euclidienne : $x^3+2 = x(x^2+1) - x + 2$. Donc $\frac{x^3+2}{x^2+1} = x + \frac{-x+2}{x^2+1}$.
$$ K = \int x\,dx + \int \frac{-x}{x^2+1}dx + 2\int \frac{dx}{x^2+1}. $$ $$ = \frac{x^2}{2} - \frac12 \ln(x^2+1) + 2\arctan(x) + C. $$ On a utilisé $\int \frac{-x}{x^2+1}dx = -\frac12\ln(x^2+1)$.
📐 Si degré(P) ≥ degré(Q), toujours commencer par la division pour simplifier l’intégration.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier la valeur absolue dans $\ln| \text{dénominateur} |$ quand celui-ci peut être négatif sur l’intervalle d’intégration.
  • Attention : Pour les racines multiples $(x-a)^n$, ne pas écrire une seule fraction $\frac{A}{(x-a)^n}$ mais la somme des termes $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + ... + \frac{A_n}{(x-a)^n}$.
  • Piège classique : Confondre la forme $\int \frac{dx}{x^2+a^2}$ avec $\int \frac{dx}{x^2-a^2}$ (celle-ci se décompose en éléments simples $\frac{1}{2a}(\frac{1}{x-a} - \frac{1}{x+a})$).
  • Vérification indispensable : Après décomposition, toujours réduire au même dénominateur pour valider les coefficients.

Extrait corrigé type

On considère la fonction $f(x) = \dfrac{x^2+3x+1}{(x+1)^2(x-2)}$. Décomposer $f$ en éléments simples puis calculer $\displaystyle\int_3^4 f(x)\,dx$.

Décomposition : $f(x) = \frac{A}{x+1} + \frac{B}{(x+1)^2} + \frac{C}{x-2}$.
Par identification : $A = \frac{2}{9},\; B = -\frac{1}{3},\; C = \frac{11}{9}$.
$$ \int f(x)dx = \frac{2}{9}\ln|x+1| + \frac{1}{3(x+1)} + \frac{11}{9}\ln|x-2| + K. $$ Puis évaluer entre 3 et 4 : on obtient une valeur exacte (souvent demandée sous forme $\ln(\dots) + \text{rationnel}$). Cet exemple illustre la puissance de la décomposition pour intégrer sur un intervalle.
✨ Astuce examen : Toujours vérifier si l’intervalle d’intégration contient des pôles (valeurs interdites). Si oui, l’intégrale est impropre.

✅ Résumé express : Pour intégrer $\frac{P(x)}{Q(x)}$ → division si nécessaire → factoriser $Q$ → décomposer en éléments simples → intégrer chaque terme (ln, puissance, arctan).
