Leçon 28 : Décomposition en éléments simples

Mathématiques · Terminale D (Sciences Expérimentales) · Synthèse pour examen

📌 Principes fondamentaux

Qu’est-ce qu’une fraction rationnelle ?

Une fraction rationnelle est le quotient de deux polynômes : $F(x) = \dfrac{P(x)}{Q(x)}$, où $Q(x) \neq 0$. Lorsque le degré de $P$ est supérieur ou égal à celui de $Q$, on effectue d’abord la division euclidienne pour obtenir une partie polynomiale (partie entière) plus une fraction propre (degré du numérateur < degré du dénominateur).

  • Partie entière : $E(x)$ (polynôme).
  • Fraction propre : $\dfrac{R(x)}{Q(x)}$ avec $\deg(R) < \deg(Q)$.
  • Objectif : décomposer cette fraction propre en éléments simples du type $\dfrac{A}{x-a}$, $\dfrac{Bx+C}{(x^2+px+q)^k}$, etc.

Types d'éléments simples (selon les racines du dénominateur)

On factorise $Q(x)$ sur $\mathbb{R}$ en produits de facteurs du premier degré et de trinômes irréductibles du second degré.

  • Pôle simple réel $a$ : $\displaystyle \frac{A}{x-a}$
  • Pôle multiple d'ordre $k$ (réel $a$) : $\displaystyle \frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \dots + \frac{A_k}{(x-a)^k}$
  • Facteur quadratique irréductible $(x^2+px+q)$ : $\displaystyle \frac{Bx+C}{x^2+px+q}$
  • Puissance d’un facteur quadratique $(x^2+px+q)^k$ : $\displaystyle \frac{B_1x+C_1}{x^2+px+q} + \frac{B_2x+C_2}{(x^2+px+q)^2} + \dots$
💡 Astuce-clé : Une fois la forme théorique posée, on détermine les coefficients par identification, en multipliant par le dénominateur commun ou en utilisant des valeurs particulières de $x$.

Méthode pratique en 4 étapes

  • 1. Vérifier le degré : Si $\deg P \ge \deg Q$, effectuer la division euclidienne pour isoler la partie entière.
  • 2. Factoriser le dénominateur $Q(x)$ sur $\mathbb{R}$ (racines réelles et trinômes irréductibles).
  • 3. Écrire la forme théorique de la décomposition en éléments simples (avec coefficients inconnus).
  • 4. Déterminer les coefficients par identification (méthode des valeurs particulières ou système linéaire).

💡 Exemples types (session baccalauréat)

Exemple 1 : Pôles simples réels

Décomposer en éléments simples sur $\mathbb{R}$ : $\displaystyle F(x) = \frac{3x+5}{(x-1)(x+2)}$.

Résolution : La fraction est déjà propre. On écrit :
$\displaystyle \frac{3x+5}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}$.
En multipliant par $(x-1)(x+2)$ : $3x+5 = A(x+2) + B(x-1)$.
Méthode des valeurs particulières :
• $x = 1$ : $3(1)+5 = A(3) + B(0) \Rightarrow 8 = 3A \Rightarrow A = \frac{8}{3}$.
• $x = -2$ : $3(-2)+5 = A(0) + B(-3) \Rightarrow -1 = -3B \Rightarrow B = \frac{1}{3}$.
Ainsi : $\displaystyle F(x) = \frac{8/3}{x-1} + \frac{1/3}{x+2}$.
On peut vérifier : $\frac{8}{3(x-1)} + \frac{1}{3(x+2)} = \frac{8(x+2)+(x-1)}{3(x-1)(x+2)} = \frac{9x+15}{3(x-1)(x+2)} = \frac{3x+5}{(x-1)(x+2)}$.
✨ Pour les pôles simples, choisir les racines du dénominateur donne directement les coefficients.

Exemple 2 : Partie entière + pôle double

Décomposer $G(x) = \dfrac{x^3 + 2x^2 - x + 1}{x^2 (x-1)}$.

Étape 1 : Division euclidienne (car degré numérateur =3, dénominateur=3) ? deg(num)=3, deg(den)=3 → même degré : $x^3+2x^2-x+1$ divisé par $x^3 - x^2$ (en développant $x^2(x-1)=x^3-x^2$). On obtient quotient $1$ et reste $3x^2 - x + 1$. Vérifions : $x^3+2x^2-x+1 = 1\cdot(x^3-x^2) + (3x^2 - x + 1)$.
Donc $G(x) = 1 + \dfrac{3x^2 - x + 1}{x^2(x-1)}$.
Étape 2 : Décomposer la fraction propre $H(x) = \dfrac{3x^2 - x + 1}{x^2(x-1)}$.
Forme : $\displaystyle \frac{A}{x-1} + \frac{B}{x} + \frac{C}{x^2}$ (pôle $x=0$ double).
Multiplions par $x^2(x-1)$ : $3x^2 - x + 1 = A x^2 + B x(x-1) + C (x-1)$.
Développons : $A x^2 + Bx^2 - Bx + Cx - C = (A+B)x^2 + (-B+C)x - C$.
Identification : \[ \begin{cases} A+B = 3 \\ -B + C = -1 \\ -C = 1 \end{cases} \Rightarrow C = -1,\; -B -1 = -1 \Rightarrow -B = 0 \Rightarrow B=0,\; A+0=3 \Rightarrow A=3. \] Donc $H(x) = \dfrac{3}{x-1} + \dfrac{0}{x} + \dfrac{-1}{x^2} = \dfrac{3}{x-1} - \dfrac{1}{x^2}$.
Finalement : $G(x) = 1 + \dfrac{3}{x-1} - \dfrac{1}{x^2}$.
📌 Ne jamais oublier la partie entière lorsque le degré du numérateur est ≥ degré dénominateur. Pôle double → terme en $\frac{C}{x^2}$.

Exemple 3 : Facteur quadratique irréductible

Décomposer $K(x) = \dfrac{2x^2 + 3x + 5}{(x^2+1)(x-2)}$.

Fraction propre. Forme théorique : $\displaystyle \frac{Ax+B}{x^2+1} + \frac{C}{x-2}$.
Multiplier par $(x^2+1)(x-2)$ : $2x^2+3x+5 = (Ax+B)(x-2) + C(x^2+1)$.
Développons : $(Ax+B)(x-2) = Ax^2 -2Ax + Bx -2B = Ax^2 + (-2A+B)x -2B$.
Ajoutons $Cx^2 + C$ : total = $(A+C)x^2 + (-2A+B)x + (-2B+C)$.
Identification : \[ \begin{cases} A+C = 2 \\ -2A+B = 3 \\ -2B+C = 5 \end{cases} \] De (1) : $C = 2-A$. Remplaçons dans (3) : $-2B + 2 - A = 5 \Rightarrow -2B - A = 3$.
Avec (2) : $B = 3+2A$. Alors $-2(3+2A) - A = -6 -4A - A = -6 -5A = 3 \Rightarrow -5A = 9 \Rightarrow A = -\dfrac{9}{5}$.
$B = 3+2(-\frac{9}{5}) = 3 - \frac{18}{5} = \frac{15-18}{5} = -\frac{3}{5}$.
$C = 2 - (-\frac{9}{5}) = 2 + \frac{9}{5} = \frac{19}{5}$.
Donc $K(x) = \dfrac{-\frac{9}{5}x - \frac{3}{5}}{x^2+1} + \dfrac{\frac{19}{5}}{x-2}$.
Soit $K(x) = -\dfrac{9x+3}{5(x^2+1)} + \dfrac{19}{5(x-2)}$.
🧠 Pour un trinôme irréductible, le numérateur est affine $Bx+C$. Penser à résoudre le système par substitution.

🔗 Lien avec l’intégration (Leçon 29)

Pourquoi décomposer ?

La décomposition en éléments simples permet d’intégrer facilement toute fraction rationnelle. Les primitives des éléments simples sont immédiates :

  • $\displaystyle \int \frac{A}{x-a} dx = A \ln|x-a| + cte$
  • $\displaystyle \int \frac{A}{(x-a)^k} dx = A \cdot \frac{(x-a)^{-k+1}}{-k+1} + cte \quad (k>1)$
  • $\displaystyle \int \frac{Bx+C}{x^2+px+q} dx$ se ramène à $\frac{B}{2}\ln|x^2+px+q| + \text{arctan}$ (après forme canonique).
Exemple rapide : $\displaystyle \int \frac{3x+5}{(x-1)(x+2)} dx = \frac{8}{3}\ln|x-1| + \frac{1}{3}\ln|x+2| + K$.

⚠️ Pièges courants & erreurs à éviter

  • Oublier la partie entière : toujours vérifier le degré avant de décomposer.
  • Mauvaise forme pour les pôles multiples : pour $(x-a)^k$, il faut tous les termes jusqu’à l’ordre $k$ (ex: $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2}$).
  • Confondre les facteurs quadratiques : si le discriminant est négatif, on garde $(x^2+px+q)$ sans factorisation réelle.
  • Erreur de calcul dans l’identification : bien développer et regrouper les coefficients de chaque puissance de $x$.
  • Ne pas simplifier au préalable : simplifier par des facteurs communs avant la décomposition (si $P$ et $Q$ ont une racine commune).
💬 Rappel examen : en Terminale D, la décomposition en éléments simples est souvent la première étape d’un calcul d’intégrale (voir session 2023 D, Exercice 5). Maîtrisez la méthode des valeurs particulières, c’est la plus rapide.

✍️ Exercice d’auto‑évaluation

À vous de jouer !

Décomposer en éléments simples sur $\mathbb{R}$ : $\displaystyle H(x) = \frac{2x^2 - x + 4}{(x^2+2x+2)(x-1)}$.
(Indice : $x^2+2x+2 = (x+1)^2+1$ est irréductible)

📖 Voir la correction détaillée

Forme : $\dfrac{Ax+B}{x^2+2x+2} + \dfrac{C}{x-1}$.
Multiplier par $(x^2+2x+2)(x-1)$ : $2x^2 - x +4 = (Ax+B)(x-1) + C(x^2+2x+2)$.
Développement : $Ax^2 - Ax + Bx - B + Cx^2+2Cx+2C = (A+C)x^2 + (-A+B+2C)x + (-B+2C)$.
Identification : \[ \begin{cases} A+C = 2 \\ -A+B+2C = -1 \\ -B+2C = 4 \end{cases} \] De (1) $C=2-A$. Substituer dans (2) : $-A+B+2(2-A) = -1 \Rightarrow -A+B+4-2A = -1 \Rightarrow -3A+B = -5$.
(3) : $-B+2(2-A)=4 \Rightarrow -B+4-2A=4 \Rightarrow -B-2A=0 \Rightarrow B = -2A$.
Remplaçons B dans $-3A + (-2A) = -5 \Rightarrow -5A = -5 \Rightarrow A=1$. Alors $B = -2$, $C=2-1=1$.
Finalement : $H(x) = \dfrac{x - 2}{x^2+2x+2} + \dfrac{1}{x-1}$.
