Leçon 17 : Décomposition en éléments simples

Terminale D · Mathématiques · Intégration et fractions rationnelles

📌 Éléments clés à retenir

🔷 Qu’est-ce qu’une fraction rationnelle ?

Une fraction rationnelle est le quotient de deux polynômes : $F(x) = \frac{P(x)}{Q(x)}$, avec $Q(x) \neq 0$. L'objectif de la décomposition en éléments simples (DES) est d'écrire cette fraction comme une somme de fractions plus simples, faciles à intégrer ou à étudier.

  • Si $\deg(P) \ge \deg(Q)$, on effectue d'abord la division euclidienne pour obtenir une partie polynômiale (entière).
  • On décompose ensuite la partie rationnelle propre ($\deg(P) < \deg(Q)$) en éléments simples de 1ʳᵉ ou 2ᵉ espèce.
  • Application majeure : intégration de fonctions rationnelles (conformément aux exigences du baccalauréat).

🧩 Formes des éléments simples (selon les racines de $Q(x)$)

Soit $Q(x)$ factorisé dans $\mathbb{R}$. Chaque facteur donne un type d’élément simple.

📌 Racine réelle simple $a$ : $ \displaystyle \frac{A}{x-a} $
📌 Racine réelle multiple d'ordre $k$ : $ \displaystyle \frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \dots + \frac{A_k}{(x-a)^k} $
📌 Facteur quadratique irréductible $x^2+px+q$ (simple) : $ \displaystyle \frac{Bx+C}{x^2+px+q} $
📌 Puissance $m$ d’un facteur quadratique irréductible : $ \displaystyle \frac{B_1x+C_1}{x^2+px+q} + \dots + \frac{B_mx+C_m}{(x^2+px+q)^m} $

⚠️ Les coefficients $A, B, C, \dots$ sont des réels à déterminer par identification ou par des valeurs particulières.

⚙️ Méthode pratique de décomposition

Suivre ces étapes garantit une décomposition correcte pour toute fraction rationnelle propre.

  • Étape 1 : Vérifier que la fraction est propre (sinon effectuer la division euclidienne).
  • Étape 2 : Factoriser complètement $Q(x)$ dans $\mathbb{R}$ (racines réelles et trinômes du second degré à discriminant $\Delta < 0$).
  • Étape 3 : Écrire la forme théorique de la décomposition avec des coefficients inconnus ($A, B, C, \dots$).
  • Étape 4 : Déterminer les coefficients :
    • Multiplier par $Q(x)$ puis identifier les coefficients (méthode par identification polynomiale).
    • Ou utiliser des valeurs particulières de $x$ (souvent les racines).
    • Combiner les deux méthodes pour gagner du temps.

💡 Exemples de compréhension

Exemple 1 : Racines réelles simples (bac 2023 D)

Décomposer en éléments simples dans $\mathbb{R}$ : $ \displaystyle F(x) = \frac{3x+5}{x^2 - 4} $.

Résolution :
1. $x^2 - 4 = (x-2)(x+2)$ (racines simples 2 et -2).
2. Forme : $\displaystyle \frac{3x+5}{(x-2)(x+2)} = \frac{A}{x-2} + \frac{B}{x+2}$.
3. On multiplie par $(x-2)(x+2)$ : $3x+5 = A(x+2) + B(x-2)$.
4. Pour $x = 2$ : $6+5 = A(4) \Rightarrow 11 = 4A \Rightarrow A = \frac{11}{4}$.
   Pour $x = -2$ : $-6+5 = B(-4) \Rightarrow -1 = -4B \Rightarrow B = \frac{1}{4}$.
5. Conclusion : $\displaystyle F(x) = \frac{11/4}{x-2} + \frac{1/4}{x+2}$.
💡 Astuce : utiliser les pôles (racines du dénominateur) donne directement les coefficients sans résoudre un système complet.

Exemple 2 : Racine multiple (ordre 2) et partie entière

Décomposer $ \displaystyle G(x) = \frac{x^3 + 2x^2 + 1}{x^2 (x+1)} $ (fraction rationnelle).

Étapes :
• $\deg(P)=3$, $\deg(Q)=3$ → fraction propre. Pas de division nécessaire (propre car degrés égaux ? attention, $\deg(P)=\deg(Q)=3$, mais on peut avoir une partie entière si $\deg(P)\ge\deg(Q)$ ; ici ils sont égaux, mais on peut aussi décomposer directement. Cependant la règle formelle : on peut aussi faire apparaître une constante. Ici on écrit la forme adaptée.)
• Factorisation : $x^2(x+1)$.
• Forme : $\displaystyle \frac{x^3+2x^2+1}{x^2(x+1)} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x+1}$ (car racine $x=0$ double, $x=-1$ simple).
• Multiplier par $x^2(x+1)$ : $x^3+2x^2+1 = A x(x+1) + B(x+1) + C x^2$.
• Développer : $A x^2 + A x + Bx + B + C x^2 = (A+C)x^2 + (A+B)x + B$.
• Identification : \[ \begin{cases} A+C = 1 & \text{(coeff de }x^3\text{? attention : membre gauche } x^3+2x^2+0x+1) \\ A+B = 0 & \text{(coeff de }x\text{)} \\ B = 1 & \text{(terme constant)} \end{cases} \] On obtient $B=1$, $A = -1$, puis $C = 2$.
• Final : $G(x) = -\dfrac{1}{x} + \dfrac{1}{x^2} + \dfrac{2}{x+1}$.
💬 Pour les racines multiples, pensez à tous les dénominateurs $ (x-a)^k, (x-a)^{k-1}, ..., (x-a) $.

Exemple 3 : Élément simple du second degré (trinôme irréductible)

Décomposer $ \displaystyle H(x) = \frac{2x^2+3}{(x^2+1)(x-1)} $.

Méthode :
• $x^2+1$ est irréductible dans $\mathbb{R}$ (discriminant négatif).
• Forme : $\displaystyle \frac{2x^2+3}{(x^2+1)(x-1)} = \frac{Ax+B}{x^2+1} + \frac{C}{x-1}$.
• Multiplier par $(x^2+1)(x-1)$ : $2x^2+3 = (Ax+B)(x-1) + C(x^2+1)$.
• Développer : $Ax^2 - Ax + Bx - B + Cx^2 + C = (A+C)x^2 + (-A+B)x + (-B+C)$.
• Identification : \[ \begin{cases} A+C = 2 \\ -A+B = 0 \\ -B + C = 3 \end{cases} \] De la 2ᵉ équation $B = A$. La 3ᵉ donne $-A + C = 3$ et la 1ʳᵉ $A+C=2$. On résout : $(-A+C) - (A+C) = 3-2 \Rightarrow -2A = 1 \Rightarrow A = -\frac12$, $B = -\frac12$, puis $C = 2 - A = 2 + \frac12 = \frac52$.
• Conclusion : $\displaystyle H(x) = \frac{-\frac12 x -\frac12}{x^2+1} + \frac{5/2}{x-1}$.
🧠 Ne jamais oublier le numérateur $Bx+C$ pour un trinôme du second degré.

🧮 Application directe : intégration des fractions rationnelles

📖 Primitives des éléments simples

Une fois la fraction décomposée, l'intégration devient immédiate grâce aux primitives classiques :

\[ \int \frac{A}{x-a} \,dx = A \ln|x-a| + C \qquad \int \frac{A}{(x-a)^k} \,dx = \frac{A}{(1-k)(x-a)^{k-1}} + C \;(k>1) \] \[ \int \frac{Bx+C}{x^2+px+q} \,dx \quad \text{(se ramène à logarithme + arctangente via forme canonique)} \]

Exemple type (bac 2023 D) : $\int \frac{3x+5}{x^2-4}dx = \frac{11}{4}\ln|x-2| + \frac{1}{4}\ln|x+2| + \text{constante}$.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier la partie entière (division euclidienne) quand $\deg(P) \ge \deg(Q)$. Cela conduit à une décomposition fausse.
  • Attention : Pour un facteur quadratique irréductible, le numérateur est de la forme $Bx+C$ (jamais une simple constante).
  • Confusion possible : Racine multiple : ne pas omettre les termes d'ordre inférieur. Exemple : pour $(x-1)^3$, il faut $\frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{C}{(x-1)^3}$.
  • Piège classique : Utiliser uniquement la méthode des valeurs particulières sans vérifier la cohérence de tous les coefficients peut générer des erreurs. Combiner avec l'identification.

🎯 Synthèse pour l’épreuve (baccalauréat)

Les sujets de Terminale D (2023, 2024, 2025) utilisent la décomposition en éléments simples dans deux contextes :

  • Calcul d'intégrales de fonctions rationnelles (souvent dans la partie Étude de fonctions).
  • Résolution d'équations différentielles ou recherche de primitives compliquées.
Maîtriser les 3 types (racines simples, multiples, trinômes) garantit de réussir les questions d’intégration.

📌 Dernier conseil : vérifiez toujours que la somme des éléments simples redonne bien la fraction initiale (en réduisant au même dénominateur) pour éviter les erreurs d’étourderie.
