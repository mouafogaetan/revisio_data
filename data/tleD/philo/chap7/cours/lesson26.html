📖 Leçon 26 : L’accord du participe passé

Français · 4ème / 3ème · Révision BAC / BEPC

📌 Éléments clés à retenir

1️⃣ Avec l’auxiliaire AVOIR

Le participe passé conjugué avec « avoir » s’accorde en genre et en nombre avec le complément d’objet direct (COD) si celui-ci est placé avant le verbe. Si le COD est après le verbe ou absent, le participe passé reste invariable.

  • COD avant → accord : Les pommes que j’ai mangées (pommes, fém. plur. → mangées).
  • COD après → pas d’accord : J’ai mangé des pommes (COD après “mangé” → invariable).
  • Astuce mnémotechnique : AVOIR → chercher le COD avant (rien avant = pas d’accord).
  • Formule à retenir : Participe passé (avoir) = accord si COD placé avant.
Illustration : la règle de l'antéposition du COD avec avoir.

2️⃣ Avec l’auxiliaire ÊTRE

Le participe passé conjugué avec « être » s’accorde toujours en genre et en nombre avec le sujet du verbe (comme un adjectif). C’est valable pour les verbes pronominaux (sauf exceptions) et les verbes de mouvement.

  • Exemple : Marie est venue (sujet féminin singulier → venue).
  • Exemple : Ils sont tombés (sujet masculin pluriel → tombés).
  • Attention : Avec les verbes pronominaux de sens passif ou réfléchi, l’accord se fait avec le COD si celui-ci est placé avant.
  • Règle générale : ÊTRE → accord systématique avec le sujet.
Mémo visuel : toujours penser au sujet quand on utilise être.

3️⃣ Cas particuliers : verbes pronominaux & accord

Avec un verbe pronominal (se laver, se parler…), l’accord dépend de la nature du pronom réfléchi (COD ou COI). Règle pratique : si le pronom « se » est COD, on accorde ; s’il est COI, pas d’accord.

  • COD → accord : Elles se sont lavées (se = les laver elles-mêmes, COD).
  • COI → pas d’accord : Elles se sont parlé (se = à elles-mêmes, COI → invariable).
  • Astuce : remplacer par “quelque chose” (COD) ou “à quelqu’un” (COI).

💡 Exemples de compréhension

Exemple 1 : La tarte que j’ai mangée…

Phrase à compléter : Les tartes que nous avons ______ (préparer) sont délicieuses.

Résolution : On identifie l’auxiliaire = avoir. Le COD est « que » (qui remplace « les tartes ») placé avant le verbe → accord. « Les tartes » féminin pluriel → participe passé « préparées ».
✅ Les tartes que nous avons préparées sont délicieuses.

💬 Astuce : Posez la question « quoi ? » après le verbe : nous avons préparé quoi ? → les tartes (COD avant). Donc on accorde !

Exemple 2 : Elles se sont écrit… ou écrites ?

Complétez : Sophie et Julie se sont ______ (écrire) des lettres toute l’année.

Détail : Verbe pronominal « s’écrire ». Le pronom « se » représente-t-il le COD ou COI ? On dit « écrire à quelqu’un » → se = COI (à elles-mêmes). Pas d’accord du participe passé → écrit.
✅ Elles se sont écrit des lettres.

📌 Si l’on avait « Elles se sont lavées » (laver quelqu’un → COD) → accord. Distinguer COI/COD est essentiel !

Exemple 3 : La route a été … (être + participe)

Voix passive : Les routes ont été ______ (construire) récemment.

Analyse : Dans la voix passive, l’auxiliaire est « être » et le participe s’accorde avec le sujet (les routes, féminin pluriel) → construites. ✅ Les routes ont été construites récemment.

💡 Quand vous voyez « être » + participe (voix passive ou verbes de mouvement), cherchez le sujet et accordez-le comme un adjectif.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier l’accord du participe passé avec « avoir » quand le COD est un pronom relatif (que, qu’) placé avant.
    ❌ Les fleurs que j’ai envoyé → ✅ Les fleurs que j’ai envoyées.
  • Attention : Confondre l’auxiliaire « être » dans les verbes pronominaux. Elle s’est lavée (accord) ≠ Elle s’est lavé les mains (ici « les mains » = COD après le verbe, pas d’accord avec « se »).
  • Ne pas oublier : Avec les verbes impersonnels (il a plu, il a neigé) → toujours invariable.
  • Le fameux piège : « Les pommes que j’ai vu tomber » → infinitif après le participe ? Règle particulière : « vu » s’accorde si le COD fait l’action de l’infinitif. « Les pommes que j’ai vues tomber » (pommes = COD, elles tombent → accord).

Moyen mnémotechnique : « Avant d’avoir, cherche l’objet ; avec être, épouse le sujet ; si tu te pronomise, demande-toi si tu es objet ou destinataire. »

✍️ Mini‑test : accorde correctement !

Auto‑évaluation rapide

1. Les lettres que j’ai ______ (écrire) sont sur la table.
2. Marine s’est ______ (téléphoner) plusieurs fois.
3. Les invités sont ______ (partir) très tard.
4. Quelle belle journée ! Nous nous sommes ______ (reposer) au soleil.

Corrections :
1. écrites (COD « que » = lettres, fém. plur., avant le verbe)
2. téléphoné (verbe pronominal « se téléphoner » → se = COI, invariable)
3. partis (auxiliaire être, sujet « les invités » masc. plur.)
4. reposés (verbe pronominal réfléchi, « se reposer » → se = COD, accord avec « nous » masculin pluriel si mixte, ou féminin selon contexte ; ici générique masculin)

💪 Astuce de pro : pour vérifier l’accord avec avoir, remplace le COD par “le/la/les” placé avant. Si ça fonctionne, accorde !
