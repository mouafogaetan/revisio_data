Leçon 8 : Variables, types et opérateurs en C

Informatique · Terminale C & D · Programme officiel (Session 2025)

📌 Éléments clés à retenir

1. Variables : boîtes nommées pour stocker des données

Une variable est un espace mémoire nommé qui peut contenir une valeur modifiable au cours du programme. En C, toute variable doit être déclarée avant d'être utilisée, avec un type précis.

  • Syntaxe de déclaration : type nom_variable; ou type nom_variable = valeur_initiale;
  • Exemple : int age; (déclaration seule) ou float moyenne = 15.5; (initialisation)
  • Règle : Les noms de variables doivent commencer par une lettre ou underscore, sans espace ni caractère spécial.
  • Affectation : age = 18; (on met la valeur à droite dans la variable de gauche)

2. Types de données en C

Le type détermine la nature de la donnée et la taille mémoire allouée. Voici les types primaires essentiels pour le baccalauréat :

TypeDescriptionExemple de déclarationSpécificateur printf/scanf
intNombre entier (positif ou négatif)int compteur = 10;%d
floatNombre à virgule flottante (décimaux)float pi = 3.14;%f
doubleNombre décimal avec double précisiondouble grand = 1.23456789;%lf
charCaractère unique (entre guillemets simples)char lettre = 'A';%c

Remarque : Le type void (absence de type) existe mais n'est pas utilisé pour les variables classiques.

3. Opérateurs en C

Les opérateurs permettent d'effectuer des calculs ou des comparaisons. On distingue :

  • Arithmétiques : + (addition), - (soustraction), * (multiplication), / (division), % (modulo, reste de division entière).
  • Relatifs (comparaison) : == (égal), != (différent), <, <=, >, >=.
  • Logiques : && (ET logique), || (OU logique), ! (NON logique).
  • Affectation composée : +=, -=, *=, /=, %= (ex: x += 5 équivaut à x = x + 5).
  • Incrémentation / décrémentation : i++ (post-incrémentation), ++i (pré-incrémentation), i--, --i.
💡 Astuce : La division entre deux entiers int donne un entier (troncature). Pour un résultat décimal, utilisez float ou (float).

💡 Exemples de compréhension

Exemple 1 : Calcul de moyenne simple

Écrire un programme C qui déclare trois notes (entiers), calcule leur moyenne (float) et affiche le résultat.

Correction commentée :
#include <stdio.h>
int main() {
    int note1 = 14, note2 = 16, note3 = 12;   // déclaration + initialisation
    float moyenne;
    moyenne = (note1 + note2 + note3) / 3.0;   // 3.0 force la division réelle
    printf("La moyenne est : %.2f\n", moyenne);
    return 0;
}

💬 Astuce : Si on divise par 3 (entier) au lieu de 3.0, le résultat serait tronqué (ex: 14+16+12=42, 42/3=14 exact, mais pour d'autres valeurs cela poserait problème).

Exemple 2 : Vérifier si un nombre est compris entre 0 et 100

À partir d’une variable int valeur, écrire une expression logique qui vaut vrai (1) si la valeur est dans l’intervalle [0,100].

Expression : (valeur >= 0 && valeur <= 100)
Exemple d’utilisation :
if (valeur >= 0 && valeur <= 100)
    printf("Valide\n");
else
    printf("Hors intervalle\n");

💡 && signifie que les deux conditions doivent être vraies. Ne pas confondre = (affectation) avec == (comparaison) !

Exemple 3 : Parité et incrémentation

Déterminer si un entier est pair (modulo 2 = 0) et afficher le compteur après incrémentation.

int nb = 7;
if (nb % 2 == 0)
    printf("%d est pair\n", nb);
else
    printf("%d est impair\n", nb);
int compteur = 5;
printf("Post-incrémentation : %d\n", compteur++); // affiche 5 puis compteur devient 6
printf("Après : %d\n", compteur); // affiche 6

⚠️ Attention à l'effet de bord : compteur++ utilise la valeur puis incrémente ; ++compteur incrémente puis utilise la nouvelle valeur.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre l'affectation = et la comparaison == dans les conditions (ex: if(x = 5) au lieu de if(x == 5)).
  • Attention : Division entière : 5 / 2 donne 2 (troncature). Pour obtenir 2.5, il faut utiliser 5 / 2.0 ou caster (float)5/2.
  • À ne pas oublier : Une variable non initialisée contient une valeur aléatoire (poubelle). Toujours initialiser avant de lire.
  • Précision : Le type char stocke un caractère, mais en réalité c’est un petit entier (code ASCII). Ex: 'A' vaut 65.

🎯 Synthèse pour l’examen (Baccalauréat)

En programmation C, maîtrisez :

  • Déclarer des variables avec int, float, double, char.
  • Utiliser les opérateurs arithmétiques pour les calculs, modulo pour la divisibilité.
  • Construire des conditions avec ==, !=, <, >, &&, ||.
  • Écrire des affectations simples et composées.
  • Savoir que printf utilise %d, %f, %c et scanf aussi (avec & pour les variables).
