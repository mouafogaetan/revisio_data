❓ Question 1 (1 pt)
En algorithmique, quelle structure permet d'exécuter un bloc d'instructions tant qu'une condition est vraie ?
  • A. Si...Alors...Sinon
  • B. Pour (For)
  • C. Tant que (While)
  • D. Selon que (Switch)

    • ✅ Réponse correcte : C. Tant que (While)

    Les boucles conditionnelles "Tant que" (While) répètent les instructions tant que la condition de test est vraie. La boucle "Pour" est itérative mais plutôt utilisée quand le nombre d'itérations est connu.

    ❓ Question 2 (1 pt)
    Lequel de ces types n'est PAS un type de base standard en langage C ?
  • A. int
  • B. float
  • C. string
  • D. char

    • ✅ Réponse correcte : C. string

    En langage C, il n'existe pas de type "string" primitif. On utilise des tableaux de caractères (char[]) pour manipuler des chaînes. Les types int, float, char sont bien des types fondamentaux.

    ❓ Question 4 (1 pt)
    Vrai ou Faux : En langage C, une variable déclarée à l'intérieur d'une fonction (variable locale) est accessible depuis n'importe quelle autre fonction du programme.
  • A. Vrai
  • B. Faux

    • ✅ Réponse correcte : B. Faux

    Une variable locale n'existe que dans le bloc où elle est définie (sa fonction). Elle n'est pas accessible depuis d'autres fonctions. Seules les variables globales le sont.

    ❓ Question 5 (1 pt)
    Quelle est l'instruction correcte pour déclarer un tableau (vecteur) de 10 entiers en langage C ?
  • A. int tab[10];
  • B. array int tab[10];
  • C. tab = int[10];
  • D. int tab(10);

    • ✅ Réponse correcte : A. int tab[10];

    La syntaxe en C pour déclarer un tableau est : type nom[Taille];. Ici, int tab[10]; déclare un tableau de 10 entiers.

    📌 PARTIE 2 : Algorithmique (6 pts)
    📝 Exercice 1 : Somme des éléments d'un tableau (3 pts)
    On considère un tableau T de N entiers (N étant un entier positif saisi par l'utilisateur). Écrire un algorithme (en langage naturel ou pseudo-code) qui calcule et affiche la somme de tous les éléments du tableau ainsi que la moyenne.

    ✅ Proposition de corrigé

    Algorithme SommeMoyenneTableau
Variables
   T: tableau d'entiers
   N, i, somme: entiers
   moyenne: réel
Début
   Ecrire("Donner la taille du tableau : ")
   Lire(N)
   // Déclaration du tableau de taille N
   Pour i de 0 à N-1 faire
       Ecrire("Donner l'élément ", i+1, " : ")
       Lire(T[i])
   FinPour
   
   somme ← 0
   Pour i de 0 à N-1 faire
       somme ← somme + T[i]
   FinPour
   
   moyenne ← somme / N
   Ecrire("La somme est : ", somme)
   Ecrire("La moyenne est : ", moyenne)
Fin

    Critères : Boucle de lecture (1 pt), calcul somme (1 pt), calcul et affichage moyenne (1 pt).

    📝 Exercice 2 : Structure conditionnelle (3 pts)

    - Moyenne < 10 → "Insuffisant"
    - 10 ≤ moyenne < 12 → "Passable"
    - 12 ≤ moyenne < 14 → "Assez bien"
    - 14 ≤ moyenne < 16 → "Bien"
    - 16 ≤ moyenne ≤ 20 → "Très bien"

    ✅ Proposition de corrigé

    Algorithme MentionNotes
Variables
   note1, note2, note3, moyenne: réel
Début
   Ecrire("Entrer les trois notes :")
   Lire(note1, note2, note3)
   moyenne ← (note1 + note2 + note3) / 3
   
   Si moyenne < 10 Alors
       Ecrire("Insuffisant")
   Sinon Si moyenne < 12 Alors
       Ecrire("Passable")
   Sinon Si moyenne < 14 Alors
       Ecrire("Assez bien")
   Sinon Si moyenne < 16 Alors
       Ecrire("Bien")
   Sinon
       Ecrire("Très bien")
   FinSi
Fin

    📌 PARTIE 3 : Langage C (9 pts)
    💻 Question 6 : Analyse de programme (3 pts)
    Soit le programme C suivant :
    #include <stdio.h>
int main() {
    int i = 1, s = 0;
    while(i <= 5) {
        s = s + i;
        i++;
    }
    printf("Résultat = %d", s);
    return 0;
}
    Que va afficher ce programme ? Expliquez brièvement l'exécution.

    ✅ Réponse : Résultat = 15

    Le programme calcule la somme des entiers de 1 à 5 : 1+2+3+4+5 = 15. La boucle while s'exécute 5 fois, incrémentant i et cumulant dans s.

    💻 Question 7 : Écrire un programme (4 pts)
    Écrire un programme en langage C qui lit un entier n, puis affiche la table de multiplication de n (de 1 à 10). Exemple : si n=3, afficher :
    3 x 1 = 3
    3 x 2 = 6
    ...
    3 x 10 = 30

    ✅ Proposition de corrigé

    #include <stdio.h>

int main() {
    int n, i;
    printf("Entrez un entier : ");
    scanf("%d", &n);
    
    for(i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", n, i, n*i);
    }
    return 0;
}

    Évaluation : Inclusion de stdio.h et fonction main (1 pt), lecture correcte (1 pt), boucle for adaptée (1 pt), affichage formaté (1 pt).

    💻 Question 8 : Compléter le code (2 pts)
    Compléter le programme C ci-dessous pour qu'il affiche "PAIR" si la variable nb est paire, sinon "IMPAIR".
    #include <stdio.h>
int main() {
    int nb;
    printf("Donner un nombre : ");
    scanf("%d", &nb);
    
    // Écrire la condition if...else ici
    
    return 0;
}

    ✅ Complétion :

    if(nb % 2 == 0) {
    printf("PAIR\n");
} else {
    printf("IMPAIR\n");
}

    L'opérateur modulo (%) donne le reste de la division par 2. Si le reste est 0 → pair, sinon impair.
