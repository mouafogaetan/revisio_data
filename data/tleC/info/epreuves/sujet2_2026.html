📌 Partie A : Évaluation des ressources (8 points)
❓ Question 1 (1 pt)
Quelle est la structure de données permettant de stocker une collection d'éléments de même type accessibles par un indice ?
  • A. Une variable simple
  • B. Un tableau (array)
  • C. Une constante
  • D. Une fonction

    • ✅ Réponse correcte : B. Un tableau (array)

    🔹 Un tableau est une structure de données qui permet de stocker plusieurs éléments de même type, accessibles via un indice (ou index).

    ❓ Question 2 (1 pt)
    Dans un tableau à une dimension (vecteur), quel est l'indice du premier élément ?
  • A. 1
  • B. 0
  • C. -1
  • D. Cela dépend du langage

    • ✅ Réponse correcte : B. 0

    🔹 En algorithmique et dans la plupart des langages de programmation (C, Java, Python...), l'indice du premier élément d'un tableau est 0.

    ❓ Question 3 (1 pt)
    La recherche séquentielle (ou linéaire) dans un tableau :
  • A. Nécessite que le tableau soit trié
  • B. Compare l'élément recherché avec chaque élément un par un
  • C. Utilise toujours la méthode "diviser pour régner"
  • D. Est plus rapide que la recherche dichotomique sur de grands tableaux

    • ✅ Réponse correcte : B. Compare l'élément recherché avec chaque élément un par un

    🔹 La recherche séquentielle parcourt le tableau élément par élément jusqu'à trouver la valeur recherchée ou atteindre la fin du tableau. Elle ne nécessite pas que le tableau soit trié.

    ❓ Question 4 (1 pt)
    Quel est le principe du tri par sélection ?
  • A. On compare des éléments adjacents et on les échange si nécessaire
  • B. On recherche l'élément minimum (ou maximum) et on le place à sa position finale
  • C. On divise le tableau en deux moitiés récursivement
  • D. On insère chaque élément à sa place dans une sous-liste triée

    • ✅ Réponse correcte : B. On recherche l'élément minimum (ou maximum) et on le place à sa position finale

    🔹 Le tri par sélection consiste à trouver le plus petit élément et à l'échanger avec le premier élément, puis à recommencer sur le reste du tableau.

    ❓ Question 5 (1 pt)
    Quelle est la complexité temporelle (dans le pire des cas) de la recherche séquentielle dans un tableau de n éléments ?
  • A. O(1)
  • B. O(log n)
  • C. O(n)
  • D. O(n²)

    • ✅ Réponse correcte : C. O(n)

    🔹 Dans le pire des cas (élément absent ou en dernière position), la recherche séquentielle effectue n comparaisons, d'où une complexité linéaire O(n).

    ❓ Question 6 (1 pt)
    Pour déclarer un tableau d'entiers de 10 éléments en langage C, on utilise :
  • A. int tab[10];
  • B. array int tab[10];
  • C. tab = int[10];
  • D. int tab(10);

    • ✅ Réponse correcte : A. int tab[10];

    🔹 En langage C, la déclaration d'un tableau se fait avec la syntaxe : type nomTableau[taille];

    ❓ Question 7 (1 pt)
    Une matrice (tableau à deux dimensions) de taille 3×4 contient combien d'éléments ?
  • A. 7
  • B. 10
  • C. 12
  • D. 14

    • ✅ Réponse correcte : C. 12

    🔹 Une matrice de dimensions 3 lignes × 4 colonnes contient 3 × 4 = 12 éléments.

    ❓ Question 8 (1 pt)
    Le tri par insertion fonctionne sur le principe suivant :
  • A. On insère chaque élément à sa place dans une séquence déjà triée
  • B. On échange des éléments adjacents jusqu'à obtenir un tableau trié
  • C. On choisit un pivot et on partitionne le tableau
  • D. On fusionne des sous-tableaux triés

    • ✅ Réponse correcte : A. On insère chaque élément à sa place dans une séquence déjà triée

    🔹 Le tri par insertion construit progressivement un tableau trié en insérant chaque nouvel élément à la position qui lui convient dans la partie déjà triée.

    📝 Exercice 1 : Recherche séquentielle (6 pts)
    On considère un tableau T de N entiers (N ≤ 100). On souhaite écrire un algorithme qui recherche la première occurrence d'une valeur X dans ce tableau.

    1. Écrire l'algorithme de recherche séquentielle qui retourne l'indice de X s'il existe, ou -1 sinon. (3 pts)

    2. Traduire cet algorithme en langage C. (3 pts)

    ✅ Corrigé

    1. Algorithme de recherche séquentielle :

    ALGORITHME recherche_sequentielle VAR T : tableau[0..99] d'Entiers N, X, i : Entiers trouve : Booléen DEBUT trouve ← FAUX i ← 0 TANT QUE (i < N) ET (trouve = FAUX) FAIRE SI T[i] = X ALORS trouve ← VRAI SINON i ← i + 1 FIN SI FIN TANT QUE SI trouve = VRAI ALORS RETOURNER i SINON RETOURNER -1 FIN SI FIN

    2. Traduction en langage C :

    #include <stdio.h> int recherche_sequentielle(int T[], int N, int X) { for(int i = 0; i < N; i++) { if(T[i] == X) { return i; } } return -1; } int main() { int T[100], N, X, resultat; printf("Entrez le nombre d'éléments : "); scanf("%d", &N); printf("Entrez les éléments du tableau :\n"); for(int i = 0; i < N; i++) { scanf("%d", &T[i]); } printf("Entrez la valeur à rechercher : "); scanf("%d", &X); resultat = recherche_sequentielle(T, N, X); if(resultat != -1) { printf("L'élément %d se trouve à l'indice %d\n", X, resultat); } else { printf("L'élément %d n'existe pas dans le tableau\n", X); } return 0; }
    📝 Exercice 2 : Manipulation de tableaux (6 pts)
    On dispose d'un tableau A de 10 entiers. Écrire un programme en langage C qui :

    1. Remplit le tableau avec des valeurs saisies par l'utilisateur. (1 pt)
    2. Calcule et affiche la somme de tous les éléments du tableau. (2 pts)
    3. Calcule et affiche la moyenne des éléments du tableau. (1 pt)
    4. Affiche le plus grand élément et sa position. (2 pts)

    ✅ Corrigé

    #include <stdio.h> int main() { int A[10]; int somme = 0; int max, position_max; float moyenne; // 1. Saisie des éléments du tableau printf("Saisissez les 10 éléments du tableau :\n"); for(int i = 0; i < 10; i++) { printf("A[%d] = ", i); scanf("%d", &A[i]); } // Initialisation pour la recherche du maximum max = A[0]; position_max = 0; // 2. Calcul de la somme et recherche du maximum for(int i = 0; i < 10; i++) { somme = somme + A[i]; if(A[i] > max) { max = A[i]; position_max = i; } } // 3. Calcul de la moyenne moyenne = (float)somme / 10; // Affichage des résultats printf("\n--- Résultats ---\n"); printf("Somme des éléments : %d\n", somme); printf("Moyenne des éléments : %.2f\n", moyenne); printf("Le plus grand élément est %d à l'indice %d\n", max, position_max); return 0; }

    Exemple d'exécution :

    Saisissez les 10 éléments du tableau : A[0] = 5 A[1] = 12 A[2] = 3 A[3] = 8 A[4] = 15 A[5] = 7 A[6] = 10 A[7] = 2 A[8] = 9 A[9] = 4 --- Résultats --- Somme des éléments : 75 Moyenne des éléments : 7.50 Le plus grand élément est 15 à l'indice 4
