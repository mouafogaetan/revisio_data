Leçon 45 : Problèmes avec équations différentielles

Terminale C · Modélisation & résolution · Bac 2023/2025

📌 Modéliser et résoudre des problèmes concrets

Équation différentielle linéaire du 1er ordre

Forme générale : $y' + a(x)\,y = b(x)$. À coefficients constants : $y' + a y = b$. Solution générale = solution homogène + solution particulière.

  • Homogène $y' + a y = 0$ → $y_h(t) = C e^{-a t}$
  • Particulière constante si $b$ constant : $y_p = \dfrac{b}{a}$ (avec $a \neq 0$)
  • Utilisation de la condition initiale pour déterminer la constante $C$.

Équation différentielle linéaire du 2nd ordre (coefficients constants)

Forme : $y'' + p y' + q y = f(t)$. Résolution en deux étapes :

  • Solution homogène : équation caractéristique $r^2 + p r + q = 0$.
  • Si $\Delta > 0$ : $y_h = C_1 e^{r_1 t} + C_2 e^{r_2 t}$
  • Si $\Delta = 0$ : $y_h = (C_1 + C_2 t) e^{r t}$
  • Si $\Delta < 0$ : $y_h = e^{\alpha t}\big(C_1 \cos(\beta t) + C_2 \sin(\beta t)\big)$
  • Solution particulière : selon la forme de $f(t)$ (constante, exponentielle, polynôme) → méthode des coefficients indéterminés.
💡 Pour un second membre constant, on cherche $y_p$ constante si $q \neq 0$.

Problèmes types (physique, biologie, économie)

Les équations différentielles traduisent des lois d'évolution :

  • Ressort / Mécanique : $x'' + \omega^2 x = 0$ (oscillations libres) ou $x'' + \frac{k}{m}x = F(t)$.
  • Propagation de maladie : $y' = k y (N-y)$ (logistique) ou $y' = a y + b$ (modèle exponentiel).
  • Circuit RC / charge : $q' + \frac{1}{RC}q = E$.
  • Croissance bactérienne : $P'(t) = \alpha P(t) + \text{apport}$.

💡 Exemples tirés des examens (Bac)

Exemple 1 : Mouvement d’un ressort (Session 2023 C, Partie B)

Un ressort horizontal sans frottement obéit à l'équation $x''(t) + \frac{k}{4}x(t) = 0$, avec $k>0$ constante. À $t=0$, $x(0)=2$ et $x'(0)=0$. Exprimer $x(t)$ et déterminer la période.

Résolution : Équation caractéristique $r^2 + \frac{k}{4} = 0$ → $r = \pm i \frac{\sqrt{k}}{2}$.
Solution générale : $x(t) = C_1 \cos\left(\frac{\sqrt{k}}{2}t\right) + C_2 \sin\left(\frac{\sqrt{k}}{2}t\right)$.
Condition $x(0)=2$ → $C_1 = 2$. $x'(t) = -C_1 \frac{\sqrt{k}}{2}\sin(...) + C_2 \frac{\sqrt{k}}{2}\cos(...)$, avec $x'(0)=0$ → $C_2 \cdot \frac{\sqrt{k}}{2} = 0$ ⇒ $C_2=0$.
Donc $\boxed{x(t) = 2\cos\left(\frac{\sqrt{k}}{2}t\right)}$. Période $T = \frac{2\pi}{\sqrt{k}/2} = \frac{4\pi}{\sqrt{k}}$.
💬 Astuce : Reconnaître $x'' + \omega^2 x = 0$ → oscillations sinusoïdales. Période $T=2\pi/\omega$.

Exemple 2 : Propagation d’une pandémie (Session 2025 C, Partie B)

Un modèle simplifié : $y'(t) = 0,03\,y(t) + 200$, où $y(t)$ est le nombre de cas actifs. À $t=0$, $y(0)=1000$. Déterminer $y(t)$ et la limite quand $t\to +\infty$.

Équation du 1er ordre : $y' - 0,03 y = 200$. Solution homogène : $y_h(t)=C e^{0,03 t}$.
Solution particulière constante : $y_p = \frac{200}{-0,03}$ ? Attention forme $y' = a y + b$ → $y_p = -\frac{b}{a}$ si $a\neq0$. Ici $y' - 0,03 y = 200$ ⇒ $a=-0,03$ ? Reécrivons : $y' = 0,03y +200$ ⇒ $y' -0,03y = 200$. Particulière constante $K$ vérifie $0 - 0,03K = 200$ ⇒ $K = -\frac{200}{0,03} \approx -6666,67$ (pas cohérent physiquement car négatif).
En réalité, pour $y' = a y + b$ ($a>0$) solution générale : $y(t)=Ce^{at} - \frac{b}{a}$.
Ici $a=0,03$, $b=200$ : $y(t)=Ce^{0,03t} - \frac{200}{0,03} = Ce^{0,03t} - 6666,\!67$.
Condition initiale $y(0)=1000$ : $1000 = C - 6666,67$ ⇒ $C = 7666,67$.
Donc $y(t)=7666,67\,e^{0,03t} - 6666,67$. Quand $t\to +\infty$, $y(t) \to +\infty$ (croissance exponentielle).
Interprétation : sans frein, le nombre de cas explose.
🧠 Mémoriser : $y' = a y + b$ (a≠0) ⇒ $y(t) = \left(y_0 + \frac{b}{a}\right) e^{a t} - \frac{b}{a}$.

Exemple 3 : Refroidissement (loi de Newton) – modélisation type

Un objet chaud refroidit selon $\theta'(t) = -k(\theta(t) - \theta_{amb})$, avec $k>0$. $\theta(0)=80°C$, $\theta_{amb}=20°C$. Résoudre et trouver le temps pour atteindre 30°C.

Posons $u(t)=\theta(t)-20$. Alors $u'(t) = \theta'(t) = -k u(t)$. Donc $u(t)=C e^{-kt}$.
$u(0)=60$ ⇒ $C=60$ ⇒ $\theta(t)=20+60e^{-kt}$.
Résoudre $20+60e^{-kt}=30$ ⇒ $60e^{-kt}=10$ ⇒ $e^{-kt}=1/6$ ⇒ $t = \frac{\ln 6}{k}$.
🔥 Penser à centrer l'équation pour se ramener à une homogène simple.

⚠️ Pièges fréquents & Conseils d’examen

  • Erreur classique : Oublier la solution homogène et donner uniquement la particulière. Toujours $y = y_h + y_p$.
  • Confusion : Dans $y' = a y + b$, la particulière constante est $-\frac{b}{a}$ (avec $a\neq0$) et non $\frac{b}{a}$. Vérifiez en remplaçant.
  • Conditions initiales : Mal exploiter $y'(0)$ dans le second ordre (dériver correctement l’expression générale).
  • Unité et modélisation : Toujours vérifier la cohérence des constantes (exponentielle positive ou négative selon le phénomène).
  • Racines complexes : Ne pas oublier la forme $\cos$ et $\sin$ et interpréter comme oscillation amortie ou non.

📐 Marche à suivre (problème concret)

Démarche systématique pour résoudre un problème d’ED

  • 1. Identifier la grandeur inconnue (température, population, position, charge).
  • 2. Traduire l’énoncé en équation : $y' = \dots$ ou $y'' = \dots$ (loi physique, taux de variation).
  • 3. Déterminer l’ordre et le type : 1er ordre linéaire / 2nd ordre / coefficients constants.
  • 4. Résoudre l’équation homogène (équation caractéristique).
  • 5. Trouver une solution particulière (constante, exponentielle, polynôme).
  • 6. Écrire la solution générale et utiliser les conditions initiales (souvent $y(0)$, $y'(0)$).
  • 7. Interpréter la solution dans le contexte (limite, période, temps caractéristique).

📖 Formulaire à maîtriser

Résolutions rapides

  • $y' + a y = 0$ → $y = C e^{-a t}$
  • $y' + a y = b$ (constante) → $y = C e^{-a t} + \dfrac{b}{a}$ (si $a \neq 0$)
  • $y'' + \omega^2 y = 0$ → $y = A\cos(\omega t) + B\sin(\omega t)$
  • $y'' + 2\lambda y' + \omega_0^2 y = 0$ : discriminant $\Delta' = \lambda^2 - \omega_0^2$ → selon signe, régime apériodique/pseudopériodique.
  • Pour $y'' + p y' + q y = K$ (constante) : si $q \neq 0$, $y_p = \frac{K}{q}$.
✨ Astuce examen : avant de résoudre, posez toujours l'équation sous forme normalisée (coefficient 1 devant $y''$ ou $y'$).
