Nombres complexes : conjugué & module

Terminale C · Mathématiques · Révision clé pour le Bac (session 2022-2025)

📌 Éléments clés à retenir

🧩 1. Conjugué d’un nombre complexe

Si $z = x + iy$ (avec $x, y \in \mathbb{R}$), alors son conjugué noté $\overline{z}$ est : \[ \overline{z} = x - iy \] Graphiquement, $\overline{z}$ est le symétrique de $z$ par rapport à l'axe des réels (axe des abscisses).

  • $\overline{\overline{z}} = z$ (l’involution)
  • $\overline{z_1 + z_2} = \overline{z_1} + \overline{z_2}$
  • $\overline{z_1 \cdot z_2} = \overline{z_1} \cdot \overline{z_2}$
  • $\overline{\left(\frac{z_1}{z_2}\right)} = \frac{\overline{z_1}}{\overline{z_2}},\quad (z_2 \neq 0)$
  • $z \in \mathbb{R} \iff \overline{z} = z$   |   $z \in i\mathbb{R} \iff \overline{z} = -z$
💡 Astuce : Le conjugué "renverse" le signe de la partie imaginaire. Très utile pour rendre un dénominateur réel.

📏 2. Module d’un nombre complexe

Le module de $z = x + iy$ est la distance de l'origine au point $M(z)$ dans le plan complexe : \[ |z| = \sqrt{x^2 + y^2} = \sqrt{z \cdot \overline{z}} \] C’est un réel positif ou nul. $|z| = 0 \iff z = 0$.

$|z_1 z_2| = |z_1||z_2|$ $\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}$ $|\overline{z}| = |z|$ $|z|^2 = z \overline{z}$ $|z^n| = |z|^n$

Inégalité triangulaire : $|z_1 + z_2| \le |z_1| + |z_2|$ (égalité ssi les vecteurs sont colinéaires de même sens).

⚡ 3. Liens entre conjugué, module et parties réelle/imaginaire

Pour tout complexe $z$ : \[ \operatorname{Re}(z) = \frac{z + \overline{z}}{2}, \qquad \operatorname{Im}(z) = \frac{z - \overline{z}}{2i} \] Ces formules sont très utiles pour transformer des équations complexes en conditions réelles.

  • $z$ est réel $\iff z = \overline{z}$
  • $z$ est imaginaire pur $\iff z = -\overline{z}$
  • $|z| = |\overline{z}|$ et $|z|^2 = z\overline{z}$

💡 Exemples de compréhension (Bac & exercices types)

Exemple 1 : Calcul avec conjugué et module

Soit $z = 3 - 4i$. Déterminer $\overline{z}$, $|z|$, puis écrire $z \cdot \overline{z}$ sous forme réelle.

Résolution :
$\overline{z} = 3 + 4i$ (changement de signe de la partie imaginaire).
$|z| = \sqrt{3^2 + (-4)^2} = \sqrt{9+16} = \sqrt{25} = 5$.
$z \cdot \overline{z} = (3-4i)(3+4i) = 9 - (4i)^2 = 9 - (-16) = 25 = |z|^2$.

💬 Astuce : Le produit $z \overline{z}$ est toujours un réel positif égal au carré du module.

Exemple 2 : Mettre sous forme algébrique avec le conjugué

Écrire $\displaystyle \frac{2+5i}{1-3i}$ sous forme $a+ib$ (avec $a,b \in \mathbb{R}$).

Méthode : multiplier numérateur et dénominateur par le conjugué du dénominateur $(1+3i)$ :
\[ \frac{2+5i}{1-3i} = \frac{(2+5i)(1+3i)}{(1-3i)(1+3i)} = \frac{2+6i+5i+15i^2}{1 - (3i)^2} \] \[ = \frac{2 + 11i -15}{1 - 9i^2} = \frac{-13 + 11i}{1 + 9} = \frac{-13}{10} + \frac{11}{10}i \] Donc $a = -\frac{13}{10},\ b = \frac{11}{10}$.

🔍 Pensez à $i^2 = -1$ et vérifiez que le dénominateur devient réel grâce au conjugué.

Exemple 3 : Utilisation du module et de l'inégalité triangulaire

On donne $|z_1| = 3$ et $|z_2| = 4$. Déterminer un encadrement possible de $|z_1 + z_2|$ et $|z_1 - z_2|$.

Par l'inégalité triangulaire : $|\,|z_1| - |z_2|\,| \le |z_1 + z_2| \le |z_1| + |z_2|$
soit $|3-4| \le |z_1+z_2| \le 3+4$ → $1 \le |z_1+z_2| \le 7$.
De même pour $z_1 - z_2$ : $|z_1 - z_2| \ge ||z_1| - |z_2|| = 1$ et $\le |z_1| + |z_2| = 7$.

🎯 Liens avec l’épreuve (Baccalauréat)

Sujets types rencontrés

Dans les sessions 2022, 2023, 2024 (Partie A, Exercice 2) les questions sur module et argument sont fréquentes : calcul de distance, interprétation géométrique, résolution d’équations $|z - a| = |z - b|$ (médiatrice) ou $|z - \omega| = R$ (cercle). Le conjugué intervient systématiquement pour montrer qu’un nombre est réel ou pour simplifier des fractions.

  • Égalité de modules : $|z - a| = |z - b|$ équivaut à l'ensemble des points équidistants de $A$ et $B$ → médiatrice.
  • Cercle : $|z - z_0| = R$ décrit un cercle de centre $z_0$ et de rayon $R$.
  • Condition de colinéarité : $\frac{z_1}{z_2} \in \mathbb{R}$ ssi $\arg(z_1) \equiv \arg(z_2) \pmod{\pi}$.
📌 Piège classique : $|z|^2 = z^2$ est faux en général ! Rappel : $|z|^2 = z \overline{z}$.

⚠️ Pièges courants & erreurs fréquentes

  • Erreur n°1 : Écrire $\sqrt{z}$ ou comparer deux complexes avec $<$ ou $>$ (les complexes ne sont pas ordonnés). Le module oui, lui est réel.
  • Attention : $\overline{z_1 + z_2} = \overline{z_1} + \overline{z_2}$ mais $\overline{z_1 \cdot z_2} = \overline{z_1} \cdot \overline{z_2}$ ; ne pas oublier que le conjugué d’une somme est la somme des conjugués.
  • Piège module : $|z|^2 = z^2$ seulement si $z$ est réel. Sinon $z^2$ peut être complexe.
  • Méprise géométrique : $|z - a|$ représente la distance entre les points $M(z)$ et $A(a)$. Ne pas confondre avec $|z| - |a|$.
  • Lors d’une division : toujours multiplier par le conjugué du dénominateur pour obtenir une forme algébrique.

📖 Formulaire à découper mentalement

Synthèse : conjugué & module

$\boxed{\overline{x+iy}=x-iy}$    $\boxed{|x+iy|=\sqrt{x^2+y^2}}$
$\boxed{z\overline{z}=|z|^2}$    $\boxed{\overline{z_1+z_2}=\overline{z_1}+\overline{z_2}}$    $\boxed{\overline{z_1z_2}=\overline{z_1}\,\overline{z_2}}$
$\boxed{|z_1z_2|=|z_1||z_2|}$    $\boxed{|\frac{z_1}{z_2}|=\frac{|z_1|}{|z_2|}}$    $\boxed{|\overline{z}|=|z|}$
$\boxed{\operatorname{Re}(z)=\frac{z+\overline{z}}{2}}$    $\boxed{\operatorname{Im}(z)=\frac{z-\overline{z}}{2i}}$

✏️ Vérifie ta compréhension (mini‑test)

À résoudre seul(e) avant l’examen

1. Soit $z = -2 + 2i$. Calculer $\overline{z}$, $|z|$ et $z\cdot \overline{z}$.
2. Déterminer l’ensemble des points $M(z)$ tels que $|z - 1 + i| = 3$.
3. Mettre $\frac{5}{2+3i}$ sous forme algébrique.
4. Vrai ou faux : $|z_1| = |z_2| \Rightarrow z_1 = \pm z_2$ ? Justifier.

📌 Voir les réponses (clique)

1. $\overline{z} = -2-2i$, $|z| = \sqrt{(-2)^2+2^2} = \sqrt{8}=2\sqrt{2}$, $z\overline{z}=|z|^2=8$.

2. $|z - (1 - i)| = 3$ → cercle de centre $\Omega(1,-1)$ et rayon $3$.

3. $\frac{5(2-3i)}{(2+3i)(2-3i)} = \frac{10-15i}{4+9} = \frac{10}{13} - \frac{15}{13}i$.

4. Faux : des modules égaux signifie que les points sont sur un même cercle centré à l’origine, mais leurs arguments peuvent être différents (ex: $z_1=1$, $z_2=-1$ ont même module $1$ mais $z_1 \neq \pm z_2$? $\pm z_2$ serait $-1$ ou $1$ ; ici $1 = -(-1)$ → si $z_2=-1$, alors $z_1 = -z_2$ ; l'affirmation exacte serait $|z_1|=|z_2|$ n’implique pas $z_1 = \pm z_2$ en général car $z_2 = e^{i\theta}z_1$).
