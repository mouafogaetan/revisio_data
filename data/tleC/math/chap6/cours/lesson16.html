Leçon 16 : Écriture trigonométrique & notation exponentielle

Mathématiques · Terminale C (Scientifique) · Nombres complexes (approche géométrique)

📌 Éléments clés à retenir

Module et argument d'un nombre complexe

Tout nombre complexe $z = x + iy$ (avec $x,y \in \mathbb{R}$) possède un module noté $|z|$ et un argument noté $\arg(z)$ (défini à $2\pi$ près).

  • Module : $|z| = \sqrt{x^2 + y^2}$. Il représente la distance du point $M(z)$ à l'origine dans le plan complexe.
  • Argument : $\theta = \arg(z)$ tel que $\cos\theta = \frac{x}{|z|}$ et $\sin\theta = \frac{y}{|z|}$. Géométriquement, c'est l'angle orienté $(\overrightarrow{OI}, \overrightarrow{OM})$.
  • Formule : $z = |z|(\cos\theta + i\sin\theta)$ (forme trigonométrique).

Notation exponentielle (formule d'Euler)

Grâce à la formule d'Euler $e^{i\theta} = \cos\theta + i\sin\theta$, tout complexe non nul s'écrit sous la forme exponentielle :

  • $$z = |z| e^{i\theta} \quad \text{avec} \quad \theta \equiv \arg(z) \pmod{2\pi}.$$
  • Cette écriture simplifie considérablement les produits, quotients et puissances.
  • Cas particuliers : $e^{i0}=1$, $e^{i\pi/2}=i$, $e^{i\pi}=-1$, $e^{-i\pi/2}=-i$.

Propriétés fondamentales (module et argument)

Pour tous $z, z' \in \mathbb{C}^*$ et $n \in \mathbb{Z}$ :

  • $|z z'| = |z| \times |z'|$ et $\arg(z z') = \arg(z) + \arg(z') \pmod{2\pi}$.
  • $\left|\frac{z}{z'}\right| = \frac{|z|}{|z'|}$ et $\arg\left(\frac{z}{z'}\right) = \arg(z) - \arg(z') \pmod{2\pi}$.
  • $|z^n| = |z|^n$ et $\arg(z^n) = n \arg(z) \pmod{2\pi}$ (Formule de Moivre généralisée).
  • Formule de Moivre : $(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$.

Passage d'une forme à l'autre

Algébrique → Trigonométrique/Exponentielle : Calculer $r = \sqrt{x^2+y^2}$ puis déterminer $\theta$ tel que $\cos\theta = x/r$, $\sin\theta = y/r$ (attention au quadrant).
Exponentielle → Algébrique : Utiliser $e^{i\theta} = \cos\theta + i\sin\theta$.

💡 Exemples de compréhension

Exemple 1 : Passage forme algébrique → exponentielle

Soit $z = 1 + i\sqrt{3}$. Écrire $z$ sous forme exponentielle.

Résolution :
1. Calcul du module : $|z| = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1+3} = 2$.
2. Calcul de l'argument : $\cos\theta = \frac{1}{2}$, $\sin\theta = \frac{\sqrt{3}}{2}$ ⇒ $\theta = \frac{\pi}{3}$ (mod $2\pi$).
3. Forme exponentielle : $z = 2 e^{i\pi/3}$.

💬 Astuce : Vérifiez toujours le quadrant ! Ici $x>0$ et $y>0$ donc $\theta$ dans $]0, \pi/2[$.

Exemple 2 : Utilisation de la formule de Moivre

Calculer $(1+i)^{10}$ sous forme algébrique.

Méthode :
1. Forme exponentielle de $1+i$ : $|1+i| = \sqrt{2}$, $\arg(1+i) = \pi/4$ ⇒ $1+i = \sqrt{2} e^{i\pi/4}$.
2. Puissance : $(1+i)^{10} = (\sqrt{2})^{10} e^{i10\pi/4} = 2^{5} e^{i5\pi/2} = 32 e^{i(5\pi/2)}$.
3. Réduction de l'angle : $5\pi/2 = 2\pi + \pi/2$ ⇒ $e^{i5\pi/2} = e^{i\pi/2} = i$.
4. Résultat : $(1+i)^{10} = 32i$.

💡 Penser à simplifier l'argument modulo $2\pi$ pour obtenir une forme plus simple.

Exemple 3 : Équation et racines complexes

Résoudre dans $\mathbb{C}$ l'équation $z^3 = -8i$.

Résolution exponentielle :
Écrire $-8i$ sous forme exponentielle : $|-8i| = 8$, $\arg(-8i) = -\frac{\pi}{2}$ (car $-i$ a pour argument $-\pi/2$). Donc $-8i = 8 e^{-i\pi/2}$.
On cherche $z = r e^{i\theta}$ tel que $r^3 e^{i3\theta} = 8 e^{-i\pi/2}$ ⇒ $r^3 = 8$ ⇒ $r=2$ (module positif).
$3\theta \equiv -\frac{\pi}{2} + 2k\pi$ ⇒ $\theta = -\frac{\pi}{6} + \frac{2k\pi}{3}$ pour $k=0,1,2$.
Racines : $z_0 = 2 e^{-i\pi/6} = \sqrt{3} - i$, $z_1 = 2 e^{i\pi/2} = 2i$, $z_2 = 2 e^{i7\pi/6} = -\sqrt{3} - i$.

✨ La notation exponentielle transforme une équation de degré $n$ en une simple équation sur le module et l'argument.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre $\arg(z)$ et $\arctan(y/x)$ sans tenir compte du quadrant. Utilisez $\cos\theta$ et $\sin\theta$ pour déterminer l'angle unique.
  • Attention : L'argument n'est pas défini pour $z=0$. Le module est nul, mais aucun argument n'est associé au complexe nul.
  • Piège classique : Oublier que la forme exponentielle est $|z| e^{i\theta}$ avec $\theta$ modulo $2\pi$. Ne pas écrire $z = r e^{i\theta}$ sans préciser que $r\ge 0$.
  • À ne pas confondre : $e^{i\theta}$ n'est pas une exponentielle réelle ; son module vaut 1 (c'est un point sur le cercle unité).
  • Formule de Moivre : $(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$ ne s'applique qu'à un complexe de module 1. Pour un module différent, factorisez d'abord $r^n$.

🎯 Lien avec l'épreuve (Session 2022 C)

Extrait type : Écriture exponentielle et géométrie

On considère les points $A$, $B$ et $C$ d'affixes respectives $z_A = 1+i$, $z_B = \sqrt{3}+i$ et $z_C = -\sqrt{3}+i$. Déterminer la nature du triangle $ABC$ en utilisant la forme exponentielle des rapports complexes.

Piste de résolution : Calculer $\frac{z_B - z_A}{z_C - z_A}$ sous forme exponentielle. Si le module est 1 et l'argument $\pm \pi/2$ ou $\pm \pi/3$, on peut caractériser un triangle rectangle ou équilatéral. L'écriture exponentielle permet de lire directement l'angle et le rapport de longueurs.

📌 Maîtriser le passage rapide de l'algébrique à l'exponentielle est essentiel pour gagner du temps le jour de l'examen.
