Leçon 15 : Image et affixe d’un complexe

Terminale C · Approche géométrique des complexes · 2e trimestre

📌 Éléments clés à retenir

1. Plan complexe : affixe et image

Tout nombre complexe $z = x + iy$ (avec $x, y \in \mathbb{R}$) est associé à un unique point $M(x; y)$ du plan muni d’un repère orthonormé $(O; \vec{u}, \vec{v})$.
Affixe de M : $z_M = x + iy$
Image de z : le point $M$ d’affixe $z$.
On note : $M(z)$ ou $M \leftrightarrow z$.

  • La partie réelle $x = \text{Re}(z)$ est l’abscisse de $M$.
  • La partie imaginaire $y = \text{Im}(z)$ est l’ordonnée de $M$.
  • L’origine $O$ a pour affixe $0$, le point $A(1;0)$ a pour affixe $1$, le point $B(0;1)$ a pour affixe $i$.
  • L’axe des abscisses est appelé axe réel, l’axe des ordonnées axe imaginaire pur.

2. Interprétation géométrique des opérations

L’addition et la multiplication par un réel se traduisent vectoriellement.

  • Si $z_1$ et $z_2$ sont les affixes de $M_1$ et $M_2$, alors l’affixe de la somme vectorielle $\overrightarrow{OM_1} + \overrightarrow{OM_2}$ est $z_1 + z_2$.
  • Pour tout $\lambda \in \mathbb{R}$, l’affixe du vecteur $\lambda \overrightarrow{OM}$ est $\lambda z$.
  • L’affixe du milieu $I$ de $[M_1M_2]$ est $z_I = \dfrac{z_1 + z_2}{2}$.
  • L’affixe du vecteur $\overrightarrow{M_1M_2}$ est $z_{M_2} - z_{M_1}$.
💡 Astuce visuelle : Dans le plan, chaque complexe est un « point étiqueté » ; l’addition correspond à la règle du parallélogramme.

3. Module et argument (rappel fondamental)

Le module $|z| = \sqrt{x^2 + y^2}$ est la distance $OM$ (norme du vecteur $\overrightarrow{OM}$). L’argument $\theta = \arg(z)$ (mod $2\pi$) est l’angle orienté $(\vec{u}, \overrightarrow{OM})$. Ces deux notions lient la géométrie et l’algèbre complexe.

Écriture trigonométrique : $z = |z|(\cos\theta + i\sin\theta)$.

📐 Représentation graphique & transformations

🖼️ Plan complexe – Affixe et image

✨ Illustration du plan complexe : tout nombre complexe $z=x+iy$ correspond au point $M(x;y)$. L’affixe est l’étiquette complexe du point.

L’image géométrique permet de « visualiser » les nombres complexes et de traduire des propriétés (alignement, distance, angle). Un outil indispensable pour les transformations (rotation, translation, similitude).

💡 Exemples de compréhension

Exemple 1 : Déterminer affixe et image

On donne $z = 3 - 2i$. Déterminer l’affixe, l’image $M$, puis la distance $OM$ et l’affixe du symétrique par rapport à l’axe réel.
Résolution :
- $z = 3 - 2i$ : partie réelle $3$, partie imaginaire $-2$. Donc $M(3; -2)$.
- Distance $OM = |z| = \sqrt{3^2 + (-2)^2} = \sqrt{9+4} = \sqrt{13}$.
- Le symétrique par rapport à l’axe réel (axe des abscisses) a pour affixe $\overline{z} = 3 + 2i$ (conjugué). Image $M'(3;2)$.
💬 Astuce : Le conjugué d’un complexe correspond à la réflexion par rapport à l’axe réel. Cela sera utile dans les problèmes de géométrie.

Exemple 2 : Utilisation des affixes pour les vecteurs

Soient $A(2+ i)$, $B(-1+3i)$. Calculer l’affixe du vecteur $\overrightarrow{AB}$, puis l’affixe du milieu $I$ de $[AB]$.
Calcul :
- Affixe de $\overrightarrow{AB}$ : $z_B - z_A = (-1+3i) - (2+i) = -3 + 2i$.
- Affixe du milieu : $z_I = \dfrac{z_A + z_B}{2} = \dfrac{(2+i)+(-1+3i)}{2} = \dfrac{1+4i}{2} = 0,5 + 2i$.
Donc $I\left(0,5 ; 2\right)$.
📐 Ce principe est fondamental pour démontrer qu’un point est le milieu ou pour caractériser un parallélogramme : $ABCD$ parallélogramme $\iff z_A + z_C = z_B + z_D$.

Exemple 3 : Lien avec la rotation (transformation géométrique)

Soit $M$ d’affixe $z = 1 + i\sqrt{3}$. On applique une rotation de centre $O$ d’angle $+\dfrac{\pi}{3}$. Déterminer l’affixe $z'$ de l’image $M'$.
Rappel : Rotation de centre $O$ d’angle $\theta$ : $z' = e^{i\theta} \cdot z$.
- Écriture exponentielle de $z$ : $|z| = \sqrt{1^2 + (\sqrt{3})^2}=2$, $\arg(z) = \frac{\pi}{3}$ car $\cos\frac{\pi}{3}=1/2$ et $\sin\frac{\pi}{3}=\sqrt{3}/2$.
- Donc $z = 2e^{i\pi/3}$.
- Après rotation : $z' = e^{i\pi/3} \times 2e^{i\pi/3} = 2e^{i2\pi/3} = 2\left(\cos\frac{2\pi}{3}+i\sin\frac{2\pi}{3}\right) = 2\left(-\frac12 + i\frac{\sqrt{3}}{2}\right) = -1 + i\sqrt{3}$.
L’image $M'$ a pour affixe $-1 + i\sqrt{3}$.
🧭 Une rotation d’angle $\theta$ se traduit par une multiplication par le nombre complexe $e^{i\theta}$ : un atout puissant de l’approche géométrique.

🔄 Correspondances complexes → transformations

Tableau des transformations fondamentales

  • Translation de vecteur $\vec{w}$ d’affixe $b$ : $z' = z + b$.
  • Rotation de centre $O$ d’angle $\theta$ : $z' = e^{i\theta}z$.
  • Homothétie de centre $O$ et de rapport $k \in \mathbb{R}^*$ : $z' = k\,z$.
  • Rotation + homothétie (similitude directe) de centre $O$ : $z' = \lambda z$ avec $\lambda = k e^{i\theta}$.
  • Symétrie axiale par rapport à l’axe réel : $z' = \overline{z}$.
  • Symétrie centrale par rapport à $O$ (demi-tour) : $z' = -z$.

Dans les problèmes d’examen (sessions 2022-2025), on utilise souvent ces correspondances pour caractériser des figures ou résoudre des configurations géométriques.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre affixe d’un point et coordonnées cartésiennes sans tenir compte de l’unité imaginaire. Rappel : $z = x+iy$ → $M(x;y)$.
  • Attention : L’affixe d’un vecteur $\overrightarrow{AB}$ est $z_B - z_A$ et non $z_A - z_B$ (ordre important pour le sens).
  • Piège : Une rotation de centre autre que l’origine ne s’écrit pas simplement $z' = e^{i\theta}z$ mais $z' - \omega = e^{i\theta}(z-\omega)$ où $\omega$ est l’affixe du centre.
  • Confusion récurrente : Le module $|z|$ est une distance (positive), l’argument est défini modulo $2\pi$ ; ne pas écrire $\arg(z_1+z_2)=\arg(z_1)+\arg(z_2)$ (faux en général).

🏆 Application type examen (Bac C)

Problème inspiré – Session 2024 C (similitude)

Le plan complexe est muni d’un repère orthonormé. On considère les points $A(2)$, $B(2i)$ et $C$ d’affixe $z_C = 2+2i$. Déterminer la nature du triangle $ABC$ puis l’affixe du centre du cercle circonscrit.
Démarche :
1. Calcul des affixes : $A(2+0i)$, $B(0+2i)$, $C(2+2i)$.
2. $AB = |z_B - z_A| = |-2+2i| = \sqrt{4+4} = \sqrt{8} = 2\sqrt{2}$ ; $AC = |z_C - z_A| = |0+2i| = 2$ ; $BC = |z_C - z_B| = |2+0i| = 2$.
3. $AC = BC = 2$, le triangle est isocèle rectangle en $C$ ? Vérifions $AB^2 = 8$, $AC^2+BC^2 = 4+4=8$ → triangle rectangle en $C$.
4. Le centre du cercle circonscrit d’un triangle rectangle est le milieu de l’hypoténuse $[AB]$.
Affixe du milieu de $[AB]$ : $z_I = \dfrac{z_A+z_B}{2} = \dfrac{2 + 2i}{2} = 1 + i$.
Conclusion : l’image $I(1;1)$ est le centre cherché.
✅ Réflexe examen : Dès qu’on lit « plan complexe », penser à utiliser les affixes pour calculer distances ($|z_2-z_1|$) et interpréter les angles via arguments.

🔗 Prolongement : Similitudes et complexes

Cette leçon est la base du chapitre 7 (Similitudes directes). L’écriture $z' = a z + b$ avec $a\in\mathbb{C}^*$ représente une similitude directe (composition d’une rotation, homothétie et translation). Maîtriser l’image et l’affixe vous permettra de résoudre les exercices de type « déterminer le centre d’une similitude » ou « reconnaître une transformation à partir d’une équation complexe ».
