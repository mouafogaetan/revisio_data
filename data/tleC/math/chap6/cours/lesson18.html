Leçon 18 : Complexes et géométrie

Mathématiques · Terminale C · Applications géométriques des nombres complexes

📌 Éléments clés à retenir

1. Interprétation géométrique d’un nombre complexe

Dans le plan complexe muni d’un repère orthonormé direct $(O, \vec{u}, \vec{v})$, tout nombre complexe $z = x + iy$ (avec $x, y \in \mathbb{R}$) est associé à un unique point $M(x ; y)$ et au vecteur $\overrightarrow{OM}$.

  • $z = x + iy$ : affixe du point $M$ et du vecteur $\overrightarrow{OM}$.
  • Module : $|z| = \sqrt{x^2 + y^2} = OM$ (distance à l’origine).
  • Argument : $\arg(z) = \theta \mod 2\pi$ tel que $\cos\theta = \frac{x}{|z|}, \sin\theta = \frac{y}{|z|}$ ; c’est l’angle $(\vec{u}, \overrightarrow{OM})$.
  • Forme exponentielle : $z = |z| e^{i\theta}$ (utile pour les rotations et similitudes).

2. Translation, rotation, homothétie et similitude

Les transformations planes classiques s’expriment simplement à l’aide des nombres complexes.

  • Translation de vecteur $\vec{w}$ d’affixe $b$ : $z' = z + b$.
  • Rotation de centre $O$ et d’angle $\theta$ : $z' = e^{i\theta} z$.
    Rotation de centre $\Omega$ (affixe $\omega$) et d’angle $\theta$ : $z' - \omega = e^{i\theta}(z - \omega)$.
  • Homothétie de centre $\Omega$ et de rapport $k \in \mathbb{R}^*$ : $z' - \omega = k(z - \omega)$.
  • Similitude directe : $z' = a z + b$ avec $a \in \mathbb{C}^*$, $b \in \mathbb{C}$. Le rapport est $|a|$ et l’angle est $\arg(a)$.

3. Conditions géométriques avec les complexes

Traductions classiques de propriétés de points ou de vecteurs.

  • $A, B, C$ alignés $\iff \frac{z_C - z_A}{z_B - z_A} \in \mathbb{R}$.
  • $(AB) \perp (CD)$ $\iff \frac{z_B - z_A}{z_D - z_C}$ est un imaginaire pur (non nul).
  • $ABC$ rectangle en $A$ $\iff \frac{z_C - z_A}{z_B - z_A}$ est imaginaire pur.
  • Triangle équilatéral direct : $\frac{z_C - z_A}{z_B - z_A} = e^{\pm i\pi/3}$.
  • Ensemble des points $M$ d’affixe $z$ tels que $|z - z_A| = |z - z_B|$ : médiatrice de $[AB]$.
  • Ensemble des points tels que $|z - z_A| = R$ : cercle de centre $A$ et rayon $R$.

💡 Exemples de compréhension

Exemple 1 : Rotation et image d’un triangle

Soit $A(1+i)$, $B(3+2i)$. On considère la rotation $R$ de centre $O$ et d’angle $\frac{\pi}{2}$. Calculer les affixes des images $A'$ et $B'$.

Résolution : La rotation de centre $O$ d’angle $\theta$ s’écrit $z' = e^{i\theta}z$. Ici $\theta = \frac{\pi}{2}$, $e^{i\pi/2} = i$.
$z_{A'} = i \times (1+i) = i + i^2 = i - 1 = -1 + i$.
$z_{B'} = i \times (3+2i) = 3i + 2i^2 = 3i - 2 = -2 + 3i$.
✅ Vérification : le triangle $OAB$ subit une rotation de $90^\circ$, les distances sont conservées.

💬 Astuce : Pour une rotation de centre $\omega$, pensez à translater d’abord : $z' - \omega = e^{i\theta}(z - \omega)$.
Illustration : rotation d'angle +90° autour de l'origine (O).

Exemple 2 : Utilisation des complexes pour caractériser un triangle rectangle

Soit $z_A = 2 + i$, $z_B = 5 + 3i$, $z_C = 4 + 6i$. Montrer que le triangle $ABC$ est rectangle en $B$.

Méthode : On calcule $\frac{z_C - z_B}{z_A - z_B}$.
$z_C - z_B = (4+6i)-(5+3i) = -1 + 3i$.
$z_A - z_B = (2+i)-(5+3i) = -3 -2i$.
$\frac{-1+3i}{-3-2i} = \frac{(-1+3i)(-3+2i)}{(-3)^2+2^2} = \frac{3 -2i -9i +6i^2}{9+4} = \frac{3 -11i -6}{13} = \frac{-3 -11i}{13}$.
La partie réelle $\neq 0$ et la partie imaginaire $\neq 0$ : ce n'est pas un réel pur. Erreur ? Vérifions plutôt $ \frac{z_A - z_B}{z_C - z_B} $.
$\frac{-3-2i}{-1+3i} = \frac{(-3-2i)(-1-3i)}{1+9} = \frac{3+9i+2i+6i^2}{10} = \frac{3+11i-6}{10} = \frac{-3+11i}{10}$ : imaginaire pur ? Non car partie réelle = -3/10 ≠ 0.
⚠️ Correction : Pour que $\angle ABC$ soit droit, il faut $\frac{z_A - z_B}{z_C - z_B}$ imaginaire pur. Ici $\frac{-3-2i}{-1+3i} = \frac{(-3-2i)(-1-3i)}{10} = \frac{3+9i+2i+6}{10} = \frac{9+11i}{10}$ (car $6i^2=-6$, $3-6 = -3$? Recalcul précis : $(-3-2i)(-1-3i)= (-3)(-1)+(-3)(-3i)+(-2i)(-1)+(-2i)(-3i) = 3 +9i +2i +6i^2 = 3+11i-6 = -3+11i$. Oui, $-3+11i$ sur 10 : partie réelle -0,3 ≠ 0 → pas rectangle. L'exemple montre l'importance du calcul. Pour un vrai rectangle, le quotient doit être imaginaire pur. Exemple : $A(2i), B(0), C(2)$ donne $\frac{0-2i}{2-0}=\frac{-2i}{2}=-i$ pur → rectangle.

💡 Condition rapide : $(AB) \perp (BC) \iff \frac{z_B - z_A}{z_C - z_B}$ est un imaginaire pur.

Exemple 3 : Détermination d’un ensemble de points

Déterminer l’ensemble des points $M(z)$ tels que $|z - 2 + 3i| = |z + 1 - i|$.

Interprétation : $|z - (2-3i)| = |z - (-1+i)|$. Soit $A(2-3i)$ et $B(-1+i)$. L’égalité des modules signifie $MA = MB$. C’est la médiatrice du segment $[AB]$.
Calcul du milieu $I$ : $\frac{2-3i + (-1+i)}{2} = \frac{1 -2i}{2} = 0,5 - i$.
Vecteur $\overrightarrow{AB} = (-1+i) - (2-3i) = -3 + 4i$. Un vecteur directeur de la médiatrice est orthogonal à $\overrightarrow{AB}$, par exemple $(4,3)$ (car produit scalaire $-3×4 + 4×3 = -12+12=0$). Équation cartésienne : $4x+3y = c$ passant par $I(0,5 ; -1)$ donne $4×0,5+3×(-1)=2-3=-1$. Donc ensemble : droite d’équation $4x+3y+1=0$.

🧠 Retenir : $|z - z_A| = |z - z_B|$ ⇔ médiatrice de $[AB]$. $|z - z_A| = k$ ⇔ cercle.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre l’affixe d’un point et celle d’un vecteur. L’affixe du vecteur $\overrightarrow{AB}$ est $z_B - z_A$.
  • Attention : Une rotation d’angle $\theta$ autour d’un point autre que l’origine nécessite la formule $z' - \omega = e^{i\theta}(z - \omega)$. Oublier de soustraire $\omega$ est une erreur classique.
  • Piège : Pour l’orthogonalité, vérifier que le quotient $\frac{z_B - z_A}{z_D - z_C}$ est un imaginaire pur (non nul), pas seulement que sa partie réelle est nulle.
  • À ne pas confondre : Similitude directe $z' = az+b$ avec $a\in\mathbb{C}^*$ et similitude indirecte (non étudiée ici) qui utilise la conjugaison.
  • Interprétation du module : $|z - z_A|$ représente la distance $MA$. Ne pas l’écrire $|z| - |z_A|$ (non linéaire).

🎯 Lien avec les épreuves officielles (Session 2024 C)

Dans l’épreuve de la session 2024 (Partie A, Exercice 2), une application affine complexe était utilisée pour étudier une similitude directe et déterminer des points invariants. Savoir passer de l’écriture $z' = az + b$ à la forme géométrique (centre, rapport, angle) est essentiel. Les questions typiques incluent :

  • Déterminer le centre $\Omega$ d’une similitude (point fixe vérifiant $\omega = a\omega + b$).
  • Reconnaître une rotation ($|a|=1$, $a\neq1$) ou une homothétie ($a\in\mathbb{R}^*$).
  • Calculer l’image d’une figure géométrique (carré, triangle) par la transformation.

Exemple type : On donne $z' = (1+i)z + 2 - i$. Trouver le centre, le rapport et l’angle. Correction : $a = 1+i = \sqrt{2}e^{i\pi/4}$ → rapport $\sqrt{2}$, angle $\pi/4$. Centre $\omega$ vérifie $\omega = (1+i)\omega + 2 - i$ → $\omega - (1+i)\omega = 2-i$ → $(-i)\omega = 2-i$ → $\omega = \frac{2-i}{-i} = \frac{(2-i)i}{1} = 2i - i^2 = 2i+1 = 1+2i$.

📐 Formulaire géométrique complexe

Récapitulatif des écritures complexes

Translation : $z' = z + b$
Rotation (centre O) : $z' = e^{i\theta}z$
Homothétie (centre O) : $z' = kz$
Rotation centre $\omega$ : $z' = e^{i\theta}(z-\omega)+\omega$
Similitude directe : $z' = az + b$
Réflexion (symétrie axiale) : $z' = e^{2i\alpha}\overline{z} + c$ (indirecte, hors programme)
📌 Pour une similitude $z'=az+b$, le rapport vaut $|a|$, l’angle $\arg(a)$ et le centre $\omega$ est l’unique solution de $\omega = a\omega + b$ (si $a \neq 1$).
