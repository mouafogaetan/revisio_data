Leçon 23 : Fonctions exponentielles de base $a$ et fonctions puissances

Mathématiques · Terminale C · Révision Bac 2025-2026

📌 Éléments clés à retenir

1. Fonction exponentielle de base $a$ (avec $a > 0$, $a \neq 1$)

La fonction exponentielle de base $a$ est définie pour tout réel $x$ par : $$ \boxed{a^x = e^{x \ln a}} $$ Elle généralise la fonction exponentielle népérienne ($a=e$). C'est une fonction continue, dérivable sur $\mathbb{R}$.

  • Domaine : $\mathbb{R}$.
  • Dérivée : $(a^x)' = a^x \cdot \ln a$.
  • Cas particuliers : Si $a > 1$, $a^x$ est strictement croissante. Si $0 < a < 1$, $a^x$ est strictement décroissante.
  • Limites :
    • Si $a > 1$ : $\displaystyle \lim_{x \to +\infty} a^x = +\infty$ ; $\displaystyle \lim_{x \to -\infty} a^x = 0$.
    • Si $0 < a < 1$ : $\displaystyle \lim_{x \to +\infty} a^x = 0$ ; $\displaystyle \lim_{x \to -\infty} a^x = +\infty$.
  • Propriétés algébriques : $a^{x+y} = a^x a^y$, $a^{x-y} = \dfrac{a^x}{a^y}$, $(a^x)^y = a^{xy}$.

2. Fonction puissance (exposant réel $\alpha$)

Pour $\alpha \in \mathbb{R}$, la fonction puissance est définie sur $\mathbb{R}_+^*$ (et parfois sur $\mathbb{R}$ si $\alpha$ est entier) par : $$ \boxed{x^\alpha = e^{\alpha \ln x}} $$ Elle étend la notion de puissance à un exposant réel quelconque.

  • Domaine : $\mathbb{R}_+^*$ (sauf cas particuliers comme $\alpha$ entier naturel où l'on peut étendre à $\mathbb{R}$).
  • Dérivée : $(x^\alpha)' = \alpha x^{\alpha-1}$ (pour $x > 0$).
  • Monotonie : Si $\alpha > 0$, $x \mapsto x^\alpha$ est strictement croissante. Si $\alpha < 0$, elle est strictement décroissante.
  • Limites remarquables : $\displaystyle \lim_{x \to 0^+} x^\alpha = 0$ si $\alpha > 0$ ; $\displaystyle \lim_{x \to 0^+} x^\alpha = +\infty$ si $\alpha < 0$ ; $\displaystyle \lim_{x \to +\infty} x^\alpha = +\infty$ si $\alpha > 0$ ; $\displaystyle \lim_{x \to +\infty} x^\alpha = 0$ si $\alpha < 0$.

3. Relation entre exponentielle de base $a$ et puissance

Pour $a > 0$ et $b$ réel, on a l'identité fondamentale : $a^b = e^{b \ln a}$. Ce pont permet d'utiliser toutes les propriétés de l'exponentielle népérienne. De plus, pour tout réel $x$ : $a^x = e^{x \ln a}$.

💡 Exemples de compréhension

Exemple 1 : Dérivée d'une exponentielle de base a

Soit $f(x) = 3^x \cdot x^2$. Calculer $f'(x)$.

Résolution : On utilise $(3^x)' = 3^x \ln 3$ et la formule de dérivation d'un produit.
$f'(x) = (3^x)' \cdot x^2 + 3^x \cdot (x^2)' = 3^x \ln 3 \cdot x^2 + 3^x \cdot 2x = 3^x \big( x^2 \ln 3 + 2x \big)$.

💬 Astuce : Ne jamais oublier le facteur $\ln a$ en dérivant $a^x$. Pour $a^x$, la dérivée est $a^x \ln a$, pas $a^x$.

Exemple 2 : Simplification d'une expression avec $a^x$

Simplifier l'expression $A = \dfrac{2^{3x} \cdot 4^{x+1}}{8^{x-1}}$ pour $x \in \mathbb{R}$.

Calcul : On écrit tout en base 2 : $4 = 2^2$, $8 = 2^3$.
$A = \dfrac{2^{3x} \cdot (2^2)^{x+1}}{(2^3)^{x-1}} = \dfrac{2^{3x} \cdot 2^{2x+2}}{2^{3x-3}} = 2^{(3x + 2x+2) - (3x-3)} = 2^{(5x+2 -3x +3)} = 2^{2x+5}$.
Donc $A = 2^{2x+5}$.

💡 Toujours ramener à la même base (ici 2) pour simplifier les produits/quotients d'exponentielles.

Exemple 3 : Étude d'une fonction puissance $x^\alpha$

Étudier les variations de $g(x) = x^{\sqrt{2}}$ sur $]0, +\infty[$.

Méthode : $g(x) = e^{\sqrt{2} \ln x}$. $g'(x) = \sqrt{2} \cdot x^{\sqrt{2}-1}$ car $\sqrt{2} > 0$.
Pour tout $x>0$, $g'(x) > 0$ donc $g$ est strictement croissante sur $\mathbb{R}_+^*$. De plus $\lim_{x \to 0^+} g(x) = 0$ et $\lim_{x \to +\infty} g(x) = +\infty$.

💬 Pour $\alpha$ réel quelconque, la formule de dérivation $(x^\alpha)' = \alpha x^{\alpha-1}$ reste valable sur $\mathbb{R}_+^*$.

📐 Formulaires à maîtriser

Synthèse des dérivées et propriétés

  • $(a^x)' = a^x \ln a$ (avec $a > 0$)
  • $(x^\alpha)' = \alpha x^{\alpha-1}$ (pour $x > 0$, $\alpha \in \mathbb{R}$)
  • $a^x = e^{x \ln a}$ et $x^\alpha = e^{\alpha \ln x}$
  • Comparaison des croissances : quand $x \to +\infty$, toute exponentielle $a^x$ (avec $a>1$) l'emporte sur toute puissance $x^\alpha$.

🎯 Application : Résolution d'équations

Résoudre $2^{x+1} = 5^{1-x}$

Trouver $x$ réel vérifiant l'égalité.

Méthode : On prend le logarithme népérien des deux membres (strictement positifs).
$\ln(2^{x+1}) = \ln(5^{1-x}) \iff (x+1)\ln 2 = (1-x)\ln 5$.
$x\ln 2 + \ln 2 = \ln 5 - x\ln 5 \iff x(\ln 2 + \ln 5) = \ln 5 - \ln 2 \iff x\ln(10) = \ln\left(\frac{5}{2}\right)$.
Donc $\boxed{x = \dfrac{\ln(5/2)}{\ln 10}}$.

💡 Penser à utiliser $\ln(a^x) = x\ln a$ pour résoudre les équations exponentielles.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre $a^x$ avec $x^a$. Par exemple, $2^3 = 8$ alors que $3^2 = 9$. La variable et la base sont inversées.
  • Attention : Dériver $a^x$ comme $x \mapsto x a^{x-1}$ (c'est faux !). La bonne formule fait intervenir $\ln a$.
  • Piège : Pour $x^\alpha$ avec $\alpha$ réel, le domaine n'est pas toujours $\mathbb{R}$. Si $\alpha$ n'est pas entier, $x^\alpha$ n'est définie que pour $x>0$.
  • Ne pas oublier : $a^0 = 1$ pour tout $a \neq 0$, et $a^1 = a$.

🏆 Focus Bac (Sessions 2022-2025)

Les fonctions exponentielles de base $a$ et les fonctions puissances apparaissent souvent dans les problèmes d'étude de fonctions (Partie A) ou dans des modélisations (Partie B). Il est essentiel de maîtriser le lien $a^x = e^{x\ln a}$ pour se ramener aux fonctions usuelles. Les questions types : calcul de dérivée, limite, recherche de primitives, résolution d'équations, comparaison de croissances.

  • Exemple typique (session 2023) : Étudier $f(x) = 2^x - x^3$ sur $\mathbb{R}$.
  • Savoir utiliser la formule $(a^x)'$ pour les problèmes d'optimisation.
