🔢 Exercice 1 | Nombres complexes (4 pts)
Pour chaque question, une seule réponse est correcte. (1 pt par bonne réponse)

1. Le conjugué du nombre complexe z = 3 - 4i est :

  • A. -3 + 4i
  • B. 3 + 4i
  • C. -3 - 4i
  • D. 4 - 3i

    • 2. Le module du nombre complexe z = -1 + i est :

  • A. 0
  • B. 1
  • C. √2
  • D. 2

    • 3. L'équation z² = -9 dans ℂ admet pour solutions :

  • A. 3 et -3
  • B. 3i et -3i
  • C. 9i et -9i
  • D. 3i seulement

    • 4. La partie réelle de z = (1 + i)² est :

  • A. 0
  • B. 1
  • C. 2
  • D. -2

    • ✅ Réponses 1. B (3+4i) — Le conjugué change le signe de la partie imaginaire.

    ✅ Réponse 2 C (√2) — |z| = √[(-1)² + 1²] = √2.

    ✅ Réponse 3 B (3i et -3i) — z² = -9 ⇔ z = ±3i.

    ✅ Réponse 4 A (0) — (1+i)² = 1 + 2i + i² = 1 + 2i -1 = 2i, partie réelle nulle.

    📐 Exercice 2 | Espaces vectoriels (5 pts)
    On considère l'application linéaire f : ℝ³ → ℝ² définie par :
    f(x ; y ; z) = (2x - y + z ; x + y - 3z).

    1. (1,5 pt) Déterminer la matrice A de f par rapport aux bases canoniques.

    2. (1,5 pt) Déterminer le noyau Ker(f). Est-ce que le vecteur u = (1 ; 1 ; 1) appartient à Ker(f) ?

    3. (2 pts) Déterminer l'image Im(f). Donner une base de Im(f).

    ✅ 1. Matrice A = [ [2, -1, 1] ; [1, 1, -3] ] (2 lignes, 3 colonnes).

    ✅ 2. Ker(f) = { (x;y;z) ∈ ℝ³ | 2x - y + z = 0 et x + y - 3z = 0 }.
    En résolvant : de la 2e équation x = 3z - y. En substituant : 2(3z-y) - y + z = 6z - 2y - y + z = 7z - 3y = 0 ⇒ y = (7/3)z. Puis x = 3z - (7/3)z = (2/3)z.
    Ker(f) = { (2z/3 ; 7z/3 ; z) / z∈ℝ } = Vect{ (2;7;3) }.
    u = (1;1;1) ⇒ 2×1 -1 +1 = 2 ≠ 0 donc u ∉ Ker(f).

    ✅ 3. Im(f) est un sous-espace vectoriel de ℝ². f est surjective car les deux vecteurs colonnes (2;1) et (-1;1) sont linéairement indépendants. Donc Im(f) = ℝ². Une base : {(1;0), (0;1)}.

    📈 Exercice 3 | Suites numériques (6 pts)
    Soit la suite (Un) définie par U0 = 1 et pour tout n ∈ ℕ, Un+1 = (1/2)Un + 2.

    1. (1 pt) Calculer U1, U2 et U3.

    2. (2 pts) Démontrer par récurrence que pour tout n ∈ ℕ, Un ≤ 4.

    3. (2 pts) On pose Vn = Un - 4. Démontrer que (Vn) est une suite géométrique. Préciser sa raison.

    4. (1 pt) En déduire l'expression de Un en fonction de n, puis la limite de (Un).

    ✅ 1. U1 = 1/2×1 + 2 = 2,5 ; U2 = 1/2×2,5 + 2 = 3,25 ; U3 = 1/2×3,25 + 2 = 3,625.

    ✅ 2. Initialisation : U0 = 1 ≤ 4. Vrai.
    Hérédité : Supposons Un ≤ 4. Alors Un+1 = (1/2)Un + 2 ≤ (1/2)×4 + 2 = 2 + 2 = 4. La propriété est héréditaire.
    Conclusion : ∀n, Un ≤ 4.

    ✅ 3. Vn+1 = Un+1 - 4 = (1/2)Un + 2 - 4 = (1/2)Un - 2 = (1/2)(Un - 4) = (1/2)Vn.
    Donc (Vn) est géométrique de raison q = 1/2 et V0 = U0 - 4 = -3.

    ✅ 4. Vn = V0 × (1/2)n = -3 × (1/2)n. Donc Un = Vn + 4 = 4 - 3×(1/2)n.
    Comme (1/2)n → 0, on a lim Un = 4.

    💡 PARTIE B : ÉVALUATION DES COMPÉTENCES (5 points)
    🏥 Situation-problème | Modélisation d'une épidémie

    Dans une ville camerounaise, le nombre de personnes infectées par un virus évolue selon la relation suivante : chaque jour, le nombre de nouveaux cas est égal à la moitié du nombre de cas de la veille, auquel s'ajoutent 50 nouveaux cas importés.

    On note In le nombre d'infectés au jour n (avec I0 = 100).

    1. (1 pt) Traduire l'énoncé par une relation de récurrence entre In+1 et In.

    2. (1,5 pts) La modélisation mène à In+1 = 0,5 In + 50. Déterminer la limite éventuelle de la suite (In).

    3. (2,5 pts) Le service sanitaire souhaite savoir si le nombre d'infectés dépassera 150. Pour cela :
      a) On pose Jn = In - 100. Montrer que (Jn) est géométrique.
      b) Exprimer In en fonction de n.
      c) Déterminer si In dépasse 150 et au bout de combien de jours cela se produit.

    Image illustrative :

    Source : Organisation mondiale de la santé (illustration)

    ✅ 1. In+1 = (1/2) In + 50.

    ✅ 2. Si la suite converge vers L, alors L = 0,5L + 50 ⇒ 0,5L = 50 ⇒ L = 100. La limite est 100.

    ✅ 3a. Jn+1 = In+1 - 100 = (0,5In + 50) - 100 = 0,5In - 50 = 0,5(In - 100) = 0,5 Jn.
    Donc (Jn) est géométrique de raison 0,5.

    ✅ 3b. J0 = I0 - 100 = 0. Donc Jn = 0 × (0,5)n = 0.
    Ainsi In = 100 pour tout n. La suite est constante !

    ✅ 3c. Comme In = 100 pour tout n, le nombre d'infectés ne dépassera jamais 150. L'épidémie reste stable.

    📌 Remarque pédagogique : Le choix de I0 = 100 donne un point fixe exact. Si I0 était différent, on observerait une convergence vers 100.
