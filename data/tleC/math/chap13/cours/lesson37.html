📐 Leçon 37 : Équations différentielles du premier ordre

Mathématiques · Terminale C (Scientifique) · Résolution & modélisation

📌 Éléments clés à retenir

1. Définition & forme générale

Une équation différentielle linéaire du premier ordre est une relation entre une fonction $y = f(x)$, sa dérivée $y'$ et une fonction continue $a(x)$, $b(x)$. Sa forme générale (avec coefficients constants ou variables) s'écrit :

  • Forme réduite : $$ y' + a(x)\,y = b(x) $$
  • Si $b(x) = 0$ : équation homogène (sans second membre).
  • Si $b(x) \neq 0$ : équation complète (avec second membre).
  • En Terminale C (programme officiel), on se concentre sur les cas où $a$ et $b$ sont des constantes réelles : $$ y' + a\,y = b \quad (a,b \in \mathbb{R}) $$

🔬 Les équations différentielles du 1er ordre modélisent de nombreux phénomènes : charge d’un condensateur, refroidissement, dynamique des populations.

2. Solution de l’équation homogène $y' + a y = 0$

La solution générale de l’équation sans second membre s’obtient par intégration directe ou séparation des variables.

  • On écrit $\frac{dy}{dx} = -a y$ → $\frac{dy}{y} = -a\,dx$.
  • Après intégration : $\ln|y| = -a x + K$.
  • D’où la solution générale : $$ y_h(x) = C\,e^{-a x} \quad (C \in \mathbb{R}) $$
  • À retenir : la solution homogène est une exponentielle dont le coefficient dans l’exposant est l’opposé du coefficient $a$.

3. Solution particulière (second membre constant)

Pour $y' + a y = b$ avec $a \neq 0$, on cherche une solution constante $y_p = k$ (car le second membre est constant).

  • Si $y_p = k$ alors $y_p' = 0$. En substituant : $0 + a k = b \;\Rightarrow\; k = \dfrac{b}{a}$.
  • Ainsi solution particulière : $$ y_p = \frac{b}{a} $$
  • Cas particulier $a = 0$ : l’équation devient $y' = b$ → $y_p = b x$ (solution particulière affine).

4. Solution générale de l’équation complète

Principe fondamental : La solution générale d’une EDL du 1er ordre est la somme de la solution homogène et d’une solution particulière.

$$ y(x) = y_h(x) + y_p(x) = C\,e^{-a x} + \frac{b}{a} \quad (a \neq 0) $$

La constante $C$ est déterminée par une condition initiale du type $y(x_0) = y_0$ (unicité de la solution).

💡 Exemples de compréhension (examen type)

📘 Exemple 1 : Résolution complète + condition initiale

Résoudre l’équation différentielle : $y' - 3y = 6$ avec la condition $y(0) = 5$.

🔎 Étapes détaillées :
1️⃣ Mettre sous la forme $y' + a y = b$ : ici $y' - 3y = 6$ ⇒ $a = -3$, $b = 6$. (Attention : $y' + (-3)y = 6$).
2️⃣ Solution homogène : $y_h = C e^{-a x} = C e^{-(-3)x} = C e^{3x}$.
3️⃣ Solution particulière constante : $y_p = \frac{b}{a} = \frac{6}{-3} = -2$.
4️⃣ Solution générale : $y(x) = C e^{3x} - 2$.
5️⃣ Condition $y(0)=5$ : $5 = C e^{0} - 2 \Rightarrow C = 7$.
6️⃣ Solution unique : $\boxed{y(x) = 7e^{3x} - 2}$.

💡 Astuce : Vérifiez toujours le signe de $a$ dans la forme standard $y'+a y = b$. Ici $a = -3$, donc l’exponentielle est $e^{-a x}=e^{3x}$.

📗 Exemple 2 : Équation avec second membre constant (a ≠ 0)

Trouver la solution générale de : $2y' + 4y = 8$.

Résolution rapide :
• Diviser par 2 pour obtenir la forme normale : $y' + 2y = 4$.
• Ici $a = 2$, $b = 4$. Solution homogène : $y_h = C e^{-2x}$.
• Solution particulière : $y_p = \frac{b}{a} = \frac{4}{2} = 2$.
• Solution générale : $y(x) = C e^{-2x} + 2$.
Interprétation : quand $x \to +\infty$, $y(x) \to 2$ (régime permanent).

💬 Pensez au comportement asymptotique : la solution tend toujours vers la solution particulière constante lorsque $a>0$.

📙 Exemple 3 : Cas particulier a = 0 (y' = b)

Résoudre $y' = 5$ avec $y(1) = 3$.

• L’équation est $y' = 5$ → solution générale $y(x) = 5x + C$.
• Condition $y(1)=3$ : $3 = 5(1) + C \Rightarrow C = -2$.
• Solution : $\boxed{y(x) = 5x - 2}$.
On retrouve bien le principe : solution particulière affine $y_p = b x$ (car $a=0$).

🎯 Lien avec l’épreuve (Session 2023 C – Partie B)

Dans l’exercice sur le ressort, on rencontre une équation différentielle du second ordre, mais les prérequis sur le premier ordre sont essentiels pour la mise en équation ou les changements de variable. Savoir résoudre $y'+ay = b$ permet de traiter des modèles de vitesse limite, charge RC, etc.

📌 Pour l’examen : la méthode de résolution (homogène + particulière) est systématique. Rédigez toujours les trois étapes : équation homogène → solution particulière → somme + condition initiale.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier le signe dans l’exposant de la solution homogène. Pour $y' + a y = 0$ → $y_h = C e^{-a x}$ (et non $e^{a x}$).
  • Attention : Confondre la solution particulière lorsque $a=0$. Dans ce cas, la particulière n’est pas constante mais affine ($y_p = b x$).
  • Piège : Ne pas diviser l’équation par le coefficient devant $y'$ avant d’identifier $a$ et $b$.
  • À surveiller : Si la condition initiale est donnée, ne pas oublier de calculer $C$ explicitement ; une solution générale sans constante n’est pas complète.
  • Modélisation : Ne pas confondre la variable (souvent $t$ pour le temps) avec $x$. Les mêmes règles s’appliquent : $y' + a y = b$.

🏁 Entraînement – sujet type (évaluation des compétences)

Problème : Modèle de refroidissement (loi de Newton)

La température $\theta(t)$ d’un corps (en °C) évolue selon l’équation : $\theta'(t) = -0,2\big(\theta(t) - 25\big)$, où $t$ est le temps en minutes. À $t=0$, $\theta(0) = 80$°C.
1) Mettre l’équation sous la forme $\theta' + a\theta = b$.
2) Résoudre l’équation différentielle.
3) Déterminer la température après 10 minutes.
4) Au bout de combien de temps la température sera-t-elle inférieure à 30°C ?

Correction :
1) $\theta' = -0,2\theta + 5$ ⇒ $\theta' + 0,2\,\theta = 5$. (donc $a = 0,2$ et $b = 5$).
2) Solution homogène : $\theta_h = C e^{-0,2 t}$. Particulière constante : $\theta_p = \frac{b}{a} = \frac{5}{0,2}=25$.
   Générale : $\theta(t) = C e^{-0,2 t} + 25$.
   Condition $\theta(0)=80$ : $80 = C + 25 \Rightarrow C = 55$.
   Donc $\boxed{\theta(t) = 55 e^{-0,2 t} + 25}$.
3) $\theta(10) = 55 e^{-2} + 25 \approx 55 \times 0,1353 + 25 \approx 7,44 + 25 = 32,44$°C.
4) Résoudre $55 e^{-0,2 t} + 25 < 30$ ⇒ $55 e^{-0,2 t} < 5$ ⇒ $e^{-0,2 t} < \frac{5}{55}=\frac{1}{11}$.
   $-0,2 t < \ln(1/11) = -\ln 11$ ⇒ $t > \frac{\ln 11}{0,2} \approx \frac{2,398}{0,2} \approx 11,99$ minutes.
   La température passe sous 30°C après environ 12 minutes.

🧠 Astuce examen : Lisez bien le contexte pour identifier la variable (ici $t$) et les valeurs initiales. La mise en forme $y'+ay=b$ est primordiale.
