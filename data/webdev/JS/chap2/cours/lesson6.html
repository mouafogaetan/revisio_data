🧮 Opérateurs arithmétiques

Ils permettent d'effectuer des calculs mathématiques classiques.

+   Addition

let somme = 10 + 5; // 15

-   Soustraction

let diff = 20 - 7; // 13

*   Multiplication

let produit = 6 * 4; // 24

/   Division

let quotient = 15 / 3; // 5

%   Modulo (reste)

let reste = 17 % 5; // 2

**   Exponentiation

let puissance = 2 ** 3; // 8 
// Exemple complet :
let a = 12, b = 4;
console.log(a + b);  // 16
console.log(a ** b); // 20736 (12⁴)
console.log(a % b);  // 0

Note : l'opérateur + sert aussi à concaténer des chaînes.

✍️ Opérateurs d'affectation

Ils servent à assigner une valeur à une variable, souvent en combinant une opération.

OpérateurExempleÉquivalent
=x = 5x = 5
+=x += 3x = x + 3
-=x -= 2x = x - 2
*=x *= 4x = x * 4
/=x /= 5x = x / 5
%=x %= 3x = x % 3
 
let score = 10;
score += 5;    // score devient 15
score *= 2;    // 30
score %= 7;    // 30 % 7 = 2
console.log(score); // 2
Astuce : les opérateurs composés rendent le code plus court et lisible.

⚖️ Comparaison : égalité et inégalité

Ils renvoient toujours un booléen : true ou false.

== égalité simple (valeur, conversion possible)
5 == "5" // true
=== égalité stricte (valeur + type)
5 === "5" // false
!= différent (valeur)
10 != "10" // false
!== différent strict
10 !== "10" // true
< , > inférieur, supérieur
<= , >= inférieur ou égal, supérieur ou égal
let age = 18;
console.log(age >= 18);   // true
console.log(age == "18"); // true (conversion)
console.log(age === "18");// false (string vs number)

⚠️ Toujours préférer === et !== pour éviter les conversions surprenantes.

🔗 Opérateurs logiques (ET, OU, NON)

Ils combinent des expressions booléennes – essentiels pour les conditions.

&& (ET logique) → vrai si les deux opérandes sont vrais.
let adulte = (age >= 18 && majeur === true);
|| (OU logique) → vrai si au moins un est vrai.
let acces = (role === "admin" || role === "moderateur");
! (NON logique) inverse la valeur booléenne.
let estConnecte = false; if(!estConnecte) { /* rediriger */ }
let a = true, b = false;
console.log(a && b);  // false
console.log(a || b);  // true
console.log(!a);      // false
💡 Les opérateurs logiques courts-circuits : && s'arrête si première expression fausse, || s'arrête si vraie.

🛠️ Applications et exemples concrets

Mélangeons arithmétique + affectation + comparaison et logique dans des mini-scénarios.

📌 Calcul du prix avec réduction (ternaire) :

let prix = 120;
let remise = 0.2; // 20%
let prixFinal = prix - (prix * remise);
let estCher = prixFinal > 100 ? "Cher" : "Abordable";
console.log(prixFinal, estCher);

// Vérification d'un nombre pair (modulo + comparaison)

let nombre = 7;
let estPair = (nombre % 2 === 0);
console.log(`${nombre} est pair ? ${estPair}`); // false

Combinaison logique : if(age >= 18 && aPermis) { "peut conduire" }

📐 Priorité des opérateurs

Comme en maths, certains opérateurs sont évalués avant d'autres. Utilisez les parenthèses pour clarifier.

let result = 10 + 5 * 2;   // 20 (multiplication d'abord)
let correct = (10 + 5) * 2; // 30  (parenthèses)
1. () parenthèses
2. ** exponentiation
3. * / % multiplication etc
4. + - addition/soustraction
5. < > <= >= comparaison
6. === !== égalité
7. && ET logique
8. || OU

📌 Bonne pratique : toujours utiliser des parenthèses pour rendre le code clair.
