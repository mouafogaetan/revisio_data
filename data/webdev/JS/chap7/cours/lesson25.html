⌨️ Événements Clavier

Utilisés pour capturer les frappes dans des champs ou sur la page.

keydown : touche enfoncée
keyup : touche relâchée
keypress (déprécié mais pédagogique)
⚡ Astuce : keydown permet d’annuler certaines touches (ex: empêcher la saisie de chiffres).

📝 Événements Formulaires

🔹 focus / blur

🔹 change / input

🔸 submit (prévention rechargement)

$('#demoForm').on('submit', function(e) {
    e.preventDefault();
    alert('Formulaire intercepté !');
});

✨ Astuces & bonnes pratiques

  • Préférez addEventListener (jQuery .on()) plutôt que les attributs HTML onclick.
  • Utilisez la délégation d’événements pour les éléments dynamiques : $('#parent').on('click', '.enfant', fn)
  • Pour les champs de formulaire, input est idéal pour la validation en temps réel.
  • Événements clavier : pensez à event.key pour identifier la touche (plus fiable que keyCode).
  • Propriétés utiles : preventDefault() pour annuler l’action par défaut, stopPropagation() pour ne pas remonter.

🎯 Exemple de délégation :

$('body').on('click', '.dynamic-btn', function() { console.log('cliqué'); });
💡 Les événements souris/clavier sont le pont entre l’utilisateur et votre logique JavaScript.
