Partie 1 : Structures conditionnelles (14 pts)

Question 1 – if/else et ternaire (5 pts)

Écrivez une fonction evaluationNote(score) qui prend un nombre (score sur 20). Si le score est ≥ 10, la fonction retourne "Validé", sinon "Échec". Réalisez d'abord une version avec if/else, puis une version avec l'opérateur ternaire.

Question 2 – switch (5 pts)

Créez une fonction obtenirMois(n) qui prend un numéro de mois (1 pour janvier, 2 pour février, etc.) et retourne le nom du mois (en français). Si le numéro est invalide (hors 1-12), retournez "Mois invalide". Utilisez une structure switch.

Question 3 – analyse de code (4 pts)

Quel sera l'affichage du code suivant ? Justifiez étape par étape.

let a = 5, b = 10;
if (a > b) console.log("A supérieur");
else if (a < b) console.log("A inférieur");
else console.log("Égalité");
if (a === b) console.log("Strictement égal");
else console.log("Différent");

Partie 2 : Boucles (20 pts)

Question 4 – boucle for (5 pts)

Écrivez une boucle for qui affiche dans la console les nombres pairs de 2 à 20 inclus. Utilisez l'incrémentation classique.

Question 5 – while et do...while (6 pts)

Écrivez un programme qui demande à l'utilisateur de saisir un nombre (via prompt()) et continue à redemander tant que le nombre saisi n'est pas un entier positif. Utilisez une boucle while.

Bonus : expliquez la différence fondamentale entre while et do...while.

Question 6 – analyse de boucle (4 pts)

Combien de fois la boucle suivante s'exécute-t-elle ? Quelle est la valeur finale de i ?

let i = 0;
while (i < 10) {
    console.log(i);
    i += 2;
}

Question 7 – table de multiplication (5 pts)

Créez une fonction tableMultiplication(n) qui affiche la table de multiplication du nombre n (de 1 à 10) dans la console en utilisant une boucle for. Par exemple, tableMultiplication(7) affichera :

7 x 1 = 7
7 x 2 = 14
...
7 x 10 = 70

Partie 3 : Break et Continue (10 pts)

Question 8 – utilisation de break (5 pts)

Écrivez une boucle for qui parcourt les nombres de 1 à 20, mais s'arrête dès qu'elle rencontre le nombre 13 (utilisation de break). Affichez chaque nombre avant l'arrêt.

Question 9 – continue (5 pts)

Écrivez une boucle while qui affiche les nombres impairs de 1 à 15 en ignorant les nombres pairs (utilisez continue). Indice : utilisez i++ et gérez la condition modulo 2.
