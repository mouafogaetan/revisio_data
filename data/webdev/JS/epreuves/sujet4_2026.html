🔍 Exercice 2 – Analyse et correction (5 pts)

5 pts

Analysez le code suivant. Identifiez les erreurs et proposez une version corrigée.

function calculerPrix(prix, taxe = 0.1925) { let total = prix * (1 + taxe); return total; } let produit = calculerPrix(1000); console.log("Prix TTC : " + produit); function appliquerReduction(prix, reduction) { return prix - (prix * reduction / 100); } let prixReduit = appliquerReduction(500); console.log(prixReduit);
Indice : Une fonction attend deux paramètres mais un seul est fourni.

🔒 Exercice 4 – Closure (3 pts)

3 pts

Créez une fonction creerCompteur() qui retourne une fonction. Cette fonction interne doit incrémenter et retourner un compteur privé à chaque appel.

// Exemple d'utilisation const compteurA = creerCompteur(); console.log(compteurA()); // 1 console.log(compteurA()); // 2 const compteurB = creerCompteur(); console.log(compteurB()); // 1 (indépendant)

📝 Éléments de correction

  • QCM : 1.1 return / 1.2 const add = (a,b) => a+b; / 1.3 undefined / 1.4 Elle permet de créer des variables privées.
  • Erreur code : appliquerReduction(500) manque le second paramètre réduction. Correction : appliquerReduction(500, 10) par exemple.
  • Fonction fléchée : const multiplier = (a, b) => a * b;
  • setTimeout : setTimeout(() => { console.log("Bonjour Cameroun !"); }, 2000);
  • Closure : function creerCompteur() { let count = 0; return () => ++count; }
  • Graphique : La fonction C est la plus rapide (barre la plus basse), la fonction A la plus lente.
