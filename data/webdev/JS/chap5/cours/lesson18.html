📌 Qu'est-ce qu'un objet ?

Un objet est une collection de propriétés. Chaque propriété associe un nom (clé) à une valeur. Cela permet de regrouper des informations qui vont ensemble.

🔹 Exemple concret : une personne possède un nom, un âge, une ville.

// Création d'un objet vide
let personne = {};

// Avec des propriétés
let etudiant = {
  nom: "Angele",
  age: 20,
  filiere: "Informatique"
};
⭐ Les objets stockent des données structurées ⭐ Les valeurs peuvent être de tout type (string, number, booléen, même tableau ou autre objet)

🛠️ Créer des objets (littéraux)

La manière la plus courante : les accolades {}.

// Objet voiture
let voiture = {
  marque: "Toyota",
  modele: "Corolla",
  annee: 2022,
  demarrer: function() {
    console.log("Vroum !");
  }
};

// Objet avec propriétés imbriquées
let etudiant = {
  prenom: "Yannick",
  notes: [15, 17, 12],
  adresse: {
    ville: "Douala",
    quartier: "Bonapriso"
  }
};

✅ Les clés peuvent être écrites sans guillemets (sauf caractères spéciaux). Les valeurs sont séparées par des virgules.

🔍 Accéder aux propriétés

Notation point (.)

Utilisée quand la clé est connue et valide.

let livre = { titre: "JavaScript", pages: 300 };
console.log(livre.titre);   // "JavaScript"
livre.pages = 320;          // modification

Crochets [ ]

Utile pour clés dynamiques ou avec espaces/tirets.

let user = { "nom complet": "Mbarga Jean" };
console.log(user["nom complet"]); 
let prop = "age";
user[prop] = 22;   // ajout dynamique
💡 objet.propriete est plus court, mais objet["propriete"] est plus puissant.

✏️ Ajouter / Modifier / Supprimer

Les objets JavaScript sont dynamiques : on peut changer leur structure à tout moment.

let telephone = {
  marque: "Samsung",
  stockage: "128Go"
};

// Ajouter une propriété
telephone.couleur = "Noir";

// Modifier
telephone.stockage = "256Go";

// Supprimer
delete telephone.marque;

console.log(telephone); 
// { stockage: "256Go", couleur: "Noir" }
⚠️ Attention : delete supprime la propriété entière. On peut aussi lui donner la valeur null ou undefined selon le besoin.

🔁 Parcourir les propriétés d’un objet

On utilise souvent for...in pour itérer sur les clés.

let eleve = { nom: "Claire", age: 18, moyenne: 14.5 };

for (let cle in eleve) {
  console.log(cle + " : " + eleve[cle]);
}
// affiche : nom : Claire, age : 18, moyenne : 14.5
✨ Bonnes pratiques : préférer const pour un objet qu’on modifie (la variable reste constante, mais le contenu peut changer). Utilisez la notation point par défaut.

📌 Récapitulons !

  • Un objet est une structure de données { clé: valeur }
  • Accès via . ou ["clé"].
  • Ajout dynamique : objet.nouvelleCle = valeur
  • Suppression : delete objet.propriete
  • Les méthodes sont des fonctions à l'intérieur d'un objet.
