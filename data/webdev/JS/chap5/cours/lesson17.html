📌 Objectifs de la leçon

  • Parcourir un tableau avec forEach()
  • Transformer chaque élément avec map() et filtrer avec filter()
  • Agréger des données avec reduce(), chercher avec find()
  • Tester des conditions avec some() / every()
  • Trier des tableaux avec sort()
💡 À la fin, vous serez capables de manipuler des collections sans écrire des boucles lourdes !

🔄 forEach() : parcourir simplement

Exécute une fonction sur chaque élément du tableau. Alternative moderne à for.

const fruits = ["mangue", "orange", "ananas"];
fruits.forEach(function(fruit, index) {
    console.log(index + " : " + fruit);
});
// Avec fonction fléchée :
fruits.forEach(fruit => console.log("J'aime la " + fruit));

🗺️ map() – créer un nouveau tableau transformé

Applique une fonction à chaque élément et retourne un tableau de même longueur.

const nombres = [1, 2, 3, 4];
const doubles = nombres.map(n => n * 2);
// doubles → [2, 4, 6, 8]
const annonces = ["chat", "chien"].map(animal => animal.toUpperCase());

🔍 filter() – extraire selon une condition

Retourne un sous-ensemble d'éléments vérifiant un test.

const ages = [12, 18, 25, 16, 30];
const majeurs = ages.filter(age => age >= 18);
// majeurs → [18, 25, 30]

⚙️ reduce() – tout agréger

Réduit un tableau à une seule valeur (somme, moyenne, objet...).

const prix = [250, 100, 75];
const total = prix.reduce((acc, courant) => acc + courant, 0);
// total = 425

🔎 find, some, every : interroger efficacement

  • find() : retourne le premier élément qui satisfait la condition.
  • some() : true si au moins un élément vérifie la condition.
  • every() : true si tous les éléments vérifient la condition.
const utilisateurs = [{nom: "Ali", actif: true}, {nom: "Mona", actif: false}];
const actif = utilisateurs.find(u => u.actif === true);
const auMoinsUnActif = utilisateurs.some(u => u.actif);   // true
const tousActifs = utilisateurs.every(u => u.actif);      // false

📊 sort() – trier comme on veut

Attention : sort() trie par défaut selon les chaînes de caractères. Pour les nombres, il faut une fonction de comparaison.

const notes = [40, 100, 1, 25];
notes.sort((a, b) => a - b); // tri croissant [1,25,40,100]
const fruits = ["banane", "ananas", "orange"];
fruits.sort(); // ["ananas","banane","orange"]
