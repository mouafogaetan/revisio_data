Déstructuration, Spread & Rest

Manipuler tableaux et objets avec élégance

Objectifs de la leçon

  • ✓ Déstructurer des tableaux et des objets facilement.
  • ✓ Utiliser l'opérateur spread (...) pour copier ou fusionner tableaux/objets.
  • ✓ Maîtriser le paramètre rest pour capturer un nombre variable d'arguments.

🎯 À la fin, vous saurez écrire du code plus propre et éviter les boucles inutiles.

📦 Destructuration de tableaux

Extraire des valeurs d'un tableau en une seule ligne !

// Sans destructuration
const fruits = ["Mangue", "Orange", "Ananas"];
let premier = fruits[0];
let deuxieme = fruits[1];

// AVEC destructuration
const [fruitA, fruitB] = fruits;
console.log(fruitA); // Mangue
console.log(fruitB); // Orange

// Ignorer des éléments avec des virgules
const [, second, ] = fruits;
console.log(second); // Orange

// Valeurs par défaut
const [x, y, z = "Citron"] = ["Goyave", "Papaye"];
console.log(z); // Citron
💡 À retenir : La destructuration suit l'ordre du tableau. Utile pour échanger des variables : [a, b] = [b, a]

🔑 Destructuration d'objets

Accéder aux propriétés d'un objet avec le même nom de variable.

const etudiant = {
    nom: "Awa",
    age: 19,
    ville: "Douala"
};

// Destructuration simple
const { nom, age } = etudiant;
console.log(nom);  // Awa
console.log(age);  // 19

// Renommer les variables
const { nom: fullName, ville: city } = etudiant;
console.log(fullName); // Awa

// Valeurs par défaut
const { niveau = "Débutant" } = etudiant;
console.log(niveau); // Débutant (propriété inexistante)
✨ Très pratique avec les APIs

⚡ Spread `...` avec les tableaux

Étaler les éléments d'un tableau. Idéal pour copier ou fusionner.

const saisons = ["Printemps", "Été"];
const autresSaisons = ["Automne", "Hiver"];

// Copie superficielle
const copieSaisons = [...saisons];
console.log(copieSaisons); // ["Printemps", "Été"]

// Fusionner deux tableaux
const toutesSaisons = [...saisons, ...autresSaisons];
console.log(toutesSaisons); // ["Printemps","Été","Automne","Hiver"]

// Ajouter un élément au début
const nouveaux = ["Nouvel An", ...saisons];
console.log(nouveaux); 

// Spread avec des chaînes
const lettres = [..."Cameroun"];
console.log(lettres); // ['C','a','m','e','r','o','u','n']

🔄 Spread `...` sur les objets (ES2018)

Cloner ou fusionner des objets proprement.

const coordonnees = { x: 10, y: 20 };
const options = { couleur: "rouge", actif: true };

// Cloner un objet
const cloneCoord = { ...coordonnees };
console.log(cloneCoord); // { x:10, y:20 }

// Fusionner (les propriétés de droite écrasent celles de gauche)
const fusion = { ...coordonnees, ...options };
console.log(fusion); // { x:10, y:20, couleur:"rouge", actif:true }

// Immuabilité: très utile en programmation fonctionnelle
const user = { nom: "Claude", age: 25 };
const userUpdated = { ...user, ville: "Yaoundé", age: 26 };
console.log(user);      // inchangé
console.log(userUpdated); // { nom:"Claude", age:26, ville:"Yaoundé" }

🛑 Paramètre Rest `...`

Regroupe les arguments restants dans un tableau.

// Récupérer tous les arguments supplémentaires
function somme(message, ...nombres) {
    console.log(message);
    return nombres.reduce((acc, val) => acc + val, 0);
}

console.log(somme("Addition:", 5, 10, 15)); 
// Addition:  -> retourne 30

// Avec plusieurs paramètres
function panier(proprietaire, ...articles) {
    return `${proprietaire} achète : ${articles.join(", ")}`;
}
console.log(panier("Fatima", "livre", "cahier", "stylo"));
// Fatima achète : livre, cahier, stylo

// Rest vs Spread : le contexte détermine tout
⚠️ Le paramètre rest doit toujours être le dernier paramètre de la fonction.

const panierBase = ["Bananes", "Mangues"];
const nouvelArticle = "Ananas";
const panierFinal = [...panierBase, nouvelArticle];
console.log(panierFinal); // ["Bananes","Mangues","Ananas"]
