🔐 VPN

Virtual Private Network

Un tunnel sécurisé sur Internet pour protéger vos données et votre identité

Cliquez sur les boutons ci-dessous pour naviguer

🎯 Objectifs pédagogiques

  • ✓ Comprendre le principe d'un VPN (tunnel chiffré)
  • ✓ Distinguer les protocoles (IPsec, OpenVPN, WireGuard, L2TP)
  • ✓ Configurer un VPN site-à-site ou client-serveur

💡 Un VPN crée un réseau privé virtuel au-dessus d'Internet public. Toutes les données sont chiffrées de bout en bout.

🛡️ Qu'est-ce qu'un VPN ?

Un VPN (Virtual Private Network) étend un réseau privé à travers un réseau public (Internet) en créant un tunnel chiffré.

🔹 Objectifs principaux :

  • Confidentialité : chiffrement des données (personne ne peut les lire)
  • Anonymat : votre adresse IP réelle est masquée
  • Intégrité : les données ne peuvent pas être modifiées en transit
  • Authentification : seuls les clients autorisés peuvent se connecter
Sans VPN :
[Client] ===== Internet (en clair) ===== [Serveur]

Avec VPN (tunnel) :
[Client]  <---[== VPN Tunnel chiffré ==]---> [Serveur VPN] --- [Ressource]

🏢 Types de VPN

🌐 VPN Client-Serveur (Remote Access)

Un utilisateur isolé se connecte au réseau de son entreprise depuis son domicile ou un café.

Télétravailleur ---📡--- [Internet] ---🔒--- [Routeur VPN Entreprise] --- [Réseau interne]

✅ Idéal pour le télétravail.

🏛️ VPN Site-à-Site (Router-to-Router)

Deux réseaux locaux distants sont interconnectés comme s'ils étaient sur le même câble.

[Site Douala] ---🔒 [Routeur] --- Internet --- [Routeur] 🔒--- [Site Yaoundé]

✅ Utilisé pour relier des succursales.

🔧 Protocoles VPN : IPsec

IPsec (IP Security) : protocole standard de la couche réseau (niveau 3). Très largement utilisé dans les routeurs professionnels.

Modes IPsec :

  • Transport : seule la charge utile est chiffrée (hors en-tête IP).
  • Tunnel : tout le paquet IP (en-tête + données) est chiffré et encapsulé. Le plus utilisé pour VPN site-à-site.
Paquet IPsec en mode tunnel :
[ NOUVEL EN-TÊTE IP (passerelles) ] [ EN-TÊTE IPsec ] [ ANCIEN PAQUET IP chiffré ]

Deux sous-protocoles : AH (authentification, pas chiffrement) et ESP (chiffrement + authentification).

🔓 OpenVPN

OpenVPN est une solution open source très populaire (couche application, SSL/TLS).

✅ Avantages
  • Très sécurisé (certificats, clés)
  • Traverse facilement les pare-feu (port 443)
  • Disponible sur tous OS (Windows, Linux, macOS, Android, iOS)
⚠️ Inconvénients
  • Configuration plus complexe
  • Mode user-space peut être moins performant
Commande client OpenVPN typique :
sudo openvpn --config client.ovpn

📌 OpenVPN est utilisé par de nombreux fournisseurs VPN commerciaux (NordVPN, ExpressVPN, ProtonVPN).

⚡ WireGuard

WireGuard est un protocole VPN récent, plus simple, plus rapide et plus sécurisé que IPsec et OpenVPN.

🔹 Points forts :

  • Code source très réduit (~4000 lignes contre ~400 000 pour OpenVPN)
  • Cryptographie moderne (Curve25519, ChaCha20, Poly1305)
  • Très faible latence, idéal pour le gaming et la VoIP
  • Intégré directement dans le noyau Linux depuis la version 5.6
Configuration minimaliste (wg0.conf) :
[Interface]
PrivateKey = abc123...
Address = 10.0.0.1/24

[Peer]
PublicKey = xyz789...
AllowedIPs = 10.0.0.2/32

📜 Protocoles VPN historiques

PPTP (Point-to-Point Tunneling Protocol) – obsolète et non sécurisé depuis 2012 (failles MS-CHAPv2). ❌ À éviter.
L2TP (Layer 2 Tunneling Protocol) – souvent combiné avec IPsec (L2TP/IPsec). Sécurisé mais double encapsulation, peut être plus lent.
SSTP (Secure Socket Tunneling Protocol) – développé par Microsoft, utilise SSL (port 443). Bonne compatibilité Windows.

⚠️ PPTP ne doit plus être utilisé. L2TP/IPsec reste acceptable mais WireGuard ou OpenVPN sont préférables.

🇨🇲 Cas d'usage des VPN au Cameroun

  • Télétravail : les employés se connectent au réseau de l'entreprise depuis leur domicile (sécurisé).
  • Succursales bancaires : les agences Bank Central, Afriland, SCB relient leurs sites via VPN pour échanger des données sensibles.
  • Administration : certains ministères utilisent des VPN site-à-site pour interconnecter leurs directions régionales.
  • Protection personnelle : les citoyens utilisent des VPN pour sécuriser leurs échanges sur les WiFi publics (aéroport, cafés).
  • Contournement de censure : accéder à des contenus géo-restreints (streaming, actualités).
⚠️ Attention : L'utilisation d'un VPN pour des activités illégales reste interdite. Respectez les lois camerounaises sur la cybersécurité.

⚙️ Exemple de configuration client OpenVPN

# Fichier .ovpn (client)
client
dev tun
proto udp
remote vpn.entreprise.cm 1194
resolv-retry infinite
nobind
persist-key
persist-tun
ca ca.crt
cert client.crt
key client.key
remote-cert-tls server
cipher AES-256-GCM
verb 3

📌 Pour se connecter : télécharger le fichier de configuration, l'importer dans l'application OpenVPN, puis cliquer sur "Connecter".

💡 Sur smartphone : des applications comme OpenVPN Connect, WireGuard ou les clients intégrés (Android VPN, iOS IKEv2) permettent une connexion en un clic.

📝 À retenir

  • Un VPN crée un tunnel chiffré sur Internet pour des communications privées.
  • Types : Client-Serveur (télétravail) et Site-à-Site (liaison entre succursales).
  • Protocoles courants : IPsec (standard, niveau réseau), OpenVPN (SSL/TLS, open source), WireGuard (moderne, très rapide).
  • PPTP est obsolète et dangereux → à proscrire.
  • Un VPN ne rend pas complètement anonyme ; il faut aussi bloquer les traces (cookies, DNS leaks).

🎓 Le saviez-vous ?

Les fournisseurs d'accès (Orange, MTN, CAMTEL) ne peuvent pas voir le contenu des paquets dans un VPN. Cependant, ils savent qu'un VPN est utilisé (volume de données, horaires).
