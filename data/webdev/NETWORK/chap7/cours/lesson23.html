💡 Le WiFi est partout : maisons, entreprises, aéroports, universités. Derrière ce nom se cache la famille de normes IEEE 802.11.

📡 Qu'est-ce que le WiFi ?

WiFi = marque déposée par la Wi-Fi Alliance. Le protocole sous-jacent est la famille IEEE 802.11 (couches physique et liaison).

🔹 Caractéristiques générales :

  • Transmission par ondes radio (support non guidé)
  • Bandes de fréquences libres (ISM : 2.4 GHz, 5 GHz, 6 GHz)
  • Portée typique : 30-100 m en intérieur, plus en extérieur
  • Utilise CSMA/CA pour éviter les collisions (écoute avant d'émettre)
📌 La bande 2.4 GHz porte plus loin mais est plus encombrée (micro-ondes, Bluetooth, téléphones sans fil). La bande 5 GHz est plus rapide et moins interférée mais porte moins loin.

📡 Bandes 2.4 GHz vs 5 GHz vs 6 GHz

2.4 GHz

  • Portée longue
  • Traverse mieux les murs
  • Encombré (micro-ondes, BT)
  • 3 canaux non chevauchants

5 GHz

  • Moins de portée
  • Moins d'interférences
  • Plus de canaux (23 non chevauchants)
  • Débits plus élevés

6 GHz (WiFi 6E/7)

  • Très large bande
  • Très faible interférence
  • Portée encore plus courte
  • Nouveaux terminaux requis

🏗️ Modes de fonctionnement WiFi

Mode Infrastructure (BSS)

Le mode le plus courant : un point d'accès (AP) central connecté au réseau filaire. Les clients (stations) s'associent à l'AP.

[Client] ---📶--- [AP] ---🔌--- [Switch/Routeur] --- Internet

✅ Avantage : gestion centralisée, extension facile, sécurité (WPA2/3).

Mode Ad-hoc (IBSS)

Communication directe entre deux stations sans point d'accès. Chaque périphérique joue le rôle de routeur pour les autres.

[Client A] <---📶---> [Client B]

⚠️ Peu utilisé aujourd'hui, remplacé par WiFi Direct ou Bluetooth.

🎲 CSMA/CA : éviter les collisions

En WiFi, on ne peut pas écouter et émettre en même temps (contrairement à Ethernet). Donc on utilise CSMA/CA (Collision Avoidance).

1. Écouter le canal (CCA) → s'il est libre, attendre DIFS

2. Envoyer une trame RTS (Request to Send)

3. Le destinataire répond CTS (Clear to Send)

4. Envoi des données + ACK

Émetteur                        Récepteur
    |--- RTS ----------------------→|
    |←--- CTS -----------------------|
    |--- DATA ---------------------→|
    |←--- ACK -----------------------|

⚠️ Le RTS/CTS réduit les collisions cachées (problème du terminal caché).

🏷️ Identifiants WiFi

  • SSID (Service Set IDentifier) : nom du réseau WiFi visible par l'utilisateur (ex: "Orange_WiFi", "MTN_Campus")
  • BSSID : adresse MAC du point d'accès (unique par AP).
  • ESSID : étendu à plusieurs AP (même SSID). Permet le roaming : se déplacer sans perdre la connexion.

Exemple d'un campus universitaire :

Plusieurs bornes WiFi avec le même SSID "UnivWifi". Votre téléphone bascule automatiquement de l'une à l'autre (roaming).

[Borne1 MAC: AA:AA] --- [Borne2 MAC: BB:BB] --- [Borne3 MAC: CC:CC]
           \_______________ même SSID "UnivWifi" _______________/

📡 Antennes WiFi

Omnidirectionnelle → signal dans toutes les directions (360°), idéale pour un point d'accès classique.
Directionnelle → signal concentré dans une direction (type parabole, panel). Portée plus longue.

🇨🇲 Le WiFi au Cameroun

  • Les box Internet (Orange, MTN, CAMTEL, YooMee) intègrent toutes un point d'accès WiFi 802.11ac (WiFi 5) ou 802.11ax (WiFi 6).
  • Dans les grandes villes (Douala, Yaoundé, Garoua), de nombreux cybercafés, hôtels et universités offrent le WiFi public.
  • Les projets de smart cities intègrent des hotspots WiFi gratuits dans certaines zones.
  • Attention : le WiFi public non sécurisé expose aux risques d'interception (évitez les transactions bancaires).
💡 Bonnes pratiques : utilisez un VPN sur les réseaux WiFi publics, et privilégiez les hotspots sécurisés (WPA2/WPA3).

📝 À retenir

  • Le WiFi = marque pour la famille IEEE 802.11
  • Normes récentes : 802.11ac (WiFi 5) et 802.11ax (WiFi 6/6E) pour la très haute densité
  • Deux bandes principales : 2.4 GHz (portée, encombrée) et 5 GHz (débit, moins de portée)
  • Mode infrastructure = avec point d'accès ; mode ad-hoc = direct entre stations
  • Le protocole d'accès est CSMA/CA (évitement de collision).

🎓 Évolution : le WiFi 7 (802.11be) arrive avec des débits théoriques de 30 Gb/s, idéal pour la réalité virtuelle et la 8K.
