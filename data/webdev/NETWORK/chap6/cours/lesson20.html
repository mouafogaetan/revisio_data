🔌 Qu'est-ce qu'un switch ?

Un switch est un équipement réseau qui relie plusieurs périphériques au sein d'un même réseau local (LAN).

Couche 2 (liaison) : utilise les adresses MAC

Intelligent : apprend quelle machine est sur quel port

Full-duplex : envoie/reçoit simultanément

📋 La table d'adresses MAC

Le switch maintient une table qui associe chaque adresse MAC à un port physique.

Adresse MACPortÂge (sec)
AA:AA:AA:AA:AA:01110
BB:BB:BB:BB:BB:0225
CC:CC:CC:CC:CC:033120

Apprentissage : quand un paquet arrive, le switch enregistre la MAC source et le port. Si la MAC destination est inconnue → inondation (broadcast) sur tous les ports sauf la source.

⚙️ Switch non géré vs Switch géré

Non géré (plug-and-play)

  • Aucune configuration
  • Prix bas
  • Petits réseaux domestiques
  • Pas de VLAN, QoS, monitoring

Géré (configurable)

  • Configuration via CLI/Web
  • VLAN, QoS, SNMP, STP
  • Port mirroring, sécurité (802.1X)
  • Réseaux d'entreprise

🏷️ Les VLAN (Virtual LAN)

Un VLAN permet de découper un switch physique en plusieurs réseaux logiques isolés.

Avantages :

  • Sécurité (isolation du trafic)
  • Réduction des broadcasts
  • Flexibilité (équipes différentes)

Exemple :

  • VLAN 10 : Comptabilité
  • VLAN 20 : Commercial
  • VLAN 30 : Direction
switchport access vlan 10 – affecte un port au VLAN 10

🎯 QoS – Qualité de service

La QoS permet de prioriser certains types de trafic sur le réseau.

VoIP (téléphonie) Priorité haute
Visioconférence Priorité haute
Transfert de fichiers Priorité normale
P2P / mails Priorité basse

🌲 STP – Spanning Tree Protocol

Problème des boucles : dans un réseau redondant, les paquets peuvent tourner en boucle.

STP bloque certains ports pour créer une topologie logique sans boucle (arbre couvrant).

Switch A ←→ Switch B ←→ Switch C

✓ Port actif

✗ Port bloqué (STP)

États STP : Blocking → Listening → Learning → Forwarding

🔄 Routage inter-VLAN

Pour que deux VLAN communiquent, il faut un routeur ou un switch de couche 3.

Solution :

  • Router-on-a-stick : un seul lien trunk vers le routeur
  • Switch L3 : commutateur avec capacités de routage
interface GigabitEthernet0/1.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0
