Ce que vous allez apprendre

  • ✓ Définir une signature de fonction avec type alias.
  • ✓ Utiliser les fonctions fléchées typées.

💡 Une signature de fonction décrit les paramètres et le type de retour, sans l'implémentation.

📝 Signature de fonction

Une signature décrit la forme d'une fonction : types des paramètres et type de retour.

(param1: type1, param2: type2) => typeRetour

Exemple :

// Signature : prend deux nombres et retourne un nombre
(a: number, b: number) => number

🏷️ Type alias pour fonction

On peut nommer une signature avec type pour la réutiliser.

type OperationMathematique = (a: number, b: number) => number;

// Utilisation
const addition: OperationMathematique = (x, y) => x + y;
const soustraction: OperationMathematique = (x, y) => x - y;
💡 Avantage : on évite de répéter la signature, et on garantit la cohérence.

⚡ Fonctions fléchées typées

Deux façons de typer une fonction fléchée :

Méthode 1 : typage direct 
const direBonjour: (nom: string) => string = (nom) => `Bonjour ${nom}`;
Méthode 2 : inférence + typage paramètres 
const direBonjour = (nom: string): string => `Bonjour ${nom}`;
✅ La méthode 2 est souvent préférée car plus lisible et TypeScript infère le type de retour.

⚠️ Éviter le type Function

Le type Function accepte n'importe quelle fonction, sans aucune vérification de sécurité.

// ❌ À éviter
let maFonction: Function;
maFonction = (a: string) => a.length; // OK
maFonction = "pas une fonction"; // ERREUR mais TypeScript ne vérifie pas les paramètres
// ✅ Recommandé
type Callback = (data: string) => number;
let traiter: Callback = (texte) => texte.length;

Le type explicite garantit que la fonction reçoit les bons paramètres et retourne le bon type.

🔧 Signatures avec paramètres optionnels

type Formattage = (texte: string, majuscule?: boolean) => string;

const formater: Formattage = (texte, majuscule = false) => {
    return majuscule ? texte.toUpperCase() : texte.toLowerCase();
};

console.log(formater("Bonjour"));      // "bonjour"
console.log(formater("Bonjour", true)); // "BONJOUR"

Le ? rend le paramètre optionnel. À l'implémentation, on peut fournir une valeur par défaut.

🧪 Exercice pratique

Mettez en pratique ce que vous avez appris :

1. Créez un type alias MapFunction qui prend un tableau de nombres et retourne un tableau de nombres.

2. Implémentez une fonction fléchée doubler de ce type qui multiplie chaque élément par 2.

3. Créez une fonction appliquer qui prend un tableau et une fonction de type MapFunction, et retourne le résultat.

📌 Solution : 
type MapFunction = (nombres: number[]) => number[];

const doubler: MapFunction = (nombres) => nombres.map(n => n * 2);

const appliquer = (nombres: number[], fn: MapFunction): number[] => fn(nombres);

console.log(appliquer([1, 2, 3], doubler)); // [2, 4, 6]
