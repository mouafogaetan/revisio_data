TypeScript | Lesson 2

Installation et Configuration

npm · tsc · tsconfig.json

Classe : Développement Web – Initiation à TypeScript
Objectifs

Ce que vous allez apprendre

  • ✓ Installer TypeScript globalement avec npm.
  • ✓ Compiler un fichier .ts avec tsc fichier.ts.
  • ✓ Créer et configurer tsconfig.json (target, strict, outDir, rootDir).
💡 TypeScript se compile en JavaScript. La configuration tsconfig.json centralise toutes les options du compilateur.

📦 Installation de TypeScript

TypeScript nécessite Node.js et npm (Node Package Manager).

# Vérifier que Node.js est installé
node --version
npm --version

# Installer TypeScript globalement
npm install -g typescript

# Vérifier l'installation
tsc --version
⚠️ Sur Linux/Mac, vous pourriez avoir besoin de sudo pour l'installation globale.

📝 Premier fichier TypeScript

Créez un fichier index.ts :

// index.ts
function saluer(nom: string): string {
    return `Bonjour ${nom} !`;
}

console.log(saluer("Cameroun"));

Compiler le fichier :

tsc index.ts

→ Génère un fichier index.js exécutable avec Node.js : node index.js

⚙️ Options du compilateur (tsc)

🔁 --watch (mode surveillance)

Recompile automatiquement à chaque modification.

tsc index.ts --watch
🚫 --noEmitOnError

Ne génère pas de JS s'il y a des erreurs TS.

tsc index.ts --noEmitOnError
🎯 --target

Version JS cible (ES5, ES6, ES2020).

tsc index.ts --target ES2020
📁 --outDir

Dossier de sortie pour les fichiers JS.

tsc index.ts --outDir dist

📄 tsconfig.json : la configuration

Générez un fichier de configuration :

tsc --init

Exemple de configuration personnalisée :

{
  "compilerOptions": {
    "target": "ES2020",
    "module": "CommonJS",
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src",
    "esModuleInterop": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}

🔍 Options importantes de tsconfig.json

  • target : Version JS générée (ES5, ES6, ES2020...)
  • module : Système de modules (CommonJS, ESNext, AMD)
  • strict : Active toutes les vérifications strictes (recommandé)
  • outDir : Dossier où placer les fichiers JS compilés
  • rootDir : Dossier racine des fichiers TS (src/ par exemple)
  • esModuleInterop : Compatibilité avec les modules CommonJS
💡 Après avoir configuré tsconfig.json, il suffit de taper tsc pour compiler tout le projet.

🧪 Exercice pratique

Mettez en pratique ce que vous avez appris :

1. Installez TypeScript globalement.

2. Créez un fichier hello.ts avec une fonction qui additionne deux nombres typés.

3. Compilez-le en JS avec tsc hello.ts.

4. Générez un tsconfig.json et configurez outDir: "./dist" et rootDir: "./src".

5. Utilisez le mode --watch pour recompiler automatiquement.

📌 Correction : 
// hello.ts
function addition(a: number, b: number): number {
    return a + b;
}
console.log(addition(5, 3));
