Partie 2 : Vrai ou Faux (10 pts)

Une interface peut étendre plusieurs autres interfaces en utilisant une virgule.
Le mot-clé readonly rend une propriété modifiable uniquement à l'initialisation.
Record<string, number> est équivalent à { [key: string]: number }.
Le type unknown peut être assigné à n'importe quelle variable sans vérification préalable.
Les mapped types permettent de transformer chaque propriété d'un type existant.

Exercice 2 - 20 pts Extension et types utilitaires

À partir de l'interface Produit suivante :

interface Produit {
  id: number;
  nom: string;
  prix: number;
  stock: number;
  description?: string;
}

Réalisez les tâches suivantes :

  1. Créez une interface ProduitAvecRemise qui étend Produit et ajoute une propriété remise (nombre, optionnelle).
  2. Créez un type ProduitPartiel qui rend toutes les propriétés de Produit optionnelles en utilisant un type utilitaire.
  3. Créez un type ProduitSansStock qui utilise Omit pour exclure la propriété stock.
Exercice 3 - 15 pts Dictionnaire typé

Vous devez créer un système de traduction dynamique pour une application mobile.

  • Définissez un type DictionnaireTraduction avec une signature d'index où les clés sont des chaînes (codes de langue comme "fr", "en", "es") et les valeurs sont des objets contenant des traductions.
  • Chaque traduction doit avoir : bonjour, auRevoir, merci (tous des chaînes).
  • Ensuite, utilisez Record<string, Traduction> pour définir le même type.
Exercice 4 - 15 pts Union, intersection et narrowing

Soient les types suivants :

interface Cercle {
  type: "cercle";
  rayon: number;
}

interface Rectangle {
  type: "rectangle";
  largeur: number;
  hauteur: number;
}

Réalisez :

  1. Définissez un type Forme qui est l'union de Cercle et Rectangle.
  2. Écrivez une fonction calculerAire(forme: Forme): number qui calcule l'aire selon le type de forme (utilisez le narrowing).
  3. Créez une intersection FormeColoree = Forme & { couleur: string }.
