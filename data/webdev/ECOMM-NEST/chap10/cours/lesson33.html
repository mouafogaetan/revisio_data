🎟️ Appliquer les coupons de réduction

Vérifier la validité, appliquer la réduction au panier/commande et générer des coupons uniques

🔍 Vérifier la validité d’un coupon

Un code promo doit être vérifié selon plusieurs critères avant application :

  • Date de validité : date_debut ≤ aujourd'hui ≤ date_fin
  • Nombre max d’utilisations : si `maxUses` atteint, coupon invalide.
  • Par utilisateur : `perUserLimit` (ex: 1 fois par client).
  • Montant minimum de panier (`minAmount`).
  • Produits spécifiques : le coupon ne s'applique qu'à certains produits.
// Exemple de validation coté service (NestJS)
async validateCoupon(code: string, userId: string, cartTotal: number) {
  const coupon = await this.couponRepo.findOne({ where: { code } });
  if (!coupon) throw new Error("Code invalide");
  const now = new Date();
  if (now < coupon.validFrom || now > coupon.validUntil)
    throw new Error("Coupon expiré");
  if (coupon.usedCount >= coupon.maxUses)
    throw new Error("Coupon épuisé");
  if (cartTotal < coupon.minAmount)
    throw new Error(`Minimum d'achat ${coupon.minAmount} FCFA requis`);
  // Vérification perUser ...
  return coupon;
}

🧮 Appliquer la réduction : panier / commande

Deux types de réduction :

📉 Pourcentage (%)

Réduction = \( \text{total} \times \frac{\text{valeur}}{100} \)

Ex: -15% → total * 0.85
💰 Montant fixe (FCFA)

Réduction = \( \min(\text{valeur}, \text{total}) \)

Ex: -2000 FCFA sur total ≥ 5000 FCFA
calculateDiscount(coupon, cartSubtotal) {
  if (coupon.type === 'percent') {
    return cartSubtotal * (coupon.value / 100);
  } else { // fixed
    return Math.min(coupon.value, cartSubtotal);
  }
}
⚠️ Important : Après application, mettre à jour le total de la commande et enregistrer le coupon utilisé.

✉️ Générer des coupons uniques pour campagne email

Pour offrir des codes personnalisés à chaque client (ex: "BIENVENUE-JOE123"), on génère des chaînes aléatoires et on les associe à un utilisateur ou un email.

import { randomBytes } from 'crypto';

generateUniqueCode(prefix = "PROMO") {
  const random = randomBytes(4).toString('hex').toUpperCase();
  return `${prefix}-${random}`;
}
// Exemple: PROMO-3F8A1C

Implémentation résumée :

  • Récupérer le panier actif de l'utilisateur.
  • Valider le coupon avec les règles vues précédemment.
  • Calculer la réduction et mettre à jour le champ `discountAmount` du panier.
  • Sauvegarder le coupon appliqué (référence).
  • Retourner le nouveau total.

🏁 Récapitulatif & Défis

✔️ Ce qu'il faut retenir

  • Validation multi-critères (date, stock, utilisateur).
  • Différence % vs montant fixe.
  • Génération de codes uniques pour personnalisation.
  • Ne jamais appliquer deux fois le même coupon sur une commande.

🚀 Défis pour les élèves

  • Créer un endpoint `GET /coupons/validate?code=...`
  • Implémenter un coupon "Livraison gratuite" (type spécial).
  • Ajouter une condition "première commande uniquement".
