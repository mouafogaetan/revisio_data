Leçon 17 : Calculs & promotions

Calculs du panier

Sous-total, taxes, frais de livraison, codes promo

🧮 Chapitre 5•⚡ Leçon 17•🏷️ Promotions & frais
🎯

Objectifs de la leçon

  • ✅ Calculer le sous-total, les taxes et les frais de livraison
  • ✅ Appliquer des codes promo (réduction % ou montant fixe)
  • ✅ Gérer les frais de port (gratuité selon seuil, poids, zone)
  • ✅ Retourner un récapitulatif clair au client
💡 Le calcul du panier est central : il détermine le montant final avant paiement.

📊 1. Structure de la réponse

export class CartResponseDto {
    items: CartItemDto[];
    subtotal: number;      // HT
    tax: number;           // TVA (ex: 19.25%)
    taxAmount: number;     // Montant de la taxe
    shippingCost: number;  // Frais de livraison
    discount: number;      // Réduction appliquée
    total: number;         // Total TTC à payer
    couponCode?: string;
}

💰 total = subtotal + taxAmount + shippingCost - discount

📈 2. Calcul du sous-total et TVA

computeSubtotal(items: CartItem[]): number {
    return items.reduce((sum, item) => 
        sum + (item.product.price * item.quantity), 0);
}

computeTaxAmount(subtotal: number, taxRate: number): number {
    return subtotal * (taxRate / 100);
}
🇨🇲 Au Cameroun, la TVA est de 19.25% pour la plupart des biens. Certains produits peuvent être exonérés (livres, produits de première nécessité).

🚚 3. Frais de livraison

computeShippingCost(subtotal: number, weight: number, zone: string): number {
    // Livraison gratuite si montant > 50 000 FCFA
    if (subtotal >= 50000) return 0;
    
    const rates = {
        'douala': 1500,
        'yaounde': 2000,
        'other': 3500
    };
    
    // Supplément poids si > 5kg
    let extra = weight > 5 ? 1000 : 0;
    return (rates[zone] || rates['other']) + extra;
}
📦 On peut aussi intégrer une API de transporteur (La Poste, DHL) pour calculer le coût en temps réel.

🎟️ 4. Application des codes promo

applyCoupon(subtotal: number, coupon: Coupon): number {
    if (coupon.type === 'percent') {
        return subtotal * (coupon.value / 100);
    } else if (coupon.type === 'fixed') {
        return Math.min(coupon.value, subtotal);
    }
    return 0;
}

// Vérification de validité
isCouponValid(coupon: Coupon, subtotal: number): boolean {
    const now = new Date();
    return coupon.validUntil > now &&
           (!coupon.minAmount || subtotal >= coupon.minAmount) &&
           coupon.usedCount < coupon.maxUses;
}

⚙️ 5. Service de calcul unifié

async getCartTotals(cartId: string, couponCode?: string, zone: string = 'douala'): Promise<CartResponseDto> {
    const cart = await this.getCartWithItems(cartId);
    const subtotal = this.computeSubtotal(cart.items);
    const taxAmount = this.computeTaxAmount(subtotal, this.configService.get('TAX_RATE'));
    
    let discount = 0;
    let coupon = null;
    if (couponCode) {
        coupon = await this.couponService.findByCode(couponCode);
        if (coupon && this.isCouponValid(coupon, subtotal)) {
            discount = this.applyCoupon(subtotal, coupon);
        }
    }
    
    const weight = cart.items.reduce((w, i) => w + (i.product.weight || 0) * i.quantity, 0);
    const shippingCost = this.computeShippingCost(subtotal, weight, zone);
    
    const total = subtotal + taxAmount + shippingCost - discount;
    
    return { items: cart.items, subtotal, taxAmount, shippingCost, discount, total, couponCode };
}

🌐 6. Endpoint de calcul

// CartController
@Get('totals')
@UseGuards(JwtAuthGuard)
async getCartTotals(
    @Req() req,
    @Query('coupon') couponCode?: string,
    @Query('zone') zone?: string
) {
    const cart = await this.cartService.getCartByUserId(req.user.id);
    return this.cartService.getCartTotals(cart.id, couponCode, zone);
}
📌 Exemple de réponse : { "subtotal": 50000, "taxAmount": 9625, "shippingCost": 0, "discount": 5000, "total": 54625 }
