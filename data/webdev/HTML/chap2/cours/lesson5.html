Initiation au HTML · Chapitre 2

Les listes en HTML

Structurer et organiser l’information avec ul, ol et bien plus.

📘 Cours enseignant au Cameroun — Niveau débutant

🎯 Objectifs d'apprentissage

  • 1 Créer des listes non ordonnées (ul) pour des éléments sans ordre particulier.
  • 2 Maîtriser les listes ordonnées (ol) avec numérotation automatique.
  • 3 Utiliser les listes de définition (dl, dt, dd) pour du vocabulaire ou des paires terme-définition.
  • 4 Imbriquer des listes pour créer des menus ou plans structurés.

📋 1. Liste non ordonnée (ul)

La balise <ul> (unordered list) affiche des puces. Chaque élément est une <li>. Idéal pour les listes de courses, caractéristiques, etc.

✨ Exemple :

<ul>
  <li>Pommes</li>
  <li>Bananes</li>
  <li>Oranges</li>
</ul>

Résultat :

  • Pommes
  • Bananes
  • Oranges

💡 On peut changer l’aspect des puces avec CSS (cercle, carré…), mais par défaut le navigateur affiche un disque plein.

🔢 2. Liste ordonnée (ol)

<ol> génère une liste numérotée (1, 2, 3, ...). Parfaite pour les étapes, classements, instructions.

📝 Exemple :

<ol>
  <li>Préchauffer le four</li>
  <li>Mélanger la farine</li>
  <li>Cuire 20 minutes</li>
</ol>

Aperçu :

  1. Préchauffer le four
  2. Mélanger la farine
  3. Cuire 20 minutes
⚙️ On peut modifier le type de numérotation avec l'attribut type : "A" pour majuscules, "i" pour chiffres romains...

📖 3. Listes de définition (dl)

Les listes de définition servent à associer un terme ( <dt> ) à sa définition ( <dd> ), le tout encadré par <dl>. Utile pour glossaire ou FAQ.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language, langage de structure.</dd>
  <dt>CSS</dt>
  <dd>Feuilles de style pour la présentation.</dd>
</dl>

Rendu visuel :

HTML
HyperText Markup Language, langage de structure.
CSS
Feuilles de style pour la présentation.

💡 Les navigateurs affichent généralement les définitions avec une légère indentation.

🌿 4. Imbriquer des listes

Une liste peut contenir d’autres listes. Cela permet de créer des menus hiérarchiques, des plans détaillés, ou des sous-catégories.

<ul>
  <li>Fruits
    <ul>
      <li>Agrumes</li>
      <li>Fruits rouges</li>
    </ul>
  </li>
  <li>Légumes
    <ul>
      <li>Tomates</li>
      <li>Carottes</li>
    </ul>
  </li>
</ul>

🎨 Résultat imbriqué :

  • Fruits
    • Agrumes
    • Fruits rouges
  • Légumes
    • Tomates
    • Carottes
✅ Important : une sous-liste doit toujours être placée à l'intérieur d'un élément <li>.

⚡ Astuces & exemples concrets

💻 Utilisez les listes pour organiser : menus de navigation, recettes, caractéristiques de produits, ou questions/réponses.

Menu simple (ul) :

  • Accueil
  • Services
  • Contact

Top 3 (ol) :

  1. Étape 1: Analyser
  2. Étape 2: Coder
  3. Étape 3: Tester
✏️ Petit dessin maison : une liste imbriquée schématisée 
├─ Cours
│  ├─ HTML
│  └─ CSS
└─ Exercices
   ├─ Quiz
   └─ Projet

Imbrication représentée avec des tirets et lignes. Cette structure est facilement réalisable avec ul et li.

📌 Rappel : les balises de liste sont très utilisées en HTML, elles améliorent l’accessibilité et le référencement (SEO).

  • Une liste ordonnée des étapes pour préparer un sandwich.
  • Une liste de définition avec deux termes HTML et leur description.
