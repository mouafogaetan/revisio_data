📝 Évaluation Séquentielle n°2

Structuration du contenu HTML : titres, listes, liens, images

📅 Séquence 2 · Durée : 60 min

Classe : Développement Web (HTML)

📚 Matière : Initiation au HTML
🎯 Compétences : Titres (h1-h6), paragraphes, listes ordonnées/non ordonnées, liens hypertextes, balises d'images
⭐ Total : 50 points

📌 Consignes : Lisez chaque énoncé. Pour les questions de code, écrivez le HTML valide correspondant. Les questions avec rédaction nécessitent des phrases complètes. Le soin et la précision du code seront évalués.

📌

Partie 1 : Titres et hiérarchie (8 pts)

Question 1 (4 pts)

Complétez la phrase :
"La balise HTML utilisée pour le titre principal d'une page est _________. Il ne devrait y avoir qu'un seul titre de niveau 1 par page pour des raisons de _________ et d'_________."

Question 2 (4 pts)

Écrivez le code HTML d'une hiérarchie de titres pour un article de blog intitulé "Les bases du HTML", avec deux sections : "Pourquoi apprendre HTML ?" (niveau 2) et "Les balises essentielles" (niveau 2), cette dernière contenant une sous-section "Les titres" (niveau 3).

📋

Partie 2 : Listes ordonnées et non ordonnées (12 pts)

Question 3 – Liste d'ingrédients (6 pts)

Créez une liste non ordonnée (puces) contenant : farine, sucre, œufs, beurre. À l'intérieur, ajoutez une sous-liste ordonnée (numérotée) pour les étapes de préparation : Mélanger les ingrédients secs, Ajouter les œufs, Incorporer le beurre fondu. Écrivez le code HTML complet.

Question 4 – Liste de définitions (6 pts)

En utilisant les balises <dl>, <dt>, <dd>, créez une liste de définitions pour trois termes : HTML (HyperText Markup Language), CSS (Cascading Style Sheets) et URL (Uniform Resource Locator).

🔗

Partie 3 : Liens hypertextes (12 pts)

Question 5 – Types de liens (6 pts)

Pour chacun des cas suivants, écrivez l'attribut href approprié :

  • Lien vers le site https://www.cameroon-web.com (nouvel onglet)
  • Lien vers une page contact.html située dans le même dossier
  • Lien vers une ancre interne nommée #section2 dans la même page
  • Lien d'envoi d'email à support@exemple.cm

Question 6 – Création d'un menu de navigation (6 pts)

Écrivez le code d'une barre de navigation (balises <nav> et <a>) contenant trois liens : "Accueil" (lien #), "Services" (services.html), "Contact" (contact.html). Le lien "Accueil" doit ouvrir dans le même onglet, les autres dans un nouvel onglet.

Partie 4 : Images (12 pts)

Question 7 – Attributs d'image (6 pts)

Expliquez l'utilité des attributs suivants pour la balise <img> : src, alt, width, height. Pourquoi l'attribut alt est-il essentiel pour l'accessibilité et le SEO ?

Question 8 – Balise figure et figcaption (6 pts)

Écrivez le code HTML pour insérer une image nommée yaounde-market.jpg présente dans un dossier images/ avec une largeur de 300 pixels. Ajoutez une légende : "Marché central de Yaoundé". Utilisez les balises sémantiques <figure> et <figcaption>.
