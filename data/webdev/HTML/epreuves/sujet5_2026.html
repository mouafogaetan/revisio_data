Listes HTML : ul, ol, dl et imbrication

Durée : 1h  |  Barème /20  |  Listes ordonnées, non ordonnées, de définition, imbrication

📌 Exercice 1 – QCM (4 points)

4 pts

1.1 Quelle balise permet de créer une liste non ordonnée (à puces) ?

1.2 Dans une liste de définition, quelle balise contient le terme à définir ?

1.3 Pour créer une liste ordonnée avec des chiffres romains (I, II, III...), quel attribut utilise-t-on ?

1.4 Lequel de ces codes crée correctement une liste imbriquée ?

🔍 Exercice 2 – Analyse et correction (5 pts)

5 pts

Analysez le code HTML suivant. Identifiez les erreurs et proposez une version corrigée.

<ul> <p>Liste de courses</p> <li>Fruits</li> <ul> <li>Mangues</li> <li>Bananes</li> </ul> <li>Légumes</li> <li>Poisson</li> </ul>
Indice : une liste ne peut contenir que des éléments <li>. Une sous-liste doit être placée à l'intérieur d'un <li>.

📖 Exercice 3 – Liste de définition (3 pts)

3 pts

Créez une liste de définition <dl> pour expliquer les balises suivantes : <ul>, <ol>, <li> et <dl>.

Pour chaque terme, fournissez une brève définition et un exemple d'utilisation.

🌐 Exercice 5 – Mini-site de recettes (4 pts)

4 pts

Vous devez créer la structure HTML d'une page "Recettes du Cameroun". La page doit contenir :

  • Un titre principal (h1) "Recettes traditionnelles du Cameroun"
  • Une liste non ordonnée des catégories de recettes (Plats principaux, Entrées, Desserts, Boissons)
  • Pour la catégorie "Plats principaux", une liste ordonnée de 3 recettes (Ndolé, Poulet DG, Eru)
  • Une liste de définition avec 3 termes culinaires (ex: "Ndolé", "Poulet DG", "Eru") et leur brève description
  • Un lien vers une page "À propos" (href="apropos.html") et un lien vers "Contact" (href="contact.html")

📝 Éléments de correction

  • QCM : 1.1 <ul> / 1.2 <dt> / 1.3 type="I" / 1.4 La première option.
  • Erreurs code : <p> ne doit pas être enfant direct de <ul> ; la sous-liste doit être dans un <li>.
  • Liste imbriquée : La sous-liste <ul> ou <ol> doit être placée à l'intérieur d'un <li> parent.
  • Liste de définition : <dl> contient des couples <dt> (terme) et <dd> (définition).
  • Organigramme Mermaid : Traduire chaque niveau par une liste imbriquée.
