📄 La balise <form> : le conteneur

Tout formulaire commence par <form>. Deux attributs clés :

  • action : URL où les données sont envoyées.
  • method : "GET" (données dans l'URL) ou "POST" (données cachées).
<form action="/inscription" method="post">
  ... tous les champs ici ...
</form>
🔍 Astuce : si action est vide, le formulaire s'envoie vers la même page.

✍️ Les champs <input> essentiels

La balise <input> avec l'attribut type définit la nature du champ.

type="text" → texte simple
type="email" → validation email basique
type="password" → masque les caractères
type="number" → chiffres avec flèches
type="date" → sélecteur calendrier

📌 Exemple :

<input type="email" placeholder="exemple@cameroun.com" required>

📋 Zone de texte & liste déroulante

📎 <textarea> : plusieurs lignes

Pour les commentaires ou messages longs.

<textarea rows="4" placeholder="Votre message..."></textarea>

📌 <select> et <option> : menu déroulant

Idéal pour choix multiples ou unique.

<select name="ville">
  <option value="douala">Douala</option>
  <option value="yaounde">Yaoundé</option>
</select>

⚡ Boutons et attributs indispensables

Boutons :

  • <input type="submit"> → Envoie le formulaire
  • <input type="reset"> → Réinitialise les champs
  • <button type="button"> → Action JS personnalisée

Attributs utiles :

  • required : champ obligatoire
  • placeholder : texte d'exemple
  • value : valeur pré-remplie
💡 Exemple : <input type="text" placeholder="Votre nom" required value="Jean">
