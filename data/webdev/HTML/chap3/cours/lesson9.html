📊 Les tableaux en HTML

Organiser et présenter des données sous forme de tableau structuré

📋 Leçon 9 ⏱️ Durée : 20 min 🎯 Niveau : Débutant

🎯 Objectifs de la leçon

  • ✓ Créer des tableaux avec les balises <table>, <tr>, <th> et <td>.
  • ✓ Fusionner des cellules avec les attributs colspan et rowspan.
  • ✓ Structurer un tableau avec <thead>, <tbody> et <tfoot>.
  • ✓ Appliquer les bonnes pratiques : accessibilité, en-têtes clairs et respect des normes.

📐 Structure de base d'un tableau

Un tableau HTML se construit avec plusieurs balises imbriquées :

  • <table> : conteneur principal
  • <tr> : ligne du tableau (Table Row)
  • <th> : cellule d'en-tête (Table Header) – gras et centré par défaut
  • <td> : cellule de données (Table Data)

📋 Exemple simple :

<table border="1">
  <tr>
    <th>Nom</th>
    <th>Âge</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>

💡 À retenir : L'attribut border n'est plus recommandé en HTML5. On préfère utiliser CSS pour les bordures. Mais il permet de visualiser facilement les contours.

🔗 Fusionner des cellules

📘 colspan : fusion horizontale

La cellule s'étend sur plusieurs colonnes.

<td colspan="2">texte sur 2 colonnes</td>

📘 rowspan : fusion verticale

La cellule s'étend sur plusieurs lignes.

<td rowspan="3">texte sur 3 lignes</td>

🖼️ Exemple : emploi du temps simplifié

HeureLundiMardi
8h-10hMathsFrançais

<tr>
  <td>8h-10h</td>
  <td rowspan="2">Maths</td>
  <td>Français</td>
</tr>
<tr>
  <td>10h-12h</td>
  <td>Histoire</td>
</tr>

🏗️ Structure avancée : thead, tbody, tfoot

Pour améliorer la sémantique et l'accessibilité, on regroupe les lignes :

  • <thead> : en-tête du tableau (généralement la première ligne).
  • <tbody> : corps principal du tableau (une ou plusieurs lignes).
  • <tfoot> : pied de tableau (totaux, récapitulatifs).
<table>
  <thead>
    <tr><th>Produit</th><th>Prix</th></tr>
  </thead>
  <tbody>
    <tr><td>Stylo</td><td>2€</td></tr>
    <tr><td>Cahier</td><td>3€</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>5€</td></tr>
  </tfoot>
</table>
✅ Bon à savoir : Le navigateur peut répéter l'en-tête lors de l'impression si le tableau est long. C'est aussi utile pour les lecteurs d'écran.

♿ Accessibilité des tableaux

  • Utiliser <th> pour les en-têtes (pas simplement du texte en gras).
  • Ajouter l'attribut scope="col" ou scope="row" pour préciser la portée des en-têtes (aides techniques).
  • Ne jamais utiliser les tableaux pour la mise en page (uniquement pour des données tabulaires).
  • Donner un titre au tableau avec la balise <caption> (située juste après <table>).
<table>
  <caption>Liste des étudiants</caption>
  <thead>
    <tr>
      <th scope="col">Nom</th>
      <th scope="col">Note</th>
    </tr>
  </thead>
  <tbody>...</tbody>
</table>

📝 Récapitulatif

🧱 Éléments clés

  • <table> + <tr> + <td>
  • <th> pour les en-têtes
  • colspan et rowspan pour les fusions
  • <thead>, <tbody>, <tfoot> pour la structure
  • <caption> pour le titre

✨ Bonnes pratiques

  • Tables réservées aux données tabulaires (pas de mise en page).
  • Attribut scope pour l'accessibilité.
  • CSS pour le style (bordures, couleurs).
  • Toujours valider son HTML avec le validateur W3C.
