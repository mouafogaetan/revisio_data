Margin vs Padding

margin: espace EXTERNE
Élément avec marge de 20px

👉 La marge repousse les autres éléments. Elle est transparente et invisible.

padding: espace INTERNE
Élément avec padding de 20px

👉 Le padding agrandit l'intérieur, pousse le contenu vers le centre sans affecter les voisins.

Astuce : Utilisez la console navigateur (inspecter) pour voir visuellement ces zones !

Block vs Inline : quel comportement ?

Le modèle de boîte change selon la nature de l'élément.

Élément BLOC (div, p, h1)

  • Occupe 100% de la largeur parent
  • Respecte width, height, margin, padding totalement
  • Passe à la ligne automatiquement

Élément INLINE (span, a, strong)

  • Largeur seulement du contenu
  • margin-top/bottom ignorées, seulement left/right fonctionnent
  • Padding fonctionne mais peut chevaucher
  • Pas de retour à la ligne

Raccourcis pratiques & largeur réelle

🔹 Raccourcis margin & padding (top right bottom left) :

margin: 10px;           /* tous les côtés */
margin: 10px 20px;      /* haut/bas 10px, droite/gauche 20px */
margin: 5px 10px 15px 20px; /* top, right, bottom, left */

🔹 Largeur totale d’un élément (sans box-sizing) :

largeur totale = width + padding(left+right) + border(left+right) + margin(left+right)

Exemple : width:200px, padding:10px, border:5px → largeur affichée = 230px (200+20+10).

À retenir : Pour maîtriser la taille finale, on peut utiliser box-sizing: border-box (vu leçon suivante).
