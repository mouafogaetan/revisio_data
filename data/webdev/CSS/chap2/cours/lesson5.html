💡 À la fin de ce cours, vous saurez ajouter des effets dynamiques et du contenu décoratif sans toucher au HTML.

📌 Pseudo-classes d'interaction

Elles s'appliquent selon l'état d'un élément : lien non visité, survol, focus clavier…

/* États des liens */
a:link { color: #1e3a8a; }        /* lien non visité */
a:visited { color: #6b21a5; }     /* déjà visité */
a:hover { text-decoration: underline; background: #eef2ff; }
button:focus, input:focus { outline: 3px solid #facc15; }
a:active { color: #b91c1c; }

✨ Exemple interactif :

🚀 Lien survolable / actif

Essayez : survolez le lien, cliquez (active), ou mettez le focus sur le bouton avec Tab.

🧩 Structurelles : cibler selon le rang

li:first-child { font-weight: bold; }
li:last-child { border-bottom: none; }
tr:nth-child(even) { background: #f1f5f9; }
p:nth-child(3n+1) { color: #0f3b5c; }

Utile pour les listes, tableaux, galeries, sans ajouter de classes.

🎨 Liste d'élèves (exemple dynamique) :

  • Fatima (1er enfant)
  • Ibrahim
  • Claire
  • Michael
  • Sophie (dernier enfant)

Premier élément en gras, dernier sans bordure (via CSS).

:nth-child(an+b) permet de créer des motifs (2n, 3n+1, odd/even).

✨ Pseudo-éléments : créer du virtuel

Ils ajoutent du style ou du contenu sans modifier le HTML. ::before et ::after sont très utiles.

h2::before {
  content: "🎯 ";
  font-size: 1.2em;
}
blockquote::first-line {
  font-weight: bold;
  color: #334155;
}
p::first-letter {
  font-size: 2rem;
  font-weight: bold;
  color: #b45309;
  margin-right: 4px;
}

Exemple : Le CSS donne vie au texte, la première lettre est spéciale.
