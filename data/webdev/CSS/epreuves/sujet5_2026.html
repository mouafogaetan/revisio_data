Partie 1

Positionnement & contexte (18 pts)

1. (6 pts) Différence absolute vs relative

3. (6 pts) sticky & fixed – cas d'usage

Partie 2

Maîtrise de Flexbox (20 pts)

🎨 Exercice interactif (8 pts) :

On souhaite réaliser une barre de navigation responsive avec Flexbox. Écrivez le CSS pour que :

  • Les éléments soient alignés horizontalement sur desktop, avec espace entre eux (space-between)
  • Sur mobile (max-width: 640px), la colonne se transforme en disposition verticale (column)
  • Ajoutez un effet de survol (background léger)

2. (6 pts) Propriétés flex enfants

Expliquez la différence entre flex-grow, flex-shrink et flex-basis. Que fait la règle flex: 1; ?

3. (6 pts) Alignement avec align-items et justify-content

Dans un conteneur flex de hauteur 300px, comment centrer parfaitement un élément horizontalement et verticalement ? Écrivez les deux propriétés principales.

Partie 3

CSS Grid & Media Queries (16 pts)

1. (6 pts) Grille responsive avec grid-template-columns

Créez une grille de 3 colonnes égales sur desktop, qui se transforme en 2 colonnes sur tablette (max-width: 768px) et 1 colonne sur mobile (max-width: 480px). Utilisez grid-template-columns et media queries.

2. (5 pts) Zones nommées (grid-template-areas)

Définissez un layout classique : header, sidebar, contenu, footer en utilisant grid-template-areas. Donnez le code CSS minimal.

3. (5 pts) Approche mobile-first vs desktop-first

Citez un avantage de l'approche mobile-first et écrivez un exemple de media query correspondant.
