2 Mise en pratique – Boîtes & positionnement

12 points
<div class="parent">
  <div class="box">Boîte 1</div>
  <div class="box special">Boîte 2</div>
</div>

En vous basant sur le code HTML ci-dessus, écrivez le CSS nécessaire pour :

  1. Définir une largeur de 300px, une hauteur de 150px, une marge extérieure de 20px et un padding intérieur de 15px pour toutes les boîtes.
  2. Appliquer un fond gris clair et une bordure solide noire de 2px.
  3. Positionner la boîte avec la classe "special" en absolute à 30px du haut et 50px de la gauche par rapport au parent qui doit être positionné en relative.
  4. Ajouter une ombre portée (box-shadow) légère sur la boîte special.

3 Flexbox – Layout responsive

8 points

Créez une barre de navigation horizontale responsive avec Flexbox. Les éléments sont : Accueil, Services, À propos, Contact.

  • Sur desktop : les liens sont alignés horizontalement avec un espace égal entre eux.
  • Sur mobile (max-width: 640px) : la navigation passe en colonne, centrée.
  • Ajoutez un effet au survol (couleur de fond et transition).

📦 box-sizing : la différence

content-box (défaut) : largeur = contenu seulement

border-box : largeur = contenu + padding + bordure

✏️ À retenir : * { box-sizing: border-box; } rend les layouts prévisibles.
