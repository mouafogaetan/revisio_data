🌍 CSS : Définition & mission

CSS (Cascading Style Sheets) est un langage qui permet de contrôler l’apparence des pages web (couleurs, polices, mise en page, animations). Sans CSS, le web serait constitué uniquement de texte brut sur fond blanc, comme dans les années 90.

🔹 HTML = Structure (squelette)
🔹 CSS = Style (habillage, peau)
Exemple concret : un même HTML peut devenir tantôt un blog élégant, tantôt un dashboard moderne uniquement en changeant le CSS.

🎨 Avantages de la séparation :

  • Maintenance facilitée
  • Même design sur plusieurs pages
  • Accessibilité et temps de chargement optimisés

📜 Évolution du CSS : une histoire de style

CSS1 (1996)

Fonctionnalités basiques : couleurs, polices, marges, alignement des textes. Première version qui pose les bases.

CSS2 (1998)

Ajout du positionnement (relatif, absolu), des media types (imprimante, écran), et du concept de z-index. Meilleur contrôle des layouts.

CSS3 (modulaire à partir de 2011)

Révolution ! Flexbox, Grid, animations, bordures arrondies, ombres, transitions, media queries (responsive). Modules indépendants qui évoluent sans cesse.

💡 Aujourd'hui, on parle souvent de "CSS moderne" avec des fonctionnalités puissantes comme les variables CSS, les sous-grilles, etc. (CSS3 et au-delà)

✏️ Aperçu de la syntaxe CSS

Une règle CSS se compose d’un sélecteur et d’un bloc de déclarations { propriété: valeur; }

/* Exemple: tous les paragraphes deviennent bleus */
p {
  color: blue;
  font-size: 16px;
}

La puissance du CSS vient des sélecteurs (balises, classes, id) et de la cascade. Nous approfondirons dans les prochaines leçons.

✅ À retenir :

CSS = règles, sélecteur + propriétés. Le navigateur lit le HTML et applique les styles correspondants.
