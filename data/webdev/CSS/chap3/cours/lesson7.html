🌈 Systèmes de couleurs

En CSS, on peut définir une couleur de plusieurs manières :

Mots-clés
red, blue, transparent, currentColor...
Hexadécimal
#FF5733, #0f3b5f
RGB / RGBA
rgb(255, 87, 51) / rgba(0,0,0,0.5)
HSL / HSLA
hsl(9, 100%, 60%) / hsla(200, 80%, 50%, 0.8)
/* Exemples de déclarations */
h1 {
  color: #2c3e66;        /* hexa */
  background: rgba(44, 62, 102, 0.1);
}
.btn {
  background: hsl(210, 80%, 50%);
  border: 2px solid hsl(210, 80%, 35%);
}
✨ Astuce : RGBA et HSLA permettent la transparence (alpha channel) pour des superpositions élégantes.

📏 Unités en CSS : absolues & relatives

Unités absolues

Elles sont fixes et ne changent pas selon l'écran ou la taille de police parente : px, cm, mm, in, pt, pc.

🔹 px (pixel) : l'unité la plus courante pour les bordures, ombres, petites marges.

font-size: 16px; width: 200px;

Unités relatives

Elles s'adaptent au contexte : taille de la police, de la fenêtre, ou de l'élément parent.

  • em : relatif à la taille de police de l'élément parent (ou lui-même).
  • rem : relatif à la racine (<html>), idéal pour l'accessibilité.
  • % : relatif au parent (width, height, font-size indirect).
  • vw / vh : 1% de la largeur / hauteur de la fenêtre (viewport).
/* Exemple pratique */
body { font-size: 18px; }
h1 { font-size: 2rem; }   /* = 36px si racine 18px */
.container { width: 80%; margin: 0 auto; }
.hero { height: 70vh; }   /* 70% de la hauteur de l'écran */

⚖️ Choisir la bonne unité

🎯 Pour les polices : rem (accessibilité, zoom navigateur). Éviter les px sauf cas particuliers.

📦 Pour les largeurs de conteneurs : % , vw, flex/grid pour le responsive.

🖼️ Pour les marges internes/externes : rem, em, parfois px.

📱 Media queries : px reste standard pour les breakpoints (ex: @media (max-width: 768px)).

🧠 Règle d'or : Pour un design adaptable, préférez les unités relatives (rem, %, vw). Les pixels (px) sont parfaits pour les détails précis (bords, petits écarts).

Largeur = 75% du parent

💡 Les unités vw / vh permettent des sections pleine largeur/hauteur d'écran.
