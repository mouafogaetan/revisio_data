📌 Placement individuel : grid-column / grid-row

Par défaut, les éléments remplissent chaque cellule. Mais on peut étendre un élément sur plusieurs colonnes ou lignes.

/* Étendre sur 2 colonnes */ .item1 { grid-column: 1 / 3; /* démarre à ligne 1, finit à ligne 3 */ } /* ou utiliser span */ .item2 { grid-row: span 2; }
💡 Rappel : Les lignes de grille sont numérotées à partir de 1. grid-column: 2 / 4 signifie de la ligne 2 à 4.

🗺️ Les zones nommées : grid-template-areas

Rendez votre code ultra lisible en donnant des noms aux régions de votre grille.

.grid-container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-areas: "header header header" "nav main aside" "footer footer footer"; } .header { grid-area: header; } .nav { grid-area: nav; } .main { grid-area: main; } .aside { grid-area: aside; } .footer { grid-area: footer; }

⚙️ Alignement et espacements dans Grid

📏 gap (row-gap / column-gap)

Définit l'espace entre les lignes/colonnes. gap: 1rem;

🎯 justify-items / align-items

Alignent les éléments à l'intérieur de leurs cellules (start, end, center, stretch).

🧭 justify-content / align-content

Quand la grille est plus petite que le conteneur, ces propriétés répartissent l'espace supplémentaire.

📐 place-items / place-content

Raccourcis pour (align + justify). Ex: place-items: center;

🏆 Mise en pratique & synthèse

💻 À vous de jouer : construisons une grille de type "galerie" avec zones nommées

/* Exemple : Page d'accueil */ .grid-galerie { display: grid; grid-template-columns: repeat(4, 1fr); grid-template-areas: "titre titre titre titre" "photo1 photo2 photo3 sidebar" "photo4 photo5 photo6 sidebar" "pied pied pied pied"; gap: 1rem; }

📌 Rappel des bonnes pratiques :

  • grid-column: span 2 pour fusionner des colonnes facilement.
  • Utilisez les zones nommées pour des layouts complexes (dashboard, portfolio).
  • Combinez justify-content: center et align-content: center pour centrer la grille entière.
  • gap évite les margins complexes.
