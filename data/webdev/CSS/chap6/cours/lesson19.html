📐 Les bases : @keyframes

Une animation CSS se définit avec la règle @keyframes où l'on décrit les étapes (0% à 100% ou from/to).

@keyframes monAnimation {
  0% { opacity: 0; transform: scale(0.8); }
  50% { opacity: 0.6; transform: scale(1.1); background: orange; }
  100% { opacity: 1; transform: scale(1); background: green; }
}

Pour appliquer l’animation à un élément :

.ma-boite {
  animation-name: monAnimation;
  animation-duration: 2s;
  animation-iteration-count: 3;
}

🧠 Astuce: On peut utiliser infinite pour une boucle sans fin. Très pratique pour des indicateurs de chargement.

⚙️ Les propriétés essentielles

🔁 animation-iteration-count

Nombre de répétitions : 1, 3, infinite

BOOM!

🔄 animation-direction

normal, reverse, alternate, alternate-reverse

↺
✨ animation-direction: alternate – l'animation va et vient, idéal pour un effet oscillant.

🎭 animation-fill-mode : maîtriser l'état avant/après

Par défaut, après une animation, l'élément revient à son style initial. fill-mode permet de figer l’état final (forwards) ou d’appliquer le premier keyframe (backwards).

🎯

forwards : garde la dernière keyframe

✨

backwards : applique le style 0% avant le début

📈 animation-timing-function : le rythme de l'animation

Choisissez la courbe d'accélération: ease, linear, ease-in, ease-out, cubic-bezier()

Cliquez sur chaque bouton pour lancer un mouvement horizontal avec différentes fonctions de timing.

🚀 Exemple complet : animation "couleur et rotation"

Clique-moi

Cette boîte utilise @keyframes avec plusieurs propriétés : background-color, scale, rotation, et elle s'anime en boucle infinie avec direction alternate et timing ease-in-out.

@keyframes magicMove {
0% { transform: scale(1) rotate(0deg); background: #6366f1; }
50% { transform: scale(1.2) rotate(10deg); background: #f43f5e; }
100% { transform: scale(1) rotate(0deg); background: #10b981; }
}

📌 Récapitulatif & bonnes pratiques

  • 🟢 @keyframes définit la séquence d’animation (étapes %).
  • 🟢 Les propriétés animation-* permettent un contrôle riche: durée, itération, délai, direction.
  • 🟢 animation-fill-mode: forwards utile pour garder l’état final.
  • 🟢 animation-timing-function ajuste le mouvement (cubic-bezier).
  • 🟢 Toujours penser aux performances: préférer transform et opacity.
