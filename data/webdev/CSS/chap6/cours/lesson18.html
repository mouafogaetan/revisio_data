1. Les transitions : fluidité en douceur

Les transitions permettent de changer progressivement la valeur d'une propriété CSS sur une durée définie. Syntaxe simplifiée :

/* Propriétés essentielles */
selector {
  transition-property: all;  /* ou une propriété spécifique */
  transition-duration: 0.3s;
  transition-timing-function: ease;
  transition-delay: 0s;
}
/* Raccourci */
selector {
  transition: transform 0.4s cubic-bezier(0.2, 0.9, 0.4, 1.1);
}

2. Transformations 2D : modifiez la forme/l'emplacement

La propriété transform accepte plusieurs fonctions :

translate(x,y)
Déplacer
rotate(deg)
Tourner
scale(x,y)
Agrandir/rétrécir
skew(deg)
Incliner
Test

Cliquez sur les boutons pour voir la transformation en direct (sans transition pour l'instant, ajoutons la transition sur la boîte)

.ma-boite {
  transform: rotate(10deg) scale(1.1);
  transition: transform 0.2s ease;
}

💡 Bonnes pratiques :

  • Privilégiez transform et opacity pour des animations 60fps (performances).
  • Évitez les transitions sur width/height/top/left si possible.
  • Utilisez will-change pour les animations complexes.
🖌️

Exemples concrets & exercice rapide

Créez un bouton qui s'agrandit et change de couleur au survol, ou une carte qui s'élève avec une ombre. Voici un mix :

Animer
🚀 hover lévitation 
.carte:hover {
  transform: translateY(-8px) scale(1.02);
  transition: transform 0.2s cubic-bezier(0.2, 0.9, 0.6, 1.1);
}

🧪 À toi de jouer : modifie le fichier CSS d'un de tes projets pour ajouter une transition subtile sur les boutons ou les cartes.

Récapitulons !

✅ transition : crée des animations basées sur les changements d'état (hover, active, class ajoutée).

✅ transform : modifie géométrie (position, rotation, échelle, inclinaison).

✅ Toujours coupler transition: transform 0.3s ease pour des animations fluides.

✅ Utiliser les transformations sur les pseudo-classes :hover, :focus.
