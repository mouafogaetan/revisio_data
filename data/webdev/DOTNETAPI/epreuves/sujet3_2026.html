Partie 2 : Validation des modèles et Data Annotations (14 pts)

Question 4 – Attributs de validation (8 pts)

On vous donne le DTO suivant pour la création d'un produit :

public class CreateProductDto
{
    public string Name { get; set; }
    public decimal Price { get; set; }
    public string Category { get; set; }
    public DateTime ReleaseDate { get; set; }
}

Ajoutez les attributs de validation appropriés (Data Annotations) pour imposer :

  • Name : requis, longueur maximale 100 caractères.
  • Price : compris entre 0 et 10000, avec 2 décimales.
  • Category : requis, doit faire partie des valeurs ["Electronics","Clothing","Books"] (utilisez RegularExpression ou validation personnalisée).
  • ReleaseDate : ne peut pas être une date future.
