❤️ Page des favoris (Wishlist)

Les favoris sont stockés dans un store Zustand avec persistance localStorage. La page /favoris liste tous les produits aimés par l'utilisateur.

// store/favoriteStore.js
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

const useFavoriteStore = create(
  persist(
    (set, get) => ({
      favorites: [],
      toggleFavorite: (product) => {
        const exists = get().favorites.some(p => p.id === product.id);
        const newFavorites = exists 
          ? get().favorites.filter(p => p.id !== product.id)
          : [...get().favorites, product];
        set({ favorites: newFavorites });
        toast.success(exists ? 'Retiré des favoris' : 'Ajouté aux favoris');
      },
      isFavorite: (id) => get().favorites.some(p => p.id === id),
    }),
    { name: 'wishlist-storage' }
  )
);

📌 Composant FavoritesPage : utilise `useFavoriteStore().favorites` et affiche les produits en grille. Si la liste est vide, message "Aucun favori".

⭐ Gestion des avis clients

L'utilisateur peut consulter, modifier ou supprimer ses avis depuis son espace compte. Chaque avis contient une note (1 à 5 étoiles), un commentaire et une image facultative.

// hooks/useUserReviews.js
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';

export const useUserReviews = () => {
  return useQuery({
    queryKey: ['user-reviews'],
    queryFn: () => api.get('/reviews/me'),
  });
};

export const useDeleteReview = () => {
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (reviewId) => api.delete(`/reviews/${reviewId}`),
    onSuccess: () => {
      queryClient.invalidateQueries(['user-reviews']);
      toast.success('Avis supprimé');
    },
  });
};

export const useUpdateReview = () => {
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: ({ reviewId, data }) => api.put(`/reviews/${reviewId}`, data),
    onSuccess: () => queryClient.invalidateQueries(['user-reviews']),
  });
};

📝 Affichage :

Chaque avis affiche le produit concerné, la note (avec étoiles), le commentaire, la date. Boutons "Modifier" (ouvre un modal) et "Supprimer" (avec confirmation).

🔔 Alertes promo sur favoris

Quand un produit favori voit son prix baisser ou entre en promotion, l'utilisateur reçoit une notification (toast + badge dans l'interface).

// Composant PromotionAlert
import { useEffect } from 'react';
import { useFavoriteStore } from '../stores/favoriteStore';
import { useProducts } from '../hooks/useProducts';

export function PromotionAlert() {
  const { favorites } = useFavoriteStore();
  const { data: products } = useProducts({ ids: favorites.map(f => f.id) });
  
  useEffect(() => {
    const discountedFavs = products?.filter(p => p.discountPercent > 0);
    if (discountedFavs?.length) {
      discountedFavs.forEach(product => {
        toast.info(`🔥 ${product.name} est en promo à ${product.discountedPrice} FCFA !`, {
          duration: 10000,
          action: { label: 'Voir', onClick: () => navigate(`/produit/${product.id}`) }
        });
      });
    }
  }, [products]);
  
  return discountedFavs?.length > 0 && (
    <div className="bg-red-100 p-2 rounded-lg text-sm">
      {discountedFavs.length} produit(s) favori(s) en promotion !
    </div>
  );
}

⚙️ Mécanisme :

Au chargement de la page d'accueil ou du compte, on compare les prix actuels des favoris avec leur prix original (ou ancien prix). Une alerte toast est déclenchée.
