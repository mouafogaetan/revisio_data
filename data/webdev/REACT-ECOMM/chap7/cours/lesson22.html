Leçon 22 : Page d'accueil

🏠 Page d'accueil e-commerce

Hero banner, produits vedettes, catégories et promotions. Créez une première impression irrésistible.

🎯 UX engageante ⚡ Composants réutilisables 📱 Responsive

🎯 Objectifs de la leçon

1️⃣

Hero banner avec CTA

Bannière principale et appel à l’action percutant.

2️⃣

Produits en vedette

Grille ou carrousel des meilleures ventes.

3️⃣

Catégories + promotions

Mise en avant des offres et rubriques.

💡 Une bonne page d’accueil convertit 30% de visiteurs en acheteurs potentiels.

🎨 Hero Banner

Composant essentiel : une grande image ou un dégradé, un texte accrocheur, un bouton (CTA).

🔥 Summer Sale

Jusqu’à -40% sur les produits sélectionnés

Le CTA (Call To Action) guide l'utilisateur vers le catalogue.

Utilisez des badges “-20%”, “Livraison offerte” ou “Nouveauté”.

🧩 Composant HomePage (extrait)

import Hero from "../components/Hero";
import FeaturedProducts from "../components/FeaturedProducts";
import Categories from "../components/Categories";
import PromoBanner from "../components/PromoBanner";

export default function HomePage() {
  return (
    <div>
      <Hero title="Summer Sale" ctaText="Découvrir" />
      <FeaturedProducts />
      <Categories />
      <PromoBanner />
    </div>
  );
}

Approche modulaire : chaque section est un composant réutilisable.

📱 Adaptation mobile

Tailwind permet d’adapter la grille : grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4.

  • Le Hero banner réduit sa hauteur sur mobile
  • Les catégories deviennent une ligne scrollable horizontalement
  • Images responsives avec w-full h-auto
💡 Toujours tester sur mobile avec les devtools Chrome.
