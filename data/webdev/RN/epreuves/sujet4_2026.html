Partie 2 : Vrai ou Faux (10 pts)

En React Native, la propriété position: 'fixed' fonctionne exactement comme sur le web.
flexWrap: 'wrap' permet aux éléments d'une ligne flex de passer à la ligne suivante.
Les valeurs de width et height en pourcentage s'expriment avec des chaînes comme '50%'.
useWindowDimensions() ne se met pas à jour lors du changement d'orientation de l'appareil.
Le style zIndex n'existe pas en React Native, on utilise elevation pour Android uniquement.

Partie 3 : Mise en pratique (70 pts)

Exercice 1 - 25 pts Layout avec Flexbox

Créez un composant React Native nommé HeaderContentFooter qui reproduit la structure suivante :

  • Un en-tête (Header) en haut : hauteur 80px, couleur bleue, texte centré "Mon Application"
  • Une zone de contenu (Content) qui prend tout l'espace restant et affiche une carte simple avec du texte
  • Un pied de page (Footer) en bas : hauteur 60px, couleur grise, texte centré "© 2025"

Contraintes : Utiliser une View principale avec flex: 1, flexDirection: 'column'. Le Content doit avoir flex: 1. Fournissez le code complet avec StyleSheet.create().

📝 Exemple de structure attendue :

<View style={styles.container}>
  <View style={styles.header}><Text>Mon App</Text></View>
  <View style={styles.content}><Text>Contenu</Text></View>
  <View style={styles.footer}><Text>Footer</Text></View>
</View>
Exercice 2 - 25 pts Adaptation responsive

Utilisez useWindowDimensions pour créer un composant ResponsiveCard qui :

  • Affiche une carte (View avec padding, borderRadius, ombre)
  • Si la largeur de l'écran > 600px, la carte occupe 50% de la largeur et s'affiche en ligne avec une autre carte
  • Si la largeur ≤ 600px, la carte occupe 90% de la largeur et s'affiche en colonne
  • Le texte doit s'adapter : taille de police 20px sur grand écran, 16px sur petit écran

🎯 Indice :

const { width } = useWindowDimensions(); const isLarge = width > 600;
Exercice 3 - 20 pts Position absolute & adaptation iOS/Android

Créez un composant FloatingButton avec un bouton flottant en bas à droite (position absolute, margin 20px).

  • Sur iOS, le bouton doit avoir une couleur bleue et une opacité au toucher
  • Sur Android, le bouton doit avoir une couleur indigo et un effet d'élévation (elevation: 5)
  • Utilisez Platform.select() pour différencier les styles
  • Ajoutez un message "Cliqué !" dans une alerte lors du clic (simulez avec Alert.alert sans l'importer réellement)
// Extrait attendu
const styles = StyleSheet.create({
  button: {
    position: 'absolute',
    bottom: 20,
    right: 20,
    ...Platform.select({
      ios: { backgroundColor: '#3b82f6', opacity: 0.9 },
      android: { backgroundColor: '#4f46e5', elevation: 5 }
    })
  }
});
