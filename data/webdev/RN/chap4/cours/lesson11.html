En React Native, tout composant View est un conteneur flex avec flexDirection: 'column'. Pour aligner horizontalement, il faut explicitement définir flexDirection: 'row'.

🧭 flexDirection

Définit l'axe principal du conteneur :

  • 'column' (défaut) : disposition verticale
  • 'row' : disposition horizontale
  • 'column-reverse' : verticale inversée
  • 'row-reverse' : horizontale inversée
<View style={{ flexDirection: 'row' }}>
  <Text>A</Text>
  <Text>B</Text>
  <Text>C</Text>
</View>

📏 justifyContent (axe principal)

Aligne les enfants le long de l'axe principal (défini par flexDirection).

flex-start : début
flex-end : fin
center : centré
space-between : espace égal entre
space-around : espace autour
space-evenly : espace uniforme
justifyContent: 'space-between'

🎯 alignItems (axe transversal)

Aligne les enfants sur l'axe transversal (perpendiculaire à flexDirection).

  • 'stretch' (défaut) : les enfants s'étendent
  • 'flex-start' : alignés au début
  • 'flex-end' : alignés à la fin
  • 'center' : centrés
  • 'baseline' : alignés sur la ligne de base du texte

📦 flexWrap

Permet aux éléments de passer à la ligne suivante si l'espace manque.

  • 'nowrap' (défaut) : pas de retour à la ligne
  • 'wrap' : retour à la ligne
  • 'wrap-reverse' : retour à la ligne inversé
flexDirection: 'row', flexWrap: 'wrap'

⚠️ alignContent s'applique sur l'axe transversal, mais seulement quand il y a plusieurs lignes (flexWrap: 'wrap').

🔄 gap (espacement)

Ajoute un espacement entre les enfants, sans avoir à gérer des marges individuelles.

  • gap: 10 : espacement identique dans les deux axes
  • rowGap: 10 : espacement entre les lignes
  • columnGap: 10 : espacement entre les colonnes
flexDirection: 'row', flexWrap: 'wrap', gap: 12

🏗️ Exemple : Header avec logo et menu

const styles = StyleSheet.create({
  header: {
    flexDirection: 'row',
    justifyContent: 'space-between',
    alignItems: 'center',
    paddingHorizontal: 16,
    height: 60,
    backgroundColor: '#fff'
  },
  logo: { fontSize: 20, fontWeight: 'bold' },
  nav: { flexDirection: 'row', gap: 16 }
});

⚠️ Pièges fréquents à éviter

  • Oublier qu'un View est déjà en flex → pas besoin de display: 'flex'
  • Utiliser justifyContent sur l'axe transversal → besoin de align* à la place
  • Ne pas définir de hauteur/largeur → un flex peut s'écraser
  • Mélanger gap et margin → les marges s'ajoutent au gap, attention au design
  • Oublier flexWrap → les éléments dépassent de l'écran sans retour à la ligne

🧪 Mini exercice

Créez une grille de 3 colonnes avec des cartes :

  • Utilisez flexDirection: 'row' et flexWrap: 'wrap'
  • Chaque carte doit prendre 1/3 de la largeur (width: '33%')
  • Ajoutez un gap de 8px
  • Centre le contenu de chaque carte
🔎 Voir solution 
<View style={{ flexDirection: 'row', flexWrap: 'wrap', gap: 8 }}>
  {cards.map(card => (
    <View key={card.id} style={{ width: '33%', alignItems: 'center', padding: 10 }}>
      <Text>{card.title}</Text>
    </View>
  ))}
</View>

📌 Récapitulatif

🧭 flexDirection – axe principal (colonne par défaut)
📏 justifyContent – alignement sur l'axe principal
🎯 alignItems – alignement sur l'axe transversal
📦 flexWrap + gap – gestion des lignes multiples et espacement
