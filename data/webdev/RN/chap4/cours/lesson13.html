📐 Dimensions API vs useWindowDimensions

React Native propose deux façons d'obtenir la taille de l'écran :

// 1. Dimensions (statique - ne réagit pas aux changements d'orientation)
import { Dimensions } from 'react-native';
const { width, height } = Dimensions.get('window');

// 2. useWindowDimensions (reactif - recommandé)
import { useWindowDimensions } from 'react-native';
const { width, height, fontScale, scale } = useWindowDimensions();

// Exemple : adapter la taille d'une carte
const cardWidth = width > 600 ? width * 0.45 : width * 0.9;

✨ useWindowDimensions se met à jour automatiquement lorsque l'utilisateur change l'orientation ou la taille de la fenêtre (multitâche sur iPad).

🎮 Platform.select : un code, deux plateformes

Différences entre iOS et Android (ombres, polices, espacements) :

import { Platform, StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  container: {
    ...Platform.select({
      ios: {
        shadowColor: '#000',
        shadowOffset: { width: 0, height: 2 },
        shadowOpacity: 0.25,
      },
      android: {
        elevation: 4,
      },
      default: { backgroundColor: 'white' } // web ou autre
    }),
    padding: Platform.OS === 'ios' ? 16 : 12,
  },
  text: {
    fontFamily: Platform.OS === 'ios' ? 'Helvetica' : 'Roboto',
  }
});
⚡ Pratique : Platform.select peut aussi retourner des composants différents !

📱 Adaptation selon la largeur (breakpoints)

const { width } = useWindowDimensions();
const isTablet = width >= 768;
const isLandscape = width > height;

// Styles dynamiques
const dynamicStyle = {
  flexDirection: isTablet ? 'row' : 'column',
  fontSize: isTablet ? 24 : 18,
  padding: isTablet ? 24 : 12,
};

// Affichage conditionnel
{isTablet && <Sidebar />}
<MainContent />

💡 Exemple concret : Une liste d'articles s'affiche en grille (2 colonnes) sur tablette, en liste simple sur smartphone.

🔄 Gérer l'orientation : portrait / paysage

import { useWindowDimensions } from 'react-native';

function ResponsiveComponent() {
  const { width, height } = useWindowDimensions();
  const isPortrait = height >= width;

  return (
    <View style={{ flexDirection: isPortrait ? 'column' : 'row' }}>
      <View style={{ flex: isPortrait ? 1 : 2 }} />
      <View style={{ flex: 1 }} />
    </View>
  );
}

On peut aussi verrouiller l'orientation dans app.json avec "orientation": "portrait" si nécessaire.

✨ Bonnes pratiques & astuces

  • Utiliser useWindowDimensions plutôt que Dimensions (réactivité).
  • Créer des constantes de breakpoints dans un fichier constants/theme.js.
  • Pour des projets très complexes, utiliser react-native-responsive-screen ou react-native-size-matters.
  • Tester sur émulateur avec différentes résolutions (Pixel 2, iPhone SE, iPad).
  • Platform.select peut être utilisé pour des valeurs simples ou des composants entiers.
📱 Testez systématiquement en mode paysage et sur tablette !
