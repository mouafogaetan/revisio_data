Localisation & Cartes

Obtenir la position GPS et afficher des cartes interactives

👨‍🏫 Classe Développement Web • Cameroun

🎯 Objectifs du cours

  • ✓ Obtenir la position GPS avec expo-location
  • ✓ Afficher une carte interactive avec react-native-maps
  • ✓ Ajouter des marqueurs personnalisés sur la carte
  • ✓ Gérer les permissions de localisation sur Android et iOS
💡 À la fin, vous saurez créer une application qui affiche la carte avec votre position actuelle !

📱 Pourquoi utiliser la localisation ?

La géolocalisation permet de créer des applications sensibles au contexte de l'utilisateur :

🚕 Applications de VTC (Uber, Heetch)
🍕 Livraison de repas (Dominos, Uber Eats)
🌤️ Météo locale
📱 Réseaux sociaux (partager position)
🗺️ Guides touristiques
📦 Suivi de colis en temps réel
🧭 Votre téléphone peut connaître votre position grâce aux satellites GPS, réseaux Wi-Fi et antennes relais.

📦 Installation (Expo)

Pour un projet Expo, installez les bibliothèques suivantes :

expo install expo-location
expo install react-native-maps

Pour React Native CLI :

npm install react-native-maps
npm install @react-native-community/geolocation
⚠️ Important : Sur Android, ajoutez la permission dans AndroidManifest.xml :
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
Sur iOS, ajoutez NSLocationWhenInUseUsageDescription dans Info.plist.

🔐 Gérer les permissions

Avant d'accéder à la position, il faut demander l'autorisation à l'utilisateur.

import * as Location from 'expo-location';

async function demandePermission() {
  let { status } = await Location.requestForegroundPermissionsAsync();
  if (status !== 'granted') {
    Alert.alert('Permission refusée', 'On ne peut pas avoir votre position');
    return false;
  }
  return true;
}
📌 Bonnes pratiques : Expliquez pourquoi vous avez besoin de la position avant de demander la permission.

📍 Obtenir la position actuelle

async function getCurrentLocation() {
  const permission = await demandePermission();
  if (!permission) return;

  let location = await Location.getCurrentPositionAsync({
    accuracy: Location.Accuracy.High
  });

  console.log(location.coords.latitude);
  console.log(location.coords.longitude);
  
  // Retourne un objet :
  // {
  //   coords: { latitude, longitude, altitude, accuracy, speed },
  //   timestamp: ...
  // }
}

⚡ `accuracy` peut être : High, Balanced, Low, Lowest

🔄 Suivi en temps réel

Pour des applications de traçage (course à pied, livraison) :

const subscription = await Location.watchPositionAsync(
  {
    accuracy: Location.Accuracy.High,
    timeInterval: 5000,   // toutes les 5 secondes
    distanceInterval: 10  // ou tous les 10 mètres
  },
  (location) => {
    console.log('Nouvelle position:', location.coords);
  }
);

// Pour arrêter le suivi :
subscription.remove();
⚠️ Attention à la batterie ! Évitez un suivi trop fréquent si ce n'est pas nécessaire.

🗺️ Carte avec react-native-maps

import MapView from 'react-native-maps';

<MapView
  style={{ flex: 1 }}
  initialRegion={{
    latitude: 3.8480,
    longitude: 11.5021,
    latitudeDelta: 0.0922,
    longitudeDelta: 0.0421,
  }}
/>

Explication :

  • latitudeDelta / longitudeDelta : niveau de zoom (plus petit = zoom avant)
  • La carte occupe tout l'espace avec flex: 1

📍 Afficher "Ma position"

const [userLocation, setUserLocation] = useState(null);

useEffect(() => {
  (async () => {
    let loc = await getCurrentLocation();
    setUserLocation(loc.coords);
  })();
}, []);

<MapView
  showsUserLocation={true}  // point bleu automatique
  followsUserLocation={true} // centre sur l'utilisateur
  initialRegion={userLocation ? {
    latitude: userLocation.latitude,
    longitude: userLocation.longitude,
    latitudeDelta: 0.01,
    longitudeDelta: 0.01
  } : undefined}
>
  {userLocation && <Marker coordinate={userLocation} title="Vous êtes ici" pinColor="red" />}
</MapView>

💡 showsUserLocation affiche un point bleu animé sans code supplémentaire !

🖱️ Gérer les interactions

const onRegionChange = (region) => {
  console.log('Nouvelle région:', region);
};

const onMarkerPress = (markerData) => {
  Alert.alert('Marqueur cliqué', markerData.nativeEvent.title);
};

<MapView
  onRegionChange={onRegionChange}
  onRegionChangeComplete={(region) => setCurrentRegion(region)}
>
  <Marker onPress={onMarkerPress} coordinate={...} title="Info" />
</MapView>
  • onRegionChange : appelé en continu pendant le déplacement
  • onRegionChangeComplete : appelé à la fin du geste (idéal pour charger des lieux)

📱 Exemple complet fonctionnel

function MapScreen() {
  const [location, setLocation] = useState(null);

  useEffect(() => {
    (async () => {
      let { status } = await Location.requestForegroundPermissionsAsync();
      if (status === 'granted') {
        let loc = await Location.getCurrentPositionAsync({});
        setLocation(loc.coords);
      }
    })();
  }, []);

  if (!location) return <Text>Chargement...</Text>;

  return (
    <MapView style={{ flex:1 }} initialRegion={{
      latitude: location.latitude,
      longitude: location.longitude,
      latitudeDelta: 0.01,
      longitudeDelta: 0.01
    }} showsUserLocation>
      <Marker coordinate={location} title="Ma position" pinColor="blue" />
    </MapView>
  );
}

✅ Bonnes pratiques

  • Toujours gérer le cas où l'utilisateur refuse la permission
  • Ne pas demander la position dès l'ouverture de l'app (attendez une interaction)
  • Pour Android, pensez à la permission en arrière-plan si nécessaire (requestBackgroundPermissionsAsync)
  • Utilisez distanceInterval ou timeInterval pour économiser la batterie
  • Affichez un indicateur de chargement pendant la récupération de la position
  • Testez sur un vrai téléphone : l'émulateur peut simuler des positions mais pas toujours précis
