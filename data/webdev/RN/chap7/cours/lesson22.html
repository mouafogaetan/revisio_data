StatusBar : La barre d'état

La StatusBar est cette fine bande en haut de l'écran qui affiche l'heure, le niveau de batterie, la connexion réseau... Avec React Native, on peut la personnaliser pour qu'elle s'intègre parfaitement à notre application.

🎨 Ce qu'on peut modifier

  • Couleur de fond (surtout sur Android)
  • Style du texte : dark-content ou light-content
  • Masquer/afficher la barre d'état
  • Rendre la barre translucide (effet de superposition)

⚙️ Utilisation de StatusBar

Exemple concret dans un composant React Native :

import { StatusBar } from 'react-native';

function MonEcran() {
  return (
    <View style={{ flex: 1 }}>
      {/* Barre d'état avec texte clair et fond bleu */}
      <StatusBar 
        barStyle="light-content" 
        backgroundColor="#2563eb" 
        translucent={false}
      />
      <Text>Mon contenu</Text>
    </View>
  );
}
📱 barStyle : "dark-content" (texte noir) ou "light-content" (texte blanc)
🎨 backgroundColor : fonctionne surtout sur Android
✨ translucent : si true, le contenu passe sous la barre d'état
👻 hidden : StatusBar hidden={true} masque la barre

📱 SafeAreaView : La solution pour les encoches

Depuis l'iPhone X, beaucoup de smartphones ont des encoches (notch) ou des bords arrondis. SafeAreaView évite que votre contenu soit caché sous ces zones.

📱 Contenu

Sans SafeAreaView
⚠️ contenu sous la zone

→
✅ Contenu visible

Avec SafeAreaView
✔️ contenu bien positionné

import { SafeAreaView, Text } from 'react-native';

function App() {
  return (
    <SafeAreaView style={{ flex: 1 }}>
      <Text>Mon contenu est en sécurité !</Text>
    </SafeAreaView>
  );
}

🔄 Platform.select() : Un style différent sur iOS et Android

Chaque plateforme a ses propres règles (ombres, hauteurs de status bar, typographie). React Native fournit l'API Platform.

import { Platform, StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: Platform.OS === 'android' ? StatusBar.currentHeight : 44,
    ...Platform.select({
      ios: { shadowColor: '#000', shadowOffset: { width: 0, height: 2 } },
      android: { elevation: 4 },
    })
  },
  text: {
    fontSize: Platform.select({ ios: 18, android: 16 }),
  }
});
📱 iOS
Ombre, hauteur spécifique
🤖 Android
Elevation, Material Design
💡 Astuce : Utilisez Platform.select() pour les icônes, les polices ou les styles de boutons !

🧩 Mettre le tout ensemble

Exemple d'un écran complet qui s'adapte parfaitement à tous les appareils :

import { StatusBar, SafeAreaView, Platform, View, Text } from 'react-native';

export default function HomeScreen() {
  return (
    <SafeAreaView style={{ flex: 1, backgroundColor: '#f8fafc' }}>
      <StatusBar 
        barStyle={Platform.OS === 'ios' ? 'dark-content' : 'light-content'}
        backgroundColor={Platform.OS === 'android' ? '#1e40af' : undefined}
      />
      <View style={{ padding: 20 }}>
        <Text style={{ fontSize: Platform.select({ ios: 24, android: 22 }) }}>
          Bienvenue sur mon app 📱
        </Text>
      </View>
    </SafeAreaView>
  );
}
✅ Résultat : application professionnelle, sans découpage, avec un rendu natif sur chaque OS !

📌 Bonnes pratiques & erreurs fréquentes

⚠️
Ne pas oublier SafeAreaView – Sur iPhone récent, votre contenu risque d'être mangé par l'encoche.
✔️
Toujours tester sur iOS et Android – Les comportements diffèrent !
✔️
Utiliser StatusBar currentHeight – Sur Android, `StatusBar.currentHeight` donne la hauteur réelle de la barre.
💡
SafeAreaView ne fonctionne que pour les zones de sécurité – Pour des marges personnalisées, combinez avec `useSafeAreaInsets` (react-native-safe-area-context).

🏋️‍♂️ Exercice pratique

À réaliser sur votre projet React Native :

  1. Créez un écran d'accueil avec un fond de couleur dégradé.
  2. Ajustez la StatusBar pour qu'elle ait un fond semi-transparent et du texte clair.
  3. Utilisez SafeAreaView pour éviter que le contenu touche les bords (notch).
  4. Ajoutez un texte qui change de taille selon la plateforme (iOS = 20, Android = 18).

💡 N'oubliez pas d'importer `StatusBar`, `SafeAreaView`, `Platform` depuis 'react-native'.
