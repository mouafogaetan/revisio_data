🎯 Objectifs de la leçon

Comprendre useSharedValue (valeur partagée entre JS et UI thread)
Utiliser useAnimatedStyle pour lier des animations
Créer des animations gestuelles avec GestureDetector
Comparer Reanimated 2 vs Animated API classique
💡 Reanimated 2 exécute les animations sur le thread natif = fluidité parfaite même lors de gestes complexes.

📦 Installation & concepts

npm install react-native-reanimated react-native-gesture-handler

🔑 Les 3 piliers :

  • useSharedValue(initial) : valeur réactive partagée entre threads.
  • useAnimatedStyle(() => ({})) : style animé basé sur les shared values.
  • GestureDetector + Gesture : reconnaît les gestes (pan, tap, pinch).
⚠️ N'oubliez pas d'ajouter babel.config.js avec le plugin 'react-native-reanimated/plugin'.

🎨 Premier exemple : opacité animée

import Animated, { useSharedValue, useAnimatedStyle, withSpring } from 'react-native-reanimated';

const opacity = useSharedValue(0.3);
const animatedStyle = useAnimatedStyle(() => ({
  opacity: opacity.value
}));

// Déclencher une animation
opacity.value = withSpring(1);

<Animated.View style={[styles.box, animatedStyle]} />

✨ withSpring, withTiming, withDecay remplacent les anciennes méthodes.

🖐️ Gestuelle : faire glisser un objet

import { GestureDetector, Gesture } from 'react-native-gesture-handler';
const translateX = useSharedValue(0);
const translateY = useSharedValue(0);

const panGesture = Gesture.Pan()
  .onUpdate((event) => {
    translateX.value = event.translationX;
    translateY.value = event.translationY;
  })
  .onEnd(() => {
    translateX.value = withSpring(0);
    translateY.value = withSpring(0);
  });

<GestureDetector gesture={panGesture}>
  <Animated.View style={[styles.box, animatedStyle]} />
</GestureDetector>
✅ Retour à la position initiale avec effet ressort (spring).

👆 Double tap et zoom

const scale = useSharedValue(1);
const tapGesture = Gesture.Tap()
  .numberOfTaps(2)
  .onEnd(() => {
    scale.value = withSpring(scale.value === 1 ? 1.5 : 1);
  });
const animatedStyle = useAnimatedStyle(() => ({
  transform: [{ scale: scale.value }]
}));

Double tap pour zoomer/dézoomer, typique des applications de visualisation d’images.

+ → reconnaît aussi pinch (pincer) pour zoomer

⚡ Pourquoi Reanimated 2 ?

Thread UI natif
Pas de passage par le bridge JS.
Gestes complexes
Pan, pinch, rotation, fling.
60 FPS garantis
Idéal pour animations lourdes.
Syntaxe déclarative
useAnimatedStyle + worklets.
🚀 L’API Animated classique (timing, spring) reste utile pour des animations simples ; Reanimated 2 est recommandé pour des interactions complexes.

📱 Application : élément suit le doigt

const x = useSharedValue(0);
const y = useSharedValue(0);
const pan = Gesture.Pan().onUpdate((e) => {
  x.value = e.translationX;
  y.value = e.translationY;
});
const rStyle = useAnimatedStyle(() => ({
  transform: [{ translateX: x.value }, { translateY: y.value }]
}));
Avec withSpring dans onEnd on peut renvoyer l’élément à sa place.

🧪 À toi de jouer

Crée un composant qui :

  • Contient un cercle jaune de 80x80.
  • Réagit au glissement (pan) : se déplace avec le doigt.
  • Au lâcher, revient en douceur à sa position initiale (avec spring).
  • Un double tap sur le cercle change sa couleur (ex: rouge ↔️ jaune).
🧩 Indice / solution
useSharedValue(0) pour translation, useSharedValue('yellow') pour couleur, avec interpolateColor ou un style conditionnel. Double tap avec Gesture.Tap().numberOfTaps(2).onEnd(() => { color.value = color.value === 'yellow' ? 'red' : 'yellow' });
