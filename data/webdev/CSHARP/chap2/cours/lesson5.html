Types valeur / référence

Manipuler les chaînes et l'interpolation

🧠 Types valeur vs Types référence

📦 Valeur (struct)
  • int, float, double, bool, char, decimal
  • Stocké dans la pile (stack) → accès rapide
  • Chaque variable possède sa propre copie des données
  • Exemple : int a = 5; int b = a; b = 10; → a reste 5
🔗 Référence (class)
  • string, array, class, List<T>
  • Stocké dans le tas (heap) ; variable contient une adresse
  • Copier une référence = partager les mêmes données
  • Exemple : int[] t1 = {1}; int[] t2 = t1; t2[0]=99; → t1[0] devient 99

💡 En résumé :

Les types valeur sont indépendants. Les types référence partagent le même objet. string est particulier : bien que référence, il est immutable (chaque modification crée une nouvelle chaîne).

   STACK (pile)               HEAP (tas)
  ┌──────────┐              ┌────────────┐
  │ int x = 5│              │   "Bonjour"│  ← référence vers chaîne
  └──────────┘              └────────────┘
  ┌──────────┐                    ▲
  │string s  │────────────────────┘
  └──────────┘

Illustration : les valeurs sont directes, les références pointent vers le tas.

📝 Les chaînes de caractères (string)

string est un alias de System.String. C’est une séquence de caractères Unicode. Important : immuable → chaque opération crée une nouvelle chaîne.

✍️ Déclaration classique

string message = "Bienvenue en C#";
char premièreLettre = message[0]; // 'B'

📁 Verbatim strings @"..."

Ignore les caractères d’échappement comme \n ou \t. Idéal pour les chemins de fichiers ou textes multilignes.

string chemin = @"C:\Dossier\Fichier.txt";
string multiligne = @"Ligne 1
Ligne 2
Ligne 3";

🔧 Échappement classique

string guillemet = "Il a dit \"Bonjour!\"";
string retour = "Première ligne\nSeconde ligne";
⚠️ Astuce : Pour comparer des chaînes, privilégiez Equals() ou string.Compare(), pas == si vous voulez ignorer la casse.

✨ Interpolation de chaînes ($)

Introduite en C# 6, elle permet d’insérer des variables ou expressions directement dans une chaîne. Plus lisible que string.Format() ou les concaténations.

Syntaxe : $"texte {expression} suite"

string nom = "Fatima";
int age = 19;
string ville = "Douala";

// interpolation simple
string presentation = $"Je m'appelle {nom}, j'ai {age} ans et j'habite {ville}.";
Console.WriteLine(presentation);
// Output: Je m'appelle Fatima, j'ai 19 ans et j'habite Douala.

// avec expressions mathématiques
double prix = 49.99;
string ticket = $"Total TTC : {prix * 1.2:C2}"; // format monétaire
Console.WriteLine(ticket); // Total TTC : 59,99 € (selon culture)

// appel de méthode ou propriété
DateTime aujourdhui = DateTime.Now;
string date = $"Nous sommes le {aujourdhui:dddd dd MMMM yyyy}";
Console.WriteLine(date);
✅ Avantages : clarté, moins d’erreurs, supports formatage (comme :N2, :C).
⚠️ Attention : Ne pas utiliser avec des données utilisateur non nettoyées (injection ?) en général c'est sûr car chaîne simple.

💻 Mise en pratique - exemples Camerounais 🇨🇲

Imaginons une petite application qui affiche la présentation d’un étudiant en génie logiciel.

string ecole = "Université de Douala";
string filiere = "Génie Informatique";
int niveau = 2;
double moyenne = 15.75;

// interpolation avec formatage
string description = $"""
    === FICHE ÉTUDIANT ===
    Établissement : {ecole}
    Filière : {filiere}
    Niveau : {niveau}ème année
    Moyenne : {moyenne:F2}/20
    """;
Console.WriteLine(description);

// Verbatim + interpolation combiné (C# 11+ possibilité $@)
string cheminProjet = $@"C:\Projets\{filiere.Replace(" ", "_")}\src";
Console.WriteLine($"Dossier projet : {cheminProjet}");

🧪 À essayer chez vous :

  • Créez une variable string nomComplet = "Awa Diallo"; et affichez "Bienvenue Awa Diallo, la session a commencé!" en utilisant l’interpolation.
  • Testez une verbatim string avec des slashs inversés : @"\\serveur\partage\fichier.cs".
  • Comparez la différence entre int x = 10; int y = x; y=20; vs int[] a = {1}; int[] b = a; b[0]=99; pour bien assimiler valeur/référence.

📌 Ce qu’il faut retenir

✔️ Valeur (int, bool, struct) → stockage pile, indépendance. Référence (string, class) → partage des données, heap.
✔️ string est immuable : chaque modification produit une nouvelle chaîne. Préférer StringBuilder pour des concaténations lourdes.
✔️ Verbatim strings @"..." : idéales pour les chemins Windows et le texte multiligne.
✔️ Interpolation $"{variable}" : plus propre, moderne, supporte la mise en forme comme {prix:C}.
