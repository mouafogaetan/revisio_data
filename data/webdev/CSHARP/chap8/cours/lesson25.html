Objectifs de la leçon

1️⃣ Syntaxe de requête

Comprendre la syntaxe déclarative from ... where ... select, proche du SQL, pour interroger facilement des collections.

2️⃣ Syntaxe de méthode

Utiliser les méthodes d'extension comme Where(), Select(), OrderBy() pour enchaîner les opérations.

3️⃣ Différencier les variantes

Savoir distinguer LINQ to Objects (collections mémoire), LINQ to SQL (base de données) et LINQ to XML.

📌 Compétence visée

Maîtriser les bases de LINQ pour filtrer, trier, projeter des données efficacement en C#.

🧠

LINQ en une minute

LINQ (Language Integrated Query) est un ensemble de fonctionnalités C# qui permet d'écrire des requêtes directement dans le langage, de manière fortement typée et lisible.

✅ Avantages

  • Moins de code comparé aux boucles classiques
  • Intégration directe au langage (intellisense)
  • Requêtes homogènes sur différentes sources

⚡ Exemple simple

string[] noms = {"Ali", "Fatima", "Jean"};
var result = from n in noms
             where n.Length > 3
             select n;
// Syntaxe méthode : noms.Where(n => n.Length > 3);

💡 La bibliothèque System.Linq est indispensable. N'oubliez pas using System.Linq; en haut de votre fichier.

📜 Syntaxe de requête

Inspirée du SQL, elle repose sur trois mots-clés essentiels : from, where, select.

// Filtrer une liste d'entiers pairs et les doubler
int[] nombres = { 1, 2, 3, 4, 5, 6 };
var pairsMultiples = from n in nombres
                     where n % 2 == 0
                     select n * 2;
// résultat : {4, 8, 12}
from variable in collection
🔹 Définit la source
where condition
🔹 Filtre les éléments
select expression
🔹 Projette le résultat
⚠️ La syntaxe de requête est plus lisible pour les opérations complexes (jointures, group by).

⚡ Syntaxe de méthode (Fluent)

Utilisez les méthodes d'extension comme Where(), Select(), OrderBy() avec des expressions lambda.

List<string> pays = new() {"Cameroun", "France", "Canada", "Bénin"};
// 1. Filtrer les pays commençant par 'C'
var filtre = pays.Where(p => p.StartsWith("C"));
// 2. Trier par longueur
var trie = pays.OrderBy(p => p.Length);
// 3. Projeter en majuscules
var maj = pays.Select(p => p.ToUpper());
// Enchaînement : pays.Where(p => p.Length > 5).OrderBy(p => p).Select(p => p);
🔁 Lambda : param => expression (ex: n => n > 0) 📦 Retardé (Deferred Execution) : La requête n'est exécutée que lors de l'énumération.

🧩 Trois visages de LINQ

💾
LINQ to Objects
Requêtes sur collections en mémoire : tableaux, List, Dictionary. Idéal pour manipuler des données locales.
🗄️
LINQ to SQL (Entity Framework)
Traduit les requêtes LINQ en SQL pour interagir avec une base de données relationnelle.
📄
LINQ to XML
Manipuler des documents XML de manière fluide (XElement, XDocument) sans utiliser DOM lourd.
🌟 Dans ce cours, nous nous concentrons sur LINQ to Objects, base indispensable pour la suite.

🇨🇲 Exemple : Étudiants camerounais

Créez une liste d'étudiants et utilisez LINQ pour interroger.

class Etudiant {
    public string Nom { get; set; }
    public string Ville { get; set; }
    public int Note { get; set; }
}
List<Etudiant> eleves = new() {
    new Etudiant{"Alice","Douala",85},
    new Etudiant{"Brice","Yaoundé",92},
    new Etudiant{"Cédric","Bafoussam",67}
};
// Requête : étudiants de Yaoundé avec note > 80
var resultat = from e in eleves
               where e.Ville == "Yaoundé" && e.Note > 80
               select e.Nom;
// Syntaxe méthode : eleves.Where(e => e.Ville=="Yaoundé" && e.Note>80)
//                     .Select(e => e.Nom);
🔍 Résultat : "Brice" 🏆 La puissance de LING évite les boucles foreach fastidieuses.

🎓 Résumé & Bonnes pratiques

  • Utilisez la syntaxe de requête pour des opérations complexes (jointures, group by) et la syntaxe de méthode pour des enchaînements simples.
  • Les requêtes LINQ sont évaluées de manière différée : n'oubliez pas d'ajouter .ToList() ou .ToArray() pour exécuter immédiatement.
  • LINQ est indispensable pour travailler avec Entity Framework Core (API / Base de données).
  • Explorez les opérateurs avancés : GroupBy, Join, Aggregate, etc. (leçons suivantes).

✨ Challenge :

À partir d'un tableau d'entiers [12, 45, 7, 32, 89, 22], écrivez une requête LINQ (syntaxe méthode) pour obtenir les nombres > 20, triés par ordre croissant.

Voir solution int[] nums = {12,45,7,32,89,22}; var result = nums.Where(n => n > 20).OrderBy(n => n);
