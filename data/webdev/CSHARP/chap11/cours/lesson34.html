Manipulation du système de fichiers

Lire, écrire des fichiers et gérer des dossiers avec .NET

📁 File / Directory 📄 StreamReader/Writer ⚙️ .NET I/O

🎯 Objectifs de la leçon

  • ✓ Utiliser File.ReadAllText et WriteAllText pour simplifier l’accès aux fichiers.
  • ✓ Travailler avec StreamReader / StreamWriter pour lire/écrire ligne par ligne.
  • ✓ Lire et écrire des fichiers binaires via BinaryReader / BinaryWriter.
  • ✓ Explorer et gérer les répertoires avec Directory, DirectoryInfo.

📌 Après cette leçon, vous saurez créer un mini explorateur de fichiers ou une sauvegarde automatique !

📄 1. Opérations simples avec la classe File

La classe System.IO.File propose des méthodes statiques rapides pour lire/écrire du texte en une ligne.

using System.IO;

// Écrire tout le texte dans un fichier (écrase si existe)
string contenu = "Bonjour Cameroun !\nBienvenue dans la manip fichiers.";
File.WriteAllText("exemple.txt", contenu);

// Lire tout le fichier d’un coup
string lu = File.ReadAllText("exemple.txt");
Console.WriteLine(lu);

// Ajouter du texte à la fin
File.AppendAllText("exemple.txt", "\nLigne ajoutée avec AppendAllText");

💡 Astuce : Utile pour les fichiers de configuration, logs simples, ou sauvegarde de petites données. Attention avec les très gros fichiers (préférez Stream).

📖 2. Lecture/écriture avancée : StreamReader & StreamWriter

Ideal pour lire ligne par ligne ou traiter des fichiers volumineux sans tout charger en mémoire.

// Écriture avec StreamWriter
using (StreamWriter sw = new StreamWriter("journal.txt"))
{
    sw.WriteLine("Première ligne");
    sw.WriteLine($"Heure actuelle : {DateTime.Now}");
}

// Lecture ligne par ligne
using (StreamReader sr = new StreamReader("journal.txt"))
{
    string ligne;
    while ((ligne = sr.ReadLine()) != null)
    {
        Console.WriteLine($"👉 {ligne}");
    }
}
✔ using garantit la fermeture du flux.
✔ ReadLine() = lecture efficace ligne par ligne.

🔢 3. Manipulation de fichiers binaires

Pour stocker des données brutes (nombres, images personnalisées, objets sérialisés).

// Écriture binaire
using (BinaryWriter bw = new BinaryWriter(File.Open("donnees.bin", FileMode.Create)))
{
    bw.Write(123);          // int
    bw.Write(3.1415);       // double
    bw.Write("Cameroun .NET"); // string
}

// Lecture binaire
using (BinaryReader br = new BinaryReader(File.Open("donnees.bin", FileMode.Open)))
{
    int entier = br.ReadInt32();
    double pi = br.ReadDouble();
    string texte = br.ReadString();
    Console.WriteLine($"{entier}, {pi}, {texte}");
}
📦 Utilisation fréquente pour sauvegarder des structures, éviter l’encodage texte.

🗂️ 4. Gestion des répertoires

Créer, lister, déplacer, supprimer des dossiers : Directory (statique) et DirectoryInfo (orienté objet).

// Créer un dossier
Directory.CreateDirectory(@"C:\CoursCSharp\Backup");

// Lister tous les fichiers .txt
string[] fichiersTxt = Directory.GetFiles(@"C:\CoursCSharp", "*.txt");
foreach (var f in fichiersTxt) Console.WriteLine(f);

// Avec DirectoryInfo (plus riche)
DirectoryInfo dirInfo = new DirectoryInfo(@"C:\CoursCSharp");
if (dirInfo.Exists)
{
    Console.WriteLine($"Créé le : {dirInfo.CreationTime}");
    // Parcourir sous-dossiers
    foreach (var sub in dirInfo.GetDirectories())
        Console.WriteLine($"📁 {sub.Name}");
}
⚠️ Attention aux droits d'accès. Sur Linux/Mac, utilisez des chemins comme "/home/user/Documents".

🛠️ Exemple pratique : Sauvegarde de notes

Petit programme qui écrit les saisies utilisateur dans un fichier et liste le contenu du dossier.

string dossier = "NotesApp";
Directory.CreateDirectory(dossier);
string cheminFichier = Path.Combine(dossier, "journal.txt");

// Écrire des notes
Console.WriteLine("Votre note :");
string note = Console.ReadLine();
File.AppendAllText(cheminFichier, $"{DateTime.Now}: {note}\n");

// Lire toutes les notes
Console.WriteLine("\n--- Vos notes sauvegardées ---");
string toutesNotes = File.ReadAllText(cheminFichier);
Console.WriteLine(toutesNotes);

// Lister tous les fichiers dans le dossier
string[] fichiers = Directory.GetFiles(dossier);
Console.WriteLine($"\nFichiers dans {dossier} :");
foreach (var f in fichiers) Console.WriteLine(Path.GetFileName(f));

🌟 À retenir : Path.Combine gère automatiquement les séparateurs de chemin selon l'OS. Toujours utiliser using pour les flux.
