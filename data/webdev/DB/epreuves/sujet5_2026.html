⚡ Optimisation des performances

🔍 Analyse avec EXPLAIN :

EXPLAIN SELECT * FROM produit WHERE categorie_id = 5;

Un seq scan signifie un parcours séquentiel. Pour accélérer, créez un index :

CREATE INDEX idx_produit_categorie ON produit(categorie_id);

Les index de type B-Tree sont efficaces pour les égalités et intervalles.
