Partie 3

SQL : Création et modification de tables (20 pts)

🎯 Contexte : Vous devez créer la table Client et Commande pour une boutique.

Question 1 (6 pts) : Écrire la requête SQL (CREATE TABLE) pour l'entité Client avec :

  • id_client (INT, clé primaire, auto-incrément)
  • nom (VARCHAR(50), NOT NULL)
  • email (VARCHAR(100), UNIQUE)
  • date_inscription (DATE, valeur par défaut = date courante)

Question 2 (6 pts) : Ajoutez une colonne telephone (VARCHAR(20)) à la table Client après la création. Puis modifiez le type de la colonne email pour le passer à VARCHAR(150).

Question 3 (8 pts) : Créez la table Commande avec les contraintes suivantes :

  • id_commande INT PK, date_commande DATETIME, montant DECIMAL(10,2) CHECK (montant > 0)
  • id_client INT, clé étrangère référençant Client(id_client) avec ON DELETE CASCADE.
