Rappel : Le hook useState

useState permet d'ajouter un état local à un composant fonctionnel.

import { useState } from 'react';

function MonComposant() {
  const [compteur, setCompteur] = useState(0);
  // compteur : la valeur actuelle (0)
  // setCompteur : fonction pour modifier la valeur

  return <div>{compteur}</div>;
}

✨ Règles à retenir :

  • useState retourne un tableau de 2 éléments : [valeur, fonction]
  • Appelez useState uniquement au niveau racine du composant
  • La mise à jour est asynchrone

🔢 Créer un compteur interactif

Voici un exemple complet de compteur avec incrément, décrément et réinitialisation.

function Compteur() {
  const [count, setCount] = useState(0);

  const incrementer = () => setCount(count + 1);
  const decrementer = () => setCount(count - 1);
  const reset = () => setCount(0);

  return (
    <div>
      <p>Compteur : {count}</p>
      <button onClick={incrementer}>+</button>
      <button onClick={decrementer}>-</button>
      <button onClick={reset}>Reset</button>
    </div>
  );
}

⚠️ Mise à jour asynchrone

Lorsqu'on utilise la valeur précédente, il faut utiliser la forme fonctionnelle.

// ❌ Problème si on appelle plusieurs fois setCount
setCount(count + 1);
setCount(count + 1); // count n'a pas encore changé → 1 au lieu de 2

// ✅ Solution : utiliser prevState
setCount(prevCount => prevCount + 1);
setCount(prevCount => prevCount + 1); // fonctionne !

📌 Règle d'or :

Quand la nouvelle valeur dépend de l'ancienne, utilisez toujours la forme fonctionnelle : setState(prev => prev + 1)

📝 Input contrôlé

Un champ est dit "contrôlé" quand sa valeur est liée à l'état React.

function FormulaireSimple() {
  const [nom, setNom] = useState('');

  const handleChange = (e) => {
    setNom(e.target.value);
  };

  return (
    <div>
      <input 
        type="text" 
        value={nom} 
        onChange={handleChange}
        placeholder="Votre nom" 
      />
      <p>Bonjour {nom || 'invité'} !</p>
    </div>
  );
}

🔘 Checkbox, Radio, Select

function Preferences() {
  const [accepte, setAccepte] = useState(false);
  const [genre, setGenre] = useState('femme');
  const [couleur, setCouleur] = useState('bleu');

  return (
    <div>
      <label>
        <input type="checkbox" checked={accepte} 
               onChange={(e) => setAccepte(e.target.checked)} />
        J'accepte les conditions
      </label>

      <select value={genre} onChange={(e) => setGenre(e.target.value)}>
        <option>homme</option>
        <option>femme</option>
      </select>
    </div>
  );
}
💡 Pour une checkbox, on utilise e.target.checked au lieu de e.target.value.

📬 Gérer la soumission (onSubmit)

function Inscription() {
  const [email, setEmail] = useState('');
  const [mdp, setMdp] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault(); // Empêche le rechargement
    console.log('Email :', email, 'Mot de passe :', mdp);
    alert(`Bienvenue ${email} !`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="email" value={email} onChange={(e) => setEmail(e.target.value)} />
      <input type="password" value={mdp} onChange={(e) => setMdp(e.target.value)} />
      <button type="submit">S'inscrire</button>
    </form>
  );
}
⚠️ N'oubliez pas e.preventDefault() pour éviter que la page ne se recharge !

✅ Bonnes pratiques

✓
Nommage explicite : [compteur, setCompteur] plutôt que [c, setC]
✓
Utiliser prevState : quand la nouvelle valeur dépend de l'ancienne
✓
Un useState par donnée : séparez les états non liés
✓
Objets complexes : utilisez le spread operator setUser({...user, nom: 'Jean'})
