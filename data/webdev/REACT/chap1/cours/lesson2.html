🎯 Objectifs de la leçon

  • Utiliser Create React App (npx create-react-app)

À la fin de ce cours, vous saurez créer et organiser n’importe quelle application React.

⚛️ Create React App (CRA)

Create React App est l'outil officiel pour initialiser un projet React sans configuration complexe.

npx create-react-app mon-app-react
cd mon-app-react
npm start
  • Génère une structure complète avec Webpack, Babel, ESLint.
  • Démarre un serveur de développement avec rechargement à chaud.
  • Commande npm run build pour la version de production.
CRA est parfait pour les débutants, mais Vite est plus rapide pour les projets récents.

⚡ Vite : l'alternative moderne

Vite offre un démarrage instantané et un rechargement ultra-rapide grâce aux modules ES natifs.

npm create vite@latest mon-app-vite -- --template react
cd mon-app-vite
npm install
npm run dev

Avantages :

  • Build plus rapide en production (Rollup).
  • Moins de dépendances, configuration simplifiée.
  • Support natif des fichiers .env et TypeScript.
CRA : robuste & éprouvé
Vite : vitesse & modernité

📁 Structure d'un projet React

📂 mon-app/
├── public/
│ └── index.html (la page principale)
├── src/
│ ├── main.jsx (point d'entrée React)
│ ├── App.jsx
│ └── index.css
├── package.json
└── vite.config.js (ou pas)

index.html : contient la div #root où React se greffe.

main.jsx : rend le composant principal <App /> dans le DOM.

src/ : tous vos composants, styles, hooks, assets.

À retenir : Le dossier public contient les fichiers statiques, src contient le code source évolutif.

📝 Exercice & Synthèse

Mise en pratique

1. Créez un projet avec Vite et un autre avec Create React App.

2. Observez la différence de temps d'installation et de démarrage.

3. Identifiez le fichier où le composant principal est rendu.

4. Modifiez le texte d’un composant et voyez le rafraîchissement automatique.

Question rapide

Quelle commande permet de lancer l’application en mode développement avec Vite ?

✅ Réponse : npm run dev

Pour aller plus loin

Essayez d’ajouter un composant simple dans src/components/ et importez-le dans App.jsx.
