Personnaliser l'apparence de l'application

🎨 Thème, couleurs, typographie et thème sombre avec Flutter

Classe: Développement Web | Matière: Flutter
Chapitre 8 : Personnalisation UI

🎯 Objectifs de la leçon

  • ✓ Configurer ThemeData dans MaterialApp.
  • ✓ Définir les couleurs principales (primarySwatch, accentColor, backgroundColor).
  • ✓ Utiliser Theme.of(context) pour accéder au thème et le réutiliser.
  • ✓ Comprendre l'importance d'un thème cohérent pour l'expérience utilisateur.

💡 Pourquoi ? Une application professionnelle offre une apparence harmonieuse. Flutter facilite la personnalisation globale des couleurs, polices et styles.

🎨 1. ThemeData : Le cœur du style

Dans Flutter, le thème se définit dans le widget MaterialApp via la propriété theme. ThemeData contient toutes les configurations visuelles globales.

MaterialApp(
  title: 'Mon App',
  theme: ThemeData(
    primarySwatch: Colors.blue,
    accentColor: Colors.orange,
    backgroundColor: Colors.grey[100],
    fontFamily: 'Poppins',
  ),
  home: HomeScreen(),
);
🔵 primarySwatch – Définit la couleur principale (appBar, boutons, sélections).
🟠 accentColor – Couleur d'accentuation (FAB, éléments interactifs).
⚪ backgroundColor – Couleur de fond générale de l'application.

🌈 2. Palettes et couleurs personnalisées

primarySwatch utilise une nuance Material Design. Vous pouvez aussi définir des couleurs précises avec primaryColor, primaryColorLight, etc.

theme: ThemeData(
  primaryColor: Color(0xFF2E5BFF),   // bleu personnalisé
  accentColor: Color(0xFFFF6B35),
  scaffoldBackgroundColor: Color(0xFFF5F7FA),
),

📐 Illustration simple : hiérarchie des couleurs Flutter

💡 Bonnes pratiques : utilisez des couleurs accessibles (contraste suffisant). Taillez votre thème pour qu’il reflète l’identité de l’application.

🔍 3. Accéder au thème avec Theme.of(context)

Grâce à Theme.of(context), vous pouvez récupérer les couleurs, styles de texte, etc., n'importe où dans l'arbre des widgets. Cela permet d’assurer la cohérence et de ne pas dupliquer les couleurs.

// Exemple : utiliser la couleur primaire pour un texte
Text(
  'Bonjour Cameroun !',
  style: TextStyle(
    color: Theme.of(context).primaryColor,
    fontWeight: FontWeight.bold,
  ),
),

// Récupérer la couleur d'accentuation pour un bouton
ElevatedButton(
  style: ElevatedButton.styleFrom(
    primary: Theme.of(context).accentColor,
  ),
  onPressed: () {},
  child: Text('Valider'),
),
📌 Astuce prof : Utilisez Theme.of(context).textTheme.bodyText1 pour adopter la typographie globale. Très pratique pour des applications évolutives!

🧪 À vous de jouer ! Mini-exercice

Modifiez le thème d'une petite application Flutter :

  • Créez un thème personnalisé avec primarySwatch: Colors.green et accentColor: Colors.pink.
  • Changez la couleur de fond (backgroundColor) en Colors.grey[200].
  • Dans un widget, affichez un texte utilisant Theme.of(context).primaryColor.

✨ Récapitulatif important :

- ThemeData : configuration globale.
- primarySwatch / primaryColor : couleur principale.
- accentColor : couleur secondaire.
- Theme.of(context) : accès dynamique au thème.
- Une bonne personnalisation rend l'application unique et professionnelle.
