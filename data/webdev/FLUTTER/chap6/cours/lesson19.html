📱 Les widgets de saisie Flutter

Flutter propose une riche collection de widgets pour interagir avec l'utilisateur. Voici les incontournables :

TextField

Champ de texte simple, idéal pour email, nom, mot de passe.

TextField(decoration: InputDecoration(labelText: "Nom"))
Checkbox

Case à cocher pour booléens (accepter conditions).

Checkbox(value: isChecked, onChanged: (val) {})
Radio

Choix unique parmi plusieurs options.

Radio(value: "Homme", groupValue: genre, onChanged: (val) {})
Switch

Interrupteur pour activer/désactiver une fonction.

Switch(value: isSwitched, onChanged: (val) {})
Slider

Curseur pour sélectionner une valeur dans un intervalle.

Slider(value: _value, min: 0, max: 100, onChanged: (newVal) {})
✨ Chaque widget peut être stylisé via decoration ou Theme.

⚡ Événements et contrôle du texte

📌 onChanged & onSubmitted

Déclenchez des actions à chaque frappe (onChanged) ou quand l’utilisateur valide (onSubmitted). Utile pour la validation en direct.

TextField(
  onChanged: (text) => print("Texte: $text"),
  onSubmitted: (value) => print("Envoyé: $value"),
)

🎮 TextEditingController

Permet de lire, modifier ou effacer le contenu d’un champ de manière programmatique.

final controller = TextEditingController();
TextField(controller: controller);
// Lire: controller.text
// Modifier: controller.text = "nouveau texte"
// Effacer: controller.clear()

💡 Indispensable pour pré-remplir des formulaires ou réinitialiser après envoi.

2
Checkbox, Radio, Switch, Slider : gérer booléens, choix multiples, plages.
3
Événements : onChanged, onSaved, onSubmitted → rend l’interface réactive.
4
TextEditingController : contrôle total du texte (lecture, modification, reset).

📝 Extrait de code complet :

final _nameController = TextEditingController();
bool _isSubscribed = false;
double _volume = 0.5;

@override
Widget build(BuildContext context) {
  return Column(children: [
    TextField(controller: _nameController),
    Switch(value: _isSubscribed, onChanged: (val) => setState(() => _isSubscribed = val)),
    Slider(value: _volume, onChanged: (val) => setState(() => _volume = val)),
    ElevatedButton(onPressed: () => print(_nameController.text), child: Text("Envoyer"))
  ]);
}
