📦 1. Installation du package image_picker

Pour accéder à la galerie et à l’appareil photo, Flutter utilise le package image_picker. Ajoutez-le dans pubspec.yaml :

dependencies:
  flutter:
    sdk: flutter
  image_picker: ^1.0.5   # dernière version stable

⚠️ Important :

Sur iOS, ajoutez dans ios/Runner/Info.plist :
NSCameraUsageDescription et NSPhotoLibraryUsageDescription
Sur Android, pas de configuration supplémentaire pour la galerie (mais la caméra nécessite l’ajout dans AndroidManifest.xml)

🖼️ 2. Implémentation pas à pas

✨ Ouvrir la galerie

final ImagePicker picker = ImagePicker();
final XFile? image = await picker.pickImage(source: ImageSource.gallery);

  setState(() => _imageFile = image);

📷 Prendre une photo

final XFile? photo = await picker.pickImage(source: ImageSource.camera);
if (photo != null) setState(() => _imageFile = photo);

🖼️ Afficher l'image

_imageFile != null 
  ? Image.file(File(_imageFile!.path))

🔐 3. Gestion des permissions (Android/iOS)

Depuis Flutter, on utilise souvent permission_handler pour vérifier les autorisations avant d’ouvrir la caméra.

import 'package:permission_handler/permission_handler.dart';

Future<bool> requestCameraPermission() async {
  var status = await Permission.camera.status;
  if (!status.isGranted) {
    status = await Permission.camera.request();
  }
  return status.isGranted;
}

✔️ Sur Android 11+ : les permissions de stockage changent, mais image_picker gère automatiquement les scoped storage. Pas besoin de READ_EXTERNAL_STORAGE pour la galerie dans la plupart des cas.

📱 iOS: ajouter dans Info.plist
🤖 Android: Vérifier minSdkVersion 21+

📱 4. Application complète + démo pratique

Voici un widget minimal qui combine galerie et caméra :

class ImagePickerDemo extends StatefulWidget {
  @override
  _ImagePickerDemoState createState() => _ImagePickerDemoState();
}
class _ImagePickerDemoState extends State<ImagePickerDemo> {
  XFile? _image;
  final ImagePicker _picker = ImagePicker();

  Future _getFromGallery() async {
    final XFile? picked = await _picker.pickImage(source: ImageSource.gallery);
    setState(() => _image = picked);
  }

  Future _takePhoto() async {
    final XFile? photo = await _picker.pickImage(source: ImageSource.camera);
    setState(() => _image = photo);
  }

  @override
  Widget build(BuildContext context) {
    return Column(children: [
      Row(mainAxisAlignment: MainAxisAlignment.center, children: [
        ElevatedButton(onPressed: _getFromGallery, child: Text('Galerie')),
        SizedBox(width: 16),
        ElevatedButton(onPressed: _takePhoto, child: Text('Caméra')),
      ]),
      SizedBox(height: 20),
      _image != null ? Image.file(File(_image!.path), height: 200) 
                     : Icon(Icons.image, size: 100, color: Colors.grey)
    ]);
  }
}
