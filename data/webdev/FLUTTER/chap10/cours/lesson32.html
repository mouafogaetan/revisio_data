4️⃣

Afficher une carte Google Maps avec un marqueur à la position de l'utilisateur.

💡 Pourquoi c'est utile ?
Les applications de livraison, de météo, de réseaux sociaux et de navigation ont toutes besoin de connaître la position de l'utilisateur. Flutter permet d'accéder facilement au GPS grâce à des packages bien maintenus.

📦 Installation des packages

Ajoutez ces dépendances dans votre fichier pubspec.yaml :

dependencies:
  flutter:
    sdk: flutter
  geolocator: ^10.1.0          # Pour la géolocalisation
  google_maps_flutter: ^2.5.0   # Pour l'affichage des cartes

Ensuite, exécutez :

flutter pub get

⚠️ Note importante :

Pour google_maps_flutter, vous aurez besoin d'une clé API Google Maps (gratuite avec limitation). Obtenez-la depuis Google Cloud Console.

📱 Configuration Android

1. Ajouter les permissions dans android/app/src/main/AndroidManifest.xml :

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />

<!-- Pour Google Maps -->
<meta-data
    android:name="com.google.android.geo.API_KEY"
    android:value="VOTRE_CLE_API_GOOGLE_MAPS" />

2. Activer l'accès à la localisation dans le manifeste.

💡 La clé API doit être ajoutée dans la section application du manifeste.

🍎 Configuration iOS

Ajouter les descriptions de permission dans ios/Runner/Info.plist :

<key>NSLocationWhenInUseUsageDescription</key>
<string>Cette application a besoin de votre position pour afficher la carte</string>

<key>NSLocationAlwaysUsageDescription</key>
<string>Nous utilisons votre position même en arrière-plan pour améliorer l'expérience</string>

<!-- Clé API Google Maps iOS -->
<key>io.flutter.embedded_views_preview</key>
<true/>

Ajoutez également votre clé API Google Maps dans AppDelegate.swift ou via un fichier de configuration séparé.

🔐 Demander les permissions

Avant d'accéder à la position, il faut demander l'autorisation à l'utilisateur :

import 'package:geolocator/geolocator.dart';

Future<LocationPermission> checkAndRequestPermission() async {
  LocationPermission permission = await Geolocator.checkPermission();
  
  if (permission == LocationPermission.denied) {
    permission = await Geolocator.requestPermission();
    if (permission == LocationPermission.denied) {
      print('Permission refusée');
      return permission;
    }
  }
  
  if (permission == LocationPermission.deniedForever) {
    print('Permission refusée définitivement');
    return permission;
  }
  
  return permission;
}

📌 Bonnes pratiques : Toujours vérifier l'état de la permission avant d'appeler la position. Si l'utilisateur refuse définitivement, redirigez-le vers les paramètres de l'application.

📍 Obtenir la position actuelle

Future<Position> getCurrentPosition() async {
  LocationPermission permission = await checkAndRequestPermission();
  
  if (permission == LocationPermission.always ||
      permission == LocationPermission.whileInUse) {
    return await Geolocator.getCurrentPosition(
      desiredAccuracy: LocationAccuracy.high,
    );
  } else {
    throw Exception('Permissions insuffisantes');
  }
}

L'objet Position contient :

  • position.latitude : la latitude (double)
  • position.longitude : la longitude (double)
  • position.altitude : l'altitude (optionnel)
  • position.speed : la vitesse (optionnel)

🗺️ Afficher une carte interactive

Créez un widget StatefulWidget qui affiche une carte et un marqueur :

class MapScreen extends StatefulWidget {
  @override
  _MapScreenState createState() => _MapScreenState();
}

class _MapScreenState extends State<MapScreen> {
  late GoogleMapController _controller;
  Position? _currentPosition;
  Set<Marker> _markers = {};

  @override
  void initState() {
    super.initState();
    _getUserLocation();
  }

  Future<void> _getUserLocation() async {
    Position position = await getCurrentPosition();
    setState(() {
      _currentPosition = position;
      _markers.add(
        Marker(
          markerId: MarkerId('user'),
          position: LatLng(position.latitude, position.longitude),
          infoWindow: InfoWindow(title: 'Vous êtes ici'),
        ),
      );
    });
  }

  @override
  Widget build(BuildContext context) {
    if (_currentPosition == null) {
      return Scaffold(
        body: Center(child: CircularProgressIndicator()),
      );
    }

    return Scaffold(
      appBar: AppBar(title: Text('Ma position')),
      body: GoogleMap(
        initialCameraPosition: CameraPosition(
          target: LatLng(_currentPosition!.latitude, _currentPosition!.longitude),
          zoom: 15,
        ),
        markers: _markers,
        onMapCreated: (controller) => _controller = controller,
      ),
    );
  }
}

📍 Aller plus loin : marqueurs interactifs

// Ajouter un marqueur au clic sur la carte
GoogleMap(
  onTap: (LatLng position) {
    setState(() {
      _markers.add(
        Marker(
          markerId: MarkerId(position.toString()),
          position: position,
          infoWindow: InfoWindow(title: 'Point sélectionné'),
        ),
      );
    });
  },
  // ... autres propriétés
)

// Obtenir la position en temps réel (suivi)
Stream<Position> getPositionStream() {
  return Geolocator.getPositionStream(
    locationSettings: LocationSettings(
      accuracy: LocationAccuracy.high,
      distanceFilter: 10, // mètres
    ),
  );
}

✨ Astuce : Utilisez StreamBuilder pour écouter les changements de position en continu (exemple : application de course à pied).
