2 Interface utilisateur (NativeWind) - 5 pts

Vous devez créer une carte produit pour l'écran d'accueil. En utilisant NativeWind (Tailwind CSS pour React Native), écrivez le composant ProductCard qui :

  • Affiche une image (carrée, 120x120) avec className="w-32 h-32 rounded-lg".
  • Affiche le nom du produit (Text, gras, 16px).
  • Affiche le prix (Text, rouge, 18px).
  • Affiche un badge "Promotion" si discount > 0 (fond rouge, texte blanc, arrondi).
  • Ajoute un bouton "Ajouter au panier" avec style Tailwind.
  • Utilise TouchableOpacity pour le clic sur la carte entière.

4 Gestion d'état (Zustand) - 5 pts

Créez un store Zustand pour la gestion du panier avec persistance (AsyncStorage). Le store doit gérer :

  • items: CartItem[] (id, name, price, quantity, image)
  • addItem(item) : ajoute ou incrémente la quantité
  • removeItem(id) : supprime l'article
  • updateQuantity(id, quantity) : modifie la quantité (min 1)
  • clearCart() : vide le panier
  • totalItems : nombre total d'articles (getter)
  • totalPrice : prix total (getter)
