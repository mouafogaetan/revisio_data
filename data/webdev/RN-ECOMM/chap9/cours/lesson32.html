🗄️ Structure d'une adresse & Store Zustand

// types/address.ts
interface Address {
  id: string;
  fullName: string;
  phone: string;
  street: string;
  city: string;
  postalCode?: string;
  latitude?: number;
  longitude?: number;
  isDefault: boolean;
}

// stores/addressStore.js
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

const useAddressStore = create(
  persist(
    (set, get) => ({
      addresses: [],
      addAddress: (addr) => set({ addresses: [...get().addresses, addr] }),
      updateAddress: (id, updated) => set({ 
        addresses: get().addresses.map(a => a.id === id ? updated : a) 
      }),
      deleteAddress: (id) => set({ 
        addresses: get().addresses.filter(a => a.id !== id) 
      }),
      setDefaultAddress: (id) => set({ 
        addresses: get().addresses.map(a => ({ ...a, isDefault: a.id === id }))
      })
    }),
    { name: 'address-storage' }
  )
);
Stockage persistant avec AsyncStorage (Zustand persist).

📝 Interface CRUD adresses

Liste des adresses avec actions (modifier, supprimer, définir par défaut).

// Composant AddressList
const AddressList = () => {
  const { addresses, deleteAddress, setDefaultAddress } = useAddressStore();

  return (
    <FlatList
      data={addresses}
      renderItem={({ item }) => (
        <View>
          <Text>{item.street}, {item.city}</Text>
          {item.isDefault && <Badge>Par défaut</Badge>}
          <Button title="Modifier" onPress={() => editAddress(item)} />
          <Button title="Supprimer" onPress={() => deleteAddress(item.id)} />
          {!item.isDefault && <Button title="Définir par défaut" onPress={() => setDefaultAddress(item.id)} />}
        </View>
      )}
    />
  );
};
L'adresse par défaut est automatiquement sélectionnée lors du checkout.

🔄 Géocodage direct & inverse

Convertir une adresse texte en coordonnées GPS, et inversement.

import { Geocoder } from 'react-native-geocoding';

// Initialisation avec clé API Google Maps
Geocoder.init('VOTRE_CLE_API');

// Adresse → coordonnées
const geocodeAddress = async (address) => {
  const result = await Geocoder.from(address);
  const { lat, lng } = result.results[0].geometry.location;
  return { latitude: lat, longitude: lng };
};

// Coordonnées → adresse
const reverseGeocode = async (lat, lng) => {
  const result = await Geocoder.from(lat, lng);
  return result.results[0].formatted_address;
};
Nécessite une clé API Google Maps (ou MapTiler/OpenStreetMap).

✨ Bonnes pratiques adresses

  • ✔️ Validation : ville, rue, téléphone obligatoires.
  • ✔️ Limiter le nombre d'adresses (ex: max 10).
  • ✔️ Synchronisation backend si l'utilisateur est connecté.
  • ✔️ Bouton "Utiliser ma position" pour ajouter automatiquement l'adresse courante.
  • ✔️ Icône de localisation sur la carte pour centrer sur l'utilisateur.

Impact UX :

Réduction des erreurs de livraison → Moins de réclamations → Fidélisation client.
