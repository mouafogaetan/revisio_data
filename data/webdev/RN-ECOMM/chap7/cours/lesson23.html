📸 Carrousel avec React Native Snap Carousel

Permet de faire défiler des images horizontalement avec gestes, pagination et zoom.

import Carousel from 'react-native-snap-carousel';

<Carousel
  data={product.images}
  renderItem={({ item }) => (
    <Image source={{ uri: item }} style={styles.image} />
  )}
  sliderWidth={sliderWidth}
  itemWidth={itemWidth}
  autoplay={true}
  loop={true}
  onSnapToItem={(index) => console.log(index)}
/>

⭐ Affichage des avis & notation

Composant réutilisable pour afficher la moyenne et la liste des commentaires.

// Composant StarRating
const StarRating = ({ rating }) => {
  return (
    <View style={{ flexDirection: 'row' }}>
      {[1,2,3,4,5].map(star => (
        <Icon key={star} name={star <= rating ? 'star' : 'star-outline'} 
              size={20} color="#fbbf24" />
      ))}
    </View>
  );
};

// Avis utilisateur
<FlatList
  data={reviews}
  renderItem={({item}) => (
    <View>
      <Text>{item.userName}</Text>
      <StarRating rating={item.rating} />
      <Text>{item.comment}</Text>
    </View>
  )}
/>
Formule de la moyenne : $$ \bar{r} = \frac{\sum_{i=1}^{n} r_i}{n} $$

📱 Mise en œuvre complète

Structure typique d'un écran produit dans une app e-commerce camerounaise.

const ProductScreen = ({ route }) => {
  const { productId } = route.params;
  const [product, setProduct] = useState(null);
  const [quantity, setQuantity] = useState(1);
  
  useEffect(() => {
    fetchProduct(productId).then(setProduct);
  }, []);

  const addToCart = () => {
    cartStore.addItem({ ...product, quantity });
    showToast("Ajouté au panier");
  };

  return product ? (
    <ScrollView>
      <Carousel images={product.images} />
      <Text>{product.name} - {product.price} FCFA</Text>
      <QuantitySelector max={product.stock} onChange={setQuantity} />
      <StarRating rating={product.avgRating} />
      <Button title="Ajouter au panier" onPress={addToCart} />
      <ReviewsList reviews={product.reviews} />
    </ScrollView>
  ) : <Spinner />;
};

✨ Bonnes pratiques pour l'écran produit

  • ✔️ Afficher le prix immédiatement en évidence.
  • ✔️ Proposer des miniatures cliquables pour le carrousel.
  • ✔️ Ajouter un bouton "favoris" 🧡 à côté du panier.
  • ✔️ Indiquer clairement le stock restant ("plus que 3 exemplaires").
  • ✔️ Pour les variantes (couleur/taille), mettre en surbrillance celle sélectionnée.
  • ✔️ Afficher les 3 premiers avis avec possibilité de "voir tout".
