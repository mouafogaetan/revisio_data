Dark Mode dynamique

React Navigation Theme + React Native Paper + Persistance AsyncStorage

🌙 Thème système 🎨 Paper Provider 💾 AsyncStorage

🎯 Objectifs de la leçon

  • ✓ Implémenter le dark mode avec React Navigation Theme (écrans + barre de navigation)
  • ✓ Utiliser React Native Paper pour le thème dynamique des composants
  • ✓ Persister la préférence utilisateur avec AsyncStorage
  • ✓ Détecter le thème système (Appearance API) et synchroniser
💡 Pourquoi le dark mode ? Réduction de la fatigue oculaire, économie d'énergie sur écrans OLED, amélioration de l'expérience utilisateur.

2. Création du thème personnalisé

// theme.js
import { DefaultTheme as NavigationDefaultTheme, DarkTheme as NavigationDarkTheme } from '@react-navigation/native';
import { MD3LightTheme, MD3DarkTheme, adaptNavigationTheme } from 'react-native-paper';

const { LightTheme, DarkTheme } = adaptNavigationTheme({
  reactNavigationLight: NavigationDefaultTheme,
  reactNavigationDark: NavigationDarkTheme,
});

export const CombinedLightTheme = {
  ...MD3LightTheme,
  ...LightTheme,
  colors: {
    ...MD3LightTheme.colors,
    ...LightTheme.colors,
    primary: '#6366f1',
    accent: '#8b5cf6',
  },
};

export const CombinedDarkTheme = {
  ...MD3DarkTheme,
  ...DarkTheme,
  colors: {
    ...MD3DarkTheme.colors,
    ...DarkTheme.colors,
    primary: '#818cf8',
    accent: '#a78bfa',
  },
};
🔁 adaptNavigationTheme synchronise les thèmes Paper et React Navigation.
💾

3. Persistance du thème (AsyncStorage)

// ThemeContext.jsx
import React, { createContext, useState, useEffect, useContext } from 'react';
import { useColorScheme } from 'react-native';
import AsyncStorage from '@react-native-async-storage/async-storage';
import { CombinedLightTheme, CombinedDarkTheme } from './theme';

const ThemeContext = createContext();

export const ThemeProvider = ({ children }) => {
  const systemScheme = useColorScheme();
  const [isDarkMode, setIsDarkMode] = useState(false);

  useEffect(() => {
    loadThemePreference();
  }, []);

  const loadThemePreference = async () => {
    const saved = await AsyncStorage.getItem('themePreference');
    if (saved === 'dark') setIsDarkMode(true);
    else if (saved === 'light') setIsDarkMode(false);
    else setIsDarkMode(systemScheme === 'dark'); // fallback système
  };

  const toggleTheme = async () => {
    const newMode = !isDarkMode;
    setIsDarkMode(newMode);
    await AsyncStorage.setItem('themePreference', newMode ? 'dark' : 'light');
  };

  const theme = isDarkMode ? CombinedDarkTheme : CombinedLightTheme;
  return (
    <ThemeContext.Provider value={{ isDarkMode, toggleTheme, theme }}>
      {children}
    </ThemeContext.Provider>
  );
};

export const useTheme = () => useContext(ThemeContext);

4. Intégration dans l'application

// App.jsx
import { NavigationContainer } from '@react-navigation/native';
import { Provider as PaperProvider } from 'react-native-paper';
import { ThemeProvider, useTheme } from './context/ThemeContext';
import AppNavigator from './navigation/AppNavigator';

function AppContent() {
  const { theme } = useTheme();
  return (
    <PaperProvider theme={theme}>
      <NavigationContainer theme={theme}>
        <AppNavigator />
      </NavigationContainer>
    </PaperProvider>
  );
}

export default function App() {
  return (
    <ThemeProvider>
      <AppContent />
    </ThemeProvider>
  );
}
✨ Le thème est automatiquement appliqué à tous les composants Paper et à la navigation.
🔄

5. Basculer le thème (Toggle)

Ajoutez un interrupteur dans un écran ou le menu latéral :

import { Switch, Text, View } from 'react-native';
import { useTheme } from '../context/ThemeContext';
import { useTheme as usePaperTheme } from 'react-native-paper';

export default function SettingsScreen() {
  const { isDarkMode, toggleTheme } = useTheme();
  const paperTheme = usePaperTheme();

  return (
    <View style={{ flex: 1, backgroundColor: paperTheme.colors.background }}>
      <View style={{ flexDirection: 'row', justifyContent: 'space-between', padding: 20 }}>
        <Text style={{ color: paperTheme.colors.onBackground }}>Mode sombre</Text>
        <Switch value={isDarkMode} onValueChange={toggleTheme} />
      </View>
    </View>
  );
}
🌙 La bascule change instantanément l'apparence de toute l'application grâce au contexte partagé.

🏁 Résumé & Bonnes pratiques

  • React Navigation Theme : gère la couleur de la barre d'état, header, arrière-plan des écrans.
  • React Native Paper : fournit des composants qui s'adaptent automatiquement au thème.
  • AsyncStorage : persistance de la préférence utilisateur.
  • useColorScheme() : détecter le thème système (Appearance API).
  • adaptNavigationTheme : synchronisation parfaite entre Paper et Navigation.
