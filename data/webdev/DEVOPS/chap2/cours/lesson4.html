🧠 Architecture Linux : le cerveau des serveurs

Linux est un système d’exploitation open source, stable et sécurisé. Il est partout : serveurs web, smartphones Android, supercalculateurs.

📌 Composants clés :

  • Noyau (Kernel) : cœur du système, dialogue avec le matériel.
  • Shell : interpréteur de commandes (bash, zsh).
  • Système de fichiers : organisation des données.
  • Distributions : Ubuntu (débutant), Debian (stable), CentOS (entreprise).

🌍 Distribution populaire au Cameroun : Ubuntu LTS (Long Term Support) facilite la prise en main.

🔌 SSH : La porte sécurisée vers vos serveurs

SSH (Secure Shell) est un protocole permettant d’exécuter des commandes sur une machine distante de façon chiffrée.

$ ssh utilisateur@192.168.1.100

$ ssh -i ma_clef.pem ubuntu@mon-serveur.com

📝 Étapes pour une connexion SSH :

  1. Le serveur distant doit avoir un service SSH actif (OpenSSH).
  2. Connaître l'adresse IP ou le nom de domaine.
  3. Utiliser un nom d'utilisateur et un mot de passe (ou une clé SSH).
  4. Commande de base : ssh user@host

⚠️ Astuce : sous Windows, utilisez PowerShell ou Git Bash ; sous Linux/macOS, terminal intégré.

📂 Où se trouve quoi ? La hiérarchie FHS

Linux suit la norme FHS (Filesystem Hierarchy Standard). Tout part de la racine /.

/home : dossiers personnels des utilisateurs.
/etc : fichiers de configuration système.
/var : données variables (logs, bases de données).
/usr : programmes et bibliothèques utilisateurs.
/bin : commandes essentielles.
/root : dossier de l'administrateur (root). 
/
├── home/
│   └── eleve/
├── etc/
│   └── ssh/
├── var/
│   └── log/
├── usr/
│   └── bin/
└── ...

📌 Naviguer dans ces répertoires est indispensable pour tout DevOps.

⌨️ Mise en pratique : Première connexion

Simulation de connexion locale : Pour tester SSH, on peut se connecter à sa propre machine via ssh localhost (si service activé).

🔐 Générer une clé SSH (sécurité renforcée) :

$ ssh-keygen -t ed25519 -C "votre-email@exemple.com"
$ ssh-copy-id utilisateur@serveur
✅ Vérifier la connexion : ssh -v user@host
📂 Transférer fichier : scp fichier user@host:~/chemin
💡 Rappel : Dans le cadre du cours au Cameroun, nous utiliserons des VM locales ou des services cloud gratuits pour pratiquer.

📚 Exercices & ressources

Mini-projet : Installer une machine virtuelle Ubuntu (via VirtualBox) et activez SSH. Connectez-vous depuis votre terminal.

🔧 Étapes conseillées :

  • Téléchargez Ubuntu Server 22.04 LTS.
  • Lancez sudo apt install openssh-server
  • Vérifiez avec systemctl status ssh
  • Connectez-vous depuis l'hôte : ssh user@ip_vm
