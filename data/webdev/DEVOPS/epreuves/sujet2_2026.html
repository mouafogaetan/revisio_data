Partie 1

Linux : fichiers, processus, scripts (20 pts)

1. (6 pts) Commandes de base et permissions

Donnez la commande pour :

  • Afficher tous les processus en temps réel (style interactif)
  • Changer le propriétaire d’un fichier script.sh pour l’utilisateur devops
  • Ajouter le droit d’exécution pour le propriétaire sur deploy.sh

2. (6 pts) Scripting bash – conditions et boucles

Écrivez un script bash qui affiche “Pair” si un nombre passé en argument est pair, “Impair” sinon. Utilisez une structure conditionnelle.

3. (8 pts) Gestion des paquets et services

Sur Ubuntu, quelle commande permet d’installer le serveur web Nginx ? Après installation, comment démarrer le service et vérifier son statut ?

Partie 2

Git : branches, merge, rebase, conflits (20 pts)

1. (6 pts) Commandes avancées

Expliquez la différence entre git merge et git rebase. Dans quel cas privilégier rebase ?

2. (6 pts) Résolution de conflit

Deux développeurs modifient la même ligne du fichier app.js. Après un git pull, Git signale un conflit. Quelles étapes suivez-vous pour résoudre le conflit manuellement ? (commandes et actions)

3. (8 pts) Workflow avec tags et hooks

Créez un tag annoté v2.0.0 sur le commit courant. Expliquez à quoi sert un Git hook pre-commit et donnez un exemple d’utilisation concrète.

Partie 3

CI/CD & écosystème (12 pts)
