Leçon 36 : Notion de couple oxydant/réducteur

Chimie – 1ère C & D | Module : Oxydoréduction en solution aqueuse

📌 Éléments clés à retenir

1. Définition d’un couple oxydant/réducteur (Ox/Red)

Un couple redox est constitué de deux espèces chimiques : un oxydant (Ox) et un réducteur (Red) qui sont liés par un transfert d’électrons. On le note conventionnellement Ox/Red. L’oxydant est la forme capable de capter des électrons, le réducteur est la forme capable de céder des électrons.

  • Ox + n e⁻ ⇌ Red : demi-équation électronique associée au couple.
  • Exemple classique : couple Cu²⁺/Cu (Cu²⁺ est l’oxydant, Cu le réducteur).
  • Autre couple fondamental : Zn²⁺/Zn, Fe³⁺/Fe²⁺, H₃O⁺/H₂.
  • Un même élément peut exister sous plusieurs degrés d’oxydation et former différents couples (ex : Fe³⁺/Fe²⁺, Fe²⁺/Fe).

2. Demi-équation électronique : écriture et signification

La demi-équation traduit l’équilibre entre l’oxydant et le réducteur, avec échange d’électrons. Elle s’écrit toujours dans le sens réduction (gain d’électrons) par convention.

  • Forme générale : $Ox + n e^- \rightleftharpoons Red$
  • n représente le nombre d’électrons échangés (il dépend des nombres d’oxydation).
  • Pour équilibrer une demi-équation : équilibrer les atomes (sauf H et O), puis équilibrer l’oxygène avec H₂O, l’hydrogène avec H⁺, et enfin la charge avec des électrons.
  • Exemples :
    - $Cu^{2+} + 2e^- \rightleftharpoons Cu$
    - $2H_3O^+ + 2e^- \rightleftharpoons H_2 + 2H_2O$
    - $Fe^{3+} + e^- \rightleftharpoons Fe^{2+}$

3. Identifier oxydant et réducteur dans un couple

Dans la notation Ox/Red :

  • Oxydant (Ox) : espèce chimique placée à gauche (ex : Cu²⁺, Fe³⁺, H₃O⁺). Elle peut gagner des électrons.
  • Réducteur (Red) : espèce chimique placée à droite (ex : Cu, Fe²⁺, H₂). Elle peut perdre des électrons.
  • Une réaction d’oxydoréduction met en jeu deux couples : l’oxydant d’un couple réagit avec le réducteur de l’autre couple.

💡 Exemples de compréhension

Exemple 1 : Le couple Zinc / Ion zinc (Zn²⁺/Zn)

On considère le couple Zn²⁺/Zn. La demi-équation associée est : $Zn^{2+} + 2e^- \rightleftharpoons Zn$.

🔍 Analyse :
  • Oxydant = Zn²⁺ (il peut capter 2 électrons pour devenir Zn métal).
  • Réducteur = Zn (il peut perdre 2 électrons pour devenir Zn²⁺).
  • Lorsque le zinc métallique est plongé dans une solution contenant des ions Zn²⁺, l’équilibre s’établit : le zinc peut se dissoudre (oxydation) ou les ions Zn²⁺ se déposer (réduction) selon les conditions.
💡 Astuce : Toujours se souvenir que l’oxydant est l’espèce qui attire les électrons (souvent ion positif ou molécule contenant un élément à haut n.o.), tandis que le réducteur est l’espèce qui cède les électrons (souvent métal ou ion à bas n.o.).

Exemple 2 : Écrire la demi-équation du couple permanganate / ion manganèse (MnO₄⁻/Mn²⁺)

En milieu acide, le couple MnO₄⁻/Mn²⁺ est très utilisé. Établissons la demi-équation de réduction.

✅ Démarche :
1. Écrire l’oxydant et le réducteur : $MnO_4^- \rightarrow Mn^{2+}$
2. Équilibrer Mn : déjà 1 atome de chaque côté.
3. Équilibrer O en ajoutant H₂O : $MnO_4^- \rightarrow Mn^{2+} + 4H_2O$
4. Équilibrer H en ajoutant H⁺ (milieu acide) : $MnO_4^- + 8H^+ \rightarrow Mn^{2+} + 4H_2O$
5. Équilibrer les charges : gauche : -1 + 8 = +7 ; droite : +2 → il faut ajouter 5 e⁻ à gauche :
$MnO_4^- + 8H^+ + 5e^- \rightarrow Mn^{2+} + 4H_2O$.
Dans l’eau, on utilise souvent H₃O⁺ : $MnO_4^- + 8H_3O^+ + 5e^- \rightarrow Mn^{2+} + 12H_2O$.
🧪 Rappel : Cette demi-équation montre que MnO₄⁻ (oxydant) gagne 5 électrons pour donner Mn²⁺ (réducteur). Le permanganate est un oxydant puissant.

Exemple 3 : Relier oxydant et réducteur dans une pile

Dans la pile Daniell (Zn/Cu²⁺), les couples mis en jeu sont Zn²⁺/Zn et Cu²⁺/Cu.

Fonctionnement :
- À l’anode (oxydation) : $Zn \rightarrow Zn^{2+} + 2e^-$ (Zn joue le rôle de réducteur).
- À la cathode (réduction) : $Cu^{2+} + 2e^- \rightarrow Cu$ (Cu²⁺ joue le rôle d’oxydant).
- On voit bien que chaque couple fournit un oxydant et un réducteur. L’oxydant Cu²⁺ capte les électrons fournis par le réducteur Zn.
⚡ Une réaction redox spontanée se produit toujours entre l’oxydant le plus fort et le réducteur le plus fort.

⭐ À mémoriser : la demi-équation permet de prévoir le nombre d’électrons échangés et de déterminer le rôle (oxydant/réducteur) dans une transformation chimique.

⚠️ Pièges courants / À ne pas confondre

  • Confondre oxydant et réducteur : Une astuce mnémotechnique : Oxydant = celui qui "prend" des électrons (donc son nombre d’oxydation diminue). Réducteur = celui qui "donne" des électrons (son n.o. augmente).
  • Oublier l’équilibre des charges dans les demi-équations : Toujours vérifier que la somme des charges électriques est équilibrée entre réactifs et produits (grâce aux électrons).
  • Écrire la demi-équation dans le mauvais sens : La convention universelle est de toujours écrire réduction (Ox + n e⁻ → Red). Cela évite les erreurs lors de la combinaison des couples.
  • Croire qu’un oxydant est toujours un cation : Certains oxydants sont des molécules neutres (ex : Cl₂, O₂) ou des anions (MnO₄⁻, Cr₂O₇²⁻).
  • Négliger le milieu (acide/basique) : Les demi-équations dépendent du pH ; en milieu acide on utilise H⁺/H₂O, en milieu basique on utilise OH⁻/H₂O.

🔬 Représentation schématique d’un couple

Légende : L’oxydant (Ox) se réduit en acceptant $n$ électrons ; le réducteur (Red) s’oxyde en cédant $n$ électrons. Les deux formes constituent un couple redox.

✏️ À retenir : Ox + n e⁻ ⇌ Red (équilibre dynamique).
