Leçon 5 : La biosynthèse des protéines

SVTEEHB · Première C & TI · Expression et transmission de l'information génétique

📌 Éléments clés à retenir

1. Qu'est-ce que la biosynthèse des protéines ?

La biosynthèse des protéines, ou traduction, est le mécanisme cellulaire qui permet la fabrication d’une protéine à partir de l’information portée par l’ARN messager (ARNm). Cette étape fait partie de l'expression des gènes et se déroule dans le cytoplasme, sur les ribosomes.

  • ADN → transcription → ARNm → traduction → Protéine (dogme central de la biologie moléculaire).
  • La séquence des acides aminés de la protéine est déterminée par la séquence de nucléotides de l'ARNm.
  • Chaque triplet de nucléotides sur l'ARNm s'appelle un codon.

2. La transcription : de l'ADN à l'ARNm

Avant la synthèse de la protéine, l'information contenue dans l'ADN doit être transcrite en ARNm.

  • Lieu : Noyau de la cellule (chez les eucaryotes).
  • Enzyme : ARN polymérase.
  • Principe : L'ARN polymérase lit le brin d'ADN et assemble des ribonucléotides complémentaires (A→U, T→A, C→G, G→C).
  • L'ARNm obtenu est une copie épissée (exons conservés, introns éliminés) chez les eucaryotes.
Figure 1 : La transcription - synthèse de l'ARNm à partir d'une matrice d'ADN.

3. La traduction : synthèse de la protéine

La traduction se déroule en trois étapes : initiation, élongation et terminaison. Elle nécessite des ribosomes, des ARNt et des acides aminés.

  • Initiation : Le petit sous‑unité du ribosome se fixe sur l'ARNm au niveau du codon START (AUG). L'ARNt portant la méthionine s'associe.
  • Élongation : Les ARNt apportent les acides aminés correspondant à chaque codon. Les acides aminés sont liés par des liaisons peptidiques.
  • Terminaison : Arrivée à un codon STOP (UAA, UAG, UGA). La protéine est libérée.
  • Rôle des ARNt : Ils possèdent un anticodon complémentaire du codon de l'ARNm et fixent l'acide aminé spécifique.
Figure 2 : Mécanisme de la traduction – le ribosome lit l'ARNm de 5' vers 3' et assemble les acides aminés.

4. Le code génétique

Le code génétique est l'ensemble des correspondances entre les codons (triplets de l'ARNm) et les acides aminés.

  • Universalité : Le code génétique est quasiment identique chez tous les êtres vivants (bactéries, plantes, animaux).
  • Redondance : Plusieurs codons différents peuvent coder le même acide aminé (ex : GCU, GCC, GCA, GCG codent l'alanine).
  • Non chevauchant : Les codons sont lus de manière linéaire et sans chevauchement.
  • Codons particuliers : AUG = démarrage (méthionine), UAA/UAG/UGA = stop.
💡 Astuce : Pour retenir le code génétique, pensez que chaque triplet correspond à une "lettre" du langage des protéines.

📍 Devenir des protéines synthétisées

Une fois synthétisées, les protéines sont acheminées vers leur lieu de fonction :

  • Protéines cytoplasmiques : restent dans le cytosol (ex : enzymes de la glycolyse).
  • Protéines membranaires ou de sécrétion : synthétisées sur les ribosomes du réticulum endoplasmique rugueux, puis modifiées dans l'appareil de Golgi avant d'être exportées (ex : hormones, anticorps).
  • Protéines de structure : forment le cytosquelette, la matrice extracellulaire, etc.

💡 Exemples de compréhension

Exemple 1 : Lecture d'un codon

Une séquence d'ARNm est la suivante : AUG - GCU - UUU - UAC - UGA. Déterminer la séquence d'acides aminés correspondante en utilisant le code génétique simplifié (AUG = Met, GCU = Ala, UUU = Phe, UAC = Tyr, UGA = Stop).

Résolution : On lit chaque codon de 5' vers 3' :
- AUG → Méthionine (Met) (démarrage)
- GCU → Alanine (Ala)
- UUU → Phénylalanine (Phe)
- UAC → Tyrosine (Tyr)
- UGA → Stop → fin de la synthèse.
La protéine formée est donc : Met - Ala - Phe - Tyr.

💬 Rappel : le codon stop n'est pas traduit en acide aminé ; il provoque l'arrêt de la traduction.

Exemple 2 : Mutation et conséquence sur la protéine

Une mutation remplace le codon GCU (Ala) par le codon GAU (Asp). Quel est l'effet sur la protéine ?

Analyse : Le codon GCU code l'alanine, tandis que GAU code l'acide aspartique (Asp). La mutation entraîne le remplacement d'un acide aminé par un autre. Selon la position dans la protéine, cela peut modifier la structure et la fonction de la protéine (par exemple, rendre une enzyme inactive).

💡 Certaines mutations sont "silencieuses" si le nouveau codon code le même acide aminé (redondance du code génétique).

⚠️ Pièges courants / À ne pas confondre

  • Confondre transcription et traduction : La transcription a lieu dans le noyau (ADN → ARNm), la traduction dans le cytoplasme (ARNm → protéine).
  • Codon vs anticodon : Le codon est sur l'ARNm, l'anticodon est sur l'ARNt. Ils sont complémentaires.
  • Oublier le rôle des ribosomes : Ce sont les "usines" à protéines, mais ce sont les ARNt qui apportent les acides aminés.
  • Penser que tous les ARNm donnent directement une protéine fonctionnelle : Chez les eucaryotes, l'ARNm subit un épissage (élimination des introns) avant la traduction.
