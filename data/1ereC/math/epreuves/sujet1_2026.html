❓ Question 1 (1 pt)
Le discriminant Δ du trinôme 2x² - 3x + 1 est égal à :
  • A. Δ = 1
  • B. Δ = 17
  • C. Δ = -7
  • D. Δ = 9

    • ✅ Réponse correcte : A. Δ = 1

    🔹 Δ = b² - 4ac = (-3)² - 4×2×1 = 9 - 8 = 1.

    ❓ Question 2 (1 pt)
    L'équation 3x² + 2x + 5 = 0 admet dans ℝ :
  • A. Deux solutions distinctes
  • B. Une solution double
  • C. Aucune solution
  • D. Une infinité de solutions

    • ✅ Réponse correcte : C. Aucune solution

    🔹 Δ = 4 - 60 = -56 < 0, donc l'équation n'a pas de solution réelle.

    ❓ Question 3 (1 pt)
    Si un polynôme P(x) admet α = 2 pour racine, alors on peut factoriser P(x) par :
  • A. (x + 2)
  • B. (x - 2)
  • C. (2x - 1)
  • D. (x + 1)

    • ✅ Réponse correcte : B. (x - 2)

    🔹 Si α est racine, alors P(α)=0, donc P(x) est divisible par (x - α).

    ❓ Question 4 (1 pt)
    L'ensemble des solutions de l'inéquation x² - 4 > 0 est :
  • A. ]-2 ; 2[
  • B. ]-∞ ; -2[ ∪ ]2 ; +∞[
  • C. ]-∞ ; -2] ∪ [2 ; +∞[
  • D. [-2 ; 2]

    • ✅ Réponse correcte : B. ]-∞ ; -2[ ∪ ]2 ; +∞[

    🔹 x² - 4 > 0 ⇔ (x-2)(x+2) > 0. Tableau de signe → solutions pour x < -2 ou x > 2.

    ❓ Question 5 (1 pt)
    Pour résoudre l'équation irrationnelle √(2x+1) = x-1, la condition d'existence est :
  • A. 2x+1 ≥ 0
  • B. x-1 ≥ 0
  • C. 2x+1 ≥ 0 ET x-1 ≥ 0
  • D. 2x+1 ≥ 0 OU x-1 ≥ 0

    • ✅ Réponse correcte : C. 2x+1 ≥ 0 ET x-1 ≥ 0

    🔹 La racine carrée √(f(x)) est définie si f(x) ≥ 0, et l'égalité √(f(x)) = g(x) impose g(x) ≥ 0 car une racine carrée est toujours positive ou nulle.

    ❓ Question 6 (1 pt)
    La somme des racines du trinôme 2x² - 8x + 6 = 0 est égale à :
  • A. -4
  • B. 4
  • C. 3
  • D. -8

    • ✅ Réponse correcte : B. 4

    🔹 S = -b/a = -(-8)/2 = 8/2 = 4.

    ❓ Question 7 (1 pt)
    Le couple (x ; y) solution du système { x + y = 5 et 2x - y = 4 } est :
  • A. (3 ; 2)
  • B. (2 ; 3)
  • C. (1 ; 4)
  • D. (4 ; 1)

    • ✅ Réponse correcte : A. (3 ; 2)

    🔹 Par addition : (x+y)+(2x-y)=5+4 → 3x=9 → x=3, puis y=5-3=2.

    ❓ Question 8 (1 pt)
    Le système { x² + y² = 25 et x + y = 7 } se ramène à une équation du second degré d'inconnue x en substituant y = 7 - x. L'équation obtenue est :
  • A. 2x² - 14x + 24 = 0
  • B. x² - 7x + 12 = 0
  • C. 2x² - 14x + 49 = 0
  • D. x² - 14x + 24 = 0

    • ✅ Réponse correcte : A. 2x² - 14x + 24 = 0

    🔹 x² + (7-x)² = 25 → x² + 49 - 14x + x² = 25 → 2x² - 14x + 24 = 0.

    📝 PARTIE B : RÉSOLUTION DE PROBLÈMES (12 points)
    Les exercices sont indépendants. Rédigez vos réponses avec soin.
    🔢 EXERCICE 1 : Équation et inéquation du second degré (4 pts)
    1. Résoudre dans ℝ l'équation : 2x² - 5x + 2 = 0. (1 pt)

    2. En déduire l'ensemble des solutions de l'inéquation : 2x² - 5x + 2 ≤ 0. (1,5 pt)

    3. Soit le polynôme P(x) = 2x³ - 5x² + 2x. Vérifier que P(x) = x(2x² - 5x + 2), puis résoudre P(x) = 0. (1,5 pt)

    ✅ Corrigé

    1. Δ = 25 - 16 = 9, √Δ = 3. Racines : x₁ = (5-3)/4 = 0,5 et x₂ = (5+3)/4 = 2. S = {0,5 ; 2}

    2. Le coefficient a=2 > 0, donc le trinôme est négatif entre les racines. S = [0,5 ; 2]

    3. P(x) = x(2x² - 5x + 2). P(x)=0 ⇔ x=0 ou 2x²-5x+2=0. S = {0 ; 0,5 ; 2}

    🔢 EXERCICE 2 : Polynôme de degré 3 et relations racines (3 pts)
    On considère le polynôme P(x) = x³ - 6x² + 11x - 6.

    1. Vérifier que 1 est une racine de P. (0,5 pt)
    2. Déterminer les réels a, b, c tels que P(x) = (x-1)(ax² + bx + c). (1,5 pt)
    3. En déduire toutes les racines de P. (1 pt)

    ✅ Corrigé

    1. P(1) = 1 - 6 + 11 - 6 = 0, donc 1 est racine.

    2. Par division euclidienne ou identification : P(x) = (x-1)(x² - 5x + 6).

    3. x² - 5x + 6 = 0 → Δ = 25-24=1 → x = 2 ou x = 3. Racines : S = {1 ; 2 ; 3}

    🔢 EXERCICE 3 : Équation irrationnelle (2 pts)
    Résoudre dans ℝ l'équation : √(x+2) = x - 4. (Indication : déterminer d'abord les conditions d'existence)

    ✅ Corrigé

    Conditions : x+2 ≥ 0 ⇔ x ≥ -2 ET x-4 ≥ 0 ⇔ x ≥ 4. Donc D = [4 ; +∞[

    Élévation au carré : (√(x+2))² = (x-4)² → x+2 = x² - 8x + 16 → 0 = x² - 9x + 14

    Δ = 81 - 56 = 25, √Δ = 5. x₁ = (9-5)/2 = 2 (non retenu car x≥4), x₂ = (9+5)/2 = 7.

    Solution : S = {7}

    🔢 EXERCICE 4 : Système d'équations (3 pts)
    Résoudre dans ℝ² le système suivant :
    { x + y = 7
    { xy = 10

    ✅ Corrigé

    Les nombres x et y sont les racines de l'équation t² - St + P = 0, soit t² - 7t + 10 = 0.

    Δ = 49 - 40 = 9, √Δ = 3. t₁ = (7-3)/2 = 2, t₂ = (7+3)/2 = 5.

    Les solutions sont les couples (2 ; 5) et (5 ; 2).
