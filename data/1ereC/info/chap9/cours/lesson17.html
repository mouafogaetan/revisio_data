UE 17 : Insérer les fichiers multimédias dans une page web

HTML & Multimédia · Première C, D, E · Probatoire

📌 Éléments clés à retenir

1. Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage standard utilisé pour créer et structurer le contenu des pages web. Ce n'est pas un langage de programmation, mais un langage de balisage qui permet d'organiser le texte, d'insérer des images, des vidéos, des sons et de créer des liens hypertextes.

  • Les pages HTML sont interprétées par les navigateurs (Chrome, Firefox, Safari, etc.).
  • Les balises sont généralement composées d'une balise ouvrante <nom> et d'une balise fermante </nom>.
  • Certaines balises sont dites "orphelines" (ou auto-fermantes) : <img>, <br>, <hr>.

2. Structure de base d'une page HTML

Toute page HTML respecte une structure minimale obligatoire pour être correctement interprétée.

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
  </head>
  <body>
    <!-- Contenu visible de la page -->
  </body>
</html>
  • <!DOCTYPE html> : Déclare la version HTML5 du document.
  • <html> : Élément racine qui contient toute la page.
  • <head> : Contient les métadonnées (titre, encodage, liens CSS, etc.).
  • <body> : Contient tout le contenu visible par l'utilisateur.

3. Insérer des images avec <img>

La balise <img> permet d'afficher une image dans une page web. C'est une balise orpheline (pas de balise fermante).

<img src="logo.png" alt="Description de l'image" width="200" height="150">
  • src (source) : Chemin vers le fichier image (URL ou chemin local). Obligatoire.
  • alt (alternative text) : Texte alternatif affiché si l'image ne charge pas. Recommandé (accessibilité).
  • width et height : Dimensions en pixels (optionnel, mais améliore le rendu).
  • Formats supportés : JPEG, PNG, GIF, SVG, WebP, etc.
💡 Astuce : Toujours renseigner l'attribut alt pour l'accessibilité (malvoyants utilisant un lecteur d'écran) et le référencement (SEO).

4. Insérer des titres (<h1> à <h6>)

Les balises de titre hiérarchisent le contenu textuel. <h1> est le plus important (titre principal), <h6> le moins important.

<h1>Titre principal de la page</h1>
<h2>Sous-titre important</h2>
<h3>Section secondaire</h3>
  • Un seul <h1> par page (recommandé pour le SEO).
  • Les navigateurs appliquent des styles par défaut (tailles différentes, gras).
  • Ces balises aident à structurer logiquement le document.

5. Insérer des fichiers audio et vidéo

HTML5 introduit les balises <audio> et <video> pour intégrer facilement du multimédia sans plugins.

<audio controls>
  <source src="musique.mp3" type="audio/mpeg">
  Votre navigateur ne supporte pas l'audio.
</audio>

<video controls width="640">
  <source src="video.mp4" type="video/mp4">
  Votre navigateur ne supporte pas la vidéo.
</video>
  • L'attribut controls ajoute les boutons de lecture/pause/volume.
  • La balise <source> permet de proposer plusieurs formats pour compatibilité.
  • Formats audio : MP3, OGG, WAV. Formats vidéo : MP4, WebM, OGG.

💡 Exemples de compréhension

Exemple 1 : Page simple avec image et titres

Créer une page HTML qui affiche un titre principal "Bienvenue au Cameroun", un sous-titre "Les paysages magnifiques", et une image du Mont Cameroun.

<!DOCTYPE html>
<html>
  <head>
    <title>Découverte du Cameroun</title>
  </head>
  <body>
    <h1>Bienvenue au Cameroun</h1>
    <h2>Les paysages magnifiques</h2>
    <img src="mont-cameroun.jpg" alt="Vue du Mont Cameroun avec sa végétation luxuriante" width="500">
  </body>
</html>
💬 Astuce : Toujours placer les images dans le même dossier que la page HTML ou dans un sous-dossier (ex: images/) pour simplifier le chemin.

Exemple 2 : Intégration d'une vidéo avec plusieurs formats

On souhaite intégrer une vidéo de présentation du système éducatif camerounais. Pour être sûr que tous les navigateurs la lisent, on fournit les formats MP4 et WebM.

<video controls width="640" height="360" poster="vignette.jpg">
  <source src="education.mp4" type="video/mp4">
  <source src="education.webm" type="video/webm">
  Votre navigateur ne supporte pas la balise vidéo.
</video>

Explication : L'attribut poster affiche une image de prévisualisation avant la lecture. Les balises <source> sont essayées dans l'ordre : si MP4 ne marche pas, le navigateur essaie WebM.

Exemple 3 : Image avec lien cliquable

Transformer une image en lien hypertexte vers un autre site.

<a href="https://www.cameroun-tourisme.com">
  <img src="drapeau-cameroun.png" alt="Drapeau du Cameroun" width="200">
</a>

Résolution : La balise <a> (ancre) encadre l'image. En cliquant sur l'image, l'utilisateur est redirigé vers le site indiqué dans href.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Oublier l'attribut alt dans la balise <img>. Cela nuit à l'accessibilité et au référencement.
  • Attention : Confondre les balises <img> (orpheline) avec <image> qui n'existe pas en HTML.
  • Piège classique : Utiliser des fichiers image trop volumineux sans spécifier width/height, ce qui ralentit le chargement et déforme la page.
  • À ne pas oublier : Les chemins des fichiers sont sensibles à la casse sur les serveurs Linux. "Image.JPG" ≠ "image.jpg".
  • Erreur : Oublier de fermer les balises de titre ou d'imbriquer correctement les éléments.

Tableau des principales balises

Balise Description Attributs essentiels
