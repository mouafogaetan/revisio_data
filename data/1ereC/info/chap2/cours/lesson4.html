Leçon 3 : Introduction à l'invite de commandes

Informatique · Première (Séries C, D, E) · Module 1: Environnement numérique

📌 Éléments clés à retenir

1. Définition de l'invite de commandes

L'invite de commandes (ou CMD pour « Command ») est une interface textuelle qui permet d'interagir directement avec le système d'exploitation en tapant des instructions. Contrairement à l'interface graphique (fenêtres, icônes, souris), on utilise ici le clavier pour saisir des commandes.

  • On l'appelle aussi console ou shell (interpréteur de commandes).
  • Elle permet d'exécuter des programmes, gérer des fichiers et configurer le système sans passer par la souris.
  • Très utilisée par les administrateurs système et les développeurs pour sa rapidité et sa puissance.

2. Comment ouvrir l'invite de commandes (Windows)

Il existe plusieurs méthodes pour lancer l'invite de commandes sous Windows. La plus courante est :

  • Appuyer sur Win + R (touche Windows + R), taper cmd et valider par Entrée.
  • Ou bien chercher "cmd" ou "invite de commandes" dans le menu Démarrer.
  • Pour les besoins avancés (installation, modification système), il faut l'exécuter en tant qu'administrateur (clic droit → "Exécuter en tant qu'administrateur").

3. Syntaxe générale d'une commande

Une commande suit une structure précise :
[commande] [paramètres] [options]

  • Commande : l'action à réaliser (ex: DIR, COPY).
  • Paramètres : les éléments sur lesquels agit la commande (ex: un nom de fichier).
  • Options (ou commutateurs) : elles modifient le comportement de la commande, souvent préfixées par / (ex: /W pour un affichage large).

Exemple : DIR /W C:\Users → la commande DIR avec l'option /W (affichage large) et le paramètre C:\Users (le dossier à lister).

4. Obtenir de l'aide : la commande /?

Pour chaque commande, vous pouvez afficher son aide et la liste de ses options en tapant la commande suivie de /?.

  • Exemple : DIR /? → affiche toutes les options de la commande DIR (affichage par pages, tri, etc.).
  • C'est la meilleure façon de découvrir une commande et ses possibilités sans internet.

💡 Exemples de compréhension

Exemple 1 : Se déplacer dans l'arborescence (CD)

Votre invite de commandes affiche C:\Users\Eleve>. Vous souhaitez accéder au dossier "Documents" qui se trouve dans votre dossier personnel.

Commande à taper : CD Documents puis Entrée.
Le prompt devient : C:\Users\Eleve\Documents>. Vous êtes maintenant dans le dossier Documents.

💬 Astuce : CD .. permet de remonter d'un niveau (dossier parent). CD \ vous ramène à la racine du disque (C:).

Exemple 2 : Afficher le contenu d'un dossier (DIR)

Vous êtes dans le dossier C:\Projets et vous voulez voir la liste de tous les fichiers et sous-dossiers qu'il contient.

Commande : DIR.
La console affichera une liste avec les noms, tailles, dates de modification. Pour un affichage plus lisible (par écran), utilisez DIR /P (Pause). Pour voir aussi les fichiers cachés : DIR /A.

💡 Pour vider l'écran et repartir de zéro, tapez simplement CLS (Clear Screen).

Exemple 3 : Créer un répertoire (MD)

Vous devez organiser vos travaux. Créez un dossier nommé "EXAMENS" à l'intérieur de C:\Users\Eleve\Documents.

Étapes :
1. Assurez-vous d'être dans C:\Users\Eleve\Documents (utilisez CD Documents si besoin).
2. Tapez MD EXAMENS (ou MKDIR EXAMENS).
3. Vérifiez avec DIR : le nouveau dossier "EXAMENS" apparaît.

⚠️ Les noms de dossiers ne doivent pas contenir d'espaces (ou alors utilisez des guillemets : MD "EXAMENS 2025").

⚠️ Pièges courants / À ne pas confondre

  • Confondre les commandes Windows et Linux : DIR est pour Windows, ls pour Linux. Sur CMD, ls ne fonctionne pas.
  • Oublier l'espace entre la commande et ses paramètres : CDDocuments est une erreur, il faut CD Documents.
  • Ne pas connaître le répertoire courant : Le prompt (ex: C:\>) vous indique où vous êtes. Si vous êtes perdu, tapez CD sans paramètre pour afficher le chemin complet.
  • Supprimer un dossier non vide avec RD : RD MonDossier ne fonctionne que si le dossier est vide. Utilisez RD /S MonDossier pour supprimer un dossier avec son contenu (attention, irréversible).

🚀 Petits trucs en plus

Historique et auto-complétion

  • Utilisez les flèches haut/bas pour naviguer dans l'historique des commandes tapées.
  • La touche TABULATION auto-complète les noms de fichiers et dossiers (gain de temps et évite les fautes de frappe).
  • Pour interrompre une commande qui tourne en boucle (ou trop longue), tapez Ctrl + C.
