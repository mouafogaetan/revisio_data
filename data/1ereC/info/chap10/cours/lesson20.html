📌 Éléments clés à retenir

1. Variables en JavaScript

Une variable est un espace mémoire nommé qui stocke une valeur (nombre, texte, booléen…). En JavaScript, on déclare une variable avec les mots-clés let, const ou var (ancien).

  • let age = 17; → variable modifiable.
  • const PI = 3.14; → constante (ne peut pas être réaffectée).
  • var nom = "Elie"; → ancienne méthode, à éviter en faveur de let.
  • JavaScript est un langage à typage dynamique : une même variable peut changer de type.

2. Fonctions prédéfinies essentielles

JavaScript fournit des fonctions intégrées pour interagir avec l'utilisateur et manipuler les données.

  • prompt(message, défaut) : Ouvre une boîte de dialogue pour saisir une valeur. Retourne une chaîne de caractères.
  • document.write(contenu) : Écrit directement dans la page HTML (à utiliser avec précaution).
  • parseFloat(chaine) : Convertit une chaîne en nombre décimal (ex: "12.5" → 12.5).
  • parseInt(chaine) : Convertit une chaîne en nombre entier (ex: "42.7" → 42).
  • alert(message) : Affiche une boîte d'alerte simple.
💡 Astuce : prompt() retourne toujours une chaîne. Pour faire des calculs, utilisez parseFloat() ou parseInt().

3. Conversion de types (typage)

Les saisies utilisateur via prompt() sont des chaînes. Les opérations mathématiques nécessitent des nombres.

  • Exemple : let nb = prompt("Entrez un nombre"); → nb est une chaîne.
  • let valeur = parseFloat(nb); → conversion en nombre réel.
  • Si la conversion échoue, parseFloat() retourne NaN (Not a Number).

💡 Exemples de compréhension

Exemple 1 : Saisie et affichage d'un message personnalisé

On demande à l'utilisateur son prénom, puis on affiche "Bonjour [prénom] !" dans la page web.

Code JavaScript :

let prenom = prompt("Quel est ton prénom ?");
document.write("<h2>Bonjour " + prenom + " !</h2>");

✔️ Explication : La variable prenom récupère la saisie. L'opérateur + concatène les chaînes. document.write() injecte le résultat dans le HTML.

💬 Astuce : Évitez document.write() après le chargement complet de la page (il peut écraser le contenu). Pour des projets réels, on préfère innerHTML.

Exemple 2 : Calcul de la moyenne de deux notes (avec conversion)

L'utilisateur saisit deux notes sur 20. Le programme calcule et affiche la moyenne avec alerte.

Code JavaScript :

let note1 = prompt("Première note :");
let note2 = prompt("Deuxième note :");
let n1 = parseFloat(note1);
let n2 = parseFloat(note2);
let moyenne = (n1 + n2) / 2;
alert("La moyenne est : " + moyenne);

✔️ Détail : parseFloat() convertit les chaînes en nombres pour que l'addition fonctionne. Si on avait écrit note1 + note2 sans conversion, cela aurait concaténé les chaînes (ex: "12"+"14" = "1214").

💡 Pensez à gérer les cas où l'utilisateur n'entre pas un nombre (vérification avec isNaN()).

Exemple 3 : Utilisation de parseInt() pour un âge

Demander l'année de naissance et calculer l'âge approximatif.

Code :

let anneeNaiss = prompt("Année de naissance :");
let anneeActuelle = 2026;
let age = anneeActuelle - parseInt(anneeNaiss);
document.write("<p>Vous avez " + age + " ans (environ).</p>");

✔️ parseInt() extrait la partie entière de la chaîne. Si l'utilisateur saisit "2005", la soustraction fonctionne correctement.

⚠️ Pièges courants / À ne pas confondre

  • Confondre concaténation et addition : "10" + "5" donne "105" (car chaînes). Utilisez parseFloat() ou parseInt() avant d'additionner.
  • Oublier que prompt() retourne toujours une chaîne : Même si l'utilisateur tape un chiffre, c'est une chaîne. Convertir systématiquement pour les maths.
  • Utiliser document.write() après le chargement complet : Cela peut remplacer toute la page. Privilégier d'autres méthodes comme innerHTML ou console.log() pour déboguer.
  • Ne pas initialiser les variables : Une variable non initialisée vaut undefined et peut causer des erreurs silencieuses.
