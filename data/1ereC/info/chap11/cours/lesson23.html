🔧 Définitions & éléments clés

🗣️ Qu'est-ce que le langage C ?

Le langage C est un langage de programmation structuré et compilé, créé dans les années 1970 par Dennis Ritchie. Il est connu pour sa performance, sa portabilité et son contrôle précis sur le matériel. Il sert de base à de nombreux langages modernes (C++, Java, C#).

  • Structuré : Organisation en blocs, fonctions et instructions séquentielles, conditionnelles ou répétitives.
  • Compilé : Le code source est transformé en langage machine via un compilateur (ex: GCC).
  • Utilisations fréquentes : Systèmes d'exploitation (Linux, Windows), jeux vidéo, embarqué, bases de données.

🏗️ Structure de base d'un programme C

Tout programme en C suit une organisation minimale : inclusion des bibliothèques, déclaration de la fonction principale main(), puis les instructions entre accolades.

/* Exemple de programme minimal */ #include <stdio.h> // Bibliothèque standard I/O int main() { printf("Bienvenue en langage C !\n"); return 0; // Indique que le programme s'est terminé correctement }
  • #include <stdio.h> : permet d'utiliser printf() et scanf().
  • int main() : point d'entrée obligatoire du programme.
  • return 0; : code de retour signifiant "succès".
  • Les instructions se terminent par ; (point-virgule).

📚 La bibliothèque stdio.h

#include <stdio.h> est une directive de préprocesseur qui intègre la bibliothèque standard d'entrée/sortie. Sans elle, les fonctions d'affichage et de saisie (printf, scanf) sont inconnues du compilateur.

💡 À retenir : stdio.h = STandarD Input/Output. Elle contient des fonctions essentielles pour dialoguer avec l'utilisateur.

💻 Exemples commentés

Exemple n°1 : Afficher un message

Écrire un programme C qui affiche "Bonjour, je découvre le C !" à l'écran.

#include <stdio.h> int main() { printf("Bonjour, je découvre le C !\n"); return 0; }

🔍 Explication :
• #include <stdio.h> rend printf disponible.
• La fonction main() s'exécute au démarrage.
• printf(...) affiche le texte. Le \n ajoute un retour à la ligne.
• return 0 termine le programme avec succès.

💬 Astuce : printf peut afficher des variables grâce aux formats comme %d (entier) ou %f (réel).

Exemple n°2 : Lire et afficher un nombre

Programme qui demande l'âge de l'utilisateur et affiche "Vous avez X ans".

#include <stdio.h> int main() { int age; printf("Entrez votre age : "); scanf("%d", &age); printf("Vous avez %d ans.\n", age); return 0; }

🔍 Explication :
• int age; déclare une variable de type entier.
• scanf("%d", &age); lit un entier au clavier et le stocke dans age. Le symbole & donne l'adresse mémoire.
• printf("%d", age) insère la valeur de age à la place de %d.

🧠 À retenir : Ne jamais oublier & devant la variable dans scanf (sauf pour les chaînes de caractères).

Exemple n°3 : Addition de deux entiers

Lire deux nombres entiers, afficher leur somme.

#include <stdio.h> int main() { int a, b, somme; printf("Donner a et b : "); scanf("%d %d", &a, &b); somme = a + b; printf("Somme = %d\n", somme); return 0; }

Étapes : déclaration de trois entiers, lecture de a et b, calcul avec l'opérateur +, puis affichage.

✨ Le C permet d'effectuer tous les opérateurs arithmétiques classiques : + - * / % (modulo).

⚠️ Pièges fréquents & bonnes pratiques

  • Erreur n°1 : Oublier #include <stdio.h> → le compilateur ne reconnaît ni printf ni scanf.
  • Erreur n°2 : Point-virgule manquant à la fin d'une instruction → erreur de syntaxe.
  • Erreur n°3 : Dans scanf, oublier le & devant une variable simple → comportement indéfini, souvent un crash.
  • Erreur n°4 : Utiliser une variable sans l'avoir initialisée → valeur aléatoire (poubelle mémoire).
  • Bonnes pratiques : Toujours indenter le code, commenter les parties importantes, et retourner 0 dans main().

🔄 Lien avec l'algorithmique

Du pseudo-code au C

En classe de première, vous avez vu des algorithmes en français structuré. Le langage C permet de traduire ces algorithmes de manière précise.

  • Algorithme : Ecrire("Bonjour") → C : printf("Bonjour");
  • Algorithme : Lire(age) → C : scanf("%d", &age);
  • Algorithme : Si condition alors ... Sinon ... → C : if (condition) { ... } else { ... }
  • Boucle POUR : pour i de 1 à N faire ... → C : for(i=1; i<=N; i++) { ... }
📢 À retenir pour l'examen : Le C est un langage compilé, très proche de la machine, mais sa logique suit exactement les structures algorithmiques étudiées (séquences, conditions, boucles).

✍️ Auto-évaluation (vérifie ta compréhension)

Question flash

Que manque-t-il dans ce programme C pour qu'il fonctionne correctement ?

int main() { int x = 10; printf("Valeur de x : %d", x); return 0; }

Réponse : Il manque la directive #include <stdio.h> au début, car printf appartient à la bibliothèque standard d’entrée/sortie. Sans elle, le compilateur signalera une erreur implicite.

💡 Astuce : Avant de coder, écrivez toujours #include <stdio.h> et la structure main().

Exercice de réflexion

On souhaite écrire un programme qui lit le rayon d’un cercle et affiche son périmètre (2 × π × rayon). Proposez la structure en C en utilisant une constante PI = 3.14159.

🔍 Voir une solution possible
#include <stdio.h> #define PI 3.14159 int main() { float rayon, perimetre; printf("Rayon : "); scanf("%f", &rayon); perimetre = 2 * PI * rayon; printf("Périmètre = %.2f\n", perimetre); return 0; }

La directive #define PI 3.14159 permet de définir une constante. On utilise %f pour lire/afficher un réel.
