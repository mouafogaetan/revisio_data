📘 Leçon 9 : Conditionnels Type 0 et Type 1

Anglais · Première CD · Révision Probatoire

📌 Éléments clés à retenir

🔹 Conditionnel Type 0 : Vérités générales & faits scientifiques

Le type 0 exprime une certitude absolue. On l'utilise pour parler de faits toujours vrais, de lois scientifiques, d'habitudes ou de conséquences automatiques.
Structure : If + présent simple , présent simple
(On peut aussi utiliser "when" à la place de "if" sans changer le sens.)

  • Exemple scientifique : If you heat ice, it melts. (Si tu chauffes de la glace, elle fond.)
  • Exemple habitude : If my brother is tired, he drinks coffee. (Si mon frère est fatigué, il boit du café.)
  • Conseil mnémotechnique : Les deux verbes sont au présent simple → une action entraîne l’autre immédiatement.

🔸 Conditionnel Type 1 : Situations réelles et probables

Le type 1 exprime une condition qui a de fortes chances de se réaliser dans le futur. C'est le conditionnel de la possibilité réelle.
Structure : If + présent simple , will + base verbale (infinitif sans to)
On peut aussi utiliser "unless" (sauf si), "as soon as", "when" pour des nuances.

  • Exemple promesse : If you study hard, you will pass the exam. (Si tu étudies sérieusement, tu réussiras l’examen.)
  • Exemple avertissement : If you don’t leave now, you will miss the bus. (Si tu ne pars pas tout de suite, tu rateras le bus.)
  • Variante avec impératif : If you see Paul, tell him the news. (Si tu vois Paul, dis-lui la nouvelle.)

💡 Exemples de compréhension (avec explications en français)

Exemple 1 - Type 0 (vérité générale)

Phrase : “If you mix blue and yellow, you get green.”

🔍 Explication pas à pas :
1. Il s’agit d’un fait scientifique toujours vrai (mélange des couleurs).
2. Les deux verbes sont au présent simple : mix (mélanger) et get (obtenir).
3. On peut remplacer "if" par "when" : “When you mix blue and yellow, you get green.” → sens identique.
✅ Traduction : Si tu mélanges du bleu et du jaune, tu obtiens du vert.
💡 Astuce : Pour reconnaître le type 0, pose-toi la question : "Est-ce que la conséquence est toujours vraie, hier, aujourd’hui et demain ?" Si oui → type 0.

Exemple 2 - Type 1 (situation future probable)

Phrase : “If it rains tomorrow, we will cancel the picnic.”

🔍 Explication pas à pas :
1. La condition (pluie demain) est possible mais pas certaine.
2. On utilise le présent simple dans la proposition avec "if" → “rains”.
3. Dans la proposition principale (conséquence), on utilise “will + base verbale” → “will cancel”.
4. Cela exprime une décision future dépendante du temps qu’il fera.
✅ Traduction : S’il pleut demain, nous annulerons le pique-nique.
💡 Rappelle-toi : Jamais de “will” dans la partie avec “if” ! On ne dit pas ❌ “If it will rain”.

Exemple 3 - Type 1 avec “unless” et conseil

Phrase : “Unless you hurry, you will be late for school.”

🔍 Explication :
“Unless” signifie “sauf si” ou “si … ne … pas”. C’est une alternative négative à “if…not”.
Ici, “Unless you hurry” = “If you don’t hurry”.
La structure reste la même : présent simple après unless, puis will + verbe.
✅ Traduction : Si tu ne te dépêches pas (sauf si tu te dépêches), tu seras en retard à l’école.
💬 Petit conseil : “Unless” est très utile à l’écrit pour éviter une double négation. Exemple : ❌ “If you don’t study…” → ✅ “Unless you study…”

📊 Récapitulatif des différences Type 0 vs Type 1

  • Type 0 : vérité universelle, certitude 100% → présent + présent. Ex: If you freeze water, it becomes ice.
  • Type 1 : situation réaliste et probable dans le futur → présent + will + infinitif. Ex: If you freeze the bottle, it will break.
  • Moyen mnémotechnique : “Zéro = zéro doute (toujours vrai) ; Un = une chance sur deux (probable mais pas sûr).”

⚠️ Pièges courants à éviter

  • Erreur fréquente : Mettre “will” dans la proposition avec “if”. ❌ “If you will come, I will be happy.” → ✅ “If you come, I will be happy.”
  • Attention : Confondre le type 0 et le type 1. Si la conséquence est une habitude ou une loi scientifique → type 0. Si c’est un événement futur dépendant d’une condition → type 1.
  • Piège : Oublier la virgule quand la proposition “if” est en début de phrase. Si “if” est en premier, on met une virgule. Ex: If you work, you will succeed. / You will succeed if you work. (pas de virgule ici)
  • À ne pas confondre : “When” (quand) exprime une certitude, “if” (si) exprime une condition possible. Ex: When he arrives (il va arriver sûrement) / If he arrives (peut-être qu’il viendra).

✍️ Petit test d'application (auto-évaluation)

Complète avec le bon conditionnel (Type 0 ou Type 1)

  • 1. If you heat water to 100°C, it _______________ (boil). → (boils – Type 0, vérité scientifique)
  • 2. If she _______________ (study) regularly, she will pass her exam. → (studies – Type 1, condition probable)
  • 3. If they don’t respect the rules, the teacher _______________ (punish) them. → (will punish – Type 1)
  • 4. If you drop an egg, it _______________ (break). → (breaks – Type 0, toujours vrai)
🧠 Pour t’entraîner : invente trois phrases sur la vie scolaire (ex: “If I wake up late, I will miss the first class”) en utilisant chaque type.

📖 Formules à mémoriser absolument

If + présent simple → présent simple (Type 0)

If + présent simple → will + base verbale (Type 1)

⭐ Conseil de prof : Apprends ces deux structures par cœur comme une table de multiplication, et tu réussiras tous les exercices sur les conditionnels !
