📦 Installation, désinstallation et mises à jour logicielles

Informatique · Classe de 3ème · Module 2 – UE 5

📌 Éléments clés à retenir

🔧 Qu'est-ce qu'une installation logicielle ?

Installer un logiciel, c'est copier et configurer ses fichiers sur un ordinateur pour qu'il soit exécutable. L'installation peut se faire à partir :

  • D'un fichier d'installation téléchargé (.exe, .dmg, .deb, .msi).
  • D'un magasin d'applications (Microsoft Store, App Store, Google Play, etc.).
  • D'un support physique (CD, DVD, clé USB).
📀 Sources d'installation : Internet, magasin d'apps, périphérique externe.

🔄 Mise à jour logicielle : pourquoi et comment ?

Une mise à jour apporte des correctifs de sécurité, de nouvelles fonctionnalités ou améliore la stabilité.

  • Mise à jour automatique : le logiciel se met à jour en arrière-plan (ex: navigateurs, antivirus).
  • Mise à jour manuelle : l'utilisateur vérifie et lance l'installation depuis le logiciel ou le site officiel.
  • Il est crucial de maintenir ses logiciels à jour pour éviter les failles de sécurité.
🔁 Exemple : paramètres Windows Update.

🗑️ Désinstallation propre d'un logiciel

Désinstaller, c'est supprimer complètement un logiciel et ses composants pour libérer de l'espace et éviter les conflits.

  • Sous Windows : "Panneau de configuration" → "Désinstaller un programme" ou via Paramètres → Applications.
  • Sous macOS : déplacer l'application vers la corbeille ou utiliser un désinstalleur.
  • Sous Linux : via le gestionnaire de paquets (apt, snap, etc.).
  • ⚠️ Éviter de supprimer manuellement les dossiers d'application sans désinstalleur officiel.

⚖️ Respect des licences et droits d'auteur

Avant d'installer un logiciel, il faut respecter sa licence. Voici les types courants :

🆓 Logiciel libre Code source accessible, modification autorisée (ex: LibreOffice, GIMP).
💰 Logiciel propriétaire Payant, utilisation soumise à contrat (ex: Microsoft Office, Adobe Photoshop).
🧪 Shareware / Trial Version d'essai limitée dans le temps ou en fonctionnalités.
🎁 Freeware Gratuit mais propriétaire (ex: Skype, certains jeux).

Piratage = utilisation illégale, passible de sanctions. Toujours télécharger depuis des sources officielles.

💡 Exemples de compréhension

✅ Installer un logiciel (VLC) sous Windows

👩‍🎓 Situation : Tu souhaites installer le lecteur multimédia VLC (gratuit) depuis son site officiel.

Étapes :

1 Se rendre sur videolan.org (site officiel).
2 Cliquer sur "Télécharger VLC" → obtenir le fichier vlc-x.x.x-win64.exe.
3 Double-cliquer sur le fichier, accepter la licence (GPL).
4 Suivre l'assistant : choisir le dossier d'installation, options (créer un raccourci).
5 Cliquer sur "Installer" et attendre la fin.
6 Lancer VLC depuis le menu Démarrer.
💡 Astuce : Pendant l’installation, méfie-toi des cases pré-cochées proposant d’installer des logiciels additionnels (adwares). Toujours choisir "Installation personnalisée" pour désactiver les offres indésirables.

🔄 Mise à jour d'un logiciel : le navigateur Firefox

🌐 Firefox affiche une notification "Une nouvelle version est disponible". Comment procéder ?

Méthode manuelle : cliquer sur le menu ≡ → "Aide" → "À propos de Firefox". Une vérification s'effectue et la mise à jour se télécharge automatiquement. Redémarrer le navigateur.

Alternative : télécharger le nouvel installeur depuis le site officiel et lancer l'installation par-dessus l'ancienne version (les données sont conservées).

🔐 La mise à jour comble souvent des failles de sécurité critiques, donc ne jamais ignorer ces alertes.

🧹 Désinstallation correcte sous Windows 11

Ton ordinateur est ralenti à cause d’un jeu que tu n’utilises plus. Tu veux le désinstaller complètement.

Procédure : Paramètres → Applications → Applications installées → rechercher le jeu → cliquer sur les trois points → "Désinstaller". Confirmer. Certains logiciels possèdent leur propre désinstalleur dans le dossier d’installation.

📦 Après désinstallation, on peut vider la corbeille et éventuellement utiliser un nettoyeur (comme BleachBit) pour supprimer les clés résiduelles du registre (pour utilisateur averti).

📜 Droits d'auteur : téléchargement responsable

Un site propose une version crackée de Photoshop sans payer. Que faire ?

Bonne attitude numérique : ne pas télécharger car c'est illégal et risqué (virus potentiels). On utilise plutôt des alternatives gratuites et libres comme GIMP (traitement d'image) ou Photopea (version navigateur). La loi camerounaise (cybercriminalité) et internationale sanctionne le piratage.

⚖️ Sanctions possibles : amendes, peines d'emprisonnement (selon gravité) pour contrefaçon. Au Cameroun, la loi n° 2010/012 relative à la cybersécurité et la lutte contre la cybercriminalité punit l'atteinte aux droits d'auteur.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Télécharger un logiciel depuis un site non officiel (ex: "telecharger-logiciel-gratuit.com") → risque de virus ou logiciel malveillant.
  • Confusion : "Freeware" n'est pas forcément "logiciel libre". Un freeware est gratuit mais son code source reste fermé.
  • Attention : Pendant l'installation, accepter rapidement "Installation express" peut ajouter des barres d'outils indésirables. Toujours privilégier l'installation personnalisée.
  • Oubli important : Ne jamais ignorer les mises à jour de sécurité (ex: système d'exploitation, navigateur). C'est une porte ouverte aux cyberattaques.
  • Idée reçue : "Désinstaller un logiciel = supprimer son raccourci". Faux, le programme reste installé. Il faut utiliser les outils système.

🖥️ Particularités selon les systèmes d'exploitation

Windows / macOS / Linux

🪟 Windows : fichiers .exe, .msi, Microsoft Store
🍎 macOS : fichiers .dmg, .pkg, App Store
🐧 Linux : gestionnaires de paquets (apt, snap, dpkg)

Sous Linux, on privilégie le terminal ou la logithèque pour garantir une installation propre et des mises à jour centralisées. Exemple : sudo apt install nom_du_logiciel. Sous macOS, après avoir ouvert un .dmg, il faut souvent glisser l'application dans le dossier "Applications". Sous Windows, les installers .exe demandent souvent des droits administrateur.

📎 Sources légales et recommandations

Pour télécharger en toute sécurité : sites officiels des éditeurs, magasins d’applications intégrés (Microsoft Store, Mac App Store, Snap Store), ou plateformes reconnues comme Framasoft (logiciels libres).

✅ Bon réflexe : avant d'installer, vérifier les avis et la réputation de l'éditeur.

📜 À retenir sur les droits d’auteur : Au Cameroun, l’OMPI et la législation nationale protègent les œuvres logicielles. Le non-respect expose à des poursuites judiciaires. L’éducation au numérique passe par l’usage de logiciels sous licence libre ou l’achat de licences légitimes.
