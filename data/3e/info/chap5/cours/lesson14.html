UE 14 : Mise en forme et mise en page d’une feuille de calcul

Module 2 : Production de contenus numériques · Niveau 3e · Tableur (LibreOffice Calc / Excel)

📌 Éléments clés à retenir

1. Mise en forme des données (Améliorer la lisibilité)

La mise en forme permet de rendre un tableau plus clair et professionnel. On peut agir sur l'apparence des cellules et de leur contenu.

  • Format de cellule : On définit le type de donnée (Nombre, Monnaie, Date, Texte, Pourcentage). Cela évite les erreurs de calcul.
  • Police et alignement : Choix de la police, taille, gras, italique. Alignement horizontal (gauche, centré) et vertical (haut, milieu).
  • Couleurs et bordures : Couleur de fond (trames) pour différencier les titres, bordures pour délimiter les zones du tableau.
  • Fusionner et centrer : Technique utilisée pour créer un titre unique sur plusieurs colonnes.

2. Création de graphiques (Visualiser les données)

Un graphique transforme des chiffres en une image compréhensible instantanément. Il est lié aux données : si les données changent, le graphique se met automatiquement à jour.

  • Graphique en colonnes/barres : Idéal pour comparer des valeurs (ex: ventes par mois).
  • Graphique en courbes (lignes) : Idéal pour montrer une évolution dans le temps (ex: température).
  • Graphique en secteurs (camembert) : Idéal pour montrer des proportions (ex: répartition d'un budget).
  • Éléments d’un graphique : Titre, axes (X et Y), légende, étiquettes de données.

3. Préparation à l'impression (Mise en page)

Avant d'imprimer, il faut préparer la feuille pour qu'elle soit lisible sur papier.

  • Marges : Espace entre le bord de la feuille et le tableau.
  • En-tête et pied de page : Texte qui s'affiche automatiquement en haut (en-tête) ou en bas (pied) de chaque page (ex: nom du fichier, numéro de page, date).
  • Zone d'impression : Définir la zone exacte à imprimer (si on ne veut pas imprimer toute la feuille).
  • Répéter les titres : Paramètre important pour que la ligne des titres (ex: "Nom", "Prix") apparaisse sur chaque page imprimée.
  • Aperçu avant impression : Toujours vérifier avant d'imprimer pour éviter les erreurs.

💡 Exemples de compréhension

Exemple 1 : Mise en forme d'un relevé de notes

Contexte : Vous avez un tableau avec les colonnes "Nom", "Maths", "Français", "Moyenne". Vous voulez le rendre plus professionnel.

Démarche :
1. Sélectionnez la ligne des titres (Nom, Maths, Français).
2. Appliquez un style "Gras" et une couleur de fond (ex: bleu clair).
3. Pour la colonne "Moyenne", choisissez le format de cellule "Nombre" avec 2 décimales.
4. Ajoutez des bordures fines autour de toutes les cellules pour délimiter le tableau.
5. Fusionnez les cellules de la première ligne sur toute la largeur pour écrire le titre "Bulletin Scolaire 2024".

💬 Astuce : Utilisez le bouton "Format Painter" (pinceau) pour copier rapidement le style d'une cellule à d'autres.

Exemple 2 : Création d'un graphique à partir de données

Contexte : Vous avez les ventes trimestrielles d'un produit : T1: 150, T2: 220, T3: 180, T4: 300.

Résolution :
1. Sélectionnez les données (Trimestres et valeurs).
2. Allez dans le menu "Insertion" puis choisissez "Graphique".
3. Choisissez un type "Colonnes" pour comparer les ventes.
4. Ajoutez un titre "Ventes annuelles" et une légende.
5. On voit immédiatement que le T4 est le trimestre le plus performant.

💡 Un graphique est dynamique. Si vous modifiez la valeur du T4 (ex: 300 → 350), le graphique se met à jour automatiquement.

Exemple 3 : Impression d'une feuille qui dépasse

Contexte : Votre tableau est très large (12 colonnes) et tient difficilement sur une page A4 en orientation portrait.

Solutions possibles :
- Modifier l'orientation de la page en "Paysage" (Menu Mise en page → Orientation).
- Ajuster la largeur des colonnes pour qu'elles tiennent sur une seule page.
- Utiliser la fonction "Mettre à l'échelle" pour réduire le contenu (ex: 90% de la taille normale) afin que tout tienne sur une largeur de page.
- Définir une "Zone d'impression" pour ne pas imprimer les colonnes vides à droite.

💬 Toujours utiliser l'aperçu avant impression pour voir le résultat exact avant de gaspiller du papier.

🖨️ Schéma récapitulatif : Mise en page pour l'impression

Voici les éléments clés à configurer dans la boîte de dialogue "Mise en page" avant d'imprimer un tableau professionnel.

  • ① En-tête : Titre du document, nom de l’auteur, logo.
  • ② Marges : Espaces blancs autour du tableau (haut, bas, gauche, droit).
  • ③ Zone d'impression : Seule la zone sélectionnée est imprimée.
  • ④ Titres répétés : La ligne 1 (Nom, Prénom, Note) apparaît sur chaque page.
  • ⑤ Pied de page : Numéro de page (Page 1/3), date d'impression.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Fusionner les cellules dans une zone qui contient des données utilisées pour des calculs ou des graphiques (cela peut générer des erreurs de référence).
  • Attention : Oublier de définir la zone d'impression et imprimer des milliers de lignes vides (problème fréquent dans Calc/Excel).
  • À ne pas confondre : Le graphique en courbes (évolution) et le graphique en colonnes (comparaison). Bien choisir le type selon le message à faire passer.
  • Bon à savoir : Un graphique n'est pas une image fixe ; il est lié aux données sources. Si on supprime les données, le graphique devient vide.

⌨️ Raccourcis et bonnes pratiques

  • Mise en forme rapide : Ctrl + B (Gras), Ctrl + I (Italique), Ctrl + 1 (Format de cellule).
  • Créer un graphique : Sélectionner les données puis appuyer sur F11 (ou Insertion > Graphique).
  • Aperçu avant impression : Ctrl + P puis cliquer sur "Aperçu".
  • Bonnes pratiques :
    • Utilisez des couleurs sobres (éviter le rouge vif pour un document officiel).
    • Préférez les polices sans sérif (Arial, Calibri) pour une meilleure lisibilité à l'écran.
    • Testez l'impression sur une seule page avant de lancer une impression massive.
