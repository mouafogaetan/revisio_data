UE 1 : Composants et rôles de l'architecture d'un microordinateur

Informatique · Classe de Troisième · Architecture et maintenance

📌 Éléments clés à retenir

1. La carte mère

La carte mère est le composant principal qui relie et permet la communication entre tous les autres composants (processeur, mémoire, disques, périphériques). Elle héberge les circuits électroniques, les ports de connexion et le BIOS.

Schéma simplifié d'une carte mère (carte de base de l'ordinateur)
  • Rôle principal : Assurer la liaison physique et électrique entre tous les composants.
  • Éléments intégrés : Chipset, slots d'extension (PCIe), connecteurs SATA, ports USB/Ethernet.
  • BIOS/UEFI : Programme stocké sur la carte mère qui lance le démarrage de l'ordinateur.

2. Le processeur (CPU - Central Processing Unit)

C'est le "cerveau" de l'ordinateur. Il exécute les instructions des programmes, effectue les calculs et contrôle les autres composants.

Un processeur (vue de dessus et de dessous)
  • Fréquence : Mesurée en GHz (gigahertz), elle indique le nombre d'opérations par seconde.
  • Cœurs : Plusieurs unités de calcul indépendantes (dual-core, quad-core...).
  • Rôle : Interpréter et exécuter les instructions, gérer les échanges de données.

3. La mémoire RAM (Random Access Memory)

Mémoire vive, volatile : elle stocke temporairement les données et les programmes en cours d'utilisation. Sa capacité est exprimée en Go (gigaoctets).

Barrettes de RAM (mémoire vive)
  • Volatilité : Les données sont effacées à l'extinction de l'ordinateur.
  • Influence : Plus la RAM est grande, plus on peut exécuter d'applications simultanément sans ralentissement.
  • Le processeur accède directement à la RAM pour lire/écrire rapidement les données.

4. Disque dur (HDD) et SSD (Solid State Drive)

Ils assurent le stockage permanent des données (système d'exploitation, documents, logiciels). Contrairement à la RAM, les données persistent après extinction.

  • HDD : Disque mécanique avec plateaux magnétiques, grande capacité mais plus lent.
  • SSD : Mémoire flash électronique, très rapide, silencieux et plus résistant aux chocs.
  • Capacité exprimée en Go ou To (téraoctets).

5. Périphériques d'entrée/sortie

Les périphériques permettent d'interagir avec l'ordinateur et d'échanger des données avec le monde extérieur.

🖱️ Entrée : Clavier, souris, microphone, scanner, webcam.
🖨️ Sortie : Écran, imprimante, haut-parleurs, casque.
📀 Mixte : Écran tactile, clé USB, modem (entrée/sortie).

Ils se connectent via des ports (USB, HDMI, VGA, RJ45, Jack audio) et sont gérés par des pilotes (drivers).

Schéma fonctionnel simplifié

Les échanges entre composants se font via la carte mère (bus de données, bus d'adresses, bus de contrôle).

Communication entre les composants

💡 Exemples de compréhension

Exemple 1 : Identifier le goulot d'étranglement

Sophie a un ordinateur avec un processeur récent (Intel Core i7), 4 Go de RAM et un disque dur HDD de 1 To. Elle constate que son ordinateur rame beaucoup lorsqu'elle a 15 onglets ouverts dans son navigateur + un logiciel de montage vidéo.

Analyse : Le processeur est puissant, mais la RAM (4 Go) est insuffisante pour exécuter plusieurs applications lourdes simultanément. Le système utilise alors le disque dur comme mémoire virtuelle, ce qui ralentit tout car le HDD est lent comparé à la RAM.
Solution envisagée : Augmenter la quantité de RAM (passer à 8 Go ou 16 Go) et remplacer le HDD par un SSD pour améliorer la réactivité générale.

💬 Astuce : La RAM est cruciale pour le multitâche, le stockage (SSD) améliore le temps de démarrage et d'ouverture des logiciels.

Exemple 2 : Associer périphérique et fonction

Lors d’un cours, l’enseignant demande : « Je veux numériser un document papier pour le modifier sur l’ordinateur. De quel périphérique ai-je besoin ? Et si je veux afficher une présentation sur un grand écran ? »

Réponse : Pour numériser un document → un scanner (périphérique d’entrée). Pour afficher sur grand écran → un vidéoprojecteur ou un écran externe connecté via HDMI/VGA (périphérique de sortie).
Ces périphériques communiquent avec la carte mère via leurs ports dédiés.

💡 À retenir : Les périphériques d'entrée envoient des informations à l'ordinateur ; les périphériques de sortie reçoivent des informations de l'ordinateur.

Exemple 3 : Rôle de la carte mère dans la communication

Lorsque tu tapes une lettre au clavier, comment l'information parvient-elle à l'écran ?

Cheminement :

  1. Le clavier (périphérique d’entrée) envoie un signal électrique via le port USB.
  2. La carte mère achemine le signal vers le processeur via les bus.
  3. Le processeur analyse l’information et demande à la RAM de stocker temporairement le caractère.
  4. Le processeur envoie ensuite l’ordre d’affichage à la carte graphique (ou graphique intégrée).
  5. L’écran (sortie) affiche la lettre.
Cet exemple illustre la coordination obligatoire entre tous les composants.

⚠️ Pièges courants / À ne pas confondre

  • Confondre RAM et stockage : La RAM est temporaire (mémoire vive) et le disque dur/SSD est permanent. On ne sauvegarde pas ses fichiers dans la RAM !
  • Croire que seul le processeur détermine la vitesse : Un ordinateur rapide nécessite un équilibre entre CPU, RAM et stockage. Un SSD accélère énormément les démarrages.
  • Oublier les périphériques internes vs externes : La carte réseau, la carte graphique (si dédiée) sont aussi des composants internes ayant des rôles précis.
  • Penser que tous les périphériques sont indispensables : Un ordinateur peut fonctionner sans imprimante, mais sans RAM il ne démarre pas.

📝 Résumé mnémotechnique

« Le Processeur calcule, la RAM garde en mémoire vive, le Disque stocke à long terme, la Carte mère relie tout, les Périphériques dialoguent avec l’utilisateur. »
