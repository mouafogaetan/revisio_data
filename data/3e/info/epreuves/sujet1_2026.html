🧠 PARTIE 1 : MAÎTRISE DES CONNAISSANCES (12 pts)
❶ (2 pts) L’architecture de Von Neumann repose sur le principe du « programme enregistré ». Lequel de ces composants n’appartient PAS au modèle de Von Neumann ?
  • A. Unité de contrôle (UC)
  • B. Unité arithmétique et logique (UAL)
  • C. Mémoire cache L3 séparée pour les instructions
  • D. Mémoire centrale (RAM)
    • ✅ Réponse : C
    Le modèle de Von Neumann (1945) ne prévoit pas de cache séparé pour instructions/données ; la mémoire est unique. Les caches séparés sont une évolution moderne (Harvard modifié).
    ❷ (1.5 pts) Vrai ou Faux ? Dans l’architecture de Von Neumann, le processeur ne peut pas lire une instruction et une donnée simultanément car elles partagent le même bus.
    ✅ Réponse : Vrai
    C'est le célèbre « goulot d'étranglement de Von Neumann » (Von Neumann bottleneck) [citation:5][citation:6].
    ❸ (2 pts) Associez chaque composant à son rôle principal (Relier par lettre) :
  • 1. Carte mère    →   A. Stockage permanent des données
  • 2. Disque dur / SSD  →   B. Connecte tous les composants internes
  • 3. Mémoire RAM   →   C. Exécute les calculs et instructions
  • 4. Processeur     →   D. Mémoire volatile pour les programmes en cours
    • ✅ Correction : 1-B ; 2-A ; 3-D ; 4-C
    La carte mère joue le rôle de colonne vertébrale, le processeur est le « cerveau ».
    ❹ (2.5 pts) Distinguer les philosophies RISC et CISC :
  • A. RISC possède un jeu d’instructions réduit, chaque instruction s’exécute généralement en un cycle.
  • B. CISC utilise des instructions complexes qui peuvent prendre plusieurs cycles.
  • C. RISC est toujours plus lent que CISC pour les calculs arithmétiques.
  • D. Les processeurs Intel Core (x86) sont généralement de type CISC.
  • E. ARM est un exemple typique d’architecture RISC.
    • (Plusieurs réponses possibles, cocher les affirmations exactes)
    ✅ Bonnes réponses : A, B, D, E
    L’affirmation C est fausse ; RISC n’est pas systématiquement plus lent, il peut être plus efficace énergétiquement.
    ❺ (2 pts) Citer deux types de maintenance informatique en donnant un exemple pour chacun.
    ✅ Exemple de réponse :
    • Maintenance préventive : nettoyer la poussière à l’intérieur du boîtier, défragmenter le disque dur.
    • Maintenance corrective : remplacer une barrette de RAM défectueuse, réinstaller un pilote après un écran bleu.
    ❻ (2 pts) Un élève se plaint que son ordinateur « rame » énormément et met du temps à démarrer. Proposez deux causes possibles (matérielles ou logicielles) et une solution de maintenance de premier niveau.
    ✅ Éléments de correction :
    • Manque de RAM → Fermer des applications ou ajouter de la RAM.
    • Trop de logiciels au démarrage → Désactiver les programmes inutiles dans le gestionnaire des tâches.
    • Disque dur fragmenté ou plein → Nettoyage de disque / défragmentation (HDD).

    ❽ (4 pts) Lisez la situation : Un ordinateur portable présente un écran noir au démarrage, le ventilateur tourne mais aucun signal vidéo. Aucun bip sonore.
  • a) (1 pt) Selon vous, la panne est-elle matérielle ou logicielle ? Justifiez.
  • b) (2 pts) Proposez trois actions de maintenance de premier niveau avant d’appeler un technicien.

    • ✅ Corrigé type :
    a) Panne matérielle probable (pas de signal vidéo mais ventilateur actif).
    b) 1. Vérifier le branchement de l’écran externe / bascule écran (Fn+F4). 2. Débrancher la batterie et tenir le bouton power 30s (réinitialisation). 3. Resserrer la mémoire RAM / changer de slot.
