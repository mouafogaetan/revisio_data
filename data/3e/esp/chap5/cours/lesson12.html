📆 Los pasados : Pretérito perfecto, indefinido e imperfecto

Espagnol · Troisième (3e) · Les 3 temps du passé à maîtriser

📌 Éléments clés à retenir

1️⃣ Pretérito perfecto : le passé récent / l'expérience

📖 En français : Il correspond au passé composé. Il exprime une action achevée dans le passé mais en lien avec le présent (expérience, résultat actuel, action récente).

🔧 Formation : verbe HABER au présent + participe passé.

  • Haber au présent : he, has, ha, hemos, habéis, han
  • Participe passé : -ado (1er groupe : hablar → hablado) / -ido (2e & 3e : comer → comido, vivir → vivido)
    ⚠️ Irréguliers fréquents : abrir → abierto, decir → dicho, escribir → escrito, hacer → hecho, ver → visto, volver → vuelto.
  • Exemple : Hoy he comido paella. (Aujourd'hui j'ai mangé une paella → action dans la journée, lien avec le présent)
💡 Astuce : On utilise le pretérito perfecto avec des marqueurs temporels comme hoy, esta semana, este año, ya, todavía no, nunca, alguna vez.

2️⃣ Pretérito indefinido : l'action ponctuelle achevée

📖 En français : Équivalent du passé simple (ou parfois du passé composé pour une action coupée du présent). Il raconte des actions ponctuelles, terminées dans un passé lointain ou sans lien avec le présent.

🔧 Conjugaison (verbes réguliers) :

-AR (hablar)-ER (comer)-IR (vivir)
hablécomíviví
hablastecomisteviviste
hablócomióvivió
hablamoscomimosvivimos
hablasteiscomisteisvivisteis
hablaroncomieronvivieron

Irréguliers très fréquents : ser/ir → fui, fuiste, fue, fuimos, fuisteis, fueron ; tener → tuve, tuviste, tuvo, tuvimos, tuvisteis, tuvieron ; hacer → hice, hiciste, hizo, hicimos, hicisteis, hicieron.

Exemple : Ayer fui al cine con mis amigos. (Hier je suis allé au cinéma → action précise et terminée, pas de lien avec aujourd'hui)

⏰ Marqueurs : ayer, la semana pasada, el año pasado, en 2020, entonces, aquel día.

3️⃣ Pretérito imperfecto : la description / habitudes passées

📖 En français : Corresponde à l'imparfait. Il sert à décrire dans le passé (physique, âge, météo), exprimer des habitudes ou des actions non terminées.

🔧 Conjugaison (réguliers) : on ajoute les terminaisons à l'infinitif (pour -AR : -aba, -abas, -aba, -ábamos, -abais, -aban) ; pour -ER / -IR : -ía, -ías, -ía, -íamos, -íais, -ían).

  • Hablar : hablaba, hablabas, hablaba, hablábamos, hablabais, hablaban
  • Comer : comía, comías, comía, comíamos, comíais, comían
  • Vivir : vivía, vivías, vivía, vivíamos, vivíais, vivían
  • Irréguliers (seulement 3) : ser → era, eras, era, éramos, erais, eran ; ir → iba, ibas, iba, íbamos, ibais, iban ; ver → veía, veías, veía, veíamos, veíais, veían

Exemple : Cuando era pequeño, jugaba en el parque todos los días. (Quand j'étais petit, je jouais au parc tous les jours → habitude)

🌟 Utilise l'imparfait pour les descriptions, les émotions, l'âge, l'heure dans le passé.

🧠 Moyen mnémotechnique :
- Perfecto = ✔️ action “parfaite” liée au présent (aujourd'hui, cette année).
- Indefinido = 🎬 action précise, achevée, “hier, l'année dernière” (film terminé).
- Imperfecto = 🎨 décoration, habitude, “chaque jour, autrefois”.

💡 Exemples de compréhension (avec explications en français)

Exemple 1 : trois phrases sur une même histoire

📖 Lis ce petit texte et observe les trois temps :

“El año pasado viajé a España. Durante el viaje, hacía buen tiempo todos los días. Esta semana he recordado mucho aquel viaje.”

🔍 Explication :
• viajé (indefinido) → action ponctuelle terminée : l'année dernière, voyage fini.
• hacía (imperfecto) → description météo dans le passé, sans début ni fin précis (habitude sur la durée).
• he recordado (perfecto) → action récente (cette semaine) en lien avec le présent : le souvenir est actuel.

💬 Astuce : Pour choisir, demande-toi : est-ce que l'action est terminée et datée précisément ? → indefinido. Est-ce une description / habitude ? → imperfecto. Est-ce une expérience ou résultat présent ? → perfecto.

Exemple 2 : les trois passés dans un dialogue

🗣️ Situation : Paul parle de son enfance et de ses vacances récentes.

— De pequeño, jugaba al fútbol cada tarde. Era muy deportista.
— ¿Y este verano?
— Este verano he ido a la playa con mi familia. Compramos muchas conchas el último día.

📝 Analyse :
• jugaba / era → imparfait : habitude (cada tarde) et description physique (deportista).
• he ido → perfecto : action récente (este verano, dans la période présente).
• compramos → indefinido (ici compramos = nous achetâmes) car action ponctuelle précise “el último día” (le dernier jour).

Exemple 3 : biographie avec les trois temps

🎭 Frida Kahlo : Frida Kahlo nació en 1907. De niña, era muy inteligente y leía mucho. Hoy, sus cuadros han viajado por todo el mundo.

🎯 Pourquoi ces choix ?
• nació (indefinido) → événement ponctuel dans le passé (date de naissance).
• era / leía (imperfecto) → description et habitude durant l'enfance.
• han viajado (perfecto) → résultat actuel (jusqu'à aujourd'hui), expérience sans date précise.

⚠️ Pièges courants & astuces de révision

  • ❌ Ne pas confondre “he” (auxiliaire) et “eh” : “He comido” ≠ “Eh comido”.
  • ⚠️ Attention aux participes irréguliers ! romper → roto, volver → vuelto, escribir → escrito, decir → dicho, hacer → hecho. Ils sont très fréquents aux examens.
  • 🧩 Piège : “Fui” peut venir de SER ou IR. Le contexte donne le sens : Fui médico (ser) vs Fui al cine (ir).
  • 📅 Ne pas utiliser le pretérito perfecto avec des marqueurs coupés du présent (ayer, el año pasado). Exemple à éviter : ✘ Ayer he comprado pan → ✔ Ayer compré pan.
  • 🎭 Imperfecto vs indefinido ensemble : l’imparfait décrit le décor, l’indéfini indique l’action soudaine. Miraba la tele cuando sonó el teléfono. (Je regardais la télé quand le téléphone a sonné).

Schéma simplifié des trois passés :

🧠 Astuce mnémotechnique visuelle : Imagine un film :
- Perfecto = une scène qui fait encore écho (caméra au présent).
- Indefinido = une action nette, une scène coupée (flash).
- Imperfecto = le décor et les habitudes (arrière-plan).

✅ Synthèse à coller dans son cahier

  • Pretérito perfecto = haber au présent + participe → pour des expériences ou actions récentes (hoy, esta semana).
  • Pretérito indefinido = terminaisons spécifiques (é, aste, ó, amos, asteis, aron ; í, iste, ió, imos, isteis, ieron) → actions terminées sans lien avec le présent (ayer, hace dos años).
  • Pretérito imperfecto = terminaisons -aba / -ía → pour décrire, parler d'habitudes, âge, météo (cuando era niño, siempre).
  • Mots-clés pour choisir : si c'est “todos los días” → imparfait ; “el otro día” → indefinido ; “nunca” → perfecto ou indefinido selon le contexte.
