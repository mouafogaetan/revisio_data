📐 Position relative de deux droites

Mathématiques · Classe de Troisième · Chapitre 14 : Équations de droite (Leçon 35)

📌 Éléments clés à retenir

1. Droites sécantes, parallèles ou confondues

Deux droites dans le plan peuvent être :

  • Sécantes : elles se coupent en un unique point. Leurs coefficients directeurs sont différents.
  • Parallèles et distinctes : elles n’ont aucun point commun. Leurs coefficients directeurs sont égaux, mais les ordonnées à l’origine sont différentes.
  • Confondues : elles sont superposées (infiniment de points communs). Même coefficient directeur et même ordonnée à l’origine.
Fig. 1 : De gauche à droite – droites sécantes, parallèles distinctes, confondues.

Équation réduite : $y = m x + p$ où $m$ est le coefficient directeur (pente) et $p$ l’ordonnée à l’origine.

2. Reconnaître la position avec les équations cartésiennes

Soit deux droites $(d_1) : ax + by + c = 0$ et $(d_2) : a'x + b'y + c' = 0$ (avec $(a,b) \neq (0,0)$ et $(a',b') \neq (0,0)$).

  • Elles sont sécantes si $\dfrac{a}{a'} \neq \dfrac{b}{b'}$ (dans le cas où $a'$ et $b'$ non nuls).
  • Elles sont parallèles (strictes ou confondues) si $\dfrac{a}{a'} = \dfrac{b}{b'} \neq \dfrac{c}{c'}$ (parallèles distinctes) ou $\dfrac{a}{a'} = \dfrac{b}{b'} = \dfrac{c}{c'}$ (confondues).
  • Pour les équations réduites : $m_1 = m_2$ → parallèles ou confondues ; si de plus $p_1 = p_2$ → confondues.

On peut aussi utiliser les vecteurs directeurs : $\vec{u}(-b;a)$ et $\vec{v}(-b';a')$ sont colinéaires $\iff$ droites parallèles ou confondues.

3. Calcul du point d’intersection (cas sécant)

Pour deux droites sécantes, on résout le système formé par leurs équations (substitution ou combinaison). La solution $(x; y)$ donne les coordonnées du point d’intersection.

💡 Astuce : Mettre les deux équations sous forme $y = ...$ ou utiliser la méthode par combinaison linéaire pour éviter les fractions.

💡 Exemples de compréhension

Exemple 1 : Droites sécantes → déterminer l’intersection

On considère $(d_1) : y = 2x - 3$ et $(d_2) : y = -x + 4$. Déterminer si elles sont sécantes ; si oui, trouver leur point d’intersection.

Résolution : Les coefficients directeurs sont $m_1 = 2$ et $m_2 = -1$. $2 \neq -1$ donc les droites sont sécantes.
On égalise les ordonnées : $2x - 3 = -x + 4$ → $2x + x = 4 + 3$ → $3x = 7$ → $x = \frac{7}{3}$.
On remplace dans $(d_1)$ : $y = 2 \times \frac{7}{3} - 3 = \frac{14}{3} - \frac{9}{3} = \frac{5}{3}$.
Le point d’intersection est $\boxed{I\left(\dfrac{7}{3}; \dfrac{5}{3}\right)}$.

💬 Vérification rapide : remplacer dans $(d_2)$ → $-7/3 + 4 = -7/3 + 12/3 = 5/3$ ✔.

Exemple 2 : Parallélisme à partir d’équations cartésiennes

Soit $(d_3) : 2x - 3y + 5 = 0$ et $(d_4) : 4x - 6y + 1 = 0$. Montrer qu’elles sont parallèles et distinctes.

Méthode : On compare les rapports des coefficients.
$\dfrac{2}{4} = \dfrac{1}{2}$, $\dfrac{-3}{-6} = \dfrac{1}{2}$, les rapports sont égaux → les droites sont parallèles ou confondues.
Vérifions le rapport des constantes : $\dfrac{5}{1} = 5$ alors que $\dfrac{2}{4} = 0,5$. Comme $\dfrac{5}{1} \neq \dfrac{1}{2}$, les droites sont parallèles distinctes (aucun point commun).

Fig. 2 : Les deux droites sont parallèles et ne se rencontrent jamais.

Exemple 3 : Droites confondues

Vérifier que $(d_5) : y = 3x + 2$ et $(d_6) : 6x - 2y + 4 = 0$ sont confondues.

Étape 1 : Mettre $(d_6)$ sous forme réduite : $6x - 2y + 4 = 0 \Rightarrow -2y = -6x - 4 \Rightarrow y = 3x + 2$.
Les deux équations sont identiques : même coefficient directeur (3) et même ordonnée à l’origine (2). Donc les droites sont confondues.

✨ Une droite peut être écrite sous plusieurs formes cartésiennes équivalentes (multiplication par un facteur non nul).

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Confondre coefficient directeur et ordonnée à l'origine. Pour deux droites parallèles, $m_1 = m_2$ est nécessaire mais pas suffisant pour dire qu'elles sont distinctes.
  • Attention : Avec les équations cartésiennes, ne pas oublier de simplifier les rapports. Par exemple $(d): 2x+4y-6=0$ peut se simplifier par 2.
  • Piège classique : Croire que si deux droites n'ont aucun point commun, elles sont forcément parallèles. Dans le plan, oui, mais attention aux droites verticales (coefficient directeur infini).
  • À ne pas oublier : Pour une droite verticale ($x = k$), on ne peut pas écrire $y = mx + p$. Dans ce cas, comparer les équations sous forme cartésienne est plus fiable.

✏️ Entraînement express

Teste-toi !

  • 1. Les droites $y = 4x - 1$ et $y = 4x + 2$ sont-elles sécantes ? Justifier.
  • 2. Soit $(d_7): 3x - y + 2 = 0$ et $(d_8): 6x - 2y + 4 = 0$. Position relative ?
  • 3. Trouver l'intersection de $y = -2x + 5$ et $y = \frac{1}{2}x - 1$.
🔍 Voir les réponses

1. Parallèles distinctes (même pente 4, ordonnées différentes).
2. Confondues car $(d_8)$ est $2 \times (3x - y + 2) = 0$ → même équation.
3. Résoudre $-2x+5 = 0.5x -1$ → $-2x-0.5x = -1-5$ → $-2.5x = -6$ → $x = 2.4$ ; $y = -2(2.4)+5 = 0.2$ → point $(2,4 ; 0,2)$.

🧠 Méthode avancée (rappel)

Utiliser les vecteurs directeurs

À partir d’une équation cartésienne $ax+by+c=0$, un vecteur directeur est $\vec{u}(-b;a)$.
Deux droites sont parallèles ou confondues si leurs vecteurs directeurs sont colinéaires, c’est-à-dire $(-b;a) = k(-b';a')$ ou encore $a b' - b a' = 0$.
Condition de colinéarité : $a b' - a' b = 0$ (déterminant nul).
Si le déterminant est non nul : droites sécantes.

📌 Exemple : $(d): 2x+3y-1=0$ et $(d'): 4x+6y+5=0$ : $2\times 6 - 4\times 3 = 12-12 = 0$ → parallèles. Puis vérifier les constantes → distinctes.
