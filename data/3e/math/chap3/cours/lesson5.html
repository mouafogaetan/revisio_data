📐 Opérations sur les racines carrées

Mathématiques · Troisième · Module 1 – Chapitre 3 (Leçon 5)

📌 Éléments clés à retenir

1. Produit de deux racines carrées

Pour tous nombres réels positifs $a$ et $b$ : \[ \sqrt{a \times b} = \sqrt{a} \times \sqrt{b} \] Cette propriété permet de décomposer ou de regrouper des racines. On peut l'utiliser aussi dans l'autre sens : $\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$.

  • Exemple : $\sqrt{9 \times 4} = \sqrt{9} \times \sqrt{4} = 3 \times 2 = 6$.
  • Astuce : Pour simplifier $\sqrt{50}$, écrire $50 = 25 \times 2$ → $\sqrt{50} = \sqrt{25} \times \sqrt{2} = 5\sqrt{2}$.
  • Attention : Cette règle ne s'applique pas à l'addition : $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$ en général.

2. Quotient de deux racines carrées

Pour $a \ge 0$ et $b > 0$ : \[ \sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}} \] Cette propriété est très utile pour simplifier des fractions contenant des racines ou pour écrire une racine d’un quotient.

  • Exemple : $\sqrt{\frac{36}{9}} = \frac{\sqrt{36}}{\sqrt{9}} = \frac{6}{3} = 2$.
  • On peut aussi l'utiliser pour rendre un dénominateur rationnel : $\frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$ (mais c'est une autre technique).

3. Simplification d’expressions avec racines carrées

Pour simplifier une expression contenant des racines carrées :

  • Décomposer chaque racine en utilisant le produit : $\sqrt{a^2 \times k} = a\sqrt{k}$ (avec $a \ge 0$).
  • Regrouper les termes semblables (même partie radicale) comme pour des termes algébriques.
  • Appliquer les règles de multiplication et division des racines.

💡 À retenir : $\sqrt{a^2} = a$ pour $a \ge 0$. Une racine carrée est toujours positive ou nulle.

💡 Exemples de compréhension

Exemple 1 : Simplifier un produit de racines

Calculer et simplifier $A = \sqrt{12} \times \sqrt{3}$.

Résolution :
On utilise la règle du produit : $\sqrt{12} \times \sqrt{3} = \sqrt{12 \times 3} = \sqrt{36} = 6$.
Ou bien on simplifie d'abord $\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$ ; alors $2\sqrt{3} \times \sqrt{3} = 2 \times (\sqrt{3})^2 = 2 \times 3 = 6$. Les deux méthodes mènent au même résultat.

✨ Astuce : Quand vous multipliez deux racines, cherchez à obtenir un carré parfait sous la racine.

Exemple 2 : Appliquer la règle du quotient

Écrire $B = \sqrt{\frac{27}{4}}$ sous forme simplifiée.

Démarche :
$\displaystyle \sqrt{\frac{27}{4}} = \frac{\sqrt{27}}{\sqrt{4}} = \frac{\sqrt{9 \times 3}}{2} = \frac{3\sqrt{3}}{2}$.
On ne peut pas simplifier davantage car $\sqrt{3}$ est irréductible.

🔍 Vérifiez toujours que le dénominateur est strictement positif pour appliquer la formule.

Exemple 3 : Additionner et soustraire des racines semblables

Simplifier $C = 3\sqrt{5} + 2\sqrt{20} - \sqrt{5}$.

Étapes :
1. Simplifier $\sqrt{20} = \sqrt{4 \times 5} = 2\sqrt{5}$.
2. Remplacer : $C = 3\sqrt{5} + 2 \times (2\sqrt{5}) - \sqrt{5} = 3\sqrt{5} + 4\sqrt{5} - \sqrt{5}$.
3. Additionner les coefficients : $(3+4-1)\sqrt{5} = 6\sqrt{5}$.
Résultat final : $C = 6\sqrt{5}$.

💬 On ne peut additionner que des racines ayant le même radicande. Ici $\sqrt{5}$ est la partie commune.
📐 Représentation géométrique : l'aire d'un carré de côté √(a×b) est égale au produit des aires de carrés de côtés √a et √b (illustration IA).

⚠️ Pièges courants / À ne pas confondre

  • Erreur classique : Écrire $\sqrt{a+b} = \sqrt{a} + \sqrt{b}$. C’est faux ! Exemple : $\sqrt{9+16} = \sqrt{25}=5$, alors que $\sqrt{9}+\sqrt{16}=3+4=7$. Les racines ne sont pas linéaires par rapport à l’addition.
  • Attention : La propriété $\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$ n’est valable que si $b>0$. Si $b$ est négatif, la racine carrée n’est pas définie dans $\mathbb{R}$ (sauf pour les nombres réels positifs).
  • Ne pas oublier : Lorsqu'on simplifie $\sqrt{a^2}$, le résultat est $|a|$ (valeur absolue). Pour $a$ positif ou nul, c'est $a$. Mais si $a$ est négatif, $\sqrt{a^2} = -a$ (car la racine est positive). En Troisième on travaille surtout avec des nombres positifs.
  • Piège fréquent : Confondre $(\sqrt{a})^2$ (qui vaut $a$ pour $a\ge 0$) et $\sqrt{a^2}$ (qui vaut $|a|$). Bien maîtriser ces deux notions.

🏆 Exercice guidé : simplification complète

Simplifier $D = 2\sqrt{18} - 3\sqrt{8} + \sqrt{32}$

Objectif : écrire $D$ sous la forme $a\sqrt{2}$ avec $a$ entier.

Correction détaillée :
• $\sqrt{18} = \sqrt{9 \times 2} = \sqrt{9} \times \sqrt{2} = 3\sqrt{2}$ → $2\sqrt{18} = 2 \times 3\sqrt{2} = 6\sqrt{2}$
• $\sqrt{8} = \sqrt{4 \times 2} = 2\sqrt{2}$ → $3\sqrt{8} = 3 \times 2\sqrt{2} = 6\sqrt{2}$
• $\sqrt{32} = \sqrt{16 \times 2} = 4\sqrt{2}$
• On remplace : $D = 6\sqrt{2} - 6\sqrt{2} + 4\sqrt{2} = (6-6+4)\sqrt{2} = 4\sqrt{2}$.
✅ Résultat : $D = 4\sqrt{2}$.

📌 Pensez toujours à décomposer le radicande en un produit incluant le plus grand carré parfait possible.
