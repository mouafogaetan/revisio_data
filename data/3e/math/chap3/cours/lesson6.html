Leçon 6 : Ensembles des nombres réels

Mathématiques · Classe de Troisième · Module 1 : Nombres réels

📌 Ensembles de nombres : hiérarchie et inclusion

🌱 Des entiers naturels aux réels

Les nombres sont classés dans des ensembles emboîtés. Du plus petit au plus grand : ℕ ⊂ ℤ ⊂ 𝔻 ⊂ ℚ ⊂ ℝ. Chaque nombre appartient à tous les ensembles plus grands qui le contiennent.

  • ℕ (Entiers naturels) : 0, 1, 2, 3, … (pas de virgule, pas de signe −).
  • ℤ (Entiers relatifs) : … , -3, -2, -1, 0, 1, 2, 3, … (positifs, négatifs ou nul).
  • 𝔻 (Décimaux) : nombres qui peuvent s’écrire avec un nombre fini de chiffres après la virgule. Ex: 3,5 ; -2,75 ; 0,25.
  • ℚ (Rationnels) : nombres pouvant s’écrire sous la forme $\frac{a}{b}$ avec $a$ et $b$ entiers, $b \neq 0$. Ex: $\frac{2}{3}$ ; $- \frac{5}{4}$ ; 0,333… = $\frac{1}{3}$.
  • ℝ (Réels) : tous les nombres (rationnels + irrationnels comme $\sqrt{2}$, $\pi$, …).

Un nombre décimal est toujours rationnel ($\frac{25}{100}=0,25$). Un nombre rationnel n’est pas toujours décimal ($\frac{1}{3}=0,333...$). Un nombre réel qui n’est pas rationnel est appelé irrationnel (ex: $\sqrt{2}$, $\pi$, $e$).

🔍 Comment situer un nombre donné ?

Pour déterminer le plus petit ensemble auquel appartient un nombre, on applique les règles suivantes :

  • Pas de signe − et pas de virgule → ℕ (ex: 45, 0, 128).
  • Signe − possible, pas de virgule → ℤ (ex: -7, 23).
  • Virgule et nombre fini de décimales → 𝔻 (ex: -3,14 ; 0,5 ; 12,0).
  • Fraction ou développement décimal illimité périodique → ℚ (ex: $\frac{2}{3}$, 0,272727...).
  • Racine carrée d’un nombre non carré parfait, $\pi$ → ℝ \ ℚ (irrationnel).

💡 Situer des nombres dans les ensembles

Exemple 1 : Classer $-2,5$ ; $0$ ; $\frac{4}{2}$ ; $\sqrt{5}$

Pour chaque nombre, indiquer le plus petit ensemble auquel il appartient parmi ℕ, ℤ, 𝔻, ℚ, ℝ.

Résolution détaillée :
• $-2,5$ → s’écrit avec une virgule et une seule décimale → c’est un décimal ($\mathbb{D}$). Il est aussi dans ℚ et ℝ.
• $0$ → entier naturel (ℕ). C’est aussi ℤ, 𝔻, ℚ, ℝ.
• $\frac{4}{2} = 2$ → entier naturel (ℕ).
• $\sqrt{5}$ → 5 n’est pas un carré parfait, la racine carrée est un irrationnel. Donc $\sqrt{5} \in \mathbb{R}$ mais $\notin \mathbb{Q}$ (et donc pas décimal ni entier).

💬 Astuce : Si une fraction se simplifie en un entier, elle est dans ℕ ou ℤ. Pensez à simplifier avant de conclure !

Exemple 2 : Vrai ou faux ? Un nombre rationnel est toujours décimal.

Analysons cette affirmation avec un contre-exemple.

Réponse : Faux. Par exemple, $\frac{1}{3} = 0,3333…$ est un nombre rationnel (car quotient de deux entiers) mais son écriture décimale est illimitée périodique. Il n’appartient donc pas à $\mathbb{D}$ car les décimaux ont un nombre fini de décimales.

💡 Pour qu’une fraction soit décimale, son dénominateur (après simplification) ne doit contenir que les facteurs premiers 2 et 5.

Exemple 3 : Situer $3,14$ ; $\frac{22}{7}$ ; $\pi$

Ces trois nombres sont proches mais leurs ensembles diffèrent.

Comparaison :
- $3,14 = \frac{314}{100}$ : nombre décimal (𝔻).
- $\frac{22}{7} \approx 3,142857…$ : fraction simple, c’est un rationnel (ℚ) mais pas décimal (périodique).
- $\pi \approx 3,141592…$ : c’est un irrationnel célèbre, donc ∈ ℝ mais ∉ ℚ.

🧠 Liens et propriétés essentielles

Récapitulatif des inclusions

On retient la chaîne d’inclusions : \[ \mathbb{N} \subset \mathbb{Z} \subset \mathbb{D} \subset \mathbb{Q} \subset \mathbb{R} \] Chaque ensemble est "plus grand" que le précédent.

  • ℕ = entiers positifs ou nuls.
  • ℤ = ℕ ∪ les entiers négatifs.
  • 𝔻 = nombres à développement décimal fini (ex: 0,5 = 5/10).
  • ℚ = ensemble des fractions $\frac{a}{b}$.
  • ℝ = tous les points de la droite numérique (rationnels + irrationnels).

Exemple typique d’examen : « Le nombre -3,5 appartient-il à ℤ ? » Non, car -3,5 n’est pas un entier relatif (il a une virgule) → il est dans 𝔻 (et donc ℚ et ℝ).

⚠️ Pièges courants et erreurs fréquentes

  • Confondre ℚ et 𝔻 : Tous les décimaux sont rationnels, mais tous les rationnels ne sont pas décimaux (ex: $\frac{1}{3}$).
  • Croire que √2 est rationnel : √2 ne peut pas s’écrire comme fraction d’entiers → c’est un irrationnel (dans ℝ mais pas ℚ).
  • Oublier que 0 est un entier naturel : 0 ∈ ℕ (en général au collège on considère ℕ = {0;1;2;...}).
  • Dire qu’un nombre décimal n’est pas rationnel : Faux, tout décimal est rationnel car il s’écrit $\frac{\text{nombre entier}}{10^k}$.
  • Négliger les nombres négatifs : -5 ∈ ℤ mais -5 n’est pas dans ℕ.

✏️ Vérifie ta compréhension

Question type (examen)

Classe chacun des nombres suivants dans le plus petit ensemble possible (ℕ, ℤ, 𝔻, ℚ, ℝ) :
$A = -12$ ; $B = \frac{25}{5}$ ; $C = 0,373737…$ (périodique) ; $D = \sqrt{36}$ ; $E = \sqrt{10}$.

Corrigé rapide :
- $A = -12$ → entier relatif (ℤ).
- $B = \frac{25}{5} = 5$ → entier naturel (ℕ).
- $C = 0,373737…$ = $\frac{37}{99}$ → rationnel (ℚ) mais pas décimal (périodique).
- $D = \sqrt{36} = 6$ → entier naturel (ℕ).
- $E = \sqrt{10}$ → 10 n’est pas un carré parfait → irrationnel, donc ℝ seulement.

⚡ Toujours simplifier les fractions ou les racines avant de décider de l’ensemble.

📈 Les réels sur la droite graduée

L’ensemble ℝ est souvent représenté par une droite graduée (infinie dans les deux sens). Chaque nombre réel correspond à un unique point sur cette droite. On retrouve les entiers, décimaux, rationnels, et aussi les irrationnels comme $\sqrt{2}$ ou $\pi$.
