📊 Leçon 6 : Insertion des graphiques et impression d'une feuille de calcul

Informatique – Première (A, ABI, AC, SH) – Module 1 : Environnement numérique et tableur

📌 Éléments clés à retenir

🔹 1. Pourquoi insérer un graphique ?

Un graphique permet de visualiser des données chiffrées de manière claire et rapide. Il transforme une série de nombres en une représentation graphique (barres, courbes, secteurs, etc.) facilitant l’analyse, la comparaison et la prise de décision.

  • Utilité : Rendre les données plus parlantes (ex: évolution des notes, répartition des dépenses).
  • Types courants : Graphique à colonnes, à barres, circulaire (secteurs), linéaire (courbes).
  • Éléments d’un graphique : Titre, axes (abscisses / ordonnées), légende, série de données.

🔹 2. Étapes d'insertion d'un graphique (Exemple : MS Excel / LibreOffice Calc)

Pour créer un graphique pertinent, il faut d’abord sélectionner les données (plages de cellules), puis utiliser l’assistant graphique.

  • 1. Sélectionner les données : Inclure les étiquettes (titres des lignes/colonnes) et les valeurs.
  • 2. Insérer le graphique : Menu « Insertion » → choisir un type (histogramme, courbe, secteur…).
  • 3. Personnaliser : Ajouter un titre, légendes, modifier les couleurs, étiqueter les axes.
  • 4. Déplacer / redimensionner : Le graphique est un objet intégré à la feuille de calcul.
💡 Astuce examen : On vous demandera souvent de créer un graphique à partir d’un tableau de notes ou de ventes. N’oubliez jamais d’inclure les EN-TÊTES de colonnes/lignes pour que la légende soit automatique.

🔹 3. Impression d'une feuille de calcul

Imprimer un tableau ou un graphique nécessite quelques réglages pour que le document soit propre et lisible.

  • Aperçu avant impression : Permet de visualiser le rendu et d’ajuster les marges.
  • Zone d’impression : Définir une plage précise à imprimer (sinon tout le classeur ou la feuille active).
  • Réglages importants : Orientation (portrait/paysage), échelle (ajuster à une page), en-têtes/pieds de page.
  • Imprimer un graphique seul : Sélectionner le graphique → Fichier → Imprimer → « Imprimer la sélection ».

💡 Exemples de compréhension

Exemple 1 : Créer un graphique à colonnes des moyennes trimestrielles

Contexte : Un professeur a relevé les moyennes d’un élève sur les trois trimestres : T1 = 12,5 ; T2 = 14 ; T3 = 15,5. Il souhaite visualiser la progression.

Résolution étape par étape :
1. Saisir dans les cellules : A1="Trimestre", B1="Moyenne" ; A2="T1", B2=12,5 ; A3="T2", B3=14 ; A4="T3", B4=15,5.
2. Sélectionner la plage A1:B4.
3. Aller dans le menu « Insertion » → Graphique → Choisir « Histogramme » (colonnes).
4. Ajouter un titre : « Progression des moyennes », légende axe des X « Trimestres », axe des Y « Notes sur 20 ».
5. Le graphique apparaît : chaque trimestre a une barre dont la hauteur correspond à la moyenne.

💬 Astuce : Si les données changent, le graphique se met automatiquement à jour (liaison dynamique).

Exemple 2 : Imprimer proprement un tableau de notes

Vous avez une feuille de calcul avec 30 lignes d’élèves et 8 colonnes (Nom, Maths, PC, Anglais, Moyenne). Vous voulez imprimer seulement le tableau (sans les colonnes vides) sur une seule page.

Procédure :
- Sélectionner la plage de cellules contenant les données (ex: A1:H31).
- Onglet « Mise en page » → « Zone d’impression » → « Définir la zone d’impression ».
- Aller dans « Aperçu avant impression » → Choisir « Ajuster la feuille à une page » dans l’échelle.
- Vérifier l’orientation : paysage si beaucoup de colonnes.
- Ajouter un en-tête (nom de la classe, date) via « Mise en page » → « En-tête/pied de page ».
- Cliquer sur « Imprimer ».

💡 Pensez à utiliser « Aperçu des sauts de page » pour voir où le tableau sera coupé.

Exemple 3 : Graphique circulaire (secteurs) pour des parts de budget

Une association dispose d’un budget mensuel : Nourriture 150 000 FCFA, Transport 80 000 FCFA, Loisirs 45 000 FCFA, Épargne 25 000 FCFA. On veut visualiser la répartition.

Méthode : Saisir les catégories (colonne A) et les montants (colonne B). Sélectionner les deux colonnes → Insertion → Graphique circulaire (camembert). Le tableur affiche des parts proportionnelles. On peut ajouter des étiquettes de pourcentage. Utile pour comparer la part de chaque poste.

🥧 Un graphique circulaire est idéal pour montrer des proportions (total = 100%). Ne pas l’utiliser avec trop de catégories.

⚠️ Pièges courants & erreurs à éviter

  • Erreur n°1 : Sélectionner des données non contiguës ou oublier les étiquettes → le graphique n’aura pas de légende claire.
  • Erreur n°2 : Ne pas vérifier l’aperçu avant impression et imprimer des colonnes vides → perte de papier et illisibilité.
  • Attention : Un graphique n’est pas modifiable comme un dessin ; il faut modifier les données source pour qu’il évolue.
  • Piège fréquent : Utiliser un type de graphique inadapté (ex: courbe pour des catégories non ordonnées). Préférer un histogramme pour comparer des quantités.
  • À ne pas confondre : « Imprimer la feuille » et « Imprimer le graphique sélectionné ». Par défaut, le tableur imprime toute la feuille active sauf si vous définissez une zone.

📝 Exercice type (Probatoire / Bac)

Sujet fréquent : Tableau des ventes trimestrielles

On donne le tableau suivant :
Produit | T1 | T2 | T3 | T4
Chaussures | 120 | 135 | 140 | 160
Sacs | 80 | 95 | 110 | 105

Questions possibles :
1. Insérer un graphique à colonnes groupées pour comparer les ventes par produit.
2. Ajouter un titre « Évolution des ventes 2024 ».
3. Définir une zone d’impression pour n’imprimer que le tableau et le graphique.
4. Donner la procédure pour imprimer en orientation paysage.

Correction rapide :
- Sélectionner les données (A1:E3).
- Insertion → Graphique à colonnes groupées.
- Modifier le titre via « Éléments de graphique ».
- Zone d’impression : sélectionner la plage incluant tableau + graphique, puis Mise en page → Zone d’impression → Définir.
- Orientation : Aperçu avant impression → Mise en page → Paysage.

🎯 Conseil pré-examen : Maîtrisez les captures d’écran et les étapes clés (sélectionner les données → insérer graphique → personnaliser → imprimer).

📖 Synthèse pour la révision

  • Graphique : représentation visuelle d’une série de données. Types : histogramme, courbe, circulaire.
  • Insertion : Données sélectionnées → Insertion → Graphique → choix du type → personnalisation.
  • Impression professionnelle : Aperçu avant impression, orientation, marges, zone d’impression, en-têtes.
  • Définir zone d’impression : Évite d’imprimer toute la feuille et économise le papier.
  • Différence : Un graphique est lié aux données source. Si vous modifiez les nombres, le graphique évolue automatiquement.
⭐ Ce qu’il faut absolument savoir faire (compétence évaluée) :
- Créer un graphique à partir d’un tableau donné.
- Modifier le titre du graphique et des axes.
- Choisir le bon type de graphique selon le contexte.
- Paramétrer l’impression pour qu’un document soit lisible (orientation, échelle).
- Sauvegarder la feuille avant impression.
