UE 18 : Quelques actions sur les fichiers images

Informatique · Première A, ABI, AC, SH · Module 3 – Infographie

📌 Opérations infographiques courantes

1. Recadrage (Crop)

Le recadrage permet de supprimer les parties inutiles ou indésirables d’une image pour recentrer l’attention sur le sujet principal. Il modifie également le format et la composition de l’image.

  • À savoir : le recadrage ne change pas la résolution globale mais réduit le nombre de pixels (l’image devient plus légère).
  • Raccourci courant : outil "Rogner" ou touche C dans Photoshop / GIMP.

Illustration : recadrage pour éliminer les bords inutiles et améliorer la composition.

2. Correction des imperfections

Supprimer les petits défauts d’une image : poussières, taches, boutons, fils parasites, etc. Plusieurs outils existent selon le logiciel.

  • Outil correcteur (spot healing) : remplace la zone à corriger par des pixels environnants.
  • Clone tampon : copie une zone source pour masquer un défaut précis.
  • Réduction du bruit : atténue le grain numérique (souvent utile pour les photos prises en faible luminosité).

3. Ajustements de couleurs et de luminosité

Modifier les paramètres visuels d’une image pour améliorer son rendu ou lui donner un style particulier.

  • Luminosité / Contraste : éclaircir ou assombrir globalement, renforcer les différences de tons.
  • Balance des blancs : corriger les dominantes de couleur (ex: photo trop bleue ou trop jaune).
  • Saturation : intensité des couleurs (une haute saturation = couleurs vives).

Fenêtre de réglages typique (ex: Niveaux, Courbes).

4. Redimensionnement

Changer les dimensions (largeur × hauteur) d’une image. Attention : agrandir trop fortement peut dégrader la qualité (pixelisation).

  • Pour le web, on réduit souvent la taille pour accélérer l’affichage.
  • Le redimensionnement doit conserver le ratio pour éviter l’écrasement (maintenir proportions).
  • Unités possibles : pixels, pouces, pourcentages.
💡 Astuce : Dans GIMP ou Photoshop, utiliser "Image > Taille de l'image" et cocher la chaîne pour garder les proportions.

5. Rotation et retournement

Redresser une image prise de travers, ou créer un effet miroir.

  • Rotation libre : pour corriger l’horizon (outil "rotation" ou "redresser").
  • Rotation à 90°, 180° : changer l’orientation.
  • Symétrie horizontale ou verticale (effet miroir).

💡 Exemples d’actions pratiques

Exemple 1 : Recadrer une photo de classe pour supprimer les bords vides

Vous avez pris une photo de groupe où des tables vides apparaissent à gauche et à droite. Vous souhaitez recentrer sur les élèves.

Étapes avec un logiciel libre (GIMP) :
1. Ouvrir l’image (Fichier → Ouvrir).
2. Activer l’outil Recadrage (icône cutter ou touche Maj+C).
3. Dessiner un rectangle autour des élèves.
4. Ajuster les poignées, puis valider (Entrée).
5. Exporter au format PNG ou JPEG.

💬 Astuce : Toujours conserver l’original avant de recadrer (travailler sur une copie).

Exemple 2 : Supprimer une tache sur un ciel

Une photo de paysage présente une petite tache sombre (poussière sur l’objectif) dans le ciel.

Méthode rapide avec l'outil correcteur (Photoshop / Photopea / GIMP) :
1. Zoomer sur la zone tachée.
2. Choisir l’outil "Correcteur" ou "Tampon de clonage".
3. Avec Ctrl+clic (sous Windows) ou Cmd+clic (Mac) sur une zone propre à côté.
4. Cliquer ou peindre sur la tache pour la remplacer.
✅ Résultat : la tache disparaît et les pixels environnants se mélangent naturellement.

Exemple 3 : Améliorer la luminosité d’une photo sous-exposée

Une photo prise en intérieur est trop sombre ; on veut l’éclaircir sans perdre les détails.

Utilisation des niveaux :
- Ouvrir le menu "Couleurs → Niveaux" (GIMP) ou "Image → Réglages → Niveaux" (Photoshop).
- Déplacer le curseur blanc vers la gauche pour éclaircir les hautes lumières.
- Ajuster le curseur noir pour renforcer les ombres.
- Valider et observer la différence.

🛠️ Logiciels de retouche d’image (rappel)

D'après l’UE 16, voici les logiciels permettant d’effectuer les actions ci-dessus :

  • Propriétaire Adobe Photoshop – standard de l’industrie (fonctions avancées).
  • Libre et gratuit GIMP (GNU Image Manipulation Program) – excellent pour l’apprentissage.
  • En ligne Photopea – similaire à Photoshop, fonctionne dans un navigateur.
  • Intégré OS Paint (Windows) ou Preview (Mac) – fonctions basiques (recadrage, rotation, redimensionnement).

⚠️ Erreurs fréquentes & conseils de pro

  • Perdre l’original : Toujours conserver l’image source intacte. Travailler sur une copie ou utiliser "Enregistrer sous".
  • Agrandir excessivement : agrandir une petite image la rend floue ou pixelisée (perte de qualité irréversible).
  • Négliger le format d’export : Pour une retouche finale, privilégier PNG (sans perte) ou JPEG (compression avec perte modérée pour le web).
  • Mauvais choix d’outil : Utiliser le tampon de clonage pour des zones uniformes peut créer des répétitions visibles. Préférer le correcteur pour les petites imperfections.
  • Oublier les calques : Travailler avec des calques permet d’annuler/modifier une retouche sans tout recommencer.

⌨️ Mémo express (sous Photoshop / GIMP)

Recadrer : C puis valider Entrée
Tampon de clonage : S (GIMP), Maj+clic pour définir source
Zoom avant/arrière : Ctrl + + / Ctrl + -
Annuler : Ctrl + Z
Rotation libre : Outil rotation (touche Maj+R dans GIMP)
Enregistrer pour le web : Ctrl+Maj+S (Photoshop)

📝 Ce qu’il faut absolument retenir (pour l’examen)

  • ✔️ Recadrer = supprimer des zones périphériques → recentrer l’image.
  • ✔️ Corriger les imperfections = outils correcteur / tampon de clonage.
  • ✔️ Ajuster les couleurs = luminosité, contraste, saturation, balance des blancs.
  • ✔️ Redimensionner = changer largeur/hauteur (attention à la perte de qualité si agrandissement).
  • ✔️ Rotation / symétrie = redresser ou créer un effet miroir.
  • ✔️ Logiciels phares : Photoshop (payant), GIMP (gratuit).
  • ✔️ Règle d’or : toujours conserver l’image originale et travailler sur une copie.
