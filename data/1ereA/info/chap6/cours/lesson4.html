📽️ Leçon 4 : Introduction à la PréAO
Cas de MS PowerPoint

Informatique · Première (A, ABI, AC, SH) · Module 1 : Environnement numérique

📌 Éléments clés à retenir

1. Qu'est-ce que la PréAO ?

La PréAO (Préparation Assistée par Ordinateur) désigne l'ensemble des techniques et logiciels permettant de créer des présentations électroniques (diaporamas) pour accompagner un exposé, une formation ou une réunion. Ces outils aident à structurer l'information et à la rendre plus attrayante.

  • Objectif : Communiquer efficacement un message à l’aide de supports visuels (texte, images, graphiques, vidéos, sons).
  • Exemples de logiciels : Microsoft PowerPoint, LibreOffice Impress, Google Slides, Canva, Prezi.
  • Usages : Exposés scolaires, réunions professionnelles, conférences, formations, soutenances de projet.

2. Structure d'une présentation PowerPoint

Une présentation est constituée d’un ensemble de diapositives (slides). Chaque diapositive peut contenir différents éléments (textes, images, formes, tableaux, graphiques, etc.).

  • Diapositive de titre : Présente le sujet et l’auteur.
  • Diapositives de contenu : Développent les idées principales (souvent avec titres, listes à puces, images).
  • Diapositive de conclusion / remerciements.
🖼️ Structure visuelle :
Titre ➡️ Introduction ➡️ Développement 1 ➡️ Développement 2 ➡️ Conclusion

3. Fonctions de base dans MS PowerPoint

Pour créer une présentation efficace, il faut maîtriser les actions suivantes :

  • Créer une nouvelle présentation : Fichier → Nouveau → Présentation vierge ou utiliser un modèle.
  • Ajouter / supprimer / dupliquer une diapositive : Utiliser le volet des diapositives ou le raccourci Ctrl+M (nouvelle diapositive).
  • Insérer des objets :
    • Texte : Zones de texte (Insertion → Zone de texte).
    • Images : Insertion → Images → Source (fichier ou en ligne).
    • Formes, SmartArt, graphiques, tableaux, vidéos, sons.
  • Appliquer un thème ou un style : Pour une charte graphique cohérente (couleurs, polices, effets).
  • Mettre en page : Utiliser les dispositions prédéfinies (Titre + contenu, Deux colonnes, etc.).
  • Lancer le diaporama : F5 (depuis le début) ou Shift+F5 (depuis la diapositive courante).

4. Animer un objet dans une diapositive

L’animation permet de rendre la présentation plus dynamique en contrôlant l’apparition, la disparition ou le mouvement des éléments (texte, images, formes) sur la diapositive.

  • Types d’animations :
    • Animation d’entrée : L’objet apparaît (ex : Apparition, Fondu, Vol).
    • Animation d’accentuation : L’objet est mis en valeur (ex : Rebond, clignotement).
    • Animation de sortie : L’objet disparaît (ex : Disparition, Fondu).
    • Trajectoire de déplacement : L’objet se déplace selon un chemin défini.
  • Comment animer un objet ?
    1. Sélectionner l’objet (texte, image, forme).
    2. Onglet Animations.
    3. Choisir un effet dans la galerie.
    4. Paramétrer les options d’effet (déclencheur, durée, délai).
    5. Utiliser le volet d’animation pour ordonner les animations.
  • Exemple : Faire apparaître une image en fondu au clic, puis la faire rebondir.
🎬 Exemple d'animation : "Au clic, le titre entre par le haut avec un effet de vol".
Clic souris ➡️ Animation d'entrée "Vol" ➡️ Durée 0,5s

5. Mise en place des transitions

Les transitions sont des effets visuels qui se produisent lors du passage d’une diapositive à la suivante. Contrairement aux animations (qui agissent sur les objets d’une même diapositive), les transitions s’appliquent à la diapositive entière.

  • Exemples de transitions : Fondu, Pousse, Volet, Traîneau, Coupure.
  • Paramètres : Durée, son d’accompagnement, déclenchement (au clic ou automatique après un délai).
  • Application : Onglet Transitions → Choisir un effet → Appliquer à toutes les diapositives si souhaité.

💡 Exemples de compréhension

Exemple concret n°1 : Créer une présentation simple sur "Les volcans"

Vous devez préparer une présentation de 5 diapositives pour un exposé en géographie. Voici les étapes à suivre avec PowerPoint :

Résolution / Explication :
  1. Ouvrir PowerPoint → Nouvelle présentation vierge.
  2. Diapositive 1 (titre) : Saisir "Les volcans" comme titre et votre nom comme sous-titre.
  3. Diapositive 2 (définition) : Utiliser la disposition "Titre + contenu". Titre : "Qu'est-ce qu'un volcan ?". Contenu : texte + image d'un volcan.
  4. Diapositive 3 (types) : Titre : "Types de volcans". Contenu : liste à puces (volcan rouge, volcan gris).
  5. Diapositive 4 (risques) : Insérer une image de coulée de lave et une courte vidéo si possible.
  6. Diapositive 5 (conclusion) : Résumer les idées principales.
  7. Appliquer un thème cohérent (onglet Conception).
  8. Ajouter des animations aux titres pour les faire apparaître en fondu.
  9. Lancer le diaporama avec F5.
💬 Astuce : Utilisez les dispositions prédéfinies pour gagner du temps et assurer une bonne mise en page.

Exemple d'application n°2 : Animer un diagramme dans une diapositive

Vous avez un graphique représentant les ventes trimestrielles. Vous souhaitez que chaque barre du graphique apparaisse l'une après l'autre au clic.

Procédure :
  1. Sélectionner le graphique.
  2. Onglet Animations → Choisir "Apparition" (ou un autre effet).
  3. Cliquer sur Volet d'animation.
  4. Dans le volet, cliquer sur la flèche déroulante de l'animation → Options d'effet.
  5. Onglet "Animation du graphique" → Choisir "Par série" ou "Par élément de la série".
  6. Valider. Désormais, chaque barre du graphique s'affiche au clic suivant.
📊 Résultat : Clic 1 → Barre T1 apparaît ; Clic 2 → Barre T2 apparaît ; etc.
💡 Pour une animation professionnelle, évitez les effets trop longs ou trop exubérants qui distraient l'audience.

Exemple n°3 : Insérer une vidéo YouTube dans une diapositive

Vous voulez montrer une courte vidéo explicative pendant votre diaporama, sans quitter PowerPoint.

Méthode (PowerPoint 2016 et ultérieur) :
  1. Aller dans l'onglet Insertion → Vidéo → Vidéo en ligne.
  2. Copier l'URL de la vidéo YouTube (ex : https://youtu.be/...).
  3. Coller l'URL et cliquer sur "Insérer".
  4. Redimensionner la vidéo sur la diapositive.
  5. Pendant le diaporama, cliquer sur la vidéo pour la lire (une connexion Internet est requise).

Alternative : Télécharger la vidéo au préalable (MP4) et utiliser Insertion → Vidéo → Ce périphérique.

⚠️ Attention : Vérifiez toujours votre connexion Internet et les droits d'auteur avant d'utiliser une vidéo en ligne.

⚠️ Pièges courants / À ne pas confondre

  • Confondre animation et transition : L'animation s'applique aux objets d'UNE diapositive ; la transition s'applique entre DEUX diapositives.
  • Surcharger la diapositive : Mettre trop de texte ou trop d'images rend la présentation illisible. Privilégiez les mots-clés et les visuels simples.
  • Oublier les raccourcis utiles : F5 pour démarrer, Esc pour quitter, Espace ou Entrée pour avancer.
  • Utiliser trop d'animations différentes : Mélanger 10 effets différents donne un rendu amateur. Préférez 2 ou 3 effets cohérents sur l'ensemble.
  • Négliger le contraste des couleurs : Un texte clair sur fond clair est illisible. Vérifiez le contraste avant de présenter.
  • Ne pas vérifier la compatibilité : Une vidéo ou police spéciale peut ne pas fonctionner sur un autre ordinateur. Exportez au format PDF ou utilisez "Emporter un CD" pour regrouper les ressources.

🎯 Conseils pour une présentation percutante

Règle des 5x5 / 7x7

Pour une diapositive lisible : maximum 5 à 7 lignes et maximum 5 à 7 mots par ligne. Cela évite de noyer l’audience.

Utilisation des images pertinentes

Une image vaut mille mots. Choisissez des images de qualité, libres de droits (ex : Pixabay, Unsplash) et en rapport direct avec le propos.

Préparation au diaporama

  • Répétez votre discours avant la présentation.
  • Utilisez le mode Présentateur (si disponible) pour voir vos notes en privé.
  • Prévoyez une solution de secours (PDF sur clé USB, version papier).
