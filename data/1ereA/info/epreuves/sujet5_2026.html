📘 Partie 1 Maîtrise des connaissances (8 points)
Exercice 1
(3 pts)
1. Définissez les termes suivants : Algorithme, Variable, Structure alternative. (1,5 pt)
2. Expliquez la différence entre une structure conditionnelle Si... Alors... Sinon... et une boucle Pour... Faire.... (1,5 pt)

✅ Correction

1. Définitions :
- Algorithme : Suite finie et ordonnée d'opérations ou d'instructions permettant de résoudre un problème.
- Variable : Zone mémoire nommée qui peut contenir une donnée susceptible d'être modifiée au cours de l'exécution d'un algorithme.
- Structure alternative : Structure de contrôle qui permet d'exécuter un bloc d'instructions si une condition est vraie, et éventuellement un autre bloc si elle est fausse.

2. Différence : La structure Si... Alors... Sinon... permet d'exécuter des instructions selon qu'une condition est vraie ou fausse (exécution unique). La boucle Pour... Faire... permet de répéter un bloc d'instructions un nombre déterminé de fois (itération).

Exercice 3
(2 pts)
Dites si les affirmations suivantes sont VRAIES ou FAUSSES. Justifiez brièvement.
a) Une instruction d'affectation permet de stocker une valeur dans une variable.
b) En HTML, la balise <br> possède une balise fermante </br> obligatoire.
c) L'organigramme est une représentation graphique d'un algorithme.
d) Un site web est un ensemble de pages web liées entre elles par des hyperliens.

✅ Correction

a) VRAI – L'affectation (ex: x ← 5) enregistre une donnée en mémoire.
b) FAUX – La balise <br> est une balise auto-fermante (void element) en HTML.
c) VRAI – L'organigramme utilise des symboles normalisés pour décrire un algorithme.
d) VRAI – Un site web est structuré en pages reliées par des hyperliens.

⚙️ Partie 2 Algorithmique & Structures de contrôle (7 points)
Exercice 4
(3,5 pts)
On considère l'algorithme suivant :
Variables A, B, C : entiers
Début
    A ← 5
    B ← 8
    C ← A + B
    Afficher "La somme est ", C
    Si C > 10 Alors
        A ← C - 2
        B ← B + 1
    Sinon
        A ← C + 3
    FinSi
    Afficher A, B
Fin
1. Faites la trace d'exécution de cet algorithme (état des variables après chaque instruction).
2. Quelles sont les valeurs finales affichées ?

✅ Trace & Résultat

Trace :
- A=5, B=8 → C = 5+8 = 13
- Affichage : "La somme est 13"
- Condition C > 10 (13>10) : VRAIE → A = 13-2 = 11 ; B = 8+1 = 9
- Fin Si → Affichage : A=11, B=9

Valeurs finales affichées : "La somme est 13" puis 11 et 9.

Exercice 5
(3,5 pts)
Un professeur souhaite calculer la moyenne de trois notes (N1, N2, N3) et afficher si l'élève est "ADMIS" (moyenne ≥ 10) ou "NON ADMIS".
Écrivez un algorithme en pseudo-code ou sous forme d'organigramme (vous pouvez décrire les symboles).

✅ Proposition d'algorithme

Variables N1, N2, N3, Moy : réels
Début
    Ecrire "Entrer les trois notes"
    Lire N1, N2, N3
    Moy ← (N1 + N2 + N3) / 3
    Si Moy >= 10 Alors
        Afficher "ADMIS"
    Sinon
        Afficher "NON ADMIS"
    FinSi
Fin

Organigramme : Début → Lecture N1,N2,N3 → Calcul Moyenne → Condition (Moy >=10) ? → Oui : Admis / Non : Non Admis → Fin.

🌐 Partie 3 Développement Web - HTML (5 points)
Exercice 6
(2,5 pts)
Écrivez le fragment de code HTML permettant d’insérer :
1. Une image nommée "cameroun.jpg" située dans le dossier "images", avec un texte alternatif "Paysage Cameroun". (1 pt)
2. Un lien hypertexte vers le site "https://www.education.cm" qui s'ouvre dans un nouvel onglet, et dont le texte cliquable est "Ministère de l'Éducation". (1,5 pt)

✅ Code HTML

<!-- 1. Insertion de l'image -->
<img src="images/cameroun.jpg" alt="Paysage Cameroun">

<!-- 2. Lien externe avec target _blank -->
<a href="https://www.education.cm" target="_blank">Ministère de l'Éducation</a>

L'attribut target="_blank" permet d'ouvrir le lien dans un nouvel onglet.
