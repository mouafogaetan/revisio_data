🎥 Leçon 20 : Traitement des vidéos

Informatique · Première (A, ABI, AC, SH) · Module 3 – Infographie, multimédia & usages socioculturels

📌 Éléments clés à retenir

🎞️ 1. Formats de fichiers vidéo

Un format vidéo définit la manière dont les données (images + son) sont encodées et compressées dans un fichier. Les formats les plus répandus permettent un bon équilibre entre qualité et taille du fichier.

  • MP4 (MPEG-4 Part 14) : format universel, idéal pour le web, smartphones, et réseaux sociaux. Bon compromis qualité/poids.
  • AVI (Audio Video Interleave) : format plus ancien, parfois non compressé → fichiers très lourds, mais bonne compatibilité.
  • MOV : développé par Apple (QuickTime), très utilisé en montage professionnel.
  • MKV (Matroska) : conteneur flexible, peut intégrer plusieurs pistes (sous-titres, audios).
  • WEBM : conçu pour le web, souvent utilisé pour les vidéos HTML5.

✂️ 2. Logiciels de montage vidéo

Le montage vidéo consiste à assembler, couper, ajouter des effets, des transitions, des titres et du son pour produire une vidéo finale. Voici des logiciels adaptés aux débutants et aux environnements scolaires :

  • 🎬 Windows Movie Maker (ancien mais classique) : simple, interface intuitive, idéal pour apprendre les bases (couper, ajouter musique, transitions).
  • ⚡ OpenShot : logiciel libre, multiplateforme (Windows, Mac, Linux), puissant pour un usage pédagogique.
  • 🎞️ Shotcut : open source, professionnel mais accessible, supporte de nombreux formats.
  • 🍎 iMovie (sur Mac/iOS) : très ergonomique, intégré dans l'écosystème Apple.
  • 📱 CapCut / InShot : applications mobiles simples pour montage rapide (réseaux sociaux).

✅ À retenir : Un logiciel de montage permet de découper des scènes, ajouter des transitions (fondu, fondu enchaîné), insérer du texte, des filtres, et exporter dans le format souhaité.

📋 3. Étapes clés de la production vidéo

Pour réaliser une vidéo de qualité, il faut suivre un processus méthodique (du projet brut au fichier final).

  • 1. Pré-production : écriture du scénario, storyboard, plan du tournage, repérage du matériel (caméra, micro, éclairage).
  • 2. Capture / Acquisition : tournage des séquences vidéo (smartphone, caméra, webcam).
  • 3. Montage : importation des rushs dans un logiciel → découpage, réorganisation, suppression des erreurs, ajout de transitions.
  • 4. Ajout d'éléments : musique, voix off, sous-titres, effets visuels, texte (générique).
  • 5. Exportation / Rendu : choisir le format (MP4 recommandé), la résolution (720p ou 1080p) et compresser pour diffusion.
  • 6. Diffusion / Partage : upload sur YouTube, réseau social, clé USB ou lecteur Cloud.
🖼️ Schéma du workflow de production vidéo : Pré-prod → Tournage → Montage → Export → Partage

💡 Exemples de compréhension

📱 Exemple 1 : Quel format pour partager sur WhatsApp ?

Contexte : Aminata a filmé un exposé avec son smartphone. Elle souhaite envoyer la vidéo à son professeur via WhatsApp et la conserver dans son Cloud sans trop de perte de qualité.

Résolution : Elle doit exporter son montage au format MP4 avec une compression H.264. Ce format est universel, léger et conserve une qualité acceptable même pour le partage mobile. Éviter AVI (trop lourd) ou MOV (pas toujours compatible sur Android).

💬 Astuce : Sur la plupart des logiciels de montage, le préréglage "YouTube 1080p" ou "MP4 HD" est parfait pour un usage scolaire.

✂️ Exemple 2 : Étapes de montage avec Windows Movie Maker (ou OpenShot)

Objectif : Créer un mini-reportage de 2 minutes à partir de 3 clips bruts et d'une chanson libre de droits.

Procédure pas à pas :
1. Importer les clips vidéo et le fichier audio dans le logiciel.
2. Glisser les séquences sur la timeline dans l'ordre souhaité.
3. Découper les parties inutiles (début/fin) avec l'outil "couper".
4. Ajouter une transition "fondu enchaîné" entre chaque clip pour un rendu fluide.
5. Ajouter un titre au début et un générique de fin.
6. Exporter au format MP4 (720p) pour un bon compromis taille/qualité.

🎯 À retenir : La timeline est l'espace de travail principal en montage vidéo. Plus on s'entraîne, plus on gagne en rapidité !

🌐 Exemple 3 : Vidéo en ligne – quel format pour une page HTML ?

Un développeur web souhaite intégrer une vidéo dans un site avec la balise HTML5 <video>.

Formats recommandés : MP4 (avec codec H.264) + WEBM (pour compatibilité maximale). Le navigateur choisira automatiquement le format supporté. AVI et MOV ne sont pas directement lisibles par les navigateurs sans plugin.

💡 Lien avec le module HTML : en cours d'informatique, la balise <video src="fichier.mp4" controls></video> permet d'insérer une vidéo.

⚠️ Pièges courants & erreurs à éviter

  • ❌ Confondre format conteneur et codec : MP4 est un conteneur, H.264 est le codec (compression). Mais en pratique, on retient que le format MP4 est le plus sûr.
  • 📁 Fichier trop lourd : Oublier de compresser la vidéo avant de l'envoyer par mail ou sur un réseau social. Toujours vérifier la taille et choisir une résolution adaptée (720p souvent suffisante).
  • 🎚️ Négliger le son : Une vidéo avec un mauvais son (grésillements, trop faible) nuit à la compréhension. Penser à ajuster le volume et supprimer les bruits parasites lors du montage.
  • ⚠️ Droits d’auteur : Utiliser une musique ou des extraits vidéo protégés sans autorisation. Pour les projets scolaires, privilégier les bibliothèques libres de droits (YouTube Audio Library, Pixabay).
  • 💾 Sauvegarde des rushs : Beaucoup d'élèves perdent leurs projets car ils n’enregistrent pas le fichier source du montage (projet .wlmp, .osp, etc.) → sauvegarder à la fois le projet modifiable ET la vidéo finale exportée.

✅ Mémo des bonnes pratiques (examen & TP)

  • 🎯 Pour l'épreuve théorique : connaître au moins 3 formats vidéo (MP4, AVI, MOV) et 2 logiciels de montage (Windows Movie Maker, OpenShot).
  • 🔁 Savoir expliquer les étapes de production : pré-production (scénario) → capture → montage → export → diffusion.
  • 📽️ Différence montage linéaire / non-linéaire : Aujourd'hui tous les logiciels sont non-linéaires (on peut modifier n'importe quelle partie sans refaire tout).
  • 📤 Export : toujours choisir MP4 pour une compatibilité maximale (ordinateur, téléphone, lecteur web).

📖 Glossaire express (traitement vidéo)

  • Rush : séquence brute non montée.
  • Timeline : ligne du temps où on organise les clips dans un logiciel de montage.
  • Transition : effet entre deux plans (fondu, fondu enchaîné, volet).
  • Codec : algorithme de compression/décompression (ex: H.264).
  • Résolution : nombre de pixels (hauteur x largeur). 1080p = 1920x1080 pixels.
