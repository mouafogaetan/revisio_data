Chapitre 6 : Statistique descriptive

Mathématiques · Première A, ABI, AC, SH · Année scolaire 2025-2026

📌 Éléments clés à retenir

1. Organisation des données (série continue)

Une série statistique continue est présentée par classes (intervalles) : \([a_1 ; a_2[, [a_2 ; a_3[, ...\). Pour chaque classe, on connaît l'effectif (\(n_i\)) ou la fréquence. On utilise le centre de classe (\(c_i = \frac{a_i + a_{i+1}}{2}\)) pour les calculs de la moyenne.

  • Centre de classe : \(c_i = \dfrac{\text{borne inf} + \text{borne sup}}{2}\)
  • Effectif total : \(N = \sum n_i\)
  • Fréquence : \(f_i = \frac{n_i}{N}\) (souvent en %)
  • Les effectifs cumulés croissants (ECC) s'obtiennent en additionnant les effectifs de la première classe jusqu'à la classe \(i\).
  • Les effectifs cumulés décroissants (ECD) s'obtiennent en additionnant les effectifs de la dernière classe jusqu'à la classe \(i\).

2. Paramètres de position

Ils permettent de résumer la série par une valeur centrale.

  • Moyenne (\(\bar{x}\)) : \(\displaystyle \bar{x} = \frac{\sum n_i c_i}{N}\) (avec \(c_i\) centre de classe).
  • Médiane (Me) : Valeur qui partage la série en deux parties d'effectifs égaux. Pour une série continue, on détermine la classe médiane (celle où l'ECC dépasse \(N/2\)) et on peut lire la médiane graphiquement ou par interpolation linéaire.
  • Mode : Classe ayant le plus grand effectif (classe modale).
💡 Astuce : Pour les exercices du Probatoire, la médiane est souvent demandée par lecture graphique sur le polygone des effectifs cumulés croissants.

3. Paramètres de dispersion

Ils mesurent l'étalement des valeurs autour de la moyenne.

  • Variance (\(V\)) : \(\displaystyle V = \frac{\sum n_i (c_i - \bar{x})^2}{N}\)
  • Écart-type (\(\sigma\)) : \(\sigma = \sqrt{V}\) (même unité que les données).
  • Étendue : \( \text{Max} - \text{Min} \) (différence entre la borne sup de la dernière classe et la borne inf de la première).

4. Représentations graphiques

Les graphiques sont essentiels pour visualiser la répartition des données.

  • Histogramme : Diagramme en bâtons pour les classes (aires proportionnelles aux effectifs).
  • Polygone des effectifs (ou fréquences) : On relie les points \((c_i, n_i)\) ou \((c_i, f_i)\).
  • Polygone des effectifs cumulés croissants : Points \(( \text{borne supérieure de classe}, \text{ECC} )\). Permet de déterminer la médiane.

💡 Exemples de compréhension

Exemple 1 : Calcul de la moyenne, ECC et médiane

On étudie les tailles (en cm) de 50 élèves. Les classes sont : [150; 160[, [160; 170[, [170; 180[, [180; 190[. Effectifs respectifs : 5, 15, 20, 10.

1. Centres de classes : 155, 165, 175, 185.
2. Moyenne : \(\bar{x} = \frac{5\times155 + 15\times165 + 20\times175 + 10\times185}{50} = \frac{775+2475+3500+1850}{50} = \frac{8600}{50} = 172\) cm.
3. Effectifs cumulés croissants (ECC) : 5, 20, 40, 50.
4. Médiane : \(N/2 = 25\). La classe médiane est [170; 180[ car ECC avant = 20, ECC après = 40. Par interpolation : \(Me = 170 + \frac{25-20}{20}\times10 = 170 + 2,5 = 172,5\) cm.

💬 Pour la médiane graphique, on reporte 25 sur l'axe des ECC et on lit l'abscisse correspondante.

Exemple 2 : Variance et écart-type

Reprenons l'exemple précédent. Calculez la variance et l'écart-type.

\(\bar{x} = 172\) cm.
\(V = \frac{5(155-172)^2 + 15(165-172)^2 + 20(175-172)^2 + 10(185-172)^2}{50}\)
= \(\frac{5(-17)^2 + 15(-7)^2 + 20(3)^2 + 10(13)^2}{50} = \frac{5\times289 + 15\times49 + 20\times9 + 10\times169}{50}\)
= \(\frac{1445 + 735 + 180 + 1690}{50} = \frac{4050}{50} = 81\).
\(\sigma = \sqrt{81} = 9\) cm.

📌 Un écart-type faible (par rapport à la moyenne) signifie que les données sont homogènes.

Exemple type Probatoire : Polygone des fréquences cumulées

À partir du tableau suivant (notes sur 20 d'une classe), construire le polygone des effectifs cumulés croissants et déterminer la médiane.
Classes : [0;4[, [4;8[, [8;12[, [12;16[, [16;20] ; Effectifs : 2, 5, 10, 8, 5.

ECC : 2, 7, 17, 25, 30. N=30 → N/2=15.
On place les points (4;2), (8;7), (12;17), (16;25), (20;30). On relie ces points.
La médiane correspond à l'abscisse du point d'ordonnée 15. D'après le graphique, elle se situe dans la classe [8;12[. Calcul : \(Me = 8 + \frac{15-7}{10}\times4 = 8 + 3,2 = 11,2\).

💡 Dans un polygone cumulé, on utilise toujours les bornes supérieures des classes en abscisse.

⚠️ Pièges courants / À ne pas confondre

  • Erreur fréquente : Utiliser les centres de classe pour les ECC au lieu des bornes supérieures. → Les ECC se lisent aux bornes supérieures !
  • Attention : Pour la médiane dans une série continue, ne pas prendre le centre de la classe médiane sans interpolation.
  • Confusion : La variance se calcule avec les centres de classe et la moyenne ; ne pas utiliser les effectifs bruts sans pondération.
  • Piège : Oublier de diviser par N (effectif total) dans les formules de moyenne et variance.
  • À retenir : L'écart-type a la même unité que la variable, pas la variance (unité au carré).

📐 Méthode pas à pas pour l'exercice de statistique

Étapes clés dans un exercice type (Probatoire)

  • 1. Compléter le tableau : centres de classe, effectifs cumulés croissants (ECC).
  • 2. Calculer la moyenne : \(\bar{x} = \frac{\sum n_i c_i}{N}\).
  • 3. Construire le polygone des ECC : points (borne sup ; ECC).
  • 4. Déterminer la médiane : graphiquement (ordonnée N/2) ou par interpolation.
  • 5. Calculer la variance et l'écart-type : utiliser \(\sum n_i (c_i-\bar{x})^2\) puis diviser par N et racine carrée.
✅ Astuce de rédaction : Présentez toujours un tableau clair avec les colonnes : Classes, Centre (\(c_i\)), Effectif (\(n_i\)), \(n_i c_i\), ECC, \(n_i (c_i-\bar{x})^2\). Cela facilite les calculs et évite les erreurs.

📖 Formulaire résumé (à découper ou à retenir)

Moyenne : \(\bar{x} = \frac{\sum n_i c_i}{N}\)

Variance : \(V = \frac{\sum n_i (c_i-\bar{x})^2}{N}\)

Écart-type : \(\sigma = \sqrt{V}\)

Médiane (interpolation) : \(Me = L + \frac{\frac{N}{2} - \text{ECC}_{\text{avant}}}{n_{\text{classe}}} \times a\)
(L = borne inf de la classe médiane, a = amplitude de la classe)
