📘 So / Such ... that (Conséquence)

Anglais · Première CD · Révision Probatoire

📌 Éléments clés à retenir

🔹 1. SO + adjectif/adverbe + THAT

On utilise so devant un adjectif (qualificatif) ou un adverbe, suivi de that pour exprimer une conséquence. Cela signifie "tellement ... que".

📌 Structure : So + adjectif / adverbe + that + proposition de conséquence

  • Exemple adjectif : The movie was so boring that I fell asleep.
    (Le film était tellement ennuyeux que je me suis endormi.)
  • Exemple adverbe : He ran so fast that nobody could catch him.
    (Il a couru si vite que personne n'a pu l'attraper.)
💡 So tired that she can't keep her eyes open → "tellement fatiguée qu'elle ..."

🔸 2. SUCH + (adjectif) + nom + THAT

Such s'utilise devant un nom (parfois accompagné d'un adjectif). On exprime également une conséquence : "un tel ... que" ou "une telle ... que".

📌 Structure : Such + (a/an) + (adjectif) + nom + that + conséquence

  • Avec nom singulier : It was such a hot day that we stayed indoors.
    (C'était une journée tellement chaude que nous sommes restés à l'intérieur.)
  • Avec nom pluriel / indénombrable : They are such kind people that everyone loves them.
    (Ce sont des gens tellement gentils que tout le monde les aime.)
  • Avec adjectif + nom : She has such a beautiful voice that she could be a singer.
✨ Such a wonderful view that ... → une vue tellement magnifique que ...

⚖️ 3. Différence essentielle SO / SUCH

Le choix dépend de ce qui suit la structure :

  • SO + adjectif / adverbe (sans nom) → so interesting, so quickly.
  • SUCH + nom (avec ou sans adjectif) → such a mess, such nice people.
  • Astuce mnémotechnique : "SO" modifie des qualités (adjectifs) ; "SUCH" modifie des choses/êtres (noms).

💡 Exemples de compréhension

Exemple 1 : So + adjectif (conséquence immédiate)

Phrase à compléter : "The test was __________ difficult __________ nobody passed."

✅ Correction : The test was so difficult that nobody passed.
Explication : "difficult" est un adjectif → on utilise "so". La conséquence est "personne n'a réussi".

💬 Astuce : Après "so", il n'y a JAMAIS de nom directement. Si vous voyez un adjectif nu → choisissez SO.

Exemple 2 : Such + (adjectif) + nom (pluriel / indénombrable)

Transformez en utilisant "such...that" : "The water was very polluted. Many fish died."

✅ Fusion : It was such polluted water that many fish died.
Explication : "water" est un nom indénombrable → on utilise "such" (sans "a"). L'adjectif "polluted" vient avant le nom.

Exemple 3 : Attention à l'article 'a/an' avec such

"C'était une telle surprise qu'elle a pleuré." Traduisez.

✅ Traduction : It was such a surprise that she cried.
Règle : Devant un nom singulier comptable, on met "such a/an". Ne dites pas "such surprise" sans "a" ici.

📌 Piège fréquent : On dit "so big a house" (littéraire) mais la forme courante est "such a big house". Maîtrisez "such a + adjectif + nom".

⚠️ Pièges courants / À ne pas confondre

  • Erreur 1 : ❌ "She is so a nice girl" → Correction : "She is such a nice girl" (car "nice girl" = nom).
  • Erreur 2 : ❌ "It was such cold that we stayed home" → Correction : "It was so cold that we stayed home" (cold = adjectif).
  • Erreur 3 : Oublier "that" dans la conséquence. Dans un anglais soigné, on conserve "that" bien qu'à l'oral il soit parfois omis. À l'écrit, gardez-le pour plus de clarté.
  • Confusion : "so much/many" exprime une grande quantité mais n'est pas identique à "such".
    Ex: "so much work that..." (correct) vs "such much work" (incorrect).

📝 Entraînement minute

✏️ À vous de jouer ! (Avec corrigés)

Complétez avec so ou such :

  1. He speaks _____ quickly that I can't understand him.
  2. They are _____ talented musicians that they won the contest.
  3. It was _____ a long journey that we arrived exhausted.
  4. The coffee is _____ hot that I can't drink it.
📖 Voir les réponses

1. so (quickly = adverbe)
2. such (musicians = nom pluriel)
3. such (such a long journey : nom singulier)
4. so (hot = adjectif)

🧠 Schéma récapitulatif : SO + ADJECTIF → conséquence ; SUCH + NOM → conséquence.

📖 Contexte : Rédiger une conséquence

Paragraphe modèle (thème : environnement)

"The air in the city was so polluted that many residents developed respiratory diseases. There was such a lack of clean water that people had to buy bottled water every day. The situation became so critical that the government declared a state of emergency."

🔍 Analyse :
- "so polluted" (adjectif) → conséquence sur la santé.
- "such a lack of clean water" (nom) → conséquence quotidienne.
- "so critical" (adjectif) → mesure gouvernementale.

💪 Pour le Probatoire, utilisez ces structures dans vos essais pour enrichir votre expression écrite. N'oubliez jamais "that" après l'adjectif ou le nom.
