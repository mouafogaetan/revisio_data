📘 Lesson 23 : So / Such...that

Exprimer la conséquence en anglais · Première CD · Révision Probatoire

📌 À retenir : So ... that / Such ... that

🔹 So + adjectif/adverbe + that

On utilise so devant un adjectif ou un adverbe pour exprimer une conséquence. La structure est : so + adjectif/adverbe + that + conséquence. En français, cela correspond à « tellement ... que ».

  • Adjectif : The movie was so boring that I fell asleep.
    → Le film était tellement ennuyeux que je me suis endormi(e).
  • Adverbe : He ran so fast that nobody could catch him.
    → Il a couru tellement vite que personne n’a pu l’attraper.
  • 📌 Astuce : « so » est toujours suivi d’un adjectif ou d’un adverbe (pas d’un nom).

🔸 Such + (adjectif) + nom + that

On utilise such devant un nom (parfois accompagné d’un adjectif) pour exprimer une conséquence. Structure : such + (adjectif) + nom + that. Cela signifie également « tellement ... que », mais on insiste sur le nom.

  • Avec adjectif + nom : It was such a hot day that we stayed inside.
    → C’était une journée tellement chaude que nous sommes restés à l’intérieur.
  • Sans adjectif (nom seul) : She has such talent that she will become famous.
    → Elle a tellement de talent que elle deviendra célèbre.
  • ⚠️ Attention au singulier : such a/an + adjectif + nom singulier (comme dans "such a hot day").
  • 📌 Pour les noms pluriels ou indénombrables, on dit such + adjectif + nom (sans a/an).
    Ex: They are such nice people that everyone loves them.

📊 Récapitulatif visuel

Structure Exemple Traduction
so + adj/adv + thatThe coffee is so hot that I can’t drink it.Le café est tellement chaud que je ne peux pas le boire.
such + (adj) + nom + thatIt’s such hot coffee that I need to wait.C’est un café tellement chaud que je dois attendre.

💡 Idée générale : So + adjectif (qualité) → Such + nom (ce dont on parle).

💡 Exemples commentés pour bien comprendre

Exemple 1 : So + adjectif

Contexte : Tu regardes un match de football au Cameroun. L’ambiance est incroyable.

Phrase : The stadium was so loud that we could not hear each other.
Explication : Ici, “loud” (bruyant) est un adjectif. On utilise so loud that pour montrer la conséquence : l’ambiance était tellement bruyante qu’on ne s’entendait plus.

💬 Astuce mnémotechnique : “So” + adjectif court → pense à “So good that...” (tellement bon que...).

Exemple 2 : Such + nom (avec adjectif)

Tu parles d’un ami très gentil.

Phrase : John is such a kind person that everybody wants to be his friend.
Explication : “a kind person” est un groupe nominal. On utilise such a + adjectif + nom pour dire que c’est une personne tellement gentille que tout le monde veut être son ami.

Exemple 3 : Such + nom indénombrable

En cours d’anglais, tu parles des progrès d’une élève.

Phrase : She has shown such progress that her teacher rewarded her.
Explication : “progress” est un nom indénombrable (on ne dit pas “a progress”). Pas de “a”. On garde such + nom. Elle a montré tellement de progrès que son professeur l’a récompensée.

💡 Petit truc : Si tu peux mettre “a/an” devant le nom, utilise “such a/an”. Sinon, “such” tout seul.

Exemple 4 : So + many/much (quantité)

Cas particulier très fréquent à l’examen.

Avec un nom dénombrable : He has so many books that his shelves are full. (tellement de livres que...)
Avec un nom indénombrable : There was so much noise that I couldn’t study. (tellement de bruit que...)
⚠️ Remarque : Devant “many” (pluriel) et “much” (singulier indénombrable), on utilise so, pas such.

⚠️ Pièges à éviter (très fréquents au probatoire)

  • ❌ Erreur : Utiliser “such” devant un adjectif seul sans nom.
    → Incorrect : It was such beautiful that...
    → Correct : It was so beautiful that... ou It was such a beautiful place that...
  • ❌ Erreur : Oublier “a/an” avec un nom singulier après “such”.
    → Incorrect : She is such good student that...
    → Correct : She is such a good student that...
  • ❌ Erreur : Confondre “so ... that” et “such ... that” quand on parle de quantité (many/much).
    → Incorrect : There were such many people that...
    → Correct : There were so many people that... (parce que “many” + nom dénombrable).
  • 💡 Rappel pour le BAC / Probatoire : Dans une phrase de conséquence, identifie d’abord si le mot qui suit est un adjectif (→ so) ou un nom (→ such).

✏️ Petit test : Complète avec so ou such

  1. It was ______ a wonderful concert that we applauded for ten minutes.
  2. The instructions were ______ confusing that nobody understood them.
  3. She has ______ much patience that she never gets angry.
  4. They are ______ lovely children that everyone adores them.
  5. He drives ______ fast that it’s dangerous.
📌 Voir les réponses

1. such (such a wonderful concert)
2. so (so confusing → adjectif)
3. so (so much patience → quantité)
4. such (such lovely children → nom pluriel)
5. so (so fast → adverbe)

🧠 Illustration mentale : la différence entre so (qualité) et such (objet/personne).
